CSS-i värv

Värvikood võib olla üks järgmistest:

Kuueteistkümnendvorm: #rrggbb

RGB-vorming: rgb (punane, roheline, sinine)

Nimevorming: nimi

Näide

Oranži värvi jaoks:

Kuueteistkümnendvorm: #FFA500

RGB-vorming: rgb(255,165,0)

Nimevorming: oranž

Elemendi värvi määramine

Konkreetse elemendi jaoks:

<element style="color: code;"></element>

Kõigi sama tüüpi elementide jaoks. Sisestage kood peajaotise märgendisse <style>:

<style>
   element { color: code; }
</style>

Näide

Lõigu teksti värvi muutmine punaseks:

Konkreetse lõigu värvi määramine:

<p style="color: #FF0000;">Some text ...</p>

Vaata:

Natuke teksti...

Kõigi lõikude värvi määramine

<style>
   p { color: #FF0000; }
</style>

Elemendi taustavärvi määramine

element { background: code; }

Näide

Lõigu taustavärvi määramine punaseks:

Konkreetse lõigu taustavärvi määramine:

<p style="background: #FF0000;">Some text ...</p>

Vaata:

Natuke teksti...

Kõigi lõikude taustavärvi määramine:

<style>
   p { background: #FF0000; }
</style>

Elemendi äärise värvi määramine

element { border-color: topcode rightcode bottomcode leftcode }

Näide

Lõigu äärise värvi määramine punaseks, roheliseks, siniseks ja mustaks:

Konkreetse lõigu äärise värvi määramine:

<p style="border-color: #FF0000 #00FF00 #0000FF #000000; border-style:solid">Some text ...</p>

Vaata:

Natuke teksti...

Kõigi lõikude ääriste värvi määramine:

<style>
   p { border-color: #FF0000 #00FF00 #0000FF #000000; }
</style>

CSS-i värvikoodide tööriista omadused

Meie CSS-värvikoodide tööriist võimaldab kasutajatel kasutada CSS-värvikoode. Mõned selle utiliidi silmapaistvad funktsioonid on selgitatud allpool.

Lihtsus

Meie CSS-värvikoodide tööriista kasutajasõbralik liides võimaldab teil seda teisendust koheselt teostada. Selle tööriista lihtsuse tõttu ei pea te enam omandama CSS-värvikoodide programmeerimisoskusi.

Registreerimist pole

CSS-i värvikoodide kasutamiseks ei pea te registreerimisprotsessi läbima. Selle utiliidi abil saate CSS-i värvikoodide tööriista tasuta kasutada nii mitu korda kui soovite.

Kaasaskantavus

Sellele CSS-i värvikoodidele pääseb juurde igast maailma nurgast. Te ei pea muretsema selle võrgutööriista ühilduvuse pärast, kuna saate sellega CSS-värvikoode sisestada mis tahes seadmest. Kõik, mida vajate, on Interneti-ühendus, et pääseda juurde ja kasutada seda CSS-i värvikoodide tööriista.

Säästab aega ja jõupingutusi

CSS-i värvikoodid võimaldavad teil sama ülesande kohe täita. Teil ei paluta järgida käsitsi toiminguid, kuna selle automatiseeritud algoritmid teevad töö teie eest ära.

Ühilduvus

CSS-i värvikoodide tööriist töötab suurepäraselt kõigis operatsioonisüsteemides. Olenemata sellest, kas teil on Maci, iOS-i, Androidi, Windowsi või Linuxi seade, saate seda veebiutiliiti hõlpsalt kasutada ilma probleemideta.

100% tasuta

Selle CSS-i värvikoodide tööriista kasutamiseks ei pea te registreerimisprotsessi läbima. Saate seda utiliiti kasutada tasuta ja teha piiramatult CSS-i värvikoode ilma piiranguteta.
