CSS-i navigeerimisriba

CSS-i navigeerimisriba – horisontaalne ja vertikaalne.

Horisontaalne navigeerimisriba

CSS-kood:

<style>
    #h_nav_bar a { padding:15px; font-weight:bold; float:left; }
    #h_nav_bar a:link { color:#d0d0d0; background-color:#0000a0; }
    #h_nav_bar a:visited { color:#c0c0c0; background-color:#0000a0; }
    #h_nav_bar a:hover { color:#ffffff; background-color:#000060; }
    #h_nav_bar a:active { color:#f0f0f0; background-color:#00ff00; }
</style>

HTML kood:

<div id="h_nav_bar">
    <a href="html-anchor-link.htm">Anchor link</a>
    <a href="html-link-color.htm">Link color</a>
    <a href="../mailto.htm">Email link</a>
    <a href="html-image-link.htm">Image link</a>
    <a href="html-text-link.htm">Text link</a>
</div>

Vaata:

Ankurlink Lingi värv Meili link Pildi link Teksti link

 

 

 

#link_bar a on lingi kõigi olekute stiil.

#link_bar a:link on tavalise lingi stiil.

#link_bar a: külastatud on külastatud lingi stiil.

#link_bar a:hover on hiirekursoriga lingi stiil.

#link_bar a: aktiivne on lingi stiil, kui seda hiirega vajutada.

CSS-i navigeerimisriba tööriista omadused

Meie CSS-i navigeerimisriba tööriist võimaldab kasutajatel kasutada CSS-i navigeerimisriba. Mõned selle utiliidi silmapaistvad funktsioonid on selgitatud allpool.

Lihtsus

Meie CSS-i navigeerimisriba tööriista kasutajasõbralik liides võimaldab teil seda teisendust koheselt teostada. Selle tööriista lihtsuse tõttu ei pea te enam CSS-i navigeerimisriba jaoks programmeerimisoskusi omandama.

Registreerimist pole

CSS-i navigeerimisriba kasutamiseks ei pea te registreerimisprotsessi läbima. Seda utiliiti kasutades saate CSS-i navigeerimisriba tööriista tasuta kasutada nii mitu korda kui soovite.

Kaasaskantavus

Sellele CSS-i navigeerimisribale pääseb juurde igast maailma nurgast. Te ei pea muretsema selle võrgutööriista ühilduvuse pärast, kuna saate sellega CSS-i navigeerimisriba kasutada mis tahes seadmest. Kõik, mida vajate, on Interneti-ühendus, et pääseda juurde ja kasutada seda CSS-i navigeerimisriba tööriista.

Säästab aega ja jõupingutusi

CSS-i navigeerimisriba võimaldab teil sama ülesande kohe täita. Teil ei paluta järgida käsitsi toiminguid, kuna selle automatiseeritud algoritmid teevad töö teie eest ära.

Ühilduvus

CSS-i navigeerimisriba tööriist töötab suurepäraselt kõigis operatsioonisüsteemides. Olenemata sellest, kas teil on Maci, iOS-i, Androidi, Windowsi või Linuxi seade, saate seda veebiutiliiti hõlpsalt kasutada ilma probleemideta.

100% tasuta

Selle CSS-i navigeerimisriba tööriista kasutamiseks ei pea te registreerimisprotsessi läbima. Saate seda utiliiti kasutada tasuta ja kasutada piiramatut CSS-i navigeerimisriba ilma piiranguteta.
