Kuidas teisendada RGB hex värviks

Kuidas teisendada RGB-värvist kuueteistkümnendsüsteemi värvikoodiks.

RGB värv

RGB värv on punase, rohelise ja sinise värvi kombinatsioon:

(R, G, B)

Punane, roheline ja sinine kasutavad igaüks 8 bitti, täisarvude väärtused jäävad vahemikku 0 kuni 255.

Seega on genereeritavate värvide arv:

256 × 256 × 256 = 16777216 = 100000016

hex värvikood

Kuueteistkümnendvärviline kood on 6-kohaline kuueteistkümnendsüsteem (alus 16):

RRGGBB 16

2 vasakpoolset numbrit tähistavad punast värvi.

Kaks keskmist numbrit tähistavad rohelist värvi.

Kaks parempoolset numbrit tähistavad sinist värvi.

rgb-kuueteistkümnend teisendamine

1. Teisendage punase, rohelise ja sinise väärtused kümnendkohalt kuueteistkümnendarvuks.
2. Kombineerige 3 punase, rohelise ja sinise kuueteistkümnendväärtust: RRGGBB.

Näide 1
Teisendage punane (255,0,0) hex värvikoodiks:

R = 25510 = FF16

G = 010 = 0016

B = 010 = 0016

Kuueteistkümnendiku värvikood on seega:

Hex = FF0000

Näide #2
Teisendage kuldne värv (255 215,0) kuueteistkümnendvärvikoodiks:

R = 25510 = FF16

G = 21510 = D716

B = 010 = 0016

Kuueteistkümnendiku värvikood on seega:

Hex = FFD700

Mida see RGB-hex muundur teeb?

See võtab sisendiks punase, rohelise ja sinise värvi väärtused vahemikus 0 kuni 255 ja teisendab need seejärel kuueteistkümnendsüsteemi stringiks, mida saab kasutada html/css-koodis värvide määramiseks. Fototöötlustarkvara esindab tavaliselt värve RGB-vormingus ja kui soovite kasutada oma fototöötlustarkvaras kasutatud värve oma HTML-elemendi taustana, peate hankima RGB-väärtuste kuueteistkümnendsüsteemi. See tööriist võimaldab teil need väärtused hankida.

Proovige meie uut värviotsingu tööriista.

Teisendage kuueteistkümnendväärtus RGB-ks

Võib-olla olete veebilehel näinud kuueteistkümnendkoodi ja soovite seda värvi kasutada oma fototöötlustarkvaras. Sel juhul vajate RGB väärtusi, kui teie fototöötlustarkvara ei toeta HEX väärtusi.
