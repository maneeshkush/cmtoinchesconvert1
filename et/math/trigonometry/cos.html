Koosinuse määratlus

Täisnurkses kolmnurgas ABC on siinus α, sin(α) defineeritud kui suhe nurga α külgneva külje ja täisnurga vastaskülje (hüpotenuus) vahel:

cos α = b / c

Näide

b = 3"

c = 5"

cos α = b / c = 3 / 5 = 0.6

Pöördkoosinusfunktsioon

X arkosinus on defineeritud kui x pöördkoosinusfunktsioon, kui -1≤x≤1.

Kui y koosinus on võrdne x-ga:

cos y = x

Siis on x arkosinus võrdne x pöördkoosinusfunktsiooniga, mis on võrdne y-ga:

arccos x = cos-1 x = y

Näide

arccos 1 = cos-1 1 = 0 rad = 0°
