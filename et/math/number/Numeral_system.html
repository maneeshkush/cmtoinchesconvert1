Numbrisüsteem

b - numbrisüsteemi alus

d n - n-s number

n - võib alata negatiivsest arvust, kui arvul on murdosa.

N +1 - numbrite arv

Binaarne numbrisüsteem – alus-2

Kahendarvud kasutavad ainult 0 ja 1 numbrit.

B tähistab binaarset eesliidet.

Näited:

101012 = 10101B = 1×24+0×23+1×22+0×21+1×20 = 16+4+1= 21

101112 = 10111B = 1×24+0×23+1×22+1×21+1×20 = 16+4+2+1= 23

1000112 = 100011B = 1×25+0×24+0×23+0×22+1×21+1×20 =32+2+1= 35

Kaheksandikarvude süsteem – alus-8

Kaheksandikarvudes kasutatakse numbreid vahemikus 0...7.

Näited:

278 = 2×81+7×80 = 16+7 = 23

308 = 3×81+0×80 = 24

43078 = 4×83+3×82+0×81+7×80= 2247

Kümnendarvude süsteem – alus-10

Kümnendarvude puhul kasutatakse numbreid 0...9.

Need on tavalised numbrid, mida me kasutame.

Näide:

253810 = 2×103+5×102+3×101+8×100

Kuueteistkümnendsüsteemi numbrisüsteem – alus-16

Kuueteistkümnendates numbrites kasutatakse numbreid 0..9 ja A..F.

H tähistab hex prefiksit.

Näited:

2816 = 28H = 2×161+8×160 = 40

2F16 = 2FH = 2×161+15×160 = 47

BC1216 = BC12H = 11×163+12×162+1×161+2×160= 48146
