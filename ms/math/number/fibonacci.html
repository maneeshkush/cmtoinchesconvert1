Nombor & Jujukan Fibonacci

Jujukan Fibonacci ialah jujukan nombor, di mana setiap nombor ialah jumlah 2 nombor sebelumnya, kecuali dua nombor pertama iaitu 0 dan 1.

Formula jujukan Fibonacci

Sebagai contoh:

F 0 = 0

F 1 = 1

F 2 = F 1 + F 0 = 1+0 = 1

F 3 = F 2 + F 1 = 1+1 = 2

F 4 = F 3 + F 2 = 2+1 = 3

F 5 = F 4 + F 3 = 3+2 = 5

...

Penumpuan nisbah emas

Nisbah dua nombor Fibonacci berjujukan, menumpu kepada nisbah emas:

φ ialah nisbah emas = (1+√ 5 ) / 2 ≈ 1.61803399

Kod C fungsi Fibonacci

Fibonacci berganda(int tidak bertanda n)

{

    ganda f_n =n;

    berganda f_n1=0.0;

    ganda f_n2=1.0;

 

    jika ( n > 1 ) {

        untuk(int k=2; k<=n; k++) {

            f_n = f_n1 + f_n2;

            f_n2 = f_n1;

            f_n1 = f_n;

        }

    }

 

    kembali f_n;

}
