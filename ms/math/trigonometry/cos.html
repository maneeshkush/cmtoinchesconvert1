Definisi kosinus

Dalam segi tiga tegak ABC sinus α, sin(α) ditakrifkan sebagai nisbah antara sisi yang bersebelahan dengan sudut α dan sisi yang bertentangan dengan sudut tegak (hipotenus):

cos α = b / c

Contoh

b = 3"

c = 5"

cos α = b / c = 3 / 5 = 0.6

Fungsi kosinus songsang

Arkosin bagi x ditakrifkan sebagai fungsi kosinus songsang bagi x apabila -1≤x≤1.

Apabila kosinus y bersamaan dengan x:

cos y = x

Maka arkosinus x adalah sama dengan fungsi kosinus songsang bagi x, yang sama dengan y:

arccos x = cos-1 x = y

Contoh

arccos 1 = cos-1 1 = 0 rad = 0°
