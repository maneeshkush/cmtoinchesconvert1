perintah mv dalam Linux/Unix

Perintah mv Linux.

Perintah mv digunakan untuk memindahkan fail dan direktori.

sintaks arahan mv

$ mv [options] source dest

pilihan arahan mv

pilihan utama arahan mv:

pilihan penerangan
mv -f bergerak paksa dengan menulis ganti fail destinasi tanpa gesaan
mv -i gesaan interaktif sebelum menulis ganti
mv -u kemas kini - bergerak apabila sumber lebih baharu daripada destinasi
mv -v verbose - sumber cetak dan fail destinasi
lelaki mv manual bantuan

contoh arahan mv

Alihkan fail main.c def.h ke direktori /home/usr/rapid/ :

$ mv main.c def.h /home/usr/rapid/

 

Pindahkan semua fail C dalam direktori semasa ke subdirektori bak :

$ mv *.c bak

 

Alihkan semua fail dalam subdirektori bak ke direktori semasa :

$ mv bak/* .

 

Namakan semula fail main.c kepada main.bak:

$ mv main.c main.bak

 

Namakan semula direktori bak kepada bak2:

$ mv bak bak2

 

Kemas kini - bergerak apabila main.c lebih baharu:

$ mv -u main.c bak
$

 

Alihkan main.c dan gesa sebelum tulis ganti bak/main.c:

$ mv -v main.c bak
'bak/main.c' -> 'bak/main.c'
$
