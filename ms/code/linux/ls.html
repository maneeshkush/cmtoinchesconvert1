Perintah ls dalam Linux/Unix

ls ialah perintah shell Linux yang menyenaraikan kandungan direktori fail dan direktori.

  • sintaks ls
  • ls pilihan
  • ls contoh
  • penjana kod ls

sintaks ls

$ ls [options] [file|dir]

Pilihan arahan ls

Pilihan utama arahan ls:

pilihan penerangan
ls -a senaraikan semua fail termasuk fail tersembunyi bermula dengan '.'
ls --warna senarai berwarna [=selalu/tidak pernah/auto]
ls -d senaraikan direktori - dengan ' */'
ls -F tambah satu aksara */=>@|kepada enteries
ls -i senarai nombor indeks inod fail
ls -l senarai dengan format panjang - tunjukkan kebenaran
ls -la senarai format panjang termasuk fail tersembunyi
ls -lh senarai format panjang dengan saiz fail yang boleh dibaca
ls -ls senarai dengan format panjang dengan saiz fail
ls -r senarai dalam susunan terbalik
ls -R senaraikan pepohon direktori secara rekursif
ls -s senarai saiz fail
ls -S susun mengikut saiz fail
ls -t susun mengikut masa & tarikh
ls -X mengisih mengikut nama sambungan

contoh perintah ls

Anda boleh menekan butang tab untuk melengkapkan secara automatik nama fail atau folder.

Senaraikan direktori Dokumen/Buku dengan laluan relatif :

$ ls Documents/Books

 

Senaraikan direktori /home/user/Documents/Books dengan laluan mutlak .

$ ls /home/user/Documents/Books

 

Senaraikan direktori akar:

$ ls /

 

Senaraikan direktori induk:

$ ls ..

 

Senaraikan direktori rumah pengguna (cth: /home/user):

$ ls ~

 

Senarai dengan format panjang:

$ ls -l

 

Tunjukkan fail tersembunyi:

$ ls -a

 

Senaraikan dengan format panjang dan tunjukkan fail tersembunyi:

$ ls -la

 

Isih mengikut tarikh/masa:

$ ls -t

 

Isih mengikut saiz fail:

$ ls -S

 

Senaraikan semua subdirektori:

$ ls *

 

Senarai pokok direktori rekursif:

$ ls -R

 

Senaraikan hanya fail teks dengan kad bebas:

$ ls *.txt

 

ls pengalihan ke fail output:

$ ls > out.txt

 

Senaraikan direktori sahaja:

$ ls -d */

 

Senaraikan fail dan direktori dengan laluan penuh:

$ ls -d $PWD/*
