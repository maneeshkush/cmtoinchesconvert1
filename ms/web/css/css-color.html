Warna CSS

Kod warna boleh menjadi salah satu daripada:

Format heks: #rrggbb

Format RGB: rgb (merah, hijau, biru)

Format nama: nama

Contoh

Untuk warna oren:

Format heks: #FFA500

Format RGB: rgb(255,165,0)

Format nama: oren

Menetapkan warna elemen

Untuk elemen tertentu:

<element style="color: code;"></element>

Untuk semua elemen dari jenis yang sama.Letakkan kod dalam teg <style> di bahagian kepala:

<style>
   element { color: code; }
</style>

Contoh

Menetapkan warna teks perenggan kepada merah:

Menetapkan warna perenggan tertentu:

<p style="color: #FF0000;">Some text ...</p>

Lihat:

Beberapa teks...

Menetapkan warna semua perenggan

<style>
   p { color: #FF0000; }
</style>

Menetapkan warna latar belakang elemen

element { background: code; }

Contoh

Menetapkan warna latar belakang perenggan kepada merah:

Menetapkan warna latar belakang perenggan tertentu:

<p style="background: #FF0000;">Some text ...</p>

Lihat:

Beberapa teks...

Menetapkan warna latar belakang semua perenggan:

<style>
   p { background: #FF0000; }
</style>

Menetapkan warna sempadan elemen

element { border-color: topcode rightcode bottomcode leftcode }

Contoh

Menetapkan warna sempadan perenggan kepada merah, hijau, biru dan hitam:

Menetapkan warna sempadan perenggan tertentu:

<p style="border-color: #FF0000 #00FF00 #0000FF #000000; border-style:solid">Some text ...</p>

Lihat:

Beberapa teks...

Menetapkan warna sempadan semua perenggan:

<style>
   p { border-color: #FF0000 #00FF00 #0000FF #000000; }
</style>
