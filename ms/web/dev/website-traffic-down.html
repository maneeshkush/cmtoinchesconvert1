Trafik laman web menurun

Mengapa trafik tapak web saya menurun?

Semak kalendar

Hari cuti dan hujung minggu mungkin mengurangkan trafik anda.

Trafik akan kembali normal apabila hari raya berakhir.

Bandingkan dengan tahun lepas

Gunakan Google Analytics untuk memaparkan graf lawatan tahun lepas.

Semak sama ada lawatan menurun setahun yang lalu juga.

Pepijat Google Analitis

Menggunakan kod Google Analitis lama dengan fail urchin.js, mungkin menunjukkan 2 hari baru-baru ini dengan trafik yang lebih rendah daripada trafik sebenar.

Trafik tidak benar-benar sesak, tetapi nampaknya hanya sesak.

Masalah pelayan

Cuba semak imbas laman web anda, jika anda tidak dapat mengaksesnya, anda mempunyai masalah pelayan web atau pelayan DNS.

Cuba akses pelayan web anda dan semak sama ada ia aktif.

Semak integriti pangkalan data atau fail html anda.

Gunakan alat ujian ping untuk menyemak respons pelayan web anda.

Cari masalah pelayan DNS baharu.Pada 9/2012, tapak web ini dengan ramai yang lain tidak dapat bertindak balas (lihat: GoDaddy digodam ).

Kedudukan hasil carian Google jatuh

Kebanyakan trafik laman web datang daripada enjin carian dan enjin carian utama ialah Google.

Jika kebanyakan lawatan tapak web anda dijana oleh satu kata kunci, ia mungkin diambil oleh persaingan.

  • Cari kata kunci dalam Google untuk menentukan sama ada terdapat tapak web lain yang jika terletak di hadapan tapak anda dan memberikan nilai yang lebih baik kepada pengguna.
  • Cari berita untuk perubahan algoritma kedudukan Google.Contohnya, kemas kini Google panda merosakkan trafik banyak laman web.

Laman web diharamkan oleh Google

Menggunakan kaedah yang dilarang untuk mempromosikan tapak anda di Google akan memastikan tapak web anda akan diharamkan oleh Google.

Cari di Google dengan kata kunci utama anda dan lihat sama ada ia muncul seperti biasa dalam hasil carian.

Jika tapak web anda tidak muncul sama sekali, anda harus:

  1. Baca garis panduan juruweb Google dan betulkan tapak web anda.
  2. Serahkan permintaan pertimbangan semula kepada Google.
