Bagaimana untuk menukar ppm kepada peratus

1% = 1/100

1ppm = 1/1000000

Jadi

1ppm = 0.0001%

Jadi tukar daripada ppm kepada peratus, bahagikan ppm dengan [10000].

x(%) = x(ppm) / 10000

 

Contoh 1

cari berapa peratus dalam 200ppm.

x(%) = 200ppm / 10000 = 0.02%

Contoh 2

cari berapa peratus dalam 400ppm.

x(%) = 400ppm / 10000 = 0.04%

Contoh 3

cari berapa peratus dalam 500ppm.

x(%) = 500ppm / 10000 = 0.05%

Contoh 4

cari berapa peratus dalam 600ppm.

x(%) = 600ppm / 10000 = 0.06%

Contoh 5

cari berapa peratus dalam 800ppm.

x(%) = 800ppm / 10000 = 0.08%
