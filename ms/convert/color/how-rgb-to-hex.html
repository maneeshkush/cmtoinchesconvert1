Bagaimana untuk menukar RGB kepada warna heks

Bagaimana untuk menukar daripada warna RGB kepada kod warna heksadesimal.

Warna RGB

Warna RGB ialah gabungan warna Merah, Hijau dan biru:

(R, G, B)

Merah, hijau dan biru masing-masing menggunakan 8 bit, dengan nilai integer antara 0 hingga 255.

Jadi bilangan warna yang boleh dihasilkan ialah:

256×256×256 = 16777216 = 100000016

kod warna hex

Kod warna heks ialah nombor heksadesimal 6 digit (asas 16):

RRGGBB 16

2 digit kiri mewakili warna merah.

2 digit tengah mewakili warna hijau.

2 digit kanan mewakili warna biru.

penukaran rgb kepada hex

1. Tukarkan nilai merah, hijau dan biru daripada perpuluhan kepada heks.
2. Gabungkan 3 nilai hex merah, hijau dan biru bersama-sama: RRGGBB.

Contoh 1
Tukar merah (255,0,0) kepada kod warna heks:

R = 25510 = FF16

G = 010 = 0016

B = 010 = 0016

Jadi kod warna hex ialah:

Hex = FF0000

Contoh #2
Tukar warna emas (255,215,0) kepada kod warna heks:

R = 25510 = FF16

G = 21510 = D716

B = 010 = 0016

Jadi kod warna hex ialah:

Hex = FFD700

Apakah yang dilakukan oleh penukar RGB ke Hex ini?

Ia mengambil sebagai input nilai warna merah, hijau dan biru antara 0 hingga 255 dan kemudian menukar nilai tersebut menjadi rentetan heksadesimal yang boleh digunakan untuk menentukan warna dalam kod html/css.Perisian penyunting foto biasanya mewakili warna dalam RGB dan oleh itu jika anda ingin menggunakan warna yang digunakan dalam perisian penyuntingan foto anda sebagai latar belakang elemen HTML anda, anda perlu mendapatkan perwakilan heksadesimal bagi nilai RGB.Alat ini membolehkan anda mendapatkan nilai tersebut.

Cuba alat carian warna baharu kami.

Tukar Nilai Hex kepada RGB

Mungkin anda pernah melihat kod heks pada halaman web dan ingin menggunakan warna itu dalam perisian penyuntingan foto anda.Dalam kes itu, anda memerlukan nilai RGB jika perisian penyuntingan foto anda tidak menyokong nilai HEX.
