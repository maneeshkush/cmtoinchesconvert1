Linux/Unix 中的 cat 命令

Linux 貓命令。

cat 命令用於顯示文本文件的內容，並將多個文件合併為一個文件。

cat 命令不接受目錄。

貓命令語法

$ cat [options] file1 [file2...]

貓命令選項

cat 命令主要選項：

選項 描述
貓-b 為非空行添加行號
貓-n 為所有行添加行號
貓 將空行壓縮為一行
貓-E 在行尾顯示 $
貓-T 顯示 ^I 而不是製表符

cat 命令示例

查看文本文件數據：

$ cat list1.txt
milk
bread
apples

$ cat list2.txt
house
car

$

 

合併 2 個文本文件：

$ cat list1.txt list2.txt
milk
bread
apples

house
car

$

 

將 2 個文本文件合併到另一個文件：

$ cat list1.txt list2.txt > todo.txt
$
