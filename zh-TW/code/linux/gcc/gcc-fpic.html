gcc -fPIC 選項標誌

gcc -fPIC 為共享庫生成位置獨立代碼 (PIC)。

句法

$ gcc -fPIC [options] [source files] [object files] -o output file

 

如果平台編譯器支持，請使用 -fpic 而不是 -fPIC 來生成更高效的代碼。

例子

編寫源文件myfile.c：

// myfile.c
#include <stdio.h>
 
int myfunc()
{
    printf("myfunc\n");
}

 

構建myfile.c生成myfile.o：

$ gcc -fPIC -c myfile.c
$
