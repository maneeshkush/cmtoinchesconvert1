電流定義

電流是 電荷在電場中的流動速率，通常在電路中。

使用水管類比，我們可以將電流想像成在管道中流動的水流。

電流以安培 (amp) 為單位測量。

電流計算

電流是通過電路中電荷流動的速率來衡量的：

i(t) = dQ(t) / dt

所以瞬時電流由電荷對時間的導數給出。

i(t) 是時間 t 處的瞬時電流I，單位為安培 (A)。

Q(t) 是以庫侖 (C) 為單位的瞬時電荷。

t 是以秒 (s) 為單位的時間。

 

當電流恆定時：

I = ΔQ / Δt

I 是以安培 (A) 為單位的電流。

ΔQ 是以庫侖 (C) 為單位的電荷，它在 Δt 的持續時間內流動。

Δt 是以秒 (s) 為單位的持續時間。

 

例子

當 5 庫侖電流流過電阻持續 10 秒時，

電流將通過以下方式計​​算：

I = Δ Q / Δ t  = 5C / 10s = 0.5A

串聯電路中的電流

因此，流過串聯電阻器的電流在所有電阻器中都是相等的——就像水流過單個管道一樣。

ITotal = I1 = I2 = I3 =...

I Total - 以安培 (A) 為單位的等效電流。

I 1 - 負載 #1 的電流，單位為安培 (A)。

I 2 - 負載 #2 的電流，單位為安培 (A)。

I 3 - 以安培 (A) 為單位的負載 #3 的電流。

並聯電路中的電流

平行流過負載的電流 - 就像水流過平行管道一樣。

所以總電流I Total是各負載並聯電流之和：

ITotal = I1 + I2 + I3 +...

I Total - 以安培 (A) 為單位的等效電流。

I 1 - 負載 #1 的電流，單位為安培 (A)。

I 2 - 負載 #2 的電流，單位為安培 (A)。

I 3 - 以安培 (A) 為單位的負載 #3 的電流。

分流器

所以並聯電阻的分流為

RT = 1 / (1/R2 + 1/R3)

或者

I1 = IT × RT / (R1+RT)

基爾霍夫電流定律 (KCL)

所以幾個電氣元件的連接點稱為節點。

所以進入節點的電流的代數和為零。

∑ Ik = 0

交流電 (AC)

交流電由正弦電壓源產生。

歐姆定律

IZ = VZ / Z

I Z   - 流過負載的電流，單位為安培 (A)

V Z - 負載上的壓降，單位為伏特 (V)

Z   - 以歐姆 (Ω) 為單位測量的負載阻抗

角頻率

ω = 2π f

ω - 角速度，單位為弧度每秒 (rad/s)

f - 以赫茲 (Hz) 為單位測量的頻率。

瞬時電流

i ( t ) = I峰值 sin ( ωt+θ )

i ( t ) - 時間 t 的瞬時電流，以安培 (A) 為單位測量。

Ipeak - 最大電流（= 正弦波幅值），以安培 (A) 為單位測量。

ω -以每秒弧度 (rad/s) 為單位測量的角頻率。

t——時間，以秒 (s) 為單位。

θ        - 以弧度 (rad) 為單位的正弦波相位。

RMS（有效）電流

I rms =  I eff =  I peak / √ 2 ≈ 0.707 I peak

峰峰值電流

I p-p = 2 I峰值

電流測量

所以電流測量是通過將電流表串聯在被測物上來完成的，所以所有被測電流都會流過電流表。

所以電流表的阻值很低，幾乎不影響被測電路。
