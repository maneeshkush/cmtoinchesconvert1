如何計算期末考試成績

期末考試成績計算。

期末考試成績計算

因此，期末考試成績等於 100% 乘以要求成績，減去 100% 減去期末考試權重 (w) 乘以當前成績 (g)，再除以期末考試權重 (w)。

Final exam grade =

=  ( 100%×required grade - (100% - w)×current grade ) / w

例子

當前成績為 70%（或 C-）。

期末考試權重為50%。

要求的成績為 80%（或 B-）。

計算

因此，期末考試成績等於 100% 乘以要求成績，減去 100% 減去期末考試權重 (w) 乘以當前成績 (g)，再除以期末考試權重 (w)。

Final exam grade =

= ( 100%×required grade - (100% - w)×current grade ) / w

= ( 100%×80% - (100% - 50%)×70% ) / 50% = 90%

所以期末考試成績應該是 90%（或 A-）。
