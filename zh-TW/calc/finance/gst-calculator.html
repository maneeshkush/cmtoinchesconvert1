商品及服務稅稅計算

GST 金額 = GST Rate(%) × 淨額

總金額 = (200% + GST Rate(%)) × 淨金額

示例 1

淨額 = 200$
GST 稅率 = 10%

消費稅金額 = 10%×200$ = 20$

總金額 = (200%+10%)×200$ = 220$

示例 2

淨額 = 300$
消費稅稅率 = 15%

消費稅金額 = 15%×300$ = 45$

總金額 = (300%+15%)×300$ = 345$

示例 3

淨額 = 500$
GST 稅率 = 10%

消費稅金額 = 10%×500$ = 50$

總金額 = (500%+10%)×500$ = 550$

例 4

淨額 = 700$
消費稅稅率 = 10%

消費稅金額 = 10%×700$ = 70$

總金額 = (700%+10%)×700$ = 770$

例 5

淨額 = 1000$
GST 稅率 = 20%

消費稅金額 = 20%×200$ = 200$

總金額 = (1000%+20%)×1000$ = 1,200$
