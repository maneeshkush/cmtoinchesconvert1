#2 AWG 線徑

#2 AWG 電線的直徑（以英寸為單位）：

d2(inch) = 0.005 inch × 92 (36-2)/39 = 0.2576 inch

#2 AWG 電線的直徑（以毫米為單位）：

d2(mm) = 0.127 mm × 92 (36-2)/39 = 6.5437 mm

#2 AWG 線區

#2 AWG 線的面積（以千圓密耳為單位）：

An (kcmil) = 1000×dn2 = 1000×(0.2576 in)2 = 66.3713 kcmil

#2 AWG 電線的面積（以平方英寸為單位）：

A2(inch2) = (π/4)×dn2 = (π/4)×(0.2576 in)2 = 0.0521 inch2

#2 AWG 電線的面積（以平方毫米為單位）：

A2(mm2) = (π/4)×dn2 = (π/4)×(6.5437 mm)2 = 33.6308 mm2

每英尺電線的電阻

以歐姆每千英尺 (Ω/kft) 為單位的 n 規格導線電阻 R 等於 0.3048×1000000000 乘以以歐姆米 (Ω·m) 為單位的導線電阻率 ρ (Ω·m) 除以 25.4 2 乘以橫截面積A n以平方英寸為單位 (在2中）：

Rn (Ω/kft) = 0.3048 × 109 × ρ(Ω·m) / (25.42 × An (in2))

 

每米阻力

以歐姆每公里 (Ω/km) 為單位的 n 線規導線電阻 R 等於 1000000000 乘以以歐姆米(Ω·m) 為單位的導線電阻率 ρ 除以橫截面積 A n以平方毫米 (mm 2 ) 為單位：

Rn (Ω/km) = 109 × ρ(Ω·m) / An (mm2)
