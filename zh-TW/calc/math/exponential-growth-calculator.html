指數增長/衰減計算器

在線指數增長/衰減計算器。

指數增長/衰減公式

x(t) = x0 × (1 + r) t

x (t) 是時間 t 的值。

x 0是時間t=0時的初始值。

r 為 r>0 時的增長率或 r<0 時的衰減率，以百分比表示。

t 是以離散間隔和選定時間單位表示的時間。

指數增長計算器

輸入初始值 ( x 0 ):  
輸入增長率/衰減率 ( r )： %
輸入時間（t）：  
   
時間 t ( x (t))的值：  

示例 1

x0 = 40

r = 3% = 0.04

t = 80 hours

x(t) = x0 × (1 + r) t = 40×(1+0.03)80 = 425.63

示例 2

x0 = 40

r = 5% = 0.05

t = 80 hours

x(t) = x0 × (1 + r) t = 40×(1+0.05)80 = 1982

示例 3

x0 = 60

r = 3% = 0.05

t = 80 hours

x(t) = x0 × (1 + r) t = 60×(1+0.05)80 = 2973
