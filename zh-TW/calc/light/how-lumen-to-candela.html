如何將流明轉換為坎德拉

如何將以流明 (lm) 為單位的光通量轉換為以坎德拉 (cd) 為單位的發光強度。

您可以計算但不能將流明轉換為坎德拉，因為坎德拉和流明不代表相同的數量。

流明到坎德拉的計算

對於均勻、各向同性的光源，以坎德拉 (cd) 為單位的發光強度 I v 等於以流明 (lm) 為單位的光通量 Φ v ，

除以  球面度 (sr) 中的立體角Ω ：

Iv(cd) = Φv(lm) / Ω(sr)

 

因此，以球面度為單位的立體角Ω  (sr) 等於 2 乘以 pi 乘以 1 減去以 度數 (°)為單位的頂角θ的一半的餘弦值。

Ω(sr) = 2π(1 - cos(θ/2))

 

所以以坎德拉 (cd) 為單位的發光強度I v 等於以流明 (lm) 為單位的光通量 Φ v ，

除以 2 乘以 pi 乘以 1 減去以度 (°)為單位的頂角 θ的一半的餘弦。

Iv(cd) = Φv(lm) / ( 2π(1 - cos(θ/2)) )

所以

candela = lumens / ( 2π(1 - cos(degrees/2)) )

或者

cd = lm / ( 2π(1 - cos(°/2)) )

示例 1

求 當以 流明 (lm) 為單位的光通量 Φ v 為 340lm，頂角為 60°時，以坎德拉 (cd) 為單位的發光強度 I v：

Iv(cd) = 340 lm / ( 2π(1 - cos(60°/2)) ) = 403.9 cd

示例 2

 求當以流明 (lm) 為單位的光通量 Φ v 為 360lm，頂角為 60°時，以坎德拉 (cd) 為單位的發光強度 I v ：

Iv(cd) = 360 lm / ( 2π(1 - cos(60°/2)) ) = 427.6 cd

示例 3

求 當以 流明 (lm) 為單位的光通量 Φ v 為 380lm，頂角為 60°時，以坎德拉 (cd) 為單位的發光強度 I v：

Iv(cd) = 380 lm / ( 2π(1 - cos(60°/2)) ) = 451.4 cd

例 4

求 當以 流明 (lm) 為單位的光通量 Φ v 為 440lm，頂角為 60°時，以坎德拉 (cd) 為單位的發光強度 I v：

Iv(cd) = 440 lm / ( 2π(1 - cos(60°/2)) ) = 522.6 cd

例 5

求 以流明 (lm) 為單位的光通量 Φ v 為 540lm，頂角為 60° 時，以坎德拉 (cd) 為單位的發光強度 I v：

Iv(cd) = 540 lm / ( 2π(1 - cos(60°/2)) ) = 641.4 cd
