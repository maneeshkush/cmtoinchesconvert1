餘弦定義

在直角三角形 ABC 中，α 的正弦，sin(α) 定義為與角 α 相鄰的邊與直角的對邊（斜邊）之間的比率：

cos α = b / c

例子

b = 3"

c = 5"

cos α = b / c = 3 / 5 = 0.6

反餘弦函數

當-1≤x≤1時，x的反餘弦定義為x的反餘弦函數。

當 y 的餘弦等於 x 時：

cos y = x

那麼x的反餘弦等於x的反餘弦函數，等於y：

arccos x = cos-1 x = y

例子

arccos 1 = cos-1 1 = 0 rad = 0°
