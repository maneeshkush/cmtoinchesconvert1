使用 Javascript 進行屏幕分辨率檢測

您可以使用screen.width和screen.height屬性獲取屏幕尺寸。

您可以使用screen.availWidth和screen.availHeight屬性獲取可用的屏幕尺寸。

例如：

<script>
alert(screen.width+' x '+screen.height);
</script>
