使用 Javascript 進行屏幕分辨率檢測

您可以使用window.outerWidth和window.outerHeight獲取外部窗口大小。

您可以使用window.innerWidth和 window.innerHeight獲取內部窗口大小。

在 IE 中，您可以通過document.body.clientWidth和 document.body.clientHeight獲得它。

您可以使用window.screenLeft和 window.screenTop獲取屏幕窗口位置。

例如：

<script>
alert(window.outerWidth+' x '+window.outerHeight);
</script>
