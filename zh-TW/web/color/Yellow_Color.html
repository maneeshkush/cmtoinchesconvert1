黃色 HTML 顏色代碼

帶有黃色字體的 HTML 段落

代碼：

<p style=" color:yellow; background:black">這些字體是黃色的！</p>

結果：

這些字體是黃色的！

或者

<p style=" color:#FFFF00; background:black">這些字體也是黃色的！</p>

結果：

這些字體也是黃色的！

或者

<p style=" color:rgb(255,255,0); background:black">這些字體也是黃色的！</p>

結果：

這些字體也是黃色的！

帶有黑色字體和黃色背景色的 HTML 段落

代碼：

<p style="color:black; background:yellow ">背景色為黃色</p>

結果：

底色為黃色
