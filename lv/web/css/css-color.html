CSS krāsa

Krāsu kods var būt viens no:

Hex formāts: #rrggbb

RGB formāts: rgb (sarkans, zaļš, zils)

Nosaukuma formāts: nosaukums

Piemērs

Oranžai krāsai:

Hex formāts: #FFA500

RGB formāts: rgb (255,165,0)

Nosaukuma formāts: oranžs

Elementa krāsas iestatīšana

Konkrētam elementam:

<element style="color: code;"></element>

Visiem viena veida elementiem.Ievietojiet kodu tagā <style> galvas sadaļā:

<style>
   element { color: code; }
</style>

Piemērs

Rindkopas teksta krāsas iestatīšana uz sarkanu:

Konkrētas rindkopas krāsas iestatīšana:

<p style="color: #FF0000;">Some text ...</p>

Skatīt:

Kaut kāds teksts...

Visu rindkopu krāsas iestatīšana

<style>
   p { color: #FF0000; }
</style>

Iestatīšanas elementa fona krāsa

element { background: code; }

Piemērs

Rindkopas fona krāsas iestatīšana uz sarkanu:

Konkrētas rindkopas fona krāsas iestatīšana:

<p style="background: #FF0000;">Some text ...</p>

Skatīt:

Kaut kāds teksts...

Visu rindkopu fona krāsas iestatīšana:

<style>
   p { background: #FF0000; }
</style>

Elementa apmales krāsas iestatīšana

element { border-color: topcode rightcode bottomcode leftcode }

Piemērs

Rindkopas apmales krāsas iestatīšana uz sarkanu, zaļu, zilu un melnu:

Konkrētas rindkopas apmales krāsas iestatīšana:

<p style="border-color: #FF0000 #00FF00 #0000FF #000000; border-style:solid">Some text ...</p>

Skatīt:

Kaut kāds teksts...

Visu rindkopu apmales krāsas iestatīšana:

<style>
   p { border-color: #FF0000 #00FF00 #0000FF #000000; }
</style>
