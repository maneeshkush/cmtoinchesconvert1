Skaitļu pievienošana ar eksponentiem

Eksponentu pievienošana tiek veikta, vispirms aprēķinot katru eksponentu un pēc tam pievienojot:

an + bm

Piemērs:

42 + 25 = 4⋅4+2⋅2⋅2⋅2⋅2 = 16+32 = 48

To pašu bāzu b un eksponentu n pievienošana:

bn + bn = 2bn

Piemērs:

42 + 42 = 2⋅42 = 2⋅4⋅4 = 32

Negatīvo eksponentu pievienošana

Negatīvo eksponentu pievienošana tiek veikta, vispirms aprēķinot katru eksponentu un pēc tam pievienojot:

a-n + b-m = 1/an + 1/bm

Piemērs:

4-2 + 2-5 = 1/42 + 1/25 = 1/(4⋅4)+1/(2⋅2⋅2⋅2⋅2) = 1/16+1/32 = 0.09375

Daļējo eksponentu pievienošana

Daļējo eksponentu pievienošana tiek veikta, vispirms paaugstinot katru eksponentu un pēc tam pievienojot:

an/m + bk/j

Piemērs:

33/2 + 25/2 = √(33) + √(25) = √(27) + √(32) = 5.196 + 5.657 = 10.853

 

To pašu bāzu b un eksponentu n/m pievienošana:

bn/m + bn/m = 2bn/m

Piemērs:

42/3 + 42/3 = 2⋅42/3 = 2 ⋅ 3√(42) = 5.04

Mainīgo pievienošana ar eksponentiem

Eksponentu pievienošana tiek veikta, vispirms aprēķinot katru eksponentu un pēc tam pievienojot:

xn + xm

Ar tiem pašiem eksponentiem:

xn + xn = 2xn

Piemērs:

x2 + x2 = 2x2
