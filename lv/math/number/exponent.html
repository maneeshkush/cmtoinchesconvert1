Kas ir eksponents

Bāze a, kas palielināta līdz pakāpei n, ir vienāda ar a reizinājumu, n reizes:

a n = a × a × ... × a

                    n reizes

a ir bāze un n ir eksponents.

Piemēri

31 = 3

32 = 3 × 3 = 9

33 = 3 × 3 × 3 = 27

34 = 3 × 3 × 3 × 3 = 81

35 = 3 × 3 × 3 × 3 × 3 = 243

Eksponentu produktu noteikumi

Produkta noteikums ar tādu pašu bāzi

an ⋅ am = an+m

Piemērs:

23 ⋅ 24 = 23+4 = 27 = 2⋅2⋅2⋅2⋅2⋅2⋅2 = 128

Produkta kārtula ar tādu pašu eksponentu

an ⋅ bn = (a ⋅ b)n

Piemērs:

32 ⋅ 42 = (3⋅4)2 = 122 = 12⋅12 = 144

Eksponentu koeficienta noteikumi

Koeficientu noteikums ar tādu pašu bāzi

an / am = an-m

Piemērs:

25 / 23 = 25-3 = 22 = 2⋅2 = 4

Koeficienta noteikums ar tādu pašu eksponentu

an / bn = (a / b)n

Piemērs:

43 / 23 = (4/2)3 = 23 = 2⋅2⋅2 = 8

Eksponentu jaudas noteikumi

Spēka noteikums I

(an) m = a n⋅m

Piemērs:

(23)2 = 23⋅2 = 26 = 2⋅2⋅2⋅2⋅2⋅2 = 64

Jaudas noteikums II

a nm = a (nm)

Piemērs:

232 = 2(32) = 2(3⋅3) = 29 = 2⋅2⋅2⋅2⋅2⋅2⋅2⋅2⋅2 = 512

Varas likums ar radikāļiem

m√(a n) = a n/m

Piemērs:

2√(26) = 26/2 = 23 = 2⋅2⋅2 = 8

Valda negatīvie eksponenti

b-n = 1 / bn

Piemērs:

2-3 = 1/23 = 1/(2⋅2⋅2) = 1/8 = 0.125
