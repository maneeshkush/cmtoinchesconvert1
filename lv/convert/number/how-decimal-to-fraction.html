Kā konvertēt decimāldaļu uz daļskaitli

Konversijas posmi

  1. Ierakstiet decimāldaļu kā daļu no cipariem, kas atrodas pa labi no decimālpunkta (skaitītājs) un pakāpē 10 (saucējs).
  2. Atrodiet skaitītāja un saucēja lielāko kopīgo dalītāju (gcd).
  3. Samaziniet daļu, dalot skaitītāju un saucēju ar gcd.

1. piemērs

Pārvērst 0,35 uz daļskaitli:

0.35 = 35/100

Tātad atrodiet skaitītāja un saucēja lielāko kopīgo dalītāju (gcd):

gcd(35,100) = 5

Tātad samaziniet daļu, dalot skaitītāju un saucēju ar gcd:

0.35 = (35/5) / (100/5) = 7/20

2. piemērs

Pārvērst 2,58 uz daļskaitli:

2.58 = 2+58/100

Tātad atrodiet skaitītāja un saucēja lielāko kopīgo dalītāju (gcd):

gcd(58,100) = 2

Tātad samaziniet daļu, dalot skaitītāju un saucēju ar gcd:

2+58/100 = 2 + (58/2) / (100/2) = 2+29/50

3. piemērs

Pārvērst 0,126 uz daļskaitli:

0.126 = 126/1000

Atrodiet skaitītāja un saucēja lielāko kopīgo dalītāju (gcd):

gcd(126,1000) = 2

Samaziniet daļu, dalot skaitītāju un saucēju ar gcd:

0.126 = (126/2)/(1000/2) = 63/500

Kā atkārtotu decimāldaļu pārveidot par daļskaitli

1. piemērs

Pārvērst 0,333333... par daļu:

x = 0.333333...

10x = 3.333333...

10x - x = 9x = 3

x = 3/9 = 1/3

2. piemērs

Pārvērst 0,0565656... par daļu:

x = 0.0565656...

100 x = 5.6565656...

100 x -  x = 99 x = 5.6

990 x = 56

x = 56/990 = 28/495
