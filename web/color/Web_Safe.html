Web Safe Colors Chart

Web safe colors are formed by the combination of red, green and blue (RGB) hex values of 0,33H,66H,99H,CCH,FFH.

This results with 6*6*6=216 colors.

Previously browsers did not support display of more than 256 colors palette at the same time. With the new web browsers it is not necessary to use only these colors.
