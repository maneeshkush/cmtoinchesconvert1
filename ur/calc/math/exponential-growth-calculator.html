ایکسپونینشل گروتھ/کیلکولیٹر

آن لائن ایکسپونینشل گروتھ/کیلکولیٹر۔

کفایتی نمو/ زوال کا فارمولا

x(t) = x0 × (1 + r) t

x (t) وقت t کی قدر ہے۔

x 0 وقت کی ابتدائی قدر ہے t=0۔

r شرح نمو ہے جب r>0 یا زوال کی شرح جب r<0، فیصد میں۔

t مجرد وقفوں اور منتخب وقت کی اکائیوں میں وقت ہے۔

تیز رفتار ترقی کیلکولیٹر

ابتدائی قدر درج کریں ( x 0 ):  
شرح نمو درج کریں ( r ): %
وقت درج کریں ( t ):  
   
وقت کی قدر t ( x (t)):  

مثال 1

x0 = 40

r = 3% = 0.04

t = 80 hours

x(t) = x0 × (1 + r) t = 40×(1+0.03)80 = 425.63

مثال 2

x0 = 40

r = 5% = 0.05

t = 80 hours

x(t) = x0 × (1 + r) t = 40×(1+0.05)80 = 1982

مثال 3

x0 = 60

r = 3% = 0.05

t = 80 hours

x(t) = x0 × (1 + r) t = 60×(1+0.05)80 = 2973
