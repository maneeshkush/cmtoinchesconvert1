لینکس / یونکس میں سی ڈی کمانڈ

cd ٹرمینل کے شیل کی ڈائریکٹری/فولڈر کو تبدیل کرنے کے لیے ایک لینکس کمانڈ ہے۔

ڈائرکٹری کا نام خود بخود مکمل کرنے کے لیے آپ ٹیب بٹن دبا سکتے ہیں۔

  • سی ڈی نحو
  • سی ڈی کی مثالیں

سی ڈی نحو

$ cd [directory]

سی ڈی کمانڈ کی مثالیں۔

ہوم ڈائرکٹری میں تبدیلی ($HOME ماحول کے متغیر کے ذریعہ طے شدہ):

$ cd

 

ہوم ڈائریکٹری میں بھی تبدیل کریں:

$ cd ~

 

روٹ ڈائرکٹری میں تبدیل کریں:

$ cd /

 

پیرنٹ ڈائرکٹری میں تبدیل کریں:

$ cd ..

 

ذیلی ڈائرکٹری دستاویزات میں تبدیل کریں:

$ cd Documents

 

ذیلی ڈائرکٹری دستاویزات/کتابوں میں تبدیل کریں :

$ cd Documents/Books

 

مطلق راستے کے ساتھ ڈائریکٹری میں تبدیل کریں /home/user/Desktop :

$ cd /home/user/Desktop

 

ڈائرکٹری کا نام سفید جگہ کے ساتھ تبدیل کریں - My Images :

$ cd My\ Images

یا

$ cd "My Images"

یا

$ cd 'My Images'
