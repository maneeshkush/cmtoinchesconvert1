عددی نظام

b - عددی نظام کی بنیاد

d n - n-واں ہندسہ

n - منفی نمبر سے شروع ہوسکتا ہے اگر نمبر کا ایک حصہ حصہ ہو۔

N +1 - ہندسوں کی تعداد

ثنائی عددی نظام - بیس -2

بائنری نمبرز صرف 0 اور 1 ہندسوں کا استعمال کرتے ہیں۔

B بائنری سابقہ ​​کو ظاہر کرتا ہے۔

مثالیں:

101012 = 10101B = 1×24+0×23+1×22+0×21+1×20 = 16+4+1= 21

101112 = 10111B = 1×24+0×23+1×22+1×21+1×20 = 16+4+2+1= 23

1000112 = 100011B = 1×25+0×24+0×23+0×22+1×21+1×20 =32+2+1= 35

آکٹل عددی نظام - بنیاد-8

اوکٹل نمبرز 0..7 سے ہندسوں کا استعمال کرتے ہیں۔

مثالیں:

278 = 2×81+7×80 = 16+7 = 23

308 = 3×81+0×80 = 24

43078 = 4×83+3×82+0×81+7×80= 2247

اعشاریہ عددی نظام - بنیاد -10

اعشاریہ نمبر 0..9 سے ہندسوں کا استعمال کرتے ہیں۔

یہ وہ باقاعدہ نمبر ہیں جو ہم استعمال کرتے ہیں۔

مثال:

253810 = 2×103+5×102+3×101+8×100

Hexadecimal Numeral System - Base-16

ہیکس نمبرز 0..9 اور A..F سے ہندسوں کا استعمال کرتے ہیں۔

H ہیکس پریفکس کو ظاہر کرتا ہے۔

مثالیں:

2816 = 28H = 2×161+8×160 = 40

2F16 = 2FH = 2×161+15×160 = 47

BC1216 = BC12H = 11×163+12×162+1×161+2×160= 48146
