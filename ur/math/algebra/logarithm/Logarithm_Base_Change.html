بنیادی اصول کی لوگارتھم تبدیلی

بیس کو b سے c میں تبدیل کرنے کے لیے، ہم بنیادی اصول کی لاگرتھم تبدیلی کا استعمال کر سکتے ہیں۔x کا بیس بی لوگارتھم x کے بیس c لوگارتھم کے برابر ہے جس کو b کے بیس c لوگارتھم سے تقسیم کیا گیا ہے:

logb(x) = logc(x) / logc(b)

مثال نمبر 1

log2(100) = log10(100) / log10(2) = 2 / 0.30103 = 6.64386

مثال نمبر 2

log3(50) = log8(50) / log8(3) = 1.8812853 / 0.5283208 = 3.5608766

ثبوت

x کے بیس بی لاگرتھم کی طاقت کے ساتھ b کو بڑھانے سے x ملتا ہے:

(1) x = blogb(x)

b کے بیس c لوگارتھم کی طاقت کے ساتھ c کو بڑھانے سے b ملتا ہے:

(2) b = clogc(b)

جب ہم (1) لیتے ہیں اور b کو c log c ( b ) (2) سے بدل دیتے ہیں تو ہمیں ملتا ہے:

(3) x = blogb(x) = (clogc(b))logb(x) = clogc(b)×logb(x)

لاگ سی () کو (3) کے دونوں اطراف پرلاگو کرکے :

logc(x) = logc(clogc(b)×logb(x))

لوگارتھم پاور رول کو لاگو کرکے :

logc(x) = [logc(b)×logb(x)] × logc(c)

چونکہ لاگ c ( c ) = 1

logc(x) = logc(b)×logb(x)

یا

logb(x) = logc(x) / logc(b)
