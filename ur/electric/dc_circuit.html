ڈی سی سرکٹ کے قوانین

اوہ کے قانون

I = V / R

جول کا قانون

P = V · I = I 2 · R = V 2 / R

سیریز سرکٹ کے قواعد

VT = V1 + V2 + V3 + ...

IT = I1 = I2 = I3 = ...

RT = R1 + R2 + R3 + ...

1/CT = 1/C1 + 1/C2 + 1/C3 + ...

LT = L1 + L2 + L3 + ...

متوازی سرکٹ کے قواعد

VT = V1 = V2 = V3 = ...

IT = I1 + I2 + I3 + ...

1/RT = 1/R1 + 1/R2 + 1/R3 + ...

CT = C1 + C2 + C3 + ...

1/LT = 1/L1 + 1/L2 + 1/L3 + ...

وولٹیج کی تقسیم

V1 = VT ⋅ R1 / (R1+R2+R3+...)

موجودہ ڈویژن

I1 = IT ⋅ (R2+R3+...) / (R1+R2+R3+...)

کرچوف کا وولٹیج قانون (KVL)

موجودہ لوپ پر وولٹیج کے قطروں کا مجموعہ صفر ہے:

∑ Vi = 0

کرچوف کا موجودہ قانون (KCL)

کئی سرکٹ عناصر کے درمیان جنکشن کو نوڈ کہا جاتا ہے۔

نوڈ پر کرنٹ کی قدروں کا مجموعہ صفر ہے:

∑ I i = 0

اہلیت

C = Q/V

متوازی پلیٹ کپیسیٹر

C = ε ⋅ A / l

فاراد فی میٹر (F/m) میں ε اجازت ہے۔

اجازت نامہ

ε = ε0 ⋅ εr

ε 0 ویکیوم میں اجازت ہے۔

ε r رشتہ دار اجازت یا جدلیاتی مستقل ہے۔

کیپسیٹر کا کرنٹ

IC(t) = C d VC(t) / dt

کیپسیٹر کا وولٹیج

V C (t) = V C (0) + 1/ C ∫ I C (t) ⋅ dt

انڈکٹر کا وولٹیج

VL(t) = L d IL(t) / dt

انڈکٹر کا کرنٹ

I L (t) = I L (0) + 1/ L ∫ V L (t)⋅ dt

کیپسیٹر کی توانائی

WC = C⋅V 2 / 2

انڈکٹر کی توانائی 

WL = L⋅I 2 / 2
