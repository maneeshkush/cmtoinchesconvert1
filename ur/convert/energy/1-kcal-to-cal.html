ہمیں ایک دن میں کتنی کیلوریز کی ضرورت ہے؟

اگر آپ یہ سمجھ چکے ہیں کہ جسم میں توانائی کی پیمائش کے لیے کیلوریز کا استعمال کیا جاتا ہے، تو یہ جاننا بھی ضروری ہے کہ ہمارے روزمرہ کے کام کرنے کے لیے کتنی توانائی کی ضرورت ہے۔سیدھے الفاظ میں، ہمیں کتنی کیلوریز کی ضرورت ہے اس کا انحصار جنس، عمر اور صحت پر ہے۔

مثال کے طور پر خواتین کو مردوں کے مقابلے میں کم توانائی کی ضرورت ہوتی ہے تاکہ وہ اپنے روزمرہ کے کام کر سکیں، جس کے مطابق عورت کو صحت مند رہنے کے لیے کم از کم 2000 کیلوریز کی ضرورت ہوتی ہے، جب کہ اگر مرد صحت مند رہنا چاہتا ہے تو کم از کم 2500 کیلوریز کی ضرورت ہوتی ہے۔

لیکن جب آپ صحت مند رہنے کے لیے برابر کیلوریز لیتے ہیں تو اس سے آپ کے جسمانی وزن میں کمی نہیں ہوتی اور نہ ہی اس سے آپ کا وزن کم ہوتا ہے، اگر آپ اپنا وزن بڑھانا چاہتے ہیں تو آپ کو کیلوریز کی مقدار میں اضافہ کرنا ہوگا اور اگر آپ کو وزن کم کرنا ہے تو آپ کو کیلوری کی مقدار کو کم کرنے کے لئے.

آئیے آپ کو بتاتے ہیں کہ صرف کیلوریز کی مقدار کم کرنے یا بڑھانے سے ہمارے جسم کی ساخت میں کوئی فرق نہیں پڑے گا۔اس سے کوئی فرق نہیں پڑتا ہے کہ آپ کتنا کھاتے ہیں یا آپ کتنا ہی روزہ رکھتے ہیں، اگر آپ کا وزن کم ہے تو آپ کے جسم کی ساخت بدل جائے گی جب آپ سخت محنت کرتے ہیں۔

جب آپ محنت کریں گے تب ہی آپ کا جسم آپ سے زیادہ کیلوریز اپنائے گا، ورنہ وہ اس کیلوریز کو استعمال نہیں کرے گا۔یہ سمجھنا ہوگا کہ ہمارا دماغ اتنا سمجھدار ہے کہ آپ کو وزن کم کرنے یا کم کرنے کی ضرورت ہے۔اگر آپ اپنا وزن بڑھانا چاہتے ہیں تو آپ کو کیلوریز کی تعداد بڑھانے کے ساتھ ساتھ اپنی محنت کے عمل کو تیز کرنا ہوگا تاکہ دماغ آپ کو دی گئی زیادہ کیلوریز استعمال کرے اور آپ کے جسم کی ساخت میں فرق آئے۔

اسی طرح اگر آپ وزن کم کرنا چاہتے ہیں تو آپ کو سخت محنت کرنا ہوگی اور کیلوریز کی مقدار کم کرنی ہوگی تاکہ جسم اضافی کیلوریز استعمال کرکے آپ کی محنت کو پورا کرے اور جسم کی ساخت میں فرق آجائے۔
