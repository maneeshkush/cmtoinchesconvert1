HTML ری ڈائریکشن

HTML ری ڈائریکٹ۔HTML میٹا ریفریش ری ڈائریکشن کوڈ۔

HTML میٹا ریفریش ری ڈائریکٹ کلائنٹ سائڈ ری ڈائریکٹ ہے اور یہ 301 مستقل ری ڈائریکٹ نہیں ہے۔

0 سیکنڈ کے وقفے کے ساتھ ایچ ٹی ایم ایل میٹا ریفریش، گوگل کی طرف سے صفحہ رینک کی منتقلی کے لیے 301 ری ڈائریکٹ کے لیے اشارہ سمجھا جاتا ہے۔

اگر آپ حقیقی 301 مستقل ری ڈائریکٹ کرنا چاہتے ہیں، تو آپ HTML فائلوں میں پی ایچ پی کوڈ کو فعال کرنے کے بعد پی ایچ پی ری ڈائریکٹ کے ساتھ کر سکتے ہیں۔

HTML میٹا ریفریش ری ڈائریکٹ

ری ڈائریکشن ہیڈ سیکشن میں میٹا ریفریش کے ساتھ کیا جاتا ہے۔

فال بیک مقاصد کے لیے باڈی سیکشن میں لنک۔

پرانے صفحہ کو ری ڈائریکشن کوڈ کے ساتھ اس صفحہ کے URL کے ساتھ تبدیل کریں جس پر آپ ری ڈائریکٹ کرنا چاہتے ہیں۔

old-page.html:

<!DOCTYPE html>
<html>
<head>
   <!-- HTML meta refresh URL redirection -->
   <meta http-equiv="refresh"
   content="0; url=http://www.mydomain.com/new-page.html">
</head>
<body>
   <p>The page has moved to:
   <a href="http://www.mydomain.com/new-page.html">this page</a></p>
</body>
</html>

ایچ ٹی ایم ایل میٹا ریفریش ری ڈائریکٹ مثال

html-redirect-test.htm:

<!DOCTYPE html>
<html>
<head>
   <!-- HTML meta refresh URL redirection -->
   <meta http-equiv="refresh"
   content="0; url=https://www.cmtoinchesconvert.com/ur/web/dev/html-redirect.htm">
</head>
<body>
   <p>The page has moved to:
   <a href="https://www.cmtoinchesconvert.com/ur/web/dev/html-redirect.htm">this page</a></p>
</body>
</html>

 

html-redirect-test.htm سے اس صفحہ پر واپس بھیجنے کے لیے اس لنک کو دبائیں:

 

HTML میٹا ریفریش ری ڈائریکٹ ٹیسٹ

HTML کینونیکل لنک ٹیگ ری ڈائریکٹ

کینونیکل لنک ترجیحی یو آر ایل پر ری ڈائریکٹ نہیں ہوتا ہے، لیکن یہ ان ویب سائٹس کے لیے یو آر ایل ری ڈائریکشن کا متبادل ہوسکتا ہے جہاں زیادہ تر ٹریفک سرچ انجنوں سے آتی ہے۔

HTML کینونیکل لنک ٹیگ اس وقت استعمال کیا جا سکتا ہے جب ایک جیسے مواد والے کئی صفحات ہوں اور آپ سرچ انجنوں کو بتانا چاہتے ہیں کہ آپ تلاش کے نتائج میں کون سا صفحہ استعمال کرنا پسند کرتے ہیں۔

کیننیکل لنک ٹیگ ایک ہی ڈومین اور کراس ڈومین سے بھی منسلک ہو سکتا ہے۔

نئے صفحہ سے لنک کرنے کے لیے پرانے صفحہ پر کینونیکل لنک ٹیگ شامل کریں۔

ان صفحات پر کینونیکل لنک ٹیگ شامل کریں جنہیں آپ ترجیحی صفحہ سے لنک کرنے کے لیے سرچ انجنوں کی ٹریفک حاصل کرنے کو ترجیح نہیں دیتے ہیں۔

کینونیکل لنک ٹیگ کو <head> سیکشن میں شامل کیا جانا چاہیے۔

old-page.html:

<link rel="canonical" href="http://www.mydomain.com/new-page.html">
