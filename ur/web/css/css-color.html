سی ایس ایس کا رنگ

رنگ کا کوڈ ان میں سے ایک ہو سکتا ہے:

ہیکس فارمیٹ: #rrggbb

آر جی بی فارمیٹ: آر جی بی (سرخ، سبز، نیلا)

نام کی شکل: نام

مثال

نارنجی رنگ کے لیے:

ہیکس فارمیٹ: #FFA500

RGB فارمیٹ: rgb(255,165,0)

نام کی شکل: سنتری

عنصر کا رنگ ترتیب دینا

مخصوص عنصر کے لیے:

<element style="color: code;"></element>

ایک ہی قسم کے تمام عناصر کے لیے۔ہیڈ سیکشن میں <style> ٹیگ میں کوڈ ڈالیں:

<style>
   element { color: code; }
</style>

مثال

پیراگراف ٹیکسٹ کا رنگ سرخ پر سیٹ کرنا:

مخصوص پیراگراف کا رنگ ترتیب دینا:

<p style="color: #FF0000;">Some text ...</p>

دیکھیں:

کچھ متن...

تمام پیراگراف کا رنگ ترتیب دینا

<style>
   p { color: #FF0000; }
</style>

عنصر کے پس منظر کا رنگ ترتیب دینا

element { background: code; }

مثال

پیراگراف کے پس منظر کا رنگ سرخ پر سیٹ کرنا:

مخصوص پیراگراف کے پس منظر کا رنگ ترتیب دینا:

<p style="background: #FF0000;">Some text ...</p>

دیکھیں:

کچھ متن...

تمام پیراگراف کے پس منظر کا رنگ ترتیب دینا:

<style>
   p { background: #FF0000; }
</style>

عنصر کی سرحد کا رنگ ترتیب دینا

element { border-color: topcode rightcode bottomcode leftcode }

مثال

پیراگراف بارڈر کا رنگ سرخ، سبز، نیلے اور سیاہ پر سیٹ کرنا:

مخصوص پیراگراف کے بارڈر کا رنگ سیٹ کرنا:

<p style="border-color: #FF0000 #00FF00 #0000FF #000000; border-style:solid">Some text ...</p>

دیکھیں:

کچھ متن...

تمام پیراگراف کے بارڈر کا رنگ سیٹ کرنا:

<style>
   p { border-color: #FF0000 #00FF00 #0000FF #000000; }
</style>

سی ایس ایس کلر کوڈز ٹول کی خصوصیات

ہمارا CSS کلر کوڈز ٹول صارفین کو CSS کلر کوڈز کی اجازت دیتا ہے۔اس افادیت کی کچھ نمایاں خصوصیات ذیل میں بیان کی گئی ہیں۔

سادگی

ہمارے CSS کلر کوڈز ٹول کا صارف دوست انٹرفیس آپ کو اس تبدیلی کو فوری طور پر انجام دینے میں آسانی فراہم کرتا ہے۔اس ٹول کی سادگی کی وجہ سے اب آپ کو CSS کلر کوڈز کے لیے پروگرامنگ کی مہارت حاصل کرنے کی ضرورت نہیں ہے۔

کوئی رجسٹریشن نہیں۔

CSS کلر کوڈز استعمال کرنے کے لیے آپ کو کسی بھی رجسٹریشن کے عمل سے گزرنے کی ضرورت نہیں ہے۔اس یوٹیلیٹی کو استعمال کرتے ہوئے، آپ CSS کلر کوڈز ٹول کو جتنی بار چاہیں مفت میں استعمال کر سکتے ہیں۔

پورٹیبلٹی

اس CSS کلر کوڈز تک دنیا کے کسی بھی کونے سے رسائی حاصل کی جا سکتی ہے۔آپ کو اس آن لائن ٹول کی مطابقت کے بارے میں فکر کرنے کی ضرورت نہیں ہے، کیونکہ آپ کسی بھی ڈیوائس سے اس کے ساتھ کلر کوڈز CSS کر سکتے ہیں۔اس CSS کلر کوڈز ٹول تک رسائی اور استعمال کرنے کے لیے آپ کو صرف ایک انٹرنیٹ کنکشن کی ضرورت ہے۔

وقت اور کوشش کو بچاتا ہے۔

CSS کلر کوڈز آپ کو اسی کام کو فوری طور پر مکمل کرنے کی اجازت دیتے ہیں۔آپ کو دستی طریقہ کار پر عمل کرنے کے لیے نہیں کہا جائے گا، کیونکہ اس کے خودکار الگورتھم آپ کے لیے کام کریں گے۔

مطابقت

CSS کلر کوڈز ٹول تمام آپریٹنگ سسٹمز پر بالکل کام کرتا ہے۔چاہے آپ کے پاس میک، آئی او ایس، اینڈرائیڈ، ونڈوز، یا لینکس ڈیوائس ہو، آپ آسانی سے اس آن لائن یوٹیلیٹی کو بغیر کسی پریشانی کے استعمال کر سکتے ہیں۔

100% مفت

اس CSS کلر کوڈز ٹول کو استعمال کرنے کے لیے آپ کو کسی بھی رجسٹریشن کے عمل سے گزرنے کی ضرورت نہیں ہے۔آپ اس یوٹیلیٹی کو مفت میں استعمال کر سکتے ہیں اور بغیر کسی پابندی کے لامحدود CSS کلر کوڈز کر سکتے ہیں۔
