CSS ਰੰਗ

ਰੰਗ ਕੋਡ ਇਹਨਾਂ ਵਿੱਚੋਂ ਇੱਕ ਹੋ ਸਕਦਾ ਹੈ:

ਹੈਕਸ ਫਾਰਮੈਟ: #rrggbb

RGB ਫਾਰਮੈਟ: rgb (ਲਾਲ, ਹਰਾ, ਨੀਲਾ)

ਨਾਮ ਫਾਰਮੈਟ: ਨਾਮ

ਉਦਾਹਰਨ

ਸੰਤਰੀ ਰੰਗ ਲਈ:

ਹੈਕਸ ਫਾਰਮੈਟ: #FFA500

RGB ਫਾਰਮੈਟ: rgb(255,165,0)

ਨਾਮ ਫਾਰਮੈਟ: ਸੰਤਰੀ

ਤੱਤ ਦਾ ਰੰਗ ਸੈੱਟ ਕਰਨਾ

ਖਾਸ ਤੱਤ ਲਈ:

<element style="color: code;"></element>

ਇੱਕੋ ਕਿਸਮ ਦੇ ਸਾਰੇ ਤੱਤਾਂ ਲਈ।ਹੈੱਡ ਸੈਕਸ਼ਨ ਵਿੱਚ <style> ਟੈਗ ਵਿੱਚ ਕੋਡ ਪਾਓ:

<style>
   element { color: code; }
</style>

ਉਦਾਹਰਨ

ਪੈਰਾਗ੍ਰਾਫ ਟੈਕਸਟ ਰੰਗ ਨੂੰ ਲਾਲ ਸੈੱਟ ਕਰਨਾ:

ਖਾਸ ਪੈਰੇ ਦਾ ਰੰਗ ਸੈੱਟ ਕਰਨਾ:

<p style="color: #FF0000;">Some text ...</p>

ਵੇਖੋ:

ਕੁਝ ਟੈਕਸਟ ...

ਸਾਰੇ ਪੈਰਿਆਂ ਦਾ ਰੰਗ ਸੈੱਟ ਕਰਨਾ

<style>
   p { color: #FF0000; }
</style>

ਤੱਤ ਬੈਕਗ੍ਰਾਊਂਡ ਰੰਗ ਸੈੱਟ ਕਰਨਾ

element { background: code; }

ਉਦਾਹਰਨ

ਪੈਰਾਗ੍ਰਾਫ ਬੈਕਗ੍ਰਾਊਂਡ ਰੰਗ ਨੂੰ ਲਾਲ 'ਤੇ ਸੈੱਟ ਕਰਨਾ:

ਖਾਸ ਪੈਰੇ ਦਾ ਪਿਛੋਕੜ ਰੰਗ ਸੈੱਟ ਕਰਨਾ:

<p style="background: #FF0000;">Some text ...</p>

ਵੇਖੋ:

ਕੁਝ ਟੈਕਸਟ ...

ਸਾਰੇ ਪੈਰਿਆਂ ਦਾ ਪਿਛੋਕੜ ਰੰਗ ਸੈੱਟ ਕਰਨਾ:

<style>
   p { background: #FF0000; }
</style>

ਤੱਤ ਬਾਰਡਰ ਦਾ ਰੰਗ ਸੈੱਟ ਕਰਨਾ

element { border-color: topcode rightcode bottomcode leftcode }

ਉਦਾਹਰਨ

ਪੈਰਾਗ੍ਰਾਫ ਬਾਰਡਰ ਰੰਗ ਨੂੰ ਲਾਲ, ਹਰਾ, ਨੀਲਾ ਅਤੇ ਕਾਲਾ ਸੈੱਟ ਕਰਨਾ:

ਖਾਸ ਪੈਰੇ ਦਾ ਬਾਰਡਰ ਰੰਗ ਸੈੱਟ ਕਰਨਾ:

<p style="border-color: #FF0000 #00FF00 #0000FF #000000; border-style:solid">Some text ...</p>

ਵੇਖੋ:

ਕੁਝ ਟੈਕਸਟ ...

ਸਾਰੇ ਪੈਰਿਆਂ ਦਾ ਬਾਰਡਰ ਰੰਗ ਸੈੱਟ ਕਰਨਾ:

CSS ਕਲਰ ਕੋਡ ਟੂਲ ਦੀਆਂ ਵਿਸ਼ੇਸ਼ਤਾਵਾਂ

ਸਾਡਾ CSS ਕਲਰ ਕੋਡ ਟੂਲ ਉਪਭੋਗਤਾਵਾਂ ਨੂੰ CSS ਕਲਰ ਕੋਡ ਦੀ ਆਗਿਆ ਦਿੰਦਾ ਹੈ।ਇਸ ਸਹੂਲਤ ਦੀਆਂ ਕੁਝ ਪ੍ਰਮੁੱਖ ਵਿਸ਼ੇਸ਼ਤਾਵਾਂ ਹੇਠਾਂ ਦੱਸੀਆਂ ਗਈਆਂ ਹਨ।

ਸਾਦਗੀ

ਸਾਡੇ CSS ਕਲਰ ਕੋਡ ਟੂਲ ਦਾ ਉਪਭੋਗਤਾ-ਅਨੁਕੂਲ ਇੰਟਰਫੇਸ ਤੁਹਾਨੂੰ ਇਸ ਪਰਿਵਰਤਨ ਨੂੰ ਤੁਰੰਤ ਕਰਨ ਦੀ ਸੌਖ ਪ੍ਰਦਾਨ ਕਰਦਾ ਹੈ।ਇਸ ਟੂਲ ਦੀ ਸਰਲਤਾ ਦੇ ਕਾਰਨ ਤੁਹਾਨੂੰ ਹੁਣ CSS ਕਲਰ ਕੋਡਾਂ ਲਈ ਪ੍ਰੋਗਰਾਮਿੰਗ ਹੁਨਰ ਹਾਸਲ ਕਰਨ ਦੀ ਲੋੜ ਨਹੀਂ ਹੈ।
