PHP ਰੀਡਾਇਰੈਕਟ

PHP ਪੰਨੇ ਤੋਂ URL ਤੱਕ ਰੀਡਾਇਰੈਕਟ ਕਰੋ।PHP 301 ਰੀਡਾਇਰੈਕਟ.

ਇਸ PHP ਰੀਡਾਇਰੈਕਸ਼ਨ ਨੂੰ HTTP ਜਵਾਬ ਸਥਿਤੀ ਕੋਡ ਵਾਪਸ ਕਰਨਾ ਚਾਹੀਦਾ ਹੈ: 301 ਸਥਾਈ ਤੌਰ 'ਤੇ ਚਲੇ ਗਏ।

ਖੋਜ ਇੰਜਣ ਪੁਰਾਣੇ URL ਤੋਂ ਨਵੇਂ URL ਵਿੱਚ ਪੇਜ ਰੈਂਕ ਨੂੰ ਟ੍ਰਾਂਸਫਰ ਕਰਨ ਲਈ 301 ਜਵਾਬ ਸਥਿਤੀ ਕੋਡ ਦੀ ਵਰਤੋਂ ਕਰਦੇ ਹਨ।

PHP ਹੈਡਰ ਰੀਡਾਇਰੈਕਟ

ਪੁਰਾਣੇ-page.php ਕੋਡ ਨੂੰ ਰੀਡਾਇਰੈਕਸ਼ਨ ਕੋਡ ਨਾਲ new-page.php ਵਿੱਚ ਬਦਲੋ।

old-page.php:

<?php
// PHP permanent URL redirection
header("Location: http://www.domain.com/new-page.php", true, 301);
exit();
?>

ਪੁਰਾਣੇ ਪੰਨੇ ਵਿੱਚ .php ਫਾਈਲ ਐਕਸਟੈਂਸ਼ਨ ਹੋਣੀ ਚਾਹੀਦੀ ਹੈ।

ਨਵਾਂ ਪੰਨਾ ਕਿਸੇ ਵੀ ਐਕਸਟੈਂਸ਼ਨ ਨਾਲ ਹੋ ਸਕਦਾ ਹੈ।

PHP ਰੀਡਾਇਰੈਕਟ ਉਦਾਹਰਨਾਂ

ਉਦਾਹਰਨ #1

php-redirect-test.php

<?php
// PHP permanent URL redirection test
header("Location: https://www.cmtoinchesconvert.com/pa/web/dev/php-redirect.html", true, 301);
exit();
?>

 

php-redirect-test.php ਤੋਂ ਇਸ ਪੰਨੇ 'ਤੇ ਵਾਪਸ ਭੇਜਣ ਲਈ ਇਸ ਲਿੰਕ ਨੂੰ ਦਬਾਓ:

 

PHP ਰੀਡਾਇਰੈਕਟ ਟੈਸਟ - PHP ਫਾਈਲ

ਉਦਾਹਰਨ #2

php-redirect-test.htm

<?php
// PHP permanent URL redirection test
header("Location: https://www.cmtoinchesconvert.com/pa/web/dev/php-redirect.html", true, 301);
exit();
?>

 

html ਫਾਈਲ php-redirect-test.htm ਤੋਂ PHP ਰੀਡਾਇਰੈਕਸ਼ਨ ਆਮ ਤੌਰ 'ਤੇ .html ਫਾਈਲ ਐਕਸਟੈਂਸ਼ਨ ਦੇ ਕਾਰਨ ਕੰਮ ਨਹੀਂ ਕਰੇਗਾ, ਜਦੋਂ ਤੱਕ ਇਹ .htaccess ਜਾਂ httpd.conf ਫਾਈਲ ਵਿੱਚ ਸਮਰੱਥ ਨਹੀਂ ਹੁੰਦਾ:

 

PHP ਰੀਡਾਇਰੈਕਟ ਟੈਸਟ - HTML ਫਾਈਲ

 

HTML ਫਾਈਲਾਂ ਵਿੱਚ PHP ਨੂੰ ਸਮਰੱਥ ਕਰਨ ਲਈ ਇਸ ਕੋਡ ਨੂੰ .htaccess ਜਾਂ httpd.conf ਫਾਈਲ ਵਿੱਚ ਸ਼ਾਮਲ ਕਰੋ:

Addtype application/x-httpd-php .htm .html
