Πρόσθεση αριθμών με εκθέτες

Η προσθήκη εκθετών γίνεται υπολογίζοντας πρώτα κάθε εκθέτη και στη συνέχεια προσθέτοντας:

an + bm

Παράδειγμα:

42 + 25 = 4⋅4+2⋅2⋅2⋅2⋅2 = 16+32 = 48

Προσθέτοντας τις ίδιες βάσεις b και τους εκθέτες n:

bn + bn = 2bn

Παράδειγμα:

42 + 42 = 2⋅42 = 2⋅4⋅4 = 32

Προσθήκη αρνητικών εκθετών

Η προσθήκη αρνητικών εκθετών γίνεται υπολογίζοντας πρώτα κάθε εκθέτη και στη συνέχεια προσθέτοντας:

a-n + b-m = 1/an + 1/bm

Παράδειγμα:

4-2 + 2-5 = 1/42 + 1/25 = 1/(4⋅4)+1/(2⋅2⋅2⋅2⋅2) = 1/16+1/32 = 0.09375

Προσθήκη κλασματικών εκθετών

Η προσθήκη κλασματικών εκθετών γίνεται ανεβάζοντας πρώτα κάθε εκθέτη και στη συνέχεια προσθέτοντας:

an/m + bk/j

Παράδειγμα:

33/2 + 25/2 = √(33) + √(25) = √(27) + √(32) = 5.196 + 5.657 = 10.853

 

Προσθέτοντας τις ίδιες βάσεις b και τους εκθέτες n/m:

bn/m + bn/m = 2bn/m

Παράδειγμα:

42/3 + 42/3 = 2⋅42/3 = 2 ⋅ 3√(42) = 5.04

Προσθήκη μεταβλητών με εκθέτες

Η προσθήκη εκθετών γίνεται υπολογίζοντας πρώτα κάθε εκθέτη και στη συνέχεια προσθέτοντας:

xn + xm

Με τους ίδιους εκθέτες:

xn + xn = 2xn

Παράδειγμα:

x2 + x2 = 2x2
