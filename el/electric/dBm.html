ορισμός dBm

Το dBm ή ντεσιμπέλ-μιλιβάτ είναι μια μονάδα ηλεκτρικής ισχύος σε ντεσιμπέλ (dB) , που αναφέρεται σε 1 milliwatt (mW).

 

Άρα Η ισχύς σε ντεσιμπέλ-μιλιβάτ ( P (dBm) ) ισούται με 10 φορές το λογάριθμο βάσης του 10 της ισχύος σε milliwatts ( P (mW) ).

P(dBm) = 10 ⋅ log10( P(mW) / 1mW)

 

Άρα, η ισχύς σε milliwatts ( P (mW) ) είναι ίση με 1mW επί 10 που αυξάνεται από την ισχύ σε decibel-milliwatts ( P (dBm) ) δια του [10].

P(mW) = 1mW ⋅ 10(P(dBm) / 10)

 

1 milliwatt ισούται με 0 dBm:

1mW = 0dBm

 

2 watt είναι ίσα με 33dBm:

2W = 2000mW = 33dBm

Πώς να μετατρέψετε mW σε dBm

Πώς να μετατρέψετε την ισχύ σε milliwatts (mW) σε dBm.

Άρα η ισχύς σε dBm είναι ίση με τον λογάριθμο βάσης 10 της ισχύος σε milliwatts (mW):

P(dBm) = 10 ⋅ log10( P(mW) / 1mW)

 

Για παράδειγμα: 1 ποια είναι η ισχύς σε dBm για κατανάλωση ισχύος 1000 mW;

Λύση:

P(dBm) = 10 ⋅ log10( 1000mW / 1mW) = 30dBm

Για παράδειγμα: 2 ποια είναι η ισχύς σε dBm για κατανάλωση ισχύος 10 mW;

Λύση:

P(dBm) = 10 ⋅ log10( 10mW / 1mW) = 10dBm

Πώς να μετατρέψετε dBm σε mW

Πώς να μετατρέψετε την ισχύ σε dBm σε milliwatts (mW).

Άρα Η ισχύς σε milliwatts ( P (mW) ) είναι ίση με 10 που αυξάνεται από την ισχύ σε dBm ( P (dBm) ) διαιρούμενη με το 10;

P(mW) = 1mW ⋅ 10(P(dBm) / 10)

 

Για παράδειγμα: 1 ποια είναι η ισχύς σε milliwatts για κατανάλωση ισχύος 10dBm;

Λύση:

P(mW) = 1mW ⋅ 10(10dBm / 10) = 10mW

Για παράδειγμα: 2 ποια είναι η ισχύς σε milliwatts για κατανάλωση ισχύος 30dBm;

Λύση:

P(mW) = 1mW ⋅ 10(30dBm / 10) = 1000mW

Πώς να μετατρέψετε Watt σε dBm

Πώς να μετατρέψετε την ισχύ σε watt (W) σε dBm.

Η ισχύς σε dBm είναι ίση με τον λογάριθμο βάσης 10 της ισχύος σε watt (W) συν 30 dB:

P(dBm) = 10 ⋅ log10( P(W) / 1W) + 30

 

Για παράδειγμα: 1 ποια είναι η ισχύς σε dBm για κατανάλωση ισχύος 10 W;

Λύση:

P(dBm) = 10 ⋅ log10( 10W / 1W ) + 30 = 40dBm

Για παράδειγμα: 2 ποια είναι η ισχύς σε dBm για κατανάλωση ισχύος 1000W;

Λύση:

P(dBm) = 10 ⋅ log10( 1000W / 1W ) + 30 = 60dBm

Πώς να μετατρέψετε dBm σε Watt

Πώς να μετατρέψετε την ισχύ σε dBm σε Watt (W).

Η ισχύς σε watt ( P (W) ) είναι ίση με 10 αυξημένη από την ισχύ σε dBm ( P (dBm) ) μείον 30 dB διαιρούμενο με 10:

P(W) = 1W ⋅ 10( (P(dBm)- 30) / 10)

 

Για παράδειγμα: 1 ποια είναι η ισχύς σε watt για κατανάλωση ισχύος 50dBm;

Λύση:

P(W) = 1W ⋅ 10( (50dBm - 30) / 10) = 100W

Για παράδειγμα: 2 ποια είναι η ισχύς σε watt για κατανάλωση ισχύος 60dBm;

Λύση:

P(W) = 1W ⋅ 10( (60dBm - 30) / 10) = 1000W

Πώς να μετατρέψετε dBW σε dBm

Πώς να μετατρέψετε την ισχύ σε dBW σε dBm.

Άρα Η ισχύς σε dBm είναι ίση με τον λογάριθμο βάσης 10 της ισχύος σε watt (W).

P(dBm) = P(dBW) + 30

 

Για παράδειγμα: ποια είναι η ισχύς σε dBm για κατανάλωση ισχύος 30 dBW;

Λύση:

P(dBm) = 30dBW + 30 = 60dBm

Πώς να μετατρέψετε dBm σε dBW

Πώς να μετατρέψετε την ισχύ σε dBm σε dBW.

Άρα, η ισχύς σε dBW ( P (dBW) ) ισούται με 10 αυξημένη από την ισχύ σε dBm ( P (dBm) ) διαιρούμενη με το 10.

P(dBW) = P(dBm) - 30

 

Για παράδειγμα: ποια είναι η ισχύς σε watt για κατανάλωση ισχύος 50dBm;

Λύση:

P(dBW) = 50dBm - 30 = 20dBW

Πώς να μετατρέψετε dB σε dBm

Το dB είναι μια σχετική μονάδα που περιγράφει το κέρδος και το dBm είναι μια απόλυτη μονάδα που αναφέρεται στο 1 milliwatt (mW).

Επομένως, δεν μπορείτε να μετατρέψετε dB σε dBm.
