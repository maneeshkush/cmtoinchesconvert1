Πώς να εξοικονομήσετε ηλεκτρική ενέργεια

Πώς να εξοικονομήσετε χρήματα στους λογαριασμούς ρεύματος. 40 συμβουλές εξοικονόμησης ηλεκτρικής ενέργειας στο σπίτι.

  1. Κλείστε τις κουρτίνες το βράδυ για να μειώσετε την απώλεια θερμότητας.
  2. Χρησιμοποιήστε έναν θερμαντήρα χώρου αντί να ενεργοποιήσετε τον θερμοστάτη για να θερμάνετε μια μικρή περιοχή.
  3. Χρησιμοποιήστε ένα άπλωμα ή σχάρα στεγνώματος αντί για στεγνωτήριο για να στεγνώσετε τα ρούχα.
  4. Χρησιμοποιήστε ένα πολύπριζο για να απενεργοποιήσετε πολλά ηλεκτρονικά ταυτόχρονα.
  5. Απενεργοποιήστε το πολύπριζο όταν δεν χρησιμοποιείτε ηλεκτρονικά.
  6. Χρησιμοποιήστε ηλεκτρική κουζίνα αντί για ηλεκτρική κουζίνα.
  7. Σβήστε τη σόμπα λίγα λεπτά πριν τελειώσει το μαγείρεμα του φαγητού για να αφήσετε την υπολειπόμενη θερμότητα να ολοκληρώσει τη δουλειά.
  8. Χρησιμοποιήστε το φούρνο μικροκυμάτων ή το φούρνο τοστιέρας αντί για τη σόμπα ή το φούρνο για μικρές εργασίες μαγειρέματος.
  9. Χρησιμοποιήστε μια αργή κουζίνα για να εξοικονομήσετε ενέργεια στο μαγείρεμα.
  10. Εγκαταστήστε ηλιακούς συλλέκτες στη στέγη σας για την παραγωγή ηλεκτρικής ενέργειας.
  11. Εγκατάσταση ηλιακού θερμοσίφωνα.
  12. Μονώστε το σπίτι σας.
  13. Τοποθετήστε παντζούρια.
  14. Τοποθετήστε διπλά τζάμια.
  15. Αγοράστε συσκευές με πιστοποίηση Energy Star.
  16. Αγοράστε συσκευές με χαμηλή κατανάλωση ρεύματος.
  17. Ελέγξτε τη θερμομόνωση του σπιτιού σας.
  18. Απενεργοποιήστε τις συσκευές και τα gadget που βρίσκονται σε κατάσταση αναμονής.
  19. Προτιμήστε τον ανεμιστήρα από το A/C
  20. Προτιμήστε τη θέρμανση A/C από την ηλεκτρική θέρμανση/αέριο/ξύλο
  21. Προτιμήστε το inverter A/C από το κανονικό on/off A/C
  22. Ρυθμίστε τον θερμοστάτη A/C σε μέτρια θερμοκρασία.
  23. Χρησιμοποιήστε A/C τοπικά για ένα δωμάτιο αντί για ολόκληρο το σπίτι.
  24. Αποφύγετε να ανοίγετε συχνά την πόρτα του ψυγείου.
  25. Αφήστε αρκετό χώρο μεταξύ του ψυγείου και του τοίχου για να επιτρέπεται ο αερισμός.
  26. Σβήστε το φως όταν φεύγετε από το δωμάτιο.
  27. Εγκαταστήστε ανιχνευτή παρουσίας για να απενεργοποιήσετε τον φωτισμό όταν βγαίνετε από το δωμάτιο.
  28. Χρησιμοποιήστε λαμπτήρες χαμηλής ισχύος.
  29. Πλύνετε τα ρούχα σας σε κρύο νερό.
  30. Χρησιμοποιήστε μικρότερο πρόγραμμα πλυντηρίου ρούχων.
  31. Γεμίστε το πλυντήριο / στεγνωτήριο / πλυντήριο πιάτων πριν τη λειτουργία.
  32. Φοράτε ρούχα που ταιριάζουν στην τρέχουσα θερμοκρασία.
  33. Φορέστε χοντρά ρούχα για να ζεσταθείτε
  34. Φορέστε ελαφριά ρούχα για να παραμείνετε δροσεροί
  35. Χρησιμοποιήστε τις σκάλες αντί για το ασανσέρ.
  36. Ορίστε λειτουργίες εξοικονόμησης ενέργειας υπολογιστή
  37. Χρησιμοποιήστε σχάρα στεγνώματος ρούχων αντί για ηλεκτρικό στεγνωτήριο ρούχων
  38. Βάλτε την ακριβή ποσότητα νερού που χρειάζεστε στον ηλεκτρικό σας βραστήρα
  39. Πήγαινε για ύπνο νωρίς.
  40. Χρησιμοποιήστε το ηλιακό φως αντί για το τεχνητό φως
  41. Αγοράστε τηλεόραση LED αντί για πλάσμα
  42. Μειώστε τη φωτεινότητα της οθόνης τηλεόρασης/οθόνης/τηλεφώνου
  43. Αγοράστε υπολογιστή με CPU/GPU χαμηλής ισχύος (TDP).
  44. Αγοράστε υπολογιστή με αποδοτική μονάδα τροφοδοσίας (PSU)
  45. Προτιμήστε το φως LED έναντι των λαμπτήρων πυρακτώσεως.
  46. Αποσυνδέστε τον ηλεκτρικό φορτιστή όταν ολοκληρωθεί η φόρτιση.
  47. Προτιμήστε το φούρνο μικροκυμάτων από το φούρνο
  48. Χρησιμοποιήστε παρακολούθηση χρήσης ηλεκτρικής ενέργειας
  49. Σβήνετε τα φώτα και τα ηλεκτρονικά όταν δεν τα χρησιμοποιείτε.
  50. Χρησιμοποιήστε ενεργειακά αποδοτικές συσκευές και λαμπτήρες.
  51. Ρυθμίστε τον θερμοστάτη σας σε χαμηλότερη θερμοκρασία το χειμώνα και υψηλότερη το καλοκαίρι.
  52. Φυτέψτε δέντρα ή εγκαταστήστε συσκευές σκίασης για να μπλοκάρετε τις ακτίνες του ήλιου και να διατηρήσετε το σπίτι σας πιο δροσερό το καλοκαίρι.
  53. Μονώστε το σπίτι σας για να το διατηρήσετε ζεστό το χειμώνα και δροσερό το καλοκαίρι.
  54. Χρησιμοποιήστε έναν προγραμματιζόμενο θερμοστάτη για να ρυθμίσετε αυτόματα τη θερμοκρασία όταν δεν είστε σπίτι.
  55. Αποσυνδέστε τα ηλεκτρονικά και τις συσκευές όταν δεν χρησιμοποιούνται, καθώς μπορούν ακόμα να χρησιμοποιούν ενέργεια όταν είναι απενεργοποιημένα αλλά συνδεδεμένα.
  56. Εγκαταστήστε κεφαλές ντους χαμηλής ροής για να μειώσετε τη χρήση νερού και ενέργειας.
  57. Διορθώστε τις διαρροές στο σπίτι σας για να μειώσετε τη σπατάλη νερού.
  58. Λειτουργήστε το πλυντήριο πιάτων και το πλυντήριο ρούχων μόνο με πλήρες φορτίο.
  59. Χρησιμοποιήστε κρύο νερό για το πλύσιμο των ρούχων για εξοικονόμηση ενέργειας στο νερό θέρμανσης.
  60. Στεγνώστε τα ρούχα έξω σε ένα άπλωμα αντί να χρησιμοποιήσετε στεγνωτήριο.
  61. Χρησιμοποιήστε χύτρα ταχύτητας ή αργή κουζίνα αντί για ηλεκτρική κουζίνα ή φούρνο για να μαγειρέψετε φαγητό.
  62. Χρησιμοποιήστε φούρνο μικροκυμάτων αντί για φούρνο για εξοικονόμηση ενέργειας όταν μαγειρεύετε μικροαντικείμενα.
  63. Χρησιμοποιήστε φούρνο τοστιέρα ή ηλεκτρικό βραστήρα αντί για εστίες μαγειρέματος για εξοικονόμηση ενέργειας όταν βράζετε νερό ή φρυγανίζετε ψωμί.
  64. Κλείστε τις συσκευές και τα φώτα όταν φεύγετε από ένα δωμάτιο.
  65. Χρησιμοποιήστε φυσικό φως αντί για τεχνητό φως όποτε είναι δυνατόν.
  66. Χρησιμοποιήστε ένα πολύπριζο για να απενεργοποιήσετε πολλά ηλεκτρονικά ταυτόχρονα.
  67. Χρησιμοποιήστε έναν ανεμιστήρα οροφής για την κυκλοφορία του αέρα αντί να ενεργοποιήσετε τον κλιματισμό.
  68. Χρησιμοποιήστε ένα άπλωμα ή σχάρα στεγνώματος αντί για στεγνωτήριο για να στεγνώσετε τα ρούχα.
  69. Χρησιμοποιήστε ένα χειροκίνητο χλοοκοπτικό αντί για ένα με αέριο.
  70. Χρησιμοποιήστε ένα επαναχρησιμοποιούμενο μπουκάλι νερού αντί για πλαστικά μπουκάλια μιας χρήσης.
  71. Ανακυκλώστε χαρτί, πλαστικό και μέταλλο για εξοικονόμηση ενέργειας στην παραγωγή.
  72. Αγοράστε μεταχειρισμένα για να παρατείνετε τη διάρκεια ζωής των προϊόντων και να μειώσετε τα απόβλητα.
  73. Υποστηρίξτε καθαρές πηγές ενέργειας όπως η ηλιακή ή η αιολική ενέργεια.
  74. Χρησιμοποιήστε δημόσια μέσα μεταφοράς, ομαδική χρήση αυτοκινήτου ή περπατήστε ή ποδήλατο αντί να οδηγείτε μόνοι.
  75. Φουσκώστε σωστά τα ελαστικά του αυτοκινήτου σας για να βελτιώσετε την απόδοση καυσίμου.
  76. Χρησιμοποιήστε το cruise control στον αυτοκινητόδρομο για εξοικονόμηση καυσίμου.
  77. Αποφύγετε να αφήνετε το αυτοκίνητό σας στο ρελαντί για μεγάλες χρονικές περιόδους.
  78. Συνδυάστε τα θελήματα σε ένα ταξίδι για να μειώσετε τον αριθμό των φορών που πρέπει να οδηγείτε.
  79. Εγκαταστήστε μια τουαλέτα χαμηλής ροής για να μειώσετε τη χρήση νερού.
  80. Διορθώστε τυχόν ρεύματα στο σπίτι σας για να μειώσετε την απώλεια θερμότητας.
  81. Χρησιμοποιήστε πώματα ρεύματος σε πόρτες και παράθυρα για να μειώσετε την απώλεια θερμότητας.
  82. Χρησιμοποιήστε χύτρα ταχύτητας για εξοικονόμηση ενέργειας στο μαγείρεμα.
  83. Χρησιμοποιήστε ψησταριά αερίου αντί για ηλεκτρική ψησταριά.
  84. Χρησιμοποιήστε πρόγραμμα περιήγησης/εφαρμογές με Dark mode
