Μέγεθος εσωτερικού παραθύρου:

Αριστερά, επάνω συντεταγμένες οθόνης του παραθύρου:

Ανίχνευση ανάλυσης οθόνης με Javascript

Μπορείτε να λάβετε το μέγεθος του εξωτερικού παραθύρου με window.outerWidth και window.outerHeight .

Μπορείτε να λάβετε το μέγεθος του εσωτερικού παραθύρου με window.innerWidth και window.innerHeight .

Με τον IE μπορείτε να το αποκτήσετε με document.body.clientWidth και document.body.clientHeight .

Μπορείτε να βρείτε τη θέση του παραθύρου της οθόνης με window.screenLeft και window.screenTop .

Για παράδειγμα:

<script>
alert(window.outerWidth+' x '+window.outerHeight);
</script>

Χαρακτηριστικά του Εργαλείου μεγέθους παραθύρου

Το Εργαλείο μεγέθους παραθύρου επιτρέπει στους χρήστες να έχουν μέγεθος παραθύρου. Μερικά από τα κύρια χαρακτηριστικά αυτού του βοηθητικού προγράμματος εξηγούνται παρακάτω.

Απλότητα

Η φιλική προς το χρήστη διεπαφή του Εργαλείου μεγέθους παραθύρου σάς παρέχει την ευκολία να πραγματοποιήσετε αυτήν τη μετατροπή άμεσα. Δεν χρειάζεται πλέον να αποκτήσετε δεξιότητες προγραμματισμού για το μέγεθος του παραθύρου λόγω της απλότητας αυτού του Εργαλείου.

Χωρίς Εγγραφή

Δεν χρειάζεται να περάσετε από καμία διαδικασία εγγραφής για να χρησιμοποιήσετε το μέγεθος του παραθύρου. Χρησιμοποιώντας αυτό το βοηθητικό πρόγραμμα, μπορείτε να χρησιμοποιήσετε το Εργαλείο μεγέθους παραθύρου όσες φορές θέλετε δωρεάν.

Φορητότητα

Αυτό το μέγεθος παραθύρου είναι προσβάσιμο από οποιαδήποτε γωνιά του κόσμου. Δεν χρειάζεται να ανησυχείτε για τη συμβατότητα αυτού του διαδικτυακού εργαλείου, καθώς μπορείτε να το μέγεθος παραθύρου με αυτό από οποιαδήποτε συσκευή. Το μόνο που χρειάζεστε είναι μια σύνδεση στο Διαδίκτυο για πρόσβαση και χρήση αυτού του Εργαλείου μεγέθους παραθύρου.

Εξοικονομεί χρόνο και κόπο

Το μέγεθος παραθύρου σάς επιτρέπει να ολοκληρώσετε την ίδια εργασία αμέσως. Δεν θα σας ζητηθεί να ακολουθήσετε μη αυτόματες διαδικασίες, καθώς οι αυτοματοποιημένοι αλγόριθμοί του θα κάνουν τη δουλειά για εσάς.

Συμβατότητα

Το Εργαλείο μεγέθους παραθύρου λειτουργεί τέλεια σε όλα τα λειτουργικά συστήματα. Είτε έχετε συσκευή Mac, iOS, Android, Windows ή Linux, μπορείτε εύκολα να χρησιμοποιήσετε αυτό το διαδικτυακό βοηθητικό πρόγραμμα χωρίς να αντιμετωπίσετε καμία ταλαιπωρία.

100% Δωρεάν

Δεν χρειάζεται να περάσετε από καμία διαδικασία εγγραφής για να χρησιμοποιήσετε αυτό το Εργαλείο μεγέθους παραθύρου. Μπορείτε να χρησιμοποιήσετε αυτό το βοηθητικό πρόγραμμα δωρεάν και να κάνετε απεριόριστο μέγεθος παραθύρου χωρίς περιορισμούς.
