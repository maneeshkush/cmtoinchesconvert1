Χρώμα CSS

Ο χρωματικός κώδικας μπορεί να είναι ένας από τους:

Μορφή δεκαεξαδικού: #rrggbb

Μορφή RGB: rgb (κόκκινο, πράσινο, μπλε)

Μορφή ονόματος: όνομα

Παράδειγμα

Για πορτοκαλί χρώμα:

Δεκαεξαδική μορφή: #FFA500

Μορφή RGB: rgb(255,165,0)

Μορφή ονόματος: πορτοκαλί

Ρύθμιση χρώματος στοιχείου

Για συγκεκριμένο στοιχείο:

<element style="color: code;"></element>

Για όλα τα στοιχεία του ίδιου τύπου. Βάλτε κώδικα στην ετικέτα <style> στην ενότητα head:

<style>
   element { color: code; }
</style>

Παράδειγμα

Ρύθμιση χρώματος κειμένου παραγράφου σε κόκκινο:

Ρύθμιση χρώματος συγκεκριμένης παραγράφου:

<p style="color: #FF0000;">Some text ...</p>

Θέα:

Κάποιο κείμενο...

Ρύθμιση χρώματος όλων των παραγράφων

<style>
   p { color: #FF0000; }
</style>

Ρύθμιση χρώματος φόντου στοιχείου

element { background: code; }

Παράδειγμα

Ρύθμιση του χρώματος φόντου της παραγράφου σε κόκκινο:

Ρύθμιση χρώματος φόντου συγκεκριμένης παραγράφου:

<p style="background: #FF0000;">Some text ...</p>

Θέα:

Κάποιο κείμενο...

Ρύθμιση χρώματος φόντου για όλες τις παραγράφους:

<style>
   p { background: #FF0000; }
</style>

Ρύθμιση χρώματος περιγράμματος στοιχείου

element { border-color: topcode rightcode bottomcode leftcode }

Παράδειγμα

Ορισμός χρώματος περιγράμματος παραγράφου σε κόκκινο, πράσινο, μπλε και μαύρο:

Ορισμός χρώματος περιγράμματος συγκεκριμένης παραγράφου:

<p style="border-color: #FF0000 #00FF00 #0000FF #000000; border-style:solid">Some text ...</p>

Θέα:

Κάποιο κείμενο...

Ορισμός χρώματος περιγράμματος όλων των παραγράφων:

<style>
   p { border-color: #FF0000 #00FF00 #0000FF #000000; }
</style>

Χαρακτηριστικά του Εργαλείου κωδικών χρωμάτων CSS

Το Εργαλείο κωδικών χρωμάτων CSS επιτρέπει στους χρήστες να χρησιμοποιούν κωδικούς χρωμάτων CSS. Μερικά από τα κύρια χαρακτηριστικά αυτού του βοηθητικού προγράμματος εξηγούνται παρακάτω.

Απλότητα

Η φιλική προς το χρήστη διεπαφή του Εργαλείου κωδικών χρωμάτων CSS σας παρέχει την ευκολία να πραγματοποιήσετε αυτήν τη μετατροπή άμεσα. Δεν χρειάζεται πλέον να αποκτήσετε δεξιότητες προγραμματισμού για χρωματικούς κώδικες CSS λόγω της απλότητας αυτού του Εργαλείου.

Χωρίς Εγγραφή

Δεν χρειάζεται να περάσετε από καμία διαδικασία εγγραφής για να χρησιμοποιήσετε τους χρωματικούς κωδικούς CSS. Χρησιμοποιώντας αυτό το βοηθητικό πρόγραμμα, μπορείτε να χρησιμοποιήσετε το Εργαλείο κωδικών χρωμάτων CSS όσες φορές θέλετε δωρεάν.

Φορητότητα

Μπορείτε να έχετε πρόσβαση σε αυτούς τους χρωματικούς κωδικούς CSS από οποιαδήποτε γωνιά του κόσμου. Δεν χρειάζεται να ανησυχείτε για τη συμβατότητα αυτού του διαδικτυακού εργαλείου, καθώς μπορείτε να κωδικοποιήσετε χρώμα CSS με αυτό από οποιαδήποτε συσκευή. Το μόνο που χρειάζεστε είναι μια σύνδεση στο διαδίκτυο για πρόσβαση και χρήση αυτού του Εργαλείου κωδικών χρωμάτων CSS.

Εξοικονομεί χρόνο και κόπο

Οι χρωματικοί κωδικοί CSS σάς επιτρέπουν να ολοκληρώσετε την ίδια εργασία αμέσως. Δεν θα σας ζητηθεί να ακολουθήσετε μη αυτόματες διαδικασίες, καθώς οι αυτοματοποιημένοι αλγόριθμοί του θα κάνουν τη δουλειά για εσάς.

Συμβατότητα

Το Εργαλείο κωδικών χρωμάτων CSS λειτουργεί τέλεια σε όλα τα λειτουργικά συστήματα. Είτε έχετε συσκευή Mac, iOS, Android, Windows ή Linux, μπορείτε εύκολα να χρησιμοποιήσετε αυτό το διαδικτυακό βοηθητικό πρόγραμμα χωρίς να αντιμετωπίσετε καμία ταλαιπωρία.

100% Δωρεάν

Δεν χρειάζεται να περάσετε από καμία διαδικασία εγγραφής για να χρησιμοποιήσετε αυτό το Εργαλείο κωδικών χρωμάτων CSS. Μπορείτε να χρησιμοποιήσετε αυτό το βοηθητικό πρόγραμμα δωρεάν και να δημιουργήσετε απεριόριστους κωδικούς χρωμάτων CSS χωρίς κανέναν περιορισμό.
