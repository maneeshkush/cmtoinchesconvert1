Γραμμή πλοήγησης CSS

Γραμμή πλοήγησης CSS - οριζόντια και κάθετη.

Οριζόντια γραμμή πλοήγησης

Κωδικός CSS:

<style>
    #h_nav_bar a { padding:15px; font-weight:bold; float:left; }
    #h_nav_bar a:link { color:#d0d0d0; background-color:#0000a0; }
    #h_nav_bar a:visited { color:#c0c0c0; background-color:#0000a0; }
    #h_nav_bar a:hover { color:#ffffff; background-color:#000060; }
    #h_nav_bar a:active { color:#f0f0f0; background-color:#00ff00; }
</style>

Κώδικας HTML:

<div id="h_nav_bar">
    <a href="html-anchor-link.htm">Anchor link</a>
    <a href="html-link-color.htm">Link color</a>
    <a href="../mailto.htm">Email link</a>
    <a href="html-image-link.htm">Image link</a>
    <a href="html-text-link.htm">Text link</a>
</div>

Θέα:

Σύνδεσμος άγκυρας Χρώμα συνδέσμου Σύνδεσμος email Σύνδεσμος εικόνας Σύνδεσμος κειμένου

 

 

 

Το #link_bar a είναι το στυλ για όλες τις καταστάσεις του συνδέσμου.

#link_bar a:link είναι το στυλ του κανονικού συνδέσμου.

#link_bar a: visited είναι το στυλ του συνδέσμου επίσκεψης.

#link_bar a:hover είναι το στυλ του συνδέσμου που τοποθετείται με το ποντίκι.

#link_bar α: ενεργό είναι το στυλ του συνδέσμου όταν πατηθεί από το ποντίκι.

Χαρακτηριστικά του CSS Navigation Bar Tool

Το Εργαλείο γραμμής πλοήγησης CSS επιτρέπει στους χρήστες τη γραμμή πλοήγησης CSS. Μερικά από τα κύρια χαρακτηριστικά αυτού του βοηθητικού προγράμματος εξηγούνται παρακάτω.

Απλότητα

Η φιλική προς το χρήστη διεπαφή του Εργαλείου γραμμής πλοήγησης CSS σας παρέχει την ευκολία να πραγματοποιήσετε αυτήν τη μετατροπή άμεσα. Δεν χρειάζεται πλέον να αποκτήσετε δεξιότητες προγραμματισμού για το CSS Navigation Bar λόγω της απλότητας αυτού του Εργαλείου.

Χωρίς εγγραφή

Δεν χρειάζεται να περάσετε από καμία διαδικασία εγγραφής για να χρησιμοποιήσετε τη γραμμή πλοήγησης CSS. Χρησιμοποιώντας αυτό το βοηθητικό πρόγραμμα, μπορείτε να CSS Navigation Bar Tool όσες φορές θέλετε δωρεάν.

Φορητότητα

Αυτή η γραμμή πλοήγησης CSS είναι προσβάσιμη από οποιαδήποτε γωνιά του κόσμου. Δεν χρειάζεται να ανησυχείτε για τη συμβατότητα αυτού του διαδικτυακού εργαλείου, καθώς μπορείτε να το CSS Navigation Bar με αυτό από οποιαδήποτε συσκευή. Το μόνο που χρειάζεστε είναι μια σύνδεση στο διαδίκτυο για πρόσβαση και χρήση αυτού του Εργαλείου γραμμής πλοήγησης CSS.

Εξοικονομεί χρόνο και κόπο

Η γραμμή πλοήγησης CSS σάς επιτρέπει να ολοκληρώσετε την ίδια εργασία αμέσως. Δεν θα σας ζητηθεί να ακολουθήσετε μη αυτόματες διαδικασίες, καθώς οι αυτοματοποιημένοι αλγόριθμοί του θα κάνουν τη δουλειά για εσάς.

Συμβατότητα

Το CSS Navigation Bar Tool λειτουργεί τέλεια σε όλα τα λειτουργικά συστήματα. Είτε έχετε συσκευή Mac, iOS, Android, Windows ή Linux, μπορείτε εύκολα να χρησιμοποιήσετε αυτό το διαδικτυακό βοηθητικό πρόγραμμα χωρίς να αντιμετωπίσετε καμία ταλαιπωρία.
