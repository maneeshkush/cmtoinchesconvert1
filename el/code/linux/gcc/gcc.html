GCC C Compiler

Το GCC είναι μικρότερο από τη συλλογή GNU Compiler, έναν μεταγλωττιστή C για Linux.

  • σύνταξη gcc
  • επιλογές gcc
  • παραδείγματα gcc
  • γεννήτρια κωδικών gcc

Σύνταξη GCC

$ gcc [options] [source files] [object files] [-o output file]

Επιλογές GCC

Κύριες επιλογές του GCC:

επιλογή περιγραφή
gcc -γ μεταγλώττιση αρχείων πηγής σε αρχεία αντικειμένων χωρίς σύνδεση
Όνομα gcc -D [ =value ] ορίστε μια μακροεντολή προεπεξεργαστή
gcc -fPIC δημιουργία κώδικα ανεξάρτητου θέσης για κοινόχρηστες βιβλιοθήκες
gcc -g επίπεδο δημιουργία πληροφοριών εντοπισμού σφαλμάτων που θα χρησιμοποιηθούν από το GDB
gcc -I σκην προσθήκη καταλόγου συμπερίληψης αρχείων κεφαλίδας
gcc -l lib σύνδεσμος με αρχείο βιβλιοθήκης
gcc -L σκην αναζητήστε στον κατάλογο για αρχεία βιβλιοθήκης
gcc -o αρχείο εξόδου εγγραφή εξόδου κατασκευής σε αρχείο εξόδου
gcc -O επίπεδο βελτιστοποίηση για το μέγεθος του κώδικα και τον χρόνο εκτέλεσης
gcc -κοινόχρηστο δημιουργία αρχείου κοινόχρηστου αντικειμένου για κοινόχρηστη βιβλιοθήκη
όνομα gcc -U δεν ορίζει μια μακροεντολή προεπεξεργαστή
gcc -w απενεργοποιήστε όλα τα προειδοποιητικά μηνύματα
gcc -Τοίχος ενεργοποιήστε όλα τα προειδοποιητικά μηνύματα
gcc -Wextra ενεργοποιήστε επιπλέον προειδοποιητικά μηνύματα

Παραδείγματα GCC

Μεταγλωττίστε τα file1.c και file2.c και συνδέστε το αρχείο execfile εξόδου :

$ gcc file1.c file2.c -o execfile

 

Εκτέλεση αρχείου εξόδου execfile :

$ ./execfile

 

Μεταγλώττιση file1.c και file2.c χωρίς σύνδεση:

$ gcc -c file1.c file2.c

 

Μεταγλώττιση του myfile.c με πληροφορίες εντοπισμού σφαλμάτων και σύνδεση στο execfile αρχείου εξόδου :

$ gcc -g myfile.c -o execfile

 

Μεταγλωττίστε το myfile.c με ενεργοποιημένα προειδοποιητικά μηνύματα και συνδέστε το αρχείο execfile εξόδου :

$ gcc -Wall myfile.c -o execfile

 

Μεταγλώττιση του myfile.c με και σύνδεση με στατική βιβλιοθήκη libmath.a που βρίσκεται στο /user/local/math για να εξάγει το αρχείο execfile :

$ gcc -static myfile.c -L/user/local/math -lmath -o execfile

 

Μεταγλώττιση του myfile.c με βελτιστοποίηση και σύνδεση στο αρχείο execfile εξόδου :

$ gcc -O myfile.c -o execfile
