Τι είναι ο κώδικας ASCII

Το ASCII (American Standard Code for Information Interchange) είναι ένας κωδικός 7-bit χαρακτήρων, με τιμές από 0 έως 127. Ο κωδικός ASCII είναι ένα υποσύνολο του κώδικα UTF-8. Ο κωδικός ASCII περιλαμβάνει χαρακτήρες ελέγχου και εκτυπώσιμους χαρακτήρες: ψηφία, κεφαλαία και πεζά γράμματα.

ASCII εναντίον Unicode

Το ASCII είναι ένας κωδικός χαρακτήρων 7 bit, με τιμές από 0 έως 7F 16 . Ο κώδικας χαρακτήρων Unicode είναι ένα υπερσύνολο του ASCII που περιέχει τον κωδικό ASCII με τιμές από 0 έως 10FFFF 16
