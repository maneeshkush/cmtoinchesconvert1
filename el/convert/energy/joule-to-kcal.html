Πώς να μετατρέψετε από joules σε kcal

Joules σε θερμοχημικές / χιλιοθερμίδες τροφίμων

2 kcalth = 4184 J

6.000 J = (6.000 ÷ 4.184) = 1,434034 kcal

 

τζάουλ σε χιλιοθερμίδες χρησιμοποιώντας τον παραπάνω τύπο. Αυτή η ενέργεια σε joules E (J)  είναι ίση με την ενέργεια σε χιλιοθερμίδες  E (kcal-th)  διαιρούμενη με 4184:

E(kcal) = E(J) / 4184

Παράδειγμα 1

Μετατρέψτε 7000 τζάουλ σε χιλιοθερμίδες.

E(kcal) = 7000 J / 4184 = 1.6730 kcal

Παράδειγμα 2

Μετατρέψτε 8000 τζάουλ σε χιλιοθερμίδες.

E(kcal) = 8000 J / 4184 = 1.9120 kcal

Παράδειγμα 3

Μετατρέψτε 18000 τζάουλ σε χιλιοθερμίδες.

E(kcal) = 18000 J / 4184 = 4.3021 kcal

Joules σε διεθνείς χιλιοθερμίδες

2 kcalIT = 4186.8 J

Η ενέργεια  σε  joules E(J) ισούται με  ενέργεια σε διεθνείς  χιλιοθερμίδες E(kcal-IT) διαιρούμενη  με 4186,8:

E(kcal-IT) = E(J) / 4186.8

Παράδειγμα 1

Μετατρέψτε 6000 τζάουλ σε χιλιοθερμίδες.

E(kcal-IT) = 6000 J / 4186.8 = 1.433 kcalIT

Παράδειγμα 2

Μετατρέψτε 8000 τζάουλ σε χιλιοθερμίδες.

E(kcal-IT) = 18000 J / 4186.8 = 1.910 kcalIT

Παράδειγμα 3

Μετατρέψτε 18000 τζάουλ σε χιλιοθερμίδες.

E(kcal-IT) = 18000 J / 4186.8 = 4.299 kcalIT

E(kcal15) = E(J) / 4185.5

Παράδειγμα 1

Παράδειγμα 3

Μετατρέψτε 18000 τζάουλ σε χιλιοθερμίδες.

οπότε η ενέργεια σε joules E (J)  είναι ίση με την ενέργεια σε 20°C χιλιοθερμίδες  E (kcal20)  διαιρούμενη με 4182:

E(kcal20) = E(J) / 4182

Πώς μετατρέπετε το J σε kcal mol;
