Πώς να μετατρέψετε mAh σε Ah

Πώς να μετατρέψετε ηλεκτρικό φορτίο milliamp-h (mAh) σε amp-h (Ah).

μετατροπή χιλιοστών σε αμπερώρα

Για να μετατρέψετε το ηλεκτρικό φορτίο σε μιλιαμπέρ ώρες Q (mAh) σε ηλεκτρικό φορτίο σε αμπέρ ώρες Q (Ah) , μπορείτε να χρησιμοποιήσετε τον ακόλουθο τύπο:

Q(Ah) = Q(mAh) / 1000

 

Άρα amp-h ισούται με milliamp-h διαιρούμενο με 1000:

ampere-hours = milliampere-hours / 1000

ή

Ah = mAh / 1000

Παράδειγμα

  • Q (Ah) είναι το ηλεκτρικό φορτίο σε αμπέρ-ώρες και
  • Q (mAh) , είναι το ηλεκτρικό φορτίο σε milliampere-ώρες.

Για να χρησιμοποιήσετε τον τύπο, απλώς αντικαταστήστε την τιμή για Q (mAh) σε χιλιοστά αμπέρ-ώρες στην εξίσωση και λύστε το Q (Ah) σε αμπέρ-ώρες.

Για παράδειγμα, εάν έχετε ηλεκτρικό φορτίο 200 χιλιοστά αμπέρ ωρών, μπορείτε να το μετατρέψετε σε αμπέρ ώρες ως εξής:

Q = 200mAh / 1000 = 0,2Ah

Αυτό σημαίνει ότι το ηλεκτρικό φορτίο είναι 0,2 αμπέρ ώρες.

Είναι σημαντικό να σημειωθεί ότι αυτός ο τύπος ισχύει μόνο για τη μετατροπή ηλεκτρικού φορτίου από μιλιαμπέρ ώρες σε αμπέρ ώρες. Εάν προσπαθείτε να μετατρέψετε ηλεκτρικό φορτίο από διαφορετική μονάδα, θα χρειαστεί να χρησιμοποιήσετε διαφορετικό τύπο.
