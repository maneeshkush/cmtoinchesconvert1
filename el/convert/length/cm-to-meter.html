Πόσα μέτρα σε ένα εκατοστό

Ένα εκατοστό ισούται με 0,01 μέτρο:

1cm = 1cm/100 = 0,01m

δύο εκατοστά ισούται με 0,01 μέτρο:

2cm = 2cm/100 = 0,02m

4cm = 4cm/100 = 0,04m

5cm = 5cm/100 = 0,05m

6cm = 6cm/100 = 0,06m

8cm = 8cm/100 = 0,08m

9cm = 9cm/100 = 0,09m

Πόσα εκατοστά σε ένα μέτρο

Ένα μέτρο ισούται με 100 εκατοστά:

1m = 100×1m = 100cm

Δύο μέτρα ισούται με 200 εκατοστά:

2m = 100×2m = 200cm

Πώς να μετατρέψετε 8 εκατοστά σε μέτρα

 Διαιρέστε λοιπόν 8 εκατοστά με 100 για να πάρετε μέτρα.

8cm = 8m/100 = 0,08m

9cm = 9m/100 = 0,09m
