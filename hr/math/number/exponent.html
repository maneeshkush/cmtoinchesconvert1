Što je eksponent

Baza a podignuta na n-potencija jednaka je množenju a, n puta:

a n = a × a × ... × a

                    n puta

a je baza, a n je eksponent.

Primjeri

31 = 3

32 = 3 × 3 = 9

33 = 3 × 3 × 3 = 27

34 = 3 × 3 × 3 × 3 = 81

35 = 3 × 3 × 3 × 3 × 3 = 243

Pravila i svojstva eksponenata

Naziv pravila Pravilo Primjer
Pravila proizvoda a n ⋅ a m = a n+m 2 3 ⋅ 2 4 = 2 3+4 = 128
a n ⋅ b n = ( a ⋅ b ) n 3 2 ⋅ 4 2 = (3⋅4) 2 = 144
Pravila kvocijenta a n / a m = a n - m 2 5 / 2 3 = 2 5-3 = 4
a n / b n = ( a / b ) n 4 3 / 2 3 = (4/2) 3 = 8
Pravila moći ( b n ) m = b n⋅m (2 3 ) 2 = 2 3⋅2 = 64
b n m = b ( n m ) 2 3 2 = 2 ( 3 2 ) = 512
m √( b n ) = b n / m 2 √(2 6 ) = 2 6/2 = 8
b 1/ n = n √ b 8 1/3 = 3 √ 8 = 2
Negativni eksponenti b -n = 1 / b n 2 -3 = 1/2 3 = 0,125
Nulta pravila b 0 = 1 5 0 = 1
0 n = 0, za n >0 0 5 = 0
Jedno pravilo b 1 = b 5 1 = 5
1 n = 1 1 5 = 1

Pravila umnoška eksponenata

Pravilo proizvoda s istom bazom

an ⋅ am = an+m

Primjer:

23 ⋅ 24 = 23+4 = 27 = 2⋅2⋅2⋅2⋅2⋅2⋅2 = 128

Pravilo umnoška s istim eksponentom

an ⋅ bn = (a ⋅ b)n

Primjer:

32 ⋅ 42 = (3⋅4)2 = 122 = 12⋅12 = 144

Pravila kvocijenta eksponenata

Pravilo kvocijenta s istom bazom

an / am = an-m

Primjer:

25 / 23 = 25-3 = 22 = 2⋅2 = 4

Pravilo kvocijenta s istim eksponentom

an / bn = (a / b)n

Primjer:

43 / 23 = (4/2)3 = 23 = 2⋅2⋅2 = 8

Vidi: Dijeljenje eksponenata

Pravila potencije eksponenata

Pravilo snage I

(an) m = a n⋅m

Primjer:

(23)2 = 23⋅2 = 26 = 2⋅2⋅2⋅2⋅2⋅2 = 64

Pravilo moći II

a nm = a (nm)

Primjer:

232 = 2(32) = 2(3⋅3) = 29 = 2⋅2⋅2⋅2⋅2⋅2⋅2⋅2⋅2 = 512

Vladavina moći s radikalima

m√(a n) = a n/m

Primjer:

2√(26) = 26/2 = 23 = 2⋅2⋅2 = 8

Pravilo negativnih eksponenata

b-n = 1 / bn

Primjer:

2-3 = 1/23 = 1/(2⋅2⋅2) = 1/8 = 0.125
