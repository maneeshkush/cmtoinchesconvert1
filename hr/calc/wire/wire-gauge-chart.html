Proračuni promjera žice

Izračun promjera žice

Promjer žice promjera n d n u inčima (in) jednak je 0,005 in puta 92 podignuto na potenciju 36 minus broj promjera n, podijeljeno s 39:

dn (in) = 0.005 in × 92(36-n)/39

Promjer žice promjera n d n u milimetrima (mm) jednak je 0,127 mm puta 92 podignuto na potenciju 36 minus broj promjera n, podijeljeno s 39:

dn (mm) = 0.127 mm × 92(36-n)/39

Izračun površine presjeka žice

Poprečno područje A n žice promjera n u kilo-kružnim milima (kcmil) jednako je 1000 puta kvadratni promjer žice d u inčima (in):

An (kcmil) = 1000×dn2 = 0.025 in2 × 92(36-n)/19.5

Dakle, poprečno područje A n žice promjera n u kvadratnim inčima (in 2 ) jednako je pi podijeljeno s 4 puta kvadratni promjer žice d u inčima (in).

An (in2) = (π/4)×dn2 = 0.000019635 in2 × 92(36-n)/19.5

Dakle, poprečno područje A n žice promjera n u kvadratnim milimetrima (mm 2 ) jednako je pi podijeljeno s 4 puta kvadratni promjer žice d u milimetrima (mm).

An (mm2) = (π/4)×dn2 = 0.012668 mm2 × 92(36-n)/19.5

Proračun otpora žice

Dakle, n otpor žice R u ohmima po kiloftu (Ω/kft) jednak je 0,3048×1000000000 puta otpor žice ρ u ohm-metrima (Ω·m) podijeljeno s 25,4 2 puta površina poprečnog presjeka A n u kvadratnim inčima (u 2 ).

Rn (Ω/kft) = 0.3048 × 109 × ρ(Ω·m) / (25.42 × An (in2))

Dakle, otpor žice promjera n R u ohmima po kilometru (Ω/km) jednak je 1000000000 puta otpor žice ρ u ohm-metrima (Ω·m) podijeljen s površinom poprečnog presjeka A n u kvadratnim milimetrima (mm 2 ).

Rn (Ω/km) = 109 × ρ(Ω·m) / An (mm2)
