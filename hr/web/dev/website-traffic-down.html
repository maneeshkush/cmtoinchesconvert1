Pad prometa na web stranici

Zašto se moj promet na web stranici smanjuje?

Provjerite kalendar

Praznici i vikendi mogu smanjiti vaš promet.

Promet će se normalizirati nakon završetka praznika.

Usporedi s prošlom godinom

Koristite Google Analyics za prikaz grafikona prošlogodišnjih posjeta.

Provjerite je li posjećenost pala i prije godinu dana.

Google Analytics bug

Korištenje starog Google Analytics koda s datotekom urchin.js može prikazati posljednja 2 dana s manjim prometom od stvarnog prometa.

Promet zapravo nije smanjen, ali se samo čini da je smanjen.

Problem s poslužiteljem

Pokušajte pregledavati svoju web stranicu, ako joj ne možete pristupiti, imate problem s web poslužiteljem ili DNS poslužiteljem.

Pokušajte pristupiti svom web poslužitelju i provjerite je li aktivan.

Provjerite integritet vaše baze podataka ili html datoteka.

Upotrijebite alat za testiranje pinga da provjerite odgovor web poslužitelja.

Traži novi problem na DNS poslužitelju. Dana 9. 2012. ova web stranica s mnogim drugima nije mogla odgovoriti (pogledajte: GoDaddy hakiran ).

Rang rezultata Google pretraživanja je pao

Većina prometa na web stranicama dolazi iz tražilica, a glavna tražilica je Google.

Ako većinu posjeta vašoj web stranici generira jedna ključna riječ, konkurencija bi je mogla preuzeti.

  • Pretražite ključnu riječ na Googleu kako biste utvrdili postoji li druga web stranica koja se nalazi ispred vaše stranice i daje bolju vrijednost korisniku.
  • Pretražite vijesti za promjenu algoritma Google rangiranja. Na primjer, ažuriranje Google pande oštetilo je promet mnogih web stranica.

Web stranicu zabranio Google

Korištenje zabranjenih metoda za promoviranje vaše web stranice na Googleu osigurat će zabranu vaše web stranice od strane Googlea.

Pretražite Google sa svojim glavnim ključnim riječima i provjerite pojavljuje li se kao i obično u rezultatima pretraživanja.

Ako se vaša web stranica uopće ne pojavi, trebali biste:

  1. Pročitajte Googleove smjernice za webmastere i popravite svoju web stranicu.
  2. Pošaljite Googleu zahtjev za ponovno razmatranje .
