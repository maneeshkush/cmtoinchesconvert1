CSS boja

Šifra boje može biti jedna od sljedećih:

Hex format: #rrggbb

RGB format: rgb (crvena, zelena, plava)

Oblik naziva: naziv

Primjer

Za narančastu boju:

Hex format: #FFA500

RGB format: rgb(255,165,0)

Oblik naziva: narančasta

Postavljanje boje elementa

Za određeni element:

<element style="color: code;"></element>

Za sve elemente iste vrste. Stavite kod u oznaku <style> u odjeljku head:

<style>
   element { color: code; }
</style>

Primjer

Postavljanje boje teksta odlomka na crvenu:

Postavljanje boje određenog odlomka:

<p style="color: #FF0000;">Some text ...</p>

Pogled:

Neki tekst...

Postavljanje boje svih odlomaka

<style>
   p { color: #FF0000; }
</style>

Postavljanje boje pozadine elementa

element { background: code; }

Primjer

Postavljanje boje pozadine paragrafa na crvenu:

Postavljanje boje pozadine određenog odlomka:

<p style="background: #FF0000;">Some text ...</p>

Pogled:

Neki tekst...

Postavljanje boje pozadine svih odlomaka:

<style>
   p { background: #FF0000; }
</style>

Postavljanje boje obruba elementa

element { border-color: topcode rightcode bottomcode leftcode }

Primjer

Postavljanje boje ruba odlomka na crvenu, zelenu, plavu i crnu:

Postavljanje boje obruba određenog odlomka:

<p style="border-color: #FF0000 #00FF00 #0000FF #000000; border-style:solid">Some text ...</p>

Pogled:

Neki tekst...

Postavljanje boje obruba svih odlomaka:

<style>
   p { border-color: #FF0000 #00FF00 #0000FF #000000; }
</style>
