Link za preuzimanje HTML-a

Kako napisati link za preuzimanje u HTML-u.

Link za preuzimanje je link koji se koristi za preuzimanje datoteke sa poslužitelja u direktorij preglednika na lokalnom disku.

Kôd veze za preuzimanje napisan je kao:

<a href="test_file.zip" download>Download File</a>

Kod će stvoriti ovu vezu:

Preuzmi datoteku

Testna datoteka bi trebala biti na web poslužitelju.

Kod ima sljedeće dijelove:

  • <a> je oznaka veze.
  • href atribut postavlja datoteku za preuzimanje.
  • Download File je tekst poveznice.
  • </a> je završna oznaka veze.
