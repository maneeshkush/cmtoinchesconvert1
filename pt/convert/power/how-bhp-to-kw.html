Como converter BHP para quilowatts

Como converter potência em cavalos de frenagem (BHP) para quilowatts (kW) .

Fórmula de conversão de BHP para kW

Uma potência de freio mecânico é igual a 0,745699872 quilowatts:

1 hp = 745.699872 W = 0.745699872 kW

Portanto, a conversão de potência de quilowatts para BHP é dada por:

P(hp) = P(kW) / 0.745699872

Exemplo 1

Converter 200 kW para BHP:

P(hp) = 200 kW /  0.745699872 = 268.204 hp

Exemplo 2

Converter 400 kW para BHP:

P(hp) = 400 kW /  0.745699872 = 536.409 hp

Exemplo 3

Converta 500 kW para BHP:

P(hp) = 500 kW /  0.745699872 = 670.511 hp

Exemplo 4

Converter 700 kW para BHP:

P(hp) = 700 kW /  0.745699872 = 938.715 hp

Exemplo 5

Converter 900 kW para BHP:

P(hp) = 900 kW /  0.745699872 = 1206.92 hp
