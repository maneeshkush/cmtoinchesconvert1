Como converter BTU/h para kW

Como converter unidades térmicas britânicas por hora (BTU/h) para quilowatts (kW) .

Um BTU/hr é igual a 0,00029307107 quilowatts:

1 BTU/hr = 0.00029307107 kW

Fórmula de conversão de BTU/h para quilowatts

A potência em quilowatts  P (kW)  é igual à potência em BTU IT  por hora  P ( BTU /hr )  dividida por 3412,142:

P(kW) = P(BTU/hr) / 3412.142

Exemplo 1

Converter 10000 BTU/h para quilowatts:

P(kW) = 10000 BTU/hr / 3412.142 = 2.930kW

Exemplo 2

Converter 30000 BTU/h para quilowatts:

P(kW) = 30000 BTU/hr / 3412.142 = 8.792kW

Exemplo 3

Converter 40000 BTU/h para quilowatts:

P(kW) = 40000 BTU/hr / 3412.142 = 11.722kW

Exemplo 4

Converter 60000 BTU/h para quilowatts:

P(kW) = 60000 BTU/hr / 3412.142 = 17.584kW

Exemplo 5

Converter 110000 BTU/h para quilowatts:

P(kW) = 110000 BTU/hr / 3412.142 = 32.237kW
