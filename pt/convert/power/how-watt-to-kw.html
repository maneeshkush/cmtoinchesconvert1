Como converter watts para quilowatts

Como converter potência em watts (W) para quilowatts (kW) .

Um quilowatt é igual a 1000 watts:

1kW = 1000W

Fórmula de conversão de Watts para kilowatts

Portanto, a potência em quilowatts P (kW)  é igual à potência em watts  P (W) dividida por [1000]

P(kW) = P(W) / 1000

Exemplo 1

Converter 1200W para quilowatts:

P(kW) = 1200W / 1000 = 1.2kW

Exemplo 2

Converter 1400W para quilowatts:

P(kW) = 1400W / 1000 = 1.4kW

Exemplo 3

Converter 1600W para quilowatts:

P(kW) = 1600W / 1000 = 1.6kW

Exemplo 4

Converter 1800W para quilowatts:

P(kW) = 1800W / 1000 = 1.8kW

Exemplo 5

Converter 1900W para quilowatts:

P(kW) = 1900W / 1000 = 1.9kW
