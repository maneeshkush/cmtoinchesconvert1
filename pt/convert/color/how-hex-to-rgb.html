Como converter cores hexadecimais para rgb

Como converter de código de cores hexadecimal para cores RGB.

código de cores
hexadecimais O código de cores hexadecimais é um número hexadecimal de 6 dígitos (base 16):

RRGBBB 16

Os 2 dígitos à esquerda representam a cor vermelha.

Os 2 dígitos do meio representam a cor verde.

Os 2 dígitos da direita representam a cor azul.

cor RGB

A cor RGB é uma combinação das cores vermelho, verde e azul:

(R,G,B)

Vermelho, verde e azul usam 8 bits cada, com valores inteiros variando de 0 a 255.

Assim, o número de cores que podem ser geradas é:

256×256×256 = 16777216 = 1000000 16

conversão hexadecimal para rgb

1. Pegue 2 dígitos à esquerda do código de cor hexadecimal e converta para o valor decimal para obter o nível de cor vermelha.
2. Obtenha 2 dígitos do meio do código de cor hexadecimal e converta para o valor decimal para obter o nível de cor verde.
3. Encontre 2 dígitos corretos do código de cor hexadecimal e converta para o valor decimal para obter o nível de cor azul.

Exemplo 1
Converta o código de cor hexadecimal vermelho FF0000 em cor RGB:

hex = FF0000

Então as cores RGB são:

R = FF16 = 25510

G = 0016 = 010

B = 0016 = 010

Ou

RGB = (255, 0, 0)

Exemplo #2
Converter código de cor hexadecimal dourado FFD700 para cor RGB:

Hex = FFD700

Então as cores RGB são:

R = FF16 = 25510

G = D716 = 21510

B = 0016 = 010

Ou

RGB = (255, 215, 0)
