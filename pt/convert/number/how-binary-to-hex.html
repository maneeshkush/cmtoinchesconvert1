Como converter de hexadecimal para binário

Exemplo 1

Converter (10101110) 2  para hexadecimal:

(1010)2 = (A)16

(1110)2 = (E)16

Então

(10101110)2 = (AE)16

Exemplo #2

Converter (0111101000000001) 2  para hexadecimal:

(0111)2 = (7)16

(1010)2 = (A)16

(0000)2 = (0)16

(0001)2 = (1)16

Então

(0111101000000001)2 = (7A01)16

Exemplo #3

Converter (0111101000000101) 2  para hexadecimal:

(0111)2 = (7)16

(1010)2 = (A)16

(0000)2 = (0)16

(0101)2 = (5)16

Então

(0111101000000101)2 = (7A05)16
