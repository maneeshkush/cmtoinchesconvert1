Compilador GCC C

GCC é uma abreviação de GNU Compiler Collection, um compilador C para Linux.

  • sintaxe gcc
  • opções gcc
  • exemplos gcc
  • gerador de código gcc

Sintaxe do GCC

$ gcc [options] [source files] [object files] [-o output file]

Opções do GCC

Principais opções do GCC:

opção descrição
gcc -c compilar arquivos de origem para arquivos de objeto sem vincular
gcc -D nome [ =valor ] definir uma macro de pré-processador
gcc -fPIC gerar código independente de posição para bibliotecas compartilhadas
nível gcc -g gerar informações de depuração para serem usadas pelo GDB
gcc -I dir adicionar diretório de inclusão de arquivos de cabeçalho
gcc -l lib link com arquivo de biblioteca
gcc -L dir procure no diretório por arquivos de biblioteca
gcc -o arquivo de saída gravar a saída da compilação no arquivo de saída
nível gcc -O otimizar o tamanho do código e o tempo de execução
gcc -shared gerar arquivo de objeto compartilhado para biblioteca compartilhada
gcc -U nome indefinir uma macro de pré-processador
gcc -w desativar todas as mensagens de aviso
gcc -Parede ativar todas as mensagens de aviso
gcc -Wextra habilitar mensagens extras de aviso

Exemplos de GCC

Compile file1.c e file2.c e vincule ao arquivo de saída execfile :

$ gcc file1.c file2.c -o execfile

 

Execute o arquivo de saída execfile :

$ ./execfile

 

Compile file1.c e file2.c sem vincular:

$ gcc -c file1.c file2.c

 

Compile myfile.c com informações de depuração e link para o arquivo de saída execfile :

$ gcc -g myfile.c -o execfile

 

Compile myfile.c com mensagens de aviso ativadas e vincule ao arquivo de saída execfile :

$ gcc -Wall myfile.c -o execfile

 

Compile myfile.c e vincule-o à biblioteca estática libmath.a localizada em /user/local/math para o arquivo de saída execfile :

$ gcc -static myfile.c -L/user/local/math -lmath -o execfile

 

Compile myfile.c com otimização e link para o arquivo de saída execfile :

$ gcc -O myfile.c -o execfile
