Como visualizar/mesclar arquivos de texto

Linux visualizar/combinar arquivos de texto.

Ver arquivo de texto

$ cat list1.txt
milk
bread
apples

$ cat list2.txt
house
car

$

Combinar arquivos de texto

$ cat list1.txt list2.txt
milk
bread
apples

house
car

$

Combine 2 arquivos de texto em outro arquivo

$ cat list1.txt list2.txt > todo.txt
$
