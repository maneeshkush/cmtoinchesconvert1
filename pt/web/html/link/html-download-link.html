Link de download HTML

Como escrever link de download em HTML.

Link de download é um link usado para baixar um arquivo do servidor para o diretório do navegador no disco local.

O código do link de download é escrito como:

<a href="test_file.zip" download>Download File</a>

O código criará este link:

⇬ Fazer download do arquivo

O arquivo de teste deve estar no servidor web.

O código tem as seguintes partes:

  • <a> é a tag do link.
  • O atributo href define o arquivo para download.
  • Download File é o texto do link.
  • </a> é a tag final do link.
