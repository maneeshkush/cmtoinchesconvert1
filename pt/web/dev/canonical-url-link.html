Link de URL canônico

Marca de URL canônica.Marca de link canônico.

Redirecionamento de tag de link canônico HTML

O link canônico não redireciona para o URL preferido, mas pode ser uma alternativa ao redirecionamento de URL para sites onde a maior parte do tráfego vem dos mecanismos de pesquisa.

A tag HTML para links canônicos pode ser usada quando há várias páginas com conteúdo semelhante e você deseja informar aos mecanismos de pesquisa qual página prefere nos resultados da pesquisa.

A tag de link canônico pode vincular ao mesmo domínio e também a vários domínios .

Adicione a tag de link canônico à página antiga para vincular à
nova página.

Adicione a tag de link canônico às páginas que você prefere não  vincular o tráfego do mecanismo de  pesquisa  à página preferida.

A tag de link canônico deve ser adicionada na seção.

Exemplo 1

Quando a página for movida, podemos adicionar um link canônico na página antiga para notificar os mecanismos de pesquisa para acessar a nova página.

página-antiga1.html:

<!DOCTYPE html>
<html>
<head>
   ...
   <link rel="canonical" href="http://www.mydomain.com/new-page1.html">
</head>
<body>
   ...
</body>
</html>

Exemplo #2

Quando houver várias páginas com um convetnt semelhante, se preferirmos que os mecanismos de pesquisa cheguem à página 3.html, devemos adicionar um link canônico para a página 3.html na seção principal da página 1.html e da página 2.html.

página1.html:

<!DOCTYPE html>
<html>
<head>
   ...
   <link rel="canonical" href="http://www.mydomain.com/page3.html">
</head>
<body>
   ...
</body>
</html>

página2.html:

<!DOCTYPE html>
<html>
<head>
   ...
   <link rel="canonical" href="http://www.mydomain.com/page3.html">
</head>
<body>
   ...
</body>
</html>
