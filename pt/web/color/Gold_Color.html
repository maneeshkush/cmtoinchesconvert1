Quais cores fazem ouro

A cor dourada RGB é gerada pela adição das cores vermelha e verde.

Código de cores RGB dourado

Código de cor RGB dourado = #FFD700 = 255*65536+215*256+0 = (255, 215, 0)

VERMELHO=255, VERDE=215, AZUL=0
