Como usar a câmera online:

  • Primeiro, permita o acesso à câmera aceitando a solicitação de acesso do navegador ou clicando no ícone da câmera na barra de endereço.
  • Pressione o botão Pause para capturar a imagem e desativar a câmera.
  • Pressione o botão Download para baixar a imagem para a pasta Downloads.
  • Pressione o botão Fullscreen para ampliar a imagem.
  • Pressione os botões sépia/tons de cinza/inverter/contraste/desfoque para alterar o estilo da imagem.
  • Pressione o botão × Redefinir para limpar o estilo da imagem.
  • Clique na barra de zoom para ampliar a imagem.
