Placar

:

Como usar o placar

  • Defina o tempo em minutos e segundos.
  • Defina os nomes dos jogadores/times.
  • Pressione o botão Iniciar para iniciar o jogo.
  • Clique na caixa de texto de pontuação para definir a pontuação.
  • Pressione o botão + para aumentar a pontuação.
  • Pressione o botão - para diminuir a pontuação.
  • Pressione o botão × para zerar a pontuação.
