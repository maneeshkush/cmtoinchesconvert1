Ajuda do bloco de notas

Existem 2 maneiras de salvar o texto

  1. O texto é salvo toda vez que você fecha a aba Bloco de Notas, no cache local do navegador.Quando você entrar novamente na página do Bloco de Notas, o texto reaparecerá.
  2. O texto é salvo/backup no disco rígido quando você pressiona o botão Salvar.Para reabrir o texto do disco rígido, pressione o botão Abrir e selecione o arquivo de texto que você criou.

Parâmetros de configuração


Informação importante

  • Se faltar o texto da sessão anterior :
    • Se existir texto, selecione e copie-o com Ctrl+C e cole-o com Ctrl+V na página do Bloco de Notas :

    • Copie o texto do arquivo de texto atualizado que foi gerado pela operação de salvamento automático para a pasta Download (se existir).
    • Verifique se você está olhando para o mesmo computador que usou anteriormente.
    • Ative os cookies e o histórico do navegador.
    • Não use o modo privado/anônimo do navegador.O armazenamento local de texto será excluído pelo navegador quando você fechar a janela do navegador.
    • Tente adicionar/remover o www da URL na barra de endereço do navegador.
  • O texto do bloco de notas não será salvo com a navegação no modo anônimo/privado !!!
  • O texto do Notpad salvo pode ser excluído quando você exclui seu histórico de navegação/cache ou executa o aplicativo de limpeza de disco (por exemplo, limpeza de disco do Windows / CCleaner) !!!
  • Se o botão Abrir arquivo não funcionar, recarregue a página e tente novamente.
  • Use o Bloco de Notas com a versão atualizada do navegador.Se você usa o Internet Explorer , copie seu texto para um navegador moderno (por exemplo, Chrome/Edge/Firefox ).
  • O texto do bloco de notas é salvo automaticamente no cache local do navegador (não é seguro).
  • O texto do bloco de notas pode ser salvo automaticamente (backup) no disco rígido, de acordo com a duração do salvamento automático nomenu Exibir > Preferências .
  • Você pode fazer backup do texto do Bloco de Notas no disco rígido, usando o botão Salvar ou o menu Arquivo > Salvar.
  • No Mac, use ⌘ Command em vez da tecla Ctrl .
  • Para abrir o arquivo salvo, procure o arquivo na pasta Downloads.
  • Se as linhas de fundo não rolarem, oculte as linhas: desmarque o menu Exibir > Preferências > Linhas de texto
  • Botão Salvar ou menu Arquivo > Salvar salva o arquivo napasta Downloads . Consulte: Para onde vão os arquivos quando baixados?
  • Se as linhas de texto não estiverem visíveis, tente usar o navegador Chrome.
  • Se a verificação ortográfica não estiver funcionando, tente ativá-la na seção Configurações>Idiomas do seu navegador.Se não estiver definido, você também pode tentar definir o inglês (Estados Unidos) na configuração de idioma do seu navegador.
