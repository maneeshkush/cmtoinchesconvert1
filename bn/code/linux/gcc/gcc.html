জিসিসি সি কম্পাইলার

GCC হল GNU কম্পাইলার সংগ্রহের একটি সংক্ষিপ্ত, লিনাক্সের জন্য একটি সি কম্পাইলার।

GCC সিনট্যাক্স

$ gcc [options] [source files] [object files] [-o output file]

GCC বিকল্প

GCC প্রধান বিকল্প:

বিকল্প বর্ণনা
gcc -c লিঙ্ক না করেই অবজেক্ট ফাইলে সোর্স ফাইল কম্পাইল করুন
gcc -D নাম [ = মান ] একটি প্রিপ্রসেসর ম্যাক্রো সংজ্ঞায়িত করুন
gcc -fPIC ভাগ করা লাইব্রেরির জন্য অবস্থান স্বাধীন কোড তৈরি করুন
gcc -g স্তর GDB ব্যবহার করার জন্য ডিবাগ তথ্য তৈরি করে
gcc - আমি পরিচালক হেডার ফাইলের ডিরেক্টরি অন্তর্ভুক্ত করুন
gcc -l lib লাইব্রেরি ফাইলের সাথে লিঙ্ক করুন
gcc -L dir লাইব্রেরি ফাইলের জন্য ডিরেক্টরি দেখুন
gcc -o আউটপুট ফাইল আউটপুট ফাইলে বিল্ড আউটপুট লিখুন
gcc -O স্তর কোড আকার এবং কার্যকর করার সময় জন্য অপ্টিমাইজ করুন
gcc - শেয়ার করা শেয়ার্ড লাইব্রেরির জন্য শেয়ার্ড অবজেক্ট ফাইল তৈরি করুন
gcc -U নাম একটি প্রিপ্রসেসর ম্যাক্রো অসংজ্ঞায়িত করুন
gcc -w সমস্ত সতর্কতা বার্তা নিষ্ক্রিয় করুন
gcc -ওয়াল সমস্ত সতর্কতা বার্তা সক্রিয় করুন
জিসিসি -ওয়েক্সট্রা অতিরিক্ত সতর্কতা বার্তা সক্ষম করুন

GCC উদাহরণ

file1.c এবং file2.c কম্পাইল করুন এবং আউটপুট ফাইল execfile লিঙ্ক করুন :

$ gcc file1.c file2.c -o execfile

 

আউটপুট ফাইল execfile চালান :

$ ./execfile

 

লিঙ্ক না করে file1.c এবং file2.c কম্পাইল করুন :

$ gcc -c file1.c file2.c

 

ডিবাগ তথ্য সহ myfile.c কম্পাইল করুন এবং আউটপুট ফাইল execfile লিঙ্ক করুন :

$ gcc -g myfile.c -o execfile

 

সতর্কতা বার্তা সক্ষম করে myfile.c কম্পাইল করুন এবং আউটপুট ফাইল execfile লিঙ্ক করুন :

$ gcc -Wall myfile.c -o execfile

 

আউটপুট ফাইল execfile করতে /user/local/ math- এ অবস্থিত স্ট্যাটিক লাইব্রেরি libmath.a- এর সাথে myfile.c কম্পাইল করুন এবং লিঙ্ক করুন :

$ gcc -static myfile.c -L/user/local/math -lmath -o execfile

 

অপ্টিমাইজেশন সহ myfile.c কম্পাইল করুন এবং আউটপুট ফাইল execfile লিঙ্ক করুন :

$ gcc -O myfile.c -o execfile
