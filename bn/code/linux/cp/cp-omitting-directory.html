cp: ডিরেক্টরি বাদ দেওয়া

cp ডিরেক্টরি লিনাক্স ত্রুটি বার্তা বাদ দেওয়া।

সমস্যা

একটি ডিরেক্টরি অনুলিপি করার চেষ্টা করার সময়, আপনি ত্রুটি বার্তা পাবেন:

$ cp srcdir destdir
cp: omitting directory 'srcdir'
$

সমাধান

রিকার্সিভ কপির জন্য অ্যাড -আর বিকল্প:

$ cp -R srcdir destdir
