লিনাক্স/ইউনিক্সে ls কমান্ড

ls হল একটি লিনাক্স শেল কমান্ড যা ফাইল এবং ডিরেক্টরির ডিরেক্টরি বিষয়বস্তু তালিকাভুক্ত করে।

  • ls সিনট্যাক্স
  • ls অপশন
  • ls উদাহরণ
  • ls কোড জেনারেটর

ls সিনট্যাক্স

$ ls [options] [file|dir]

ls কমান্ড অপশন

ls কমান্ড প্রধান বিকল্প:

বিকল্প বর্ণনা
ls -a '.' দিয়ে শুরু হওয়া লুকানো ফাইল সহ সমস্ত ফাইল তালিকাভুক্ত করুন।
ls --রঙ রঙিন তালিকা [=সর্বদা/কখনও না/স্বয়ংক্রিয়]
ls -d তালিকা ডিরেক্টরি - '*/' সহ
ls -F */=>@| এর একটি অক্ষর যোগ করুন প্রবেশের জন্য
ls - i তালিকা ফাইলের ইনোড সূচক নম্বর
ls -l দীর্ঘ বিন্যাস সহ তালিকা - অনুমতি দেখান
ls -la লুকানো ফাইল সহ দীর্ঘ বিন্যাস তালিকা
ls -lh পঠনযোগ্য ফাইল আকার সহ দীর্ঘ বিন্যাস তালিকা
ls -ls ফাইল আকার সহ দীর্ঘ বিন্যাস সহ তালিকা
ls -r বিপরীত ক্রমে তালিকা
ls -আর পুনরাবৃত্তভাবে ডিরেক্টরি গাছ তালিকা
ls -s তালিকা ফাইলের আকার
ls -S ফাইলের আকার অনুসারে সাজান
ls -t সময় এবং তারিখ অনুসারে সাজান
ls -X এক্সটেনশন নাম অনুসারে সাজান

ls কমান্ডের উদাহরণ

ফাইল বা ফোল্ডারের নাম স্বয়ংক্রিয়ভাবে সম্পূর্ণ করতে আপনি ট্যাব বোতাম টিপুন ।

আপেক্ষিক পথ সহ ডিরেক্টরি নথি/বই তালিকা করুন:

$ ls Documents/Books

 

পরম পথ সহ ডিরেক্টরি /home/user/Documents/Books তালিকা করুন।

$ ls /home/user/Documents/Books

 

তালিকা রুট ডিরেক্টরি:

$ ls /

 

মূল ডিরেক্টরি তালিকাভুক্ত করুন:

$ ls ..

 

ব্যবহারকারীর হোম ডিরেক্টরি তালিকাভুক্ত করুন (যেমন: /home/user):

$ ls ~

 

দীর্ঘ বিন্যাস সহ তালিকা:

$ ls -l

 

গোপন ফাইলগুলো দেখুন:

$ ls -a

 

দীর্ঘ বিন্যাস সহ তালিকা করুন এবং লুকানো ফাইলগুলি দেখান:

$ ls -la

 

তারিখ/সময় অনুসারে সাজান:

$ ls -t

 

ফাইলের আকার অনুসারে সাজান:

$ ls -S

 

সমস্ত সাবডিরেক্টরি তালিকাভুক্ত করুন:

$ ls *

 

পুনরাবৃত্ত ডিরেক্টরি গাছ তালিকা:

$ ls -R

 

ওয়াইল্ডকার্ড সহ শুধুমাত্র পাঠ্য ফাইল তালিকাভুক্ত করুন:

$ ls *.txt

 

আউটপুট ফাইলে ls পুনঃনির্দেশ:

$ ls > out.txt

 

শুধুমাত্র ডিরেক্টরি তালিকাভুক্ত করুন:

$ ls -d */

 

সম্পূর্ণ পাথ সহ ফাইল এবং ডিরেক্টরি তালিকাভুক্ত করুন:

$ ls -d $PWD/*

ls কোড জেনারেটর

ls অপশন নির্বাচন করুন এবং জেনারেট কোড বোতাম টিপুন:

অপশন 
  দীর্ঘ তালিকা বিন্যাস (-l)
  সমস্ত ফাইল / লুকানো ফাইল তালিকাভুক্ত করুন (-a)
  বারবার তালিকাভুক্ত ডিরেক্টরি গাছ (-R)
  বিপরীত ক্রমে তালিকা (-r)
  সম্পূর্ণ পথ সহ তালিকা (-d $PWD/*)
ক্রমানুসার:
ফাইল / ফোল্ডার
নথি পত্র:
ফোল্ডার:
আউটপুট পুনর্নির্দেশ
কোড নির্বাচন করতে টেক্সটবক্সে ক্লিক করুন, তারপর কপি করে টার্মিনালে পেস্ট করুন
