লিনাক্স/ইউনিক্সে সিডি কমান্ড

cd টার্মিনালের শেলের ডিরেক্টরি/ফোল্ডার পরিবর্তন করার জন্য একটি লিনাক্স কমান্ড।

ডিরেক্টরির নামটি স্বয়ংক্রিয়ভাবে সম্পূর্ণ করার জন্য আপনি ট্যাব বোতাম টিপুন।

  • সিডি সিনট্যাক্স
  • সিডি উদাহরণ

সিডি সিনট্যাক্স

$ cd [directory]

cd কমান্ডের উদাহরণ

হোম ডিরেক্টরিতে পরিবর্তন করুন ($HOME পরিবেশ পরিবর্তনশীল দ্বারা নির্ধারিত):

$ cd

 

এছাড়াও হোম ডিরেক্টরিতে পরিবর্তন করুন:

$ cd ~

 

রুট ডিরেক্টরিতে পরিবর্তন করুন:

$ cd /

 

মূল ডিরেক্টরিতে পরিবর্তন করুন:

$ cd ..

 

সাবডিরেক্টরি নথিতে পরিবর্তন করুন :

$ cd Documents

 

সাবডিরেক্টরি নথি/বই- এ পরিবর্তন করুন :

$ cd Documents/Books

 

পরম পথ /home/user/Desktop সহ ডিরেক্টরিতে পরিবর্তন করুন :

$ cd /home/user/Desktop

 

সাদা স্থান সহ ডিরেক্টরির নাম পরিবর্তন করুন - আমার ছবি :

$ cd My\ Images

বা

$ cd "My Images"

বা

$ cd 'My Images'
