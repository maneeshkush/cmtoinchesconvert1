ক্যানোনিকাল URL লিঙ্ক

ক্যানোনিকাল URL ট্যাগ। ক্যানোনিকাল লিঙ্ক ট্যাগ।

HTML ক্যানোনিকাল লিঙ্ক ট্যাগ পুনঃনির্দেশ

ক্যানোনিকাল লিঙ্কটি পছন্দের URL-এ পুনঃনির্দেশিত করে না, তবে এটি এমন ওয়েবসাইটগুলির জন্য URL পুনঃনির্দেশের বিকল্প হতে পারে যেখানে বেশিরভাগ ট্র্যাফিক সার্চ ইঞ্জিন থেকে আসে।

ক্যানোনিকাল লিঙ্কগুলির জন্য HTML ট্যাগ ব্যবহার করা যেতে পারে যখন একই বিষয়বস্তু সহ একাধিক পৃষ্ঠা থাকে এবং আপনি অনুসন্ধান ইঞ্জিনগুলিকে বলতে চান যে আপনি অনুসন্ধান ফলাফলে কোন পৃষ্ঠা পছন্দ করেন৷

ক্যানোনিকাল লিঙ্ক ট্যাগ একই ডোমেনে এবং একাধিক ডোমেনে লিঙ্ক করতে পারে ।


নতুন পৃষ্ঠায় লিঙ্ক করতে পুরানো পৃষ্ঠায় ক্যানোনিকাল লিঙ্ক ট্যাগ যোগ করুন ।

 যে পৃষ্ঠাগুলিতে আপনি সার্চ  ইঞ্জিন  ট্র্যাফিককে পছন্দের পৃষ্ঠায় লিঙ্ক করতে চান না সেগুলিতে ক্যানোনিকাল লিঙ্ক ট্যাগ যুক্ত করুন  ৷

ক্যানোনিকাল লিঙ্ক ট্যাগ বিভাগে যোগ করা উচিত.

উদাহরণ # 1

পৃষ্ঠাটি সরানো হলে আমরা পুরানো পৃষ্ঠায় ক্যানোনিকাল লিঙ্ক যোগ করতে পারি যাতে নতুন পৃষ্ঠায় পৌঁছানোর জন্য সার্চ ইঞ্জিনগুলিকে অবহিত করা যায়।

old-page1.html:

<!DOCTYPE html>
<html>
<head>
   ...
   <link rel="canonical" href="http://www.mydomain.com/new-page1.html">
</head>
<body>
   ...
</body>
</html>

উদাহরণ #2

যখন একই ধরনের কনভেটেন্ট সহ বেশ কয়েকটি পৃষ্ঠা থাকে, আমরা যদি page3.html-এ পৌঁছানোর জন্য সার্চ ইঞ্জিন পছন্দ করি, তাহলে page1.html এবং page2.html-এর প্রধান বিভাগে page3.html-এ ক্যানোনিকাল লিঙ্ক যোগ করা উচিত।

page1.html:

<!DOCTYPE html>
<html>
<head>
   ...
   <link rel="canonical" href="http://www.mydomain.com/page3.html">
</head>
<body>
   ...
</body>
</html>

page2.html:

<!DOCTYPE html>
<html>
<head>
   ...

</head>

</html>
