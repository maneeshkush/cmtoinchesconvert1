ডিসি সার্কিটের নিয়ম

ওম এর আইন

I = V / R

জুলের আইন

P = V · I = I 2 · R = V 2 / R

সিরিজ সার্কিট নিয়ম

VT = V1 + V2 + V3 + ...

IT = I1 = I2 = I3 = ...

RT = R1 + R2 + R3 + ...

1/CT = 1/C1 + 1/C2 + 1/C3 + ...

LT = L1 + L2 + L3 + ...

সমান্তরাল সার্কিট নিয়ম

VT = V1 = V2 = V3 = ...

IT = I1 + I2 + I3 + ...

1/RT = 1/R1 + 1/R2 + 1/R3 + ...

CT = C1 + C2 + C3 + ...

1/LT = 1/L1 + 1/L2 + 1/L3 + ...

ভোল্টেজ বিভাগ

V1 = VT ⋅ R1 / (R1+R2+R3+...)

বর্তমান বিভাগ

I1 = IT ⋅ (R2+R3+...) / (R1+R2+R3+...)

Kirchhoff এর ভোল্টেজ আইন (KVL)

বর্তমান লুপে ভোল্টেজ ড্রপের যোগফল শূন্য:

∑ Vi = 0

Kirchhoff এর বর্তমান আইন (KCL)

কয়েকটি সার্কিট উপাদানের মধ্যে সংযোগস্থলকে নোড বলে।

একটি নোডে কারেন্টের মানের যোগফল শূন্য:

∑ I i = 0

ক্যাপাসিট্যান্স

C = Q/V

সমান্তরাল প্লেট ক্যাপাসিটর

C = ε ⋅ A / l

ε হল ফ্যারাড প্রতি মিটারে (F/m) অনুমতি।

পারমিটিভিটি

ε = ε0 ⋅ εr

ε 0 হল ভ্যাকুয়ামের পারমিটিভিটি।

ε r হল আপেক্ষিক অনুমতি বা দ্বান্দ্বিক ধ্রুবক।

ক্যাপাসিটরের কারেন্ট

IC(t) = C d VC(t) / dt

ক্যাপাসিটরের ভোল্টেজ

V C (t) = V C (0) + 1/ C ∫ I C (t)⋅ dt

ইন্ডাক্টরের ভোল্টেজ

VL(t) = L d IL(t) / dt

প্রবর্তক বর্তমান

I L (t) = I L (0) + 1/ L ∫ V L (t)⋅ dt

ক্যাপাসিটরের শক্তি

WC = C⋅V 2 / 2

ইন্ডাক্টরের শক্তি 

WL = L⋅I 2 / 2
