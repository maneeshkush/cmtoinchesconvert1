কিভাবে শক্তি সঞ্চয় করা যায়

কিভাবে শক্তি খরচ সংরক্ষণ করা যায়. কিভাবে বিদ্যুৎ এবং জ্বালানী সাশ্রয় করা যায়।

জ্বালানী খরচ কমান

  • একা ড্রাইভ করার পরিবর্তে পাবলিক ট্রান্সপোর্ট, কারপুল, বা হাঁটা বা বাইক ব্যবহার করুন।
  • জ্বালানি দক্ষতা উন্নত করতে আপনার গাড়ির টায়ার সঠিকভাবে স্ফীত করুন।
  • জ্বালানি বাঁচাতে হাইওয়েতে ক্রুজ নিয়ন্ত্রণ ব্যবহার করুন।
  • দীর্ঘ সময়ের জন্য আপনার গাড়ী অলস এড়িয়ে চলুন.
  • আপনাকে যতবার গাড়ি চালাতে হবে তার সংখ্যা কমাতে কাজগুলিকে এক ট্রিপে একত্রিত করুন।
  • আপনি বাড়িতে না থাকলে স্বয়ংক্রিয়ভাবে তাপমাত্রা সামঞ্জস্য করতে একটি প্রোগ্রামেবল থার্মোস্ট্যাট ব্যবহার করুন।
  • ইলেকট্রনিক্স এবং যন্ত্রপাতিগুলি যখন ব্যবহারে না থাকে তখন আনপ্লাগ করুন, কারণ তারা বন্ধ থাকলেও প্লাগ ইন থাকা অবস্থায়ও শক্তি ব্যবহার করতে পারে৷
  • জল এবং শক্তি ব্যবহার কমাতে কম প্রবাহিত শাওয়ারহেড ইনস্টল করুন।
  • জলের অপচয় কমাতে আপনার বাড়িতে ফুটো ঠিক করুন।
  • সম্পূর্ণ লোড সহ শুধুমাত্র ডিশওয়াশার এবং ওয়াশিং মেশিন চালান।
  • জল গরম করার শক্তি বাঁচাতে কাপড় ধোয়ার জন্য ঠান্ডা জল ব্যবহার করুন।
  • ড্রায়ার ব্যবহার না করে বাইরে কাপড়ের লাইনে শুকিয়ে নিন।
  • খাবার রান্না করতে চুলা বা চুলার পরিবর্তে প্রেসার কুকার বা স্লো কুকার ব্যবহার করুন।
  • ছোট আইটেম রান্না করার সময় শক্তি সঞ্চয় করতে চুলার পরিবর্তে মাইক্রোওয়েভ ব্যবহার করুন।
  • পানি ফুটানোর সময় বা রুটি টোস্ট করার সময় শক্তি বাঁচাতে চুলার পরিবর্তে টোস্টার ওভেন বা বৈদ্যুতিক কেটলি ব্যবহার করুন।
  • আপনি যখন একটি রুম ছেড়ে যান যন্ত্রপাতি এবং আলো বন্ধ করুন.
  • যখনই সম্ভব কৃত্রিম আলোর পরিবর্তে প্রাকৃতিক আলো ব্যবহার করুন।
  • একবারে একাধিক ইলেকট্রনিক্স বন্ধ করতে একটি পাওয়ার স্ট্রিপ ব্যবহার করুন।
  • এয়ার কন্ডিশনার চালু না করে বাতাস চলাচলের জন্য সিলিং ফ্যান ব্যবহার করুন।
  • কাপড় শুকানোর জন্য ড্রায়ারের পরিবর্তে ক্লোথলাইন বা ড্রাইং র্যাক ব্যবহার করুন।
  • একটি গ্যাস চালিত একটি পরিবর্তে একটি ম্যানুয়াল লন ঘষার যন্ত্র ব্যবহার করুন.
  • নিষ্পত্তিযোগ্য প্লাস্টিকের বোতলের পরিবর্তে পুনরায় ব্যবহারযোগ্য পানির বোতল ব্যবহার করুন।
  • উত্পাদনে শক্তি সঞ্চয় করতে কাগজ, প্লাস্টিক এবং ধাতু পুনর্ব্যবহার করুন।
  • পণ্যের আয়ু বাড়াতে এবং অপচয় কমাতে সেকেন্ডহ্যান্ড কেনাকাটা করুন।
  • সৌর বা বায়ু শক্তির মতো পরিষ্কার শক্তির উত্সগুলিকে সমর্থন করুন।
  • লাইট এবং ইলেকট্রনিক্স ব্যবহার না হলে বন্ধ করুন।
  • শক্তি-দক্ষ যন্ত্রপাতি এবং লাইট বাল্ব ব্যবহার করুন।
  • আপনার থার্মোস্ট্যাটকে শীতকালে কম তাপমাত্রায় এবং গ্রীষ্মে উচ্চ তাপমাত্রায় সেট করুন।
  • সূর্যের রশ্মি আটকাতে এবং গ্রীষ্মে আপনার ঘরকে ঠান্ডা রাখতে গাছ লাগান বা ছায়াযুক্ত ডিভাইস ইনস্টল করুন।
  • শীতকালে গরম এবং গ্রীষ্মে ঠাণ্ডা রাখতে আপনার ঘরকে অন্তরণ করুন।

বিদ্যুৎ খরচ কমান

  • লাইট এবং ইলেকট্রনিক্স ব্যবহার না হলে বন্ধ করুন।
  • শক্তি-দক্ষ যন্ত্রপাতি এবং লাইট বাল্ব ব্যবহার করুন।
  • আপনার থার্মোস্ট্যাটকে শীতকালে কম তাপমাত্রায় এবং গ্রীষ্মে উচ্চ তাপমাত্রায় সেট করুন।
  • সূর্যের রশ্মি আটকাতে এবং গ্রীষ্মে আপনার ঘরকে ঠান্ডা রাখতে গাছ লাগান বা ছায়াযুক্ত ডিভাইস ইনস্টল করুন।
  • শীতকালে গরম এবং গ্রীষ্মে ঠাণ্ডা রাখতে আপনার ঘরকে অন্তরণ করুন।
  • আপনি বাড়িতে না থাকলে স্বয়ংক্রিয়ভাবে তাপমাত্রা সামঞ্জস্য করতে একটি প্রোগ্রামেবল থার্মোস্ট্যাট ব্যবহার করুন।
  • ইলেকট্রনিক্স এবং যন্ত্রপাতিগুলি যখন ব্যবহারে না থাকে তখন আনপ্লাগ করুন, কারণ তারা বন্ধ থাকলেও প্লাগ ইন থাকা অবস্থায়ও শক্তি ব্যবহার করতে পারে৷
  • জল এবং শক্তি ব্যবহার কমাতে কম প্রবাহিত শাওয়ারহেড ইনস্টল করুন।
  • জলের অপচয় কমাতে আপনার বাড়িতে ফুটো ঠিক করুন।
  • সম্পূর্ণ লোড সহ শুধুমাত্র ডিশওয়াশার এবং ওয়াশিং মেশিন চালান।
  • জল গরম করার শক্তি বাঁচাতে কাপড় ধোয়ার জন্য ঠান্ডা জল ব্যবহার করুন।
  • ড্রায়ার ব্যবহার না করে বাইরে কাপড়ের লাইনে শুকিয়ে নিন।
  • খাবার রান্না করতে চুলা বা চুলার পরিবর্তে প্রেসার কুকার বা স্লো কুকার ব্যবহার করুন।
  • ছোট আইটেম রান্না করার সময় শক্তি সঞ্চয় করতে চুলার পরিবর্তে মাইক্রোওয়েভ ব্যবহার করুন।
  • পানি ফুটানোর সময় বা রুটি টোস্ট করার সময় শক্তি বাঁচাতে চুলার পরিবর্তে টোস্টার ওভেন বা বৈদ্যুতিক কেটলি ব্যবহার করুন।
  • আপনি যখন একটি রুম ছেড়ে যান যন্ত্রপাতি এবং আলো বন্ধ করুন.
  • যখনই সম্ভব কৃত্রিম আলোর পরিবর্তে প্রাকৃতিক আলো ব্যবহার করুন।
  • একবারে একাধিক ইলেকট্রনিক্স বন্ধ করতে একটি পাওয়ার স্ট্রিপ ব্যবহার করুন।
  • এয়ার কন্ডিশনার চালু না করে বাতাস চলাচলের জন্য সিলিং ফ্যান ব্যবহার করুন।
  • কাপড় শুকানোর জন্য ড্রায়ারের পরিবর্তে ক্লোথলাইন বা ড্রাইং র্যাক ব্যবহার করুন।
  • একটি গ্যাস চালিত একটি পরিবর্তে একটি ম্যানুয়াল লন ঘষার যন্ত্র ব্যবহার করুন.
  • নিষ্পত্তিযোগ্য প্লাস্টিকের বোতলের পরিবর্তে পুনরায় ব্যবহারযোগ্য পানির বোতল ব্যবহার করুন।
  • উত্পাদনে শক্তি সঞ্চয় করতে কাগজ, প্লাস্টিক এবং ধাতু পুনর্ব্যবহার করুন।
  • পণ্যের আয়ু বাড়াতে এবং অপচয় কমাতে সেকেন্ডহ্যান্ড কেনাকাটা করুন।
  • সৌর বা বায়ু শক্তির মতো পরিষ্কার শক্তির উত্সগুলিকে সমর্থন করুন।
  • একা ড্রাইভ করার পরিবর্তে পাবলিক ট্রান্সপোর্ট, কারপুল, বা হাঁটা বা বাইক ব্যবহার করুন।
  • জ্বালানি দক্ষতা উন্নত করতে আপনার গাড়ির টায়ার সঠিকভাবে স্ফীত করুন।
  • জ্বালানি বাঁচাতে হাইওয়েতে ক্রুজ নিয়ন্ত্রণ ব্যবহার করুন।
  • দীর্ঘ সময়ের জন্য আপনার গাড়ী অলস এড়িয়ে চলুন.
  • আপনাকে যতবার গাড়ি চালাতে হবে তার সংখ্যা কমাতে কাজগুলিকে এক ট্রিপে একত্রিত করুন।
