বেস নিয়মের লগারিদম পরিবর্তন

b থেকে c তে ভিত্তি পরিবর্তন করার জন্য, আমরা বেস নিয়মের লগারিদম পরিবর্তন ব্যবহার করতে পারি। x এর বেস b লগারিদম x এর বেস c লগারিদমের সমান b এর বেস c লগারিদম দ্বারা ভাগ করা হয়:

logb(x) = logc(x) / logc(b)

উদাহরণ # 1

log2(100) = log10(100) / log10(2) = 2 / 0.30103 = 6.64386

উদাহরণ #2

log3(50) = log8(50) / log8(3) = 1.8812853 / 0.5283208 = 3.5608766

প্রমাণ

x এর বেস b লগারিদমের শক্তি দিয়ে b বাড়ালে x পাওয়া যায়:

(1) x = blogb(x)

b এর বেস c লগারিদমের শক্তি দিয়ে c বাড়ালে b পাওয়া যায়:

(2) b = clogc(b)

যখন আমরা (1) নিই এবং c লগ c ( b ) (2) দিয়ে b প্রতিস্থাপন করি, তখন আমরা পাই:

(3) x = blogb(x) = (clogc(b))logb(x) = clogc(b)×logb(x)

(3) এর উভয় পাশে লগ c () প্রয়োগ করে:

logc(x) = logc(clogc(b)×logb(x))

লগারিদম পাওয়ার নিয়ম প্রয়োগ করে :

logc(x) = [logc(b)×logb(x)] × logc(c)

যেহেতু লগ c ( c ) = 1

logc(x) = logc(b)×logb(x)
