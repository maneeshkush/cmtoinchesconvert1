สัญลักษณ์อินฟินิตี้

สัญลักษณ์อินฟินิตี้เป็นสัญลักษณ์ทางคณิตศาสตร์ที่ใช้แทนจำนวนนับไม่ถ้วน

สัญลักษณ์อินฟินิตี้เขียนด้วยสัญลักษณ์เลมนิสเกต:

∞

มันแสดงถึงจำนวนมหาศาลบวกไม่สิ้นสุด

เมื่อเราต้องการเขียนจำนวนลบอนันต์ เราควรเขียนดังนี้

-∞

เมื่อเราต้องการเขียนจำนวนที่น้อยมาก เราควรเขียน:

1/∞

อินฟินิตี้เป็นจำนวนจริงหรือไม่?

อินฟินิตี้ไม่ใช่ตัวเลขมันไม่ได้แสดงถึงจำนวนเฉพาะ แต่เป็นปริมาณที่มากเป็นอนันต์

กฎและคุณสมบัติของอินฟินิตี้

ชื่อ ประเภทกุญแจ
อินฟินิตี้บวก ∞
อินฟินิตี้เชิงลบ -∞
ความแตกต่างที่ไม่มีที่สิ้นสุด ∞ - ∞ ไม่ได้กำหนด
สินค้าเป็นศูนย์ 0 ⋅ ∞ ไม่ได้กำหนด
ความฉลาดทางอินฟินิตี้ ∞ / ∞ ไม่ได้กำหนด
ผลรวมของจำนวนจริง x + ∞ = ∞ สำหรับx ∈ℝ
ผลิตภัณฑ์จำนวนบวก x ⋅ ∞ = ∞ สำหรับx >0

Aleph-null ( ) คือจำนวนองค์ประกอบที่ไม่สิ้นสุด (จำนวนสมาชิก) ของชุดจำนวนธรรมชาติ ( )

Aleph-one ( ) เป็นจำนวนอนันต์ขององค์ประกอบ (จำนวนสมาชิก) ของชุดตัวเลขลำดับที่นับได้ ( ω 1 )
