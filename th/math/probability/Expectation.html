ค่าความคาดหวัง

ในความน่าจะเป็นและสถิติความคาดหวังหรือค่าคาดหวังคือค่าเฉลี่ยถ่วงน้ำหนักของตัวแปรสุ่ม

ความคาดหวังของตัวแปรสุ่มต่อเนื่อง

E ( X ) คือค่าคาดหวังของตัวแปรสุ่มต่อเนื่องX

xคือค่าของตัวแปรสุ่มต่อเนื่องX

P ( x ) คือฟังก์ชันความหนาแน่นของความน่าจะเป็น

ความคาดหวังของตัวแปรสุ่มแบบไม่ต่อเนื่อง

E ( X ) คือค่าคาดหวังของตัวแปรสุ่มต่อเนื่องX

xคือค่าของตัวแปรสุ่มต่อเนื่องX

P ( x ) คือฟังก์ชันมวลของความน่าจะเป็นของX

คุณสมบัติของความคาดหวัง

เชิงเส้น

เมื่อ a เป็นค่าคงที่และ X,Y เป็นตัวแปรสุ่ม:

E(aX) = aE(X)

E(X+Y) = E(X) + E(Y)

คงที่

เมื่อ c เป็นค่าคงที่:

E(c) = c

ผลิตภัณฑ์

เมื่อ X และ Y เป็นตัวแปรสุ่มอิสระ:

E(X ⋅Y) = E(X) ⋅ E(Y)

ความคาดหวังแบบมีเงื่อนไข
