ppm คืออะไร?

ppm เป็นตัวย่อของส่วนในล้านส่วนppm คือค่าที่แสดงส่วนของจำนวนเต็มในหน่วย 1/1000000

ppm คือปริมาณไร้มิติ อัตราส่วนของ 2 ปริมาณในหน่วยเดียวกันตัวอย่างเช่น มก./กก.

หนึ่ง ppm เท่ากับ 1/1000000 ของทั้งหมด:

1ppm = 1/1000000 = 0.000001 = 1×10-6

 

หนึ่ง ppm เท่ากับ 0.0001%:

1ppm = 0.0001%

พีพีเอ็มดับเบิลยู

ppmw เป็นตัวย่อของส่วนต่อล้านน้ำหนัก หน่วยย่อยของ ppm ที่ใช้สำหรับส่วนของน้ำหนัก เช่น มิลลิกรัมต่อกิโลกรัม (มก./กก.)

พีพีเอ็มวี

ppmv เป็นตัวย่อของส่วนต่อล้านปริมาตร ซึ่งเป็นหน่วยย่อยของ ppm ที่ใช้สำหรับส่วนของปริมาตร เช่น มิลลิลิตรต่อลูกบาศก์เมตร (มล./ม. 3 )

ความเข้มข้นของสารเคมี

ppm ใช้ในการวัดความเข้มข้นของสารเคมี โดยปกติจะอยู่ในสารละลายของน้ำ

ความเข้มข้นของตัวถูกละลาย 1 ppm คือความเข้มข้นของตัวถูกละลาย 1/1000000 ของสารละลาย

ความเข้มข้น C ในหน่วย ppm คำนวณจากตัวถูกละลายมวล m ตัวถูกละลาย ในหน่วยมิลลิกรัมและสารละลายมวล m ในหน่วยมิลลิกรัม

C(ppm) = 1000000 × msolute / (msolution + msolute)

 

โดยปกติแล้วตัวถูกละลายมวล m ตัวถูกละลายจะน้อยกว่าสารละลายมวล m มาก

msolute ≪ msolution

 

จากนั้นความเข้มข้น C ในหน่วย ppm จะเท่ากับ 1000000 เท่าของตัวถูกละลายมวลm ตัวถูกละลาย ในหน่วยมิลลิกรัม (มก.) หารด้วยสารละลายมวลmในหน่วยมิลลิกรัม (มก.):

C(ppm) = 1000000 × msolute (mg) / msolution (mg)

 

ความเข้มข้น C ใน ppm ยังเท่ากับตัวถูกละลายมวลm ตัวถูกละลายในหน่วยมิลลิกรัม (มก.) หารด้วยสารละลายมวลmใน หน่วยกิโลกรัม (กก.):

C(ppm) = msolute (mg) / msolution (kg)

 

เมื่อสารละลายเป็นน้ำ ปริมาตรของมวล 1 กิโลกรัมจะอยู่ที่ประมาณ 1 ลิตร

ความเข้มข้น C ในหน่วย ppm ยังเท่ากับมวลของตัวถูกละลายm ตัวถูกละลาย ในหน่วยมิลลิกรัม (มก.) หารด้วยปริมาตรสารละลายน้ำVใน หน่วยลิตร (ลิตร):

C(ppm) = msolute (mg) / Vsolution (l)

 

ความเข้มข้นของ CO 2

ความเข้มข้นของก๊าซคาร์บอนไดออกไซด์ (CO 2 ) ในชั้นบรรยากาศอยู่ที่ประมาณ 388ppm

ความเสถียรของความถี่

ความเสถียรของความถี่ของส่วนประกอบออสซิลเลเตอร์อิเล็กทรอนิกส์สามารถวัดได้ในหน่วย ppm

ความแปรผันของความถี่สูงสุด Δ fหารด้วยความถี่ f เท่ากับความเสถียรของความถี่

Δf(Hz) / f(Hz) = FS(ppm) / 1000000

 
ตัวอย่าง

ออสซิลเลเตอร์ที่มีความถี่ 32MHz และความแม่นยำ ±200ppm มีความแม่นยำของความถี่

Δf(Hz) = ±200ppm × 32MHz / 1000000 = ±6.4kHz

ดังนั้นออสซิลเลเตอร์จึงสร้างสัญญาณนาฬิกาภายในช่วง 32MHz±6.4kHz

ความแปรผันของความถี่ที่ให้มาเกิดจากการเปลี่ยนแปลงของอุณหภูมิ อายุ แรงดันไฟฟ้าของแหล่งจ่าย และการเปลี่ยนแปลงของโหลด

การแปลง PPM

วิธีแปลง ppm เป็นเศษส่วนทศนิยม

ส่วน P ในทศนิยมเท่ากับส่วน P ใน ppm หารด้วย 1000000:

P(decimal) = P(ppm) / 1000000

ตัวอย่าง

ค้นหาเศษส่วนทศนิยมของ 300ppm:

P(decimal) = 300ppm / 1000000 = 0.0003

วิธีแปลงเศษส่วนทศนิยมเป็น ppm

ส่วน P ใน ppm เท่ากับส่วน P ในทศนิยมคูณ 1000000:

P(ppm) = P(decimal) × 1000000

ตัวอย่าง

ค้นหาจำนวน ppm ใน 0.0034:

P(ppm) = 0.0034 × 1000000 = 3400ppm

วิธีแปลง ppm เป็นเปอร์เซ็นต์

ส่วน P เป็นเปอร์เซ็นต์ (%) เท่ากับส่วน P ใน ppm หารด้วย 10,000:

P(%) = P(ppm) / 10000

ตัวอย่าง

ค้นหาจำนวนเปอร์เซ็นต์ใน 6ppm:

P(%) = 6ppm / 10000 = 0.0006%

วิธีแปลงเปอร์เซ็นต์เป็น ppm

ส่วน P ใน ppm เท่ากับส่วน P ในเปอร์เซ็นต์ (%) คูณ 10,000:

P(ppm) = P(%) × 10000

ตัวอย่าง

ค้นหาจำนวน ppm ใน 6%:

P(ppm) = 6% × 10000 = 60000ppm

วิธีแปลง ppb เป็น ppm

ส่วน P ใน ppm เท่ากับส่วน P ใน ppb หารด้วย 1,000:

P(ppm) = P(ppb) / 1000

ตัวอย่าง

ค้นหาจำนวน ppm ใน 6ppb:

P(ppm) = 6ppb / 1000 = 0.006ppm

วิธีแปลง ppm เป็น ppb

ส่วน P ใน ppb เท่ากับส่วน P ใน ppm คูณ 1,000:

P(ppb) = P(ppm) × 1000

ตัวอย่าง

ค้นหาจำนวน ppb ใน 6ppm:

P(ppb) = 6ppm × 1000 = 6000ppb

วิธีแปลง มิลลิกรัมต่อลิตร เป็น ppm

ความเข้มข้น C ในส่วนต่อล้านส่วน (ppm) เท่ากับความเข้มข้น C ในหน่วยมิลลิกรัมต่อกิโลกรัม (มก./กก.) และเท่ากับ 1,000 เท่าของความเข้มข้น C ในหน่วยมิลลิกรัมต่อลิตร (มก./ลิตร) หารด้วยความหนาแน่นของสารละลาย ρ เป็นกิโลกรัมต่อลูกบาศก์เมตร (กก./ม. 3 ):

C(ppm) = C(mg/kg) = 1000 × C(mg/L) / ρ(kg/m3)

In water solution, the concentration C in parts-per million (ppm) is equal to 1000 times the concentration C in milligrams per liter (mg/L) divided by the water solution density at temperature of 20ºC, 998.2071 in kilograms per cubic meter (kg/m3) and approximately equal to the concentration C in milligrams per liter (mg/L):

C(ppm) = 1000 × C(mg/L) / 998.2071(kg/m3) ≈ 1(L/kg) × C(mg/L)

How to convert grams/liter to ppm

The concentration C in parts-per million (ppm) is equal to 1000 times the concentration C in grams per kilogram (g/kg) and equal to 1000000 times the concentration C in grams per liter (g/L), divided by the solution density ρ in kilograms per cubic meter (kg/m3):

C(ppm) = 1000 × C(g/kg) = 106 × C(g/L) / ρ(kg/m3)

ในสารละลายน้ำ ความเข้มข้น C ส่วนในล้านส่วน (ppm) เท่ากับ 1,000 เท่าของความเข้มข้น C ในหน่วยกรัมต่อกิโลกรัม (g/kg) และเท่ากับ 1000000 เท่าของความเข้มข้น C ในหน่วยกรัมต่อลิตร (g/L) หารด้วยความหนาแน่นของสารละลายน้ำที่อุณหภูมิ 20ºC 998.2071 หน่วยเป็นกิโลกรัมต่อลูกบาศก์เมตร (กก./ม. 3 ) และประมาณเท่ากับ 1,000 เท่าของความเข้มข้น C ในหน่วยมิลลิกรัมต่อลิตร (มก./ลิตร):

C(ppm) = 1000 × C(g/kg) = 106 × C(g/L) / 998.2071(kg/m3) ≈ 1000 × C(g/L)

วิธีแปลง โมล/ลิตร เป็น ppm

ความเข้มข้น C ในหน่วยส่วนต่อล้านส่วน (ppm) เท่ากับความเข้มข้น C ในหน่วยมิลลิกรัมต่อกิโลกรัม (มก./กก.) และเท่ากับ 1000000 เท่าของความเข้มข้นโมล (โมลาริตี) c ในโมลต่อลิตร (โมล/ลิตร) คูณ มวลโมลาร์ของตัวละลายมีหน่วยเป็นกรัมต่อโมล (กรัม/โมล) หารด้วยความหนาแน่นของสารละลาย ρ ในหน่วยกิโลกรัมต่อลูกบาศก์เมตร (กก./ม. 3 ):

C(ppm) = C(mg/kg) = 106 × c(mol/L) × M(g/mol) / ρ(kg/m3)

ในสารละลายน้ำ ความเข้มข้น C ในส่วนต่อล้านส่วน (ppm) จะเท่ากับความเข้มข้น C ในหน่วยมิลลิกรัมต่อกิโลกรัม (มก./กก.) และเท่ากับ 1000000 เท่าของความเข้มข้นของโมล (โมลาริตี) c ในโมลต่อลิตร (โมล/ลิตร) ) คูณมวลโมลาร์ของตัวถูกละลายในหน่วยกรัมต่อโมล (g/mol) หารด้วยความหนาแน่นของสารละลายน้ำที่อุณหภูมิ 20ºC 998.2071 ในหน่วยกิโลกรัมต่อลูกบาศก์เมตร (กก./ม. 3 ):

C(ppm) = C(mg/kg) = 106 × c(mol/L) × M(g/mol) / 998.2071(kg/m3) ≈ 1000 × c(mol/L) × M(g/mol)

วิธีแปลง ppm เป็น Hz

การเปลี่ยนแปลงความถี่ในหน่วยเฮิรตซ์ (Hz) เท่ากับความเสถียรของความถี่ FS ในหน่วย ppm คูณด้วยความถี่ในหน่วยเฮิรตซ์ (Hz) หารด้วย 1000000:

Δf(Hz) = ± FS(ppm) × f(Hz) / 1000000

ตัวอย่าง

ออสซิลเลเตอร์ที่มีความถี่ 32MHz และความแม่นยำ ±200ppm มีความแม่นยำของความถี่

Δf(Hz) = ±200ppm × 32MHz / 1000000 = ±6.4kHz

ดังนั้นออสซิลเลเตอร์จึงสร้างสัญญาณนาฬิกาภายในช่วง 32MHz±6.4kHz
