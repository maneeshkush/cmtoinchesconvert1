การทำให้เลขชี้กำลังเศษส่วนอย่างง่าย

ฐาน b ยกกำลัง n/m เท่ากับ:

bn/m = (m√b)n = m√(bn)

ตัวอย่าง:

ฐาน 2 ยกกำลัง 3/2 เท่ากับ 1 หารด้วยฐาน 2 ยกกำลัง 3:

23/2 = 2√(23) = 2.828

เศษส่วนอย่างง่ายด้วยเลขยกกำลัง

เศษส่วนที่มีเลขยกกำลัง:

(a / b)n = an / bn

ตัวอย่าง:

(4/3)3 = 43 / 33 = 64 / 27 = 2.37

เลขชี้กำลังที่เป็นลบ

ฐาน b ยกกำลังลบ n/m เท่ากับ 1 หารด้วยฐาน b ยกกำลัง n/m:

b-n/m = 1 / bn/m = 1 / (m√b)n

ตัวอย่าง:

ฐาน 2 ยกกำลังลบ 1/2 เท่ากับ 1 หารด้วยฐาน 2 ยกกำลัง 1/2:

2-1/2 = 1/21/2 = 1/√2 = 0.7071

เศษส่วนที่มีเลขชี้กำลังเป็นลบ

ฐาน a/b ยกกำลังลบ n เท่ากับ 1 หารด้วยฐาน a/b ยกกำลัง n:

(a/b)-n = 1 / (a/b)n = 1 / (an/bn) = bn/an

ตัวอย่าง:

ฐาน 2 ยกกำลังลบ 3 เท่ากับ 1 หารด้วยฐาน 2 ยกกำลัง 3:

(2/3)-2 = 1 / (2/3)2 = 1 / (22/32) = 32/22 = 9/4 = 2.25

การคูณเลขชี้กำลังที่เป็นเศษส่วน

การคูณเลขชี้กำลังเศษส่วนด้วยเลขชี้กำลังเศษส่วนเดียวกัน:

a n/m ⋅ b n/m = (a ⋅ b) n/m

ตัวอย่าง:

23/2 ⋅ 33/2 = (2⋅3)3/2 = 63/2 = √(63) = √216 = 14.7

 

การคูณเลขชี้กำลังที่เป็นเศษส่วนด้วยฐานเดียวกัน:

a n/m ⋅ a k/j = a (n/m)+(k/j)

ตัวอย่าง:

23/2 ⋅ 24/3 = 2(3/2)+(4/3) = 7.127

 

การคูณเลขชี้กำลังที่เป็นเศษส่วนด้วยเลขชี้กำลังและเศษส่วนที่แตกต่างกัน:

a n/m ⋅ b k/j

ตัวอย่าง:

23/2 ⋅ 34/3 = √(23) ⋅ 3√(34) = 2.828 ⋅ 4.327 = 12.237

การคูณเศษส่วนด้วยเลขยกกำลัง

การคูณเศษส่วนด้วยเลขชี้กำลังที่มีฐานเศษส่วนเท่ากัน:

(a / b) n ⋅ (a / b) m = (a / b) n+m

ตัวอย่าง:

(4/3)3 ⋅ (4/3)2 = (4/3)3+2 = (4/3)5 = 45 / 35 = 4.214

 

การคูณเศษส่วนด้วยเลขยกกำลังที่มีเลขชี้กำลังเดียวกัน:

(a / b) n ⋅ (c / d) n = ((a / b)⋅(c / d)) n

ตัวอย่าง:

(4/3)3 ⋅ (3/5)3 = ((4/3)⋅(3/5))3 = (4/5)3 = 0.83 = 0.8⋅0.8⋅0.8 = 0.512

 

การคูณเศษส่วนด้วยเลขยกกำลังที่มีฐานและเลขยกกำลังต่างกัน:

(a / b) n ⋅ (c / d) m

ตัวอย่าง:

(4/3)3 ⋅ (1/2)2 = 2.37 ⋅ 0.25 = 0.5925

การหารเลขชี้กำลังที่เป็นเศษส่วน

การหารเลขชี้กำลังเศษส่วนด้วยเลขชี้กำลังเศษส่วนเดียวกัน:

a n/m / b n/m = (a / b) n/m

ตัวอย่าง:

33/2 / 23/2 = (3/2)3/2 = 1.53/2 = √(1.53) = √3.375 = 1.837

 

การหารเลขชี้กำลังที่เป็นเศษส่วนด้วยฐานเดียวกัน:

a n/m / a k/j = a (n/m)-(k/j)

ตัวอย่าง:

23/2 / 24/3 = 2(3/2)-(4/3) = 2(1/6) = 6√2 = 1.122

 

การหารเลขชี้กำลังที่เป็นเศษส่วนด้วยเลขชี้กำลังและเศษส่วนที่แตกต่างกัน:

a n/m / b k/j

ตัวอย่าง:

23/2 / 34/3 = √(23) / 3√(34) = 2.828 / 4.327 = 0.654

การหารเศษส่วนด้วยเลขยกกำลัง

การหารเศษส่วนด้วยเลขชี้กำลังที่มีฐานเศษส่วนเท่ากัน:

(a / b)n / (a / b)m = (a / b)n-m

ตัวอย่าง:

(4/3)3 / (4/3)2 = (4/3)3-2 = (4/3)1 = 4/3 = 1.333

 

การหารเศษส่วนด้วยเลขยกกำลังที่มีเลขชี้กำลังเดียวกัน:

(a / b)n / (c / d)n = ((a / b)/(c / d))n = ((a⋅d / b⋅c))n

ตัวอย่าง:

(4/3)3 / (3/5)3 = ((4/3)/(3/5))3 = ((4⋅5)/(3⋅3))3 = (20/9)3 = 10.97

 

การหารเศษส่วนด้วยเลขยกกำลังที่มีฐานและเลขยกกำลังต่างกัน:

(a / b) n / (c / d) m

ตัวอย่าง:

(4/3)3 / (1/2)2 = 2.37 / 0.25 = 9.481

การบวกเลขชี้กำลังที่เป็นเศษส่วน

การเพิ่มเลขชี้กำลังที่เป็นเศษส่วนทำได้โดยการเพิ่มเลขชี้กำลังแต่ละตัวก่อนแล้วจึงเพิ่ม:

an/m + bk/j

ตัวอย่าง:

33/2 + 25/2 = √(33) + √(25) = √(27) + √(32) = 5.196 + 5.657 = 10.853

 

การเพิ่มฐาน b และเลขชี้กำลังเดียวกัน n/m:

bn/m + bn/m = 2bn/m

ตัวอย่าง:

42/3 + 42/3 = 2⋅42/3 = 2 ⋅ 3√(42) = 5.04

การลบเลขชี้กำลังที่เป็นเศษส่วน

การลบเลขชี้กำลังที่เป็นเศษส่วนทำได้โดยการเพิ่มเลขชี้กำลังแต่ละตัวก่อนแล้วจึงลบ:

an/m - bk/j

ตัวอย่าง:

33/2 - 25/2 = √(33) - √(25) = √(27) - √(32) = 5.196 - 5.657 = -0.488

 

การลบฐานเดียวกัน b และเลขยกกำลัง n/m:

3bn/m - bn/m = 2bn/m

ตัวอย่าง:

3⋅42/3 - 42/3 = 2⋅42/3 = 2 ⋅ 3√(42) = 5.04
