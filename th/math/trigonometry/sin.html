คำจำกัดความของไซน์

ในรูปสามเหลี่ยมมุมฉาก ABC ไซน์ของ α, sin(α) ถูกกำหนดให้เป็นอัตราส่วนระหว่างด้านตรงข้ามกับมุม α และด้านตรงข้ามกับมุมฉาก (ด้านตรงข้ามมุมฉาก):

sin α = a / c

ตัวอย่าง

a = 3"

c = 5"

sin α = a / c = 3 / 5 = 0.6

ฟังก์ชันไซน์ผกผัน

อาร์กไซน์ของ x ถูกกำหนดให้เป็นฟังก์ชันไซน์ผกผันของ x เมื่อ -1≤x≤1

เมื่อไซน์ของ y เท่ากับ x:

sin y = x

แล้วอาร์กไซน์ของ x เท่ากับฟังก์ชันไซน์ผกผันของ x ซึ่งเท่ากับ y:

arcsin x = sin-1(x) = y
