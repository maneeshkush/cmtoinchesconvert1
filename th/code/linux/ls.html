คำสั่ง ls ใน Linux/Unix

lsเป็นคำสั่งเชลล์ของ Linux ที่แสดงเนื้อหาไดเร็กทอรีของไฟล์และไดเร็กทอรี

  • ไวยากรณ์ ls
  • ตัวเลือก ls
  • ตัวอย่าง
  • เครื่องกำเนิดรหัส ls

ไวยากรณ์ ls

$ ls [options] [file|dir]

ตัวเลือกคำสั่ง ls

คำสั่ง ls ตัวเลือกหลัก:

ตัวเลือก คำอธิบาย
ls -a รายการไฟล์ทั้งหมดรวมถึงไฟล์ที่ซ่อนที่ขึ้นต้นด้วย '.'
ls -- สี รายการสี [=เสมอ/ไม่เคย/อัตโนมัติ]
ls -d รายการไดเร็กทอรี - ด้วย ' */'
ls -F เพิ่มอักขระ */=>@| หนึ่งตัวเพื่อเข้า
ls -i หมายเลขดัชนีไอโหนดของไฟล์รายการ
ls -l รายการที่มีรูปแบบยาว - แสดงการอนุญาต
ls -la รายการรูปแบบยาวรวมถึงไฟล์ที่ซ่อนอยู่
ls -lh รายการรูปแบบยาวที่มีขนาดไฟล์ที่อ่านได้
ls -ls รายการที่มีรูปแบบยาวพร้อมขนาดไฟล์
ls -r รายการในลำดับย้อนกลับ
ls -R รายการแผนผังไดเร็กทอรีแบบเรียกซ้ำ
ls -s รายการขนาดไฟล์
ls -S จัดเรียงตามขนาดไฟล์
ls -t จัดเรียงตามเวลาและวันที่
ls -X เรียงตามชื่อนามสกุล

ตัวอย่างคำสั่ง ls

คุณสามารถกดปุ่มแท็บเพื่อเติมชื่อไฟล์หรือโฟลเดอร์โดยอัตโนมัติ

รายการไดเร็กทอรีเอกสาร/หนังสือที่มี พาธ สัมพัทธ์ :

$ ls Documents/Books

 

รายการไดเร็กทอรี/home/user/Documents/Booksที่มีพาธสัมบูรณ์

$ ls /home/user/Documents/Books

 

รายการไดเรกทอรีราก:

$ ls /

 

รายการไดเร็กทอรีหลัก:

$ ls ..

 

แสดงโฮมไดเร็กทอรีของผู้ใช้ (เช่น: /home/user):

$ ls ~

 

รายการที่มีรูปแบบยาว:

$ ls -l

 

แสดงไฟล์ที่ซ่อน:

$ ls -a

 

รายการที่มีรูปแบบยาวและแสดงไฟล์ที่ซ่อน:

$ ls -la

 

เรียงตามวันที่/เวลา:

$ ls -t

 

เรียงตามขนาดไฟล์:

$ ls -S

 

รายการไดเรกทอรีย่อยทั้งหมด:

$ ls *

 

รายการแผนผังไดเร็กทอรีแบบเรียกซ้ำ:

$ ls -R

 

แสดงเฉพาะไฟล์ข้อความที่มีสัญลักษณ์แทน:

$ ls *.txt

 

ls เปลี่ยนเส้นทางไปยังไฟล์เอาต์พุต:

$ ls > out.txt

 

รายการไดเร็กทอรีเท่านั้น:

$ ls -d */

 

แสดงรายการไฟล์และไดเร็กทอรีพร้อมเส้นทางแบบเต็ม:

$ ls -d $PWD/*

เครื่องกำเนิดรหัส ls

เลือก ตัวเลือก lsแล้วกด ปุ่ม สร้างรหัส :

ตัวเลือก 
  รูปแบบรายการยาว (-l)
  รายการไฟล์ทั้งหมด / ไฟล์ที่ซ่อน (-a)
  แผนผังไดเร็กทอรีรายการแบบเรียกซ้ำ (-R)
  รายการในลำดับย้อนกลับ (-r)
  รายการที่มีเส้นทางแบบเต็ม (-d $PWD/*)
จัดเรียงตาม:
ไฟล์/โฟลเดอร์

โฟลเดอร์:
