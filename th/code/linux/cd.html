คำสั่ง cd ใน Linux/Unix

cdเป็นคำสั่ง Linux เพื่อเปลี่ยนไดเร็กทอรี/โฟลเดอร์ของเชลล์ของเทอร์มินัล

คุณสามารถกดปุ่มแท็บเพื่อให้ชื่อไดเร็กทอรีสมบูรณ์โดยอัตโนมัติ

  • ไวยากรณ์ซีดี
  • ตัวอย่างซีดี

ไวยากรณ์ซีดี

$ cd [directory]

ตัวอย่างคำสั่ง cd

เปลี่ยนเป็นโฮมไดเร็กทอรี (กำหนดโดย $HOME ตัวแปรสภาพแวดล้อม):

$ cd

 

เปลี่ยนเป็นโฮมไดเร็กตอรี่ด้วย:

$ cd ~

 

เปลี่ยนเป็นไดเรกทอรีราก:

$ cd /

 

เปลี่ยนเป็นไดเร็กทอรีหลัก:

$ cd ..

 

เปลี่ยนเป็นเอกสาร ไดเรกทอรีย่อย :

$ cd Documents

 

เปลี่ยนเป็นไดเรกทอรีย่อยDocuments/Books :

$ cd Documents/Books

 

เปลี่ยนเป็นไดเรกทอรีด้วยเส้นทางที่แน่นอน/home/user/Desktop :

$ cd /home/user/Desktop

 

เปลี่ยนชื่อไดเร็กทอรีด้วยช่องว่าง - รูปภาพของฉัน :

$ cd My\ Images

หรือ

$ cd "My Images"

หรือ

$ cd 'My Images'
