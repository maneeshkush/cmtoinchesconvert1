คอมไพเลอร์ GCC C

GCCย่อมาจาก GNU Compiler Collection ซึ่งเป็นคอมไพเลอร์ C สำหรับ Linux

  • ไวยากรณ์ gcc
  • ตัวเลือก gcc
  • ตัวอย่าง gcc
  • เครื่องกำเนิดรหัส gcc

ไวยากรณ์ GCC

$ gcc [options] [source files] [object files] [-o output file]

ตัวเลือก GCC

ตัวเลือกหลักของ GCC:

ตัวเลือก คำอธิบาย
gcc -c รวบรวมไฟล์ต้นฉบับไปยังไฟล์วัตถุโดยไม่ต้องเชื่อมโยง
gcc -D ชื่อ [ =value ] กำหนดมาโครตัวประมวลผลล่วงหน้า
gcc -fPIC สร้างรหัสตำแหน่งอิสระสำหรับไลบรารีที่ใช้ร่วมกัน
gcc -g ระดับ สร้างข้อมูลการแก้ปัญหาที่จะใช้โดย GDB
gcc - ฉันเป็นผู้อำนวยการ เพิ่มไดเร็กทอรีรวมของไฟล์ส่วนหัว
gcc -l lib เชื่อมโยงกับไฟล์ไลบรารี
gcc -L ผบ ค้นหาไฟล์ไลบรารีในไดเร็กทอรี
ไฟล์เอาต์พุต gcc -o เขียน build เอาต์พุตไปยังไฟล์เอาต์พุต
gcc -O ระดับ ปรับให้เหมาะสมสำหรับขนาดโค้ดและเวลาดำเนินการ
gcc - แบ่งปัน สร้างไฟล์วัตถุที่ใช้ร่วมกันสำหรับไลบรารีที่ใช้ร่วมกัน
gcc - ชื่อ U เลิกกำหนดมาโครตัวประมวลผลล่วงหน้า
gcc -ว ปิดใช้งานข้อความเตือนทั้งหมด
gcc -ผนัง เปิดใช้งานข้อความเตือนทั้งหมด
gcc -Wextra เปิดใช้งานข้อความเตือนเพิ่มเติม

ตัวอย่าง GCC

คอมไพล์file1.cและfile2.cและลิงก์ไปยังไฟล์เอาต์พุตexecfile :

$ gcc file1.c file2.c -o execfile

 

เรียกใช้ไฟล์เอาต์พุตexecfile :

$ ./execfile

 

รวบรวมfile1.cและfile2.cโดยไม่ต้องเชื่อมโยง:

$ gcc -c file1.c file2.c

 

คอมไพล์myfile.cพร้อมข้อมูลการดีบักและลิงก์ไปยังไฟล์เอาต์พุตexecfile :

$ gcc -g myfile.c -o execfile

 

คอมไพล์myfile.cโดยเปิดใช้งานข้อความเตือนและลิงก์ไปยังไฟล์เอาต์พุตexecfile :

$ gcc -Wall myfile.c -o execfile

 

คอมไพล์myfile.cด้วยและเชื่อมโยงกับไลบรารีแบบคงที่libmath.aซึ่งอยู่ใน/user/local/mathไปยังไฟล์เอาต์พุตexecfile :

$ gcc -static myfile.c -L/user/local/math -lmath -o execfile

 

คอมไพล์myfile.cพร้อมการเพิ่มประสิทธิภาพและลิงก์ไปยังไฟล์เอาต์พุตexecfile :

$ gcc -O myfile.c -o execfile
