แฟล็กตัวเลือก gcc -L / -l

gcc -l เชื่อมโยงกับไฟล์ไลบรารี

gcc -L ค้นหาไฟล์ไลบรารีในไดเร็กทอรี

ไวยากรณ์

$ gcc [options] [source files] [object files] [-Ldir] -llibname [-o outfile]

 

ลิงก์ -l กับชื่อไลบรารีโดยไม่มีคำนำหน้า lib และนามสกุล.aหรือ . so

ตัวอย่าง

ตัวอย่างที่ 1

สำหรับไฟล์ไลบรารีแบบสแตติกlibmath การใช้-lmath :

$ gcc -static myfile.c -lmath -o myfile

 
ตัวอย่างที่ 2

สำหรับไฟล์ไลบรารีที่ใช้ร่วมกันlibmath ดังนั้นใช้-lmath :

$ gcc myfile.c -lmath -o myfile

 
ตัวอย่างที่ 3

ไฟล์ 1.c:

// file1.c
#include <stdio.h>

void main()
{
    printf("main() run!\n");
    myfunc();
}

 

ไฟล์ 2.c:

// file2.c
#include <stdio.h>

void myfunc()
{
    printf("myfunc() run!\n");
}

 

สร้างfile2.cคัดลอกไฟล์วัตถุfile2.oไปยัง ไดเร็กทอรี libs และเก็บถาวรไปยังไลบรารีแบบสแตติกlibmylib.a :

$ gcc -c file2.c
$ mkdir libs
$ cp file2.o libs
$ cd libs
$ ar rcs libmylib.a file2.o

 

สร้างfile1.cด้วยไลบรารีlibmylib.a แบบคงที่ ใน ไดเร็กทอรีlibs

สร้างโดยไม่มีผลลัพธ์ -L โดยมีข้อผิดพลาด:

$ gcc file1.c -lmylib -o outfile
/usr/bin/ld: cannot find -llibs
collect2: ld returned 1 exit status
$

สร้างด้วย -L ​​และเรียกใช้:

$ gcc file1.c -Llibs -lmylib -o outfile
$ ./outfile
main() run!
myfunc() run!
$
