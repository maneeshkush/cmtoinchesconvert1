วิธีแปลง BTU/ชม. เป็น กิโลวัตต์

วิธีแปลง หน่วยความร้อนบริติชต่อชั่วโมง (BTU/ชม.) เป็น กิโลวัตต์ (kW )

หนึ่ง BTU/ชม. เท่ากับ 0.00029307107 กิโลวัตต์:

1 BTU/hr = 0.00029307107 kW

สูตรการแปลง BTU/ชม. เป็นกิโลวัตต์

กำลังไฟฟ้าเป็นกิโลวัตต์  P (kW) เท่ากับกำลังไฟฟ้าในหน่วย BTU IT  ต่อชั่วโมง  P ( BTU /ชม. ) หารด้วย 3412.142:

P(kW) = P(BTU/hr) / 3412.142

ตัวอย่างที่ 1

แปลง 10,000 BTU/ชม. เป็น กิโลวัตต์:

P(kW) = 10000 BTU/hr / 3412.142 = 2.930kW

ตัวอย่างที่ 2

แปลง 30000 BTU/ชม. เป็น กิโลวัตต์:

P(kW) = 30000 BTU/hr / 3412.142 = 8.792kW

ตัวอย่างที่ 3

แปลง 40000 BTU/ชม. เป็น กิโลวัตต์:

P(kW) = 40000 BTU/hr / 3412.142 = 11.722kW

ตัวอย่างที่ 4

แปลง 60000 BTU/ชม. เป็น กิโลวัตต์:

P(kW) = 60000 BTU/hr / 3412.142 = 17.584kW

ตัวอย่างที่ 5

แปลง 110000 BTU/ชม. เป็น กิโลวัตต์:

P(kW) = 110000 BTU/hr / 3412.142 = 32.237kW
