การแปลง ppm เป็น มก./ลิตร

โมลต่อลิตร (mol/L) เป็นมิลลิกรัมต่อลิตร (mg/L) เป็น ppm เครื่องคำนวณการแปลง

ตัวแปลงสารละลายน้ำ ความเข้มข้นโมลาร์ (โมลาริตี) เป็น มิลลิกรัมต่อลิตรเป็นส่วนในล้านส่วน (ppm)

ความเข้มข้นของโมลาร์ (โมลาริตี): ค(โมล/ลิตร) = นางสาว
มวลโมลาร์ของตัวถูกละลาย: M (กรัม/โมล) = กรัม/โมล  
มิลลิกรัมต่อลิตร: C (มก./ลิตร) = มก./ลิตร
อุณหภูมิของน้ำ: ที(°ซ) = องศาเซลเซียส  
ส่วนในล้าน: C (มก./กก.) = หน้าต่อนาที

ppm และ mg/l คืออะไร?

PPM และ mg/L เป็นมาตรวัดความเข้มข้นของสารที่แตกต่างกันสองแบบ

PPM หรือส่วนในล้านส่วนคือจำนวนส่วนของสารในสารละลายหรือของผสมหนึ่งล้านส่วนตัวอย่างเช่น สมมติว่าคุณต้องการวัดความเค็มของน้ำPPM คือจำนวนส่วนของเกลือต่อล้านส่วนของสารละลายที่สมบูรณ์ของทั้งน้ำและเกลือ


Mg/L หรือมิลลิกรัมต่อลิตรเป็นหน่วยวัดความเข้มข้นมันบอกว่าสามารถพบสารได้กี่มิลลิกรัมในสารละลายหรือของผสมหนึ่งลิตร

PPM และ มก./ลิตรแปลงระหว่าง

ความสัมพันธ์ระหว่าง PPM และ mg/L ขึ้นอยู่กับความหนาแน่นของตัวถูกละลายลองนึกภาพว่าคุณเติมสาร 10 กรัมลงในน้ำหนึ่งลิตรหากสารมีความหนาแน่นเท่ากับน้ำมัน จะมีปริมาตรน้อยลง และเป็นผลให้อัตราส่วน ppm ของสารละลายมีขนาดเล็กลงสำหรับสารที่มีความหนาแน่นต่ำ (เช่น แอลกอฮอล์) อัตราส่วน ppm จะสูงขึ้นมาก แม้ว่าอัตราส่วน mg/L จะคงที่ก็ตาม

ในการแปลง PPM เป็น mg/L ให้ทำตามขั้นตอนเหล่านี้:

1. เลือกตัวถูกละลาย - สารนี้เจือจางด้วยน้ำ อะซิโตน หรืออย่างอื่นหรือไม่?เลือกน้ำมันที่มีความหนาแน่นเท่ากับ 920 กก./ม.

2. ตั้งค่า ppmสำหรับโซลูชันของคุณสมมติว่าคุณทำสารละลายด้วยน้ำมัน 1,230 ppm

3. ใช้สูตรต่อไปนี้เพื่อหาอัตราส่วน mg/L:

milligrams per liter = PPM * density / 1,000

4. ในกรณีนี้

milligrams per liter = 1,230 * 920 / 1,000 = 1,131.6 mg/L

ซึ่งหมายความว่า 1,230 ppm เทียบเท่ากับ 1,131.6 มก./ล. ของน้ำมันในน้ำ

กรณีพิเศษ: น้ำ

คุณต้องทราบว่าความหนาแน่นของน้ำเท่ากับ 1,000 กก./ม.หมายความว่า น้ำทุกลูกบาศก์เมตรมีน้ำหนัก 1,000 กิโลกรัมลองคำนวณหน่วยใหม่:

1,000 kg/m³
= 1,000,000 g/m³
= 1,000,000,000 mg/m³
= 1,000,000 mg/dm³
= 1,000,000 mg/L

ซึ่งหมายความว่าน้ำทุกลิตรมีน้ำหนึ่งล้านมิลลิกรัมพอดีหมายความว่า ถ้าความหนาแน่นของสารที่ละลายในน้ำเท่ากับหรือเกือบเท่ากับความหนาแน่นของน้ำ คุณก็สันนิษฐานได้ว่า 1ppm = 1 mg/L.

โปรดจำไว้ว่าการเปรียบเทียบนี้ใช้ได้กับเงื่อนไขที่เฉพาะเจาะจงมากเท่านั้น นั่นคือน้ำบริสุทธิ์ที่ความดันและอุณหภูมิมาตรฐาน

การคำนวณความเข้มข้นของโมลาร์

หากคุณทราบอัตราส่วน mg/L ของสารละลายแล้ว คุณสามารถใช้ตัวแปลง PPM เป็น mg/L เพื่อคำนวณโมลาริตีได้พารามิเตอร์นี้อธิบายจำนวนโมลในสารละลายหนึ่งลิตรและแสดงเป็นฟันกราม(1 M = mol/L).

ในการค้นหาโมลาริตี คุณจำเป็นต้องทราบพารามิเตอร์เพิ่มเติม - มวลโมลาร์ของตัวถูกละลายของคุณ (ปริมาณน้ำที่ละลายในนั้น)แสดงเป็นกรัมต่อโมลใช้สูตรต่อไปนี้:

molarity = milligrams per liter / (molar mass * 1,000)

ตัวอย่างเช่น มวลโมลาร์ของน้ำมันเท่ากับ 900 กรัม/โมลเราสามารถใช้อัตราส่วนมิลลิกรัมต่อลิตรที่คำนวณไว้ก่อนหน้านี้เพื่อหาค่าโมลาริตี:

molarity = 1,131.6 / (900 * 1,000) = 0.00126 M

ppm คืออะไรและใช้อย่างไร?

PPM ย่อมาจาก "parts per million" และเป็นหน่วยของความเข้มข้นที่มักใช้ในวิชาเคมีและวิทยาศาสตร์สิ่งแวดล้อมหมายถึงจำนวนส่วนของสารเฉพาะต่อล้านส่วนของสารละลายหรือของผสมPPM มักใช้เพื่อแสดงความเข้มข้นของสารปนเปื้อนหรือมลพิษในน้ำ อากาศ หรือดิน

มก./ลิตร คืออะไร และใช้อย่างไร?

มิลลิกรัม/ลิตร หรือที่เรียกว่ามิลลิกรัมต่อลิตร เป็นหน่วยของความเข้มข้นที่ใช้แสดงปริมาณของสารเฉพาะในสารละลายหรือของผสมหมายถึงจำนวนมิลลิกรัมของสารที่มีอยู่ในสารละลายหนึ่งลิตรเช่นเดียวกับ ppm มก./ลิตร มักใช้เพื่อแสดงความเข้มข้นของสารปนเปื้อนหรือมลพิษในน้ำ อากาศ หรือดิน

ฉันจะแปลง ppm เป็น มก./ลิตร ได้อย่างไร

ในการแปลง ppm เป็น mg/liter คุณสามารถใช้สูตรต่อไปนี้:

มก./ลิตร = (ppm * น้ำหนักโมเลกุลของสาร) / 1,000

ตัวอย่างเช่น หากคุณต้องการแปลงความเข้มข้น 50 ppm ของสารที่มีน้ำหนักโมเลกุล 100 กรัม/โมล การแปลงจะเป็น:

มก./ลิตร = (50 ppm * 100 ก./โมล) / 1,000 = 5 มก./ลิตร

ฉันสามารถใช้เครื่องคิดเลขหรือเครื่องมือแปลงออนไลน์เพื่อทำการแปลงได้หรือไม่

ใช่ มีเครื่องมือแปลงและเครื่องคิดเลขออนไลน์มากมายที่สามารถทำการแปลงจาก ppm เป็น mg/liter ให้คุณได้เพียงป้อนความเข้มข้นในหน่วย ppm และน้ำหนักโมเลกุลของสาร จากนั้นเครื่องมือจะคำนวณความเข้มข้นเทียบเท่าในหน่วย มก./ลิตร

หน่วยความเข้มข้น ppm และ mg/ลิตร ใช้แทนกันได้หรือไม่

ในขณะที่สามารถใช้ ppm และ mg/liter เพื่อแสดงความเข้มข้นได้ แต่ก็ไม่ใช่หน่วยที่ใช้แทนกันได้ppm อ้างอิงจากอัตราส่วนของส่วนต่อล้าน ส่วน mg/ลิตร อ้างอิงจากอัตราส่วนของมิลลิกรัมต่อลิตรสิ่งสำคัญคือต้องใช้หน่วยความเข้มข้นที่ถูกต้องตามบริบทและประเภทของการวัดที่กำลังดำเนินการ

ppm และ mg/l สัมพันธ์กันอย่างไร?

ความสัมพันธ์ระหว่าง ppm และ มก./ลิตร ขึ้นอยู่กับน้ำหนักโมเลกุลของสารที่กำลังวัดการแปลงจาก ppm เป็น มก./ลิตร เกี่ยวข้องกับการคูณความเข้มข้นใน ppm ด้วยน้ำหนักโมเลกุลของสารและหารด้วย 1,000 ความเข้มข้นที่ได้ในหน่วย มก./ลิตร จะสูงหรือต่ำขึ้นอยู่กับน้ำหนักโมเลกุลของสาร
