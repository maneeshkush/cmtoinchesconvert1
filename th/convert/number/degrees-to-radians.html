แปลงเรเดียนเป็นองศา ►

วิธีแปลงองศาเป็นเรเดียน

Pi เรเดียนเท่ากับ 180 องศา:

π rad = 180°

หนึ่งองศาเท่ากับ 0.01745329252 เรเดียน:

1° = π/180° = 0.005555556π = 0.01745329252 rad

ดังนั้น มุม α ในหน่วยเรเดียนจึงเท่ากับมุม α ในหน่วยองศา คูณ ค่าคงที่ pi หารด้วย [180] องศา

α(radians) = α(degrees) × π / 180°

หรือ

radians = degrees × π / 180°

ตัวอย่างที่ 1

แปลงมุม 35 องศาเป็นเรเดียน:

α(radians) = α(degrees) × π / 180° = 35° × 3.14159 / 180° = 0.6108652382 rad

ตัวอย่างที่ 2

แปลงมุม 90 องศาเป็นเรเดียน:

α(radians) = α(degrees) × π / 180° = 90° × 3.14159 / 180° = 1.5707963268 rad

ตัวอย่างที่ 3

แปลงมุม 45 องศาเป็นเรเดียน:

α(radians) = α(degrees) × π / 180° = 45° × 3.14159 / 180° = 0.7853981634 rad

ตัวอย่างที่ 4

แปลงมุม 60 องศาเป็นเรเดียน:

α(radians) = α(degrees) × π / 180° = 60° × 3.14159 / 180° = 1.0471975512 rad
