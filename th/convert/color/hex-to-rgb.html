การแปลงฐานสิบหกเป็น RGB

  1. รับรหัสสีฐานสิบหก 2 หลักซ้ายและแปลงเป็นค่าทศนิยมเพื่อให้ได้ระดับสีแดง
  2. รับรหัสสีฐานสิบหก 2 หลักตรงกลางและแปลงเป็นค่าทศนิยมเพื่อให้ได้ระดับสีเขียว
  3. รับรหัสสีฐานสิบหก 2 หลักที่ถูกต้องและแปลงเป็นค่าทศนิยมเพื่อให้ได้ระดับสีฟ้า

ตัวอย่าง #1

แปลงรหัสสี hex สีแดง FF0000 เป็นสี RGB:

Hex = FF0000

ดังนั้นสี RGB คือ:

R = FF16 = 25510

G = 0016 = 010

B = 0016 = 010

หรือ

RGB = (255, 0, 0)

ตัวอย่าง #2

แปลงรหัสสี gold hex FFD700 เป็นสี RGB:

Hex = FFD700

ดังนั้นสี RGB คือ:

R = FF16 = 25510

G = D716 = 21510

B = 0016 = 010

หรือ

RGB = (255, 215, 0)

  • รหัสสี RGB

คุณสมบัติของเครื่องมือแปลงสี Hex เป็น RGB

  1. แปลงรหัสสีฐานสิบหกเป็น RGB: เครื่องมือควรสามารถแปลงรหัสสีฐานสิบหก เช่น #FF0000 เป็นค่า RGB ที่สอดคล้องกัน ซึ่งจะเป็น (255, 0, 0)

  2. แปลง RGB เป็นเลขฐานสิบหก: เครื่องมือควรสามารถแปลงค่า RGB เช่น (255, 0, 0) เป็นรหัสสีเลขฐานสิบหกที่สอดคล้องกัน ซึ่งจะเป็น #FF0000

  3. รองรับรูปแบบสีที่หลากหลาย: เครื่องมือควรรองรับรูปแบบสีที่หลากหลาย รวมถึงรหัสเลขฐานสิบหก 3 หลักและ 6 หลัก ตลอดจนค่า RGB ในรูปแบบต่างๆ (เช่น มีหรือไม่มีวงเล็บ มีหรือไม่มีเครื่องหมายจุลภาค)

  4. แสดงตัวอย่างสี: เครื่องมือควรแสดงตัวอย่างสีที่กำลังแปลง เพื่อให้ผู้ใช้สามารถดูสีที่ได้ก่อนทำการแปลง

  5. อนุญาตให้ผู้ใช้ป้อนสีด้วยตนเอง: เครื่องมือควรอนุญาตให้ผู้ใช้ป้อนรหัสสีหรือค่าในรูปแบบที่ต้องการด้วยตนเอง แทนที่จะกำหนดให้ใช้ตัวเลือกสีหรือเครื่องมือภายนอกอื่นๆ

  6. รองรับการคัดลอกและวาง: เครื่องมือควรอนุญาตให้ผู้ใช้คัดลอกและวางรหัสสีหรือค่าจากแหล่งอื่นๆ เช่น โปรแกรมแก้ไขรูปภาพหรือเว็บไซต์สำหรับการแปลง

  7. ให้ผลลัพธ์ที่ชัดเจนและกระชับ: เครื่องมือควรแสดงรหัสสีหรือค่าผลลัพธ์ในลักษณะที่ชัดเจนและรัดกุม ทำให้ผู้ใช้เข้าใจและใช้ผลลัพธ์การแปลงได้ง่าย

  8. เป็นมิตรกับผู้ใช้: เครื่องมือควรใช้งานง่ายพร้อมอินเทอร์เฟซที่เรียบง่ายและใช้งานง่ายซึ่งทำให้ผู้ใช้ทำการแปลงได้ง่าย
