วิธีแปลง hex เป็นสี rgb

วิธีแปลงรหัสสีฐานสิบหกเป็นสี RGB

รหัสสีฐานสิบหก รหัส
สีฐานสิบหกคือเลขฐานสิบหก 6 หลัก (ฐาน 16):

RRGGBB 16

ตัวเลขด้านซ้าย 2 หลักแสดงถึงสีแดง

ตัวเลขตรงกลาง 2 ตัวแสดงถึงสีเขียว

ตัวเลขด้านขวา 2 หลักแสดงถึงสีฟ้า

สี RGB

สี RGB คือการรวมกันของสีแดง สีเขียว และสีน้ำเงิน:

(อาร์, จี, บี)

สีแดง เขียว และน้ำเงิน แต่ละตัวใช้ 8 บิต โดยมีค่าจำนวนเต็มตั้งแต่ 0 ถึง 255

ดังนั้นจำนวนสีที่สามารถสร้างได้คือ:

256×256×256 = 16777216 = 1000000 16

การแปลงฐานสิบหกเป็น rgb

1. นำรหัสสีฐานสิบหก 2 หลักซ้ายและแปลงเป็นค่าทศนิยมเพื่อให้ได้ระดับสีแดง
2. รับรหัสสีฐานสิบหก 2 หลักกลางและแปลงเป็นค่าทศนิยมเพื่อให้ได้ระดับสีเขียว
3. ค้นหารหัสสีฐานสิบหกที่ถูกต้อง 2 หลักและแปลงเป็นค่าทศนิยมเพื่อให้ได้ระดับสีฟ้า

ตัวอย่างที่ 1
แปลงรหัสสีฐานสิบหกสีแดง FF0000 เป็นสี RGB:

เลขฐานสิบหก = FF0000

ดังนั้นสี RGB คือ:

R = FF16 = 25510

G = 0016 = 010

B = 0016 = 010

หรือ

RGB = (255, 0, 0)

ตัวอย่าง #2
แปลงรหัสสี Gold Hex FFD700 เป็นสี RGB:

Hex = FFD700

ดังนั้นสี RGB คือ:

R = FF16 = 25510

G = D716 = 21510

B = 0016 = 010

หรือ

RGB = (255, 215, 0)
