สี CSS

รหัสสีสามารถเป็นหนึ่งใน:

รูปแบบฐานสิบหก: #rrggbb

รูปแบบ RGB: rgb (แดง, เขียว, น้ำเงิน)

รูปแบบชื่อ: ชื่อ

ตัวอย่าง

สำหรับสีส้ม:

รูปแบบเลขฐานสิบหก: #FFA500

รูปแบบ RGB: rgb(255,165,0)

รูปแบบชื่อ: ส้ม

การตั้งค่าสีองค์ประกอบ

สำหรับองค์ประกอบเฉพาะ:

<element style="color: code;"></element>

สำหรับองค์ประกอบประเภทเดียวกันทั้งหมดใส่โค้ดในแท็ก <style> ในส่วนหัว:

<style>
   element { color: code; }
</style>

ตัวอย่าง

การตั้งค่าสีข้อความย่อหน้าเป็นสีแดง:

การตั้งค่าสีของย่อหน้าเฉพาะ:

<p style="color: #FF0000;">Some text ...</p>

ดู:

ข้อความบางส่วน ...

การกำหนดสีของย่อหน้าทั้งหมด

<style>
   p { color: #FF0000; }
</style>

การตั้งค่าสีพื้นหลังขององค์ประกอบ

element { background: code; }

ตัวอย่าง

การตั้งค่าสีพื้นหลังย่อหน้าเป็นสีแดง:

การตั้งค่าสีพื้นหลังของย่อหน้าเฉพาะ:

<p style="background: #FF0000;">Some text ...</p>

ดู:

ข้อความบางส่วน ...

การตั้งค่าสีพื้นหลังของย่อหน้าทั้งหมด:

<style>
   p { background: #FF0000; }
</style>

การตั้งค่าสีเส้นขอบขององค์ประกอบ

element { border-color: topcode rightcode bottomcode leftcode }

ตัวอย่าง

การตั้งค่าสีขอบย่อหน้าเป็นสีแดง เขียว น้ำเงิน และดำ:

การตั้งค่าสีเส้นขอบของย่อหน้าเฉพาะ:

<p style="border-color: #FF0000 #00FF00 #0000FF #000000; border-style:solid">Some text ...</p>

ดู:

ข้อความบางส่วน ...

การตั้งค่าสีขอบของทุกย่อหน้า:

<style>
   p { border-color: #FF0000 #00FF00 #0000FF #000000; }
</style>

ดูสิ่งนี้ด้วย

  • แถบนำทาง CSS

คุณสมบัติของเครื่องมือรหัสสี CSS

เครื่องมือรหัสสี CSS ของเราอนุญาตให้ผู้ใช้รหัสสี CSSคุณสมบัติที่โดดเด่นบางประการของยูทิลิตี้นี้มีคำอธิบายด้านล่าง

ความเรียบง่าย

อินเทอร์เฟซที่ใช้งานง่ายของเครื่องมือรหัสสี CSS ของเราช่วยให้คุณดำเนินการแปลงนี้ได้อย่างง่ายดายในทันทีคุณไม่จำเป็นต้องได้รับทักษะการเขียนโปรแกรมสำหรับรหัสสี CSS อีกต่อไปเนื่องจากความเรียบง่ายของเครื่องมือนี้

ไม่มีการลงทะเบียน

คุณไม่จำเป็นต้องผ่านขั้นตอนการลงทะเบียนเพื่อใช้รหัสสี CSSเมื่อใช้ยูทิลิตีนี้ คุณสามารถ CSS color codes Tool ได้ฟรีหลายครั้งตามที่คุณต้องการ

พกพาสะดวก

รหัสสี CSS นี้สามารถเข้าถึงได้จากทุกมุมโลกคุณไม่จำเป็นต้องกังวลเกี่ยวกับความเข้ากันได้ของเครื่องมือออนไลน์นี้ เนื่องจากคุณสามารถเขียนโค้ดสี CSS กับเครื่องมือนี้จากอุปกรณ์ใดก็ได้เพียงคุณมีการเชื่อมต่ออินเทอร์เน็ตเพื่อเข้าถึงและใช้เครื่องมือรหัสสี CSS นี้

ประหยัดเวลาและความพยายาม

รหัสสี CSS ช่วยให้คุณทำงานเดียวกันให้เสร็จได้ทันทีคุณจะไม่ถูกขอให้ทำตามขั้นตอนด้วยตนเอง เนื่องจากอัลกอริทึมอัตโนมัติจะทำงานให้คุณ

ความเข้ากันได้

เครื่องมือรหัสสี CSS ทำงานได้อย่างสมบูรณ์แบบบนระบบปฏิบัติการทั้งหมดไม่ว่าคุณจะมีอุปกรณ์ Mac, iOS, Android, Windows หรือ Linux คุณก็สามารถใช้ยูทิลิตี้ออนไลน์นี้ได้อย่างง่ายดายโดยไม่ต้องเผชิญกับความยุ่งยากใดๆ

ฟรี 100%

คุณไม่จำเป็นต้องผ่านขั้นตอนการลงทะเบียนใดๆ เพื่อใช้เครื่องมือรหัสสี CSS นี้คุณสามารถใช้ยูทิลิตีนี้ได้ฟรีและทำโค้ดสี CSS ได้ไม่จำกัดโดยไม่มีข้อจำกัดใดๆ
