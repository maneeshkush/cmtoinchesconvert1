ขนาดหน้าต่างปัจจุบันของเบราว์เซอร์

ขนาดหน้าต่างของคุณคือ

ขนาดหน้าต่างด้านนอก:

ขนาดหน้าต่างด้านใน:

ซ้าย, พิกัดหน้าจอด้านบนของหน้าต่าง:

การตรวจจับความละเอียดหน้าจอด้วย Javascript

คุณสามารถรับขนาดหน้าต่างด้านนอกด้วยwindow.outerWidthและwindow.outerHeight

คุณสามารถรับขนาดหน้าต่างด้านในด้วยwindow.innerWidthและ window.innerHeight

ด้วย IE คุณจะ ได้รับdocument.body.clientWidthและ document.body.clientHeight

คุณสามารถรับตำแหน่งหน้าต่างหน้าจอด้วยwindow.screenLeftและ window.screenTop

ตัวอย่างเช่น:

<script>
alert(window.outerWidth+' x '+window.outerHeight);
</script>

คุณสมบัติของเครื่องมือขนาดหน้าต่าง

เครื่องมือขนาดหน้าต่างของเราช่วยให้ผู้ใช้สามารถปรับขนาดหน้าต่างได้คุณสมบัติที่โดดเด่นบางประการของยูทิลิตี้นี้มีคำอธิบายด้านล่าง

ความเรียบง่าย

ส่วนต่อประสานที่ใช้งานง่ายของเครื่องมือวัดขนาดหน้าต่างของเราช่วยให้คุณดำเนินการแปลงนี้ได้อย่างง่ายดายในทันทีคุณไม่จำเป็นต้องได้รับทักษะการเขียนโปรแกรมสำหรับขนาดหน้าต่างอีกต่อไปเนื่องจากความเรียบง่ายของเครื่องมือนี้

ไม่มีการลงทะเบียน

คุณไม่จำเป็นต้องผ่านขั้นตอนการลงทะเบียนเพื่อใช้ขนาดหน้าต่างเมื่อใช้ยูทิลิตีนี้ คุณสามารถใช้เครื่องมือปรับขนาดหน้าต่างได้หลายครั้งตามที่คุณต้องการได้ฟรี

พกพาสะดวก

ขนาดหน้าต่างนี้สามารถเข้าถึงได้จากทุกมุมโลกคุณไม่จำเป็นต้องกังวลเกี่ยวกับความเข้ากันได้ของเครื่องมือออนไลน์นี้ เนื่องจากคุณสามารถปรับขนาดหน้าต่างจากอุปกรณ์ใดก็ได้สิ่งที่คุณต้องมีคือการเชื่อมต่ออินเทอร์เน็ตเพื่อเข้าถึงและใช้เครื่องมือวัดขนาดหน้าต่างนี้

ประหยัดเวลาและความพยายาม

ขนาดหน้าต่างช่วยให้คุณทำงานเดียวกันให้เสร็จได้ทันทีคุณจะไม่ถูกขอให้ทำตามขั้นตอนด้วยตนเอง เนื่องจากอัลกอริทึมอัตโนมัติจะทำงานให้คุณ

ความเข้ากันได้

เครื่องมือปรับขนาดหน้าต่างทำงานได้อย่างสมบูรณ์แบบบนระบบปฏิบัติการทั้งหมดไม่ว่าคุณจะมีอุปกรณ์ Mac, iOS, Android, Windows หรือ Linux คุณก็สามารถใช้ยูทิลิตี้ออนไลน์นี้ได้อย่างง่ายดายโดยไม่ต้องเผชิญกับความยุ่งยากใดๆ

ฟรี 100%

คุณไม่จำเป็นต้องผ่านขั้นตอนการลงทะเบียนใดๆ เพื่อใช้เครื่องมือวัดขนาดหน้าต่างนี้คุณสามารถใช้ยูทิลิตีนี้ได้ฟรีและไม่จำกัดขนาดหน้าต่างโดยไม่มีข้อจำกัดใดๆ
