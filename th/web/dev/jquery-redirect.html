เปลี่ยนเส้นทาง jQuery

วิธีเปลี่ยนเส้นทางหน้าไปยัง URL ด้วย jQuery

เครื่องมือค้นหาใช้รหัสสถานะ 301 เพื่อโอนอันดับของหน้าจาก URL เก่าไปยัง URL ใหม่

การเปลี่ยนเส้นทาง jQuery ส่งคืนรหัสสถานะการตอบสนอง http: 200 ตกลง

ดังนั้นการเปลี่ยนเส้นทาง jQuery เช่นการเปลี่ยนเส้นทาง Javascript จึงไม่เป็นมิตรกับเครื่องมือค้นหา และควรใช้วิธีการเปลี่ยนเส้นทางอื่นๆ ที่ส่งคืนรหัสสถานะ 301 ย้ายถาวร

การเปลี่ยนเส้นทาง jQuery เป็นการเปลี่ยนเส้นทางJavascriptจริงๆการเปลี่ยนเส้นทางด้วย jQuery นั้นถือเป็นการโอเวอร์คิลเพราะคุณสามารถเปลี่ยนเส้นทาง Javascript ได้อย่างแท้จริง โดยใช้โค้ดน้อยลง

เปลี่ยนเส้นทาง jQuery

แทนที่หน้าเก่าด้วยรหัสเปลี่ยนเส้นทางด้วย URL ของหน้าที่คุณต้องการเปลี่ยนเส้นทาง

old-page.html:

<!DOCTYPE html>
<html>
<body>
<script src="http://ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script>
<ประเภทสคริปต์ ="text/javascript">
   // การเปลี่ยนเส้นทาง URL ของ jQuery
   $(document).ready( function() {
      url = "http://www.mydomain.com/new-page.html";
      $( location ).attr( "href", url);
   });
</script>
</body>
</html>

ตัวอย่างการเปลี่ยนเส้นทาง jQuery

jquery-redirect-test.htm

<!DOCTYPE html>
<html>
<body>
<script src="//ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script>

      url = "https://www.cmtoinchesconvert.com/th/web/dev/jquery-redirect.htm";

   });

</html>
