ลิงก์ Canonical URL

แท็ก Canonical URLแท็กลิงก์ Canonical

เปลี่ยนเส้นทางแท็กลิงก์มาตรฐาน HTML

ลิงก์ตามรูปแบบบัญญัติไม่ได้เปลี่ยนเส้นทางไปยัง URL ที่ต้องการ แต่อาจเป็นทางเลือกแทนการเปลี่ยนเส้นทาง URL สำหรับเว็บไซต์ที่การเข้าชมส่วนใหญ่มาจากเครื่องมือค้นหา

แท็ก HTML สำหรับลิงก์ตามรูปแบบบัญญัติสามารถใช้ได้เมื่อมีหลายหน้าที่มีเนื้อหาคล้ายกัน และคุณต้องการบอกเครื่องมือค้นหาว่าคุณต้องการหน้าใดในผลการค้นหา

แท็กลิงก์แบบบัญญัติสามารถเชื่อมโยงไปยังโดเมนเดียวกันและไปยังหลายโดเมนได้

เพิ่มแท็กลิงก์ตามรูปแบบบัญญัติในหน้าเก่าเพื่อลิงก์ไปยัง
หน้าใหม่

เพิ่มแท็กลิงก์ตามรูปแบบบัญญัติไปยังหน้าที่คุณไม่ต้องการ  เชื่อมโยง การเข้าชม ของเครื่องมือ  ค้นหา  ไปยังหน้าที่ต้องการ

ควรเพิ่มแท็กลิงก์มาตรฐานในส่วน

ตัวอย่าง #1

เมื่อย้ายหน้าแล้ว เราสามารถเพิ่มลิงก์บัญญัติในหน้าเก่าเพื่อแจ้งให้เครื่องมือค้นหาเข้าถึงหน้าใหม่

old-page1.html:

<!DOCTYPE html>
<html>
<head>
   ...
   <link rel="canonical" href="http://www.mydomain.com/new-page1.html">
</head>
<body>
   ...
</body>
</html>

ตัวอย่าง #2

เมื่อมีหลายหน้าที่มีเนื้อหาคล้ายคลึงกัน หากเราต้องการให้เครื่องมือค้นหาเข้าถึง page3.html เราควรเพิ่มลิงก์ตามรูปแบบบัญญัติไปยัง page3.html ในส่วนหัวของ page1.html และ page2.html

page1.html:

<!DOCTYPE html>
<html>
<head>
   ...
   <link rel="canonical" href="http://www.mydomain.com/page3.html">
</head>
<body>
   ...
</body>
</html>

page2.html:

<!DOCTYPE html>
<html>
<head>
   ...
   <link rel="canonical" href="http://www.mydomain.com/page3.html">
</head>
<body>
   ...
</body>
</html>
