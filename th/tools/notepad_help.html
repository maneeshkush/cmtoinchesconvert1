วิธีใช้แผ่นจดบันทึก

มี 2 ​​วิธีในการบันทึกข้อความ

  1. ข้อความจะถูกบันทึกทุกครั้งที่คุณปิดแท็บ Notepad ในแคชภายในเครื่องของเบราว์เซอร์เมื่อคุณเข้าสู่หน้า Notepad อีกครั้ง ข้อความจะปรากฏขึ้นอีกครั้ง
  2. ข้อความจะถูกบันทึก/สำรองข้อมูลไปยังฮาร์ดไดรฟ์เมื่อคุณกดปุ่มบันทึกหากต้องการเปิดข้อความจากฮาร์ดไดรฟ์อีกครั้ง ให้กดปุ่ม Open แล้วเลือกไฟล์ข้อความที่คุณสร้างขึ้น

พารามิเตอร์การกำหนดค่า


ข้อมูลสำคัญ

  • หากข้อความของเซสชันก่อนหน้าหายไป :
    • หากมีข้อความอยู่ ให้เลือกและคัดลอกด้วย Ctrl+C แล้ววางด้วย Ctrl+V ใน หน้า Notepad :

    • คัดลอกข้อความจากไฟล์ข้อความที่อัปเดตซึ่งสร้างโดยการดำเนินการบันทึกอัตโนมัติไปยังโฟลเดอร์ดาวน์โหลด (ถ้ามี)
    • ตรวจสอบว่าคุณกำลังมองหาคอมพิวเตอร์เครื่องเดิมที่คุณใช้ก่อนหน้านี้หรือไม่
    • เปิดใช้งานคุกกี้และประวัติของเบราว์เซอร์
    • อย่าใช้โหมดส่วนตัว/ไม่ระบุตัวตนของเบราว์เซอร์เบราว์เซอร์จะลบที่จัดเก็บข้อความในเครื่องเมื่อคุณปิดหน้าต่างเบราว์เซอร์
    • ลองเพิ่ม/ลบ www ออกจาก URL ในแถบที่อยู่ของเบราว์เซอร์
  • ข้อความของ Notepad จะไม่ถูกบันทึกด้วยการเรียกดูในโหมดไม่ระบุตัวตน/ส่วนตัว !!!
  • ข้อความของ Notpad ที่บันทึกไว้อาจถูกลบเมื่อคุณลบประวัติการท่องเว็บ/แคชหรือเรียกใช้แอปพลิเคชันล้างดิสก์ (เช่น Windows Disk cleanup / CCleaner) !!!
  • หาก ปุ่ม เปิด ไฟล์ ไม่ทำงาน โปรดโหลดหน้านี้ซ้ำแล้วลองอีกครั้ง
  • ใช้Notepadกับเบราว์เซอร์เวอร์ชันที่อัปเดตหากคุณใช้Internet Explorerให้คัดลอกข้อความของคุณไปยังเบราว์เซอร์สมัยใหม่ (เช่นChrome/Edge/Firefox )
  • ข้อความของแผ่นจดบันทึกจะถูกบันทึกโดยอัตโนมัติไปยังแคชในเครื่องของเบราว์เซอร์ (ไม่ปลอดภัย)
  • ข้อความของแผ่นจดบันทึกสามารถบันทึกอัตโนมัติ (สำรองข้อมูล) ลงในฮาร์ดไดรฟ์ ตามระยะเวลาบันทึกอัตโนมัติในเมนูView > Preferences
  • คุณสามารถสำรองข้อความของ Notepad ไปยังฮาร์ดไดรฟ์ได้โดยใช้ปุ่มบันทึกหรือเมนู ไฟล์ > บันทึก
  • สำหรับ Mac ให้ใช้⌘ Commandแทนปุ่มCtrl
  • หากต้องการเปิดไฟล์ที่บันทึกไว้ ให้ค้นหาไฟล์ในโฟลเดอร์ Downloads
  • หากเส้นพื้นหลังไม่เลื่อน ให้ซ่อนเส้น: ยกเลิกการเลือกเมนูView > Preferences > Text lines
  • ปุ่มบันทึกหรือเมนูไฟล์ > บันทึกบันทึกไฟล์ไปยังโฟลเดอร์ดาวน์โหลด ดู: ไฟล์จะไปที่ไหนเมื่อดาวน์โหลด
  • หาก มองไม่เห็น บรรทัดข้อความโปรดลองใช้เบราว์เซอร์ Chrome
  • หากการตรวจสอบตัวสะกดไม่ทำงาน ให้ลองเปิดใช้งานในส่วนการตั้งค่า>ภาษาของเบราว์เซอร์หากไม่ได้กำหนด คุณยังสามารถลองตั้งค่าภาษาอังกฤษ (สหรัฐอเมริกา)ในการตั้งค่าภาษาของเบราว์เซอร์ของคุณ

ตารางปุ่มลัด

การดำเนินการ ปุ่มลัด คำอธิบาย
ใหม่   ล้างพื้นที่ข้อความ
เปิด Ctrl + O เปิดไฟล์ข้อความจากฮาร์ดดิสก์
บันทึก Ctrl + S บันทึกข้อความไปยังไฟล์ปัจจุบันในฮาร์ดดิสก์
บันทึกเป็น...   บันทึกข้อความเป็นไฟล์ใหม่ในฮาร์ดดิสก์
พิมพ์ Ctrl + P พิมพ์ข้อความ
ตัด Ctrl + X คัดลอกและลบข้อความที่เลือก
สำเนา Ctrl + C คัดลอกข้อความที่เลือก
แปะ Ctrl + V วางข้อความที่ถูกตัดหรือคัดลอก
ลบ ลบ ลบข้อความที่เลือก
เลือกทั้งหมด Ctrl + A เลือกข้อความทั้งหมด
เลิกทำ Ctrl + Z เลิกทำการเปลี่ยนแปลงแก้ไขล่าสุด
ทำซ้ำ Ctrl + Y ทำการแก้ไขการเปลี่ยนแปลงอีกครั้ง
ซูมออก   ลดขนาดตัวอักษร
ขยายเข้า   เพิ่มขนาดตัวอักษร
ช่วย   แสดงหน้านี้
