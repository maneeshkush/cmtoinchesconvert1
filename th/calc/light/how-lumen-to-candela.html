วิธีแปลงลูเมนเป็นแคนเดลา

วิธีแปลงฟลักซ์การส่องสว่างในหน่วยลูเมน (lm) เป็นความเข้มของการส่องสว่างในหน่วยแคนเดลา (cd)

คุณสามารถคำนวณแต่ไม่สามารถแปลงลูเมนเป็นแคนเดลาได้ เนื่องจากแคนเดลาและลูเมนไม่ได้แสดงปริมาณที่เท่ากัน

การคำนวณลูเมนถึงแคนเดลา

สำหรับแหล่งกำเนิดแสงแบบไอโซโทรปิกที่สม่ำเสมอ ความเข้มของการส่องสว่าง  I v  ในแคนเดลา (cd) จะเท่ากับฟลักซ์การส่องสว่าง Φ v ในหน่วยลูเมน (lm)

หารด้วยมุมทึบ  Ω  ในหน่วยสเตอเรเดียน (sr):

Iv(cd) = Φv(lm) / Ω(sr)

 

ดังนั้น มุมทึบΩ  ในสเตอเรเดียน (sr) เท่ากับ 2 คูณ pi คูณ 1 ลบโคไซน์ของครึ่งหนึ่งของมุมยอด  θในหน่วยองศา (°)

Ω(sr) = 2π(1 - cos(θ/2))

 

ดังนั้น ความเข้มของการส่องสว่างI v  ในแคนเดลา (cd) จึงเท่ากับฟลักซ์การส่องสว่าง Φ v ในหน่วยลูเมน (lm)

หารด้วย 2 คูณ pi คูณ 1 ลบโคไซน์ของครึ่งมุมยอด  θ หน่วย เป็นองศา (°)

Iv(cd) = Φv(lm) / ( 2π(1 - cos(θ/2)) )

ดังนั้น

candela = lumens / ( 2π(1 - cos(degrees/2)) )

หรือ

cd = lm / ( 2π(1 - cos(°/2)) )

ตัวอย่างที่ 1

ค้นหาความเข้มของการส่องสว่าง  I v  ในแคนเดลา (cd) เมื่อฟลักซ์การส่องสว่าง Φ v ในลูเมน (lm) คือ 340lm และมุมยอดคือ 60°:

Iv(cd) = 340 lm / ( 2π(1 - cos(60°/2)) ) = 403.9 cd

Iv(cd) = 360 lm / ( 2π(1 - cos(60°/2)) ) = 427.6 cd
