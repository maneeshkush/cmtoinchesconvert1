การคำนวณ DC / เฟสเดียว

แรงดันตก V ในหน่วยโวลต์ (V) เท่ากับกระแสลวด I ในหน่วยแอมป์ (A) คูณ 2 คูณความยาวสายทางเดียว L ในหน่วยฟุต (ฟุต) คูณความต้านทานสายต่อ 1,000 ฟุต R ในหน่วยโอห์ม (Ω/kft) หารด้วย ภายใน 1,000:

Vdrop (V) = Iwire (A) × Rwire(Ω)

= I สาย (A) × (2 × L (ฟุต) × R สาย (Ω/kft) / 1,000 (ft/kft) )

แรงดันตก V ในหน่วยโวลต์ (V) เท่ากับกระแสลวด I ในหน่วยแอมป์ (A) คูณ 2 คูณความยาวสายทางเดียว L ในหน่วยเมตร (ม.) คูณความต้านทานสายต่อ 1,000 เมตร R ในหน่วยโอห์ม (Ω/กม.) หารด้วย ภายใน 1,000:

Vdrop (V) = Iwire (A) × Rwire(Ω)

= I สาย (A) × (2 × L (ม.) × R สาย (Ω/กม.) / 1,000 (ม./กม.) )

การคำนวณ 3 เฟส

แรงดันไลน์ต่อไลน์ตก V ในหน่วยโวลต์ (V) เท่ากับสแควร์รูทของ 3 เท่าของกระแสสาย I ในหน่วยแอมป์ (A) คูณความยาวสายทางเดียว L ในหน่วยฟุต (ฟุต) คูณความต้านทานสายต่อ 1,000 ฟุต R ในหน่วยโอห์ม (Ω/kft) หารด้วย 1,000:

Vdrop (V) = √3 × Iwire (A) × Rwire (Ω)

= 1.732 × I สาย (A) × ( L (ฟุต) × R สาย (Ω/kft) / 1,000 (ft/kft) )

แรงดันไลน์ต่อไลน์ตก V ในหน่วยโวลต์ (V) เท่ากับสแควร์รูทของ 3 เท่าของกระแสสาย I ในหน่วยแอมป์ (A) คูณความยาวสายทางเดียว L ในหน่วยเมตร (ม.) คูณความต้านทานสายต่อ 1,000 เมตร R ในหน่วยโอห์ม (Ω/กม.) หารด้วย 1,000:

Vdrop (V) = √3 × Iwire (A) × Rwire (Ω)

= 1.732 × สาย I (A) × ( L (ม.) × สาย R (Ω/กม.) / 1,000 (ม./กม.) )

การคำนวณเส้นผ่านศูนย์กลางของลวด

ดังนั้น เส้นผ่านศูนย์กลางเส้นลวด n เกจ dn ในหน่วยนิ้ว (นิ้ว) เท่ากับ 0.005ในคูณ 92 ยกกำลัง 36 ลบเลขเกจ n หารด้วย 39

dn (in) = 0.005 in × 92(36-n)/39

ดังนั้น เส้นผ่านศูนย์กลางลวด n เกจ dn ในหน่วยมิลลิเมตร (มม.) เท่ากับ 0.127 มม. คูณ 92 ยกกำลัง 36 ลบเลขเกจ n หารด้วย 39

dn (mm) = 0.127 mm × 92(36-n)/39

การคำนวณพื้นที่หน้าตัดลวด

พื้นที่ตัดขวางของเส้นลวด n เกจ A nในหน่วยกิโล-วงกลม mils (kcmil) เท่ากับ 1,000 เท่าของเส้นผ่านศูนย์กลางลวดสี่เหลี่ยม d ในหน่วยนิ้ว (นิ้ว):

An (kcmil) = 1000×dn2 = 0.025 in2 × 92(36-n)/19.5

ดังนั้น พื้นที่ตัดขวางของเส้นลวด n เกจ A nในตารางนิ้ว (ใน2 ) เท่ากับ pi หารด้วย 4 คูณของเส้นผ่านศูนย์กลางลวดสี่เหลี่ยม d ในหน่วยนิ้ว (นิ้ว)

An (in2) = (π/4)×dn2 = 0.000019635 in2 × 92(36-n)/19.5

ดังนั้น พื้นที่ตัดขวางของลวด n เกจ A n ในตารางมิลลิเมตร (มม. 2 ) เท่ากับ pi หารด้วย 4 คูณของเส้นผ่านศูนย์กลางลวดสี่เหลี่ยม d ในหน่วยมิลลิเมตร (มม.)

An (mm2) = (π/4)×dn2 = 0.012668 mm2 × 92(36-n)/19.5

การคำนวณความต้านทานของสายไฟ

ดังนั้น ค่าความต้านทานของเส้นลวด n เกจ R ในหน่วยโอห์มต่อกิโลฟุต (Ω/kft) เท่ากับ 0.3048×1000000000 คูณความต้านทานของเส้นลวด ρ ในหน่วยโอห์มเมตร(Ω·m) หารด้วย 25.4 2เท่าของพื้นที่หน้าตัด A nในหน่วยตารางนิ้ว (ใน2 ):

Rn (Ω/kft) = 0.3048 × 109 × ρ(Ω·m) / (25.42 × An (in2))

ดังนั้น ค่าความต้านทานของเส้นลวด n เกจ R ในหน่วยโอห์มต่อกิโลเมตร (Ω/km) เท่ากับ 1000000000 เท่าของความต้านทานของเส้นลวด ρ ในหน่วยโอห์มเมตร(Ω·m) หารด้วยพื้นที่หน้าตัด A nในหน่วยตารางมิลลิเมตร (มม. 2 )

Rn (Ω/km) = 109 × ρ(Ω·m) / An (mm2)

คุณสมบัติของเครื่องคิดเลขแรงดันตก

เครื่องคำนวณแรงดันตกของเราช่วยให้ผู้ใช้สามารถคำนวณแรงดันตกได้คุณสมบัติที่โดดเด่นบางประการของยูทิลิตี้นี้มีคำอธิบายด้านล่าง

ไม่มีการลงทะเบียน

คุณไม่จำเป็นต้องผ่านขั้นตอนการลงทะเบียนใดๆ เพื่อใช้เครื่องคำนวณแรงดันไฟตกเมื่อใช้ยูทิลิตีนี้ ผู้ใช้จะคำนวณแรงดันตกคร่อมได้มากเท่าที่คุณต้องการฟรี

การแปลงอย่างรวดเร็ว

เครื่องคำนวณแรงดันไฟฟ้าตกนี้ให้ผู้ใช้คำนวณได้เร็วที่สุดเมื่อผู้ใช้ป้อนค่า Voltage Drop ในช่องอินพุตแล้วคลิกปุ่ม Calculate ยูทิลิตีจะเริ่มกระบวนการแปลงและส่งคืนผลลัพธ์ทันที

ประหยัดเวลาและความพยายาม

ขั้นตอนด้วยตนเองของ Calculator Voltage Drop ไม่ใช่เรื่องง่ายคุณต้องใช้เวลาและความพยายามอย่างมากในการทำงานนี้ให้สำเร็จเครื่องคำนวณแรงดันตกช่วยให้คุณทำงานเดียวกันให้เสร็จได้ทันทีคุณจะไม่ถูกขอให้ทำตามขั้นตอนด้วยตนเอง เนื่องจากอัลกอริทึมอัตโนมัติจะทำงานให้คุณ

ความแม่นยำ

แม้จะใช้เวลาและความพยายามในการคำนวณด้วยตนเอง คุณอาจไม่สามารถได้ผลลัพธ์ที่ถูกต้องไม่ใช่ทุกคนที่แก้ปัญหาคณิตศาสตร์เก่ง แม้ว่าคุณจะคิดว่าคุณเป็นมือโปร แต่ก็ยังมีโอกาสดีที่คุณจะได้ผลลัพธ์ที่ถูกต้องสถานการณ์นี้สามารถจัดการได้อย่างชาญฉลาดด้วยความช่วยเหลือของเครื่องคำนวณแรงดันไฟตกคุณจะได้รับผลลัพธ์ที่แม่นยำ 100% จากเครื่องมือออนไลน์นี้

ความเข้ากันได้

ตัวแปลง Voltage Drop แบบออนไลน์ทำงานได้อย่างสมบูรณ์แบบบนระบบปฏิบัติการทั้งหมดไม่ว่าคุณจะมีอุปกรณ์ Mac, iOS, Android, Windows หรือ Linux คุณก็สามารถใช้เครื่องมือออนไลน์นี้ได้อย่างง่ายดายโดยไม่ต้องเผชิญกับความยุ่งยากใดๆ

ฟรี 100%

คุณไม่จำเป็นต้องผ่านขั้นตอนการลงทะเบียนใดๆ เพื่อใช้เครื่องคำนวณแรงดันไฟตกนี้คุณสามารถใช้ยูทิลิตีนี้ได้ฟรีและคำนวณแรงดันตกไม่จำกัดโดยไม่มีข้อจำกัดใดๆ
