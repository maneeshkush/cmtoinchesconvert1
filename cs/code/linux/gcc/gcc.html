Kompilátor GCC C

GCC je zkratka GNU Compiler Collection, kompilátoru C pro Linux.

  • možnosti gcc

Syntaxe GCC

$ gcc [options] [source files] [object files] [-o output file]

možnosti GCC

Hlavní možnosti GCC:

volba popis
gcc -c kompilovat zdrojové soubory do objektových souborů bez propojení
gcc -D název [ =value ] definovat makro preprocesoru
gcc -fPIC generovat pozičně nezávislý kód pro sdílené knihovny
úroveň gcc -g generovat informace o ladění, které má GDB použít
gcc -I dir add include adresář hlavičkových souborů
gcc -l lib odkaz na soubor knihovny
gcc -L dir vyhledejte v adresáři soubory knihovny
výstupní soubor gcc -o zápis sestavení výstupu do výstupního souboru
úroveň gcc -O optimalizovat velikost kódu a dobu provádění
gcc -sdíleno generovat sdílený objektový soubor pro sdílenou knihovnu
gcc -U jméno zrušte definici makra preprocesoru
gcc -w zakázat všechny varovné zprávy
gcc -Zeď povolit všechny varovné zprávy
gcc -Wextra povolit další varovné zprávy

Příklady GCC

Zkompilujte soubor1.c a soubor2.c a odkažte na výstupní soubor execfile :

$ gcc file1.c file2.c -o execfile

 

Spusťte execfile výstupního souboru :

$ ./execfile

 

Zkompilujte soubor1.c a soubor2.c bez propojení:

$ gcc -c file1.c file2.c

 

Kompilace myfile.c s informacemi o ladění a odkazem na výstupní soubor execfile :

$ gcc -g myfile.c -o execfile

 

Kompilace myfile.c s povolenými varovnými zprávami a odkazem na výstupní soubor execfile :

$ gcc -Wall myfile.c -o execfile

 

Zkompilujte myfile.c se statickou knihovnou libmath.a umístěnou v /user/local/math a propojte se s ní , abyste získali výstupní soubor execfile :

$ gcc -static myfile.c -L/user/local/math -lmath -o execfile

 

Kompilace myfile.c s optimalizací a odkazem na výstupní soubor execfile :

$ gcc -O myfile.c -o execfile
