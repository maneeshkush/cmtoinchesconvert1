CSS barva

Kód barvy může být jeden z:

Hexadecimální formát: #rrggbb

Formát RGB: rgb (červená, zelená, modrá)

Formát jména: jméno

Příklad

Pro oranžovou barvu:

Hexadecimální formát: #FFA500

Formát RGB: rgb (255,165,0)

Formát jména: oranžový

Nastavení barvy prvku

Pro konkrétní prvek:

<element style="color: code;"></element>

Pro všechny prvky stejného typu. Vložte kód do tagu <style> v sekci head:

<style>
   element { color: code; }
</style>

Příklad

Nastavení barvy textu odstavce na červenou:

Nastavení barvy konkrétního odstavce:

<p style="color: #FF0000;">Some text ...</p>

Pohled:

Nějaký text...

Nastavení barvy všech odstavců

<style>
   p { color: #FF0000; }
</style>

Nastavení barvy pozadí prvku

element { background: code; }

Příklad

Nastavení barvy pozadí odstavce na červenou:

Nastavení barvy pozadí konkrétního odstavce:

<p style="background: #FF0000;">Some text ...</p>

Pohled:

Nějaký text...

Nastavení barvy pozadí všech odstavců:

<style>
   p { background: #FF0000; }
</style>

Nastavení barvy ohraničení prvku

element { border-color: topcode rightcode bottomcode leftcode }

Příklad

Nastavení barvy okraje odstavce na červenou, zelenou, modrou a černou:

Nastavení barvy ohraničení konkrétního odstavce:

<p style="border-color: #FF0000 #00FF00 #0000FF #000000; border-style:solid">Some text ...</p>

Pohled:

Nějaký text...

Nastavení barvy ohraničení všech odstavců:

<style>
   p { border-color: #FF0000 #00FF00 #0000FF #000000; }
</style>
