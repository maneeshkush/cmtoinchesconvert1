Jaké barvy dělají zlato

Zlatá RGB barva je generována přidáním červené a zelené barvy.

Zlatý RGB barevný kód

Kód zlaté barvy RGB = #FFD700 = 255*65536+215*256+0 = (255, 215, 0)

ČERVENÁ=255, ZELENÁ=215, MODRÁ=0
