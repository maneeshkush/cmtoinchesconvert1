Web Bezpečné barvy graf

Webové bezpečné barvy jsou tvořeny kombinací červené, zelené a modré (RGB) hexadecimálních hodnot 0,33H,66H,99H,CCH,FFH.

Výsledkem je 6*6*6=216 barev.

Dříve prohlížeče nepodporovaly zobrazení více než 256 palety barev současně. S novými webovými prohlížeči není nutné používat pouze tyto barvy.
