Symbol nekonečna

Symbol nekonečna je matematický symbol, který představuje nekonečně velké číslo.

Symbol nekonečna je napsán symbolem Lemniscate:

∞

Představuje nekonečně kladné velké číslo.

Když chceme napsat nekonečně záporné číslo, měli bychom napsat:

-∞

Když chceme napsat nekonečně malé číslo, měli bychom napsat:

1/∞

Je nekonečno reálné číslo?

Nekonečno není číslo. Nepředstavuje konkrétní číslo, ale nekonečně velké množství.

název Typ klíče

Součet reálných čísel x + ∞ = ∞, pro x ∈ℝ
Kladné číslo produktu x ⋅ ∞ = ∞, pro x >0

Nekonečno v teorii množin

Aleph-null ( ) je nekonečný počet prvků (kardinalita) množiny přirozených čísel ( ).

Aleph-one ( ) je nekonečný počet prvků (kardinalita) spočetné množiny ordinálních čísel (ω 1 ).
