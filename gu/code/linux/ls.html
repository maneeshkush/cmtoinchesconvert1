Linux/Unix માં ls આદેશ

ls એ Linux શેલ કમાન્ડ છે જે ફાઇલો અને ડિરેક્ટરીઓની ડિરેક્ટરી સામગ્રીઓની યાદી આપે છે.

  • ls વાક્યરચના
  • ls વિકલ્પો
  • ls ઉદાહરણો
  • ls કોડ જનરેટર

ls વાક્યરચના

$ ls [options] [file|dir]

ls આદેશ વિકલ્પો

ls આદેશ મુખ્ય વિકલ્પો:

વિકલ્પ વર્ણન
ls -a '.' થી શરૂ થતી છુપી ફાઇલ સહિત તમામ ફાઇલોની યાદી બનાવો.
ls --રંગ રંગીન સૂચિ [=હંમેશા/ક્યારેય નહીં/ઓટો]
ls -d સૂચિ ડિરેક્ટરીઓ - '*/' સાથે
ls -F */=>@| નો એક અક્ષર ઉમેરો પ્રવેશો માટે
ls -i લિસ્ટ ફાઇલનો inode ઇન્ડેક્સ નંબર
ls -l લાંબા ફોર્મેટ સાથે સૂચિ - પરવાનગીઓ બતાવો
ls -la છુપાયેલ ફાઇલો સહિત લાંબા ફોર્મેટની સૂચિ
ls -lh વાંચી શકાય તેવી ફાઇલ કદ સાથે સૂચિ લાંબી ફોર્મેટ
ls -ls ફાઈલ માપ સાથે લાંબા ફોર્મેટ સાથે યાદી
ls -r વિપરીત ક્રમમાં યાદી
ls -આર પુનરાવર્તિત ડિરેક્ટરી વૃક્ષની યાદી
ls -s સૂચિ ફાઇલ કદ
ls -S ફાઇલ કદ દ્વારા સૉર્ટ કરો
ls -t સમય અને તારીખ દ્વારા સૉર્ટ કરો
ls -X એક્સ્ટેંશન નામ દ્વારા સૉર્ટ કરો

ls આદેશ ઉદાહરણો

તમે ફાઇલ અથવા ફોલ્ડરના નામોને સ્વતઃ પૂર્ણ કરવા માટે ટેબ બટન દબાવી શકો છો.

સંબંધિત પાથ સાથે દસ્તાવેજો/પુસ્તકોની સૂચિ બનાવો:

$ ls Documents/Books

 

સંપૂર્ણ પાથ સાથે ડિરેક્ટરી /home/user/Documents/Book ને સૂચિબદ્ધ કરો.

$ ls /home/user/Documents/Books

 

રુટ ડિરેક્ટરીની સૂચિ કરો:

$ ls /

 

પિતૃ નિર્દેશિકાની સૂચિ બનાવો:

$ ls ..

 

વપરાશકર્તાની હોમ ડિરેક્ટરીની સૂચિ બનાવો (દા.ત.: /home/user):

$ ls ~

 

લાંબા ફોર્મેટ સાથે સૂચિ:

$ ls -l

 

છુપાયેલ ફાઇલો બતાવો:

$ ls -a

 

લાંબા ફોર્મેટ સાથે સૂચિ બનાવો અને છુપાયેલી ફાઇલો બતાવો:

$ ls -la

 

તારીખ/સમય દ્વારા સૉર્ટ કરો:

$ ls -t

 

ફાઇલ કદ દ્વારા સૉર્ટ કરો:

$ ls -S

 

બધી સબડાયરેક્ટરીઝની સૂચિ બનાવો:

$ ls *

 

પુનરાવર્તિત ડિરેક્ટરી વૃક્ષ સૂચિ:

$ ls -R

 

વાઇલ્ડકાર્ડ સાથે ફક્ત ટેક્સ્ટ ફાઇલોની સૂચિ બનાવો:

$ ls *.txt

 

ls આઉટપુટ ફાઈલ પર રીડાયરેકશન:

$ ls > out.txt

 

ફક્ત ડિરેક્ટરીઓની સૂચિ બનાવો:

$ ls -d */

 

સંપૂર્ણ પાથ સાથે ફાઇલો અને ડિરેક્ટરીઓની સૂચિ બનાવો:

$ ls -d $PWD/*

ls કોડ જનરેટર

ls વિકલ્પો પસંદ કરો અને જનરેટ કોડ બટન દબાવો:

વિકલ્પો 
  લાંબી સૂચિ ફોર્મેટ (-l)
  બધી ફાઇલો / છુપાયેલી ફાઇલોની સૂચિ બનાવો (-a)
  વારંવાર સૂચિબદ્ધ ડિરેક્ટરી વૃક્ષ (-R)
  વિપરીત ક્રમમાં સૂચિ (-r)
  સંપૂર્ણ પાથ સાથેની સૂચિ (-d $PWD/*)
દ્વારા સૉર્ટ કરો:
ફાઇલો / ફોલ્ડર્સ
ફાઈલો:
ફોલ્ડર્સ:
આઉટપુટ રીડાયરેક્શન
કોડ પસંદ કરવા માટે ટેક્સ્ટબોક્સ પર ક્લિક કરો, પછી તેને ટર્મિનલમાં કૉપિ કરીને પેસ્ટ કરો
