Linux/Unix માં ફાઇલોને કેવી રીતે ખસેડવી

યુનિક્સ/લિનક્સ ફાઇલો ખસેડો.

 

ફાઇલ સિન્ટેક્સ ખસેડો:

$ mv [options] sourcefiles destdir

 

main.c def.h ફાઇલોને /home/usr/rapid/ ડિરેક્ટરીમાં ખસેડો

$ mv main.c def.h /home/usr/rapid/

 

વર્તમાન ડિરેક્ટરીમાંની બધી C ફાઇલોને સબડિરેક્ટરી બકમાં ખસેડો

$ mv *.c bak
