cp: ડિરેક્ટરી અવગણવું

cp ડિરેક્ટરી Linux error mesge ને છોડી દે છે.

સમસ્યા

ડિરેક્ટરીની નકલ કરવાનો પ્રયાસ કરતી વખતે, તમને ભૂલ સંદેશ મળે છે:

$ cp srcdir destdir
cp: omitting directory 'srcdir'
$

ઉકેલ

પુનરાવર્તિત નકલ માટે ઉમેરો -R વિકલ્પ:

$ cp -R srcdir destdir

 

આ પણ જુઓ
