GCC C કમ્પાઇલર

GCC એ GNU કમ્પાઇલર કલેક્શનનો ટૂંકો છે, જે Linux માટે C કમ્પાઇલર છે.

  • gcc વાક્યરચના
  • જીસીસી વિકલ્પો
  • જીસીસી ઉદાહરણો
  • જીસીસી કોડ જનરેટર

GCC વાક્યરચના

$ gcc [options] [source files] [object files] [-o output file]

GCC વિકલ્પો

GCC મુખ્ય વિકલ્પો:

વિકલ્પ વર્ણન
gcc -c લિંક કર્યા વિના ઑબ્જેક્ટ ફાઇલોમાં સ્રોત ફાઇલોને કમ્પાઇલ કરો
gcc -D નામ [ =મૂલ્ય ] પ્રીપ્રોસેસર મેક્રો વ્યાખ્યાયિત કરો
gcc -fPIC વહેંચાયેલ પુસ્તકાલયો માટે સ્થિતિ સ્વતંત્ર કોડ જનરેટ કરો
gcc -g સ્તર GDB દ્વારા ઉપયોગમાં લેવા માટે ડીબગ માહિતી બનાવો
gcc -I dir હેડર ફાઇલોની ડિરેક્ટરી શામેલ કરો
gcc -l lib લાઇબ્રેરી ફાઇલ સાથે લિંક
gcc -L dir લાઇબ્રેરી ફાઇલો માટે ડિરેક્ટરીમાં જુઓ
gcc -o આઉટપુટ ફાઇલ આઉટપુટ ફાઇલમાં બિલ્ડ આઉટપુટ લખો
gcc -O સ્તર કોડ કદ અને અમલના સમય માટે ઑપ્ટિમાઇઝ કરો
gcc - વહેંચાયેલ શેર્ડ લાઇબ્રેરી માટે શેર્ડ ઓબ્જેક્ટ ફાઇલ જનરેટ કરો
gcc -U નામ પ્રીપ્રોસેસર મેક્રોને અવ્યાખ્યાયિત કરો
gcc -w બધા ચેતવણી સંદેશાઓ અક્ષમ કરો
gcc - દિવાલ બધા ચેતવણી સંદેશાઓને સક્ષમ કરો
જીસીસી -વેક્સ્ટ્રા વધારાના ચેતવણી સંદેશાઓને સક્ષમ કરો

GCC ઉદાહરણો

file1.c અને file2.c કમ્પાઇલ કરો અને આઉટપુટ ફાઇલ execfile સાથે લિંક કરો :

$ gcc file1.c file2.c -o execfile

 

આઉટપુટ ફાઈલ execfile ચલાવો :

$ ./execfile

 

લિંક કર્યા વિના file1.c અને file2.c કમ્પાઇલ કરો :

$ gcc -c file1.c file2.c

 

ડીબગ માહિતી સાથે myfile.c કમ્પાઇલ કરો અને આઉટપુટ ફાઇલ execfile સાથે લિંક કરો :

$ gcc -g myfile.c -o execfile

 

ચેતવણી સંદેશા સક્ષમ સાથે myfile.c કમ્પાઇલ કરો અને આઉટપુટ ફાઇલ execfile સાથે લિંક કરો :

$ gcc -Wall myfile.c -o execfile

 

આઉટપુટ ફાઇલ execfile માટે /user/local/math માં સ્થિત સ્ટેટિક લાઇબ્રેરી libmath.a સાથે myfile.c કમ્પાઇલ કરો અને લિંક કરો :

$ gcc -static myfile.c -L/user/local/math -lmath -o execfile

 

myfile.c ને ઓપ્ટિમાઇઝેશન સાથે કમ્પાઇલ કરો અને આઉટપુટ ફાઇલ execfile સાથે લિંક કરો :

$ gcc -O myfile.c -o execfile
