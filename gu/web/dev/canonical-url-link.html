કેનોનિકલ URL લિંક

કેનોનિકલ URL ટૅગ. કેનોનિકલ લિંક ટેગ.

HTML કેનોનિકલ લિંક ટેગ રીડાયરેક્ટ

પ્રામાણિક લિંક પસંદગીના URL પર રીડાયરેક્ટ કરતી નથી, પરંતુ તે વેબસાઇટ્સ માટે URL રીડાયરેક્ટનો વિકલ્પ બની શકે છે જ્યાં મોટાભાગનો ટ્રાફિક સર્ચ એન્જિનમાંથી આવે છે.

કેનોનિકલ લિંક્સ માટેના HTML ટેગનો ઉપયોગ ત્યારે થઈ શકે છે જ્યારે સમાન સામગ્રીવાળા બહુવિધ પૃષ્ઠો હોય અને તમે શોધ એન્જિનને જણાવવા માંગતા હોવ કે તમે શોધ પરિણામોમાં કયું પૃષ્ઠ પસંદ કરો છો.

કેનોનિકલ લિંક ટેગ એક જ ડોમેન અને બહુવિધ ડોમેન્સ સાથે પણ લિંક કરી શકે છે .


નવા પૃષ્ઠ સાથે લિંક કરવા માટે જૂના પૃષ્ઠ પર પ્રમાણભૂત લિંક ટેગ ઉમેરો .

કેનોનિકલ લિંક ટૅગને તે પૃષ્ઠો પર ઉમેરો કે જે તમે   શોધ  એન્જિન  ટ્રાફિકને પસંદગીના પૃષ્ઠ પર લિંક કરવાનું પસંદ કરતા નથી.

વિભાગમાં પ્રમાણભૂત લિંક ટેગ ઉમેરવો જોઈએ.

ઉદાહરણ #1

જ્યારે પૃષ્ઠ ખસેડવામાં આવે છે ત્યારે અમે નવા પૃષ્ઠ સુધી પહોંચવા માટે શોધ એન્જિનને સૂચિત કરવા માટે જૂના પૃષ્ઠમાં પ્રમાણભૂત લિંક ઉમેરી શકીએ છીએ.

old-page1.html:

<!DOCTYPE html>
<html>
<head>
   ...
   <link rel="canonical" href="http://www.mydomain.com/new-page1.html">
</head>
<body>
   ...
</body>
</html>

ઉદાહરણ #2

જ્યારે સમાન કન્વેન્ટ સાથે ઘણા પૃષ્ઠો હોય, તો જો આપણે page3.html સુધી પહોંચવા માટે શોધ એન્જિનને પ્રાધાન્ય આપીએ, તો આપણે page1.html અને page2.html ના હેડ વિભાગમાં page3.html પર પ્રમાણભૂત લિંક ઉમેરવી જોઈએ.

page1.html:

<!DOCTYPE html>
<html>
<head>
   ...
   <link rel="canonical" href="http://www.mydomain.com/page3.html">
</head>
<body>
   ...
</body>
</html>

page2.html:

<!DOCTYPE html>
<html>
<head>
   ...
   <link rel="canonical" href="http://www.mydomain.com/page3.html">
</head>
<body>
   ...
</body>
</html>
