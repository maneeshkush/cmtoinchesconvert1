નવી વિંડો અથવા નવી ટેબમાં લિંક કેવી રીતે ખોલવી.

નવી વિન્ડો અથવા ટેબમાં લિંક ખોલો

નવી વિન્ડો/ટેબમાં લિંક ખોલવા માટે, <a> ટેગની અંદર target="_blank" ઉમેરો:

<a href="../html-link.htm" target="_blank">Open page in new window</a>

કોડ આ લિંક બનાવશે:

નવી વિંડોમાં પૃષ્ઠ ખોલો

નવી વિન્ડો અથવા નવી ટેબ

તમે સેટ કરી શકતા નથી કે લિંક નવી વિન્ડોમાં ખોલવામાં આવશે કે નવી ટેબમાં. તે બ્રાઉઝરની સેટિંગ્સ પર આધાર રાખે છે. 

ઉલ્લેખિત કદ સાથે નવી વિંડોમાં લિંક ખોલો

નવી વિંડોમાં લિંક ખોલવા માટે, <a> ટેગની અંદર Javascript આદેશ onclick="window.open('text-link.htm', 'name','width=600,height=400') ઉમેરો:

<a href="../html-link.htm" target="popup" onclick="window.open('../html-link.htm','name','width=600,height=400')">Open page in new window</a>
