રંગ કોડ આમાંથી એક હોઈ શકે છે:

હેક્સ ફોર્મેટ: #rrggbb

RGB ફોર્મેટ: rgb (લાલ, લીલો, વાદળી)

નામ ફોર્મેટ: નામ

ઉદાહરણ

નારંગી રંગ માટે:

હેક્સ ફોર્મેટ: #FFA500

RGB ફોર્મેટ: rgb(255,165,0)

નામ ફોર્મેટ: નારંગી

તત્વ રંગ સુયોજિત કરી રહ્યા છીએ

ચોક્કસ તત્વ માટે:

<element style="color: code;"></element>

સમાન પ્રકારના તમામ ઘટકો માટે. હેડ વિભાગમાં <style> ટેગમાં કોડ મૂકો:

<style>
   element { color: code; }
</style>

ઉદાહરણ

ચોક્કસ ફકરાનો રંગ સેટ કરી રહ્યો છે:

<p style="color: #FF0000;">Some text ...</p>

અમુક લખાણ...

આ પણ જુઓ

  • CSS નેવિગેશન બાર

CSS કલર કોડ ટૂલની વિશેષતાઓ

અમારું CSS કલર કોડ ટૂલ વપરાશકર્તાઓને CSS કલર કોડ્સ માટે પરવાનગી આપે છે. આ ઉપયોગિતાના કેટલાક મુખ્ય લક્ષણો નીચે વર્ણવેલ છે.

સરળતા

અમારા CSS કલર કોડ્સ ટૂલનું વપરાશકર્તા મૈત્રીપૂર્ણ ઇન્ટરફેસ તમને આ રૂપાંતરણને તરત જ કરવામાં સરળતા પ્રદાન કરે છે. આ ટૂલની સરળતાને કારણે તમારે હવે CSS કલર કોડ્સ માટે પ્રોગ્રામિંગ કૌશલ્ય પ્રાપ્ત કરવાની જરૂર નથી.

કોઈ નોંધણી નથી

CSS કલર કોડ્સનો ઉપયોગ કરવા માટે તમારે કોઈપણ નોંધણી પ્રક્રિયામાંથી પસાર થવાની જરૂર નથી. આ યુટિલિટીનો ઉપયોગ કરીને, તમે CSS કલર કોડ્સ ટૂલને તમે ઇચ્છો તેટલી વખત મફતમાં આપી શકો છો.

પોર્ટેબિલિટી

આ CSS કલર કોડ વિશ્વના કોઈપણ ખૂણેથી એક્સેસ કરી શકાય છે. તમારે આ ઓનલાઈન ટૂલની સુસંગતતા વિશે ચિંતા કરવાની જરૂર નથી, કારણ કે તમે કોઈપણ ઉપકરણથી તેની સાથે કલર કોડ્સ CSS કરી શકો છો. આ CSS કલર કોડ્સ ટૂલને ઍક્સેસ કરવા અને તેનો ઉપયોગ કરવા માટે તમારે ફક્ત ઇન્ટરનેટ કનેક્શનની જરૂર છે.

સમય અને પ્રયત્ન બચાવે છે

CSS કલર કોડ્સ તમને તે જ કાર્ય તરત જ પૂર્ણ કરવાની મંજૂરી આપે છે. તમને મેન્યુઅલ પ્રક્રિયાઓને અનુસરવાનું કહેવામાં આવશે નહીં, કારણ કે તેના સ્વચાલિત અલ્ગોરિધમ્સ તમારા માટે કાર્ય કરશે.

સુસંગતતા

CSS કલર કોડ્સ ટૂલ બધી ઓપરેટિંગ સિસ્ટમ્સ પર સંપૂર્ણ રીતે કામ કરે છે. ભલે તમારી પાસે Mac, iOS, Android, Windows, અથવા Linux ઉપકરણ હોય, તમે કોઈપણ મુશ્કેલીનો સામનો કર્યા વિના સરળતાથી આ ઑનલાઇન ઉપયોગિતાનો ઉપયોગ કરી શકો છો.

100% મફત

આ CSS કલર કોડ્સ ટૂલનો ઉપયોગ કરવા માટે તમારે કોઈપણ નોંધણી પ્રક્રિયામાંથી પસાર થવાની જરૂર નથી. તમે આ ઉપયોગિતાનો મફતમાં ઉપયોગ કરી શકો છો અને કોઈપણ મર્યાદા વિના અમર્યાદિત CSS કલર કોડ્સ કરી શકો છો.
