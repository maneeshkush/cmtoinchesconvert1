Linux/Unix-ൽ ls കമാൻഡ്

ഫയലുകളുടെയും ഡയറക്‌ടറികളുടെയും ഡയറക്‌ടറി ഉള്ളടക്കങ്ങൾ പട്ടികപ്പെടുത്തുന്ന ഒരു ലിനക്‌സ് ഷെൽ കമാൻഡാണ്ls .

  • ls വാക്യഘടന
  • ls ഓപ്ഷനുകൾ
  • ഉദാഹരണങ്ങൾ
  • ls കോഡ് ജനറേറ്റർ

ls വാക്യഘടന

$ ls [options] [file|dir]

ls കമാൻഡ് ഓപ്ഷനുകൾ

ls കമാൻഡ് പ്രധാന ഓപ്ഷനുകൾ:

ഓപ്ഷൻ വിവരണം
ls -a '.' ൽ ആരംഭിക്കുന്ന മറഞ്ഞിരിക്കുന്ന ഫയൽ ഉൾപ്പെടെ എല്ലാ ഫയലുകളും ലിസ്റ്റ് ചെയ്യുക
ls --നിറം നിറമുള്ള ലിസ്റ്റ് [=എപ്പോഴും/ഒരിക്കലും/ഓട്ടോ]
ls -d ലിസ്റ്റ് ഡയറക്ടറികൾ - ' */' ഉപയോഗിച്ച്
ls -F */=>@| എന്നതിന്റെ ഒരു പ്രതീകം ചേർക്കുകഎൻട്രികളിലേക്ക്
ls -i ലിസ്റ്റ് ഫയലിന്റെ ഐനോഡ് സൂചിക നമ്പർ
ls -l നീണ്ട ഫോർമാറ്റിലുള്ള ലിസ്റ്റ് - അനുമതികൾ കാണിക്കുക
ls -la മറഞ്ഞിരിക്കുന്ന ഫയലുകൾ ഉൾപ്പെടെ നീണ്ട ഫോർമാറ്റ് ലിസ്റ്റ് ചെയ്യുക
ls -lh വായിക്കാനാകുന്ന ഫയൽ വലുപ്പമുള്ള ദൈർഘ്യമേറിയ ഫോർമാറ്റ് ലിസ്റ്റ് ചെയ്യുക
ls -ls ഫയൽ വലുപ്പത്തോടുകൂടിയ നീണ്ട ഫോർമാറ്റിലുള്ള ലിസ്റ്റ്
ls -r വിപരീത ക്രമത്തിൽ പട്ടിക
ls -R ലിസ്റ്റ് ആവർത്തന ഡയറക്ടറി ട്രീ
ls -s ലിസ്റ്റ് ഫയൽ വലിപ്പം
ls -എസ് ഫയൽ വലുപ്പം അനുസരിച്ച് അടുക്കുക
ls -t സമയവും തീയതിയും അനുസരിച്ച് അടുക്കുക
ls -X വിപുലീകരണ നാമം അനുസരിച്ച് അടുക്കുക

ls കമാൻഡ് ഉദാഹരണങ്ങൾ

ഫയൽ അല്ലെങ്കിൽ ഫോൾഡർ പേരുകൾ സ്വയമേവ പൂർത്തിയാക്കാൻനിങ്ങൾക്ക് ടാബ് ബട്ടൺ അമർത്താം.

ആപേക്ഷിക പാതയുള്ള ഡയറക്‌ടറി ഡോക്യുമെന്റുകൾ/ബുക്കുകൾ ലിസ്റ്റ് ചെയ്യുക:

$ ls Documents/Books

 

സമ്പൂർണ്ണ പാതയുള്ള ഡയറക്ടറി /ഹോം/ഉപയോക്താവ്/രേഖകൾ/പുസ്തകങ്ങൾ ലിസ്റ്റ് ചെയ്യുക.

$ ls /home/user/Documents/Books

 

ലിസ്റ്റ് റൂട്ട് ഡയറക്ടറി:

$ ls /

 

പാരന്റ് ഡയറക്ടറി ലിസ്റ്റ് ചെയ്യുക:

$ ls ..

 

ഉപയോക്താവിന്റെ ഹോം ഡയറക്ടറി ലിസ്റ്റ് ചെയ്യുക (ഉദാ: /home/user):

$ ls ~

 

ദൈർഘ്യമേറിയ ഫോർമാറ്റിലുള്ള ലിസ്റ്റ്:

$ ls -l

 

മറഞ്ഞിരിക്കുന്ന ഫയലുകൾ കാണിക്കുക:

$ ls -a

 

ദൈർഘ്യമേറിയ ഫോർമാറ്റിലുള്ള ലിസ്റ്റ്, മറഞ്ഞിരിക്കുന്ന ഫയലുകൾ കാണിക്കുക:

$ ls -la

 

തീയതി/സമയം അനുസരിച്ച് അടുക്കുക:

$ ls -t

 

ഫയൽ വലുപ്പം അനുസരിച്ച് അടുക്കുക:

$ ls -S

 

എല്ലാ ഉപഡയറക്‌ടറികളും ലിസ്റ്റുചെയ്യുക:

$ ls *

 

റിക്കർസീവ് ഡയറക്ടറി ട്രീ ലിസ്റ്റ്:

$ ls -R

 

വൈൽഡ്കാർഡ് ഉള്ള ടെക്സ്റ്റ് ഫയലുകൾ മാത്രം ലിസ്റ്റ് ചെയ്യുക:

$ ls *.txt

 

ഔട്ട്പുട്ട് ഫയലിലേക്കുള്ള ls റീഡയറക്ഷൻ:

$ ls > out.txt

 

ലിസ്റ്റ് ഡയറക്ടറികൾ മാത്രം:

$ ls -d */

 

പൂർണ്ണ പാത്ത് ഉള്ള ഫയലുകളും ഡയറക്‌ടറികളും ലിസ്റ്റ് ചെയ്യുക:

$ ls -d $PWD/*

ls കോഡ് ജനറേറ്റർ

ls ഓപ്ഷനുകൾതിരഞ്ഞെടുത്ത് കോഡ് സൃഷ്ടിക്കുക ബട്ടൺ അമർത്തുക:

ഓപ്ഷനുകൾ 
  നീണ്ട ലിസ്റ്റ് ഫോർമാറ്റ് (-l)
  എല്ലാ ഫയലുകളും / മറഞ്ഞിരിക്കുന്ന ഫയലുകളും ലിസ്റ്റ് ചെയ്യുക (-എ)
  ആവർത്തിച്ചുള്ള ലിസ്റ്റ് ഡയറക്ടറി ട്രീ (-R)
  ലിസ്റ്റ് വിപരീത ക്രമത്തിൽ (-r)
  മുഴുവൻ പാതയും ഉള്ള ലിസ്റ്റ് (-d $PWD/*)
ഇങ്ങനെ അടുക്കുക:
ഫയലുകൾ / ഫോൾഡറുകൾ
ഫയലുകൾ:
ഫോൾഡറുകൾ:
ഔട്ട്പുട്ട് റീഡയറക്ഷൻ
കോഡ് തിരഞ്ഞെടുക്കുന്നതിന് ടെക്സ്റ്റ്ബോക്സിൽ ക്ലിക്ക് ചെയ്യുക, തുടർന്ന് അത് ടെർമിനലിലേക്ക് പകർത്തി ഒട്ടിക്കുക
