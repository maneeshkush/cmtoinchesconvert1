സ്ലോ വെർച്വൽബോക്സ് ഉബുണ്ടു

VirtualBox-ന് കീഴിൽ ഉബുണ്ടു പ്രവർത്തനം മന്ദഗതിയിലാക്കുന്നു.

പ്രശ്നം

VirtualBox-ന് കീഴിലുള്ള ഉബുണ്ടു 13.04 വളരെ സാവധാനത്തിലാണ് പ്രവർത്തിക്കുന്നത്.

വിൻഡോകൾ വളരെ പതുക്കെ തുറക്കുകയോ അടയ്ക്കുകയോ ചെയ്യുന്നു.

കാരണം

12.04 വിതരണത്തിൽ നിന്ന് ഉബുണ്ടുവിന്റെ ഭാഗമായ പുതിയ യൂണിറ്റി ഡെസ്‌ക്‌ടോപ്പിന്റെ മന്ദഗതിയിലുള്ള പ്രവർത്തനമാണ് പ്രശ്‌നത്തിന് കാരണം.

പരിഹാരം

  1. VirtualBox മാനേജർ തുറക്കുക.
  2. സഹായം > അപ്‌ഡേറ്റുകൾക്കായി പരിശോധിക്കുക മെനുവിൽ നിന്ന് ഏറ്റവും പുതിയ പതിപ്പിനായി VirtualBox അപ്‌ഗ്രേഡുചെയ്യുക.
  3. ക്രമീകരണ ടൂൾബാർ ബട്ടൺ അമർത്തുക.
  4. വീഡിയോ ടാബിൽ, വിപുലീകരിച്ച ഫീച്ചറുകൾ, പ്രവർത്തനക്ഷമമാക്കുക 3D ആക്സിലറേഷൻ ചെക്ക്ബോക്സ് സജ്ജമാക്കുക.
  5. ഉബുണ്ടു പുനരാരംഭിക്കുക.

 

PC BIOS ക്രമീകരണങ്ങളിൽ ഹാർഡ്‌വെയർ വിർച്ച്വലൈസേഷൻ പ്രവർത്തനക്ഷമമാക്കിയിട്ടുണ്ടോ എന്നും നിങ്ങൾക്ക് പരിശോധിക്കാം.
