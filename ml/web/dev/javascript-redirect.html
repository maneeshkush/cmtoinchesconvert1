Javascript റീഡയറക്‌ട് URL

Javascript ഉപയോഗിച്ച് URL പേജ് എങ്ങനെ റീഡയറക്‌ട് ചെയ്യാം.

Javascript റീഡയറക്‌ട് 301 സ്ഥിരമായ റീഡയറക്‌ട് സ്റ്റാറ്റസ് കോഡ് നൽകുന്നില്ല.

Javscript റീഡയറക്‌ട്

നിങ്ങൾ റീഡയറക്‌ട് ചെയ്യേണ്ട പേജിന്റെ URL ഉപയോഗിച്ച് റീഡയറക്ഷൻ കോഡ് ഉപയോഗിച്ച് പഴയ പേജ് മാറ്റിസ്ഥാപിക്കുക.

old-page.html:

<!DOCTYPE html>
<html>
<body>
<script type="text/javascript">
    // Javascript URL redirection
    window.location.replace("http://www.mydomain.com/new-page.html");
</script>
</body>
</html>

 

പഴയ URL-ൽ നിന്ന് പുതിയ URL-ലേക്ക് പേജ് റാങ്ക് കൈമാറാൻ തിരയൽ എഞ്ചിനുകൾ 301 സ്റ്റാറ്റസ് കോഡ് ഉപയോഗിക്കുന്നു.

Javascript റീഡയറക്ഷൻ റിട്ടേൺ http പ്രതികരണ സ്റ്റാറ്റസ് കോഡ്: 200 ശരി.

അതിനാൽ ജാവാസ്ക്രിപ്റ്റ് റീഡയറക്ഷൻ സെർച്ച് എഞ്ചിൻ സൗഹൃദമല്ല, സ്റ്റാറ്റസ് കോഡ് നൽകുന്ന മറ്റ് റീഡയറക്ഷൻ രീതികൾ ഉപയോഗിക്കുന്നതാണ് നല്ലത്: 301 ശാശ്വതമായി നീക്കി.

ജാവാസ്ക്രിപ്റ്റ് റീഡയറക്‌ട് ഉദാഹരണം

javascript-redirect-test.htm

<!DOCTYPE html>
<html>
<body>
<script type="text/javascript">
// Javascript URL redirection
window.location.replace("https://www.cmtoinchesconvert.com/ml/web/dev/javascript-redirect.htm");
</script>
</body>
</html>

 

javascript-redirect-test.htm-ൽ നിന്ന് ഈ പേജിലേക്ക് തിരിച്ചുവിടാൻ ഈ ലിങ്ക് അമർത്തുക:
