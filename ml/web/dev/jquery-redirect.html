jQuery റീഡയറക്‌ട്

jQuery ഉപയോഗിച്ച് URL-ലേക്ക് ഒരു പേജ് എങ്ങനെ റീഡയറക്‌ട് ചെയ്യാം.

പഴയ URL-ൽ നിന്ന് പുതിയ URL-ലേക്ക് പേജ് റാങ്ക് കൈമാറാൻ തിരയൽ എഞ്ചിനുകൾ 301 സ്റ്റാറ്റസ് കോഡ് ഉപയോഗിക്കുന്നു.

jQuery റീഡയറക്ഷൻ റിട്ടേൺ http പ്രതികരണ സ്റ്റാറ്റസ് കോഡ്: 200 ശരി.

അതിനാൽ ജാവാസ്ക്രിപ്റ്റ് റീഡയറക്‌ഷൻ പോലെയുള്ള jQuery റീഡയറക്‌ഷനും സെർച്ച് എഞ്ചിൻ സൗഹൃദമല്ല, 301 എന്ന സ്റ്റാറ്റസ് കോഡ് സ്ഥിരമായി നീക്കിയിരിക്കുന്ന മറ്റ് റീഡയറക്ഷൻ രീതികൾ ഉപയോഗിക്കുന്നതാണ് നല്ലത്.

jQuery റീഡയറക്ഷൻ യഥാർത്ഥത്തിൽ Javascript റീഡയറക്ഷൻ ആണ് .jQuery ഉപയോഗിച്ച് റീഡയറക്‌ടുചെയ്യുന്നത് ഓവർകില്ലായി കണക്കാക്കപ്പെടുന്നു, കാരണം നിങ്ങൾക്ക് കുറഞ്ഞ കോഡ് ഉപയോഗിച്ച് ശുദ്ധമായ ജാവാസ്ക്രിപ്റ്റ് റീഡയറക്ഷൻ ചെയ്യാൻ കഴിയും.
