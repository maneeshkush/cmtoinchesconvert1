HTML ഇമേജ് ലിങ്ക്

ഒരു ചിത്രം എങ്ങനെ ലിങ്ക് ആക്കാം.

HTML ഇമേജ് ലിങ്ക് കോഡ്

അല്ലെങ്കിൽ വീതിയും ഉയരവും നിർണ്ണയിക്കാൻ css സ്റ്റൈലിംഗ് ഉപയോഗിക്കുന്നതാണ് നല്ലത്.

കോഡ് ഈ ലിങ്ക് സൃഷ്ടിക്കും:

 

കോഡിന് ഇനിപ്പറയുന്ന ഭാഗങ്ങളുണ്ട്:

  • <a> എന്നത് ലിങ്ക് ടാഗ് ആണ്.
  • href ആട്രിബ്യൂട്ട് ലിങ്ക് ചെയ്യാൻ URL സജ്ജമാക്കുന്നു.
  • <img> എന്നത് ഇമേജ് സ്റ്റാർട്ട് ടാഗ് ആണ്.
  • src ആട്രിബ്യൂട്ട് ഇമേജ് ഫയൽ സജ്ജമാക്കുന്നു.
  • ടൈറ്റിൽ ആട്രിബ്യൂട്ട് ഇമേജ് ടൂൾടിപ്പ് ടെക്സ്റ്റ് സജ്ജമാക്കുന്നു.
  • alt എന്നത് ഇമേജ് ടാഗ് ആൾട്ട് ടെക്സ്റ്റ് ആട്രിബ്യൂട്ടാണ്.
  • ചിത്രത്തിന്റെ വീതിയും ഉയരവും css ഉള്ളസ്റ്റൈൽ ആട്രിബ്യൂട്ട് സെറ്റുകൾ.
  • </a> എന്നത് ലിങ്ക് എൻഡ് ടാഗ് ആണ്.
