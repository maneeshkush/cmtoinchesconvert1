ആങ്കർ ലിങ്ക് ലിങ്ക് നിറം ഇമെയിൽ ലിങ്ക് ചിത്ര ലിങ്ക് ടെക്സ്റ്റ് ലിങ്ക്

 

 

 

#link_bar a എന്നത് ലിങ്കിന്റെ എല്ലാ സ്റ്റേറ്റുകളുടെയും ശൈലിയാണ്.

#link_bar a:link എന്നത് സാധാരണ ലിങ്കിന്റെ ശൈലിയാണ്.

#link_bar a: visit എന്നത് സന്ദർശിച്ച ലിങ്കിന്റെ ശൈലിയാണ്.

#link_bar a:hover എന്നത് മൗസ് ഹോവർഡ് ലിങ്കിന്റെ ശൈലിയാണ്.

#link_bar a: മൗസ് അമർത്തിയാൽ ലിങ്കിന്റെ ശൈലി സജീവമാണ്.
