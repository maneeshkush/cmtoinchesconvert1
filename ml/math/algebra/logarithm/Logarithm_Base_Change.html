അടിസ്ഥാന നിയമത്തിന്റെ ലോഗരിതം മാറ്റം

ബിയിൽ നിന്ന് സിയിലേക്ക് ബേസ് മാറ്റുന്നതിന്, ബേസ് റൂളിന്റെ ലോഗരിതം മാറ്റം നമുക്ക് ഉപയോഗിക്കാം.x ന്റെ അടിസ്ഥാന b ലോഗരിതം x ന്റെ അടിസ്ഥാന c ലോഗരിതം b യുടെ അടിസ്ഥാന c ലോഗരിതം കൊണ്ട് ഹരിച്ചാൽ തുല്യമാണ്:

logb(x) = logc(x) / logc(b)

ഉദാഹരണം #1

log2(100) = log10(100) / log10(2) = 2 / 0.30103 = 6.64386

ഉദാഹരണം #2

log3(50) = log8(50) / log8(3) = 1.8812853 / 0.5283208 = 3.5608766

തെളിവ്

x ന്റെ ബേസ് ബി ലോഗരിതം ഉപയോഗിച്ച് b ഉയർത്തുന്നത് x നൽകുന്നു:

(1) x = blogb(x)

b യുടെ അടിസ്ഥാന c ലോഗരിതം ശക്തി ഉപയോഗിച്ച് c ഉയർത്തുന്നത് b നൽകുന്നു:

(2) b = clogc(b)

നമ്മൾ (1) എടുത്ത് b ന് പകരം c ലോഗ് c ( b ) (2) നൽകുമ്പോൾ, നമുക്ക് ലഭിക്കുന്നത്:

(3) x = blogb(x) = (clogc(b))logb(x) = clogc(b)×logb(x)

(3) ന്റെ ഇരുവശത്തുംലോഗ് സി () പ്രയോഗിക്കുന്നതിലൂടെ:

logc(x) = logc(clogc(b)×logb(x))

ലോഗരിതം പവർ റൂൾ പ്രയോഗിക്കുന്നതിലൂടെ :

logc(x) = [logc(b)×logb(x)] × logc(c)

ലോഗ് സി ( സി )=1 മുതൽ

logc(x) = logc(b)×logb(x)

അഥവാ

logb(x) = logc(x) / logc(b)
