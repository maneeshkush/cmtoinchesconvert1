സാധ്യതയും സ്ഥിതിവിവരക്കണക്കുകളും

ക്രമരഹിതമായ സംഭവങ്ങളെ വിവരിക്കുന്ന ഒരു ഗണിത സിദ്ധാന്തമാണ് പ്രോബബിലിറ്റി.

  • അടിസ്ഥാന സാധ്യത
  • പ്രതീക്ഷിത മൂല്യം ( ഇ )
  • വ്യത്യാസം ( Var )
  • സ്റ്റാൻഡേർഡ് ഡീവിയേഷൻ ( എസ്ടിഡി )
  • പ്രോബബിലിറ്റി ഡിസ്ട്രിബ്യൂഷൻ
  • സാധാരണ വിതരണം
  • പ്രോബബിലിറ്റി & സ്റ്റാറ്റിസ്റ്റിക്സ് ചിഹ്നങ്ങൾ

ഇതും കാണുക

  • സ്റ്റാറ്റിസ്റ്റിക്കൽ ടെസ്റ്റ് കാൽക്കുലേറ്ററുകൾ
