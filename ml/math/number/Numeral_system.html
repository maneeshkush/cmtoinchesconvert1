സംഖ്യാ സംവിധാനങ്ങൾ

  • സംഖ്യാ സംവിധാനം
  • ബൈനറി ന്യൂമറൽ സിസ്റ്റം
  • ഒക്ടൽ ന്യൂമറൽ സിസ്റ്റം
  • ഡെസിമൽ ന്യൂമറൽ സിസ്റ്റം
  • ഹെക്സ് ന്യൂമറൽ സിസ്റ്റം

സംഖ്യാ സംവിധാനം

b - സംഖ്യാ സിസ്റ്റം അടിസ്ഥാനം

d n - n-ാമത്തെ അക്കം

n - സംഖ്യയ്ക്ക് ഒരു ഭിന്നസംഖ്യയുണ്ടെങ്കിൽ നെഗറ്റീവ് നമ്പറിൽ നിന്ന് ആരംഭിക്കാം.

N +1 - അക്കങ്ങളുടെ എണ്ണം

ബൈനറി ന്യൂമറൽ സിസ്റ്റം - ബേസ്-2

ബൈനറി നമ്പറുകൾ 0, 1 അക്കങ്ങൾ മാത്രമേ ഉപയോഗിക്കുന്നുള്ളൂ.

B എന്നത് ബൈനറി പ്രിഫിക്‌സിനെ സൂചിപ്പിക്കുന്നു.

ഉദാഹരണങ്ങൾ:

101012 = 10101B = 1×24+0×23+1×22+0×21+1×20 = 16+4+1= 21

101112 = 10111B = 1×24+0×23+1×22+1×21+1×20 = 16+4+2+1= 23

1000112 = 100011B = 1×25+0×24+0×23+0×22+1×21+1×20 =32+2+1= 35

ഒക്ടൽ ന്യൂമറൽ സിസ്റ്റം - ബേസ്-8

ഒക്ടൽ നമ്പറുകൾ 0..7 മുതൽ അക്കങ്ങൾ ഉപയോഗിക്കുന്നു.

ഉദാഹരണങ്ങൾ:

278 = 2×81+7×80 = 16+7 = 23

308 = 3×81+0×80 = 24

43078 = 4×83+3×82+0×81+7×80= 2247

ഡെസിമൽ ന്യൂമറൽ സിസ്റ്റം - ബേസ്-10

ദശാംശ സംഖ്യകൾ 0..9 മുതൽ അക്കങ്ങൾ ഉപയോഗിക്കുന്നു.

നമ്മൾ ഉപയോഗിക്കുന്ന സ്ഥിരം നമ്പറുകളാണിത്.

ഉദാഹരണം:

253810 = 2×103+5×102+3×101+8×100

ഹെക്സാഡെസിമൽ ന്യൂമറൽ സിസ്റ്റം - ബേസ്-16

ഹെക്‌സ് നമ്പറുകൾ 0..9, A.F എന്നിവയിൽ നിന്നുള്ള അക്കങ്ങൾ ഉപയോഗിക്കുന്നു.

എച്ച് ഹെക്സ് പ്രിഫിക്സിനെ സൂചിപ്പിക്കുന്നു.

ഉദാഹരണങ്ങൾ:

2816 = 28H = 2×161+8×160 = 40

2F16 = 2FH = 2×161+15×160 = 47

BC1216 = BC12H = 11×163+12×162+1×161+2×160= 48146
