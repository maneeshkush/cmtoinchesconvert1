അവസാന ഗ്രേഡ് കണക്കുകൂട്ടൽ

ഉദാഹരണം #1

നിലവിലെ ഗ്രേഡ് 70% (അല്ലെങ്കിൽ സി-) ആണ്.

അവസാന പരീക്ഷയുടെ ഭാരം 50% ആണ്.

ആവശ്യമായ ഗ്രേഡ് 80% (അല്ലെങ്കിൽ ബി-) ആണ്.

കണക്കുകൂട്ടല്:

അതിനാൽ അവസാന പരീക്ഷാ ഗ്രേഡ് ആവശ്യമായ ഗ്രേഡിന് തുല്യമാണ്, മൈനസ് 100% മൈനസ് അവസാന പരീക്ഷയുടെ ഭാരം (w) നിലവിലെ ഗ്രേഡ് (g), അവസാന പരീക്ഷാ ഭാരം (w) കൊണ്ട് ഹരിക്കുന്നു.

അവസാന പരീക്ഷ ഗ്രേഡ് =

= (ആവശ്യമുള്ള ഗ്രേഡ് - (100% - w)× നിലവിലെ ഗ്രേഡ് ) / w

= (80% - (100% - 50%)×70% ) / 50% = 90%

അതിനാൽ അവസാന പരീക്ഷ ഗ്രേഡ് 90% (അല്ലെങ്കിൽ A-) ആയിരിക്കണം.

ഉദാഹരണം #2

നിയമനം 1: ഭാരം1=50%, ഗ്രേഡ്1=20ൽ 16.

നിയമനം 2: ഭാരം2=30%, പരമാവധി ഗ്രേഡ്=30.

നിയമനം 3: ഭാരം3=20%, പരമാവധി ഗ്രേഡ്=40.

85% ക്ലാസ് ഗ്രേഡ് ലഭിക്കാൻ ആവശ്യമായ 2, 3 അസൈൻമെന്റുകളിൽ ശരാശരി ഗ്രേഡ് കണ്ടെത്തുക.

കണക്കുകൂട്ടല്:

നിലവിലെ ഗ്രേഡ് = അസൈൻമെന്റ് 1 ഗ്രേഡ് = ഗ്രേഡ്1 / പരമാവധി ഗ്രേഡ്1 = 16/20 = 0.8 = 80%

ആവശ്യമായ ഗ്രേഡ് = 85%

അവസാന പരീക്ഷയുടെ ഭാരം = w = ഭാരം2+ ഭാരം3 = 30%+20% = 50%

അവസാന പരീക്ഷ ഗ്രേഡ് =

= ( ആവശ്യമുള്ള ഗ്രേഡ് - (100% - w )× നിലവിലെ ഗ്രേഡ് ) / w

= (85% - 50%×80%) / 50% = 90%

85% ക്ലാസ് ഗ്രേഡ് ലഭിക്കുന്നതിന് 2, 3 അസൈൻമെന്റുകളിൽ നിങ്ങൾക്ക് ശരാശരി 90% ഗ്രേഡ് ലഭിക്കണം എന്നാണ് ഇതിനർത്ഥം.

അസൈൻമെന്റ് 2 ഗ്രേഡ് = 90% × പരമാവധി ഗ്രേഡ് = 90% × 30 = 27

അസൈൻമെന്റ് 3 ഗ്രേഡ് = 90% × പരമാവധി ഗ്രേഡ് = 90% × 40 = 36
