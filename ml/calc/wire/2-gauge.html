#2 ഗേജ് വയർ

#2 അമേരിക്കൻ വയറിംഗ് ഗേജ് (AWG) പ്രോപ്പർട്ടികൾ: വ്യാസം, ഏരിയ, പ്രതിരോധം.

  • #2 AWG വ്യാസം
  • #2 AWG ഏരിയ
  • #2 AWG പ്രതിരോധം

#2 AWG വയർ വ്യാസം

ഇഞ്ചിൽ #2 AWG വയറിന്റെ വ്യാസം:

d2(inch) = 0.005 inch × 92 (36-2)/39 = 0.2576 inch

#2 AWG വയറിന്റെ വ്യാസം മില്ലിമീറ്ററിൽ:

d2(mm) = 0.127 mm × 92 (36-2)/39 = 6.5437 mm

#2 AWG വയർ ഏരിയ

കിലോ-വൃത്താകൃതിയിലുള്ള മില്ലുകളിൽ #2 AWG വയറിന്റെ വിസ്തീർണ്ണം:

An (kcmil) = 1000×dn2 = 1000×(0.2576 in)2 = 66.3713 kcmil

ചതുരശ്ര ഇഞ്ചിൽ #2 AWG വയറിന്റെ വിസ്തീർണ്ണം:

A2(inch2) = (π/4)×dn2 = (π/4)×(0.2576 in)2 = 0.0521 inch2

ചതുരശ്ര മില്ലിമീറ്ററിൽ #2 AWG വയറിന്റെ വിസ്തീർണ്ണം:

A2(mm2) = (π/4)×dn2 = (π/4)×(6.5437 mm)2 = 33.6308 mm2

ഓരോ അടിയിലും വയറിന്റെ പ്രതിരോധം

ഓം മീറ്ററിൽ (Ω/kft) n ഗേജ് വയർ റെസിസ്റ്റൻസ് R എന്നത് 0.3048×1000000000 മടങ്ങ് വയർ റെസിസ്റ്റിവിറ്റി ρ ഓം-മീറ്ററിൽ (Ω·m) 25.4 2 മടങ്ങ് ക്രോസ് സെക്ഷണൽ ഏരിയ A n ചതുരശ്ര ഇഞ്ചിൽ ഹരിച്ചാൽ (Ω/kft) തുല്യമാണ്. 2 -ൽ):

Rn (Ω/kft) = 0.3048 × 109 × ρ(Ω·m) / (25.42 × An (in2))

 

ഒരു മീറ്ററിന് പ്രതിരോധം

n ഗേജ് വയർ റെസിസ്റ്റൻസ് R ഓരോ കിലോമീറ്ററിലും (Ω/km) 1000000000 മടങ്ങ് വയർ റെസിസ്റ്റിവിറ്റി ρ ഓം-മീറ്ററിൽ (Ω·m) ക്രോസ് സെക്ഷണൽ ഏരിയ A n കൊണ്ട് ചതുരശ്ര മില്ലിമീറ്ററിൽ (mm 2 ) ഹരിച്ചാൽ തുല്യമാണ്:

Rn (Ω/km) = 109 × ρ(Ω·m) / An (mm2)
