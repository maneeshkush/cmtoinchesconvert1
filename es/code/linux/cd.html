comando cd en Linux/Unix

cd es un comando de Linux para cambiar el directorio/carpeta del shell de la terminal.

Puede presionar el botón de tabulación para completar automáticamente el nombre del directorio.

  • sintaxis de CD
  • ejemplos de discos compactos

sintaxis de CD

$ cd [directory]

ejemplos de comandos cd

Cambiar al directorio de inicio (determinado por la variable de entorno $HOME):

$ cd

 

También cambie al directorio de inicio:

$ cd ~

 

Cambiar al directorio raíz:

$ cd /

 

Cambiar al directorio principal:

$ cd ..

 

Cambiar al subdirectorio Documentos :

$ cd Documents

 

Cambiar al subdirectorio Documentos/Libros :

$ cd Documents/Books

 

Cambie al directorio con la ruta absoluta /home/user/Desktop :

$ cd /home/user/Desktop

 

Cambiar al nombre del directorio con espacios en blanco - Mis imágenes :

$ cd My\ Images

O

$ cd "My Images"

O

$ cd 'My Images'
