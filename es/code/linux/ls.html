Comando ls en Linux/Unix

ls es un comando de shell de Linux que enumera el contenido del directorio de archivos y directorios.

  • ls sintaxis
  • ls opciones
  • ls ejemplos
  • generador de código ls

ls sintaxis

$ ls [options] [file|dir]

opciones de comando ls

Opciones principales del comando ls:

opción descripción
ls-a enumere todos los archivos, incluido el archivo oculto que comienza con '.'
ls --color lista de colores [=siempre/nunca/automático]
ls-d lista de directorios - con ' */'
ls -F agregue un carácter de */=>@| a las entradas
ls -i enumerar el número de índice de inodo del archivo
ls-l lista con formato largo - mostrar permisos
ls-la lista de formato largo que incluye archivos ocultos
ls-lh lista de formato largo con tamaño de archivo legible
ls-ls lista con formato largo con tamaño de archivo
ls-r lista en orden inverso
ls -R enumerar el árbol de directorio recursivamente
ls-s tamaño de archivo de lista
ls -S ordenar por tamaño de archivo
ls-t ordenar por fecha y hora
ls -X ordenar por nombre de extensión

ejemplos de comandos ls

Puede presionar el botón de tabulación para completar automáticamente los nombres de archivos o carpetas.

Listar el directorio Documentos/Libros con ruta relativa :

$ ls Documents/Books

 

Liste el directorio /inicio/usuario/Documentos/Libros con ruta absoluta .

$ ls /home/user/Documents/Books

 

Listar el directorio raíz:

$ ls /

 

Listar el directorio principal:

$ ls ..

 

Enumere el directorio de inicio del usuario (p. ej.: /inicio/usuario):

$ ls ~

 

Lista con formato largo:

$ ls -l

 

Mostrar archivos ocultos:

$ ls -a

 

Lista con formato largo y muestra archivos ocultos:

$ ls -la

 

Ordenar por fecha/hora:

$ ls -t

 

Ordenar por tamaño de archivo:

$ ls -S

 

Listar todos los subdirectorios:

$ ls *

 

Lista recursiva del árbol de directorios:

$ ls -R

 

Enumere solo archivos de texto con comodines:

$ ls *.txt

 

ls redirección al archivo de salida:

$ ls > out.txt

 

Listar directorios solamente:

$ ls -d */

 

Lista de archivos y directorios con la ruta completa:

$ ls -d $PWD/*
