Comando mv en Linux/Unix

Comando mv de linux.

El comando mv se usa para mover archivos y directorios.

sintaxis del comando mv

$ mv [options] source dest

opciones de comando mv

Opciones principales del comando mv:

opción descripción
mv-f forzar el movimiento sobrescribiendo el archivo de destino sin aviso
mv-yo aviso interactivo antes de sobrescribir
mv -u actualizar: mover cuando la fuente es más nueva que el destino
mv-v verbose - imprimir archivos de origen y destino
hombre mv manual de ayuda

ejemplos de comandos mv

Mueva los archivos main.c def.h al directorio /home/usr/rapid/ :

$ mv main.c def.h /home/usr/rapid/

 

Mueva todos los archivos C en el directorio actual al subdirectorio bak:

$ mv *.c bak

 

Mueva todos los archivos en el subdirectorio bak al directorio actual:

$ mv bak/* .

 

Cambie el nombre del archivo main.c a main.bak:

$ mv main.c main.bak

 

Cambiar el nombre del directorio bak a bak2:

$ mv bak bak2

 

Actualizar: mover cuando main.c sea más nuevo:

$ mv -u main.c bak
$

 

Mueva main.c y pregunte antes de sobrescribir bak/main.c:

$ mv -v main.c bak
'bak/main.c' -> 'bak/main.c'
$
