Cómo obtener el directorio de trabajo actual

Unix/Linux obtiene el directorio de trabajo actual.

Para obtener el directorio de trabajo actual, use el comando pwd.

Por ejemplo, si cambiamos el directorio a /home/user, pwd imprimirá /home/user como el directorio de trabajo actual:

$ cd /home/user
$ pwd
/home/user

 

En el script de shell Bash, puede obtener el directorio de trabajo actual de la siguiente manera:

dir=$(PWD)
