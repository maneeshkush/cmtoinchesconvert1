Código de color RGB amarillo

Código RGB amarillo = 255*65536+255*256+0 = #FFFF00

ROJO=255, VERDE=255, AZUL=0

Código de color HTML amarillo

Párrafo HTML con fuentes amarillas

Código:

<p style=" color:amarillo; fondo:negro">¡Estas fuentes son amarillas!</p>

Resultado:

¡Estas fuentes son amarillas!

O

<p style=" color:#FFFF00; background:black">¡Estas fuentes también son amarillas!</p>

Resultado:

¡Estas fuentes también son amarillas!

O

<p style=" color:rgb(255,255,0); background:black">¡Estas fuentes también son amarillas!</p>

Resultado:

¡Estas fuentes también son amarillas!

Párrafo HTML con fuentes negras y color de fondo amarillo

Código:

<p style="color:black; background:yellow ">El color de fondo es amarillo</p>

Resultado:

el color de fondo es amarillo
