Detección de resolución de pantalla con Javascript

Puede obtener el tamaño de la pantalla con las propiedades screen.width y screen.height .

Puede obtener el tamaño de pantalla disponible con las propiedades screen.availWidth y screen.availHeight .

Por ejemplo:

<script>
alert(screen.width+' x '+screen.height);
</script>
