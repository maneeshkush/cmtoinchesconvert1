Enlace URL canónico

Etiqueta de URL canónica. Etiqueta de enlace canónico.

Redirección de etiqueta de enlace canónico HTML

El enlace canónico no redirige a la URL preferida, pero puede ser una alternativa a la redirección de URL para sitios web donde la mayor parte del tráfico proviene de motores de búsqueda.

La etiqueta HTML para enlaces canónicos se puede usar cuando hay varias páginas con contenido similar y desea decirle a los motores de búsqueda qué página prefiere en los resultados de búsqueda.

La etiqueta de enlace canónico puede vincular al mismo dominio y también a múltiples dominios .

Agregue la etiqueta de enlace canónico a la página anterior para vincular a la
nueva página.

Agregue la etiqueta de enlace canónico a las páginas que prefiera no  vincular el tráfico del motor de  búsqueda   a la página preferida.

La etiqueta de enlace canónico debe agregarse en la sección.

Ejemplo 1

Cuando la página se ha movido, podemos agregar un enlace canónico en la página anterior para notificar a los motores de búsqueda que lleguen a la nueva página.

antigua-pagina1.html:

<!DOCTYPE html>
<html>
<head>
   ...
   <link rel="canonical" href="http://www.mydomain.com/new-page1.html">
</head>
<body>
   ...
</body>
</html>

Ejemplo #2

Cuando hay varias páginas con un interés similar, si preferimos que los motores de búsqueda lleguen a la página3.html, debemos agregar un enlace canónico a la página3.html en la sección principal de la página1.html y la página2.html.

pagina1.html:

<!DOCTYPE html>
<html>
<head>
   ...
   <link rel="canonical" href="http://www.mydomain.com/page3.html">
</head>
<body>
   ...
</body>
</html>

página2.html:

<!DOCTYPE html>
<html>
<head>
   ...
   <link rel="canonical" href="http://www.mydomain.com/page3.html">
</head>
<body>
   ...
</body>
</html>
