redirección jQuery

Cómo redirigir una página a URL con jQuery.

Los motores de búsqueda utilizan el código de estado 301 para transferir el ranking de la página de la antigua URL a la nueva URL.

La redirección jQuery devuelve el código de estado de respuesta http: 200 OK.

Por lo tanto, la redirección de jQuery, como la redirección de Javascript, no es compatible con los motores de búsqueda y es mejor usar otros métodos de redirección que devuelvan el código de estado 301 Movido permanentemente.

La redirección de jQuery es en realidad una redirección de Javascript . La redirección con jQuery se considera una exageración porque puede hacer una redirección de Javascript pura con menos código.

redirección jQuery

Reemplace la página anterior con el código de redirección con la URL de la página a la que desea redirigir.

<html>
