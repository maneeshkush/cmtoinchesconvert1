Símbolo infinito

El símbolo de infinito es un símbolo matemático que representa un número infinitamente grande.

El símbolo de infinito se escribe con el símbolo de la Lemniscata:

∞

Representa un gran número infinitamente positivo.

Cuando queremos escribir un número infinitamente negativo debemos escribir:

-∞

Cuando queremos escribir un número infinitamente pequeño debemos escribir:

1/∞

¿El infinito es un número real?

El infinito no es un número. No representa un número específico, sino una cantidad infinitamente grande.

Reglas y propiedades del infinito

Nombre tipo de clave
infinito positivo ∞
infinito negativo -∞
diferencia infinita ∞ - ∞ no está definido
producto cero 0 ⋅ ∞ no está definido
cociente infinito ∞ / ∞ no está definido
Suma de números reales x + ∞ = ∞, para x ∈ℝ
Producto de número positivo x ⋅ ∞ = ∞, para x >0
