Simplificar exponentes racionales

La base b elevada a la potencia de n/m es igual a:

bn/m = (m√b)n = m√(bn)

Ejemplo:

La base 2 elevada a la potencia de 3/2 es igual a 1 dividido por la base 2 elevada a la potencia de 3:

23/2 = 2√(23) = 2.828

Simplificar fracciones con exponentes

Fracciones con exponentes:

(a / b)n = an / bn

Ejemplo:

(4/3)3 = 43 / 33 = 64 / 27 = 2.37

Simplificar exponentes negativos

La base b elevada a la potencia de menos n es igual a 1 dividido por la base b elevada a la potencia de n:

b-n = 1 / bn

Ejemplo:

La base 2 elevada a la potencia de menos 3 es igual a 1 dividido por la base 2 elevada a la potencia de 3:

2-3 = 1/23 = 1/(2⋅2⋅2) = 1/8 = 0.125

Simplificar radicales con exponentes

Para radical con exponente:

(m√a)n = an/m

Ejemplo:

(√5)4 = 54/2 = 52 = 25
