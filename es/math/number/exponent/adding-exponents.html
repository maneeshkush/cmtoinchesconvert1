Sumar exponentes

Sumar números con exponentes

La suma de exponentes se realiza calculando primero cada exponente y luego sumando:

an + bm

Ejemplo:

42 + 25 = 4⋅4+2⋅2⋅2⋅2⋅2 = 16+32 = 48

Sumando las mismas bases b y exponentes n:

bn + bn = 2bn

Ejemplo:

42 + 42 = 2⋅42 = 2⋅4⋅4 = 32

Sumar exponentes negativos

La suma de exponentes negativos se realiza calculando primero cada exponente y luego sumando:

a-n + b-m = 1/an + 1/bm

Ejemplo:

4-2 + 2-5 = 1/42 + 1/25 = 1/(4⋅4)+1/(2⋅2⋅2⋅2⋅2) = 1/16+1/32 = 0.09375

Sumar exponentes fraccionarios

La suma de exponentes fraccionarios se realiza elevando primero cada exponente y luego sumando:

an/m + bk/j

Ejemplo:

33/2 + 25/2 = √(33) + √(25) = √(27) + √(32) = 5.196 + 5.657 = 10.853

 

Sumando las mismas bases b y exponentes n/m:

bn/m + bn/m = 2bn/m

Ejemplo:

42/3 + 42/3 = 2⋅42/3 = 2 ⋅ 3√(42) = 5.04

Sumar variables con exponentes

La suma de exponentes se realiza calculando primero cada exponente y luego sumando:

xn + xm

Con los mismos exponentes:

xn + xn = 2xn

Ejemplo:

x2 + x2 = 2x2
