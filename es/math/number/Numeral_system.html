sistema numeral

b - base del sistema numérico

d n - el n-ésimo dígito

n - puede comenzar desde un número negativo si el número tiene una parte fraccionaria.

N +1 - el número de dígitos

Sistema numérico binario - Base-2

Los números binarios usan solo 0 y 1 dígitos.

B denota prefijo binario.

Ejemplos:

101012 = 10101B = 1×24+0×23+1×22+0×21+1×20 = 16+4+1= 21

101112 = 10111B = 1×24+0×23+1×22+1×21+1×20 = 16+4+2+1= 23

1000112 = 100011B = 1×25+0×24+0×23+0×22+1×21+1×20 =32+2+1= 35

Sistema de numeración octal - Base-8

Los números octales utilizan dígitos del 0 al 7.

Ejemplos:

278 = 2×81+7×80 = 16+7 = 23

308 = 3×81+0×80 = 24

43078 = 4×83+3×82+0×81+7×80= 2247

Sistema numérico decimal - Base-10

Los números decimales utilizan dígitos del 0 al 9.

Estos son los números regulares que usamos.

Ejemplo:

253810 = 2×103+5×102+3×101+8×100

Sistema numérico hexadecimal - Base-16

Los números hexadecimales usan dígitos de 0..9 y A..F.

H denota prefijo hexadecimal.

Ejemplos:

2816 = 28H = 2×161+8×160 = 40

2F16 = 2FH = 2×161+15×160 = 47

BC1216 = BC12H = 11×163+12×162+1×161+2×160= 48146
