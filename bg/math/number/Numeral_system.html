Бройна система

b - основа на бройната система

d n - n-тата цифра

n - може да започне от отрицателно число, ако числото има дробна част.

N +1 - броят на цифрите

Двоична бройна система - основа-2

Двоичните числа използват само 0 и 1 цифри.

B обозначава двоичен префикс.

Примери:

101012 = 10101B = 1×24+0×23+1×22+0×21+1×20 = 16+4+1= 21

101112 = 10111B = 1×24+0×23+1×22+1×21+1×20 = 16+4+2+1= 23

1000112 = 100011B = 1×25+0×24+0×23+0×22+1×21+1×20 =32+2+1= 35

Осмична бройна система - основа-8

Осмичните числа използват цифри от 0..7.

Примери:

278 = 2×81+7×80 = 16+7 = 23

308 = 3×81+0×80 = 24

43078 = 4×83+3×82+0×81+7×80= 2247

Десетична бройна система - основа-10

Десетичните числа използват цифри от 0..9.

Това са обичайните числа, които използваме.

Пример:

253810 = 2×103+5×102+3×101+8×100

Шестнадесетична бройна система - основа-16

Шестнадесетичните числа използват цифри от 0..9 и A..F.

H обозначава шестнадесетичен префикс.

Примери:

2816 = 28H = 2×161+8×160 = 40

2F16 = 2FH = 2×161+15×160 = 47

BC1216 = BC12H = 11×163+12×162+1×161+2×160= 48146
