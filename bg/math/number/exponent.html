Какво е степенен показател

Основата a, повдигната на степен n, е равна на умножението на a, n пъти:

a n = a × a × ... × a

                    n пъти

a е основата, а n е степента.

Примери

31 = 3

32 = 3 × 3 = 9

33 = 3 × 3 × 3 = 27

34 = 3 × 3 × 3 × 3 = 81

35 = 3 × 3 × 3 × 3 × 3 = 243

Правила за произведение на показателите

Продуктово правило със същата основа

an ⋅ am = an+m

Пример:

23 ⋅ 24 = 23+4 = 27 = 2⋅2⋅2⋅2⋅2⋅2⋅2 = 128

Правило за произведение с еднакъв показател

an ⋅ bn = (a ⋅ b)n

Пример:

32 ⋅ 42 = (3⋅4)2 = 122 = 12⋅12 = 144

Правила за коефициент на експоненти

Правило за частно със същата основа

an / am = an-m

Пример:

25 / 23 = 25-3 = 22 = 2⋅2 = 4

Правило за частно с еднакъв показател

an / bn = (a / b)n

Пример:

43 / 23 = (4/2)3 = 23 = 2⋅2⋅2 = 8

Правила за степен на експоненти

Силово правило I

(an) m = a n⋅m

Пример:

(23)2 = 23⋅2 = 26 = 2⋅2⋅2⋅2⋅2⋅2 = 64

Силово правило II

a nm = a (nm)

Пример:

232 = 2(32) = 2(3⋅3) = 29 = 2⋅2⋅2⋅2⋅2⋅2⋅2⋅2⋅2 = 512

Силово правило с радикали

m√(a n) = a n/m

Пример:

2√(26) = 26/2 = 23 = 2⋅2⋅2 = 8

Правило за отрицателни показатели

b-n = 1 / bn

Пример:

2-3 = 1/23 = 1/(2⋅2⋅2) = 1/8 = 0.125
