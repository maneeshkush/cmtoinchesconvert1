Келвин

Келвин е единицата за измерване на температурата.

Точката на замръзване/топене на водата при налягане от 1 атмосфера е около 273,15 К.

Символът за Келвин е К.

Преобразуване на Келвин в Целзий

Така че 0 Келвин е равно на -273,15 градуса по Целзий :

0 K = -273.15 °C

Така температурата T  в градуси  по Целзий (°C)  е равна на температурата  T в Келвин (K) минус [273.15].

T(°C) = T(K) - 273.15

Пример 1

Преобразувайте 250 келвина в градуси по Целзий:

T(°C) = 250K - 273.15 = -23.15 °C

Пример 2

Преобразувайте 330 келвина в градуси по Целзий:

T(°C) = 330K - 273.15 = 56.85 °C

Пример 3

Преобразувайте 360 келвина в градуси по Целзий:

T(°C) = 360K - 273.15 = 86.85 °C

Преобразуване на Келвин във Фаренхайт

И така, температурата T  в градуси  по Фаренхайт (°F)  е равна на температурата  T в Келвин (K) по 9/5, минус [459,67].

T(°F) = T(K) × 9/5 - 459.67

Пример 1

Преобразувайте 250 келвина в градуси по Фаренхайт:

T(°F) = 250K × 9/5 - 459.67 = -9.67 °F

Пример 2

Преобразувайте 330 келвина в градуси по Фаренхайт:

T(°F) = 330K × 9/5 - 459.67 = 134.33 °F

Пример 3

Преобразувайте 360 келвина в градуси по Фаренхайт:

T(°F) = 360K × 9/5 - 459.67 = 188.33 °F

Преобразуване на Келвин в Ранкин

И така, температурата T  в градуси Ранкин (°R) е равна на температурата  T в Келвин (K), умножена по [9/5].

T(°R) = T(K) × 9/5

Пример 1

Преобразувайте 250 Келвина в градуси Ранкин:

T(°R) = 250K × 9/5 = 450 °R

Пример 2

Преобразувайте 330 келвина в градуси Ранкин:

T(°R) = 330K × 9/5 = 594 °R

Пример 3

Преобразувайте 360 келвина в градуси Ранкин:

T(°R) = 360K × 9/5 = 648 °R
