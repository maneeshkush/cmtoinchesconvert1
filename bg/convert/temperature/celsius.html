Целзий

Целзий или Целзий е единица за измерване на температурата.

Точката на замръзване/топене на водата при налягане от 1 атмосфера е приблизително нула градуса по Целзий (0 °C).

Точката на кипене на водата при налягане от 1 атмосфера е около сто градуса по Целзий (100 °C).

Точните стойности зависят от състава на водата (обикновено количеството сол) и налягането на въздуха.

Морската вода съдържа сол и точката на замръзване пада под 0 °C.

Кипящата вода на планина над морското равнище понижава точката на кипене със 100 °C.

Символът за градуса по Целзий е °C.

Преобразуване от Целзий до Фаренхайт

Така че 0 градуса по Целзий е равно на 32 градуса по Фаренхайт.

0 °C = 32 °F

И така, температурата T  в градуси  по Фаренхайт (°F)  е равна на температурата  T  в градуси по Целзий (°C) по 9/5 плюс 32:

T(°F) = T(°C) × 9/5 + 32

Пример 1

Преобразувайте 15 градуса по Целзий в градуси по Фаренхайт:

T(°F) = 15°C × 9/5 + 32 = 59 °F

Пример 2

Преобразувайте 26 градуса по Целзий в градуси по Фаренхайт:

T(°F) = 26°C × 9/5 + 32 = 78.8 °F

Пример 3

Преобразувайте 30 градуса по Целзий в градуси по Фаренхайт:

T(°F) = 30°C × 9/5 + 32 = 86 °F

Преобразуване на Целзий в Келвин

0 градуса по Целзий е равно на 273,15 градуса по  Келвин :

0 °C = 273.15 K

Така температурата T  в  Келвин (K)  е равна на температурата  T в градуси Целзий (°C) плюс [273,15]

T(K) = T(°C) + 273.15

Пример 1

Преобразувайте 15 градуса по Целзий в Келвин:

T(K) = 15°C + 273.15 = 288.15 K

Пример 2

Преобразувайте 25 градуса по Целзий в Келвин:

T(K) = 25°C + 273.15 = 298.15 K

Пример 3

Преобразувайте 30 градуса по Целзий в Келвин:

T(K) = 30°C + 273.15 = 303.15 K

Преобразуване на Целзий в Ранкин

И така, температурата Т  в градуси Ранкин (°R) е равна на температурата  Т в градуси Целзий (°C) плюс 273,15, умножено по 9/5.

T(°R) = (T(°C) + 273.15) × 9/5

Пример 1

Преобразуване на 15 градуса по Целзий в градуси Ранкин:

T(°R) = (15°C + 273.15) × 9/5 = 518.67 °R

Пример 2

Преобразувайте 25 градуса по Целзий в градуси Ранкин:

T(°R) = (25°C + 273.15) × 9/5 = 536.67 °R

Пример 3

Преобразуване на 30 градуса по Целзий в градуси Ранкин:

T(°R) = (30°C + 273.15) × 9/5 = 545.67 °R
