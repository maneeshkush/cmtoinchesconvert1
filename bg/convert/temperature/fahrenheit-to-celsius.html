Как да конвертирате Фаренхайт в Целзий

Така че 0 градуса по Фаренхайт е равно на -17,77778 градуса по Целзий.

0 °F = -17.77778 °C

T(°C) = (T(°F) - 32) × 5/9

или

T(°C) = (T(°F) - 32) / (9/5)

или

T(°C) = (T(°F) - 32) / 1.8

Преобразувайте 60 градуса по Фаренхайт в градуси по Целзий:

Пример 3

Преобразувайте 90 градуса по Фаренхайт в градуси по Целзий:
