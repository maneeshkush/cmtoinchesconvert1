Фаренхайт

Фаренхайт е единицата за измерване на температурата.

Точката на замръзване/топене на водата при налягане от 1 атмосфера е около 32 °F.

Символът за градуса по Фаренхайт е °F.

Преобразуване на Фаренхайт в Целзий

Така че 0 градуса по Фаренхайт е равно на -17,77778 градуса по Целзий.

0 °F = -17.77778 °C

И така, температурата T  в градуси  по Целзий (°C)  е равна на температурата  T в градуси по Фаренхайт (°F) минус 32, по [5/9].

T(°C) = (T(°F) - 32) × 5/9

Пример 1

Преобразувайте 70 градуса по Фаренхайт в градуси по Целзий:

T(°C) = (70°F - 32) × 5/9 = 21.1111 °C

Пример 2

Преобразувайте 80 градуса по Фаренхайт в градуси по Целзий:

T(°C) = (80°F - 32) × 5/9 = 26.6667 °C

Пример 3

Преобразувайте 100 градуса по Фаренхайт в градуси по Целзий:

T(°C) = (100°F - 32) × 5/9 = 37.7778 °C

Преобразуване на Фаренхайт в Ранкин

Така че 0 градуса по Фаренхайт е равно на [459,67] градуса по Ранкин.

0 °F = 459.67 °R

Температурата  T  в градуси на Ранкин (°R) е равна на температурата  T в градуси по Фаренхайт (°F) плюс [459,67].

T(°R) = T(°F) + 459.67

Пример 1

Преобразуване на 70 градуса по Фаренхайт в градуси Ранкин:

T(°R) = 70°F + 459.67 = 529.67 °R

Пример 2

Преобразувайте 90 градуса по Фаренхайт в градуси Ранкин:

T(°R) = 90°F + 459.67 = 549.67 °R

Пример 3

Преобразувайте 100 градуса по Фаренхайт в градуси Ранкин:

T(°R) = 100°F + 459.67 = 559.67 °R

Преобразуване на Фаренхайт в Келвин

И така, температурата T  в  Келвин (K)  е равна на температурата  T в градуси по Фаренхайт (°F) плюс 459,67, по [5/9].

T(K) = (T(°F) + 459.67)× 5/9

Пример 1

Преобразувайте 65 градуса по Фаренхайт в градуси Келвин:

T(K) = (65°F + 459.67)× 5/9 = 291.483 K

Пример 2

Преобразувайте 80 градуса по Фаренхайт в градуси Келвин:

T(K) = (80°F + 459.67)× 5/9 = 299.817 K

Пример 3

Преобразувайте 100 градуса по Фаренхайт в градуси Келвин:

T(K) = (100°F + 459.67)× 5/9 = 310.928 K
