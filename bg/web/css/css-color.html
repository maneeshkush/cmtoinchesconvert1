CSS цвят

Цветният код може да бъде един от:

Шестнадесетичен формат: #rrggbb

RGB формат: rgb (червено, зелено, синьо)

Формат на името: име

Пример

За оранжев цвят:

Шестнадесетичен формат: #FFA500

RGB формат: rgb(255,165,0)

Формат на името: оранжев

Задаване на цвят на елемента

За конкретен елемент:

<element style="color: code;"></element>

За всички елементи от същия тип. Поставете кода в тага <style> в секцията head:

<style>
   element { color: code; }
</style>

Задаване на цвят на конкретен параграф:

<p style="color: #FF0000;">Some text ...</p>

<style>

Задаване на цвят на фона на елемента

element { background: code; }

Задаване на червен цвят на фона на параграфа:

Задаване на цвят на фона на конкретен параграф:

<p style="background: #FF0000;">Some text ...</p>

<style>
   p { background: #FF0000; }
</style>

Задаване на цвят на границата на конкретен параграф:
