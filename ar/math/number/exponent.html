ما هو الأس

القاعدة a المرفوعة للقوة n تساوي ضرب a ، n مرة:

a n = a × a × ... × a

                    مرات n

a هو الأساس و n هو الأس.

أمثلة

31 = 3

32 = 3 × 3 = 9

33 = 3 × 3 × 3 = 27

34 = 3 × 3 × 3 × 3 = 81

35 = 3 × 3 × 3 × 3 × 3 = 243

قواعد منتج الدعاة

حكم المنتج مع نفس القاعدة

an ⋅ am = an+m

مثال:

23 ⋅ 24 = 23+4 = 27 = 2⋅2⋅2⋅2⋅2⋅2⋅2 = 128

قاعدة المنتج مع نفس الأس

an ⋅ bn = (a ⋅ b)n

مثال:

32 ⋅ 42 = (3⋅4)2 = 122 = 12⋅12 = 144

قواعد حاصل الأسس

قاعدة حاصل القسمة مع نفس القاعدة

an / am = an-m

مثال:

25 / 23 = 25-3 = 22 = 2⋅2 = 4

قاعدة حاصل القسمة مع نفس الأس

an / bn = (a / b)n

مثال:

43 / 23 = (4/2)3 = 23 = 2⋅2⋅2 = 8

انظر: قسمة الأسس

قواعد قوة الدعاة

حكم القوة أنا

(an) m = a n⋅m

مثال:

(23)2 = 23⋅2 = 26 = 2⋅2⋅2⋅2⋅2⋅2 = 64

حكم القوة II

a nm = a (nm)

مثال:

232 = 2(32) = 2(3⋅3) = 29 = 2⋅2⋅2⋅2⋅2⋅2⋅2⋅2⋅2 = 512

حكم القوة مع الراديكاليين

m√(a n) = a n/m

مثال:

2√(26) = 26/2 = 23 = 2⋅2⋅2 = 8

قاعدة الأس السالب

b-n = 1 / bn

مثال:

2-3 = 1/23 = 1/(2⋅2⋅2) = 1/8 = 0.125
