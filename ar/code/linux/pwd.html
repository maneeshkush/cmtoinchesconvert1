أمر pwd في Linux / Unix

أمر Unix / Linux pwd.

pwd - دليل عمل الطباعة ، هو أمر Linux للحصول على دليل العمل الحالي.

  • بناء الجملة pwd
  • أمثلة pwd

بناء الجملة pwd

$ pwd [option]

أمثلة الأمر pwd

قم بتغيير الدليل إلى دليل / usr / src وطباعة دليل العمل:

$ cd /usr/src
$ pwd
/user/src

 

قم بتغيير الدليل إلى الدليل الرئيسي وطباعة دليل العمل:

$ cd ~
$ pwd
/home/user

 

قم بتغيير الدليل إلى الدليل الأصل للدليل الرئيسي ودليل عمل الطباعة:

$ cd ~/..
$ pwd
/home

 

تغيير الدليل إلى الدليل الجذر وطباعة دليل العمل:

$ cd /
$ pwd
/
