شريط CSS navigaiton - أفقي وعمودي.

شريط التنقل الأفقي

كود CSS:

<style>
    #h_nav_bar a { padding:15px; font-weight:bold; float:left; }
    #h_nav_bar a:link { color:#d0d0d0; background-color:#0000a0; }
    #h_nav_bar a:visited { color:#c0c0c0; background-color:#0000a0; }
    #h_nav_bar a:hover { color:#ffffff; background-color:#000060; }
    #h_nav_bar a:active { color:#f0f0f0; background-color:#00ff00; }
</style>

كود HTML:

<div id="h_nav_bar">
    <a href="html-anchor-link.htm">Anchor link</a>
    <a href="html-link-color.htm">Link color</a>
    <a href="../mailto.htm">Email link</a>
    <a href="html-image-link.htm">Image link</a>
    <a href="html-text-link.htm">Text link</a>
</div>

رأي:

رابط المرساة لون الارتباط ارتباط البريد الإلكتروني رابط الصورة رابط نصي

 

 

 

#link_bar a هو النمط لجميع حالات الارتباط.

#link_bar a: link هو نمط الارتباط العادي.

#link_bar a: تمت زيارته هو نمط الارتباط الذي تمت زيارته.

#link_bar a: hover هو نمط الارتباط الذي يتم تمريره بالماوس.

#link_bar a: active هو نمط الارتباط عند الضغط عليه بالماوس.

ميزات أداة شريط التنقل CSS

تتيح أداة CSS Navigation Bar Tool للمستخدمين استخدام CSS Navigation Bar. يتم شرح بعض الميزات البارزة لهذه الأداة أدناه.

بساطة

توفر لك الواجهة سهلة الاستخدام لأداة CSS Navigation Bar Tool الخاصة بنا سهولة إجراء هذا التحويل على الفور. لم تعد بحاجة إلى اكتساب مهارات البرمجة لـ CSS Navigation Bar نظرًا لبساطة هذه الأداة.

رقم التسجيل

لا تحتاج إلى إجراء أي عملية تسجيل لاستخدام CSS Navigation Bar. باستخدام هذه الأداة المساعدة ، يمكنك CSS Navigation Bar Tool عدة مرات كما تريد مجانًا.

قابلية التنقل

يمكن الوصول إلى شريط التنقل CSS هذا من أي ركن من أركان العالم. لا داعي للقلق بشأن توافق هذه الأداة عبر الإنترنت ، حيث يمكنك استخدام CSS Navigation Bar معها من أي جهاز. كل ما تحتاجه هو اتصال بالإنترنت للوصول إلى أداة شريط التنقل CSS هذه واستخدامها.

يوفر الوقت والجهد

يتيح لك CSS Navigation Bar إكمال نفس المهمة على الفور. لن يُطلب منك اتباع الإجراءات اليدوية ، حيث ستقوم الخوارزميات الآلية بالعمل نيابةً عنك.
