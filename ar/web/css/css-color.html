لون CSS

يمكن أن يكون رمز اللون واحدًا مما يلي:

تنسيق سداسي: #rrggbb

تنسيق RGB: RGB (أحمر ، أخضر ، أزرق)

تنسيق الاسم: الاسم

مثال

للون البرتقالي:

تنسيق سداسي عشري: # FFA500

تنسيق RGB: rgb (255،165،0)

تنسيق الاسم: برتقالي

ضبط لون العنصر

لعنصر معين:

<element style="color: code;"></element>

لجميع العناصر من نفس النوع. ضع الكود في علامة <style> في قسم الرأس:

<style>
   element { color: code; }
</style>

مثال

ضبط لون نص الفقرة على اللون الأحمر:

تحديد لون فقرة معينة:

<p style="color: #FF0000;">Some text ...</p>

رأي:

بعض النصوص ...

تحديد لون جميع الفقرات

<style>
   p { color: #FF0000; }
</style>

ضبط لون خلفية العنصر

element { background: code; }

مثال

ضبط لون خلفية الفقرة على اللون الأحمر:

تعيين لون الخلفية لفقرة معينة:

<p style="background: #FF0000;">Some text ...</p>

رأي:

بعض النصوص ...

ضبط لون الخلفية لجميع الفقرات:

<style>
   p { background: #FF0000; }
</style>

ضبط لون حد العنصر

element { border-color: topcode rightcode bottomcode leftcode }

مثال

ضبط لون حدود الفقرة على الأحمر والأخضر والأزرق والأسود:

ضبط لون حدود فقرة معينة:

<p style="border-color: #FF0000 #00FF00 #0000FF #000000; border-style:solid">Some text ...</p>

رأي:

بعض النصوص ...

ضبط لون حدود كل الفقرات:

<style>
   p { border-color: #FF0000 #00FF00 #0000FF #000000; }
</style>
