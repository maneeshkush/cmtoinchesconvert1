إعادة توجيه HTML

إعادة توجيه HTML. كود إعادة توجيه تحديث HTML الوصفي.

إعادة توجيه تحديث HTML meta هي إعادة توجيه من جانب العميل وليست 301 إعادة توجيه دائمة.

يعتبر تحديث HTML meta بفاصل زمني 0 ثانية من قبل Google كتلميح لإعادة التوجيه 301 لنقل صفحات pagerank.

إذا كنت تريد القيام بإعادة توجيه 301 دائمة حقيقية ، فيمكنك القيام بذلك باستخدام إعادة توجيه PHP بعد تمكين كود PHP في ملفات HTML.

إعادة توجيه تحديث بيانات HTML الوصفية

تتم إعادة التوجيه باستخدام تحديث التعريف في قسم الرأس.

الرابط الموجود في قسم الجسم لأغراض احتياطية.

استبدل الصفحة القديمة بكود إعادة التوجيه بعنوان URL للصفحة التي تريد إعادة التوجيه إليها.

old-page.html:

<!DOCTYPE html>
<html>
<head>
   <!-- HTML meta refresh URL redirection -->
   <meta http-equiv="refresh"
   content="0; url=http://www.mydomain.com/new-page.html">
</head>
<body>
   <p>The page has moved to:
   <a href="http://www.mydomain.com/new-page.html">this page</a></p>
</body>
</html>

مثال على إعادة توجيه تحديث HTML الوصفي

html-redirect-test.htm:

<!DOCTYPE html>
<html>
<head>
   <!-- HTML meta refresh URL redirection -->
   <meta http-equiv="refresh"
   content="0; url=https://www.cmtoinchesconvert.com/ar/web/dev/html-redirect.htm">
</head>
<body>
   <p>The page has moved to:
   <a href="https://www.cmtoinchesconvert.com/ar/web/dev/html-redirect.htm">this page</a></p>
</body>
</html>

إعادة توجيه علامة الارتباط الأساسي بتنسيق HTML

لا يقوم الرابط الأساسي بإعادة التوجيه إلى عنوان URL المفضل ، ولكن يمكن أن يكون بديلاً لإعادة توجيه عنوان URL لمواقع الويب التي تصل معظم حركة المرور إليها من محركات البحث.

يمكن استخدام علامة الارتباط الأساسية بتنسيق HTML عندما يكون هناك العديد من الصفحات ذات المحتوى المتشابه وتريد إخبار محركات البحث بالصفحة التي تفضل استخدامها في نتائج البحث.

يمكن لعلامة الارتباط الأساسية الارتباط بالنطاق نفسه وكذلك عبر النطاق.

أضف علامة الارتباط الأساسية إلى الصفحة القديمة للربط بالصفحة الجديدة.

أضف علامة الارتباط المتعارف عليه إلى الصفحات التي تفضل عدم جذب زيارات محركات البحث إلى الصفحة المفضلة.

يجب إضافة علامة الارتباط الأساسية في قسم <head>.

old-page.html:

<link rel="canonical" href="http://www.mydomain.com/new-page.html">
