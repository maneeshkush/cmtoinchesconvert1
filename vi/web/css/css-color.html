màu CSS

Mã màu có thể là một trong:

Định dạng hex: #rrggbb

Định dạng RGB: rgb (đỏ, lục, lam)

Định dạng tên: tên

Ví dụ

Đối với màu cam:

Định dạng hex: #FFA500

Định dạng RGB: rgb(255,165,0)

Định dạng tên: cam

Đặt màu phần tử

Đối với yếu tố cụ thể:

<element style="color: code;"></element>

Đối với tất cả các phần tử cùng loại.Đặt mã vào thẻ <style> trong phần đầu:

<style>
   element { color: code; }
</style>

Ví dụ

Đặt màu văn bản đoạn thành màu đỏ:

Đặt màu của đoạn cụ thể:

<p style="color: #FF0000;">Some text ...</p>

Xem:

Một số tiếp theo ...

Đặt màu cho tất cả các đoạn văn

<style>
   p { color: #FF0000; }
</style>

Đặt màu nền phần tử

element { background: code; }

Ví dụ

Đặt màu nền của đoạn văn thành màu đỏ:

Đặt màu nền của đoạn cụ thể:

<p style="background: #FF0000;">Some text ...</p>

Xem:

Một số tiếp theo ...

Đặt màu nền của tất cả các đoạn:

<style>
   p { background: #FF0000; }
</style>

Đặt màu viền phần tử

element { border-color: topcode rightcode bottomcode leftcode }

Ví dụ

Đặt màu đường viền đoạn thành đỏ, lục, lam và đen:

Đặt màu đường viền của đoạn cụ thể:

<p style="border-color: #FF0000 #00FF00 #0000FF #000000; border-style:solid">Some text ...</p>

Xem:

Một số tiếp theo ...

Đặt màu viền cho tất cả các đoạn:

<style>
   p { border-color: #FF0000 #00FF00 #0000FF #000000; }
</style>

Các tính năng của Công cụ mã màu CSS

Công cụ mã màu CSS của chúng tôi cho phép người dùng mã màu CSS.Một số tính năng nổi bật của tiện ích này được giải thích dưới đây.

Sự đơn giản

Giao diện thân thiện với người dùng của Công cụ mã màu CSS của chúng tôi giúp bạn dễ dàng thực hiện chuyển đổi này ngay lập tức.Bạn không cần phải học các kỹ năng lập trình mã màu CSS nữa do tính đơn giản của Công cụ này.

Không đăng ký

Bạn không cần thực hiện bất kỳ quy trình đăng ký nào để sử dụng mã màu CSS.Sử dụng tiện ích này, bạn có thể sử dụng miễn phí Công cụ mã màu CSS bao nhiêu lần tùy thích.

tính di động

Mã màu CSS này có thể được truy cập từ mọi nơi trên thế giới.Bạn không cần lo lắng về khả năng tương thích của công cụ trực tuyến này vì bạn có thể sử dụng mã màu CSS với công cụ này từ bất kỳ thiết bị nào.Tất cả những gì bạn cần là kết nối internet để truy cập và sử dụng Công cụ mã màu CSS này.

Tiết kiệm thời gian và công sức

Mã màu CSS cho phép bạn hoàn thành tác vụ tương tự ngay lập tức.Bạn sẽ không được yêu cầu làm theo các quy trình thủ công vì các thuật toán tự động của nó sẽ thực hiện công việc cho bạn.

khả năng tương thích

Công cụ mã màu CSS hoạt động hoàn hảo trên tất cả các hệ điều hành.Cho dù bạn có thiết bị Mac, iOS, Android, Windows hay Linux, bạn có thể dễ dàng sử dụng tiện ích trực tuyến này mà không gặp bất kỳ rắc rối nào.

100% miễn phí
