Thay đổi logarit của quy tắc cơ sở

Để đổi cơ số từ b sang c, chúng ta có thể sử dụng quy tắc đổi cơ số logarit.Lôgarit cơ số b của x bằng lôgarit cơ số c của x chia cho lôgarit cơ số c của b:

logb(x) = logc(x) / logc(b)

Ví dụ 1

log2(100) = log10(100) / log10(2) = 2 / 0.30103 = 6.64386

Ví dụ #2

log3(50) = log8(50) / log8(3) = 1.8812853 / 0.5283208 = 3.5608766

Bằng chứng

Nâng b với lũy thừa logarit cơ số b của x cho x:

(1) x = blogb(x)

Nâng c với lũy thừa logarit cơ số c của b cho b:

(2) b = clogc(b)

Khi chúng ta lấy (1) và thay thế b bằng c log c ( b ) (2), chúng ta có:

(3) x = blogb(x) = (clogc(b))logb(x) = clogc(b)×logb(x)

Bằng cách áp dụng log c () vào cả hai vế của (3):

logc(x) = logc(clogc(b)×logb(x))

Bằng cách áp dụng quy tắc lũy thừa logarit :

logc(x) = [logc(b)×logb(x)] × logc(c)

Vì log c ( c )=1

logc(x) = logc(b)×logb(x)

Hoặc

logb(x) = logc(x) / logc(b)
