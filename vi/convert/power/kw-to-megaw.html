Công thức chuyển đổi Kilowatt sang MW

Vì vậy, Công suất tính bằng megawatt P (MW)  bằng công suất tính bằng kilowatt P (kW ) chia cho [1000].

P(MW) = P(kW) / 1000

ví dụ 1

Chuyển đổi 2kW sang MW:

P(MW) = 2kW / 1000 = 0.002MW

ví dụ 2

Chuyển đổi 3kW sang MW:

P(MW) = 3kW / 1000 = 0.003MW

ví dụ 3

Chuyển đổi 4kW sang MW:

P(MW) = 4kW / 1000 = 0.004MW

Ví dụ 4

Chuyển đổi 6kW sang MW:

P(MW) = 6kW / 1000 = 0.006MW

Ví dụ 5

Chuyển đổi 8kW sang MW:

P(MW) = 8kW / 1000 = 0.008MW
