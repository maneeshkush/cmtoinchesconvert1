độ F

Fahrenheit là đơn vị đo nhiệt độ.

Điểm đóng băng/nóng chảy của nước ở áp suất 1 atm là khoảng 32 °F.

Biểu tượng cho độ Fahrenheit là °F.

Chuyển đổi độ F sang độ C

Vậy 0 độ F bằng -17,77778 độ C.

0 °F = -17.77778 °C

Vì vậy, nhiệt độ T  tính bằng độ  C (°C)  bằng với nhiệt độ  T tính bằng độ F (°F) trừ đi 32, lần [5/9].

T(°C) = (T(°F) - 32) × 5/9

ví dụ 1

Chuyển đổi 70 độ F sang độ C:

T(°C) = (70°F - 32) × 5/9 = 21.1111 °C

ví dụ 2

Chuyển đổi 80 độ F sang độ C:

T(°C) = (80°F - 32) × 5/9 = 26.6667 °C

ví dụ 3

Chuyển đổi 100 độ F sang độ C:

T(°C) = (100°F - 32) × 5/9 = 37.7778 °C

Chuyển đổi Fahrenheit sang Rankine

Vậy 0 độ F bằng [459,67] độ Rankine.

0 °F = 459.67 °R

Nhiệt độ  T  tính theo độ Rankine (°R) bằng nhiệt độ  T tính theo độ F (°F) cộng với [459,67].

T(°R) = T(°F) + 459.67

ví dụ 1

Chuyển đổi 70 độ F sang độ Rankine:

T(°R) = 70°F + 459.67 = 529.67 °R

ví dụ 2

Chuyển đổi 90 độ F sang độ Rankine:

T(°R) = 90°F + 459.67 = 549.67 °R

ví dụ 3

Chuyển đổi 100 độ F sang độ Rankine:

T(°R) = 100°F + 459.67 = 559.67 °R

Chuyển đổi Fahrenheit sang Kelvin

Vì vậy, nhiệt độ T  tính bằng  Kelvin (K)  bằng với nhiệt độ  T tính bằng độ F (°F) cộng 459,67, lần [5/9].

T(K) = (T(°F) + 459.67)× 5/9

ví dụ 1

Chuyển đổi 65 độ F sang độ Kelvin:

T(K) = (65°F + 459.67)× 5/9 = 291.483 K

ví dụ 2

Chuyển đổi 80 độ F sang độ Kelvin:

T(K) = (80°F + 459.67)× 5/9 = 299.817 K

ví dụ 3

Chuyển đổi 100 độ Fahrenheit sang độ Kelvin:

T(K) = (100°F + 459.67)× 5/9 = 310.928 K
