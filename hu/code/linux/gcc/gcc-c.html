gcc -c opció jelzője

A gcc -c hivatkozás nélkül fordítja le a forrásfájlokat.

Szintaxis

$ gcc -c [options] [source files]

Példa

Írja be a myfile.c forrásfájlt :

// myfile.c
#include <stdio.h>
 
void main()
{
    printf("Program run\n");
}

 

Fordítsa le a myfile.c fájlt :

$ gcc -c myfile.c

Ez az összeállítás létrehozta a myfile.o objektumfájlt.
