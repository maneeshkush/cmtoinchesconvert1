Javascript átirányítási URL

URL-oldal átirányítása Javascript segítségével.

A Javascript-átirányítás nem ad vissza 301-es állandó átirányítási állapotkódot.

Javascript átirányítás

Cserélje le a régi oldalt átirányítási kóddal annak az oldalnak az URL-jével, amelyre át szeretné irányítani.

old-page.html:

<!DOCTYPE html>
<html>
<body>
<script type="text/javascript">
    // Javascript URL redirection
    window.location.replace("http://www.mydomain.com/new-page.html");
</script>
</body>
</html>

 

A keresőmotorok a 301-es állapotkódot használják az oldal rangsorának átvitelére a régi URL-ről az új URL-re.

Javascript-átirányítás visszaküldi a http válasz állapotkódját: 200 OK.

Tehát a Javascript átirányítás nem keresőbarát, és jobb más átirányítási módszereket használni, amelyek állapotkódot adnak vissza: 301 Véglegesen áthelyezve.

Javascript átirányítási példa

javascript-redirect-test.htm

<!DOCTYPE html>
<html>
<body>
<script type="text/javascript">
// Javascript URL redirection
window.location.replace("https://www.cmtoinchesconvert.com/hu/web/dev/javascript-redirect.htm");
</script>
</body>
</html>
