HTML link új ablakban

Hivatkozás megnyitása új ablakban vagy új lapon.

Nyisson meg egy hivatkozást új ablakban vagy lapon

Ha egy linket új ablakban/lapon szeretne megnyitni, adja hozzá a target="_blank" elemet az <a> címkéhez:

<a href="../html-link.htm" target="_blank">Open page in new window</a>

A kód ezt a linket hozza létre:

Oldal megnyitása új ablakban

Új ablak vagy új lap

Nem állíthatja be, hogy a hivatkozás új ablakban vagy új lapon nyíljon meg.Ez a böngésző beállításaitól függ. 

Nyissa meg a hivatkozást egy új ablakban megadott mérettel

Ha egy hivatkozást új ablakban szeretne megnyitni, adja hozzá az onclick="window.open('text-link.htm', 'name','width=600,height=400') Javascript parancsot az <a> címkén belülre:

<a href="../html-link.htm" target="popup" onclick="window.open('../html-link.htm','name','width=600,height=400')">Open page in new window</a>

A kód ezt a linket hozza létre:
