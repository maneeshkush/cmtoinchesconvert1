CSS navigációs sáv

CSS navigációs sáv - vízszintes és függőleges.

Vízszintes navigációs sáv

CSS kód:

<style>
    #h_nav_bar a { padding:15px; font-weight:bold; float:left; }
    #h_nav_bar a:link { color:#d0d0d0; background-color:#0000a0; }
    #h_nav_bar a:visited { color:#c0c0c0; background-color:#0000a0; }
    #h_nav_bar a:hover { color:#ffffff; background-color:#000060; }
    #h_nav_bar a:active { color:#f0f0f0; background-color:#00ff00; }
</style>

HTML kód:

<div id="h_nav_bar">
    <a href="html-anchor-link.htm">Anchor link</a>
    <a href="html-link-color.htm">Link color</a>
    <a href="../mailto.htm">Email link</a>
    <a href="html-image-link.htm">Image link</a>
    <a href="html-text-link.htm">Text link</a>
</div>

Kilátás:

Horgony link Link színe E-mail link Kép linkje Szöveges hivatkozás

 

 

 

A #link_bar a a hivatkozás összes állapotának stílusa.

#link_bar a:link a normál hivatkozás stílusa.

#link_bar a: A látogatott a meglátogatott hivatkozás stílusa.

#link_bar a:hover az egérmutató hivatkozás stílusa.

#link_bar a: az aktív a hivatkozás stílusa, amikor megnyomja az egérrel.

A CSS Navigation Bar eszköz jellemzői

A CSS navigációs sáv eszközünk lehetővé teszi a felhasználók számára a CSS navigációs sáv használatát.Az alábbiakban ismertetjük a segédprogram néhány kiemelkedő funkcióját.

Egyszerűség

A CSS Navigációs eszköztárunk felhasználóbarát felülete lehetővé teszi az átalakítás azonnali végrehajtását.Az eszköz egyszerűsége miatt többé nem kell programozási ismereteket szereznie a CSS navigációs sávhoz.

Nincs regisztráció

A CSS navigációs sáv használatához nem kell regisztrációs folyamaton keresztülmennie.Ezzel a segédprogrammal annyiszor használhatja a CSS Navigation Bar Tool eszközt, ahányszor csak akarja ingyen.

Hordozhatóság

Ez a CSS navigációs sáv a világ bármely sarkából elérhető.Nem kell aggódnia ennek az online eszköznek a kompatibilitása miatt, mivel bármilyen eszközről CSS navigációs sávot használhat vele.Csak internetkapcsolatra van szüksége a CSS navigációs eszköztár eléréséhez és használatához.

Időt és erőfeszítést takarít meg

A CSS navigációs sáv lehetővé teszi, hogy ugyanazt a feladatot azonnal elvégezze.Nem kell manuális eljárásokat követnie, mivel az automatizált algoritmusok elvégzik a munkát Ön helyett.

Kompatibilitás

A CSS Navigation Bar Tool tökéletesen működik minden operációs rendszeren.Akár Mac-, iOS-, Android-, Windows- vagy Linux-eszközzel rendelkezik, könnyedén, gond nélkül használhatja ezt az online segédprogramot.

100% ingyenes

A CSS navigációs eszköztár használatához nem kell regisztrációs folyamaton keresztülmennie.Ezt a segédprogramot ingyenesen használhatja, és korlátlanul használhatja a CSS navigációs sávot korlátozások nélkül.
