Huzalmérő számítások

Huzalátmérő számítások

Az n méretű huzalátmérő d n hüvelykben (inch) egyenlő 0,005 hüvelyk 92-vel 36 mínusz n mérőszám hatványára emelve, osztva 39-el:

dn (in) = 0.005 in × 92(36-n)/39

Az n méretű huzalátmérő d n milliméterben (mm) egyenlő 0,127 mm-rel szorozva 92-vel, 36 mínusz n mérőszám hatványára emelve, osztva 39-cel:

dn (mm) = 0.127 mm × 92(36-n)/39

Vezeték keresztmetszeti terület számítások

Az n méretű vezeték A n keresztmetszete kilo-körmilben ( kcmil ) egyenlő a d négyzethuzal-átmérő 1000-szeresével hüvelykben (inch):

An (kcmil) = 1000×dn2 = 0.025 in2 × 92(36-n)/19.5

Tehát az n méretű vezeték A n négyzethüvelykben (in 2 ) megadott keresztirányú területeegyenlő pi-vel osztva a négyzetes huzal d hüvelykben (inch) négyzetes átmérőjével.

An (in2) = (π/4)×dn2 = 0.000019635 in2 × 92(36-n)/19.5

Tehát az n méretű huzal keresztmetszete A n négyzetmilliméterben (mm 2 ) egyenlő pi-vel osztva a négyzetes huzal d milliméterben (mm) megadott átmérőjével.

An (mm2) = (π/4)×dn2 = 0.012668 mm2 × 92(36-n)/19.5

Vezetékellenállás számítások

Tehát az n méretű vezeték ellenállása ohm/kilolábban (Ω/kft) egyenlő a huzal ρ ellenállásának ohmméterben ( Ω·m) 0,3048 × 1000000000-szerese osztvaa négyzethüvelykben megadott An keresztmetszeti terület 25,4-szeresével . (2 -ben ).

Rn (Ω/kft) = 0.3048 × 109 × ρ(Ω·m) / (25.42 × An (in2))

Tehát az n jelű vezetékellenállás ohm/km-ben (Ω/km) egyenlő a huzal ohmméterben (Ω·m) mért ρ ellenállásának 1000000000-szerese osztvaa négyzetmilliméterben (mm2 ) megadott An keresztmetszeti területtel.

Rn (Ω/km) = 109 × ρ(Ω·m) / An (mm2)
