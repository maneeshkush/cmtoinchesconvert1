jQuery-uudelleenohjaus

Sivun uudelleenohjaaminen URL-osoitteeseen jQuerylla.

Hakukoneet käyttävät 301-tilakoodia sivusijoituksen siirtämiseen vanhasta URL-osoitteesta uuteen URL-osoitteeseen.

jQuery-uudelleenohjaus palauttaa http-vastauksen tilakoodin: 200 OK.

Joten jQuery-uudelleenohjaus, kuten Javascript-uudelleenohjaus, ei ole hakukoneystävällinen, ja on parempi käyttää muita uudelleenohjausmenetelmiä, jotka palauttavat tilakoodin 301 Siirretty pysyvästi.

jQuery-uudelleenohjaus on itse asiassa Javascript-uudelleenohjaus . Uudelleenohjausta jQuerylla pidetään ylivoimaisena, koska voit tehdä puhtaan Javascript-uudelleenohjauksen vähemmällä koodilla.

jQuery-uudelleenohjaus

Korvaa vanha sivu uudelleenohjauskoodilla sen sivun URL-osoitteella, jolle haluat ohjata.

vanha-sivu.html:

<!DOCTYPE html>
<html>
<body>
<script src="http://ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script>
<script type ="teksti/javascript">
   // jQuery URL-uudelleenohjaus
   $(document).ready( function() {
      url = "http://www.omaverkkotunnus.fi/uusi-sivu.html";
      $( sijainti ).attr( "href", url);
   });
</script>
</body>
</html>

Esimerkki jQuery-uudelleenohjauksesta

jquery-redirect-test.htm

<!DOCTYPE html>
<html>
<body>
<script src="//ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script>
<script type="text/javascript">
   $(document).ready( function() {
      url = "https://www.cmtoinchesconvert.com/fi/web/dev/jquery-redirect.htm";
      $(location).attr("href", url);
   });
</script>
</body>
</html>
