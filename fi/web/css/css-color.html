CSS-väri

Värikoodi voi olla jokin seuraavista:

Hex-muoto: #rrggbb

RGB-muoto: rgb (punainen, vihreä, sininen)

Nimen muoto: nimi

Esimerkki

Oranssille värille:

Hex-muoto: #FFA500

RGB-muoto: rgb(255,165,0)

Nimen muoto: oranssi

Elementin värin asetus

Tietylle elementille:

<element style="color: code;"></element>

Kaikille samantyyppisille elementeille. Laita koodi head-osion <style>-tunnisteeseen:

<style>
   element { color: code; }
</style>

Esimerkki

Kappaleen tekstin värin asettaminen punaiseksi:

Tietyn kappaleen värin asettaminen:

<p style="color: #FF0000;">Some text ...</p>

Näytä:

Jotain tekstiä...

Kaikkien kappaleiden värin asettaminen

<style>
   p { color: #FF0000; }
</style>

Asetuselementin taustaväri

element { background: code; }

Esimerkki

Kappaleen taustavärin asettaminen punaiseksi:

Tietyn kappaleen taustavärin asettaminen:

<p style="background: #FF0000;">Some text ...</p>

Näytä:

Jotain tekstiä...

Kaikkien kappaleiden taustavärin asettaminen:

<style>
   p { background: #FF0000; }
</style>

Asetetaan elementin reunan väri

element { border-color: topcode rightcode bottomcode leftcode }

Esimerkki

Kappaleen reunan värin asettaminen punaiseksi, vihreäksi, siniseksi ja mustaksi:

Tietyn kappaleen reunavärin asettaminen:

<p style="border-color: #FF0000 #00FF00 #0000FF #000000; border-style:solid">Some text ...</p>

Näytä:

Jotain tekstiä...

Kaikkien kappaleiden reunavärin asettaminen:

<style>
   p { border-color: #FF0000 #00FF00 #0000FF #000000; }
</style>

CSS-värikoodityökalun ominaisuudet

CSS-värikoodityökalumme antaa käyttäjille mahdollisuuden CSS-värikoodeihin. Joitakin tämän apuohjelman merkittäviä ominaisuuksia selitetään alla.

Yksinkertaisuus

CSS-värikoodityökalumme käyttäjäystävällinen käyttöliittymä tarjoaa sinulle helpon muuntamisen välittömästi. Sinun ei enää tarvitse hankkia ohjelmointitaitoja CSS-värikoodeja varten tämän työkalun yksinkertaisuuden vuoksi.

Ei rekisteröintiä

Sinun ei tarvitse käydä läpi mitään rekisteröintiprosessia käyttääksesi CSS-värikoodeja. Tämän apuohjelman avulla voit CSS-värikoodityökalulla niin monta kertaa kuin haluat ilmaiseksi.

Siirrettävyys

Nämä CSS-värikoodit ovat käytettävissä mistä tahansa maailman kolkasta. Sinun ei tarvitse huolehtia tämän online-työkalun yhteensopivuudesta, sillä voit CSS-värikoodeja käyttää sillä miltä tahansa laitteelta. Tarvitset vain Internet-yhteyden tämän CSS-värikoodityökalun käyttämiseen ja käyttämiseen.

Säästää aikaa ja vaivaa

CSS-värikoodien avulla voit suorittaa saman tehtävän välittömästi. Sinua ei pyydetä noudattamaan manuaalisia toimenpiteitä, koska sen automatisoidut algoritmit tekevät työn puolestasi.

Yhteensopivuus

CSS-värikoodityökalu toimii täydellisesti kaikissa käyttöjärjestelmissä. Olipa sinulla Mac-, iOS-, Android-, Windows- tai Linux-laite, voit käyttää tätä verkkoapuohjelmaa helposti ilman ongelmia.

100% ilmainen

Sinun ei tarvitse käydä läpi mitään rekisteröintiprosessia käyttääksesi tätä CSS-värikoodityökalua. Voit käyttää tätä apuohjelmaa ilmaiseksi ja tehdä rajattomasti CSS-värikoodeja ilman rajoituksia.
