Hidas VirtualBox Ubuntu

Hidas Ubuntun toiminta VirtualBoxissa.

Ongelma

Ubuntu 13.04 VirtualBoxissa toimii hyvin hitaasti.

Ikkunat avautuvat tai sulkeutuvat hyvin hitaasti.

Syy

Ongelma johtuu Ubuntuun 12.04-jakelun jälkeen kuuluvan uuden Unity-työpöydän hitaasta toiminnasta.

Ratkaisu

  1. Avaa VirtualBox Manager.
  2. Päivitä VirtualBox uusimpaan versioon Ohje > Tarkista päivitykset -valikosta.
  3. Paina Asetukset-työkalurivin painiketta.
  4. Aseta Video-välilehden Laajennetut ominaisuudet -valintaruutu Ota käyttöön 3D-kiihdytys.
  5. Käynnistä Ubuntu uudelleen.

 

Voit myös tarkistaa, että laitteiston virtualisointi on otettu käyttöön tietokoneen BIOS-asetuksista.
