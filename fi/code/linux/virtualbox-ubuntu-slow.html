Hidas VirtualBox Ubuntu

Hidas Ubuntun toiminta VirtualBoxissa.

Ongelma

Ubuntu 13.04 VirtualBoxissa toimii hyvin hitaasti.

Ikkunat avautuvat tai sulkeutuvat hyvin hitaasti.

Syy

Ongelma johtuu Ubuntuun 12.04-jakelun jlkeen kuuluvan uuden Unity-typydn hitaasta toiminnasta.

Ratkaisu

  1. Avaa VirtualBox Manager.
  2. Pivit VirtualBox uusimpaan versioon Ohje > Tarkista pivitykset -valikosta.
  3. Paina Asetukset-tykalurivin painiketta.
  4. Aseta Video-vlilehden Laajennetut ominaisuudet -valintaruutu Ota kyttn 3D-kiihdytys.
  5. Kynnist Ubuntu uudelleen.

Voit mys tarkistaa, ett laitteiston virtualisointi on otettu kyttn tietokoneen BIOS-asetuksista.
