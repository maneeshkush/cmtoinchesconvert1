ls-komento Linuxissa/Unixissa

ls on Linuxin komentotulkkikomento, joka luettelee tiedostojen ja hakemistojen hakemistosisällöt.

  • ls syntaksi
  • Vaihtoehdot

ls syntaksi

$ ls [options] [file|dir]

ls-komentovaihtoehdot

ls-komennon päävaihtoehdot:

vaihtoehto kuvaus
ls -a luettele kaikki tiedostot, mukaan lukien piilotetut tiedostot, jotka alkavat kirjaimella "."
ls -- väri värillinen luettelo [=aina/ei koskaan/automaattisesti]
ls -d luettelo hakemistoista - ' */'
ls -F lisää yksi merkki */=>@| sisääntuloihin
ls -i listatiedoston inode-indeksinumero
ls -l luettelo pitkällä muodossa - näyttää käyttöoikeudet
ls-la luettelo pitkässä muodossa, mukaan lukien piilotetut tiedostot
ls -lh luettelon pitkä muoto luettavalla tiedostokoolla
ls -ls luettelo, jossa on pitkä muoto ja tiedostokoko
ls -r luettelo käänteisessä järjestyksessä
ls -R listaa rekursiivisesti hakemistopuu
ls -s listatiedoston koko
ls -S lajitella tiedostokoon mukaan
ls -t lajitella ajan ja päivämäärän mukaan
ls -X lajitella laajennuksen nimen mukaan

ls komento esimerkkejä

Voit täydentää tiedostojen tai kansioiden nimet automaattisesti painamalla sarkainpainiketta .

Listaa asiakirjat/kirjat hakemistoon suhteellisella polulla :

$ ls Documents/Books

 

Listaa hakemisto /home/user/Documents/Books absoluuttisella polulla .

$ ls /home/user/Documents/Books

 

Listaa juurihakemisto:

$ ls /

 

Luettelo ylätason hakemistosta:

$ ls ..

 

Listaa käyttäjän kotihakemisto (esim. /home/user):

$ ls ~

 

Lista pitkällä muodossa:

$ ls -l

 

Näytä piilotetut tiedostot:

$ ls -a

 

Luettelo pitkällä muodossa ja näytä piilotetut tiedostot:

$ ls -la

 

Lajittele päivämäärän/ajan mukaan:

$ ls -t

 

Lajittele tiedostokoon mukaan:

$ ls -S

 

Listaa kaikki alihakemistot:

$ ls *

 

Rekursiivinen hakemistopuuluettelo:

$ ls -R

 

Luettele vain tekstitiedostot jokerimerkillä:

$ ls *.txt

 

ls uudelleenohjaus tulostiedostoon:

$ ls > out.txt

 

Listaa vain hakemistot:

$ ls -d */

 

Listaa tiedostot ja hakemistot täydellä polulla:

$ ls -d $PWD/*
