Kuinka muuntaa asteet radiaaneiksi

Asteista radiaaneiksi muunnoskaava

Yksi aste on yhtä suuri kuin 0,01745329252 radiaania:

1° = π/180° = 0.005555556π = 0.01745329252 rad

Joten kulma α radiaaneina on yhtä suuri kuin kulma α asteina kertaa pi vakio jaettuna 180 astetta:

α(radians) = α(degrees) × π / 180°

tai

radians = degrees × π / 180°

Esimerkki 1

Muunna 25 asteen kulma radiaaneiksi:

α(radians) = α(degrees) × π / 180° = 25° × 3.14159 / 180° = 0.436332313 rad

Esimerkki 2

Muunna 50 asteen kulma radiaaneiksi:

α(radians) = α(degrees) × π / 180° = 50° × 3.14159 / 180° = 0.872664626 rad

Esimerkki 3

Muunna 90 asteen kulma radiaaneiksi:

α(radians) = α(degrees) × π / 180° = 90° × 3.14159 / 180° = 1.5707963268 rad

Asteesta radiaaneihin pi-arvona

Joten kulma α radiaaneina on yhtä suuri kuin kulma α asteina kertaa pi vakio jaettuna [180] astetta.

α(radians) = (α(degrees) / 180°) × π

Esimerkki 1

Muunna 40 asteen kulma radiaaneiksi pi:llä:

α(radians) = (α(degrees) / 180°) × π = (40° / 180°) × π

 = (1/6) × π = π/6 rad = 0.22222222222π rad

Esimerkki 2

Muunna 90 asteen kulma radiaaneiksi pi:llä:

α(radians) = (α(degrees) / 180°) × π = (90° / 180°) × π

 = (1/6) × π = π/6 rad = 0.5π rad

Esimerkki 3

Muunna 120 asteen kulma radiaaneiksi pi:llä:

α(radians) = (α(degrees) / 180°) × π = (120° / 180°) × π

 = (1/6) × π = π/6 rad = 0.66666666667π rad
