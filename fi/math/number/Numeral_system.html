d n - n:s numero

n - voi alkaa negatiivisesta luvusta, jos luvussa on murto-osa.

N +1 - numeroiden lukumäärä

Binäärinen numerojärjestelmä - Base-2

Binääriluvuissa käytetään vain numeroita 0 ja 1.

B tarkoittaa binäärietuliitettä.

Esimerkkejä:

101012 = 10101B = 1×24+0×23+1×22+0×21+1×20 = 16+4+1= 21

101112 = 10111B = 1×24+0×23+1×22+1×21+1×20 = 16+4+2+1= 23

1000112 = 100011B = 1×25+0×24+0×23+0×22+1×21+1×20 =32+2+1= 35

Octal Numeral System - Base-8

Oktaaliluvuissa käytetään numeroita 0...7.

Esimerkkejä:

278 = 2×81+7×80 = 16+7 = 23

308 = 3×81+0×80 = 24

43078 = 4×83+3×82+0×81+7×80= 2247

Desimaalilukujärjestelmä – kanta-10

Desimaaliluvuissa käytetään numeroita 0...9.

Nämä ovat tavallisia käyttämiämme numeroita.

Esimerkki:

253810 = 2×103+5×102+3×101+8×100

Heksadesimaalilukujärjestelmä – kanta-16

Hex-luvut käyttävät numeroita 0..9 ja A..F.

H tarkoittaa heksa-etuliitettä.

Esimerkkejä:

2816 = 28H = 2×161+8×160 = 40

2F16 = 2FH = 2×161+15×160 = 47

BC1216 = BC12H = 11×163+12×162+1×161+2×160= 48146
