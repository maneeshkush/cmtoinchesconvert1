Negatiiviset eksponentit hallitsevat

Kanta b korotettuna potenssiin miinus n on yhtä suuri kuin 1 jaettuna kantalla b korotettuna potenssiin n:

b-n = 1 / bn

Esimerkki negatiivista eksponenttia

Kanta 2 korotettuna potenssiin miinus 3 on yhtä suuri kuin 1 jaettuna kanta 2:lla, joka on korotettu potenssiin 3:

2-3 = 1/23 = 1/(2⋅2⋅2) = 1/8 = 0.125

Negatiiviset murtolukueksponentit

Kanta b korotettuna potenssiin miinus n/m on yhtä suuri kuin 1 jaettuna kantalla b korotettuna potenssiin n/m:

b-n/m = 1 / bn/m = 1 / (m√b)n

Kanta 2 nostettuna potenssiin miinus 1/2 on yhtä suuri kuin 1 jaettuna kantalla 2 korotettuna potenssiin 1/2:

2-1/2 = 1/21/2 = 1/√2 = 0.7071

Murtoluvut negatiivisilla eksponenteilla

Kanta a/b korotettuna potenssiin miinus n on yhtä suuri kuin 1 jaettuna kantalla a/b korotettuna potenssiin n:

(a/b)-n = 1 / (a/b)n = 1 / (an/bn) = bn/an

Kanta 2 korotettuna potenssiin miinus 3 on yhtä suuri kuin 1 jaettuna kanta 2:lla, joka on korotettu potenssiin 3:

(2/3)-2 = 1 / (2/3)2 = 1 / (22/32) = 32/22 = 9/4 = 2.25

Negatiivisten eksponentien kertominen

Eksponenteille, joilla on sama kanta, voimme lisätä eksponentit:

a -n ⋅ a -m = a -(n+m) = 1 / a n+m

Esimerkki:

2-3 ⋅ 2-4 = 2-(3+4) = 2-7 = 1 / 27 = 1 / (2⋅2⋅2⋅2⋅2⋅2⋅2) = 1 / 128 = 0.0078125

 

Kun kannat ovat erilaisia ​​ja a:n ja b:n eksponentit ovat samat, voimme kertoa a:n ja b:n ensin:

a -n ⋅ b -n = (a ⋅ b) -n

Esimerkki:

3-2 ⋅ 4-2 = (3⋅4)-2 = 12-2 = 1 / 122 = 1 / (12⋅12) = 1 / 144 = 0.0069444

 

Kun kantakannat ja eksponentit ovat erilaiset, meidän on laskettava jokainen eksponentti ja kerrottava sitten:

a -n ⋅ b -m

Esimerkki:

3-2 ⋅ 4-3 = (1/9) ⋅ (1/64) = 1 / 576 = 0.0017361

 

Negatiivisten eksponentien jakaminen

Eksponenteille, joilla on sama kanta, meidän tulee vähentää eksponentit:

a n / a m = a n-m

Esimerkki:

26 / 23 = 26-3 = 23 = 2⋅2⋅2 = 8

 

Kun kannat ovat erilaisia ​​ja a:n ja b:n eksponentit ovat samat, voimme jakaa a:n ja b:n ensin:

a n / b n = (a / b) n

Esimerkki:

63 / 23 = (6/2)3 = 33 = 3⋅3⋅3 = 27

 

Kun kantakannat ja eksponentit ovat erilaiset, meidän on laskettava jokainen eksponentti ja jaettava sitten:

a n / b m

Esimerkki:

62 / 33 = 36 / 27 = 1.333
