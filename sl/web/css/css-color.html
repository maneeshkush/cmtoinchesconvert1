Barva CSS

Barvna koda je lahko ena od naslednjih:

Šestnajstiška oblika: #rrggbb

RGB format: rgb (rdeča, zelena, modra)

Oblika imena: ime

Primer

Za oranžno barvo:

Šestnajstiška oblika: #FFA500

Format RGB: rgb(255,165,0)

Oblika imena: oranžna

Nastavitev barve elementa

Za določen element:

<element style="color: code;"></element>

Za vse elemente iste vrste.Vstavite kodo v oznako <style> v razdelku glave:

<style>
   element { color: code; }
</style>

Nastavitev barve določenega odstavka:
