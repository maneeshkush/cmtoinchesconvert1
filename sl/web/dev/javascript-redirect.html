Javascript preusmeritveni URL

Kako preusmeriti stran URL z Javascriptom.

Preusmeritev Javascript ne vrne kode stanja trajne preusmeritve 301.

Preusmeritev Javascript

Zamenjajte staro stran s kodo za preusmeritev z URL-jem strani, na katero želite preusmeriti.

stara-stran.html:

<!DOCTYPE html>
<html>
<body>
<script type="text/javascript">
    // Javascript URL redirection
    window.location.replace("http://www.mydomain.com/new-page.html");
</script>
</body>
</html>

 

Iskalniki uporabljajo statusno kodo 301 za prenos uvrstitve strani s starega URL-ja na nov URL.

Preusmeritev Javascript vrne statusno kodo odziva http: 200 OK.

Preusmeritev Javascript torej ni prijazna do iskalnikov in bolje je uporabiti druge metode preusmeritve, ki vrnejo statusno kodo: 301 Moved Permanently.
