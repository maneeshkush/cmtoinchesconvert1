PHP preusmeritev

PHP preusmeritev s strani na URL.PHP 301 preusmeritev.

Ta preusmeritev PHP bi morala vrniti statusno kodo odziva HTTP: 301 Trajno premaknjeno.

Iskalniki uporabljajo statusno kodo odgovora 301 za prenos uvrstitve strani s starega URL-ja na nov URL.

Preusmeritev glave PHP

Zamenjajte kodo old-page.php s kodo za preusmeritev na new-page.php.

stara-stran.php:

<?php
// PHP permanent URL redirection
header("Location: http://www.domain.com/new-page.php", true, 301);
exit();
?>

Stara stran mora imeti pripono datoteke .php.

Nova stran je lahko s poljubno razširitvijo.

Primeri preusmeritev PHP

Primer #1

php-redirect-test.php

<?php
// PHP permanent URL redirection test
header("Location: https://www.cmtoinchesconvert.com/sl/web/dev/php-redirect.html", true, 301);
exit();
?>

 

Pritisnite to povezavo za preusmeritev iz php-redirect-test.php nazaj na to stran:

 

Preizkus preusmeritve PHP - datoteka PHP

Primer #2

php-redirect-test.htm

<?php
// PHP permanent URL redirection test
header("Location: https://www.cmtoinchesconvert.com/sl/web/dev/php-redirect.html", true, 301);
exit();
?>

 

Preusmeritev PHP iz datoteke html php-redirect-test.htm običajno ne bo delovala zaradi pripone datoteke .html, razen če je omogočena v datoteki .htaccess ali httpd.conf:

 

Preizkus preusmeritve PHP - datoteka HTML

 

Če želite omogočiti PHP v datotekah HTML, dodajte to kodo v datoteko .htaccess ali httpd.conf:

Addtype application/x-httpd-php .htm .html
