Javascript ಮರುನಿರ್ದೇಶನ URL

ಜಾವಾಸ್ಕ್ರಿಪ್ಟ್‌ನೊಂದಿಗೆ URL ಪುಟವನ್ನು ಮರುನಿರ್ದೇಶಿಸುವುದು ಹೇಗೆ.

Javascript ಮರುನಿರ್ದೇಶನವು 301 ಶಾಶ್ವತ ಮರುನಿರ್ದೇಶನ ಸ್ಥಿತಿ ಕೋಡ್ ಅನ್ನು ಹಿಂತಿರುಗಿಸುವುದಿಲ್ಲ.

ಜಾವ್‌ಸ್ಕ್ರಿಪ್ಟ್ ಮರುನಿರ್ದೇಶನ

ನೀವು ಮರುನಿರ್ದೇಶಿಸಲು ಬಯಸುವ ಪುಟದ URL ನೊಂದಿಗೆ ಮರುನಿರ್ದೇಶನ ಕೋಡ್‌ನೊಂದಿಗೆ ಹಳೆಯ ಪುಟವನ್ನು ಬದಲಾಯಿಸಿ.

old-page.html:

<!DOCTYPE html>
<html>
<body>
<script type="text/javascript">
    // Javascript URL redirection
    window.location.replace("http://www.mydomain.com/new-page.html");
</script>
</body>
</html>

 

ಹಳೆಯ URL ನಿಂದ ಹೊಸ URL ಗೆ ಪುಟ ಶ್ರೇಣಿಯನ್ನು ವರ್ಗಾಯಿಸಲು ಹುಡುಕಾಟ ಇಂಜಿನ್‌ಗಳು 301 ಸ್ಥಿತಿ ಕೋಡ್ ಅನ್ನು ಬಳಸುತ್ತವೆ.

Javascript ಮರುನಿರ್ದೇಶನ ಹಿಂತಿರುಗಿಸುವಿಕೆ http ಪ್ರತಿಕ್ರಿಯೆ ಸ್ಥಿತಿ ಕೋಡ್: 200 ಸರಿ.

ಆದ್ದರಿಂದ ಜಾವಾಸ್ಕ್ರಿಪ್ಟ್ ಮರುನಿರ್ದೇಶನವು ಹುಡುಕಾಟ ಎಂಜಿನ್ ಸ್ನೇಹಿಯಾಗಿಲ್ಲ ಮತ್ತು ಸ್ಥಿತಿ ಕೋಡ್ ಅನ್ನು ಹಿಂದಿರುಗಿಸುವ ಇತರ ಮರುನಿರ್ದೇಶನ ವಿಧಾನಗಳನ್ನು ಬಳಸುವುದು ಉತ್ತಮ: 301 ಶಾಶ್ವತವಾಗಿ ಸರಿಸಲಾಗಿದೆ.

ಜಾವಾಸ್ಕ್ರಿಪ್ಟ್ ಮರುನಿರ್ದೇಶನ ಉದಾಹರಣೆ

<html>
<body>
<script type="text/javascript">
// Javascript URL redirection
window.location.replace("https://www.cmtoinchesconvert.com/kn/web/dev/javascript-redirect.htm");
</script>
</body>
</html>
