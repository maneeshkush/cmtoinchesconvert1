CSS ಬಣ್ಣ

ಬಣ್ಣ ಕೋಡ್ ಇವುಗಳಲ್ಲಿ ಒಂದಾಗಿರಬಹುದು:

ಹೆಕ್ಸ್ ಫಾರ್ಮ್ಯಾಟ್: #rrggbb

RGB ಸ್ವರೂಪ: rgb (ಕೆಂಪು, ಹಸಿರು, ನೀಲಿ)

ಹೆಸರಿನ ಸ್ವರೂಪ: ಹೆಸರು

ಉದಾಹರಣೆ

ಕಿತ್ತಳೆ ಬಣ್ಣಕ್ಕಾಗಿ:

ಹೆಕ್ಸ್ ಫಾರ್ಮ್ಯಾಟ್: #FFA500

RGB ಸ್ವರೂಪ: rgb(255,165,0)

ಹೆಸರಿನ ಸ್ವರೂಪ: ಕಿತ್ತಳೆ

ಅಂಶದ ಬಣ್ಣವನ್ನು ಹೊಂದಿಸಲಾಗುತ್ತಿದೆ

ನಿರ್ದಿಷ್ಟ ಅಂಶಕ್ಕಾಗಿ:

<element style="color: code;"></element>

ಒಂದೇ ರೀತಿಯ ಎಲ್ಲಾ ಅಂಶಗಳಿಗೆ.ಹೆಡ್ ವಿಭಾಗದಲ್ಲಿ <style> ಟ್ಯಾಗ್‌ನಲ್ಲಿ ಕೋಡ್ ಅನ್ನು ಹಾಕಿ:

<style>
   element { color: code; }
</style>

ಉದಾಹರಣೆ

ಪ್ಯಾರಾಗ್ರಾಫ್ ಪಠ್ಯದ ಬಣ್ಣವನ್ನು ಕೆಂಪು ಬಣ್ಣಕ್ಕೆ ಹೊಂದಿಸಲಾಗುತ್ತಿದೆ:

ನಿರ್ದಿಷ್ಟ ಪ್ಯಾರಾಗ್ರಾಫ್‌ನ ಬಣ್ಣವನ್ನು ಹೊಂದಿಸುವುದು:

<p style="color: #FF0000;">Some text ...</p>

ನೋಟ:

ಕೆಲವು ಪಠ್ಯ ...

ಎಲ್ಲಾ ಪ್ಯಾರಾಗಳ ಬಣ್ಣವನ್ನು ಹೊಂದಿಸಲಾಗುತ್ತಿದೆ

<style>
   p { color: #FF0000; }
</style>

ಅಂಶದ ಹಿನ್ನೆಲೆ ಬಣ್ಣವನ್ನು ಹೊಂದಿಸಲಾಗುತ್ತಿದೆ

element { background: code; }

   p { background: #FF0000; }
</style>

ಅಂಶದ ಗಡಿ ಬಣ್ಣವನ್ನು ಹೊಂದಿಸಲಾಗುತ್ತಿದೆ

element { border-color: topcode rightcode bottomcode leftcode }

ಉದಾಹರಣೆ

ಪ್ಯಾರಾಗ್ರಾಫ್ ಅಂಚು ಬಣ್ಣವನ್ನು ಕೆಂಪು, ಹಸಿರು, ನೀಲಿ ಮತ್ತು ಕಪ್ಪು ಬಣ್ಣಕ್ಕೆ ಹೊಂದಿಸುವುದು:

<p style="border-color: #FF0000 #00FF00 #0000FF #000000; border-style:solid">Some text ...</p>

ನೋಟ:

ಕೆಲವು ಪಠ್ಯ ...

ಎಲ್ಲಾ ಪ್ಯಾರಾಗಳ ಗಡಿ ಬಣ್ಣವನ್ನು ಹೊಂದಿಸುವುದು:

<style>
   p { border-color: #FF0000 #00FF00 #0000FF #000000; }
</style>
