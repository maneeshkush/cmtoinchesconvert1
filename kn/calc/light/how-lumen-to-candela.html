ಮುಖಪುಟ › ಕ್ಯಾಲ್ಕುಲೇಟರ್‌ಗಳು › ಲೈಟಿಂಗ್ ಕ್ಯಾಲ್ಕುಲೇಟರ್‌ಗಳು › ಲುಮೆನ್‌ಗಳನ್ನು ಕ್ಯಾಂಡೆಲಾಗೆ ಪರಿವರ್ತಿಸುವುದು ಹೇಗೆ

ಲುಮೆನ್‌ಗಳನ್ನು ಕ್ಯಾಂಡೆಲಾಗೆ ಪರಿವರ್ತಿಸುವುದು ಹೇಗೆ

ಲ್ಯೂಮೆನ್ಸ್‌ನಲ್ಲಿ (ಎಲ್‌ಎಂ) ಹೊಳೆಯುವ ಹರಿವನ್ನು ಕ್ಯಾಂಡೆಲಾದಲ್ಲಿ (ಸಿಡಿ) ಪ್ರಕಾಶಕ ತೀವ್ರತೆಗೆ ಪರಿವರ್ತಿಸುವುದು ಹೇಗೆ.

ಕ್ಯಾಂಡೆಲಾ ಮತ್ತು ಲ್ಯುಮೆನ್‌ಗಳು ಒಂದೇ ಪ್ರಮಾಣವನ್ನು ಪ್ರತಿನಿಧಿಸುವುದಿಲ್ಲವಾದ್ದರಿಂದ ನೀವು ಲೆಕ್ಕಾಚಾರ ಮಾಡಬಹುದು ಆದರೆ ಕ್ಯಾಂಡೆಲಾಗೆ ಲುಮೆನ್‌ಗಳನ್ನು ಪರಿವರ್ತಿಸಲು ಸಾಧ್ಯವಿಲ್ಲ.

ಕ್ಯಾಂಡೆಲಾ ಲೆಕ್ಕಾಚಾರಕ್ಕೆ ಲುಮೆನ್ಸ್

ಏಕರೂಪದ, ಐಸೊಟ್ರೊಪಿಕ್ ಬೆಳಕಿನ ಮೂಲಕ್ಕಾಗಿ, ಕ್ಯಾಂಡೆಲಾದಲ್ಲಿ (ಸಿಡಿ) ಪ್ರಕಾಶಕ ತೀವ್ರತೆ  I v ಲುಮೆನ್ಸ್ (lm) ನಲ್ಲಿನ ಪ್ರಕಾಶಕ ಫ್ಲಕ್ಸ್ Φ v   ಗೆ ಸಮಾನವಾಗಿರುತ್ತದೆ ,

 ಸ್ಟೆರಾಡಿಯನ್ಸ್ (sr) ನಲ್ಲಿ ಘನ ಕೋನ Ω ನಿಂದ ಭಾಗಿಸಿ  :

Iv(cd) = Φv(lm) / Ω(sr)

 

ಆದ್ದರಿಂದ ಸ್ಟೆರಾಡಿಯನ್‌ಗಳಲ್ಲಿ (sr)  ಘನ ಕೋನ Ω 2 ಬಾರಿ pi ಬಾರಿ 1 ಮೈನಸ್ ಕೊಸೈನ್‌ನ ಅರ್ಧ ತುದಿಯ ಕೋನ θ ಡಿಗ್ರಿಗಳಲ್ಲಿ (°) ಸಮಾನವಾಗಿರುತ್ತದೆ.

Ω(sr) = 2π(1 - cos(θ/2))

 

ಆದ್ದರಿಂದ ಕ್ಯಾಂಡೆಲಾದಲ್ಲಿ (ಸಿಡಿ) ಪ್ರಕಾಶಕ ತೀವ್ರತೆ I v ಲುಮೆನ್ಸ್ (lm) ನಲ್ಲಿನ ಪ್ರಕಾಶಕ ಫ್ಲಕ್ಸ್ Φ v   ಗೆ ಸಮನಾಗಿರುತ್ತದೆ ,

2 ಬಾರಿ pi ಬಾರಿ 1 ಮೈನಸ್ ಕೊಸೈನ್ ಅರ್ಧ ತುದಿಯ ಕೋನ  θ ಡಿಗ್ರಿಗಳಲ್ಲಿ (°) ಭಾಗಿಸಲಾಗಿದೆ.

Iv(cd) = Φv(lm) / ( 2π(1 - cos(θ/2)) )

ಆದ್ದರಿಂದ

candela = lumens / ( 2π(1 - cos(degrees/2)) )

ಅಥವಾ

cd = lm / ( 2π(1 - cos(°/2)) )

ಉದಾಹರಣೆ 1

ಲ್ಯೂಮೆನ್ಸ್ (lm) ನಲ್ಲಿನ ಪ್ರಕಾಶಕ ಫ್ಲಕ್ಸ್ Φ v  340lm ಮತ್ತು ತುದಿಯ ಕೋನ 60 ° ಆಗಿರುವಾಗ ಕ್ಯಾಂಡೆಲಾ (cd) ನಲ್ಲಿಪ್ರಕಾಶಕ ತೀವ್ರತೆಯನ್ನು  I v ಅನ್ನು ಕಂಡುಹಿಡಿಯಿರಿ:

Iv(cd) = 340 lm / ( 2π(1 - cos(60°/2)) ) = 403.9 cd

ಲ್ಯೂಮೆನ್ಸ್ (lm) ನಲ್ಲಿನ ಪ್ರಕಾಶಕ ಫ್ಲಕ್ಸ್ Φ v  360lm ಮತ್ತು ತುದಿಯ ಕೋನ 60 ° ಆಗಿರುವಾಗ ಕ್ಯಾಂಡೆಲಾದಲ್ಲಿ (cd)ಪ್ರಕಾಶಕ ತೀವ್ರತೆಯನ್ನು  I v ಅನ್ನು ಕಂಡುಹಿಡಿಯಿರಿ:

Iv(cd) = 360 lm / ( 2π(1 - cos(60°/2)) ) = 427.6 cd
