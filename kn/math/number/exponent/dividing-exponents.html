ಘಾತಗಳನ್ನು ವಿಭಜಿಸುವುದು

ಘಾತಾಂಕಗಳನ್ನು ಹೇಗೆ ವಿಭಜಿಸುವುದು.

  • ಘಾತಾಂಕಗಳನ್ನು ಒಂದೇ ಬೇಸ್‌ನೊಂದಿಗೆ ವಿಭಜಿಸುವುದು
  • ವಿಭಿನ್ನ ನೆಲೆಗಳೊಂದಿಗೆ ಘಾತಾಂಕಗಳನ್ನು ವಿಭಜಿಸುವುದು
  • ಋಣಾತ್ಮಕ ಘಾತಾಂಕಗಳನ್ನು ವಿಭಜಿಸುವುದು
  • ಘಾತಗಳೊಂದಿಗೆ ಭಿನ್ನರಾಶಿಗಳನ್ನು ಭಾಗಿಸುವುದು
  • ಭಾಗಶಃ ಘಾತಾಂಕಗಳನ್ನು ವಿಭಜಿಸುವುದು
  • ಘಾತಗಳೊಂದಿಗೆ ಅಸ್ಥಿರಗಳನ್ನು ವಿಭಜಿಸುವುದು
  • ವರ್ಗಮೂಲಗಳನ್ನು ಘಾತಗಳೊಂದಿಗೆ ಭಾಗಿಸುವುದು

ಘಾತಾಂಕಗಳನ್ನು ಒಂದೇ ಬೇಸ್‌ನೊಂದಿಗೆ ವಿಭಜಿಸುವುದು

ಒಂದೇ ಬೇಸ್ ಹೊಂದಿರುವ ಘಾತಾಂಕಗಳಿಗಾಗಿ, ನಾವು ಘಾತಾಂಕಗಳನ್ನು ಕಳೆಯಬೇಕು:

a n / a m = a n-m

ಉದಾಹರಣೆ:

26 / 23 = 26-3 = 23 = 2⋅2⋅2 = 8

ವಿಭಿನ್ನ ನೆಲೆಗಳೊಂದಿಗೆ ಘಾತಾಂಕಗಳನ್ನು ವಿಭಜಿಸುವುದು

ಬೇಸ್‌ಗಳು ವಿಭಿನ್ನವಾಗಿರುವಾಗ ಮತ್ತು a ಮತ್ತು b ನ ಘಾತಗಳು ಒಂದೇ ಆಗಿರುವಾಗ, ನಾವು ಮೊದಲು a ಮತ್ತು b ಅನ್ನು ವಿಭಜಿಸಬಹುದು:

a n / b n = (a / b) n

ಉದಾಹರಣೆ:

63 / 23 = (6/2)3 = 33 = 3⋅3⋅3 = 27

 

ಆಧಾರಗಳು ಮತ್ತು ಘಾತಾಂಕಗಳು ವಿಭಿನ್ನವಾಗಿರುವಾಗ ನಾವು ಪ್ರತಿ ಘಾತವನ್ನು ಲೆಕ್ಕ ಹಾಕಬೇಕು ಮತ್ತು ನಂತರ ಭಾಗಿಸಬೇಕು:

a n / b m

ಉದಾಹರಣೆ:

62 / 33 = 36 / 27 = 1.333

ಋಣಾತ್ಮಕ ಘಾತಾಂಕಗಳನ್ನು ವಿಭಜಿಸುವುದು

ಒಂದೇ ಬೇಸ್ ಹೊಂದಿರುವ ಘಾತಾಂಕಗಳಿಗಾಗಿ, ನಾವು ಘಾತಾಂಕಗಳನ್ನು ಕಳೆಯಬಹುದು:

a-n / a-m = a-n-(-m) = am-n

ಉದಾಹರಣೆ:

2-3 / 2-5 = 25-3 = 22 = 2⋅2 = 4

 

ಬೇಸ್‌ಗಳು ವಿಭಿನ್ನವಾಗಿರುವಾಗ ಮತ್ತು a ಮತ್ತು b ನ ಘಾತಗಳು ಒಂದೇ ಆಗಿರುವಾಗ, ನಾವು ಮೊದಲು a ಮತ್ತು b ಅನ್ನು ಗುಣಿಸಬಹುದು:

a-n / b-n = (a/b)-n = 1 / (a/b)n = (b/a)n

ಉದಾಹರಣೆ:

3-2 / 4-2 = (4/3)2 = 1.7778

 

ಆಧಾರಗಳು ಮತ್ತು ಘಾತಾಂಕಗಳು ವಿಭಿನ್ನವಾಗಿರುವಾಗ ನಾವು ಪ್ರತಿ ಘಾತವನ್ನು ಲೆಕ್ಕ ಹಾಕಬೇಕು ಮತ್ತು ನಂತರ ಭಾಗಿಸಬೇಕು:

a-n / b-m = bm / an

ಉದಾಹರಣೆ:

3-2 / 4-3 = 43 / 32 = 64 / 9 = 7.111

ಘಾತಗಳೊಂದಿಗೆ ಭಿನ್ನರಾಶಿಗಳನ್ನು ಭಾಗಿಸುವುದು

ಒಂದೇ ಭಿನ್ನರಾಶಿ ಬೇಸ್ನೊಂದಿಗೆ ಘಾತಾಂಕಗಳೊಂದಿಗೆ ಭಿನ್ನರಾಶಿಗಳನ್ನು ಭಾಗಿಸುವುದು:

(a / b)n / (a / b)m = (a / b)n-m

ಉದಾಹರಣೆ:

(4/3)3 / (4/3)2 = (4/3)3-2 = (4/3)1 = 4/3 = 1.333

 

ಭಿನ್ನರಾಶಿಗಳನ್ನು ಒಂದೇ ಘಾತಾಂಕದೊಂದಿಗೆ ಘಾತಾಂಕಗಳೊಂದಿಗೆ ಭಾಗಿಸುವುದು:

(a / b)n / (c / d)n = ((a / b)/(c / d))n = ((a⋅d / b⋅c))n

ಉದಾಹರಣೆ:

(4/3)3 / (3/5)3 = ((4/3)/(3/5))3 = ((4⋅5)/(3⋅3))3 = (20/9)3 = 10.97

 

ವಿಭಿನ್ನ ನೆಲೆಗಳು ಮತ್ತು ಘಾತಾಂಕಗಳೊಂದಿಗೆ ಘಾತಾಂಕಗಳೊಂದಿಗೆ ಭಿನ್ನರಾಶಿಗಳನ್ನು ವಿಭಜಿಸುವುದು:

(a / b) n / (c / d) m

ಉದಾಹರಣೆ:

(4/3)3 / (1/2)2 = 2.37 / 0.25 = 9.481

ಭಾಗಶಃ ಘಾತಾಂಕಗಳನ್ನು ವಿಭಜಿಸುವುದು

ಆಂಶಿಕ ಘಾತಾಂಕಗಳನ್ನು ಅದೇ ಆಂಶಿಕ ಘಾತಾಂಕದೊಂದಿಗೆ ಭಾಗಿಸುವುದು:

a n/m / b n/m = (a / b) n/m

ಉದಾಹರಣೆ:

33/2 / 23/2 = (3/2)3/2 = 1.53/2 = √(1.53) = √3.375 = 1.837

 

ಭಾಗಶಃ ಘಾತಾಂಕಗಳನ್ನು ಒಂದೇ ಬೇಸ್‌ನೊಂದಿಗೆ ವಿಭಜಿಸುವುದು:

a n/m / a k/j = a (n/m)-(k/j)

ಉದಾಹರಣೆ:

23/2 / 24/3 = 2(3/2)-(4/3) = 2(1/6) = 6√2 = 1.122

 

ವಿಭಿನ್ನ ಘಾತಾಂಕಗಳು ಮತ್ತು ಭಿನ್ನರಾಶಿಗಳೊಂದಿಗೆ ಭಾಗಶಃ ಘಾತಾಂಕಗಳನ್ನು ವಿಭಜಿಸುವುದು:

a n/m / b k/j

ಉದಾಹರಣೆ:

23/2 / 24/3 = √(23) / 3√(24) = 2.828 / 2.52 = 1.1222

ಘಾತಗಳೊಂದಿಗೆ ಅಸ್ಥಿರಗಳನ್ನು ವಿಭಜಿಸುವುದು

ಒಂದೇ ಬೇಸ್ ಹೊಂದಿರುವ ಘಾತಾಂಕಗಳಿಗಾಗಿ, ನಾವು ಘಾತಾಂಕಗಳನ್ನು ಕಳೆಯಬಹುದು:

xn / xm = xn-m

ಉದಾಹರಣೆ:

x5 / x3 = (x⋅x⋅x⋅x⋅x) / (x⋅x⋅x) = x5-3 = x2
