ಘಾತಗಳನ್ನು ಸರಳಗೊಳಿಸುವುದು

ಘಾತಾಂಕಗಳನ್ನು ಸರಳೀಕರಿಸುವುದು ಹೇಗೆ.

  • ತರ್ಕಬದ್ಧ ಘಾತಾಂಕಗಳನ್ನು ಸರಳಗೊಳಿಸುವುದು
  • ಘಾತಾಂಕಗಳೊಂದಿಗೆ ಭಿನ್ನರಾಶಿಗಳನ್ನು ಸರಳಗೊಳಿಸುವುದು
  • ಋಣಾತ್ಮಕ ಘಾತಾಂಕಗಳನ್ನು ಸರಳಗೊಳಿಸುವುದು
  • ಘಾತಾಂಕಗಳೊಂದಿಗೆ ರಾಡಿಕಲ್‌ಗಳನ್ನು ಸರಳಗೊಳಿಸುವುದು

ತರ್ಕಬದ್ಧ ಘಾತಾಂಕಗಳನ್ನು ಸರಳಗೊಳಿಸುವುದು

n/m ನ ಶಕ್ತಿಗೆ ಏರಿಸಿದ ಬೇಸ್ b ಇದಕ್ಕೆ ಸಮಾನವಾಗಿರುತ್ತದೆ:

bn/m = (m√b)n = m√(bn)

ಉದಾಹರಣೆ:

3/2 ರ ಶಕ್ತಿಗೆ ಏರಿಸಲಾದ ಬೇಸ್ 2 ಅನ್ನು 3 ರ ಶಕ್ತಿಗೆ ಏರಿಸಿದ ಬೇಸ್ 2 ರಿಂದ ಭಾಗಿಸಿದ 1 ಗೆ ಸಮಾನವಾಗಿರುತ್ತದೆ:

23/2 = 2√(23) = 2.828

ಘಾತಾಂಕಗಳೊಂದಿಗೆ ಭಿನ್ನರಾಶಿಗಳನ್ನು ಸರಳಗೊಳಿಸುವುದು

ಘಾತಗಳೊಂದಿಗೆ ಭಿನ್ನರಾಶಿಗಳು:

(a / b)n = an / bn

ಉದಾಹರಣೆ:

(4/3)3 = 43 / 33 = 64 / 27 = 2.37

ಋಣಾತ್ಮಕ ಘಾತಾಂಕಗಳನ್ನು ಸರಳಗೊಳಿಸುವುದು

ಮೈನಸ್ n ನ ಶಕ್ತಿಗೆ ಏರಿಸಲಾದ ಬೇಸ್ 1 ಗೆ ಸಮಾನವಾಗಿರುತ್ತದೆ b ಬೇಸ್ ನಿಂದ ಭಾಗಿಸಿ n ನ ಶಕ್ತಿಗೆ ಏರಿಸಲಾಗುತ್ತದೆ:

b-n = 1 / bn

ಉದಾಹರಣೆ:

ಮೈನಸ್ 3 ರ ಶಕ್ತಿಗೆ ಏರಿಸಲಾದ ಬೇಸ್ 2 ಅನ್ನು 1 ಕ್ಕೆ ಸಮನಾಗಿರುತ್ತದೆ, ಇದನ್ನು ಬೇಸ್ 2 ರಿಂದ 3 ರ ಶಕ್ತಿಗೆ ಹೆಚ್ಚಿಸಲಾಗಿದೆ:

2-3 = 1/23 = 1/(2⋅2⋅2) = 1/8 = 0.125

ಘಾತಾಂಕಗಳೊಂದಿಗೆ ರಾಡಿಕಲ್‌ಗಳನ್ನು ಸರಳಗೊಳಿಸುವುದು

ಘಾತದೊಂದಿಗೆ ಆಮೂಲಾಗ್ರಕ್ಕಾಗಿ:

(m√a)n = an/m

ಉದಾಹರಣೆ:

(√5)4 = 54/2 = 52 = 25
