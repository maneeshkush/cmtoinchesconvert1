ಸೊನ್ನೆಯ ಲಾಗರಿಥಮ್

ಸೊನ್ನೆಯ ಲಾಗರಿಥಮ್ ಎಂದರೇನು?ಲಾಗ್ (0) ಅನ್ನು ಏಕೆ ವ್ಯಾಖ್ಯಾನಿಸಲಾಗಿಲ್ಲ.

ನಿಜವಾದ ಲಾಗರಿಥಮಿಕ್ ಫಂಕ್ಷನ್ ಲಾಗ್ b (x) ಅನ್ನು x>0 ಗೆ ಮಾತ್ರ ವ್ಯಾಖ್ಯಾನಿಸಲಾಗಿದೆ.

ನಮಗೆ x ಸಂಖ್ಯೆಯನ್ನು ಕಂಡುಹಿಡಿಯಲಾಗಲಿಲ್ಲ, ಆದ್ದರಿಂದ x ನ ಶಕ್ತಿಗೆ ಏರಿಸಿದ ಮೂಲ b ಶೂನ್ಯಕ್ಕೆ ಸಮಾನವಾಗಿರುತ್ತದೆ:

b x = 0 , x does not exist

ಆದ್ದರಿಂದ ಶೂನ್ಯದ ಮೂಲ b ಲಾಗರಿಥಮ್ ಅನ್ನು ವ್ಯಾಖ್ಯಾನಿಸಲಾಗಿಲ್ಲ.

logb(0) is not defined

ಉದಾಹರಣೆಗೆ 0 ನ ಮೂಲ 10 ಲಾಗರಿಥಮ್ ಅನ್ನು ವ್ಯಾಖ್ಯಾನಿಸಲಾಗಿಲ್ಲ:

log10(0) is not defined

x ನ ಮೂಲ b ಲಾಗರಿಥಮ್‌ನ ಮಿತಿ, x ಧನಾತ್ಮಕ ಬದಿಯಿಂದ (0+) ಶೂನ್ಯವನ್ನು ಸಮೀಪಿಸಿದಾಗ, ಮೈನಸ್ ಅನಂತವಾಗಿರುತ್ತದೆ:
