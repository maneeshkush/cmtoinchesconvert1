ಲಿನಕ್ಸ್/ಯುನಿಕ್ಸ್‌ನಲ್ಲಿ cd ಆಜ್ಞೆ

cd ಎಂಬುದು ಟರ್ಮಿನಲ್‌ನ ಶೆಲ್‌ನ ಡೈರೆಕ್ಟರಿ/ಫೋಲ್ಡರ್ ಅನ್ನು ಬದಲಾಯಿಸಲು ಲಿನಕ್ಸ್ ಆಜ್ಞೆಯಾಗಿದೆ.

ಡೈರೆಕ್ಟರಿ ಹೆಸರನ್ನು ಸ್ವಯಂ ಪೂರ್ಣಗೊಳಿಸಲು ನೀವು ಟ್ಯಾಬ್ ಬಟನ್ ಅನ್ನು ಒತ್ತಬಹುದು.

  • ಸಿಡಿ ಸಿಂಟ್ಯಾಕ್ಸ್
  • ಸಿಡಿ ಉದಾಹರಣೆಗಳು

ಸಿಡಿ ಸಿಂಟ್ಯಾಕ್ಸ್

$ cd [directory]

ಸಿಡಿ ಆಜ್ಞೆಯ ಉದಾಹರಣೆಗಳು

ಹೋಮ್ ಡೈರೆಕ್ಟರಿಗೆ ಬದಲಾಯಿಸಿ ($HOME ಪರಿಸರ ವೇರಿಯಬಲ್ ನಿರ್ಧರಿಸುತ್ತದೆ):

$ cd

 

ಹೋಮ್ ಡೈರೆಕ್ಟರಿಗೆ ಸಹ ಬದಲಾಯಿಸಿ:

$ cd ~

 

ರೂಟ್ ಡೈರೆಕ್ಟರಿಗೆ ಬದಲಾಯಿಸಿ:

$ cd /

 

ಪೋಷಕ ಡೈರೆಕ್ಟರಿಗೆ ಬದಲಾಯಿಸಿ:

$ cd ..

 

ಉಪ ಡೈರೆಕ್ಟರಿ ದಾಖಲೆಗಳಿಗೆ ಬದಲಾಯಿಸಿ :

$ cd Documents

 

ಉಪ ಡೈರೆಕ್ಟರಿ ದಾಖಲೆಗಳು/ಪುಸ್ತಕಗಳಿಗೆ ಬದಲಾಯಿಸಿ :

$ cd Documents/Books

 

ಸಂಪೂರ್ಣ ಮಾರ್ಗ /ಮನೆ/ಬಳಕೆದಾರ/ಡೆಸ್ಕ್‌ಟಾಪ್‌ನೊಂದಿಗೆ ಡೈರೆಕ್ಟರಿಗೆ ಬದಲಾಯಿಸಿ:

$ cd /home/user/Desktop

 

ವೈಟ್ ಸ್ಪೇಸ್‌ನೊಂದಿಗೆ ಡೈರೆಕ್ಟರಿ ಹೆಸರಿಗೆ ಬದಲಾಯಿಸಿ - ನನ್ನ ಚಿತ್ರಗಳು :

$ cd My\ Images

ಅಥವಾ

$ cd "My Images"

ಅಥವಾ

$ cd 'My Images'
