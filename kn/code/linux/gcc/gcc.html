ಜಿಸಿಸಿ ಸಿ ಕಂಪೈಲರ್

GCC ಎನ್ನುವುದು GNU ಕಂಪೈಲರ್ ಕಲೆಕ್ಷನ್‌ನ ಚಿಕ್ಕದಾಗಿದೆ, ಇದು Linux ಗಾಗಿ C ಕಂಪೈಲರ್ ಆಗಿದೆ.

GCC ಸಿಂಟ್ಯಾಕ್ಸ್

$ gcc [options] [source files] [object files] [-o output file]

GCC ಆಯ್ಕೆಗಳು

GCC ಮುಖ್ಯ ಆಯ್ಕೆಗಳು:

ಆಯ್ಕೆಯನ್ನು ವಿವರಣೆ
gcc -c ಲಿಂಕ್ ಮಾಡದೆಯೇ ಆಬ್ಜೆಕ್ಟ್ ಫೈಲ್‌ಗಳಿಗೆ ಮೂಲ ಫೈಲ್‌ಗಳನ್ನು ಕಂಪೈಲ್ ಮಾಡಿ
gcc -D ಹೆಸರು [ =ಮೌಲ್ಯ ] ಪ್ರಿಪ್ರೊಸೆಸರ್ ಮ್ಯಾಕ್ರೋ ಅನ್ನು ವ್ಯಾಖ್ಯಾನಿಸಿ
gcc -fPIC ಹಂಚಿದ ಲೈಬ್ರರಿಗಳಿಗಾಗಿ ಸ್ಥಾನ ಸ್ವತಂತ್ರ ಕೋಡ್ ಅನ್ನು ರಚಿಸಿ
gcc -g ಮಟ್ಟ GDB ಯಿಂದ ಬಳಸಬೇಕಾದ ಡೀಬಗ್ ಮಾಹಿತಿಯನ್ನು ರಚಿಸಿ
gcc - ನಾನು ನಿರ್ದೇಶಕ ಹೆಡರ್ ಫೈಲ್‌ಗಳ ಡೈರೆಕ್ಟರಿಯನ್ನು ಸೇರಿಸಿ
gcc -l ಲಿಬ್ ಲೈಬ್ರರಿ ಫೈಲ್‌ನೊಂದಿಗೆ ಲಿಂಕ್ ಮಾಡಿ
gcc -L dir ಲೈಬ್ರರಿ ಫೈಲ್‌ಗಳಿಗಾಗಿ ಡೈರೆಕ್ಟರಿಯಲ್ಲಿ ನೋಡಿ
gcc -o ಔಟ್‌ಪುಟ್ ಫೈಲ್ ಔಟ್ಪುಟ್ ಫೈಲ್ಗೆ ಬಿಲ್ಡ್ ಔಟ್ಪುಟ್ ಅನ್ನು ಬರೆಯಿರಿ
gcc -O ಮಟ್ಟ ಕೋಡ್ ಗಾತ್ರ ಮತ್ತು ಕಾರ್ಯಗತಗೊಳಿಸುವ ಸಮಯಕ್ಕೆ ಉತ್ತಮಗೊಳಿಸಿ
gcc-ಹಂಚಿಕೊಂಡಿದೆ ಹಂಚಿದ ಲೈಬ್ರರಿಗಾಗಿ ಹಂಚಿದ ವಸ್ತು ಫೈಲ್ ಅನ್ನು ರಚಿಸಿ
gcc -U ಹೆಸರು ಪ್ರಿಪ್ರೊಸೆಸರ್ ಮ್ಯಾಕ್ರೋ ಅನ್ನು ವ್ಯಾಖ್ಯಾನಿಸಬೇಡಿ
gcc -w ಎಲ್ಲಾ ಎಚ್ಚರಿಕೆ ಸಂದೇಶಗಳನ್ನು ನಿಷ್ಕ್ರಿಯಗೊಳಿಸಿ
gcc -ಗೋಡೆ ಎಲ್ಲಾ ಎಚ್ಚರಿಕೆ ಸಂದೇಶಗಳನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಿ
ಜಿಸಿಸಿ -ವೆಕ್ಸ್ಟ್ರಾ ಹೆಚ್ಚುವರಿ ಎಚ್ಚರಿಕೆ ಸಂದೇಶಗಳನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಿ

GCC ಉದಾಹರಣೆಗಳು

file1.c ಮತ್ತು file2.c ಅನ್ನು ಕಂಪೈಲ್ ಮಾಡಿ ಮತ್ತುಔಟ್‌ಪುಟ್ ಫೈಲ್‌ಗೆ ಲಿಂಕ್ ಮಾಡಿ execfile :

$ gcc file1.c file2.c -o execfile

 

ಔಟ್ಪುಟ್ ಫೈಲ್ ಎಕ್ಸೆಕ್ಫೈಲ್ ಅನ್ನು ರನ್ ಮಾಡಿ :

$ ./execfile

 

ಲಿಂಕ್ ಮಾಡದೆಯೇ file1.c ಮತ್ತು file2.c ಅನ್ನು ಕಂಪೈಲ್ ಮಾಡಿ:

$ gcc -c file1.c file2.c

 

ಡೀಬಗ್ ಮಾಹಿತಿಯೊಂದಿಗೆ myfile.c ಅನ್ನು ಕಂಪೈಲ್ ಮಾಡಿ ಮತ್ತು ಔಟ್‌ಪುಟ್ ಫೈಲ್ ಎಕ್ಸೆಕ್‌ಫೈಲ್‌ಗೆ ಲಿಂಕ್ ಮಾಡಿ :

$ gcc -g myfile.c -o execfile

 

myfile.c ಅನ್ನು ಕಂಪೈಲ್ಮಾಡಿ ಎಚ್ಚರಿಕೆ ಸಂದೇಶಗಳನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಲಾಗಿದೆ ಮತ್ತು ಔಟ್‌ಪುಟ್ ಫೈಲ್ ಎಕ್ಸೆಕ್‌ಫೈಲ್‌ಗೆ ಲಿಂಕ್ ಮಾಡಿ :

$ gcc -Wall myfile.c -o execfile

 

myfile.c ಅನ್ನುಕಂಪೈಲ್ ಮಾಡಿ ಮತ್ತು ಸ್ಟ್ಯಾಟಿಕ್ ಲೈಬ್ರರಿ libmath.a ನೊಂದಿಗೆ ಲಿಂಕ್ ಮಾಡಿ / user/local/math ನಲ್ಲಿ ಫೈಲ್ ಎಕ್ಸೆಕ್‌ಫೈಲ್ ಅನ್ನು ಔಟ್‌ಪುಟ್ ಮಾಡಲು :

$ gcc -static myfile.c -L/user/local/math -lmath -o execfile

 

ಆಪ್ಟಿಮೈಸೇಶನ್‌ನೊಂದಿಗೆ myfile.c ಅನ್ನು ಕಂಪೈಲ್ ಮಾಡಿ ಮತ್ತು ಔಟ್‌ಪುಟ್ ಫೈಲ್ ಎಕ್ಸೆಕ್‌ಫೈಲ್‌ಗೆ ಲಿಂಕ್ ಮಾಡಿ:

$ gcc -O myfile.c -o execfile
