प्रतिपादक क्या है

आधार a की घात n की घात a, n गुणा के गुणन के बराबर है:

a n = a × a × ... × a

                    एन बार

a आधार है और n घातांक है।

उदाहरण

31 = 3

32 = 3 × 3 = 9

33 = 3 × 3 × 3 = 27

34 = 3 × 3 × 3 × 3 = 81

35 = 3 × 3 × 3 × 3 × 3 = 243

एक्सपोनेंट उत्पाद नियम

समान आधार वाला उत्पाद नियम

an ⋅ am = an+m

उदाहरण:

23 ⋅ 24 = 23+4 = 27 = 2⋅2⋅2⋅2⋅2⋅2⋅2 = 128

समान घातांक वाला गुणन नियम

an ⋅ bn = (a ⋅ b)n

उदाहरण:

32 ⋅ 42 = (3⋅4)2 = 122 = 12⋅12 = 144

देखें: गुणा करने वाले घातांक

घातांक भागफल नियम

समान आधार वाला भागफल नियम

an / am = an-m

उदाहरण:

25 / 23 = 25-3 = 22 = 2⋅2 = 4

समान घातांक वाला भागफल नियम

an / bn = (a / b)n

उदाहरण:

43 / 23 = (4/2)3 = 23 = 2⋅2⋅2 = 8

देखें: डिवाइडिंग एक्सपोनेंट्स

घातांक शक्ति नियम

शक्ति नियम मैं

(an) m = a n⋅m

उदाहरण:

(23)2 = 23⋅2 = 26 = 2⋅2⋅2⋅2⋅2⋅2 = 64

शक्ति नियम द्वितीय

a nm = a (nm)

उदाहरण:

232 = 2(32) = 2(3⋅3) = 29 = 2⋅2⋅2⋅2⋅2⋅2⋅2⋅2⋅2 = 512

रेडिकल्स के साथ पावर रूल

m√(a n) = a n/m

उदाहरण:

2√(26) = 26/2 = 23 = 2⋅2⋅2 = 8

नकारात्मक घातांक नियम

b-n = 1 / bn

उदाहरण:

2-3 = 1/23 = 1/(2⋅2⋅2) = 1/8 = 0.125
