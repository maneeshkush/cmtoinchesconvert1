एचटीएमएल पुनर्निर्देशन

एचटीएमएल रीडायरेक्ट। HTML मेटा रीफ्रेश रीडायरेक्शन कोड।

HTML मेटा रीफ्रेश रीडायरेक्ट क्लाइंट साइड रीडायरेक्ट है और 301 स्थायी रीडायरेक्ट नहीं है।

0 सेकंड के समय अंतराल के साथ HTML मेटा रीफ्रेश, Google द्वारा पेजरैंक ट्रांसफर के लिए 301 रीडायरेक्ट के संकेत के रूप में माना जाता है।

यदि आप वास्तविक 301 स्थायी पुनर्निर्देशन करना चाहते हैं, तो आप इसे HTML फ़ाइलों में PHP कोड को सक्षम करने के बाद PHP पुनर्निर्देशन के साथ कर सकते हैं।

एचटीएमएल मेटा रीफ्रेश रीडायरेक्ट

पुनर्निर्देशन हेड सेक्शन में मेटा रिफ्रेश के साथ किया जाता है।

फ़ॉलबैक उद्देश्यों के लिए बॉडी सेक्शन में लिंक।

जिस पृष्ठ पर आप पुनर्निर्देशित करना चाहते हैं, उसके URL के साथ पुराने पृष्ठ को पुनर्निर्देशन कोड से बदलें।

ओल्ड-पेज.एचटीएमएल:

<!DOCTYPE html>
<html>
<head>
   <!-- HTML meta refresh URL redirection -->
   <meta http-equiv="refresh"
   content="0; url=http://www.mydomain.com/new-page.html">
</head>
<body>
   <p>The page has moved to:
   <a href="http://www.mydomain.com/new-page.html">this page</a></p>
</body>
</html>

HTML मेटा रीफ्रेश रीडायरेक्ट उदाहरण

html-रीडायरेक्ट-test.htm:

<!DOCTYPE html>
<html>
<head>
   <!-- HTML meta refresh URL redirection -->
   <meta http-equiv="refresh"
   content="0; url=https://www.cmtoinchesconvert.com/hi/web/dev/html-redirect.htm">
</head>
<body>
   <p>The page has moved to:
   <a href="https://www.cmtoinchesconvert.com/hi/web/dev/html-redirect.htm">this page</a></p>
</body>
</html>

 

html-redirect-test.htm से इस पेज पर वापस रीडायरेक्ट करने के लिए इस लिंक को दबाएं:

 

HTML मेटा रीफ्रेश रीडायरेक्ट टेस्ट

एचटीएमएल कैनोनिकल लिंक टैग रीडायरेक्ट

प्रामाणिक लिंक पसंदीदा URL पर पुनर्निर्देशित नहीं करता है, लेकिन यह उन वेबसाइटों के लिए URL पुनर्निर्देशन का विकल्प हो सकता है, जहां अधिकांश ट्रैफ़िक खोज इंजनों से आता है।

HTML कैनोनिकल लिंक टैग का उपयोग तब किया जा सकता है जब समान सामग्री वाले कई पृष्ठ हों और आप खोज इंजनों को बताना चाहते हों कि आप खोज परिणामों में किस पृष्ठ का उपयोग करना पसंद करते हैं।

कैनोनिकल लिंक टैग एक ही डोमेन और क्रॉस-डोमेन से भी लिंक कर सकता है।

नए पेज से लिंक करने के लिए पुराने पेज में कैनोनिकल लिंक टैग जोड़ें।

उन पेजों में कैनोनिकल लिंक टैग जोड़ें जिन्हें आप पसंदीदा पेज से लिंक करने के लिए खोज इंजन ट्रैफ़िक प्राप्त नहीं करना चाहते हैं।

कैनोनिकल लिंक टैग को <head> सेक्शन में जोड़ा जाना चाहिए।

ओल्ड-पेज.एचटीएमएल:

<link rel="canonical" href="http://www.mydomain.com/new-page.html">
