jQuery रीडायरेक्ट

JQuery के साथ किसी पृष्ठ को URL पर पुनर्निर्देशित कैसे करें।

सर्च इंजन पेज रैंक को पुराने URL से नए URL में स्थानांतरित करने के लिए 301 स्थिति कोड का उपयोग करते हैं।

jQuery पुनर्निर्देशन वापसी http प्रतिक्रिया स्थिति कोड: 200 ठीक है।

तो jQuery पुनर्निर्देशन, जावास्क्रिप्ट पुनर्निर्देशन की तरह, खोज इंजन के अनुकूल नहीं है और अन्य पुनर्निर्देशन विधियों का उपयोग करना बेहतर है जो 301 स्थानांतरित स्थायी रूप से स्थिति कोड लौटाते हैं।

jQuery पुनर्निर्देशन वास्तव में जावास्क्रिप्ट पुनर्निर्देशन है । JQuery के साथ पुनर्निर्देशन को ओवरकिल माना जाता है क्योंकि आप कम कोड के साथ शुद्ध जावास्क्रिप्ट पुनर्निर्देशन कर सकते हैं।

jQuery रीडायरेक्ट

जिस पृष्ठ पर आप पुनर्निर्देशित करना चाहते हैं, उसके URL के साथ पुराने पृष्ठ को पुनर्निर्देशन कोड से बदलें।

ओल्ड-पेज.एचटीएमएल:

<!DOCTYPE html>
<html>
<body>
<script src="http://ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script>
<script type ="टेक्स्ट/जावास्क्रिप्ट">
   // जेक्वेरी यूआरएल रीडायरेक्शन
   $(दस्तावेज़) .तैयार(फ़ंक्शन() {
      url = "http://www.mydomain.com/new-page.html";
      $(स्थान ).attr( "href", url);
   });
</script>
</body>
</html>

jQuery रीडायरेक्ट उदाहरण

jquery-redirect-test.htm

<!DOCTYPE html>
<html>
<body>
<script src="//ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script>
<script type="text/javascript">
   $(document).ready( function() {
      url = "https://www.cmtoinchesconvert.com/hi/web/dev/jquery-redirect.htm";
      $(location).attr("href", url);
   });
</script>
</body>
</html>
