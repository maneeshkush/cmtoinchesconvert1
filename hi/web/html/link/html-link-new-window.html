एक नई विंडो में HTML लिंक

किसी लिंक को नई विंडो या नए टैब में कैसे खोलें।

एक नई विंडो या टैब में एक लिंक खोलें

लिंक को नई विंडो/टैब में खोलने के लिए , <a> टैग के अंदर target="_blank" जोड़ें:

<a href="../html-link.htm" target="_blank">Open page in new window</a>

कोड इस लिंक को बनाएगा:

नई विंडो या नया टैब

आप यह सेट नहीं कर सकते कि लिंक नई विंडो में खुलेगा या नए टैब में। यह ब्राउजर की सेटिंग पर निर्भर करता है। 

निर्दिष्ट आकार के साथ एक नई विंडो में एक लिंक खोलें

लिंक को नई विंडो में खोलने के लिए <a> टैग के अंदर Javascript कमांड onclick="window.open('text-link.htm', 'name','width=600,height=400') जोड़ें:

<a href="../html-link.htm" target="popup" onclick="window.open('../html-link.htm','name','width=600,height=400')">Open page in new window</a>
