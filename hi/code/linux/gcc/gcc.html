जीसीसी सी संकलक

GCC , GNU कंपाइलर कलेक्शन, Linux के लिए एक C कंपाइलर है।

जीसीसी सिंटैक्स

$ gcc [options] [source files] [object files] [-o output file]

जीसीसी विकल्प

जीसीसी मुख्य विकल्प:

विकल्प विवरण
जीसीसी -सी स्रोत फ़ाइलों को लिंक किए बिना फ़ाइलों को ऑब्जेक्ट करने के लिए संकलित करें
जीसीसी -डी नाम [ = मान ] प्रीप्रोसेसर मैक्रो को परिभाषित करें
जीसीसी -fPIC साझा पुस्तकालयों के लिए स्थिति स्वतंत्र कोड उत्पन्न करें
जीसीसी-जी स्तर GDB द्वारा उपयोग की जाने वाली डिबग जानकारी उत्पन्न करें
जीसीसी -मैं दिर ऐड में हेडर फाइलों की निर्देशिका शामिल है
जीसीसी -एल lib लाइब्रेरी फ़ाइल के साथ लिंक करें
जीसीसी-एल दिर पुस्तकालय फ़ाइलों के लिए निर्देशिका में देखें
जीसीसी -o आउटपुट फ़ाइल आउटपुट फ़ाइल में बिल्ड आउटपुट लिखें
जीसीसी-ओ स्तर कोड आकार और निष्पादन समय के लिए अनुकूलित करें
जीसीसी-साझा साझा पुस्तकालय के लिए साझा वस्तु फ़ाइल उत्पन्न करें
जीसीसी -यू नाम प्रीप्रोसेसर मैक्रो को अपरिभाषित करें
जीसीसी -डब्ल्यू सभी चेतावनी संदेशों को अक्षम करें
जीसीसी-दीवार सभी चेतावनी संदेशों को सक्षम करें
जीसीसी-वेक्स्ट्रा अतिरिक्त चेतावनी संदेश सक्षम करें

जीसीसी उदाहरण

File1.c और file2.c संकलित करें और आउटपुट फ़ाइल execfile से लिंक करें :

$ gcc file1.c file2.c -o execfile

 

आउटपुट फ़ाइल निष्पादित करें:

$ ./execfile

 

बिना लिंक के file1.c और file2.c संकलित करें :

$ gcc -c file1.c file2.c

 

डिबग जानकारी के साथ myfile.c संकलित करें और आउटपुट फ़ाइल execfile से लिंक करें :

$ gcc -g myfile.c -o execfile

 

सक्षम चेतावनी संदेशों के साथ myfile.c संकलित करें और आउटपुट फ़ाइल execfile से लिंक करें :

$ gcc -Wall myfile.c -o execfile

 

myfile.c को स्थिर पुस्तकालय libmath.a के साथ संकलित करें और /user/local/math में स्थित आउटपुट फ़ाइल execfile के साथ लिंक करें :

$ gcc -static myfile.c -L/user/local/math -lmath -o execfile

 

अनुकूलन के साथ myfile.c संकलित करें और आउटपुट फ़ाइल execfile से लिंक करें :

$ gcc -O myfile.c -o execfile
