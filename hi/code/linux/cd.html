लिनक्स/यूनिक्स में सीडी कमांड

सीडी टर्मिनल के खोल की निर्देशिका/फ़ोल्डर को बदलने के लिए एक लिनक्स कमांड है।

निर्देशिका नाम को स्वत: पूर्ण करने के लिए आप टैब बटन दबा सकते हैं।

  • सीडी सिंटैक्स
  • सीडी उदाहरण

सीडी सिंटैक्स

$ cd [directory]

सीडी कमांड उदाहरण

होम डायरेक्टरी में बदलें ($ HOME पर्यावरण चर द्वारा निर्धारित):

$ cd

 

होम डायरेक्टरी में भी बदलें:

$ cd ~

 

रूट डायरेक्टरी में बदलें:

$ cd /

 

मूल निर्देशिका में बदलें:

$ cd ..

 

उपनिर्देशिका दस्तावेज़ में बदलें :

$ cd Documents

 

उपनिर्देशिका दस्तावेज़ों/पुस्तकों में बदलें :

$ cd Documents/Books

 

निरपेक्ष पथ के साथ निर्देशिका में बदलें /home/user/Desktop :

$ cd /home/user/Desktop

 

सफेद स्थान के साथ निर्देशिका नाम में बदलें - मेरी छवियां :

$ cd My\ Images

या

$ cd "My Images"

या

$ cd 'My Images'
