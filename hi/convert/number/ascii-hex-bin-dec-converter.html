1. ASCII कोड: यह क्या है और आप इसका उपयोग कैसे करते हैं?

ASCII (American Standard Code for Information Interchange) is a code that assigns a unique number to each letter, number, and symbol on a computer keyboard. This code allows computers to communicate with each other across the internet.

The ASCII code is also used to create text files. These files are composed of a series of letters, numbers, and symbols that are assigned a unique number by the ASCII code. When a computer opens an ASCII text file, it will translate the numbers back in to letters, numbers, and symbols.

2. Hex code: what is it and how do you use it?

हेक्स कोड एक 6-अंकीय कोड है जिसका उपयोग डिजिटल मीडिया में रंगों का प्रतिनिधित्व करने के लिए किया जाता है। रंगों को परिभाषित करने के लिए HTML, CSS और अन्य वेब विकास भाषाओं में हेक्स कोड का उपयोग किया जाता है।

HTML या CSS में हेक्स कोड का उपयोग करने के लिए, आपको उस रंग का नाम जानना होगा जिसका आप उपयोग करना चाहते हैं। आप रंग नामों की एक सूची यहां पा सकते हैं: https://www.w3schools.com/colors/colors_names.asp

एक बार जब आप रंग का नाम जान लेते हैं, तो आप इस तालिका में उस रंग के लिए हेक्स कोड ढूंढ सकते हैं। : https://www.w3schools.com/tags/ref_color_hex.asp

HTML या CSS में एक हेक्स कोड का उपयोग करने के लिए, आपको इस प्रारूप में कोड का उपयोग करने की आवश्यकता है: "#xxxxxx"

उदाहरण के लिए, लाल रंग के लिए हेक्स कोड है " #FF0000"। इसलिए, यदि आप HTML या CSS में लाल रंग का उपयोग करना चाहते हैं, तो आप इस कोड का उपयोग करेंगे

3. बाइनरी कोड: यह क्या है और आप इसका उपयोग कैसे करते हैं?

बाइनरी कोड एक नंबरिंग सिस्टम है जो केवल अंकों 0 और 1 का उपयोग करता है। बाइनरी कोड में, प्रत्येक संख्या को इन दो अंकों के संयोजन द्वारा दर्शाया जाता है। उदाहरण के लिए, संख्या 12 को इस प्रकार दर्शाया जा सकता है:

4. दशमलव कोड: यह क्या है और आप इसका उपयोग कैसे करते हैं?

दशमलव कोड एक प्रणाली है जिसका उपयोग 0-9 अंकों का उपयोग करके संख्याओं का प्रतिनिधित्व करने के लिए किया जाता है। इस प्रणाली में, प्रत्येक अंक को एक स्तंभ में रखा जाता है, और अंक के मान को स्तंभ की स्थिति द्वारा इंगित 10 की शक्ति से गुणा किया जाता है। उदाहरण के लिए, संख्या 1234 को दशमलव कोड का उपयोग करके दर्शाया जा सकता है:

1 x 10^3 + 2 x 10^2 + 3 x 10^1 + 4 x 10^0

इस कोड का उपयोग किसी भी बड़ी या बड़ी संख्या को दर्शाने के लिए किया जा सकता है। छोटा। किसी संख्या को दशमलव कोड से मानक अंकन में बदलने के लिए, संख्या में अंकों को केवल बाएँ से दाएँ पढ़ा जाता है। उदाहरण के लिए, संख्या 987 को इसे पढ़कर मानक अंकन में परिवर्तित किया जा सकता है:

9 x 10^2 + 8 x 10^1 + 7 x 10^0

दशमलव कोड का उपयोग अंशों का प्रतिनिधित्व करने के लिए भी किया जा सकता है। इस मामले में, संख्या में प्रत्येक अंक को एक में रखा गया है

5. Converting between ASCII, Hex, Binary and Decimal: how do you do it?

ASCII is the most common representation of text data. Hexadecimal is a base 16 number system, which is used in computer programming to represent binary data. Binary is a base 2 number system, which is used to represent the states of computer memory. Decimal is the number system that we use in everyday life, with base 10.

To convert from ASCII to Hex, divide the ASCII value by 16, and take the remainder. The Hex value is the first six characters of the remainder. For example, the ASCII value for "A" is 65, so the Hex value is "A".

To convert from Hex to ASCII, divide the Hex value by 16, and take the remainder. The ASCII value is the first six characters of the remainder. For example, the Hex value for "A" is "41", so the ASCII value is "A".

6. बेस64 क्या है

बेस 64 एन्कोडिंग बाइनरी डेटा को टेक्स्ट फॉर्मेट में बदलने की एक प्रक्रिया है जिसे इंटरनेट पर आसानी से प्रसारित किया जा सकता है। पाठ प्रारूप 64 वर्णों की एक श्रृंखला से बना है, जिनमें से प्रत्येक 6-बिट बाइनरी नंबर से मेल खाता है। यह एक बाइनरी फ़ाइल को पाठ वर्णों की एक श्रृंखला में एन्कोड करना संभव बनाता है जिसे डेटा भ्रष्टाचार के जोखिम के बिना इंटरनेट पर आसानी से प्रसारित किया जा सकता है।
