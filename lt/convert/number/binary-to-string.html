Paversti dvejetainį tekstą

Konvertuoti dvejetainį ASCII kodą į tekstą:

  1. Gaukite dvejetainį baitą
  2. Konvertuoti dvejetainį baitą į dešimtainį
  3. Gaukite ASCII kodo simbolius iš ASCII lentelės
  4. Pereikite prie kito baito

Pavyzdys

Konvertuoti  „01010000 01101100 01100001 01101110 01110100 00100000 01110100 01110010 01100101 01100101 01100101 0111 tekstinį kodą  : Abinary 1SC

Sprendimas:

 Norėdami gauti simbolį iš ASCII kodo,naudokite  ASCII lentelę .

01010000 2  = 2 6 +2 4  = 64 + 16 = 80 => "P"

01101100 2  = 2 6 +2 5 +2 3 +2 2  = 64 + 32 + 8 + 4 = 108 => "l"

01100001 2  = 2 6 +2 5 +2 0  = 64 + 32 + 1 = 97 => "a"
