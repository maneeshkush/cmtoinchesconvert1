Laplaso transformacijos pavyzdiai

1 pavyzdys

Raskite f(t) transformacij:

f (t) = 3t + 2t2

Sprendimas:

{t} = 1/s2

{t2} = 2/s3

F(s) = {f (t)} = {3t + 2t2} = 3{t} + 2{t2} = 3/s2 + 4/s3

2 pavyzdys

Raskite atvirktin F(-i) transformacij:

F(s) = 3 / (s2 + s - 6)

Sprendimas:

Nordami rasti atvirktin transformacij, turime pakeisti s domeno funkcij paprastesn form:

F(s) = 3 / (s2 + s - 6) = 3 / [(s-2)(s+3)] = a / (s-2) + b / (s+3)

[a(s+3) + b(s-2)] / [(s-2)(s+3)] = 3 / [(s-2)(s+3)]

a(s+3) + b(s-2) = 3

Nordami rasti a ir b, gauname 2 lygtis - vien i s koeficient ir antr i likusi:

(a+b)s + 3a-2b = 3

a+b = 0 , 3a-2b = 3

a = 3/5 , b = -3/5

F(s) = 3 / 5(s-2) - 3 / 5(s+3)

Dabar F(-us) galima lengvai transformuoti naudojant eksponent funkcijos transformacij lentel:

f (t) = (3/5)e2t - (3/5)e-3t
