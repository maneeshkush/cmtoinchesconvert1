Skaičių sistema

b - skaičių sistemos bazė

d n – n-tas skaitmuo

n – gali prasidėti nuo neigiamo skaičiaus, jei skaičius turi trupmeninę dalį.

N +1 – skaitmenų skaičius

Dvejetainė skaičių sistema – bazė-2

Dvejetainiai skaičiai naudoja tik 0 ir 1 skaitmenis.

B žymi dvejetainį priešdėlį.

Pavyzdžiai:

101012 = 10101B = 1×24+0×23+1×22+0×21+1×20 = 16+4+1= 21

101112 = 10111B = 1×24+0×23+1×22+1×21+1×20 = 16+4+2+1= 23

1000112 = 100011B = 1×25+0×24+0×23+0×22+1×21+1×20 =32+2+1= 35

Aštuontainė skaičių sistema – bazė-8

Aštuntainiai skaičiai naudoja skaitmenis nuo 0...7.

Pavyzdžiai:

278 = 2×81+7×80 = 16+7 = 23

308 = 3×81+0×80 = 24

43078 = 4×83+3×82+0×81+7×80= 2247

Dešimtainė skaičių sistema – bazė-10

Dešimtainiuose skaičiuose naudojami skaitmenys nuo 0...9.

Tai yra įprasti skaičiai, kuriuos naudojame.

Pavyzdys:

253810 = 2×103+5×102+3×101+8×100

Šešioliktainė skaičių sistema – bazė-16

Šešioliktainiai skaičiai naudoja skaitmenis nuo 0..9 ir A..F.

H žymi šešioliktainį priešdėlį.

Pavyzdžiai:

2816 = 28H = 2×161+8×160 = 40

2F16 = 2FH = 2×161+15×160 = 47

BC1216 = BC12H = 11×163+12×162+1×161+2×160= 48146
