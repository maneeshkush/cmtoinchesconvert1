CSS spalva

Spalvos kodas gali būti vienas iš:

Šešioliktainis formatas: #rrggbb

RGB formatas: rgb (raudona, žalia, mėlyna)

Vardo formatas: vardas

Pavyzdys

Oranžinei spalvai:

Šešioliktainis formatas: #FFA500

RGB formatas: rgb (255,165,0)

Pavadinimo formatas: oranžinė

Elemento spalvos nustatymas

Konkrečiam elementui:

<element style="color: code;"></element>

Visiems to paties tipo elementams.Įdėkite kodą į <style> žymą antraštės skiltyje:

<style>
   element { color: code; }
</style>

Pavyzdys

Pastraipos teksto spalvos nustatymas į raudoną:

Konkrečios pastraipos spalvos nustatymas:

<p style="color: #FF0000;">Some text ...</p>

Žiūrėti:

Kažkoks tekstas...

Visų pastraipų spalvos nustatymas

<style>
   p { color: #FF0000; }
</style>

Nustatymo elemento fono spalva

element { background: code; }

Pavyzdys

Pastraipos fono spalvos nustatymas į raudoną:

Konkrečios pastraipos fono spalvos nustatymas:

<p style="background: #FF0000;">Some text ...</p>

Žiūrėti:

Kažkoks tekstas...

Visų pastraipų fono spalvos nustatymas:

<style>
   p { background: #FF0000; }
</style>

Elemento kraštinės spalvos nustatymas

element { border-color: topcode rightcode bottomcode leftcode }

Pavyzdys

Pastraipos kraštinės spalvos nustatymas į raudoną, žalią, mėlyną ir juodą:

Konkrečios pastraipos kraštinės spalvos nustatymas:

<p style="border-color: #FF0000 #00FF00 #0000FF #000000; border-style:solid">Some text ...</p>

Žiūrėti:

Kažkoks tekstas...

Visų pastraipų kraštinės spalvos nustatymas:

<style>
   p { border-color: #FF0000 #00FF00 #0000FF #000000; }
</style>
