HTML atsisiuntimo nuoroda

Kaip parašyti atsisiuntimo nuorodą HTML.

Atsisiuntimo nuoroda yra nuoroda, kuri naudojama failui iš serverio atsisiųsti į naršyklės katalogą vietiniame diske.

Atsisiuntimo nuorodos kodas parašytas taip:

<a href="test_file.zip" download>Download File</a>

Kodas sukurs šią nuorodą:

Atsisiųsti failą

Bandomasis failas turi būti žiniatinklio serveryje.

Kodą sudaro šios dalys:

  • <a> yra nuorodos žyma.
  • href atributas nustato atsisiunčiamą failą.
  • Atsisiųsti failą yra nuorodos tekstas.
  • </a> yra nuorodos pabaigos žyma.
