cd skipun í Linux/Unix

cd er Linux skipun til að breyta möppu/möppu í skel flugstöðvarinnar.

Þú getur ýtt á flipahnappinn til að fylla sjálfkrafa út nafn möppunnar.

  • cd setningafræði
  • CD dæmi

cd setningafræði

$ cd [directory]

cd skipanir dæmi

Breyta í heimaskrá (ákvarðað af $HOME umhverfisbreytu):

$ cd

 

Breyttu líka í heimaskrá:

$ cd ~

 

Breyta í rótarskrá:

$ cd /

 

Breyta í foreldraskrá:

$ cd ..

 

Breyta í undirskrá Skjöl :

$ cd Documents

 

Breyta í undirskrá Skjöl/Bækur :

$ cd Documents/Books

 

Breyttu í möppu með algjörri slóð /heima/notandi/skrifborð :

$ cd /home/user/Desktop

 

Breyta í möppuheiti með hvítu bili - Mínar myndir :

$ cd My\ Images

Eða

$ cd "My Images"

Eða

$ cd 'My Images'
