Canonical URL hlekkur

Canonical URL tag. Canonical hlekkjamerki.

HTML kanónískt tenglamerki tilvísun

Kanóníski hlekkurinn vísar ekki á valinn vefslóð, en hann getur verið valkostur við tilvísun vefslóðar fyrir vefsíður þar sem mest af umferðinni kemur frá leitarvélum.

HTML merkið fyrir kanóníska tengla er hægt að nota þegar það eru margar síður með svipað efni og þú vilt segja leitarvélum hvaða síðu þú kýst í leitarniðurstöðum.

The canonical link tag getur tengt við sama lén og einnig við mörg lén .

Bættu kanóníska tenglamerkinu við gömlu síðuna til að tengja við
nýju síðuna.

Bættu kanóníska tenglamerkinu við síður sem þú vilt helst ekki  tengja leitarvélaumferð við  valinn   síðu.

Bæta ætti við kanóníska tenglamerkinu í hlutanum.

Dæmi #1

Þegar síðan hefur færst getum við bætt við kanónískum hlekk á gömlu síðunni til að láta leitarvélar vita um að komast á nýju síðuna.

old-page1.html:

<!DOCTYPE html>
<html>
<head>
   ...
   <link rel="canonical" href="http://www.mydomain.com/new-page1.html">
</head>
<body>
   ...
</body>
</html>

Dæmi #2

Þegar það eru nokkrar síður með svipaða eiginleika, ef við kjósum að leitarvélar nái til síðu3.html, ættum við að bæta við kanónískum tengli við síðu3.html í höfuðhluta síðu1.html og síðu2.html.

síða1.html:

<!DOCTYPE html>
<html>
<head>
   ...
   <link rel="canonical" href="http://www.mydomain.com/page3.html">
</head>
<body>
   ...
</body>
</html>

síða2.html:

<!DOCTYPE html>
<html>
<head>
   ...
   <link rel="canonical" href="http://www.mydomain.com/page3.html">
</head>
<body>
   ...
</body>
</html>
