Javascript tilvísunarslóð

Hvernig á að beina vefslóð með Javascript.

Javascript tilvísun skilar ekki 301 varanlegum tilvísunarstöðukóða.

Javascript tilvísun

Skiptu út gamalli síðu fyrir tilvísunarkóða fyrir slóð síðunnar sem þú vilt beina á.

old-page.html:

<!DOCTYPE html>
<html>
<body>
<script type="text/javascript">
    // Javascript URL redirection
    window.location.replace("http://www.mydomain.com/new-page.html");
</script>
</body>
</html>

 

Leitarvélar nota 301 stöðukóðann til að flytja síðuröðun frá gömlu vefslóðinni yfir á nýju vefslóðina.

Javascript tilvísun skilar http svar stöðukóði: 200 OK.

Þannig að Javascript endurvísun er ekki leitarvélavæn og það er betra að nota aðrar tilvísunaraðferðir sem skila stöðukóða: 301 Fært varanlega.
