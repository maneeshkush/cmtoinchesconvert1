ערך ציפייה

בהסתברות ובסטטיסטיקה, התוחלת או הערך הצפוי , הוא הערך הממוצע המשוקלל של משתנה מקרי.

ציפייה למשתנה מקרי מתמשך

E ( X ) הוא ערך התוחלת של המשתנה האקראי הרציף X

x הוא הערך של המשתנה האקראי הרציף X

P ( x ) היא פונקציית צפיפות ההסתברות

ציפייה למשתנה אקראי בדיד

E ( X ) הוא ערך התוחלת של המשתנה האקראי הרציף X

x הוא הערך של המשתנה האקראי הרציף X

P ( x ) היא פונקציית מסת ההסתברות של X

מאפיינים של ציפייה

ליניאריות

כאשר a קבוע ו-X,Y הם משתנים אקראיים:

E(aX) = aE(X)

E(X+Y) = E(X) + E(Y)

קָבוּעַ

כאשר c קבוע:

E(c) = c

מוצר

כאשר X ו-Y הם משתנים אקראיים בלתי תלויים:

E(X ⋅Y) = E(X) ⋅ E(Y)
