b - בסיס מערכת ספרות

d n - הספרה ה-n

n - יכול להתחיל ממספר שלילי אם למספר יש חלק שבר.

N +1 - מספר הספרות

מערכת ספרות בינארית - בסיס-2

מספרים בינאריים משתמשים רק בספרות 0 ו-1.

B מציין קידומת בינארית.

דוגמאות:

101012 = 10101B = 1×24+0×23+1×22+0×21+1×20 = 16+4+1= 21

101112 = 10111B = 1×24+0×23+1×22+1×21+1×20 = 16+4+2+1= 23

1000112 = 100011B = 1×25+0×24+0×23+0×22+1×21+1×20 =32+2+1= 35

מערכת ספרות אוקטלית - בסיס-8

מספרים אוקטליים משתמשים בספרות מ-0..7.

דוגמאות:

278 = 2×81+7×80 = 16+7 = 23

308 = 3×81+0×80 = 24

43078 = 4×83+3×82+0×81+7×80= 2247

מערכת ספרות עשרונית - בסיס-10

מספרים עשרוניים משתמש בספרות מ-0..9.

אלה המספרים הרגילים שבהם אנו משתמשים.

דוגמא:

253810 = 2×103+5×102+3×101+8×100

מערכת ספרות הקסדצימלית - בסיס-16

מספרי hex משתמש בספרות מ-0..9 ו-A..F.

H מציין קידומת hex.

דוגמאות:

2816 = 28H = 2×161+8×160 = 40

2F16 = 2FH = 2×161+15×160 = 47

BC1216 = BC12H = 11×163+12×162+1×161+2×160= 48146
