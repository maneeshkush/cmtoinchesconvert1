הגדרה טנגנטית

במשולש ישר זווית ABC הטנגנס של α, tan(α) מוגדר כיחס בין הצלע המנוגדת לזווית α והצלע הסמוכה לזווית α:

tan α = a / b

דוגמא

a = 3"

b = 4"

tan α = a / b = 3 / 4 = 0.75

חוקים משיקים

שם החוק כְּלָל
סִימֶטרִיָה

tan(-θ) = -tan θ
סִימֶטרִיָה tan(90°- θ ) = cot θ
  tan θ = sin θ / cos θ
  tan θ = 1 / cot θ
זווית כפולה tan 2 θ = 2 tan θ / (1 - tan 2 θ )
סכום זוויות tan( α + β ) = (tan α + tan β ) / (1 - tan α tan β )
הבדל זוויות tan( α - β ) = (tan α - tan β ) / (1 + tan α tan β )
נגזר tan' x = 1 / cos 2 ( x )
בלתי נפרד ∫ tan x d x = - ln |cos x | + C
הנוסחה של אוילר tan x = ( e ix - e - ix ) / i ( e ix + e - ix )

פונקציית משיק הפוכה

הארקטנג'נט של x מוגדר כפונקציית המשיק ההפוכה של x כאשר x הוא ממשי (x ∈ℝ ) .

כאשר הטנגנס של y שווה ל-x:

tan y = x

אז הארקטנג'נט של x שווה לפונקציית המשיק ההפוכה של x, ששווה ל-y:

arctan x = tan-1 x = y

דוגמא

arctan 1 = tan-1 1 = π/4 rad = 45°
