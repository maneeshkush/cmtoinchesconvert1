הגדרה של לוגריתם טבעי

מתי

e y = x

אז בסיס e לוגריתם של x הוא

ln(x) = loge(x) = y

 

הקבוע e או מספר אוילר הוא:

e ≈ 2.71828183

Ln כפונקציה הפוכה של פונקציה אקספוננציאלית

פונקציית הלוגריתם הטבעית ln(x) היא הפונקציה ההפוכה של הפונקציה המעריכית e x .

עבור x>0,

f (f -1(x)) = eln(x) = x

אוֹ

f -1(f (x)) = ln(ex) = x

חוקי לוגריתם טבעיים ומאפיינים

שם החוק כְּלָל דוגמא
חוק מוצר

ln(x ∙ y) = ln(x) + ln(y)

ln(3 ∙ 7) = ln(3) + ln(7)
כלל כמות

ln(x / y) = ln(x) - ln(y)

ln(3 / 7) = ln(3) - ln(7)
שלטון כוח

ln(x y) = y ∙ ln(x)

ln(28) = 8∙ ln(2)
בנגזרת
 f ( x ) = ln( x ) ⇒ f ' ( x ) = 1 / x  
באינטגרל
 ∫ ln( x ) dx = x ∙ (ln( x ) - 1) + C  
ln של מספר שלילי
 ln( x ) אינו מוגדר כאשר x ≤ 0  
אין של אפס
 ln(0) אינו מוגדר  
 
באחד
 ln(1) = 0  
באין סוף
 lim ln( x ) = ∞ , כאשר x →∞  
זהותו של אוילר ln(-1) = iπ  

 

כלל תוצר לוגריתם

הלוגריתם של הכפל של x ו-y הוא סכום הלוגריתם של x והלוגריתם של y.

logb(x ∙ y) = logb(x) + logb(y)

לדוגמה:

log10(3 ∙ 7) = log10(3) + log10(7)

כלל מנה לוגריתם

הלוגריתם של החלוקה של x ו-y הוא ההבדל בין הלוגריתם של x והלוגריתם של y.

logb(x / y) = logb(x) - logb(y)

לדוגמה:

log10(3 / 7) = log10(3) - log10(7)

כלל כוח לוגריתם

הלוגריתם של x מועלה בחזקת y הוא y כפול הלוגריתם של x.

logb(x y) = y ∙ logb(x)

לדוגמה:

log10(28) = 8∙ log10(2)

נגזרת של לוגריתם טבעי

הנגזרת של פונקציית הלוגריתם הטבעית היא הפונקציה ההדדית.

מתי

f (x) = ln(x)

הנגזרת של f(x) היא:

f ' (x) = 1 / x

אינטגרל של לוגריתם טבעי

האינטגרל של פונקציית הלוגריתם הטבעי ניתן על ידי:

מתי

f (x) = ln(x)

האינטגרל של f(x) הוא:

∫ f (x)dx = ∫ ln(x)dx = x ∙ (ln(x) - 1) + C

Ln של 0

הלוגריתם הטבעי של אפס אינו מוגדר:

ln(0) is undefined

הגבול ליד 0 של הלוגריתם הטבעי של x, כאשר x מתקרב לאפס, הוא מינוס אינסוף:

Ln של 1

הלוגריתם הטבעי של אחד הוא אפס:

ln(1) = 0

Ln של אינסוף

הגבול של הלוגריתם הטבעי של האינסוף, כאשר x מתקרב לאינסוף שווה לאינסוף:

lim ln(x) = ∞, when x→∞

לוגריתם מורכב

עבור מספר מרוכב z:

z = reiθ = x + iy

הלוגריתם המורכב יהיה (n = ...-2,-1,0,1,2,...):

Log z = ln(r) + i(θ+2nπ) = ln(√(x2+y2)) + i·arctan(y/x))
