כיצד להעביר קבצים בלינוקס/יוניקס

Unix/Linux העבר קבצים.

 

תחביר העברה של קובץ:

$ mv [options] sourcefiles destdir

 

העבר קבצי main.c def.h לספריית / home/usr/rapid/

$ mv main.c def.h /home/usr/rapid/

 

העבר את כל קבצי C בספרייה הנוכחית לתת-ספריית bak

$ mv *.c bak
