פקודת pwd בלינוקס/יוניקס

פקודת Unix/Linux pwd.

pwd - הדפס ספריית עבודה, היא פקודת לינוקס כדי לקבל את ספריית העבודה הנוכחית.

  • תחביר pwd
  • דוגמאות של pwd

תחביר pwd

$ pwd [option]

דוגמאות לפקודות pwd

שנה את הספרייה לספריית /usr/src והדפיס את ספריית העבודה:

$ cd /usr/src
$ pwd
/user/src

 

שנה את הספרייה לספריית הבית והדפיס את ספריית העבודה:

$ cd ~
$ pwd
/home/user

 

שנה את הספרייה לספריית האב של ספריית הבית והדפיס את ספריית העבודה:

$ cd ~/..
$ pwd
/home

 

שנה את הספרייה לספריית שורש והדפס את ספריית העבודה:

$ cd /
$ pwd
/
