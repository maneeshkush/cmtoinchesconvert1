מהדר GCC C

GCC הוא קיצור של GNU Compiler Collection, מהדר C עבור לינוקס.

תחביר GCC

$ gcc [options] [source files] [object files] [-o output file]

אפשרויות GCC

אפשרויות עיקריות של GCC:

אוֹפְּצִיָה תיאור
gcc -c הידור קובצי מקור לקבצי אובייקט ללא קישור
gcc -D שם [ =value ] להגדיר מאקרו קדם-מעבד
gcc -fPIC יצירת קוד בלתי תלוי במיקום עבור ספריות משותפות
רמת gcc -g ליצור מידע באגים לשימוש על ידי GDB
gcc -I dir add include directory של קבצי כותרת
gcc -l lib קישור עם קובץ הספרייה
gcc -L dir חפש בספרייה עבור קבצי ספרייה
קובץ הפלט gcc -o כתוב פלט בנייה לקובץ פלט
רמת gcc -O בצע אופטימיזציה עבור גודל הקוד וזמן הביצוע
gcc -משותף ליצור קובץ אובייקט משותף לספרייה משותפת
שם gcc -U בטל הגדרה של מאקרו קדם-מעבד
gcc -w השבת את כל הודעות האזהרה
gcc -קיר אפשר את כל הודעות האזהרה
gcc -Wextra אפשר הודעות אזהרה נוספות

דוגמאות של GCC

קומפיל את file1.c ו- file2.c וקשר לקובץ הפלט execfile :

$ gcc file1.c file2.c -o execfile

 

הפעל את קובץ הפלט execfile :

$ ./execfile

 

קומפיל את file1.c ו- file2.c ללא קישור:

$ gcc -c file1.c file2.c

 

הידור myfile.c עם מידע באגים וקישור לקובץ הפלט execfile :

$ gcc -g myfile.c -o execfile

 

קומפיל את myfile.c עם הודעות אזהרה מופעלות וקישור לקובץ הפלט execfile :

$ gcc -Wall myfile.c -o execfile

 

הידור myfile.c עם וקשר עם הספרייה הסטטית libmath.a הממוקמת ב- /user/local/math כדי לפלט קובץ execfile :

$ gcc -static myfile.c -L/user/local/math -lmath -o execfile

 

הידור myfile.c עם אופטימיזציה וקישור לקובץ הפלט execfile :

$ gcc -O myfile.c -o execfile
