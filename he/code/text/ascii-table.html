מהו קוד ASCII

ASCII (American Standard Code for Information Interchange) הוא קוד של 7 סיביות תווים, עם ערכים מ-0 עד 127. קוד ASCII הוא תת-קבוצה של קוד UTF-8. קוד ASCII כולל תווי בקרה ותווים הניתנים להדפסה: ספרות, אותיות רישיות ואותיות קטנות.

ASCII לעומת Unicode

ASCII הוא קוד של 7 סיביות, עם ערכים מ-0 עד 7F 16 . קוד תווי Unicode הוא ערכת-על של ASCII המכילה את קוד ASCII עם ערכים מ-0 עד 10FFFF 16
