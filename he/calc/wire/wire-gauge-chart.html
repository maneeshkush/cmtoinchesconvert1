חישובי מד חוט

חישובי קוטר חוט

קוטר החוט n מד d n באינצ'ים (אינץ') שווה ל-0.005in כפול 92 מורם בחזקת 36 פחות מד מספר n, חלקי 39:

dn (in) = 0.005 in × 92(36-n)/39

קוטר חוט n gauge d n במילימטרים (מ"מ) שווה ל-0.127 מ"מ כפול 92 מורם בחזקת 36 פחות מד מספר n, חלקי 39:

dn (mm) = 0.127 mm × 92(36-n)/39

חישובי שטח חתך תיל

שטח החוט הצולבי של חוט n n בקילו - מילים מעגליים (kcmil) שווה ל-1000 פעמים קוטר החוט המרובע d באינצ'ים (ב):

An (kcmil) = 1000×dn2 = 0.025 in2 × 92(36-n)/19.5

אז השטח הצולבי של חוט n מד A n באינצ'ים ריבועיים (ב -2 ) שווה ל-pi חלקי פי 4 מקוטר החוט המרובע d באינצ'ים (אינץ').

An (in2) = (π/4)×dn2 = 0.000019635 in2 × 92(36-n)/19.5

אז השטח הצולבי של החוט A n במילימטרים רבועים (מ"מ 2 ) שווה ל-pi חלקי פי 4 מקוטר החוט המרובע d במילימטרים (מ"מ).

An (mm2) = (π/4)×dn2 = 0.012668 mm2 × 92(36-n)/19.5

חישובי התנגדות חוטים

אז התנגדות חוט n מד R באוהם לקילו רגל (Ω/kft) שווה ל-0.3048×1000000000 פעמים ההתנגדות של החוט ρ במטר אוהם (Ω·m) חלקי 25.4 פי 2 משטח החתך A n באינטימטרים רבועים (ב -2 ).

Rn (Ω/kft) = 0.3048 × 109 × ρ(Ω·m) / (25.42 × An (in2))

אז התנגדות חוט n מד R באוהם לקילומטר (Ω/ק"מ) שווה ל-1000000000 פעמים התנגדות החוט ρ במטר אוהם (Ω·m) חלקי שטח החתך A n במילימטרים רבועים (מ"מ 2 ).

Rn (Ω/km) = 109 × ρ(Ω·m) / An (mm2)
