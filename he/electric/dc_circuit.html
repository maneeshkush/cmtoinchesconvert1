חוקי מעגל DC

חוק אוהם

I = V / R

חוק ג'ול

P = V · I = I 2 · R = V 2 / R

חוקי מעגל סדרה

VT = V1 + V2 + V3 + ...

IT = I1 = I2 = I3 = ...

RT = R1 + R2 + R3 + ...

1/CT = 1/C1 + 1/C2 + 1/C3 + ...

LT = L1 + L2 + L3 + ...

חוקי מעגל מקבילים

VT = V1 = V2 = V3 = ...

IT = I1 + I2 + I3 + ...

1/RT = 1/R1 + 1/R2 + 1/R3 + ...

CT = C1 + C2 + C3 + ...

1/LT = 1/L1 + 1/L2 + 1/L3 + ...

חלוקת מתח

V1 = VT ⋅ R1 / (R1+R2+R3+...)

החלוקה הנוכחית

I1 = IT ⋅ (R2+R3+...) / (R1+R2+R3+...)

חוק המתח של קירכהוף (KVL)

סכום נפילות המתח בלולאת זרם הוא אפס:

∑ Vi = 0

החוק הנוכחי של קירכהוף (KCL)

החיבור בין מספר אלמנטים במעגל נקרא צומת.

סכום ערכי הזרמים בצומת הוא אפס:

∑ I i = 0

קיבול

C = Q / V

קבל צלחת מקבילה

C = ε ⋅ A / l

ε הוא הפריטטיביות בפאראד למטר (F/m).

מתירנות

ε = ε0 ⋅ εr

ε 0 היא הפריטטיביות בוואקום.

ε r הוא השרירנות היחסית או הקבוע הדיאלקטי.

זרם של קבל

IC(t) = C d VC(t) / dt

מתח של קבלים

V C (t) = V C (0) + 1/ C ∫ I C (t)⋅ dt

מתח של משרן

VL(t) = L d IL(t) / dt

זרם של משרן

I L (t) = I L (0) + 1/ L ∫ V L (t)⋅ dt

אנרגיה של קבלים

WC = C⋅V 2 / 2

אנרגיה של משרן 

WL = L⋅I 2 / 2
