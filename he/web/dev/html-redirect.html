הפניית HTML

הפניית HTML. HTML מטא רענון קוד ניתוב מחדש.

הפנייה מחדש של HTML meta refresh היא הפניה מחדש בצד הלקוח ואינה הפניה קבועה מסוג 301.

רענון המטא של HTML עם מרווח זמן של 0 שניות, נחשב על ידי גוגל כרמז להפניה מחדש של 301 להעברת דירוג דף.

אם אתה רוצה לבצע הפניה קבועה 301 אמיתית, אתה יכול לעשות זאת עם הפניית PHP לאחר הפעלת קוד PHP בקובצי HTML.

הפניה מחדש של מטא רענון HTML

ההפניה מחדש מתבצעת עם מטא רענון בקטע הראש.

הקישור בקטע הגוף למטרות חזרה.

החלף את הדף הישן בקוד ניתוב מחדש בכתובת האתר של הדף שאליו ברצונך להפנות מחדש.

old-page.html:

<!DOCTYPE html>
<html>
<head>
   <!-- HTML meta refresh URL redirection -->
   <meta http-equiv="refresh"
   content="0; url=http://www.mydomain.com/new-page.html">
</head>
<body>
   <p>The page has moved to:
   <a href="http://www.mydomain.com/new-page.html">this page</a></p>
</body>
</html>

דוגמה להפניה מחדש של HTML meta refresh

html-redirect-test.htm:

<!DOCTYPE html>
<html>
<head>
   <!-- HTML meta refresh URL redirection -->
   <meta http-equiv="refresh"
   content="0; url=https://www.cmtoinchesconvert.com/he/web/dev/html-redirect.htm">

   <p>The page has moved to:

</html>

הפניה מחדש של תג קישור קנוני HTML

הקישור הקנוני אינו מפנה לכתובת ה-URL המועדפת, אך הוא יכול להוות חלופה להפניה מחדש של כתובות אתרים שרוב התעבורה מגיעה ממנועי חיפוש.

ניתן להשתמש בתג קישור קנוני ב-HTML כאשר ישנם מספר דפים עם תוכן דומה ואתם רוצים לומר למנועי החיפוש באיזה עמוד אתם מעדיפים להשתמש בתוצאות החיפוש.

תג קישור קנוני יכול לקשר לאותו דומיין וגם חוצה דומיינים.

הוסף את תג הקישור הקנוני לדף הישן כדי לקשר לדף החדש.

הוסף את תג הקישור הקנוני לדפים שאתה מעדיף שלא כדי לגרום לתנועה של מנועי החיפוש לקשר לעמוד המועדף.

יש להוסיף את תג הקישור הקנוני בקטע <head>.

old-page.html:

<link rel="canonical" href="http://www.mydomain.com/new-page.html">
