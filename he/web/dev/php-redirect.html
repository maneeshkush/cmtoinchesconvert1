הפניה מחדש של PHP

הפניית PHP מדף לכתובת אתר. הפניה מחדש של PHP 301.

הפנייה מחדש של PHP אמורה להחזיר קוד סטטוס תגובת HTTP: 301 Moved Permanently.

מנועי חיפוש משתמשים בקוד סטטוס התגובה 301 כדי להעביר את דירוג הדף מכתובת האתר הישנה לכתובת האתר החדשה.

הפניית כותרת PHP

החלף את קוד old-page.php בקוד ניתוב מחדש ל-new-page.php.

old-page.php:

<?php
// PHP permanent URL redirection
header("Location: http://www.domain.com/new-page.php", true, 301);
exit();
?>

הדף הישן חייב להיות בעל סיומת קובץ php.

הדף החדש יכול להיות עם כל הרחבה.

דוגמאות להפניה מחדש של PHP

דוגמה מס' 1

php-redirect-test.php

<?php
// PHP permanent URL redirection test
header("Location: https://www.cmtoinchesconvert.com/he/web/dev/php-redirect.html", true, 301);
exit();
?>
