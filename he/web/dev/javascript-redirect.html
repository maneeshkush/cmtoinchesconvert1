כתובת אתר להפניה מחדש של Javascript

כיצד להפנות דף כתובת אתר עם Javascript.

הפנייה מחדש של Javascript אינה מחזירה את קוד סטטוס ההפניה הקבועה 301.

הפניית javscript

החלף את הדף הישן בקוד ניתוב מחדש בכתובת האתר של הדף שאליו ברצונך להפנות מחדש.

old-page.html:

<!DOCTYPE html>
<html>
<body>
<script type="text/javascript">
    // Javascript URL redirection
    window.location.replace("http://www.mydomain.com/new-page.html");
</script>
</body>
</html>

 

מנועי חיפוש משתמשים בקוד הסטטוס 301 כדי להעביר את דירוג הדף מכתובת האתר הישנה לכתובת ה-URL החדשה.

קוד סטטוס תגובת http: 200 אישור.

אז ניתוב מחדש של Javascript אינו ידידותי למנועי חיפוש ועדיף להשתמש בשיטות הפנייה אחרות המחזירות קוד סטטוס: 301 Moved Permanently.

<script type="text/javascript">
// Javascript URL redirection

</script>
