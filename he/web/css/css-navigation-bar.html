סרגל ניווט אופקי

קוד CSS:

<style>
    #h_nav_bar a { padding:15px; font-weight:bold; float:left; }
    #h_nav_bar a:link { color:#d0d0d0; background-color:#0000a0; }
    #h_nav_bar a:visited { color:#c0c0c0; background-color:#0000a0; }
    #h_nav_bar a:hover { color:#ffffff; background-color:#000060; }
    #h_nav_bar a:active { color:#f0f0f0; background-color:#00ff00; }
</style>

</div>

קישור עוגן צבע קישור קישור למייל קישור לתמונה קישור טקסט

 

 

 

#link_bar a הוא הסגנון עבור כל המצבים של הקישור.

#link_bar a:link הוא הסגנון של קישור רגיל.

#link_bar א: ביקר הוא סגנון הקישור שבו ביקר.

#link_bar a:hover הוא הסגנון של קישור מרחף בעכבר.

#link_bar a: פעיל הוא סגנון הקישור בלחיצה על ידי העכבר.
