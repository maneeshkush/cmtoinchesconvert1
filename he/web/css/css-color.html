צבע CSS

קוד הצבע יכול להיות אחד מ:

פורמט hex: #rrggbb

פורמט RGB: rgb (אדום, ירוק, כחול)

פורמט שם: שם

דוגמא

עבור צבע כתום:

פורמט hex: #FFA500

פורמט RGB: rgb(255,165,0)

פורמט השם: כתום

הגדרת צבע אלמנט

עבור אלמנט ספציפי:

<element style="color: code;"></element>

עבור כל האלמנטים מאותו סוג. שים קוד בתג <style> בקטע head:

<style>
   element { color: code; }
</style>

דוגמא

הגדרת צבע הטקסט של הפסקה לאדום:

הגדרת צבע של פסקה ספציפית:

<p style="color: #FF0000;">Some text ...</p>

נוף:

קצת טקסט...

הגדרת צבע של כל הפסקאות

<style>
   p { color: #FF0000; }
</style>

הגדרת צבע רקע של אלמנט

element { background: code; }

דוגמא

הגדרת צבע הרקע של הפסקה לאדום:

הגדרת צבע רקע של פסקה ספציפית:

<p style="background: #FF0000;">Some text ...</p>

נוף:

קצת טקסט...

הגדרת צבע הרקע של כל הפסקאות:

<style>
   p { background: #FF0000; }
</style>

הגדרת צבע גבול של אלמנט

element { border-color: topcode rightcode bottomcode leftcode }

דוגמא

הגדרת צבע גבול הפסקה לאדום, ירוק, כחול ושחור:

הגדרת צבע גבול של פסקה ספציפית:

<p style="border-color: #FF0000 #00FF00 #0000FF #000000; border-style:solid">Some text ...</p>

נוף:

קצת טקסט...

הגדרת צבע הגבול של כל הפסקאות:

<style>
   p { border-color: #FF0000 #00FF00 #0000FF #000000; }
</style>

תכונות של כלי קודי צבע CSS

כלי קודי הצבע שלנו ב-CSS מאפשר למשתמשים להשתמש בקודי צבע ב-CSS. כמה מהתכונות הבולטות של כלי השירות הזה מוסברים להלן.

פַּשְׁטוּת

הממשק הידידותי למשתמש של כלי קודי הצבע שלנו ב-CSS מספק לך את הקלות של ביצוע המרה זו באופן מיידי. אינך צריך עוד לרכוש מיומנויות תכנות עבור קודי צבע CSS בשל הפשטות של כלי זה.

אין רישום

אינך צריך לעבור שום תהליך רישום כדי להשתמש בקודי הצבע של CSS. באמצעות כלי זה, אתה יכול להשתמש ב-CSS צבעוני בכלי כמה פעמים שתרצה בחינם.

הִטַלטְלוּת

ניתן לגשת לקודי צבע CSS זה מכל פינה בעולם. אתה לא צריך לדאוג לגבי התאימות של הכלי המקוון הזה, מכיוון שאתה יכול להשתמש בו בקודי CSS מכל מכשיר. כל מה שאתה צריך הוא חיבור לאינטרנט כדי לגשת ולשימוש בכלי קודי צבע CSS זה.

חוסך זמן ומאמץ

קודי הצבע של CSS מאפשרים לך לבצע את אותה משימה באופן מיידי. לא תתבקש לבצע נהלים ידניים, שכן האלגוריתמים האוטומטיים שלו יעשו את העבודה בשבילך.

תְאִימוּת

כלי קודי הצבע CSS עובד בצורה מושלמת על כל מערכות ההפעלה. בין אם יש לך מכשיר Mac, iOS, Android, Windows או Linux, אתה יכול להשתמש בקלות בכלי השירות המקוון הזה מבלי להתמודד עם כל טרחה.

100% חינם

אינך צריך לעבור שום תהליך רישום כדי להשתמש בכלי קודי צבע CSS זה. אתה יכול להשתמש בכלי זה בחינם ולעשות קודי צבע CSS ללא הגבלה ללא כל הגבלה.
