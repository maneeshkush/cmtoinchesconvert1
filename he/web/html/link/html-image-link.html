קישור לתמונה ב-HTML

איך להפוך תמונה לקישור.

קוד קישור לתמונה HTML

<a href="../html-link.htm"><img src="flower.jpg" width="82" height="86" title="White flower" alt="Flower"></a>

לחלופין, השתמש בעיצוב CSS כדי לקבוע את הרוחב והגובה.

<a href="../html-link.htm"><img src="flower.jpg" style="width:82px; height:86px" title="White flower" alt="Flower"></a>

הקוד ייצור את הקישור הזה:

 

לקוד יש את החלקים הבאים:

  • <a> הוא תג הקישור.
  • תכונת href מגדירה את כתובת האתר שאליה יש קישור.
  • <img> הוא תג ההתחלה של התמונה.
  • תכונת src מגדירה את קובץ התמונה.
  • תכונת title מגדירה את טקסט תיאור הכלי של התמונה.
  • alt הוא תכונת הטקסט האלטרנטי של תג התמונה.
  • תכונת style קובעת עם css את הרוחב והגובה של התמונה.
  • </a> הוא תג סיום הקישור.
