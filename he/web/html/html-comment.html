דוגמאות להערות HTML

הערות טקסט אינפורמטיביות

השתמש בהערות טקסט אינפורמטיביות כאשר אתה רוצה להוסיף הסברים לקוד.

קוד ה-HTML:

<!-- this is a single line comment -->
<p> some text ... </p>
<!--
    this is
    a multiline
    comment
-->
<p> some more text ... </p>

 

תצוגת דף התוצאות:

some text ...
some more text ...

 

הערות קוד

השתמש בהערות קוד כאשר אתה רוצה להשאיר את הקוד לא פעיל.

קוד ה-HTML:

<!-- <p> some text ... </p> -->
<!--
<p> some text ... </p>
<p> some text ... </p>
-->
<p> some more text ... </p>

 

תצוגת דף התוצאות:

some more text ...
