cp -R-Befehl

cp -R Befehl unter Linux/Unix.

Der Befehl cp -R wird zum rekursiven Kopieren aller Dateien und Verzeichnisse im Quellverzeichnisbaum verwendet.

Syntax

$ cp -R srcdir destdir

Beispiel

Mit ausführlich (-v):

$ cp -Rv dev bak
'dev/main.c' -> 'bak/dev/main.c'
'dev/test.c' -> 'bak/dev/test.c'
$
