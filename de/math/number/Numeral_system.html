Zahlensystem

b - Basis des Zahlensystems

d n - die n-te Ziffer

n - kann mit einer negativen Zahl beginnen, wenn die Zahl einen Bruchteil hat.

N +1 - die Anzahl der Ziffern

Binäres Zahlensystem - Basis-2

Binärzahlen verwenden nur die Ziffern 0 und 1.

B bezeichnet binäres Präfix.

Beispiele:

101012 = 10101B = 1×24+0×23+1×22+0×21+1×20 = 16+4+1= 21

101112 = 10111B = 1×24+0×23+1×22+1×21+1×20 = 16+4+2+1= 23

1000112 = 100011B = 1×25+0×24+0×23+0×22+1×21+1×20 =32+2+1= 35

Oktales Zahlensystem - Basis-8

Oktalzahlen verwenden Ziffern von 0..7.

Beispiele:

278 = 2×81+7×80 = 16+7 = 23

308 = 3×81+0×80 = 24

43078 = 4×83+3×82+0×81+7×80= 2247

Dezimalzahlensystem - Basis-10

Dezimalzahlen verwenden Ziffern von 0..9.

Dies sind die regulären Nummern, die wir verwenden.

Beispiel:

253810 = 2×103+5×102+3×101+8×100

Hexadezimales Zahlensystem - Basis-16

Hex-Zahlen verwenden Ziffern von 0..9 und A..F.

H bezeichnet Hex-Präfix.

Beispiele:

2816 = 28H = 2×161+8×160 = 40

2F16 = 2FH = 2×161+15×160 = 47

BC1216 = BC12H = 11×163+12×162+1×161+2×160= 48146
