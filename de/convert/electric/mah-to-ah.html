So konvertieren Sie mAh in Ah

So wandeln Sie die elektrische Ladung von Milliamperestunden (mAh) in Amperestunden (Ah) um.

Umrechnung von Milliamperestunden in Amperestunden

Um elektrische Ladung in Milliamperestunden Q (mAh) in elektrische Ladung in Amperestunden Q (Ah) umzurechnen , können Sie die folgende Formel verwenden:

Q(Ah) = Q(mAh) / 1000

 

Eine Amperestunde ist also gleich einer Milliamperestunde geteilt durch 1000:

ampere-hours = milliampere-hours / 1000

oder

Ah = mAh / 1000

Beispiel

  • Q (Ah) ist die elektrische Ladung in Amperestunden und
  • Q (mAh) ist die elektrische Ladung in Milliamperestunden.

Um die Formel zu verwenden, ersetzen Sie einfach den Wert für Q (mAh) in Milliamperestunden in die Gleichung und lösen Sie für Q (Ah) in Amperestunden auf.

Wenn Sie beispielsweise eine elektrische Ladung von 200 Milliamperestunden haben, können Sie diese wie folgt in Amperestunden umrechnen:

Q = 200 mAh / 1000 = 0,2 Ah

Das bedeutet, dass die elektrische Ladung 0,2 Amperestunden beträgt.

Es ist wichtig zu beachten, dass diese Formel nur für die Umrechnung elektrischer Ladung von Milliamperestunden in Amperestunden gilt. Wenn Sie versuchen, die elektrische Ladung von einer anderen Einheit umzurechnen, müssen Sie eine andere Formel verwenden.
