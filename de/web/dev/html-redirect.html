HTML-Umleitung

HTML-Umleitung. Umleitungscode für die HTML-Metaaktualisierung.

Die HTML-Meta-Aktualisierungsumleitung ist eine clientseitige Umleitung und keine permanente 301-Umleitung.

Der HTML-Meta-Refresh mit 0-Sekunden-Zeitintervall wird von Google als Hinweis auf eine 301-Weiterleitung zur Pagerank-Übertragung gewertet.

Wenn Sie eine echte permanente 301-Umleitung durchführen möchten, können Sie dies mit der PHP-Umleitung tun, nachdem Sie den PHP-Code in HTML-Dateien aktiviert haben.

HTML-Meta-Aktualisierungsumleitung

Die Umleitung erfolgt mit dem Meta-Refresh im Head-Bereich.

Der Link im Body-Abschnitt für Fallback-Zwecke.

Ersetzen Sie alte Seite mit Umleitungscode durch die URL der Seite, auf die Sie umleiten möchten.

alte-seite.html:

<!DOCTYPE html>
<html>
<head>
   <!-- HTML meta refresh URL redirection -->
   <meta http-equiv="refresh"
   content="0; url=http://www.mydomain.com/new-page.html">
</head>
<body>
   <p>The page has moved to:
   <a href="http://www.mydomain.com/new-page.html">this page</a></p>
</body>
</html>

Beispiel für eine HTML-Meta-Refresh-Weiterleitung

html-weiterleitungstest.htm:

<!DOCTYPE html>
<html>
<head>
   <!-- HTML meta refresh URL redirection -->
   <meta http-equiv="refresh"
   content="0; url=https://www.cmtoinchesconvert.com/de/web/dev/html-redirect.htm">
</head>
<body>
   <p>The page has moved to:
   <a href="https://www.cmtoinchesconvert.com/de/web/dev/html-redirect.htm">this page</a></p>
</body>
</html>

HTML-Canonical-Link-Tag-Weiterleitung

Der kanonische Link leitet nicht auf die bevorzugte URL um, aber er kann eine Alternative zur URL-Umleitung für Websites sein, auf denen der meiste Traffic von Suchmaschinen kommt.

HTML Canonical Link Tag kann verwendet werden, wenn es mehrere Seiten mit ähnlichen Inhalten gibt und Sie den Suchmaschinen mitteilen möchten, welche Seite Sie in den Suchergebnissen bevorzugen.

Das kanonische Link-Tag kann auf dieselbe Domäne und auch domänenübergreifend verlinken.

Fügen Sie der alten Seite das kanonische Link-Tag hinzu, um auf die neue Seite zu verlinken.

Fügen Sie das Tag für kanonische Links zu den Seiten hinzu, die Sie bevorzugen, um keinen Suchmaschinenverkehr zu erhalten, um auf die bevorzugte Seite zu verlinken.

Das kanonische Link-Tag sollte im Abschnitt <head> hinzugefügt werden.

alte-seite.html:

<link rel="canonical" href="http://www.mydomain.com/new-page.html">
