CSS-Farbe

Der Farbcode kann einer der folgenden sein:

Hex-Format: #rrggbb

RGB-Format: rgb (rot, grün, blau)

Namensformat: Name

Beispiel

Für orange Farbe:

Hex-Format: #FFA500

RGB-Format: rgb(255,165,0)

Namensformat: orange

Elementfarbe einstellen

Für bestimmtes Element:

<element style="color: code;"></element>

Für alle Elemente des gleichen Typs. Fügen Sie den Code in das Tag <style> im Head-Bereich ein:

<style>
   element { color: code; }
</style>

Beispiel

Textfarbe des Absatzes auf Rot setzen:

Einstellen der Farbe eines bestimmten Absatzes:

<p style="color: #FF0000;">Some text ...</p>

Farbe aller Absätze einstellen

<style>
   p { color: #FF0000; }
</style>

Hintergrundfarbe des Elements festlegen

element { background: code; }

Beispiel

Hintergrundfarbe des Absatzes auf Rot setzen:

Einstellen der Hintergrundfarbe eines bestimmten Absatzes:

<p style="background: #FF0000;">Some text ...</p>

Sicht:

Etwas Text ...

Hintergrundfarbe aller Absätze einstellen:

<style>
   p { background: #FF0000; }
</style>

Rahmenfarbe des Elements einstellen

element { border-color: topcode rightcode bottomcode leftcode }

Beispiel

Einstellen der Absatzrandfarbe auf Rot, Grün, Blau und Schwarz:

Rahmenfarbe eines bestimmten Absatzes einstellen:

<p style="border-color: #FF0000 #00FF00 #0000FF #000000; border-style:solid">Some text ...</p>
