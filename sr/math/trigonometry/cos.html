Дефиниција косинуса

У правоуглом троуглу АБЦ синус од α, син(α) је дефинисан као однос између странице суседне угла α и стране супротне од правог угла (хипотенузе):

cos α = b / c

Пример

b = 3"

c = 5"

cos α = b / c = 3 / 5 = 0.6

Инверзна косинусна функција

Аркосинусод к је дефинисан као инверзна косинусна функција к када је -1≤к≤1 .

Када је косинус од и једнак к:

cos y = x

Тада је аркосинус од к једнак инверзној косинусној функцији к, која је једнака и:

arccos x = cos-1 x = y

Пример

arccos 1 = cos-1 1 = 0 rad = 0°
