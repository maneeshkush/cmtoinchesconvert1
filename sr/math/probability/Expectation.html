Очекивана вредност

У вероватноћи и статистици, очекивање или очекивана вредност је пондерисана просечна вредност случајне променљиве.

Очекивање континуиране случајне променљиве

Е ( Кс ) је очекивана вредност континуиране случајне променљиве Кс

к је вредност континуиране случајне променљиве Кс

П ( к ) је функција густине вероватноће

Очекивање дискретне случајне променљиве

Е ( Кс ) је очекивана вредност континуиране случајне променљиве Кс

к је вредност континуиране случајне променљиве Кс

П ( к ) је функција масе вероватноће Кс

Својства очекивања

Линеарност

Када је а константа и Кс,И су случајне променљиве:

E(aX) = aE(X)

E(X+Y) = E(X) + E(Y)

Константно

Када је ц константно:

E(c) = c

Производ

Када су Кс и И независне случајне променљиве:

E(X ⋅Y) = E(X) ⋅ E(Y)
