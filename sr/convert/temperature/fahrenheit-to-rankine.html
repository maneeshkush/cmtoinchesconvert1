Како претворити Фаренхајт у Ранкине

Дакле, 0 степени Фаренхајта је једнако [459,67] степени Ранкина.

0 °F = 459.67 °R

Дакле, температура Т  у степенима Ранкине (°Р) једнака је температури  Т у степенима Фаренхајта (°Ф) плус [459,67].

T(°R) = T(°F) + 459.67

Пример 1

Претворите 55 степени Фаренхајта у степене Ранкине:

T(°R) = 55°F + 459.67 = 514.67 °R

Пример 2

Пример 3

Пример 4
