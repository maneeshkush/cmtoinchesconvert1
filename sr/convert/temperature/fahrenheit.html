Фаренхајта

Фаренхајт је јединица мерења температуре.

Тачка смрзавања / топљења воде при притиску од 1 атмосфере је око 32 °Ф.

Симбол за степен Фаренхајта је °Ф.

Претворба Фаренхајта у Целзијус

Дакле, 0 степени Фаренхајта је једнако -17,77778 степени Целзијуса.

0 °F = -17.77778 °C

Дакле, температура Т  у степенима  Целзијуса (°Ц)  једнака је температури  Т у степенима Фаренхајта (°Ф) минус 32 пута [5/9].

T(°C) = (T(°F) - 32) × 5/9

Пример 1

Претворите 70 степени Фаренхајта у степене Целзијуса:

T(°C) = (70°F - 32) × 5/9 = 21.1111 °C

Пример 2

Претворите 80 степени Фаренхајта у степене Целзијуса:

T(°C) = (80°F - 32) × 5/9 = 26.6667 °C

Пример 3

Претворите 100 степени Фаренхајта у степене Целзијуса:

T(°C) = (100°F - 32) × 5/9 = 37.7778 °C

Претворба Фаренхајта у Ренкин

Дакле, 0 степени Фаренхајта је једнако [459,67] степени Ранкина.

0 °F = 459.67 °R

Температура  Т  у степенима Ранкине (°Р) једнака је температури  Т у степенима Фаренхајта (°Ф) плус [459,67].

T(°R) = T(°F) + 459.67

Пример 1

Претворите 70 степени Фаренхајта у степене Ранкине:

T(°R) = 70°F + 459.67 = 529.67 °R

Пример 2

Претворите 90 степени Фаренхајта у степене Ранкине:

T(°R) = 90°F + 459.67 = 549.67 °R

Пример 3

Претворите 100 степени Фаренхајта у степене Ранкине:

T(°R) = 100°F + 459.67 = 559.67 °R

Претворба Фаренхајта у Келвин

Дакле, температура Т  у  Келвинима (К)  је једнака температури  Т у степенима Фаренхајта (°Ф) плус 459,67 пута [5/9].

T(K) = (T(°F) + 459.67)× 5/9

Пример 1

Претворите 65 степени Фаренхајта у степени Келвина:

T(K) = (65°F + 459.67)× 5/9 = 291.483 K

Пример 2

T(K) = (80°F + 459.67)× 5/9 = 299.817 K

Пример 3

Претворите 100 степени Фаренхајта у степени Келвина:

T(K) = (100°F + 459.67)× 5/9 = 310.928 K
