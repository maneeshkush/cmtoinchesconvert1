гцц -И опција заставица

гцц -И додаје укључи директоријум датотека заглавља.

Синтакса

$ gcc -Idir [options] [source files] [object files] [-o output file]

Пример

прој/срц/михеадер.х:

// myheader.h
#define NUM1 5

 

мифиле.ц:

// myfile.c
#include <stdio.h>
#include "myheader.h"
 
void main()
{
    int num = NUM1;
    printf("num=%d\n", num);
}

 

Направи мифиле.ц без укључивања директоријума прој/срц :

$ gcc myfile.c -o myfile
myfile.c:2:22: fatal error: myheader.h: No such file or directory
compilation terminated.
$

 

Направи мифиле.ц са укљученим директоријумом прој/срц :

$ gcc -Iproj/src myfile.c -o myfile
$ ./myfile
num=5
$
