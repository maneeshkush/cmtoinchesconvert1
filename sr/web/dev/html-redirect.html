ХТМЛ преусмеравање

ХТМЛ преусмеравање.ХТМЛ мета освежавајући код за преусмеравање.

ХТМЛ мета освежавање преусмеравање је преусмеравање на страни клијента и није 301 трајно преусмеравање.

ХТМЛ мета освежавање са временским интервалом од 0 секунди, Гоогле сматра наговештајем за 301 преусмеравање за пренос ранга странице.

Ако желите да извршите право 301 трајно преусмеравање, можете то учинити помоћу ПХП преусмеравања након што омогућите ПХП код у ХТМЛ датотекама.

ХТМЛ мета освежавање преусмеравање

Преусмеравање се врши помоћу мета освежавања у одељку главе.

Веза у одељку тела за резервне сврхе.

Замените стару страницу кодом за преусмеравање УРЛ-ом странице на коју желите да преусмерите.

олд-паге.хтмл:

<!DOCTYPE html>
<html>
<head>
   <!-- HTML meta refresh URL redirection -->
   <meta http-equiv="refresh"
   content="0; url=http://www.mydomain.com/new-page.html">
</head>
<body>
   <p>The page has moved to:
   <a href="http://www.mydomain.com/new-page.html">this page</a></p>
</body>
</html>

Пример преусмеравања мета освежавања ХТМЛ-а

хтмл-редирецт-тест.хтм:

<!DOCTYPE html>
<html>
<head>
   <!-- HTML meta refresh URL redirection -->
   <meta http-equiv="refresh"
   content="0; url=https://www.cmtoinchesconvert.com/sr/web/dev/html-redirect.htm">
</head>
<body>
   <p>The page has moved to:
   <a href="https://www.cmtoinchesconvert.com/sr/web/dev/html-redirect.htm">this page</a></p>
</body>
</html>

 

Притисните ову везу да бисте преусмерили са хтмл-редирецт-тест.хтм назад на ову страницу:

 

Тест преусмеравања мета освежавања ХТМЛ-а

Преусмеравање канонске ознаке везе ХТМЛ

Канонска веза не преусмерава на жељени УРЛ, али може бити алтернатива преусмеравању УРЛ-а за веб локације на које највећи део саобраћаја стиже са претраживача.

ХТМЛ канонска ознака везе се може користити када постоји неколико страница са сличним садржајем и желите да кажете претраживачима коју страницу желите да користите у резултатима претраге.

Ознака канонске везе може да се повезује са истим доменом и такође унакрсним доменом.

Додајте канонску ознаку везе на стару страницу да бисте повезали нову страницу.

Додајте канонску ознаку везе на странице за које не желите да саобраћај претраживача води ка жељеној страници.

Ознаку канонске везе треба додати у одељак <хеад>.

олд-паге.хтмл:

<link rel="canonical" href="http://www.mydomain.com/new-page.html">
