Јавасцрипт УРЛ за преусмеравање

Како да преусмерите УРЛ страницу помоћу Јавасцрипт-а.

Јавасцрипт преусмеравање не враћа 301 стални статусни код преусмеравања.

Јавсцрипт преусмеравање

Замените стару страницу кодом за преусмеравање УРЛ-ом странице на коју желите да преусмерите.

олд-паге.хтмл:

<!DOCTYPE html>
<html>
<body>
<script type="text/javascript">
    // Javascript URL redirection
    window.location.replace("http://www.mydomain.com/new-page.html");
</script>
</body>
</html>

 

Претраживачи користе статусни код 301 да пренесу ранг странице са старог УРЛ-а на нови УРЛ.

Јавасцрипт преусмеравање враћа хттп статусни код одговора: 200 ОК.

Дакле, Јавасцрипт преусмеравање није прилагођено претраживачима и боље је користити друге методе преусмеравања које враћају статусни код: 301 Трајно померено.

Пример Јавасцрипт преусмеравања

јавасцрипт-редирецт-тест.хтм

<!DOCTYPE html>
<html>
<body>
<script type="text/javascript">
// Javascript URL redirection
window.location.replace("https://www.cmtoinchesconvert.com/sr/web/dev/javascript-redirect.htm");
</script>
</body>
</html>
