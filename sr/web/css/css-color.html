ЦСС боја

Код боје може бити један од:

Хексадецимални формат: #ррггбб

РГБ формат: ргб (црвена, зелена, плава)

Формат имена: име

Пример

За наранџасту боју:

Хек формат: #ФФА500

РГБ формат: ргб(255,165,0)

Формат имена: наранџаста

Подешавање боје елемента

За одређени елемент:

<element style="color: code;"></element>

За све елементе истог типа.Ставите код у ознаку <стиле> у одељку хеад:

<style>
   element { color: code; }
</style>

Пример

Постављање боје текста пасуса на црвену:

Подешавање боје одређеног пасуса:

<p style="color: #FF0000;">Some text ...</p>

Поглед:

Неки текст...

Подешавање боје свих пасуса

<style>
   p { color: #FF0000; }
</style>

Подешавање боје позадине елемента

element { background: code; }

Пример

Постављање боје позадине пасуса на црвену:

Подешавање боје позадине одређеног пасуса:

<p style="background: #FF0000;">Some text ...</p>

Поглед:

Неки текст...

Подешавање боје позадине свих пасуса:

<style>
   p { background: #FF0000; }
</style>

Подешавање боје ивице елемента

element { border-color: topcode rightcode bottomcode leftcode }

Пример

Постављање боје границе пасуса на црвену, зелену, плаву и црну:

Подешавање боје ивице одређеног пасуса:

<p style="border-color: #FF0000 #00FF00 #0000FF #000000; border-style:solid">Some text ...</p>

Поглед:

Неки текст...

Подешавање боје ивице свих пасуса:

<style>
   p { border-color: #FF0000 #00FF00 #0000FF #000000; }
</style>

Карактеристике алата ЦСС кодова боја

Наш ЦСС алат за кодирање боја омогућава корисницима да ЦСС кодови боја.Неке од истакнутих карактеристика овог услужног програма су објашњене у наставку.

Једноставност

Корисничко сучеље нашег ЦСС алата за кодове боја пружа вам лакоћу да одмах извршите ову конверзију.Више не морате да стичете вештине програмирања за ЦСС кодове боја због једноставности овог алата.

Нема регистрације

Не морате да прођете било какав процес регистрације да бисте користили ЦСС кодове боја.Користећи овај услужни програм, можете бесплатно користити ЦСС алатку за кодирање боја колико год пута желите.

Преносивост

Овим ЦСС кодовима боја може се приступити из било ког угла света.Не морате да бринете о компатибилности овог онлајн алата, јер помоћу њега можете да кодирате ЦСС боје са било ког уређаја.Све што вам треба је интернет веза за приступ и коришћење овог ЦСС алата за кодове боја.

Штеди време и труд

ЦСС кодови боја вам омогућавају да одмах завршите исти задатак.Од вас се неће тражити да следите ручне процедуре, јер ће аутоматизовани алгоритми обавити посао уместо вас.

Компатибилност

ЦСС алатка за кодове боја савршено ради на свим оперативним системима.Без обзира да ли имате Мац, иОС, Андроид, Виндовс или Линук уређај, можете лако да користите овај онлајн услужни програм без икаквих проблема.

100% бесплатно

Не морате да прођете било какав процес регистрације да бисте користили ову ЦСС алатку за кодове боја.Можете користити овај услужни програм бесплатно и правити неограничене ЦСС кодове боја без икаквих ограничења.
