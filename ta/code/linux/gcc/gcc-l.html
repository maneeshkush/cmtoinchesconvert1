gcc -L / -l விருப்பம் கொடிகள்

gcc -l நூலகக் கோப்புடன் இணைக்கிறது.

gcc -L லைப்ரரி கோப்புகளுக்கான கோப்பகத்தில் தெரிகிறது.

தொடரியல்

$ gcc [options] [source files] [object files] [-Ldir] -llibname [-o outfile]

 

Lib முன்னொட்டு மற்றும் .a அல்லது .so நீட்டிப்புகள் இல்லாமல் நூலகப் பெயருடன் இணைப்பு -l.

எடுத்துக்காட்டுகள்

உதாரணம்1

நிலையான நூலக கோப்பு libmath க்கான. ஒரு பயன்பாடு -lmath :

$ gcc -static myfile.c -lmath -o myfile

 
எடுத்துக்காட்டு2

பகிரப்பட்ட நூலகக் கோப்பு libmathக்கு. எனவே -lmath பயன்படுத்தவும்:

$ gcc myfile.c -lmath -o myfile

 
உதாரணம்3

file1.c:

// file1.c
#include <stdio.h>

void main()
{
    printf("main() run!\n");
    myfunc();
}

 

file2.c:

// file2.c
#include <stdio.h>

void myfunc()
{
    printf("myfunc() run!\n");
}

 

file2.c ஐ உருவாக்கவும் , பொருள் கோப்பு file2.o ஐ libs கோப்பகத்திற்குநகலெடுத்து , அதை நிலையான நூலகத்தில் காப்பகப்படுத்தவும் libmylib.a :

$ gcc -c file2.c
$ mkdir libs
$ cp file2.o libs
$ cd libs
$ ar rcs libmylib.a file2.o

 

libs கோப்பகத்தில்நிலையான நூலகம் libmylib.a உடன் file1.c ஐ உருவாக்கவும் .

பிழையுடன் -L முடிவுகள் இல்லாமல் உருவாக்கவும்:

$ gcc file1.c -lmylib -o outfile
/usr/bin/ld: cannot find -llibs
collect2: ld returned 1 exit status
$

-L உடன் உருவாக்கி இயக்கவும்:

$ gcc file1.c -Llibs -lmylib -o outfile
$ ./outfile
main() run!
myfunc() run!
$
