cp மேலெழுதும்

cp Linux/Unix இல் கோப்புகள் / கோப்பகங்களை மேலெழுதுகிறது.

 

வழக்கமான சிபி பொதுவாக இலக்கு கோப்புகள் மற்றும் கோப்பகங்களை மேலெழுதுகிறது:

$ cp test.c bak

 

மேலெழுதுவதற்கு முன் ஊடாடும் வரியைச் சேர்க்க -i விருப்பத்தைப் பயன்படுத்தவும் மற்றும் மேலெழுத 'y' ஐ அழுத்தவும்:

$ cp -i test.c bak
cp: overwrite 'bak/test.c'? y

 

மேலெழுதுவதைத் தவிர்க்க -n விருப்பத்தைப் பயன்படுத்தவும்:

$ cp -n test.c bak

 

ப்ராம்ட் இல்லாமல் எப்போதும் மேலெழுத:

$ \cp test.c bak

 

 

மேலும் பார்க்கவும்

  • cp அடைவு பிழை செய்தியைத் தவிர்க்கிறது
  • cp -R கட்டளை
  • cp கட்டளை
