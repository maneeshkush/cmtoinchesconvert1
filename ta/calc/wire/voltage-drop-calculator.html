எனவே அங்குலங்களில் (in) உள்ள n கேஜ் கம்பி விட்டம் d n என்பது 0.005in மடங்கு 92 க்கு சமமாக 36 மைனஸ் கேஜ் எண் n இன் சக்திக்கு உயர்த்தப்பட்டு 39 ஆல் வகுக்கப்படுகிறது.

dn (in) = 0.005 in × 92(36-n)/39

எனவே மில்லிமீட்டர்களில் (மிமீ) உள்ள n கேஜ் கம்பி விட்டம் d n என்பது 0.127mm மடங்கு 92 க்கு சமமாக 36 மைனஸ் கேஜ் எண் n இன் சக்திக்கு உயர்த்தப்பட்டு, 39 ஆல் வகுக்கப்படுகிறது.

dn (mm) = 0.127 mm × 92(36-n)/39

An (in2) = (π/4)×dn2 = 0.000019635 in2 × 92(36-n)/19.5

எனவே n கேஜ் வயரின் குறுக்குவெட்டுப் பகுதி A n என்பது சதுர மில்லிமீட்டரில் (mm 2 ) pi க்கு சமம் என்பது 4 மடங்கு சதுர கம்பி விட்டம் d மில்லிமீட்டரில் (mm) வகுக்கப்படும்.

எங்கள் மின்னழுத்த துளி கால்குலேட்டர் பயனர்கள் மின்னழுத்த வீழ்ச்சியைக் கணக்கிட அனுமதிக்கிறது.இந்த பயன்பாட்டின் சில முக்கிய அம்சங்கள் கீழே விளக்கப்பட்டுள்ளன.

பதிவு இல்லை

வோல்டேஜ் டிராப் கால்குலேட்டரைப் பயன்படுத்த, நீங்கள் எந்தப் பதிவுச் செயல்முறையிலும் செல்ல வேண்டியதில்லை.இந்த பயன்பாட்டைப் பயன்படுத்தி, பயனர்கள் மின்னழுத்த வீழ்ச்சியை எத்தனை முறை வேண்டுமானாலும் இலவசமாகக் கணக்கிடலாம்.

வேகமான மாற்றம்

இந்த வோல்டேஜ் டிராப் கால்குலேட்டர் பயனர்களுக்கு வேகமான கணக்கீட்டை வழங்குகிறது.பயனர் உள்ளீட்டு புலத்தில் மின்னழுத்த துளி மதிப்புகளை உள்ளிட்டு, கணக்கிடு பொத்தானைக் கிளிக் செய்தவுடன், பயன்பாடு மாற்றும் செயல்முறையைத் தொடங்கி உடனடியாக முடிவுகளை வழங்கும்.

நேரத்தையும் முயற்சியையும் மிச்சப்படுத்துகிறது
