ஹெக்ஸ் இருந்து RGB மாற்றம்

  1. ஹெக்ஸ் வண்ணக் குறியீட்டின் 2 இடது இலக்கங்களைப் பெற்று, சிவப்பு நிற அளவைப் பெற, தசம மதிப்புக்கு மாற்றவும்.
  2. ஹெக்ஸ் வண்ணக் குறியீட்டின் 2 நடுத்தர இலக்கங்களைப் பெற்று, பச்சை நிற அளவைப் பெற, தசம மதிப்புக்கு மாற்றவும்.
  3. ஹெக்ஸ் வண்ணக் குறியீட்டின் 2 வலது இலக்கங்களைப் பெற்று, நீல வண்ண அளவைப் பெற தசம மதிப்புக்கு மாற்றவும்.

எடுத்துக்காட்டு #1

சிவப்பு ஹெக்ஸ் வண்ணக் குறியீட்டை FF0000 RGB நிறமாக மாற்றவும்:

Hex = FF0000

எனவே RGB நிறங்கள்:

R = FF16 = 25510

G = 0016 = 010

B = 0016 = 010

அல்லது

RGB = (255, 0, 0)

எடுத்துக்காட்டு #2

தங்க ஹெக்ஸ் வண்ணக் குறியீட்டை FFD700 RGB நிறமாக மாற்றவும்:

Hex = FFD700

எனவே RGB நிறங்கள்:

R = FF16 = 25510

G = D716 = 21510

B = 0016 = 010

அல்லது

RGB = (255, 215, 0)

  • RGB முதல் ஹெக்ஸ் மாற்றி
  • ஹெக்ஸ் முதல் தசம மாற்றி
  • RGB வண்ணக் குறியீடுகள்

ஹெக்ஸ் முதல் RGB வண்ண மாற்றி கருவியின் அம்சங்கள்

  1. ஹெக்ஸாடெசிமல் வண்ணக் குறியீடுகளை RGBக்கு மாற்றவும்: கருவியானது #FF0000 போன்ற ஹெக்ஸாடெசிமல் வண்ணக் குறியீடுகளை அவற்றின் தொடர்புடைய RGB மதிப்புகளுக்கு மாற்ற முடியும், அது (255, 0, 0).

  2. RGB ஐ ஹெக்ஸாடெசிமலாக மாற்றவும்: கருவியானது RGB மதிப்புகளை (255, 0, 0) அவற்றுடன் தொடர்புடைய ஹெக்ஸாடெசிமல் வண்ணக் குறியீடுகளாக மாற்ற முடியும், அது #FF0000 ஆக இருக்கும்.

  3. பல வண்ண வடிவங்களை ஆதரிக்கவும்: கருவியானது 3-இலக்க மற்றும் 6-இலக்க ஹெக்ஸாடெசிமல் குறியீடுகள் உட்பட பல்வேறு வண்ண வடிவங்களை ஆதரிக்க வேண்டும், அத்துடன் பல்வேறு வடிவங்களில் RGB மதிப்புகள் (எ.கா., அடைப்புக்குறிகளுடன் அல்லது இல்லாமல், காற்புள்ளிகளுடன் அல்லது இல்லாமல்).

  4. வண்ண மாதிரிக்காட்சியைக் காண்பி: கருவி மாற்றப்படும் வண்ணத்தின் மாதிரிக்காட்சியைக் காண்பிக்க வேண்டும், இதனால் பயனர்கள் மாற்றுவதற்கு முன் அதன் விளைவாக வரும் வண்ணத்தைப் பார்க்க முடியும்.

  5. வண்ணங்களை கைமுறையாக உள்ளிட பயனர்களை அனுமதிக்கவும்: வண்ணத் தேர்வு அல்லது பிற வெளிப்புறக் கருவியைப் பயன்படுத்துவதற்குப் பதிலாக, விரும்பிய வடிவத்தில் வண்ணக் குறியீடுகள் அல்லது மதிப்புகளை கைமுறையாக உள்ளிடுவதற்கு கருவி பயனர்களை அனுமதிக்க வேண்டும்.

  6. நகலெடுத்து ஒட்டுவதை ஆதரிக்கவும்: மாற்றத்திற்காக பட எடிட்டர் அல்லது இணையதளம் போன்ற பிற மூலங்களிலிருந்து வண்ணக் குறியீடுகள் அல்லது மதிப்புகளை நகலெடுத்து ஒட்டுவதற்கு கருவி பயனர்களை அனுமதிக்க வேண்டும்.

  7. தெளிவான மற்றும் சுருக்கமான முடிவுகளை வழங்கவும்: கருவி அதன் விளைவாக வரும் வண்ணக் குறியீடு அல்லது மதிப்பை தெளிவான மற்றும் சுருக்கமான முறையில் காண்பிக்க வேண்டும், இதனால் பயனர்கள் மாற்ற முடிவுகளைப் புரிந்துகொள்வதையும் பயன்படுத்துவதையும் எளிதாக்குகிறது.

  8. பயனர் நட்புடன் இருங்கள்: கருவியானது எளிமையான மற்றும் உள்ளுணர்வு இடைமுகத்துடன் பயன்படுத்த எளிதானது
