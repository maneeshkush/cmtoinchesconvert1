பாரன்ஹீட்

பாரன்ஹீட் என்பது வெப்பநிலையை அளவிடும் அலகு.

1 வளிமண்டல அழுத்தத்தில் நீரின் உறைதல்/உருகுநிலை சுமார் 32 °F ஆகும்.

பாரன்ஹீட் பட்டத்தின் குறியீடு °F ஆகும்.

ஃபாரன்ஹீட்டிலிருந்து செல்சியஸுக்கு மாறுதல்

எனவே 0 டிகிரி ஃபாரன்ஹீட் -17.77778 டிகிரி செல்சியஸுக்குச் சமம்.

0 °F = -17.77778 °C

எனவே டிகிரி  செல்சியஸில் (°C) வெப்பநிலை T என்பது டிகிரி ஃபாரன்ஹீட் (°F) மைனஸ் 32 ல் உள்ள வெப்பநிலை  T க்கு சமம், மடங்கு [5/9].

T(°C) = (T(°F) - 32) × 5/9

எடுத்துக்காட்டு 1

70 டிகிரி பாரன்ஹீட்டை டிகிரி செல்சியஸாக மாற்றவும்:

T(°C) = (70°F - 32) × 5/9 = 21.1111 °C

உதாரணம் 2

80 டிகிரி பாரன்ஹீட்டை டிகிரி செல்சியஸாக மாற்றவும்:

T(°C) = (80°F - 32) × 5/9 = 26.6667 °C

எடுத்துக்காட்டு 3

100 டிகிரி பாரன்ஹீட்டை டிகிரி செல்சியஸாக மாற்றவும்:

T(°C) = (100°F - 32) × 5/9 = 37.7778 °C

ஃபாரன்ஹீட்டிலிருந்து ரேங்கின் மாற்றம்

எனவே 0 டிகிரி ஃபாரன்ஹீட் என்பது [459.67] டிகிரி ரேங்கினுக்கு சமம்.

0 °F = 459.67 °R

 டிகிரி ரேங்கின் (°R) இல் உள்ள வெப்பநிலை T  என்பது டிகிரி ஃபாரன்ஹீட் (°F) மற்றும் [459.67] இல் உள்ள வெப்பநிலை T க்கு சமம்.

T(°R) = T(°F) + 459.67

எடுத்துக்காட்டு 1

70 டிகிரி ஃபாரன்ஹீட்டை டிகிரி ரேங்கினுக்கு மாற்றவும்:

T(°R) = 70°F + 459.67 = 529.67 °R

உதாரணம் 2

90 டிகிரி ஃபாரன்ஹீட்டை டிகிரி ரேங்கினுக்கு மாற்றவும்:

T(°R) = 90°F + 459.67 = 549.67 °R

எடுத்துக்காட்டு 3

100 டிகிரி ஃபாரன்ஹீட்டை டிகிரி ரேங்கினுக்கு மாற்றவும்:

T(°R) = 100°F + 459.67 = 559.67 °R

ஃபாரன்ஹீட்டிலிருந்து கெல்வின் மாற்றம்

எனவே கெல்வின் (K)  இல் உள்ள வெப்பநிலை T  என்பது டிகிரி ஃபாரன்ஹீட் (°F) மற்றும் 459.67, மடங்கு [5/9] வெப்பநிலை  T க்கு சமம் .

T(K) = (T(°F) + 459.67)× 5/9

எடுத்துக்காட்டு 1

65 டிகிரி பாரன்ஹீட்டை டிகிரி கெல்வின் ஆக மாற்றவும்:

T(K) = (65°F + 459.67)× 5/9 = 291.483 K

உதாரணம் 2

80 டிகிரி பாரன்ஹீட்டை டிகிரி கெல்வின் ஆக மாற்றவும்:

T(K) = (80°F + 459.67)× 5/9 = 299.817 K

எடுத்துக்காட்டு 3

100 டிகிரி பாரன்ஹீட்டை டிகிரி கெல்வின் ஆக மாற்றவும்:

T(K) = (100°F + 459.67)× 5/9 = 310.928 K
