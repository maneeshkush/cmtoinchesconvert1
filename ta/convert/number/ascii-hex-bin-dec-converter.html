1. ASCII குறியீடு: அது என்ன, அதை எப்படிப் பயன்படுத்துகிறீர்கள்?

ASCII (American Standard Code for Information Interchange) is a code that assigns a unique number to each letter, number, and symbol on a computer keyboard. This code allows computers to communicate with each other across the internet.

The ASCII code is also used to create text files. These files are composed of a series of letters, numbers, and symbols that are assigned a unique number by the ASCII code. When a computer opens an ASCII text file, it will translate the numbers back in to letters, numbers, and symbols.

4. தசம குறியீடு: அது என்ன, அதை எப்படிப் பயன்படுத்துகிறீர்கள்?

தசம குறியீடு என்பது 0-9 இலக்கங்களைப் பயன்படுத்தி எண்களைக் குறிக்கப் பயன்படும் ஒரு அமைப்பாகும்.இந்த அமைப்பில், ஒவ்வொரு இலக்கமும் ஒரு நெடுவரிசையில் வைக்கப்படுகிறது, மேலும் இலக்கத்தின் மதிப்பு நெடுவரிசையின் நிலையால் சுட்டிக்காட்டப்பட்ட 10 இன் சக்தியால் பெருக்கப்படுகிறது.எடுத்துக்காட்டாக, தசமக் குறியீட்டைப் பயன்படுத்தி 1234 என்ற எண்ணைக் குறிப்பிடலாம்:

1 x 10^3 + 2 x 10^2 + 3 x 10^1 + 4 x 10^0

இந்தக் குறியீட்டைப் பெரிய அல்லது பெரிய எண்ணைக் குறிக்கப் பயன்படுத்தலாம். சிறிய.ஒரு எண்ணை தசமக் குறியீட்டிலிருந்து நிலையான குறியீடாக மாற்ற, எண்ணில் உள்ள இலக்கங்கள் இடமிருந்து வலமாகப் படிக்கப்படுகின்றன.எடுத்துக்காட்டாக, 987 என்ற எண்ணை நிலையான குறியீடாக மாற்றலாம்:

9 x 10^2 + 8 x 10^1 + 7 x 10^0

பின்னங்களைக் குறிக்கவும் தசம குறியீடு பயன்படுத்தப்படலாம்.இந்த வழக்கில், எண்ணில் உள்ள ஒவ்வொரு இலக்கமும் a இல் வைக்கப்படும்

5. Converting between ASCII, Hex, Binary and Decimal: how do you do it?

ASCII என்பது உரை தரவுகளின் மிகவும் பொதுவான பிரதிநிதித்துவமாகும்.ஹெக்ஸாடெசிமல் என்பது ஒரு அடிப்படை 16 எண் அமைப்பு, இது பைனரி தரவைக் குறிக்க கணினி நிரலாக்கத்தில் பயன்படுத்தப்படுகிறது.பைனரி என்பது ஒரு அடிப்படை 2 எண் அமைப்பு, இது கணினி நினைவகத்தின் நிலைகளைக் குறிக்கப் பயன்படுகிறது.தசமம் என்பது நாம் அன்றாட வாழ்வில் பயன்படுத்தும் எண் அமைப்பு, அடிப்படை 10.

ASCII இலிருந்து Hex ஆக மாற்ற, ASCII மதிப்பை 16 ஆல் வகுத்து, மீதமுள்ளதை எடுத்துக் கொள்ளுங்கள்.ஹெக்ஸ் மதிப்பு என்பது மீதமுள்ளவற்றின் முதல் ஆறு எழுத்துக்களாகும்.எடுத்துக்காட்டாக, "A"க்கான ASCII மதிப்பு 65, எனவே ஹெக்ஸ் மதிப்பு "A" ஆகும்.

Hex இலிருந்து ASCII க்கு மாற்ற, Hex மதிப்பை 16 ஆல் வகுத்து, மீதமுள்ளதை எடுத்துக் கொள்ளுங்கள்.ASCII மதிப்பு என்பது மீதமுள்ளவற்றின் முதல் ஆறு எழுத்துக்களாகும்.எடுத்துக்காட்டாக, "A"க்கான ஹெக்ஸ் மதிப்பு "41" ஆகும், எனவே ASCII மதிப்பு "A" ஆகும்.

இந்த ASCII, Hex, Base64 மாற்றி பயனர்களுக்கு விரைவான மாற்றத்தை வழங்குகிறது.பயனர் உள்ளீட்டு புலத்தில் ASCII மதிப்புகளை உள்ளிட்டு, மாற்று பொத்தானைக் கிளிக் செய்தவுடன், பயன்பாடு மாற்றும் செயல்முறையைத் தொடங்கி உடனடியாக முடிவுகளை வழங்கும்.

துல்லியமான முடிவுகள்

இந்த டெசிமல், Base64 மூலம் உருவாக்கப்பட்ட முடிவுகள் 100% துல்லியமானவை.இந்த பயன்பாட்டால் பயன்படுத்தப்படும் மேம்பட்ட வழிமுறைகள் பயனர்களுக்கு பிழையற்ற முடிவுகளை வழங்குகின்றன.இந்த பயன்பாட்டினால் வழங்கப்பட்ட முடிவுகளின் நம்பகத்தன்மையை நீங்கள் உறுதிசெய்தால், அவற்றைச் சரிபார்க்க நீங்கள் எந்த முறையையும் பயன்படுத்தலாம்.

இணக்கத்தன்மை

ASCII, தசம மாற்றி அனைத்து வகையான சாதனங்களுக்கும் இணக்கமானது.நீங்கள் ஸ்மார்ட்போன், டேப்லெட், டெஸ்க்டாப், லேப்டாப் அல்லது மேக்கைப் பயன்படுத்தினாலும், இந்த பைனரி, டெசிமல், பேஸ்64 மாற்றியை எளிதாகப் பயன்படுத்தலாம்.
