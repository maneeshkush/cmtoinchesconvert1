CSS வழிசெலுத்தல் பட்டி

CSS நேவிகெய்டன் பார் - கிடைமட்ட மற்றும் செங்குத்து.

கிடைமட்ட வழிசெலுத்தல் பட்டி

CSS குறியீடு:

<style>
    #h_nav_bar a { padding:15px; font-weight:bold; float:left; }
    #h_nav_bar a:link { color:#d0d0d0; background-color:#0000a0; }
    #h_nav_bar a:visited { color:#c0c0c0; background-color:#0000a0; }
    #h_nav_bar a:hover { color:#ffffff; background-color:#000060; }
    #h_nav_bar a:active { color:#f0f0f0; background-color:#00ff00; }
</style>

HTML குறியீடு:

<div id="h_nav_bar">
    <a href="html-anchor-link.htm">Anchor link</a>
    <a href="html-link-color.htm">Link color</a>
    <a href="../mailto.htm">Email link</a>
    <a href="html-image-link.htm">Image link</a>
    <a href="html-text-link.htm">Text link</a>
</div>

காண்க:

ஆங்கர் இணைப்பு இணைப்பு நிறம் மின்னஞ்சல் இணைப்பு பட இணைப்பு உரை இணைப்பு

 

 

 

#link_bar a என்பது இணைப்பின் அனைத்து நிலைகளுக்கான நடை.

#link_bar a:link என்பது வழக்கமான இணைப்பின் பாணி.

#link_bar a: visit என்பது பார்வையிட்ட இணைப்பின் பாணி.

#link_bar a:hover என்பது மவுஸ் ஹோவர் செய்யப்பட்ட இணைப்பின் பாணி.

#link_bar a: சுட்டி மூலம் அழுத்தும் போது செயலில் உள்ள இணைப்பின் பாணி.

CSS வழிசெலுத்தல் பார் கருவியின் அம்சங்கள்

எங்கள் CSS வழிசெலுத்தல் பட்டை கருவி பயனர்களை CSS வழிசெலுத்தல் பட்டிக்கு அனுமதிக்கிறது.இந்த பயன்பாட்டின் சில முக்கிய அம்சங்கள் கீழே விளக்கப்பட்டுள்ளன.

எளிமை

எங்களின் CSS நேவிகேஷன் பார் கருவியின் பயனர் நட்பு இடைமுகம், இந்த மாற்றத்தை உடனடியாக மேற்கொள்ள உங்களுக்கு எளிதாக வழங்குகிறது.இந்தக் கருவியின் எளிமையின் காரணமாக நீங்கள் இனி CSS வழிசெலுத்தல் பட்டிக்கான நிரலாக்கத் திறன்களைப் பெற வேண்டியதில்லை.

பதிவு இல்லை

CSS வழிசெலுத்தல் பட்டியைப் பயன்படுத்த, நீங்கள் எந்தப் பதிவுச் செயல்முறையிலும் செல்ல வேண்டியதில்லை.இந்த பயன்பாட்டைப் பயன்படுத்தி, நீங்கள் CSS வழிசெலுத்தல் பட்டை கருவியை எத்தனை முறை வேண்டுமானாலும் இலவசமாகப் பயன்படுத்தலாம்.

பெயர்வுத்திறன்

இந்த CSS வழிசெலுத்தல் பட்டியை உலகின் எந்த மூலையிலிருந்தும் அணுகலாம்.இந்த ஆன்லைன் கருவியின் இணக்கத்தன்மையைப் பற்றி நீங்கள் கவலைப்படத் தேவையில்லை, ஏனெனில் நீங்கள் எந்தச் சாதனத்திலிருந்தும் CSS வழிசெலுத்தல் பட்டியை செய்யலாம்.இந்த CSS நேவிகேஷன் பார் கருவியை அணுகுவதற்கும் பயன்படுத்துவதற்கும் இணைய இணைப்பு மட்டுமே உங்களுக்குத் தேவை.

நேரத்தையும் முயற்சியையும் மிச்சப்படுத்துகிறது

CSS வழிசெலுத்தல் பட்டி, அதே பணியை உடனடியாக முடிக்க உங்களை அனுமதிக்கிறது.அதன் தானியங்கு வழிமுறைகள் உங்களுக்காக வேலை செய்யும் என்பதால், கையேடு நடைமுறைகளைப் பின்பற்றும்படி உங்களிடம் கேட்கப்படாது.

இணக்கத்தன்மை
