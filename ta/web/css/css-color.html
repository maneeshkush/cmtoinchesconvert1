CSS நிறம்

வண்ணக் குறியீடு இவற்றில் ஒன்றாக இருக்கலாம்:

ஹெக்ஸ் வடிவம்: #rrggbb

RGB வடிவம்: rgb(சிவப்பு, பச்சை, நீலம்)

பெயர் வடிவம்: பெயர்

உதாரணமாக

ஆரஞ்சு நிறத்திற்கு:

ஹெக்ஸ் வடிவம்: #FFA500

RGB வடிவம்: rgb(255,165,0)

பெயர் வடிவம்: ஆரஞ்சு

உறுப்பு நிறத்தை அமைத்தல்

குறிப்பிட்ட உறுப்புக்கு:

<element style="color: code;"></element>

ஒரே வகை அனைத்து உறுப்புகளுக்கும்.தலைப் பிரிவில் உள்ள <style> குறிச்சொல்லில் குறியீட்டை வைக்கவும்:

<style>
   element { color: code; }
</style>

உதாரணமாக

பத்தி உரை நிறத்தை சிவப்பு நிறமாக அமைத்தல்:

குறிப்பிட்ட பத்தியின் நிறத்தை அமைத்தல்:

<p style="color: #FF0000;">Some text ...</p>

காண்க:

சில உரை...

அனைத்து பத்திகளின் நிறத்தையும் அமைத்தல்

<style>
   p { color: #FF0000; }
</style>

உறுப்பு பின்னணி வண்ணத்தை அமைத்தல்

element { background: code; }

உதாரணமாக

பத்தியின் பின்னணி நிறத்தை சிவப்பு நிறமாக அமைத்தல்:

குறிப்பிட்ட பத்தியின் பின்னணி நிறத்தை அமைத்தல்:

<p style="background: #FF0000;">Some text ...</p>

காண்க:

சில உரை...

அனைத்து பத்திகளின் பின்னணி நிறத்தை அமைத்தல்:

<style>
   p { background: #FF0000; }
</style>

உறுப்பு எல்லை நிறத்தை அமைத்தல்

element { border-color: topcode rightcode bottomcode leftcode }

உதாரணமாக

பத்தி பார்டர் நிறத்தை சிவப்பு, பச்சை, நீலம் மற்றும் கருப்பு என அமைத்தல்:

குறிப்பிட்ட பத்தியின் எல்லை நிறத்தை அமைத்தல்:

<p style="border-color: #FF0000 #00FF00 #0000FF #000000; border-style:solid">Some text ...</p>

காண்க:

சில உரை...

அனைத்து பத்திகளின் எல்லை நிறத்தை அமைத்தல்:

<style>
   p { border-color: #FF0000 #00FF00 #0000FF #000000; }
</style>
