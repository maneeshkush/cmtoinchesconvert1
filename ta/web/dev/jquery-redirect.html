jQuery வழிமாற்று

jQuery மூலம் ஒரு பக்கத்தை URL க்கு திருப்பிவிடுவது எப்படி.

தேடுபொறிகள் 301 நிலைக் குறியீட்டைப் பயன்படுத்தி பழைய URLலிருந்து புதிய URLக்கு பக்கத் தரவரிசையை மாற்றும்.

jQuery திருப்பியனுப்புதல் http மறுமொழி நிலைக் குறியீடு: 200 சரி.

எனவே jQuery திசைதிருப்பல், ஜாவாஸ்கிரிப்ட் திசைதிருப்பல் போன்றது, தேடுபொறிக்கு ஏற்றதாக இல்லை, மேலும் 301 நிரந்தரமாக நகர்த்தப்பட்ட நிலைக் குறியீட்டை வழங்கும் பிற வழிமாற்று முறைகளைப் பயன்படுத்துவது நல்லது.

jQuery திசைதிருப்பல் உண்மையில் ஜாவாஸ்கிரிப்ட் திசைதிருப்பல் ஆகும் . jQuery மூலம் திருப்பிவிடுவது ஓவர்கில் என்று கருதப்படுகிறது, ஏனெனில் நீங்கள் குறைந்த குறியீட்டில் சுத்தமான ஜாவாஸ்கிரிப்ட் திசைதிருப்பலை செய்யலாம் .
