ஜாவாஸ்கிரிப்ட் வழிமாற்று URL

ஜாவாஸ்கிரிப்ட் மூலம் URL பக்கத்தை எப்படி திருப்பிவிடுவது.

Javascript திசைதிருப்பல் 301 நிரந்தர வழிமாற்று நிலைக் குறியீட்டை வழங்காது.

ஜாவ்ஸ்கிரிப்ட் வழிமாற்று

நீங்கள் திருப்பிவிட விரும்பும் பக்கத்தின் URL ஐக் கொண்டு, பழைய பக்கத்தைத் திருப்பிவிடவும்.

old-page.html:

<!DOCTYPE html>
<html>
<body>
<script type="text/javascript">
    // Javascript URL redirection
    window.location.replace("http://www.mydomain.com/new-page.html");
</script>
</body>
</html>

 

தேடுபொறிகள் 301 நிலைக் குறியீட்டைப் பயன்படுத்தி பழைய URLலிருந்து புதிய URLக்கு பக்கத் தரவரிசையை மாற்றும்.

ஜாவாஸ்கிரிப்ட் திருப்பிவிடுதல் http மறுமொழி நிலைக் குறியீடு: 200 சரி.

எனவே ஜாவாஸ்கிரிப்ட் திசைதிருப்பல் தேடுபொறிக்கு ஏற்றதாக இல்லை, மேலும் நிலைக் குறியீட்டை வழங்கும் பிற வழிமாற்று முறைகளைப் பயன்படுத்துவது நல்லது: 301 நிரந்தரமாக நகர்த்தப்பட்டது.

ஜாவாஸ்கிரிப்ட் வழிமாற்று உதாரணம்

<!DOCTYPE html>
<html>
<body>
<script type="text/javascript">
// Javascript URL redirection
window.location.replace("https://www.cmtoinchesconvert.com/ta/web/dev/javascript-redirect.htm");
</script>
</body>
</html>
