அடிப்படை விதியின் மடக்கை மாற்றம்

b இலிருந்து cக்கு அடிப்படையை மாற்ற, அடிப்படை விதியின் மடக்கை மாற்றத்தைப் பயன்படுத்தலாம்.x இன் அடிப்படை b மடக்கையானது, b இன் அடிப்படை c மடக்கையால் வகுக்கப்படும் x இன் அடிப்படை c மடக்கைக்கு சமம்:

logb(x) = logc(x) / logc(b)

எடுத்துக்காட்டு #1

log2(100) = log10(100) / log10(2) = 2 / 0.30103 = 6.64386

எடுத்துக்காட்டு #2

log3(50) = log8(50) / log8(3) = 1.8812853 / 0.5283208 = 3.5608766

ஆதாரம்

x இன் அடிப்படை b மடக்கையின் சக்தியுடன் b ஐ உயர்த்தினால் x கிடைக்கிறது:

(1) x = blogb(x)

b இன் அடிப்படை c மடக்கையின் சக்தியுடன் c ஐ உயர்த்துவது b ஐ அளிக்கிறது:

(2) b = clogc(b)

நாம் (1) ஐ எடுத்து c log c ( b ) (2)உடன் b ஐ மாற்றும்போது , ​​​​நாம் பெறுகிறோம்:

(3) x = blogb(x) = (clogc(b))logb(x) = clogc(b)×logb(x)

(3) இன் இருபுறங்களிலும்பதிவு c () ஐப் பயன்படுத்துவதன் மூலம்:

logc(x) = logc(clogc(b)×logb(x))

மடக்கை சக்தி விதியைப் பயன்படுத்துவதன் மூலம் :

logc(x) = [logc(b)×logb(x)] × logc(c)

பதிவு c ( c )=1 என்பதால்

logc(x) = logc(b)×logb(x)

அல்லது

logb(x) = logc(x) / logc(b)
