இயற்கை மடக்கை விதிகள் & பண்புகள்

 

விதி பெயர் விதி உதாரணமாக
தயாரிப்பு விதி

ln(x ∙ y) = ln(x) + ln(y)

ln(3 ∙ 7) = ln(3) + ln(7)
அளவு விதி

ln(x / y) = ln(x) - ln(y)

ln(3 / 7) = ln(3) - ln(7)
அதிகார விதி

ln(x y) = y ∙ ln(x)

ln(28) = 8 ∙ ln(2)
Ln வழித்தோன்றல்

f (x) = ln(x) ⇒ f ' (x) = 1 / x

 
Ln ஒருங்கிணைந்த

∫ ln(x)dx = x ∙ (ln(x) - 1) + C

  
எதிர்மறை எண்ணின் Ln

ln(x) is undefined when x ≤ 0

  
பூஜ்ஜியத்தின் Ln

ln(0) is undefined

 
 
ஒருவரின் Ln

ln(1) = 0

  
முடிவிலியின் Ln

lim ln(x) = ∞ , when x→∞

  

 

இயற்கை மடக்கை (ln) செயல்பாட்டின் வழித்தோன்றல்

இயற்கை மடக்கை செயல்பாட்டின் வழித்தோன்றல் பரஸ்பர செயல்பாடு ஆகும்.

எப்பொழுது

f (x) = ln(x)

f(x) இன் வழித்தோன்றல்:

f ' (x) = 1 / x

 

இயற்கை மடக்கை (ln) செயல்பாட்டின் ஒருங்கிணைப்பு

இயற்கை மடக்கை செயல்பாட்டின் ஒருங்கிணைப்பு பின்வருமாறு வழங்கப்படுகிறது:

எப்பொழுது

f (x) = ln(x)

f(x) இன் ஒருங்கிணைப்பு:

∫ f (x)dx = ∫ ln(x)dx = x ∙ (ln(x) - 1) + C
