எதிர்பார்ப்பு மதிப்பு

நிகழ்தகவு மற்றும் புள்ளிவிவரங்களில், எதிர்பார்ப்பு அல்லது எதிர்பார்க்கப்படும் மதிப்பு , ஒரு சீரற்ற மாறியின் எடையுள்ள சராசரி மதிப்பாகும்.

தொடர்ச்சியான சீரற்ற மாறியின் எதிர்பார்ப்பு

E ( X ) என்பது தொடர்ச்சியான சீரற்ற மாறி X இன் எதிர்பார்ப்பு மதிப்பாகும்

x என்பது தொடர்ச்சியான சீரற்ற மாறி X இன் மதிப்பு

P ( x ) என்பது நிகழ்தகவு அடர்த்தி சார்பு

தனித்த சீரற்ற மாறியின் எதிர்பார்ப்பு

E ( X ) என்பது தொடர்ச்சியான சீரற்ற மாறி X இன் எதிர்பார்ப்பு மதிப்பாகும்

x என்பது தொடர்ச்சியான சீரற்ற மாறி X இன் மதிப்பு

P ( x ) என்பது X இன் நிகழ்தகவு நிறை செயல்பாடாகும்

எதிர்பார்ப்பின் பண்புகள்

நேர்கோட்டுத்தன்மை

a மாறிலி மற்றும் X,Y சீரற்ற மாறிகள் ஆகும் போது:

E(aX) = aE(X)

E(X+Y) = E(X) + E(Y)

நிலையான

c நிலையானதாக இருக்கும்போது:

E(c) = c

தயாரிப்பு

X மற்றும் Y ஆகியவை சுயாதீன சீரற்ற மாறிகளாக இருக்கும்போது:

E(X ⋅Y) = E(X) ⋅ E(Y)

நிபந்தனை எதிர்பார்ப்பு
