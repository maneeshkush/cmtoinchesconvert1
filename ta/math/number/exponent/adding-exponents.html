அடுக்குகளைச் சேர்த்தல்

அடுக்குகளை எவ்வாறு சேர்ப்பது.

  • அடுக்குகளுடன் எண்களைச் சேர்த்தல்

அடுக்குகளுடன் எண்களைச் சேர்த்தல்

அடுக்குகளைச் சேர்ப்பது ஒவ்வொரு அடுக்குகளையும் முதலில் கணக்கிட்டு பின்னர் சேர்ப்பதன் மூலம் செய்யப்படுகிறது:

an + bm

உதாரணமாக:

42 + 25 = 4⋅4+2⋅2⋅2⋅2⋅2 = 16+32 = 48

அதே அடிப்படைகள் b மற்றும் அடுக்குகள் n ஐச் சேர்த்தல்:

bn + bn = 2bn

உதாரணமாக:

42 + 42 = 2⋅42 = 2⋅4⋅4 = 32

எதிர்மறை அடுக்குகளைச் சேர்த்தல்

எதிர்மறை அடுக்குகளைச் சேர்ப்பது ஒவ்வொரு அடுக்குகளையும் முதலில் கணக்கிட்டு பின்னர் சேர்ப்பதன் மூலம் செய்யப்படுகிறது:

a-n + b-m = 1/an + 1/bm

உதாரணமாக:

4-2 + 2-5 = 1/42 + 1/25 = 1/(4⋅4)+1/(2⋅2⋅2⋅2⋅2) = 1/16+1/32 = 0.09375

பகுதியளவு அடுக்குகளைச் சேர்த்தல்

பகுதியளவு அடுக்குகளைச் சேர்ப்பது ஒவ்வொரு அடுக்குகளையும் முதலில் உயர்த்தி பின்னர் சேர்ப்பதன் மூலம் செய்யப்படுகிறது:

an/m + bk/j

உதாரணமாக:

33/2 + 25/2 = √(33) + √(25) = √(27) + √(32) = 5.196 + 5.657 = 10.853

 

அதே அடிப்படைகள் b மற்றும் அடுக்குகள் n/m சேர்த்தல்:

bn/m + bn/m = 2bn/m

உதாரணமாக:

42/3 + 42/3 = 2⋅42/3 = 2 ⋅ 3√(42) = 5.04

அடுக்குகளுடன் மாறிகளைச் சேர்த்தல்

அடுக்குகளைச் சேர்ப்பது ஒவ்வொரு அடுக்குகளையும் முதலில் கணக்கிட்டு பின்னர் சேர்ப்பதன் மூலம் செய்யப்படுகிறது:

xn + xm

அதே அடுக்குகளுடன்:

xn + xn = 2xn

உதாரணமாக:

x2 + x2 = 2x2
