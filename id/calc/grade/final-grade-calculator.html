Perhitungan nilai akhir

Contoh 1

Nilai saat ini adalah 70% (atau C-).

Bobot ujian akhir adalah 50%.

Nilai yang dibutuhkan adalah 80% (atau B-).

Perhitungan:

Jadi nilai ujian akhir sama dengan nilai wajib, dikurangi 100% dikurangi bobot ujian akhir (w) dikalikan nilai saat ini (g), dibagi dengan bobot ujian akhir (w).

Nilai ujian akhir =

= ( nilai wajib - (100% - b)× nilai saat ini ) / b

= ( 80% - (100% - 50%)×70% ) / 50% = 90%

Jadi nilai ujian akhir harus 90% (atau A-).

Contoh #2

Tugas 1: bobot1=50%, nilai1=16 dari 20.

Tugas 2: bobot2=30%, nilai maks=30.

Tugas 3: bobot3=20%, nilai maks=40.

Temukan nilai rata-rata dalam tugas 2 dan 3 yang diperlukan untuk mendapatkan nilai kelas 85%.

Perhitungan:

Nilai saat ini = Tugas 1 nilai = nilai1 / maks nilai1 = 16/20 = 0,8 = 80%

Nilai yang dibutuhkan = 85%

Bobot ujian akhir = w = bobot2+bobot3 = 30%+20% = 50%

Nilai ujian akhir =

= ( nilai wajib - (100% - b )× nilai saat ini ) / b

= ( 85% - 50%×80% ) / 50% = 90%

Artinya, Anda harus mendapatkan nilai rata-rata 90% pada tugas 2 dan 3 untuk mendapatkan nilai kelas 85%.

Tugas 2 nilai = 90% × nilai maksimal = 90% × 30 = 27

Tugas 3 nilai = 90% × nilai maksimal = 90% × 40 = 36
