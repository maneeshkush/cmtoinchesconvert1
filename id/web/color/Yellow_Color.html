Kode warna RGB kuning

Kode RGB kuning = 255*65536+255*256+0 = #FFFF00

MERAH=255, HIJAU=255, BIRU=0

Paragraf HTML dengan font kuning

Kode:

<p style=" color:yellow; background:black">Font ini berwarna kuning!</p>

Hasil:

Font ini berwarna kuning!

ATAU

<p style=" color:#FFFF00; background:black">Font ini juga berwarna kuning!</p>

Hasil:

Font-font ini juga berwarna kuning!

ATAU

<p style=" color:rgb(255,255,0); background:black">Font ini juga berwarna kuning!</p>

Hasil:

Font-font ini juga berwarna kuning!
