URL pengalihan Javascript

Cara mengarahkan ulang halaman URL dengan Javascript.

Pengalihan Javascript tidak mengembalikan 301 kode status pengalihan permanen.

Pengalihan JavaScript

Ganti halaman lama dengan kode pengalihan dengan URL halaman yang ingin Anda alihkan.

halaman-lama.html:

<!DOCTYPE html>
<html>
<body>
<script type="text/javascript">
    // Javascript URL redirection
    window.location.replace("http://www.mydomain.com/new-page.html");
</script>
</body>
</html>

 

Mesin pencari menggunakan kode status 301 untuk mentransfer peringkat halaman dari URL lama ke URL baru.

Pengalihan Javascript mengembalikan kode status respons http: 200 OK.

Jadi pengalihan Javascript tidak ramah mesin pencari dan lebih baik menggunakan metode pengalihan lain yang mengembalikan kode status: 301 Dipindahkan Secara Permanen.

Contoh pengalihan Javascript
