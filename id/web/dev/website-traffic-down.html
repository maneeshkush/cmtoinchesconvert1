Lalu lintas situs web turun

Mengapa lalu lintas situs web saya turun?

Periksa kalender

Hari libur dan akhir pekan dapat menurunkan lalu lintas Anda.

Lalu lintas akan kembali normal saat hari raya usai.

Bandingkan dengan tahun lalu

Gunakan Google Analytics untuk menampilkan grafik kunjungan tahun lalu.

Periksa apakah kunjungan turun setahun yang lalu juga.

bug Google Analytics

Menggunakan kode Google Analytics lama dengan file urchin.js, mungkin menampilkan 2 hari terakhir dengan lalu lintas lebih rendah daripada lalu lintas sebenarnya.

Lalu lintas tidak benar-benar turun, tetapi hanya tampak turun.

Masalah server

Coba jelajahi situs web Anda, jika Anda tidak dapat mengaksesnya, Anda memiliki masalah server web atau server DNS.

Coba akses server web Anda dan periksa apakah sudah aktif.

Periksa integritas database atau file html Anda.

Gunakan alat uji ping untuk memeriksa respons server web Anda.

Cari yang baru di masalah server DNS. Pada 9/2012, situs web ini dengan banyak lainnya tidak dapat merespons (lihat: GoDaddy diretas ).

Peringkat hasil pencarian Google turun

Sebagian besar lalu lintas situs web berasal dari mesin pencari dan mesin pencari utamanya adalah Google.

Jika sebagian besar kunjungan situs web Anda dihasilkan oleh satu kata kunci, mungkin diambil oleh pesaing.

  • Cari kata kunci di Google untuk menentukan apakah ada situs web lain yang jika terletak di depan situs Anda dan memberi nilai lebih baik kepada pengguna.
  • Cari berita untuk perubahan algoritme peringkat Google. Misalnya, pembaruan Google panda merusak lalu lintas banyak situs web.

Situs web dilarang oleh Google

Menggunakan metode terlarang untuk mempromosikan situs Anda di Google akan memastikan situs web Anda akan dilarang oleh Google.

Cari Google dengan kata kunci utama Anda dan lihat apakah muncul seperti biasa di hasil pencarian.

Jika situs web Anda tidak muncul sama sekali, Anda harus:

  1. Baca panduan webmaster Google dan perbaiki situs web Anda.
  2. Kirim permintaan pertimbangan ulang ke Google.
