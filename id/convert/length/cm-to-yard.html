Bagaimana mengkonversi sentimeter ke yard

  Jadi 1 sentimeter sama dengan 0,01093613298 yard:

1cm = 100cm/(2.54cm/in)/(36in/yd) = 0.010936132983yd

2cm = 200cm/(2,54cm/inci)/(36inci/yd) = 0,021872265967yd

Jadi jarak d dalam yard (yd) sama dengan jarak  d dalam centimeter (cm) dibagi dengan [91,44].

d(yd) =  d(cm) / 91.44

Contoh 1

Ubah 10 cm menjadi yard:

d(yd) = 10cm / 91.44 = 0.10936132983yd

Contoh 2

Ubah 30 cm menjadi yard:

d(yd) = 30cm / 91.44 = 0.3280839895yd

Contoh 3

Ubah 50 cm menjadi yard:

d(yd) = 50cm / 91.44 = 0.54680664917yd

Contoh 4

Ubah 80 cm menjadi yard:

d(yd) = 80cm / 91.44 = 0.87489063867yd

Contoh 5

Ubah 100 cm menjadi yard:

d(yd) = 100cm / 91.44 = 1.0936132983yd

Berapa yard dalam satu sentimeter

Satu sentimeter sama dengan 0,010936132983yd.

1cm = 1m / 91,44yd/cm = 0,010936132983yd

Dua sentimeter sama dengan 0,021872265967yd.

2cm = 2m / 91,44yd/cm = 0,021872265967yd

3cm = 3m / 91,44yd/cm = 0,03280839895yd

4cm = 4m / 91,44yd/cm = 0,043744531934yd

Berapa sentimeter dalam satu yard

 Jadi Satu yard sama dengan 91,44 sentimeter:

1yd = 91,44×1yd = 91,44cm

Bagaimana mengkonversi 10cm ke yard

 Jadi Kalikan 10 sentimeter dengan 91,44 untuk mendapatkan yard:

10cm = 10cm / 91,44cm/yd = 0,10936132983yd

50cm = 50cm / 91,44cm/yd = 0,54680664917yd

500cm = 500cm / 91,44cm/yd = 5,4680664917yd
