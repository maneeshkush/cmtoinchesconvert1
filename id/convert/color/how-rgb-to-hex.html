Bagaimana mengkonversi RGB ke warna hex

Bagaimana mengkonversi dari warna RGB ke kode warna heksadesimal.

Warna RGB

Warna RGB adalah kombinasi warna Merah, Hijau dan biru:

(R, G, B)

Merah, hijau, dan biru masing-masing menggunakan 8 bit, dengan nilai integer mulai dari 0 hingga 255.

Sehingga banyaknya warna yang dapat dihasilkan adalah :

256×256×256 = 16777216 = 100000016

kode warna hex

Kode warna hex adalah angka heksadesimal 6 digit (basis 16):

RRGGBB 16

2 digit kiri mewakili warna merah.

2 digit tengah mewakili warna hijau.

2 digit kanan mewakili warna biru.

konversi rgb ke hex

1. Ubah nilai merah, hijau, dan biru dari desimal menjadi heksadesimal.
2. Gabungkan 3 nilai hex merah, hijau dan biru menjadi satu: RRGGBB.

Contoh 1
Ubah kode warna merah (255,0,0) menjadi hex:

R = 25510 = FF16

G = 010 = 0016

B = 010 = 0016

Jadi kode warna hex adalah:

Hex = FF0000

Contoh #2
Ubah warna emas (255.215,0) menjadi kode warna hex:

R = 25510 = FF16

G = 21510 = D716

B = 010 = 0016

Jadi kode warna hex adalah:

Hex = FFD700

Apa yang dilakukan konverter RGB ke Hex ini?

Dibutuhkan sebagai input nilai warna merah, hijau dan biru mulai dari 0 hingga 255 dan kemudian mengubah nilai tersebut menjadi string heksadesimal yang dapat digunakan untuk menentukan warna dalam kode html/css. Perangkat lunak pengedit foto biasanya mewakili warna dalam RGB, jadi jika Anda ingin menggunakan warna yang digunakan dalam perangkat lunak pengedit foto sebagai latar belakang elemen HTML, Anda perlu mendapatkan representasi heksadesimal dari nilai RGB. Alat ini memungkinkan Anda untuk mendapatkan nilai-nilai tersebut.

Mengkonversi Nilai Hex ke RGB

Mungkin Anda pernah melihat kode hex di halaman web dan ingin menggunakan warna itu di perangkat lunak pengedit foto Anda. Dalam hal ini Anda memerlukan nilai RGB jika perangkat lunak pengedit foto Anda tidak mendukung nilai HEX.
