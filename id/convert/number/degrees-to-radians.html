Cara mengubah derajat ke radian

Pi radian sama dengan 180 derajat:

π rad = 180°

Satu derajat sama dengan 0,01745329252 radian:

1° = π/180° = 0.005555556π = 0.01745329252 rad

Jadi sudut α dalam radian sama dengan sudut α dalam derajat dikali konstanta pi dibagi [180] derajat.

α(radians) = α(degrees) × π / 180°

atau

radians = degrees × π / 180°

Contoh 1

Mengkonversi sudut 35 derajat ke radian:

α(radians) = α(degrees) × π / 180° = 35° × 3.14159 / 180° = 0.6108652382 rad

Contoh 2

Mengkonversi sudut 90 derajat ke radian:

α(radians) = α(degrees) × π / 180° = 90° × 3.14159 / 180° = 1.5707963268 rad

Contoh 3

Mengkonversi sudut 45 derajat ke radian:

α(radians) = α(degrees) × π / 180° = 45° × 3.14159 / 180° = 0.7853981634 rad

Contoh 4

Mengkonversi sudut 60 derajat ke radian:

α(radians) = α(degrees) × π / 180° = 60° × 3.14159 / 180° = 1.0471975512 rad
