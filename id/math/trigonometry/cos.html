Definisi kosinus

Pada segitiga siku-siku ABC, sinus dari α, sin(α) didefinisikan sebagai rasio antara sisi yang berdekatan dengan sudut α dan sisi yang berlawanan dengan sudut siku-siku (hipotenusa):

cos α = b / c

Contoh

b = 3"

c = 5"

cos α = b / c = 3 / 5 = 0.6

Fungsi cosinus terbalik

Arccosine dari x didefinisikan sebagai fungsi cosinus terbalik dari x ketika -1≤x≤1.

Ketika kosinus y sama dengan x:

cos y = x

Maka arccosine dari x sama dengan fungsi cosinus terbalik dari x, yang sama dengan y:

arccos x = cos-1 x = y

Contoh

arccos 1 = cos-1 1 = 0 rad = 0°
