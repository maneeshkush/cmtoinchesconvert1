अंशांचे रेडियनमध्ये रूपांतर कसे करावे

पाई रेडियन 180 अंशांच्या समान आहेत:

π rad = 180°

एक अंश ०.०१७४५३२९२५२ रेडियन समान आहे:

1° = π/180° = 0.005555556π = 0.01745329252 rad

तर रेडियन्समधील कोन α हा कोन α अंश गुणा pi स्थिरांक भागिले [180] अंशांइतका असतो.

α(radians) = α(degrees) × π / 180°

किंवा

radians = degrees × π / 180°

उदाहरण १

35 अंश कोन रेडियनमध्ये रूपांतरित करा:

α(radians) = α(degrees) × π / 180° = 35° × 3.14159 / 180° = 0.6108652382 rad

उदाहरण २

90 अंश कोन रेडियनमध्ये रूपांतरित करा:

α(radians) = α(degrees) × π / 180° = 90° × 3.14159 / 180° = 1.5707963268 rad

उदाहरण ३

45 अंश कोन रेडियनमध्ये रूपांतरित करा:

α(radians) = α(degrees) × π / 180° = 45° × 3.14159 / 180° = 0.7853981634 rad

उदाहरण ४

60 अंश कोन रेडियनमध्ये रूपांतरित करा:

α(radians) = α(degrees) × π / 180° = 60° × 3.14159 / 180° = 1.0471975512 rad
