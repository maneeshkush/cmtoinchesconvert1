अपूर्णांक घातांक सरलीकृत करणे

बेस b n/m च्या बळावर वाढवलेला आहे:

bn/m = (m√b)n = m√(bn)

उदाहरण:

3/2 च्या बळावर वाढवलेला बेस 2 हा 1 भागाकार बेस 2 ने 3 च्या घात केला आहे:

23/2 = 2√(23) = 2.828

घातांकांसह अपूर्णांक सरलीकृत करणे

घातांकांसह अपूर्णांक:

(a / b)n = an / bn

उदाहरण:

(4/3)3 = 43 / 33 = 64 / 27 = 2.37

ऋणात्मक अपूर्णांक घातांक

बेस b उणे n/m च्या बळावर वाढवलेला 1 भागिले n/m च्या बळावर बेस b ने वाढवलेला आहे:

b-n/m = 1 / bn/m = 1 / (m√b)n

उदाहरण:

वजा 1/2 च्या बळावर वाढवलेला आधार 2 हा 1 भागिले बेस 2 ने 1/2 च्या घातापर्यंत वाढवला आहे:

2-1/2 = 1/21/2 = 1/√2 = 0.7071

ऋण घातांक असलेले अपूर्णांक

वजा n च्या बळावर वाढवलेला आधार a/b हा n च्या बळावर वाढवलेल्या बेस a/b ने भागलेल्या 1 च्या बरोबरीचा आहे:

(a/b)-n = 1 / (a/b)n = 1 / (an/bn) = bn/an

उदाहरण:

वजा 3 च्या बळावर आधार 2 ने 1 भागिले बेस 2 ने 3 च्या बळावर वाढवलेला आहे:

(2/3)-2 = 1 / (2/3)2 = 1 / (22/32) = 32/22 = 9/4 = 2.25

अपूर्णांक घातांकांचा गुणाकार

अपूर्णांक घातांकांचा समान अपूर्णांक घातांकासह गुणाकार करणे:

a n/m ⋅ b n/m = (a ⋅ b) n/m

उदाहरण:

23/2 ⋅ 33/2 = (2⋅3)3/2 = 63/2 = √(63) = √216 = 14.7

 

अपूर्णांक घातांकांचा समान पायासह गुणाकार करणे:

a n/m ⋅ a k/j = a (n/m)+(k/j)

उदाहरण:

23/2 ⋅ 24/3 = 2(3/2)+(4/3) = 7.127

 

भिन्न घातांक आणि अपूर्णांकांसह अपूर्णांक घातांकांचा गुणाकार:

a n/m ⋅ b k/j

उदाहरण:

23/2 ⋅ 34/3 = √(23) ⋅ 3√(34) = 2.828 ⋅ 4.327 = 12.237

घातांकांसह अपूर्णांकांचा गुणाकार

समान अपूर्णांक आधार असलेल्या घातांकांसह अपूर्णांकांचा गुणाकार:

(a / b) n ⋅ (a / b) m = (a / b) n+m

उदाहरण:

(4/3)3 ⋅ (4/3)2 = (4/3)3+2 = (4/3)5 = 45 / 35 = 4.214

 

समान घातांकासह घातांकासह अपूर्णांकांचा गुणाकार:

(a / b) n ⋅ (c / d) n = ((a / b)⋅(c / d)) n

उदाहरण:

(4/3)3 ⋅ (3/5)3 = ((4/3)⋅(3/5))3 = (4/5)3 = 0.83 = 0.8⋅0.8⋅0.8 = 0.512

 

भिन्न पाया आणि घातांक असलेल्या घातांकांसह अपूर्णांकांचा गुणाकार:

(a / b) n ⋅ (c / d) m

उदाहरण:

(4/3)3 ⋅ (1/2)2 = 2.37 ⋅ 0.25 = 0.5925

अपूर्णांक घातांक विभागणे

अपूर्णांक घातांकांना समान अपूर्णांक घातांकाने विभाजित करणे:

a n/m / b n/m = (a / b) n/m

उदाहरण:

33/2 / 23/2 = (3/2)3/2 = 1.53/2 = √(1.53) = √3.375 = 1.837

 

अपूर्णांक घातांकांना समान बेससह विभाजित करणे:

a n/m / a k/j = a (n/m)-(k/j)

उदाहरण:

23/2 / 24/3 = 2(3/2)-(4/3) = 2(1/6) = 6√2 = 1.122

 

भिन्न घातांक आणि अपूर्णांकांसह अपूर्णांक घातांक विभागणे:

a n/m / b k/j

उदाहरण:

23/2 / 34/3 = √(23) / 3√(34) = 2.828 / 4.327 = 0.654

घातांकांसह अपूर्णांकांचे विभाजन करणे

समान अपूर्णांक आधार असलेल्या घातांकांसह अपूर्णांकांचे विभाजन करणे:

(a / b)n / (a / b)m = (a / b)n-m

उदाहरण:

(4/3)3 / (4/3)2 = (4/3)3-2 = (4/3)1 = 4/3 = 1.333

 

समान घातांकासह घातांकासह अपूर्णांकांचे विभाजन करणे:

(a / b)n / (c / d)n = ((a / b)/(c / d))n = ((a⋅d / b⋅c))n

उदाहरण:

(4/3)3 / (3/5)3 = ((4/3)/(3/5))3 = ((4⋅5)/(3⋅3))3 = (20/9)3 = 10.97

 

भिन्न पाया आणि घातांक असलेल्या घातांकांसह अपूर्णांकांचे विभाजन करणे:

(a / b) n / (c / d) m

उदाहरण:

(4/3)3 / (1/2)2 = 2.37 / 0.25 = 9.481

अपूर्णांक घातांक जोडत आहे

अपूर्णांक घातांक जोडणे प्रथम प्रत्येक घातांक वाढवून आणि नंतर जोडून केले जाते:

an/m + bk/j

उदाहरण:

33/2 + 25/2 = √(33) + √(25) = √(27) + √(32) = 5.196 + 5.657 = 10.853

 

समान आधार b आणि घातांक n/m जोडणे:

bn/m + bn/m = 2bn/m

उदाहरण:

42/3 + 42/3 = 2⋅42/3 = 2 ⋅ 3√(42) = 5.04

अपूर्णांक घातांक वजा करणे

अपूर्णांक घातांक वजा करणे प्रथम प्रत्येक घातांक वाढवून आणि नंतर वजा करून केले जाते:

an/m - bk/j

उदाहरण:

33/2 - 25/2 = √(33) - √(25) = √(27) - √(32) = 5.196 - 5.657 = -0.488

 

समान पाया b आणि घातांक n/m वजा करणे:

3bn/m - bn/m = 2bn/m

उदाहरण:

3⋅42/3 - 42/3 = 2⋅42/3 = 2 ⋅ 3√(42) = 5.04
