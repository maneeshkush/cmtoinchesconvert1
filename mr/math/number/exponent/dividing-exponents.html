समान पायासह घातांक भागणे

समान आधार असलेल्या घातांकांसाठी, आपण घातांक वजा केले पाहिजेत:

a n / a m = a n-m

उदाहरण:

26 / 23 = 26-3 = 23 = 2⋅2⋅2 = 8

वेगवेगळ्या बेससह घातांकांचे विभाजन करणे

जेव्हा पाया भिन्न असतात आणि a आणि b चे घातांक समान असतात, तेव्हा आपण प्रथम a आणि b ला विभाजित करू शकतो:

a n / b n = (a / b) n

उदाहरण:

63 / 23 = (6/2)3 = 33 = 3⋅3⋅3 = 27

 

जेव्हा पाया आणि घातांक भिन्न असतात तेव्हा आपल्याला प्रत्येक घातांकाची गणना करावी लागते आणि नंतर विभाजित करावे लागते:

a n / b m

उदाहरण:

62 / 33 = 36 / 27 = 1.333

ऋण घातांक विभागणे

समान आधार असलेल्या घातांकांसाठी, आपण घातांक वजा करू शकतो:

a-n / a-m = a-n-(-m) = am-n

उदाहरण:

2-3 / 2-5 = 25-3 = 22 = 2⋅2 = 4

 

जेव्हा पाया भिन्न असतात आणि a आणि b चे घातांक समान असतात, तेव्हा आपण प्रथम a आणि b चा गुणाकार करू शकतो:

a-n / b-n = (a/b)-n = 1 / (a/b)n = (b/a)n

उदाहरण:

3-2 / 4-2 = (4/3)2 = 1.7778

 

जेव्हा पाया आणि घातांक भिन्न असतात तेव्हा आपल्याला प्रत्येक घातांकाची गणना करावी लागते आणि नंतर विभाजित करावे लागते:

a-n / b-m = bm / an

उदाहरण:

3-2 / 4-3 = 43 / 32 = 64 / 9 = 7.111

घातांकांसह अपूर्णांकांचे विभाजन करणे

समान अपूर्णांक आधार असलेल्या घातांकांसह अपूर्णांकांचे विभाजन करणे:

(a / b)n / (a / b)m = (a / b)n-m

उदाहरण:

(4/3)3 / (4/3)2 = (4/3)3-2 = (4/3)1 = 4/3 = 1.333

 

समान घातांकासह घातांकासह अपूर्णांकांचे विभाजन करणे:

(a / b)n / (c / d)n = ((a / b)/(c / d))n = ((a⋅d / b⋅c))n

उदाहरण:

(4/3)3 / (3/5)3 = ((4/3)/(3/5))3 = ((4⋅5)/(3⋅3))3 = (20/9)3 = 10.97

 

भिन्न पाया आणि घातांक असलेल्या घातांकांसह अपूर्णांकांचे विभाजन करणे:

(a / b) n / (c / d) m

उदाहरण:

(4/3)3 / (1/2)2 = 2.37 / 0.25 = 9.481

अपूर्णांक घातांक विभागणे

अपूर्णांक घातांकांना समान अपूर्णांक घातांकाने विभाजित करणे:

a n/m / b n/m = (a / b) n/m

उदाहरण:

33/2 / 23/2 = (3/2)3/2 = 1.53/2 = √(1.53) = √3.375 = 1.837

 

अपूर्णांक घातांकांना समान बेससह विभाजित करणे:

a n/m / a k/j = a (n/m)-(k/j)

उदाहरण:

23/2 / 24/3 = 2(3/2)-(4/3) = 2(1/6) = 6√2 = 1.122

 

भिन्न घातांक आणि अपूर्णांकांसह अपूर्णांक घातांक विभागणे:

a n/m / b k/j

उदाहरण:

23/2 / 24/3 = √(23) / 3√(24) = 2.828 / 2.52 = 1.1222

घातांकांसह चलांचे विभाजन करणे

समान आधार असलेल्या घातांकांसाठी, आपण घातांक वजा करू शकतो:

xn / xm = xn-m

उदाहरण:

x5 / x3 = (x⋅x⋅x⋅x⋅x) / (x⋅x⋅x) = x5-3 = x2
