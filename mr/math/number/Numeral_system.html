संख्या प्रणाली

b - संख्या प्रणाली बेस

d n - n-वा अंक

n - संख्येमध्ये अपूर्णांक असल्यास ऋण संख्येपासून प्रारंभ होऊ शकतो.

N +1 - अंकांची संख्या

बायनरी न्युमरल सिस्टम - बेस-2

बायनरी संख्या फक्त 0 आणि 1 अंक वापरतात.

B हा बायनरी उपसर्ग दर्शवतो.

उदाहरणे:

101012 = 10101B = 1×24+0×23+1×22+0×21+1×20 = 16+4+1= 21

101112 = 10111B = 1×24+0×23+1×22+1×21+1×20 = 16+4+2+1= 23

1000112 = 100011B = 1×25+0×24+0×23+0×22+1×21+1×20 =32+2+1= 35

अष्टांक प्रणाली - बेस-8

ऑक्टल संख्या 0..7 पासून अंक वापरतात.

उदाहरणे:

278 = 2×81+7×80 = 16+7 = 23

308 = 3×81+0×80 = 24

43078 = 4×83+3×82+0×81+7×80= 2247

दशांश संख्या प्रणाली - बेस-10

दशांश संख्या 0..9 पासून अंक वापरते.

हे नियमित क्रमांक आहेत जे आपण वापरतो.

उदाहरण:

253810 = 2×103+5×102+3×101+8×100

हेक्साडेसिमल संख्या प्रणाली - बेस-16

हेक्स क्रमांक 0..9 आणि A..F मधील अंक वापरतात.

H हेक्स उपसर्ग दर्शवितो.

उदाहरणे:

2816 = 28H = 2×161+8×160 = 40

2F16 = 2FH = 2×161+15×160 = 47

BC1216 = BC12H = 11×163+12×162+1×161+2×160= 48146
