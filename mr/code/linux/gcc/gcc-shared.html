gcc -shared पर्याय ध्वज

gcc -shared सामायिक लायब्ररीसाठी सामायिक ऑब्जेक्ट फाइल व्युत्पन्न करते.

मांडणी

$ gcc -shared [options] [source files] [object files] [-o output file]

उदाहरण???

myfile.c :

// myfile.c
#include <stdio.h>

int main()
{
    printf("Program run!\n");
    int i=10;
}

 

myfile.c चे नियमित बिल्डकोणतेही संदेश देत नाही:

$ gcc myfile.c -o myfile
$

 

-वॉलसह myfile.c चे बिल्ड:

$ gcc -Wall myfile.c -o myfile
myfile.c In function 'main':
myfile.c:6:6: warning: unused variable 'i'
myfile.c:7:1: warning: control reaches end of non-void function
$
