jQuery omdirigering

Sådan omdirigeres en side til URL med jQuery.

Søgemaskiner bruger 301-statuskoden til at overføre sideplaceringen fra den gamle URL til den nye URL.

jQuery-omdirigering returnerer http-svarstatuskode: 200 OK.

Så jQuery-omdirigering, ligesom Javascript-omdirigering, er ikke søgemaskinevenlig, og det er bedre at bruge andre omdirigeringsmetoder, der returnerer statuskoden 301 Moved Permanently.

jQuery-omdirigering er faktisk Javascript-omdirigering . Omdirigering med jQuery betragtes som overkill, fordi du kan lave ren Javascript-omdirigering med mindre kode.

jQuery omdirigering

Erstat gammel side med omdirigeringskode med URL'en på den side, du vil omdirigere til.

old-page.html:

<!DOCTYPE html>
<html>
<body>
<script src="http://ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script>
<scripttype ="text/javascript">
   // jQuery URL-omdirigering
   $(document).ready( function() {
      url = "http://www.mydomain.com/new-page.html";
      $( location ).attr( "href", url);
   });
</script>
</body>
</html>
