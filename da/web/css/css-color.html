CSS farve

Farvekoden kan være en af:

Hex-format: #rrggbb

RGB-format: rgb (rød, grøn, blå)

Navneformat: navn

Eksempel

For orange farve:

Hex-format: #FFA500

RGB-format: rgb(255,165,0)

Navneformat: orange

Indstillingselementets farve

For specifikt element:

<element style="color: code;"></element>

For alle elementer af samme type. Indsæt kode i <style> tag i hovedsektionen:

<style>
   element { color: code; }
</style>

Eksempel

Indstilling af afsnitstekstfarve til rød:

Indstilling af farve for specifikt afsnit:

<p style="color: #FF0000;">Some text ...</p>

Indstilling af farve på alle afsnit

<style>
   p { color: #FF0000; }
</style>

Indstillingselementets baggrundsfarve

element { background: code; }

Eksempel

Indstilling af afsnits baggrundsfarve til rød:

Indstilling af baggrundsfarve for specifikt afsnit:

<p style="background: #FF0000;">Some text ...</p>

Udsigt:

Lidt tekst...

Indstilling af baggrundsfarve for alle afsnit:

<style>
   p { background: #FF0000; }
</style>

Indstillingselementets kantfarve

element { border-color: topcode rightcode bottomcode leftcode }
