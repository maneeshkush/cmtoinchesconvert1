Definition af naturlig logaritme

Hvornår

e y = x

Så er grundtallet e-logaritmen af ​​x

ln(x) = loge(x) = y

 

e - konstanten eller Eulers tal er:

e ≈ 2,71828183

Ln som invers funktion af eksponentiel funktion

Den naturlige logaritmefunktion ln(x) er den inverse funktion af eksponentialfunktionen e x .

For x>0,

f (f -1(x)) = eln(x) = x

Eller

f -1(f (x)) = ln(ex) = x

Naturlige logaritmeregler og egenskaber

Regelnavn Herske Eksempel
Produktregel

ln(x ∙ y) = ln(x) + ln(y)

ln(3 ∙ 7) = ln(3) + ln(7)
Kvotientregel

ln(x / y) = ln(x) - ln(y)

ln(3 / 7) = ln(3) - ln(7)
Magt reglen

ln(x y) = y ∙ ln(x)

ln(28) = 8∙ ln(2)

 f ( x ) = ln( x ) ⇒ f ' ( x ) = 1 / x

 ln( x ) er udefineret, når x ≤ 0

 ln(0) er udefineret

Logaritmeproduktregel

Logaritmen af ​​multiplikationen af ​​x og y er summen af ​​logaritmen af ​​x og logaritmen af ​​y.

logb(x ∙ y) = logb(x) + logb(y)

For eksempel:

log10(3 ∙ 7) = log10(3) + log10(7)

Logaritmekvotientregel

Logaritmen for divisionen af ​​x og y er forskellen mellem logaritmen af ​​x og logaritmen af ​​y.

logb(x / y) = logb(x) - logb(y)

For eksempel:

log10(3 / 7) = log10(3) - log10(7)

Reglen for logaritmemagt

Logaritmen af ​​x hævet til y potens er y gange logaritmen af ​​x.

logb(x y) = y ∙ logb(x)

For eksempel:

log10(28) = 8∙ log10(2)

Afledt af naturlig logaritme

Den afledte af den naturlige logaritmefunktion er den reciproke funktion.

Hvornår

f (x) = ln(x)

Den afledte af f(x) er:

f ' (x) = 1 / x

Integral af naturlig logaritme

Integralet af den naturlige logaritmefunktion er givet ved:

Hvornår

f (x) = ln(x)

Integralet af f(x) er:

∫ f (x)dx = ∫ ln(x)dx = x ∙ (ln(x) - 1) + C

Ln på 0

Den naturlige logaritme af nul er udefineret:

ln(0) is undefined

Grænsen nær 0 for den naturlige logaritme af x, når x nærmer sig nul, er minus uendelig:

Ln af 1

Den naturlige logaritme af en er nul:

ln(1) = 0

Ln af uendelighed

Grænsen for naturlig logaritme af uendelighed, når x nærmer sig uendeligheden er lig med uendelig:

lim ln(x) = ∞, when x→∞

Kompleks logaritme

For komplekst tal z:

z = reiθ = x + iy

Den komplekse logaritme vil være (n = ...-2,-1,0,1,2,...):

Log z = ln(r) + i(θ+2nπ) = ln(√(x2+y2)) + i·arctan(y/x))
