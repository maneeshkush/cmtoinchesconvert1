Vægtet karakterberegning

Den vægtede karakter er lig med summen af ​​produktet af vægtene (w) i procent (%) gange karakteren (g):

Weighted grade = w1×g1+ w2×g2+ w3×g3+...

Når vægtene ikke er i procent (timer eller point...), skal du også dividere med summen af ​​vægtene:

Weighted grade = (w1×g1+ w2×g2+ w3×g3+...) / (w1+w2+w3+...)

Eksempel 1

3 point matematik kursus med karakteren 74.

5 point Biologikursus med karakteren 87.

2 point Historiekursus med karakteren 71.

Den vægtede gennemsnitskarakter beregnes ved:

Weighted grade =

 = (w1×g1+ w2×g2+ w3×g3) / (w1+w2+w3)

 = (3×74+ 5×87+ 2×71) / (3+5+2) = 79.90

Eksempel 2

3 point matematik kursus med karakteren 72.

5 point biologi kursus med karakteren 88.

2 point Historiekursus med karakteren 70.

Den vægtede gennemsnitskarakter beregnes ved:

Weighted grade =

 = (w1×g1+ w2×g2+ w3×g3) / (w1+w2+w3)

 = (3×72+ 5×88+ 2×70) / (3+5+2) = 79.60
