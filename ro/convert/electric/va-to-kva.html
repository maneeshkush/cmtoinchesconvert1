Cum se transformă VA în kVA

Cum se transformă puterea aparentă de la volți-amperi (VA) în kilovolti-amperi (kVA).

Formula de calcul Volt-amperi la kVA

Deci puterea aparentă S în kilovolt-amperi (kVA) este egală cu puterea aparentă S în volți-amperi (VA), împărțită la 1000:

S(kVA) =  S(VA) / 1000

 

Deci, kilovolti-amperi sunt egali cu volți-amperi împărțiți la 1000:

kilovolt-amps = volt-amps / 1000

sau

kVA = VA / 1000

Exemplul 1

Care este puterea aparentă în kilovolt-amperi când puterea aparentă în volți-amperi este de 30000VA?

Soluţie:

S = 30000VA / 1000 = 30kVA

Exemplul 2

Care este puterea aparentă în kilovolt-amperi când puterea aparentă în volți-amperi este de 40000 VA?

Soluţie:

S = 40000VA / 1000 = 40kVA

Exemplul 3

Care este puterea aparentă în kilovolt-amperi când puterea aparentă în volți-amperi este de 50000VA?

Soluţie:

S = 50000VA / 1000 = 50kVA

Exemplul 4

Care este puterea aparentă în kilovolt-amperi când puterea aparentă în volți-amperi este de 80000 VA?

Soluţie:

S = 80000VA / 1000 = 80kVA
