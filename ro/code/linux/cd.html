comanda cd în Linux/Unix

cd este o comandă Linux pentru a schimba directorul/folderul shell-ului terminalului.

Puteți apăsa butonul Tab pentru a completa automat numele directorului.

  • sintaxa cd
  • exemple de cd

sintaxa cd

$ cd [directory]

exemple de comandă cd

Schimbați în directorul principal (determinat de variabila de mediu $HOME):

$ cd

 

De asemenea, schimbați în directorul principal:

$ cd ~

 

Schimbați în directorul rădăcină:

$ cd /

 

Schimbați în directorul părinte:

$ cd ..

 

Schimbați în subdirectorul Documente :

$ cd Documents

 

Schimbați în subdirectorul Documente/Cărți :

$ cd Documents/Books

 

Schimbați în directorul cu calea absolută /home/user/Desktop :

$ cd /home/user/Desktop

 

Schimbați la numele directorului cu spațiu alb - Imaginile mele :

$ cd My\ Images

Sau

$ cd "My Images"

Sau

$ cd 'My Images'
