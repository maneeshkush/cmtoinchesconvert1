Reguli și proprietăți ale logaritmului natural

 

Numele regulii Regulă Exemplu
Regula produsului

ln(x ∙ y) = ln(x) + ln(y)

ln(3 ∙ 7) = ln(3) + ln(7)
Regula coeficientului

ln(x / y) = ln(x) - ln(y)

ln(3 / 7) = ln(3) - ln(7)
Regula puterii

ln(x y) = y ∙ ln(x)

ln(28) = 8 ∙ ln(2)
derivata Ln

f (x) = ln(x) ⇒ f ' (x) = 1 / x

 
Ln integrală

∫ ln(x)dx = x ∙ (ln(x) - 1) + C

  
Ln de număr negativ

ln(x) is undefined when x ≤ 0

  
Ln de zero

ln(0) is undefined

 
 
Ln de unul

ln(1) = 0

  
Ln de infinit

lim ln(x) = ∞ , when x→∞

  

 

Derivată a funcției logaritmului natural (ln).

Derivata funcției logaritmului natural este funcția reciprocă.

Când

f (x) = ln(x)

Derivata lui f(x) este:

f ' (x) = 1 / x

 

Integrală a funcției logaritmului natural (ln).

Integrala funcției logaritmului natural este dată de:

Când

f (x) = ln(x)

Integrala lui f(x) este:

∫ f (x)dx = ∫ ln(x)dx = x ∙ (ln(x) - 1) + C
