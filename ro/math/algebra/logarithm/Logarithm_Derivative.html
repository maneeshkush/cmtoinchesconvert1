Derivată a logaritmului

Când funcția logaritmică este dată de:

f (x) = logb(x)

Derivata functiei logaritmice este data de:

f ' (x) = 1 / ( x ln(b) )

x este argumentul funcției.

b este baza logaritmului.

ln b este logaritmul natural al lui b.

 

De exemplu când:

f (x) = log2(x)

f ' (x) = 1 / ( x ln(2) )
