Sistemul numeric

b - baza sistemului numeric

d n - a n-a cifră

n - poate începe de la un număr negativ dacă numărul are o parte fracțională.

N +1 - numărul de cifre

Sistem numeric binar - Baza-2

Numerele binare folosesc doar 0 și 1 cifre.

B înseamnă prefix binar.

Exemple:

101012 = 10101B = 1×24+0×23+1×22+0×21+1×20 = 16+4+1= 21

101112 = 10111B = 1×24+0×23+1×22+1×21+1×20 = 16+4+2+1= 23

1000112 = 100011B = 1×25+0×24+0×23+0×22+1×21+1×20 =32+2+1= 35

Sistemul numeric octal - Baza-8

Numerele octale folosesc cifre de la 0..7.

Exemple:

278 = 2×81+7×80 = 16+7 = 23

308 = 3×81+0×80 = 24

43078 = 4×83+3×82+0×81+7×80= 2247

Sistem numeral zecimal - Baza-10

Numerele zecimale folosesc cifre de la 0..9.

Acestea sunt numerele obișnuite pe care le folosim.

Exemplu:

253810 = 2×103+5×102+3×101+8×100

Sistem numeral hexazecimal - Baza-16

Numerele hexadecimale folosesc cifre de la 0..9 și A..F.

H desemnează prefixul hexadecimal.

Exemple:

2816 = 28H = 2×161+8×160 = 40

2F16 = 2FH = 2×161+15×160 = 47

BC1216 = BC12H = 11×163+12×162+1×161+2×160= 48146
