Procent (%)

Procentul este la sută, ceea ce înseamnă părți la sută.

Un procent este egal cu fracția 1/100:

1% = 1/100 = 0.01

Zece procente este egal cu fracția 10/100:

10% = 10/100 = 0.1

Cincizeci la sută este egal cu fracția 50/100:

50% = 50/100 = 0.5

Sută la sută este egal cu fracția 100/100:

100% = 100/100 = 1

O sută zece la sută este egal cu fracția 110/100:

110% = 110/100 = 1.1

Semn de procente

Semnul procentual este simbolul: %

Este scris în partea dreaptă a numărului: 50%

Definiție procentuală

Procentul este o valoare care reprezintă proporția dintre un număr și un alt număr.

1 procent reprezintă fracțiunea 1/100.

100% (100%) dintr-un număr este același număr:

100% × 80 = 100/100×80 = 80

50% (50%) dintr-un număr este jumătate din număr:

50% × 80 = 50/100×80 = 40

Deci 40 este 50% din 80.

Calcul procentual al unei valori

x% din y se calculează prin formula:

percentage value = x% × y = (x/100) × y

Exemplu:

Găsiți 40% din 200.

40% × 200 = (40 / 100) × 200 = 80

Calcul procentual

Procentul lui x din y, se calculează prin formula:

percentage = (x / y) × 100%

Exemplu:

Procentul de 30 din 60.

(30 / 60) × 100% = 50%

Modificare procentuală (creștere/scădere)

Modificarea procentuală de la x 1 la x 2 se calculează prin formula:

percentage change = 100% × (x2 - x1) / x1

Când rezultatul este pozitiv, avem o creștere sau o creștere procentuală.

Exemplu:

Modificare procentuală de la 60 la 80 (creștere).

100% × (80 - 60) / 60 = 33.33%

Când rezultatul este negativ, avem o scădere procentuală.

Exemplu:

Modificare procentuală de la 80 la 60 (scădere).

100% × (60 - 80) / 80 = -25%
