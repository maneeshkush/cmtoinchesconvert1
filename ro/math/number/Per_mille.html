pe milă (‰)

Per-mil sau per-mil înseamnă părți la mie.

Unul pe mil este egal cu 1/1000 fracție:

1‰ = 1/1000 = 0.001

Zece pe mil este egal cu fracția 10/1000:

10‰ = 10/1000 = 0.01

O sută pe mil este egal cu fracția 100/1000:

100‰ = 100/1000 = 0.1

O mie pe mil este egal cu fracția 1000/1000:

1000‰ = 1000/1000 = 1

Exemplu

Cât este 30 pe milă de 80 USD?

30‰ × 80$ = 0,030 × 80$ = 2,4$

Semnul per mile

Semnul per milă este simbolul: ‰

Este scris în partea dreaptă a numărului.de exemplu: 600‰

Per-mil - conversie procentuală

Unul pe mil este egal cu 0,1%:

1‰ = 0.1%

Un procent este egal cu 10 pe mil:

1% = 10‰
