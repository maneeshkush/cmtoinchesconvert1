Culoare CSS

Codul de culoare poate fi unul dintre:

Format hexadecimal: #rrggbb

Format RGB: rgb (rosu, verde, albastru)

Format nume: nume

Exemplu

Pentru culoarea portocalie:

Format hexadecimal: #FFA500

Format RGB: rgb(255,165,0)

Format nume: portocaliu

Setarea culorii elementului

Pentru un anumit element:

<element style="color: code;"></element>

Pentru toate elementele de același tip.Pune codul în eticheta <style> în secțiunea head:

<style>
   element { color: code; }
</style>

Exemplu

Setarea culorii textului paragrafului la roșu:

Setarea culorii unui anumit paragraf:

<p style="color: #FF0000;">Some text ...</p>

Vedere:

Un text...

Setarea culorii tuturor paragrafelor

<style>
   p { color: #FF0000; }
</style>

Setarea culorii de fundal a elementului

element { background: code; }

Exemplu

Setarea culorii de fundal a paragrafului la roșu:

Setarea culorii de fundal a unui anumit paragraf:

<p style="background: #FF0000;">Some text ...</p>

Vedere:

Un text...

Setarea culorii de fundal a tuturor paragrafelor:

<style>
   p { background: #FF0000; }
</style>

Setarea culorii marginii elementului

element { border-color: topcode rightcode bottomcode leftcode }

Exemplu

Setarea culorii marginii paragrafului la roșu, verde, albastru și negru:

Setarea culorii marginii unui anumit paragraf:

<p style="border-color: #FF0000 #00FF00 #0000FF #000000; border-style:solid">Some text ...</p>

Vedere:

Un text...

Setarea culorii chenarului tuturor paragrafelor:

<style>
   p { border-color: #FF0000 #00FF00 #0000FF #000000; }
</style>
