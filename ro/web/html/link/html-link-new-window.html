Link HTML într-o fereastră nouă

Cum se deschide un link într-o fereastră nouă sau o filă nouă.

Deschideți un link într-o fereastră sau filă nouă

Pentru a deschide un link într-o fereastră/filă nouă, adăugați target="_blank" în interiorul etichetei <a>:

<a href="../html-link.htm" target="_blank">Open page in new window</a>

Deschideți pagina într-o fereastră nouă

Fereastră nouă sau filă nouă

Nu puteți seta dacă linkul va fi deschis într-o fereastră nouă sau o filă nouă.Depinde de setările browserului. 

Deschideți un link într-o fereastră nouă cu dimensiunea specificată

Pentru a deschide un link într-o fereastră nouă, adăugați comanda Javascript onclick="window.open('text-link.htm', 'name','width=600,height=400') în interiorul etichetei <a>:

<a href="../html-link.htm" target="popup" onclick="window.open('../html-link.htm','name','width=600,height=400')">Open page in new window</a>
