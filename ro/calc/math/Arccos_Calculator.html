Definiția Arccos

Funcția arccosinus este funcția inversă a cos(x).

arccos(x) = cos-1(x)

 

De exemplu, dacă cosinusul de 60° este 0,5:

cos(60°) = 0,5

Atunci arcul lui 0,5 este de 60°:

arccos(0,5) = cos -1 (0,5) = 60°
