Calculul valorii viitoare

Suma viitoare după n ani A n este egală cu suma inițială A 0 ori unu plus rata anuală a dobânzii r împărțită la numărul de perioade de capitalizare dintr-un an m ridicat la puterea de m ori n:

A n  este suma după n ani (valoarea viitoare).

A 0  este suma inițială (valoarea actuală).

r este rata nominală anuală a dobânzii.

m este numărul de perioade de compus dintr-un an.

n este numărul de ani.

Exemplul #1:

Calculați valoarea viitoare după 10 ani valoarea actuală de 3.000 USD cu dobândă anuală de 4%.

Soluţie:

A 0 = 3.000 USD

r  = 4% = 4/100 = 0,04

m  = 1

n  = 10

A10 = $3,000·(1+0.04/1)(1·10) = $4,440.73

Exemplul #2:

Calculați valoarea viitoare după 8 ani, valoarea actuală de 40.000 USD cu o dobândă anuală de 3% compusă lunar.

Soluţie:

A 0 = 40.000 USD

r  = 3% = 3/100 = 0,03

m  = 12

n  = 8

A8 = $40,000·(1+0.03/12)(12·8) = $50,834.74
