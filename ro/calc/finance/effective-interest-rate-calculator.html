Calculul ratei dobânzii perioadei efective

Deci Rata dobânzii pe perioada efectivă este egală cu rata dobânzii anuală nominală împărțită la numărul de perioade pe an n:

Rata perioadei efective  = Rata anuală nominală / n

Exemplul 1

Care este rata efectivă a dobânzii pe perioada pentru rata dobânzii nominale anuale de 3% compusă lunar?

Soluţie:

Effective Period Rate = 3% / 12months = 0.03 / 12 = 0.250%

Exemplul 2

Care este rata dobânzii pe perioada efectivă pentru rata dobânzii anuale nominale de 7% compusă lunar?

Soluţie:

Effective Period Rate = 7% / 12months = 0.07 / 12 = 0.583%

Exemplul 3

Care este rata dobânzii pe perioada efectivă pentru rata dobânzii anuale nominale de 9% compusă lunar?

Soluţie:

Effective Period Rate = 9% / 12months = 0.9 / 12 = 0.750%

Calcul efectiv al ratei dobânzii anuale

Deci, rata dobânzii anuale efective este egală cu 1 plus rata dobânzii nominale în procente împărțit la numărul de periode compuse pe an n, la puterea lui n, minus 1.

Effective Rate = (1 +  Nominal Rate /  n)n - 1

Exemplul 1

Care este rata dobânzii anuale efective pentru rata dobânzii anuale nominale de 3% compusă lunar?

Soluţie:

Effective Rate = (1 + 3% / 12)12 - 1

      = (1 + 0,03 / 12) 12  - 1

      = 0,03042 = 3,042%

Exemplul 2

Care este rata dobânzii anuale efective pentru rata dobânzii anuale nominale de 7% compusă lunar?

Soluţie:

Effective Rate = (1 + 7% / 12)12 - 1

      = (1 + 0,07 / 12) 12  - 1

      = 0,07229 = 7,229%

Exemplul 3

Care este rata dobânzii anuale efective pentru rata dobânzii anuale nominale de 9% compusă lunar?

Soluţie:

Effective Rate = (1 + 9% / 12)12 - 1

      = (1 + 0,9 / 12) 12  - 1

      = 0,09381 = 9,381%
