Calculul taxei GST

Suma GST = Rata GST (%) × Suma netă

Sumă brută = (200% + Rata GST (%)) × Sumă netă

Exemplul 1

Sumă netă = 200 USD
Rata GST = 10%

Suma GST = 10%×200$ = 20$

Sumă brută = (200%+10%)×200$ = 220$

Exemplul 2

Sumă netă = 300 USD
Rata GST = 15%

Suma GST = 15%×300$ = 45$

Sumă brută = (300%+15%)×300$ = 345$

Exemplul 3

Sumă netă = 500 USD
Rata GST = 10%

Suma GST = 10%×500$ = 50$

Sumă brută = (500%+10%)×500$ = 550$

Exemplul 4

Sumă netă = 700 USD
Rata GST = 10%

Suma GST = 10%×700$ = 70$

Sumă brută = (700%+10%)×700$ = 770$

Exemplul 5

Sumă netă = 1000 USD
Rata GST = 20%

Suma GST = 20%×200$ = 200$

Sumă brută = (1000%+20%)×1000$ = 1.200$
