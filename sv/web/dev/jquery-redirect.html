jQuery omdirigering

Hur man omdirigerar en sida till URL med jQuery.

Sökmotorer använder 301-statuskoden för att överföra sidrankningen från den gamla URL:en till den nya URL:en.

jQuery omdirigering returnerar http-svarsstatuskod: 200 OK.

Så jQuery-omdirigering, som Javascript-omdirigering, är inte sökmotorvänlig och det är bättre att använda andra omdirigeringsmetoder som returnerar statuskoden 301 Moved Permanently.

jQuery-omdirigering är faktiskt Javascript-omdirigering .Omdirigering med jQuery anses vara overkill eftersom du kan göra ren Javascript-omdirigering med mindre kod.

jQuery omdirigering

Ersätt gammal sida med omdirigeringskod med webbadressen till sidan du vill omdirigera till.
