Detektering av skärmupplösning med Javascript

Du kan få den yttre fönsterstorleken med window.outerWidth och window.outerHeight .

Du kan få den inre fönsterstorleken med window.innerWidth och window.innerHeight .

Med IE kan du få det med document.body.clientWidth och document.body.clientHeight .

Du kan få skärmfönstrets position med window.screenLeft och window.screenTop .

Till exempel:

<script>
alert(window.outerWidth+' x '+window.outerHeight);
</script>
