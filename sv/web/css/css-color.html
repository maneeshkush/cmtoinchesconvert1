CSS-färg

Färgkoden kan vara en av:

Hexformat: #rrggbb

RGB-format: rgb (röd, grön, blå)

Namnformat: namn

Exempel

För orange färg:

Hexformat: #FFA500

RGB-format: rgb(255,165,0)

Namnformat: orange

Inställningselementets färg

För specifikt element:

<element style="color: code;"></element>

För alla element av samma typ.Lägg kod i <style>-taggen i rubriken:

<style>
   element { color: code; }
</style>

Exempel

Ställa in stycketextfärg till röd:

Ställa in färg för ett specifikt stycke:

<p style="color: #FF0000;">Some text ...</p>

Se:

Lite text...

Ställa in färg på alla stycken

<style>
   p { color: #FF0000; }
</style>

Inställningselementets bakgrundsfärg

element { background: code; }

Exempel

Ställ in styckebakgrundsfärg till röd:

Ställa in bakgrundsfärg för ett specifikt stycke:

<p style="background: #FF0000;">Some text ...</p>

Se:

Lite text...

Ställa in bakgrundsfärg för alla stycken:

<style>
   p { background: #FF0000; }
</style>

Inställningselementets kantfärg

element { border-color: topcode rightcode bottomcode leftcode }

Exempel

Ställa in styckekantsfärg till röd, grön, blå och svart:

Ställa in kantfärg för ett specifikt stycke:

<p style="border-color: #FF0000 #00FF00 #0000FF #000000; border-style:solid">Some text ...</p>

Se:

Lite text...

Ställa in kantfärg för alla stycken:

<style>
   p { border-color: #FF0000 #00FF00 #0000FF #000000; }
</style>
