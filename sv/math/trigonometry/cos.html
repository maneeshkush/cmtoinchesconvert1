Cosinus definition

I en rätvinklig triangel ABC definieras sinus för α, sin(α) som förhållandet mellan den sida som gränsar till vinkeln α och den sida som är motsatt den räta vinkeln (hypotenus):

cos α = b / c

Exempel

b = 3"

c = 5"

cos α = b / c = 3 / 5 = 0.6

Omvänd cosinusfunktion

Arccosinusför x definieras som den inversa cosinusfunktionen för x när -1≤x≤1 .

När cosinus för y är lika med x:

cos y = x

Då är arccosinus för x lika med den inversa cosinusfunktionen för x, som är lika med y:

arccos x = cos-1 x = y

Exempel

arccos 1 = cos-1 1 = 0 rad = 0°
