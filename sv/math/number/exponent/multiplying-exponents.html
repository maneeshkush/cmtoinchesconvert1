Multiplicera exponenter med samma bas

För exponenter med samma bas bör vi lägga till exponenterna:

a n ⋅ a m = a n+m

Exempel:

23 ⋅ 24 = 23+4 = 27 = 2⋅2⋅2⋅2⋅2⋅2⋅2 = 128

Multiplicera exponenter med olika baser

När baserna är olika och exponenterna för a och b är lika, kan vi multiplicera a och b först:

a n ⋅ b n = (a ⋅ b) n

Exempel:

32 ⋅ 42 = (3⋅4)2 = 122 = 12⋅12 = 144

 

När baserna och exponenterna är olika måste vi beräkna varje exponent och sedan multiplicera:

a n ⋅ b m

Exempel:

32 ⋅ 43 = 9 ⋅ 64 = 576

Multiplicera negativa exponenter

För exponenter med samma bas kan vi lägga till exponenterna:

a -n ⋅ a -m = a -(n+m) = 1 / a n+m

Exempel:

2-3 ⋅ 2-4 = 2-(3+4) = 2-7 = 1 / 27 = 1 / (2⋅2⋅2⋅2⋅2⋅2⋅2) = 1 / 128 = 0.0078125

 

När baserna är olika och exponenterna för a och b är lika, kan vi multiplicera a och b först:

a -n ⋅ b -n = (a ⋅ b) -n

Exempel:

3-2 ⋅ 4-2 = (3⋅4)-2 = 12-2 = 1 / 122 = 1 / (12⋅12) = 1 / 144 = 0.0069444

 

När baserna och exponenterna är olika måste vi beräkna varje exponent och sedan multiplicera:

a -n ⋅ b -m

Exempel:

3-2 ⋅ 4-3 = (1/9) ⋅ (1/64) = 1 / 576 = 0.0017361

Multiplicera bråk med exponenter

Multiplicera bråk med exponenter med samma bråkbas:

(a / b) n ⋅ (a / b) m = (a / b) n+m

Exempel:

(4/3)3 ⋅ (4/3)2 = (4/3)3+2 = (4/3)5 = 45 / 35 = 4.214

 

Multiplicera bråk med exponenter med samma exponent:

(a / b) n ⋅ (c / d) n = ((a / b)⋅(c / d)) n

Exempel:

(4/3)3 ⋅ (3/5)3 = ((4/3)⋅(3/5))3 = (4/5)3 = 0.83 = 0.8⋅0.8⋅0.8 = 0.512

 

Multiplicera bråk med exponenter med olika baser och exponenter:

(a / b) n ⋅ (c / d) m

Exempel:

(4/3)3 ⋅ (1/2)2 = 2.37 ⋅ 0.25 = 0.5925

Multiplicera bråkexponenter

Multiplicera bråkexponenter med samma bråkexponent:

a n/m ⋅ b n/m = (a ⋅ b) n/m

Exempel:

23/2 ⋅ 33/2 = (2⋅3)3/2 = 63/2 = √(63) = √216 = 14.7

 

Multiplicera bråkexponenter med samma bas:

a (n/m) ⋅ a (k/j) = a [(n/m)+(k/j)]

Exempel:

2(3/2) ⋅ 2(4/3) = 2[(3/2)+(4/3)] = 7.127

 

Multiplicera bråkexponenter med olika exponenter och bråk:

a n/m ⋅ b k/j

Exempel:

2 3/2 ⋅ 24/3 = √(23) ⋅ 3√(24) = 2.828 ⋅ 2.52 = 7.127

Multiplicera kvadratrötter med exponenter

För exponenter med samma bas kan vi lägga till exponenterna:

(√a)n ⋅ (√a)m = a(n+m)/2

Exempel:

(√5)2 ⋅ (√5)4 = 5(2+4)/2 = 56/2 = 53 = 125

Multiplicera variabler med exponenter

För exponenter med samma bas kan vi lägga till exponenterna:

xn ⋅ xm = xn+m

Exempel:

x2 ⋅ x3 = (x⋅x) ⋅ (x⋅x⋅x) = x2+3 = x5
