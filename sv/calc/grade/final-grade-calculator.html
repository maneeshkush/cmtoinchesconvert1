Slutbetygsberäkning

Exempel #1

Nuvarande betyg är 70% (eller C-).

Slutprovets vikt är 50%.

Betygskrav är 80 % (eller B-).

Beräkning:

Så Det slutliga betyget är lika med det obligatoriska betyget, minus 100 % minus slutprovets vikt (w) gånger aktuellt betyg (g), dividerat med slutprovets vikt (w).

Slutprovsbetyg =

= ( krävs betyg - (100 % - w)× nuvarande betyg ) / w

= ( 80 % - (100 % - 50 %)×70 % ) / 50 % = 90 %

Så slutbetyget på tentamen ska vara 90% (eller A-).

Exempel #2

Uppgift 1: vikt1=50 %, betyg1=16 av 20.

Uppgift 2: vikt2=30%, maxbetyg=30.

Uppgift 3: vikt3=20%, maxbetyg=40.

Hitta medelbetyg i uppgifter 2 och 3 som behövs för att få klassbetyg på 85 %.

Beräkning:

Nuvarande betyg = Uppgift 1 betyg = betyg1 / maxbetyg1 = 16/20 = 0,8 = 80 %

Betygskrav = 85%

Slutprovets vikt = w = vikt2+vikt3 = 30%+20% = 50%

Slutprovsbetyg =

= ( krävs betyg - (100 % - w )× nuvarande betyg ) / w

= ( 85 % - 50 %×80 % ) / 50 % = 90 %

Det innebär att du måste få ett snittbetyg på 90% i inlämningsuppgifter 2 och 3 för att få klassbetyg på 85%.

Uppgift 2 betyg = 90 % × maxbetyg = 90 % × 30 = 27

Uppgift 3 betyg = 90 % × maxbetyg = 90 % × 40 = 36

Snabb konvertering
