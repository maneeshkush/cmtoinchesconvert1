Hur man konverterar RGB till hex-färg

Hur man konverterar från RGB-färg till hexadecimal färgkod.

RGB-färg

RGB-färg är en kombination av röd, grön och blå färg:

(R, G, B)

Rött, grönt och blått använder vardera 8 bitar, med heltalsvärden från 0 till 255.

Så antalet färger som kan genereras är:

256×256×256 = 16777216 = 100000016

hex färgkod

Hexfärgkoden är ett sexsiffrigt hexadecimalt tal (bas 16):

RRGGBB 16

De två vänstra siffrorna representerar den röda färgen.

De två mittersta siffrorna representerar den gröna färgen.

De två högra siffrorna representerar den blå färgen.

rgb till hex konvertering

1. Konvertera de röda, gröna och blå värdena från decimal till hex.
2. Kombinera de 3 hexvärdena rött, grönt och blått tillsammans: RRGGBB.

Exempel 1
Konvertera röd (255,0,0) till hexadecimal färgkod:

R = 25510 = FF16

G = 010 = 0016

B = 010 = 0016

Så hex-färgkoden är:

Hex = FF0000

Exempel #2
Konvertera guldfärg (255,215,0) till hexadecimal färgkod:

R = 25510 = FF16

G = 21510 = D716

B = 010 = 0016

Så hex-färgkoden är:

Hex = FFD700

Vad gör denna RGB till Hex-omvandlare?

Den tar som ingång röda, gröna och blå färgvärden som sträcker sig från 0 till 255 och konverterar sedan dessa värden till en hexadecimal sträng som kan användas för att specificera färger i html/css-kod.Fotoredigeringsprogram representerar vanligtvis färg i RGB och så om du vill använda färgerna som används i ditt fotoredigeringsprogram som bakgrund för ditt HTML-element måste du få den hexadecimala representationen av RGB-värden.Detta verktyg låter dig få dessa värden.
