HSL till RGB-konverteringsformel

När 0 ≤ H < 360, 0 ≤ S ≤ 1 och 0 ≤ L ≤ 1:

C = (1 - |2L - 1|) × S

X = C × (1 - |(H / 60°) mod 2 - 1|)

m = L - C/2

(R,G,B) = ((R'+m)×255, (G'+m)×255,(B'+m)×255)

HSL till RGB färgkonvertering

RGB-färgrymd är en additiv färgrymd som använder tre primärfärger, röd, grön och blå, för att skapa alla andra färger.RGB-färgvärden anges med tre 8-bitars heltal, ett för varje primärfärg.Detta skapar en rad möjliga färger från 0 (inget ljus) till 255 (fullt ljus).

HSL (hue, saturation, lightness) är en färgrymd som är ett mer intuitivt sätt att specificera färger än RGB.HSL-värden anges med tre flyttal, ett för varje komponent.Området för möjliga HSL-värden är från 0 (ingen nyans) till 1 (full mättnad och ljushet).

RGB till HSL-konvertering är en process för att konvertera RGB-färgvärden till HSL-värden.RGB till HSL-konverteringsformeln är:

nyans = (röd - grön) / (röd + grön + blå)
mättnad = (blå - grön) / (blå + grön + röd)

Funktioner i HSL till RGB Color Converter Tool

  1. HSL-inmatning (Hue, Saturation och Lightness): Verktyget låter dig mata in färger i HSL-färgrymden, som är baserad på de tre egenskaperna nyans, mättnad och ljushet.

  2. RGB-utdata (röd, grön, blå): Verktyget konverterar HSL-färger till RGB-färgrymden, som är baserad på de tre primärfärgerna rött, grönt och blått.

  3. Färgförhandsgranskning: Verktyget inkluderar vanligtvis en färgförhandsgranskningsfunktion som låter dig se en representation av den valda HSL-färgen som den kommer att visas i RGB-färgrymden.

  4. Justerbara skjutreglage: Många HSL till RGB färgkonverteringsverktyg inkluderar justerbara skjutreglage eller inmatningsfält som låter dig finjustera värdena för HSL-färgen för att få önskad RGB-utgång.

  5. Hexadecimal utdata: Verktyget kan också tillhandahålla den resulterande RGB-färgen i det hexadecimala färgformatet, som är en standardrepresentation av färger som används i webbdesign och andra digitala applikationer.

  6. Färgpalett: Vissa konverteringsverktyg från HSL till RGB inkluderar en färgpalettfunktion som låter dig välja från en rad förinställda färger eller skapa dina egna anpassade färger.

  7. Färghistorik: Vissa verktyg kan också ha en färghistorikfunktion som låter dig hålla reda på färgerna du har konverterat, vilket gör det lättare att använda samma färger i flera projekt.

  8. Kompatibilitet med olika färgrymder: Vissa HSL till RGB-konverteringsverktyg är kompatibla med andra färgrymder som CMYK (Cyan, Magenta, Yellow och Black) eller HSB (Hue, Saturation och Brightness), vilket gör att du kan konvertera mellan dessa olika färger modeller också.
