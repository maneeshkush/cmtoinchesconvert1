Hex till RGB-konvertering

  1. Hämta de två vänstra siffrorna i hex-färgkoden och konvertera till decimalvärdet för att få den röda färgnivån.
  2. Hämta de två mittersta siffrorna i hex-färgkoden och konvertera till decimalvärde för att få den gröna färgnivån.
  3. Hämta de två rätta siffrorna i hex-färgkoden och konvertera till decimalvärde för att få den blå färgnivån.

Exempel #1

Konvertera röd hex-färgkod FF0000 till RGB-färg:

Hex = FF0000

Så RGB-färgerna är:

R = FF16 = 25510

G = 0016 = 010

B = 0016 = 010

Eller

RGB = (255, 0, 0)

Exempel #2

Konvertera guld hex färgkod FFD700 till RGB färg:

Hex = FFD700

Så RGB-färgerna är:

R = FF16 = 25510

G = D716 = 21510

B = 0016 = 010

Eller

RGB = (255, 215, 0)
