cd-kommando i Linux/Unix

cd er en Linux-kommando for å endre katalogen/mappen til terminalens skall.

Du kan trykke på tabulatorknappen for å fullføre katalognavnet automatisk.

  • cd syntaks
  • eksempler på cd

cd syntaks

$ cd [directory]

eksempler på cd-kommandoer

Bytt til hjemmekatalog (bestemt av miljøvariabelen $HOME):

$ cd

 

Bytt også til hjemmekatalog:

$ cd ~

 

Bytt til rotkatalog:

$ cd /

 

Bytt til overordnet katalog:

$ cd ..

 

Bytt til underkatalog Dokumenter :

$ cd Documents

 

Bytt til underkatalog Dokumenter/Bøker :

$ cd Documents/Books

 

Bytt til katalog med absolutt bane /home/user/Desktop :

$ cd /home/user/Desktop

 

Bytt til katalognavn med mellomrom - Mine bilder :

$ cd My\ Images

Eller

$ cd "My Images"

Eller

$ cd 'My Images'
