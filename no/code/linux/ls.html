ls kommando i Linux/Unix

ls er en Linux-shell-kommando som viser kataloginnholdet i filer og kataloger.

  • ls syntaks
  • ls alternativer
  • er eksempler
  • ls kodegenerator

ls syntaks

$ ls [options] [file|dir]

ls kommandoalternativer

ls kommando hovedalternativer:

alternativ beskrivelse
ls -a liste alle filer inkludert skjult fil som starter med '.'
ls --farge farget liste [=alltid/aldri/auto]
ls -d liste kataloger - med ' */'
ls -F legg til ett tegn av */=>@|til enteries
ls -i listefilens inodeindeksnummer
ls -l liste med langt format - vis tillatelser
ls -la liste langt format inkludert skjulte filer
ls -lh liste langt format med lesbar filstørrelse
ls -ls liste med langt format med filstørrelse
ls -r liste i omvendt rekkefølge
ls -R liste rekursivt katalogtre
ls -s liste filstørrelse
ls -S sorter etter filstørrelse
ls -t sorter etter tid og dato
ls -X sorter etter utvidelsesnavn

Eksempler på kommandoer

Du kan trykke på tabulatorknappen for å fullføre fil- eller mappenavnene automatisk.

Listekatalog Dokumenter/bøker med relativ bane:

$ ls Documents/Books

 

Listekatalog /hjem/bruker/Dokumenter/Bøker med absolutt bane.

$ ls /home/user/Documents/Books

 

Liste over rotkatalogen:

$ ls /

 

List over overordnet katalog:

$ ls ..

 

List opp brukerens hjemmekatalog (f.eks: /hjem/bruker):

$ ls ~

 

Liste med langt format:

$ ls -l

 

Vis skjulte filer:

$ ls -a

 

List med langt format og vis skjulte filer:

$ ls -la

 

Sorter etter dato/klokkeslett:

$ ls -t

 

Sorter etter filstørrelse:

$ ls -S

 

Vis alle underkataloger:

$ ls *

 

Treliste for rekursiv katalog:

$ ls -R

 

Vis bare tekstfiler med jokertegn:

$ ls *.txt

 

ls omdirigering til utdatafil:

$ ls > out.txt

 

List kun kataloger:

$ ls -d */

 

Liste filer og kataloger med full bane:

$ ls -d $PWD/*
