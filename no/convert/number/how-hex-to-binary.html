Eksempel #1

Konverter (4D) 16  til binær:

(4)16 = (0100)2

(D)16 = (1101)2

Så

(4D)16 = (01001101)2

Eksempel #2

Konverter (4A0D) 16  til binær:

(4)16 = (0100)2

(A)16 = (1010)2

(0)16 = (0000)2

(D)16 = (1101)2

Så

(4A0D)16 = (0100101000001101)2

Eksempel #3

Konverter (EA0D) 16  til binær:

(E)16 = (1110)2

(A)16 = (1010)2

(0)16 = (0000)2

(D)16 = (1101)2

Så

(EA0D)16 = (1110101000001101)2
