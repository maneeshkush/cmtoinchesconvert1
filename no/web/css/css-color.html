CSS-farge

Fargekoden kan være en av:

Hex-format: #rrggbb

RGB-format: rgb (rød, grønn, blå)

Navneformat: navn

Eksempel

For oransje farge:

Hexformat: #FFA500

RGB-format: rgb(255,165,0)

Navneformat: oransje

Innstilling av elementfarge

For spesifikt element:

<element style="color: code;"></element>

For alle elementer av samme type.Sett kode i <style>-taggen i head-delen:

<style>
   element { color: code; }
</style>

Eksempel

Sette avsnittstekstfargen til rød:

Angi farge på et bestemt avsnitt:

<p style="color: #FF0000;">Some text ...</p>

Utsikt:

Litt tekst...

Angi farge på alle avsnitt

<style>
   p { color: #FF0000; }
</style>

Innstillingselementets bakgrunnsfarge

element { background: code; }

Eksempel

Sette avsnitts bakgrunnsfarge til rød:

Angi bakgrunnsfarge for et bestemt avsnitt:

<p style="background: #FF0000;">Some text ...</p>

Utsikt:

Litt tekst...

Angi bakgrunnsfarge for alle avsnitt:

<style>
   p { background: #FF0000; }
</style>

Innstillingselementets kantfarge

element { border-color: topcode rightcode bottomcode leftcode }

Eksempel

Sette avsnittskantfarge til rød, grønn, blå og svart:

Angi kantfarge for et bestemt avsnitt:

<p style="border-color: #FF0000 #00FF00 #0000FF #000000; border-style:solid">Some text ...</p>

Utsikt:

Litt tekst...

Angi kantfarge for alle avsnitt:

<style>
   p { border-color: #FF0000 #00FF00 #0000FF #000000; }
</style>
