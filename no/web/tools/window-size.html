Skjermoppløsningsdeteksjon med Javascript

Du kan få den ytre vindusstørrelsen med window.outerWidth og window.outerHeight .

Du kan få innvendig vindusstørrelse med window.innerWidth og window.innerHeight .

Med IE kan du få det med document.body.clientWidth og document.body.clientHeight .

Du kan få skjermvinduposisjonen med window.screenLeft og window.screenTop .

For eksempel:

<script>
alert(window.outerWidth+' x '+window.outerHeight);
</script>

Funksjoner av Window size Tool

Vårt verktøy for vindustørrelse lar brukerne få vindustørrelse.Noen av de fremtredende funksjonene til dette verktøyet er forklart nedenfor.

Enkelhet

Det brukervennlige grensesnittet til verktøyet for vindustørrelse gir deg enkelt å utføre denne konverteringen umiddelbart.Du trenger ikke lenger å tilegne deg programmeringskunnskaper for vindusstørrelse på grunn av dette verktøyets enkelhet.

Ingen registrering

Du trenger ikke gå gjennom noen registreringsprosess for å bruke Window-størrelsen.Ved å bruke dette verktøyet kan du gratis Window size Tool så mange ganger du vil.

Bærbarhet

Denne vindusstørrelsen kan nås fra ethvert hjørne av verden.Du trenger ikke å bekymre deg for kompatibiliteten til dette nettbaserte verktøyet, siden du kan vindusstørrelse med det fra hvilken som helst enhet.Alt du trenger er en internettforbindelse for å få tilgang til og bruke dette verktøyet for vindustørrelse.

Sparer tid og krefter

Vindustørrelsen lar deg fullføre den samme oppgaven umiddelbart.Du vil ikke bli bedt om å følge manuelle prosedyrer, siden dens automatiserte algoritmer vil gjøre jobben for deg.

Kompatibilitet

Vindusstørrelsesverktøyet fungerer perfekt på alle operativsystemer.Enten du har en Mac-, iOS-, Android-, Windows- eller Linux-enhet, kan du enkelt bruke dette nettbaserte verktøyet uten problemer.

100 % gratis

Du trenger ikke gå gjennom noen registreringsprosess for å bruke dette Vindusstørrelsesverktøyet.Du kan bruke dette verktøyet gratis og gjøre ubegrenset vindusstørrelse uten noen begrensninger.
