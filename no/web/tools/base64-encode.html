Tekst til base64-kodingseksempel

Tekstinndata:

Tekstinndata blir først kodet som binær bitstrøm av ASCII- koder for hvert tegn.

Hver 6 biter av bitstrømmen er kodet til base64-siffer.

Bilde til base64-kodingseksempel

Bildeinndata:

Bildedata URI-skjema med base64-kodet datautgang:

 

data:image/jpeg;base64,er data-URI-skjemaoverskriften.

 

/9j/4AAQSkZ...er de kodede base64-dataene.

 

HTML <img> med base64 data URI

<img src="data:image/jpeg;base64,/9j/4AAQSkZ...">

 

CSS-bakgrunn med base64-data-URI

body {
    background-image: url('data:image/jpeg;base64,/9j/4AAQSkZ...');
}
