Skjermoppløsningsdeteksjon med Javascript

Du kan få skjermstørrelsen med egenskaper for screen.width og screen.height .

Du kan få tilgjengelig skjermstørrelse med egenskaper for screen.availWidth og screen.availHeight .

For eksempel:

<script>
alert(screen.width+' x '+screen.height);
</script>
