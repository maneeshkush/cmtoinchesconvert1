Tallsystem

b - tallsystembase

d n - det n-te sifferet

n - kan starte fra negativt tall hvis tallet har en brøkdel.

N +1 - antall sifre

Binært tallsystem - Base-2

Binære tall bruker bare 0 og 1 sifre.

B angir binært prefiks.

Eksempler:

101012 = 10101B = 1×24+0×23+1×22+0×21+1×20 = 16+4+1= 21

101112 = 10111B = 1×24+0×23+1×22+1×21+1×20 = 16+4+2+1= 23

1000112 = 100011B = 1×25+0×24+0×23+0×22+1×21+1×20 =32+2+1= 35

Oktalt tallsystem - Base-8

Oktale tall bruker sifre fra 0...7.

Eksempler:

278 = 2×81+7×80 = 16+7 = 23

308 = 3×81+0×80 = 24

43078 = 4×83+3×82+0×81+7×80= 2247

Desimaltallsystem - Base-10

Desimaltall bruker sifre fra 0..9.

Dette er de vanlige tallene vi bruker.

Eksempel:

253810 = 2×103+5×102+3×101+8×100

Heksadesimalt tallsystem - Base-16

Hex-tall bruker sifre fra 0..9 og A..F.

H angir sekskantprefiks.

Eksempler:

2816 = 28H = 2×161+8×160 = 40

2F16 = 2FH = 2×161+15×160 = 47

BC1216 = BC12H = 11×163+12×162+1×161+2×160= 48146
