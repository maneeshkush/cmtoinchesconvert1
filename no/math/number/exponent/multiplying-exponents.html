Multiplisere eksponenter

Multiplisere eksponenter med samme base

For eksponenter med samme base bør vi legge til eksponentene:

a n ⋅ a m = a n+m

Eksempel:

23 ⋅ 24 = 23+4 = 27 = 2⋅2⋅2⋅2⋅2⋅2⋅2 = 128

Multiplisere eksponenter med forskjellige baser

Når basene er forskjellige og eksponentene til a og b er like, kan vi multiplisere a og b først:

a n ⋅ b n = (a ⋅ b) n

Eksempel:

32 ⋅ 42 = (3⋅4)2 = 122 = 12⋅12 = 144

 

Når basene og eksponentene er forskjellige, må vi beregne hver eksponent og deretter multiplisere:

a n ⋅ b m

Eksempel:

32 ⋅ 43 = 9 ⋅ 64 = 576

Multiplisere negative eksponenter

For eksponenter med samme base kan vi legge til eksponentene:

a -n ⋅ a -m = a -(n+m) = 1 / a n+m

Eksempel:

2-3 ⋅ 2-4 = 2-(3+4) = 2-7 = 1 / 27 = 1 / (2⋅2⋅2⋅2⋅2⋅2⋅2) = 1 / 128 = 0.0078125

 

Når basene er forskjellige og eksponentene til a og b er like, kan vi multiplisere a og b først:

a -n ⋅ b -n = (a ⋅ b) -n

Eksempel:

3-2 ⋅ 4-2 = (3⋅4)-2 = 12-2 = 1 / 122 = 1 / (12⋅12) = 1 / 144 = 0.0069444

 

Når basene og eksponentene er forskjellige, må vi beregne hver eksponent og deretter multiplisere:

a -n ⋅ b -m

Eksempel:

3-2 ⋅ 4-3 = (1/9) ⋅ (1/64) = 1 / 576 = 0.0017361

Multiplisere brøker med eksponenter

Multiplisere brøker med eksponenter med samme brøkgrunnlag:

(a / b) n ⋅ (a / b) m = (a / b) n+m

Eksempel:

(4/3)3 ⋅ (4/3)2 = (4/3)3+2 = (4/3)5 = 45 / 35 = 4.214

 

Multiplisere brøker med eksponenter med samme eksponent:

(a / b) n ⋅ (c / d) n = ((a / b)⋅(c / d)) n

Eksempel:

(4/3)3 ⋅ (3/5)3 = ((4/3)⋅(3/5))3 = (4/5)3 = 0.83 = 0.8⋅0.8⋅0.8 = 0.512

 

Multiplisere brøker med eksponenter med forskjellige baser og eksponenter:

(a / b) n ⋅ (c / d) m

Eksempel:

(4/3)3 ⋅ (1/2)2 = 2.37 ⋅ 0.25 = 0.5925

Multiplisere brøkeksponenter

Multiplisere brøkeksponenter med samme brøkeksponent:

a n/m ⋅ b n/m = (a ⋅ b) n/m

Eksempel:

23/2 ⋅ 33/2 = (2⋅3)3/2 = 63/2 = √(63) = √216 = 14.7

 

Multiplisere brøkeksponenter med samme base:

a (n/m) ⋅ a (k/j) = a [(n/m)+(k/j)]

Eksempel:

2(3/2) ⋅ 2(4/3) = 2[(3/2)+(4/3)] = 7.127

 

Multiplisere brøkeksponenter med forskjellige eksponenter og brøker:

a n/m ⋅ b k/j

Eksempel:

2 3/2 ⋅ 24/3 = √(23) ⋅ 3√(24) = 2.828 ⋅ 2.52 = 7.127

Multiplisere kvadratrøtter med eksponenter

For eksponenter med samme base kan vi legge til eksponentene:

(√a)n ⋅ (√a)m = a(n+m)/2

Eksempel:

(√5)2 ⋅ (√5)4 = 5(2+4)/2 = 56/2 = 53 = 125

Multiplisere variabler med eksponenter

For eksponenter med samme base kan vi legge til eksponentene:

xn ⋅ xm = xn+m

Eksempel:

x2 ⋅ x3 = (x⋅x) ⋅ (x⋅x⋅x) = x2+3 = x5
