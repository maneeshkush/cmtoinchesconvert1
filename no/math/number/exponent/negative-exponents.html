Negative eksponenter regjerer

Grunntallet b hevet til minus n potens er lik 1 delt på grunntallet b hevet til n:

b-n = 1 / bn

Eksempel på negativ eksponent

Grunntallet 2 hevet til minus 3 er lik 1 delt på grunntallet 2 hevet til 3 potens:

2-3 = 1/23 = 1/(2⋅2⋅2) = 1/8 = 0.125

Negative brøkeksponenter

Grunnlaget b hevet til potensen minus n/m er lik 1 delt på grunntallet b hevet til n/m potens:

b-n/m = 1 / bn/m = 1 / (m√b)n

Grunntallet 2 hevet til minus 1/2 er lik 1 delt på grunntallet 2 hevet til 1/2:

2-1/2 = 1/21/2 = 1/√2 = 0.7071

Brøker med negative eksponenter

Grunntallet a/b hevet til minus n er lik 1 delt på grunntallet a/b hevet til n:

(a/b)-n = 1 / (a/b)n = 1 / (an/bn) = bn/an

Grunntallet 2 hevet til minus 3 er lik 1 delt på grunntallet 2 hevet til 3 potens:

(2/3)-2 = 1 / (2/3)2 = 1 / (22/32) = 32/22 = 9/4 = 2.25

Multiplisere negative eksponenter

For eksponenter med samme base kan vi legge til eksponentene:

a -n ⋅ a -m = a -(n+m) = 1 / a n+m

Eksempel:

2-3 ⋅ 2-4 = 2-(3+4) = 2-7 = 1 / 27 = 1 / (2⋅2⋅2⋅2⋅2⋅2⋅2) = 1 / 128 = 0.0078125

 

Når basene er forskjellige og eksponentene til a og b er like, kan vi multiplisere a og b først:

a -n ⋅ b -n = (a ⋅ b) -n

Eksempel:

3-2 ⋅ 4-2 = (3⋅4)-2 = 12-2 = 1 / 122 = 1 / (12⋅12) = 1 / 144 = 0.0069444

 

Når basene og eksponentene er forskjellige, må vi beregne hver eksponent og deretter multiplisere:

a -n ⋅ b -m

Eksempel:

3-2 ⋅ 4-3 = (1/9) ⋅ (1/64) = 1 / 576 = 0.0017361

 

Deling av negative eksponenter

For eksponenter med samme grunntall bør vi trekke fra eksponentene:

a n / a m = a n-m

Eksempel:

26 / 23 = 26-3 = 23 = 2⋅2⋅2 = 8

 

Når basene er forskjellige og eksponentene til a og b er like, kan vi dele a og b først:

a n / b n = (a / b) n

Eksempel:

63 / 23 = (6/2)3 = 33 = 3⋅3⋅3 = 27

 

Når basene og eksponentene er forskjellige, må vi beregne hver eksponent og deretter dele:

a n / b m

Eksempel:

62 / 33 = 36 / 27 = 1.333
