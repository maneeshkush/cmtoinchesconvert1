Legge til eksponenter

Legge til tall med eksponenter

Å legge til eksponenter gjøres ved å beregne hver eksponent først og deretter legge til:

an + bm

Eksempel:

42 + 25 = 4⋅4+2⋅2⋅2⋅2⋅2 = 16+32 = 48

Legge til samme baser b og eksponenter n:

bn + bn = 2bn

Eksempel:

42 + 42 = 2⋅42 = 2⋅4⋅4 = 32

Legge til negative eksponenter

Å legge til negative eksponenter gjøres ved å beregne hver eksponent først og deretter legge til:

a-n + b-m = 1/an + 1/bm

Eksempel:

4-2 + 2-5 = 1/42 + 1/25 = 1/(4⋅4)+1/(2⋅2⋅2⋅2⋅2) = 1/16+1/32 = 0.09375

Legge til brøkeksponenter

Å legge til brøkeksponenter gjøres ved å heve hver eksponent først og deretter legge til:

an/m + bk/j

Eksempel:

33/2 + 25/2 = √(33) + √(25) = √(27) + √(32) = 5.196 + 5.657 = 10.853

 

Legge til samme baser b og eksponenter n/m:

bn/m + bn/m = 2bn/m

Eksempel:

42/3 + 42/3 = 2⋅42/3 = 2 ⋅ 3√(42) = 5.04

Legge til variabler med eksponenter

Å legge til eksponenter gjøres ved å beregne hver eksponent først og deretter legge til:

xn + xm

Med samme eksponenter:

xn + xn = 2xn

Eksempel:

x2 + x2 = 2x2
