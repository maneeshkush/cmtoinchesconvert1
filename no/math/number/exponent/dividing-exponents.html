Å dele eksponenter med samme base

For eksponenter med samme grunntall bør vi trekke fra eksponentene:

a n / a m = a n-m

Eksempel:

26 / 23 = 26-3 = 23 = 2⋅2⋅2 = 8

Dele eksponenter med forskjellige baser

Når basene er forskjellige og eksponentene til a og b er like, kan vi dele a og b først:

a n / b n = (a / b) n

Eksempel:

63 / 23 = (6/2)3 = 33 = 3⋅3⋅3 = 27

 

Når basene og eksponentene er forskjellige, må vi beregne hver eksponent og deretter dele:

a n / b m

Eksempel:

62 / 33 = 36 / 27 = 1.333

Deling av negative eksponenter

For eksponenter med samme grunntall kan vi trekke fra eksponentene:

a-n / a-m = a-n-(-m) = am-n

Eksempel:

2-3 / 2-5 = 25-3 = 22 = 2⋅2 = 4

 

Når basene er forskjellige og eksponentene til a og b er like, kan vi multiplisere a og b først:

a-n / b-n = (a/b)-n = 1 / (a/b)n = (b/a)n

Eksempel:

3-2 / 4-2 = (4/3)2 = 1.7778

 

Når basene og eksponentene er forskjellige, må vi beregne hver eksponent og deretter dele:

a-n / b-m = bm / an

Eksempel:

3-2 / 4-3 = 43 / 32 = 64 / 9 = 7.111

Dele brøker med eksponenter

Å dele brøker med eksponenter med samme brøkgrunnlag:

(a / b)n / (a / b)m = (a / b)n-m

Eksempel:

(4/3)3 / (4/3)2 = (4/3)3-2 = (4/3)1 = 4/3 = 1.333

 

Å dele brøker med eksponenter med samme eksponent:

(a / b)n / (c / d)n = ((a / b)/(c / d))n = ((a⋅d / b⋅c))n

Eksempel:

(4/3)3 / (3/5)3 = ((4/3)/(3/5))3 = ((4⋅5)/(3⋅3))3 = (20/9)3 = 10.97

 

Dele brøker med eksponenter med forskjellige baser og eksponenter:

(a / b) n / (c / d) m

Eksempel:

(4/3)3 / (1/2)2 = 2.37 / 0.25 = 9.481

Deling av brøkeksponenter

Å dele brøkeksponenter med samme brøkeksponent:

a n/m / b n/m = (a / b) n/m

Eksempel:

33/2 / 23/2 = (3/2)3/2 = 1.53/2 = √(1.53) = √3.375 = 1.837

 

Deling av brøkeksponenter med samme base:

a n/m / a k/j = a (n/m)-(k/j)

Eksempel:

23/2 / 24/3 = 2(3/2)-(4/3) = 2(1/6) = 6√2 = 1.122

 

Dele brøkeksponenter med forskjellige eksponenter og brøker:

a n/m / b k/j

Eksempel:

23/2 / 24/3 = √(23) / 3√(24) = 2.828 / 2.52 = 1.1222

Dele variabler med eksponenter

For eksponenter med samme grunntall kan vi trekke fra eksponentene:

xn / xm = xn-m

Eksempel:

x5 / x3 = (x⋅x⋅x⋅x⋅x) / (x⋅x⋅x) = x5-3 = x2
