Derivat av logaritmen

Når den logaritmiske funksjonen er gitt av:

f (x) = logb(x)

Den deriverte av den logaritmiske funksjonen er gitt av:

f ' (x) = 1 / ( x ln(b) )

x er funksjonsargumentet.

b er logaritmegrunnlaget.

ln b er den naturlige logaritmen til b.

 

For eksempel når:

f (x) = log2(x)

f ' (x) = 1 / ( x ln(2) )
