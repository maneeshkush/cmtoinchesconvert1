Endelig karakterberegning

Eksempel #1

Gjeldende karakter er 70 % (eller C-).

Avsluttende eksamensvekt er 50 %.

Karakterkrav er 80 % (eller B-).

Beregning:

Så den endelige eksamenskarakteren er lik den obligatoriske karakteren, minus 100 % minus den endelige eksamensvekten (w) ganger gjeldende karakter (g), delt på den endelige eksamensvekten (w).

Avsluttende eksamenskarakter =

= ( obligatorisk karakter - (100 % - w)× gjeldende karakter ) / w

= ( 80 % - (100 % - 50 %) × 70 % ) / 50 % = 90 %

Så den endelige eksamenskarakteren skal være 90 % (eller A-).

Eksempel #2

Oppgave 1: vekt1=50 %, karakter1=16 av 20.

Oppgave 2: vekt2=30 %, maks karakter=30.

Oppgave 3: vekt3=20 %, maks karakter=40.

Finn gjennomsnittskarakteren i oppgave 2 og 3 som trengs for å få klassekarakteren på 85 %.

Beregning:

Gjeldende karakter = Oppgave 1 karakter = karakter1 / maks karakter1 = 16/20 = 0,8 = 80 %

Påkrevd karakter = 85 %

Avsluttende eksamensvekt = w = vekt2+vekt3 = 30 %+20 % = 50 %

Avsluttende eksamenskarakter =

= ( obligatorisk karakter - (100 % - w )× gjeldende karakter ) / w

= ( 85 % - 50 % × 80 % ) / 50 % = 90 %

Dette betyr at du må få en snittkarakter på 90 % i oppgave 2 og 3 for å få klassekarakter på 85 %.

Oppgave 2 karakter = 90 % × maks karakter = 90 % × 30 = 27

Oppgave 3 karakter = 90 % × maks karakter = 90 % × 40 = 36
