最终成绩计算

示例#1

当前成绩为 70%（或 C-）。

期末考试权重为50%。

要求的成绩为 80%（或 B-）。

计算：

所以期末考试成绩等于要求的成绩减去期末考试权重（w）乘以当前成绩（g）除以期末考试权重（w）后的100%。

期末考试成绩=

= ( 要求成绩 - (100% - w)×当前成绩) / w

= ( 80% - (100% - 50%)×70% ) / 50% = 90%

所以期末考试成绩应该是 90%（或 A-）。

例子#2

作业 1：权重 1=50%，等级 1=16（满分 20）。

作业 2：权重 2 = 30%，最高等级 = 30。

作业 3：权重 3 = 20%，最高等级 = 40。

求作业 2 和 3 的平均成绩，以获得 85% 的班级成绩。

计算：

当前成绩 = 作业 1 成绩 = 成绩 1 / 最高成绩 1 = 16/20 = 0.8 = 80%

要求的成绩 = 85%

期末考试权重= w=权重2+权重3 =30%+20%=50%

期末考试成绩=

= (要求成绩- (100% - w )×当前成绩) / w

= ( 85% - 50% × 80% ) / 50% = 90%

这意味着您必须在作业 2 和 3 中获得 90% 的平均成绩才能获得 85% 的班级成绩。

作业 2 成绩 = 90% ×最高成绩= 90% × 30 = 27

作业 3 成绩 = 90% ×最高成绩= 90% × 40 = 36
