加权成绩计算

加权等级等于以百分比 (%) 表示的权重 (w) 乘以等级 (g) 的乘积之和：

Weighted grade =  w1×g1+ w2×g2+ w3×g3+...

示例 1

数学课程，成绩为 77，权重为 30%。

生物学课程，成绩为 85，权重为 50%。

历史课程，成绩为 73，权重为 20%。

加权平均成绩计算如下：

Weighted grade =

 =  w1×g1+ w2×g2+ w3×g3

 = (30×77+50×85+20×73) = 80.20

示例 2

数学课程，成绩为 79，权重为 30%。

生物学课程，成绩为 89，权重为 50%。

历史课程，成绩为 76，权重为 20%。

加权平均成绩计算如下：

Weighted grade =

 =  w1×g1+ w2×g2+ w3×g3

 = (30×79+50×89+20×76) = 83.40

 

当权重不是百分比（小时或点......）时，你还应该除以权重的总和：

Weighted grade = (w1×g1+ w2×g2+ w3×g3+...) / (w1+w2+w3+...)

示例 1

3 分数学课程，成绩为 77。

生物学课程 5 分，成绩为 85。

历史课程 2 分，成绩为 73。

加权平均成绩计算如下：

Weighted grade =

 = (w1×g1+ w2×g2+ w3×g3) / (w1+w2+w3)

 = (3×77+ 5×85+ 2×73) / (3+5+2) = 80.20

示例 2

3 分数学课程，成绩为 79。

生物学课程 5 分，成绩为 79。

历史课程 2 分，成绩为 76。

加权平均成绩计算如下：

Weighted grade =

 = (w1×g1+ w2×g2+ w3×g3) / (w1+w2+w3)

 = (3×79+ 5×89+ 2×76) / (3+5+2) = 83.40
