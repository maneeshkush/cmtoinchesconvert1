Linux/Unix 中的 cd 命令

cd是一个 Linux 命令，用于更改终端 shell 的目录/文件夹。

您可以按选项卡按钮以自动完成目录名称。

  • cd语法
  • CD范例

cd语法

$ cd [directory]

cd命令示例

更改到主目录（由 $HOME 环境变量确定）：

$ cd

 

同时更改到主目录：

$ cd ~

 

切换到根目录：

$ cd /

 

更改到父目录：

$ cd ..

 

更改为子目录Documents：

$ cd Documents

 

更改为子目录Documents/Books：

$ cd Documents/Books

 

切换到具有绝对路径/home/user/Desktop 的目录：

$ cd /home/user/Desktop

 

更改为带空格的目录名称 -我的图片：

$ cd My\ Images

或者

$ cd "My Images"

或者

$ cd 'My Images'
