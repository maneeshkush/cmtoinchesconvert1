正弦定义

在直角三角形 ABC 中，α 的正弦，sin(α) 定义为角 α 的对边与直角对边（斜边）的比值：

sin α = a / c

例子

a = 3"

c = 5"

sin α = a / c = 3 / 5 = 0.6

反正弦函数

当-1≤x≤1时，x的反正弦定义为x的反正弦函数。

当 y 的正弦等于 x 时：

sin y = x

那么x的反正弦等于x的反正弦函数，等于y：

arcsin x = sin-1(x) = y
