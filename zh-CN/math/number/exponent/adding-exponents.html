添加指数

如何添加指数。

  • 将数字与指数相加
  • 添加负指数
  • 添加分数指数
  • 添加带有指数的变量

将数字与指数相加

添加指数是通过先计算每个指数然后添加：

an + bm

例子：

42 + 25 = 4⋅4+2⋅2⋅2⋅2⋅2 = 16+32 = 48

添加相同的底数 b 和指数 n：

bn + bn = 2bn

例子：

42 + 42 = 2⋅42 = 2⋅4⋅4 = 32

添加负指数

添加负指数是通过先计算每个指数然后添加：

a-n + b-m = 1/an + 1/bm

例子：

4-2 + 2-5 = 1/42 + 1/25 = 1/(4⋅4)+1/(2⋅2⋅2⋅2⋅2) = 1/16+1/32 = 0.09375

添加分数指数

通过首先提高每个指数然后添加：

an/m + bk/j

例子：

33/2 + 25/2 = √(33) + √(25) = √(27) + √(32) = 5.196 + 5.657 = 10.853

 

添加相同的底数 b 和指数 n/m：

bn/m + bn/m = 2bn/m

例子：

42/3 + 42/3 = 2⋅42/3 = 2 ⋅ 3√(42) = 5.04

添加带有指数的变量

添加指数是通过先计算每个指数然后添加：

xn + xm

具有相同的指数：

xn + xn = 2xn

例子：

x2 + x2 = 2x2
