直流电路规则

欧姆定律

I = V / R

焦耳定律

P = V · I = I 2 · R = V 2 / R

串联电路规则

VT = V1 + V2 + V3 + ...

IT = I1 = I2 = I3 = ...

RT = R1 + R2 + R3 + ...

1/CT = 1/C1 + 1/C2 + 1/C3 + ...

LT = L1 + L2 + L3 + ...

并联电路规则

VT = V1 = V2 = V3 = ...

IT = I1 + I2 + I3 + ...

1/RT = 1/R1 + 1/R2 + 1/R3 + ...

CT = C1 + C2 + C3 + ...

1/LT = 1/L1 + 1/L2 + 1/L3 + ...

分压

V1 = VT ⋅ R1 / (R1+R2+R3+...)

当前部门

I1 = IT ⋅ (R2+R3+...) / (R1+R2+R3+...)

基尔霍夫电压定律 (KVL)

电流回路的压降总和为零：

∑ Vi = 0

基尔霍夫电流定律 (KCL)

几个电路元件之间的连接点称为节点。

节点处的电流值之和为零：

∑ 我我= 0

电容

C = Q / V

平行板电容器

C = ε ⋅ A / l

ε 是以法拉每米 (F/m) 为单位的介电常数。

介电常数

ε = ε0 ⋅ εr

ε 0是真空中的介电常数。

ε r是相对介电常数或介电常数。

电容电流

IC(t) = C d VC(t) / dt

电容电压

V C (t) = V C (0) + 1/ C ∫ I C (t)⋅ dt

电感电压

VL(t) = L d IL(t) / dt

电感电流

I L (t) = I L (0) + 1/ L ∫ V L (t)⋅ dt

电容器的能量

WC = C⋅V 2 / 2

电感能量 

WL = L⋅I 2 / 2
