如何将二进制转换为十进制

所以十进制数等于二进制数字 (d n ) 乘以它们的 2 次幂 (2 n ) 的总和：

decimal = d0×20 + d1×21 + d2×22 + ...

示例#1

求 111001 2的十进制值：

二进制数：1个1个1个001个
2 的幂：2 52 42 32 22 12 0

1110012 = 1⋅25+1⋅24+1⋅23+0⋅22+0⋅21+1⋅20 = 5710

例子#2

求出 100011 2的十进制值：

二进制数：1个0001个1个
2 的幂：2 52 42 32 22 12 0

1000112 = 1⋅25+0⋅24+0⋅23+0⋅22+1⋅21+1⋅20 = 3510
