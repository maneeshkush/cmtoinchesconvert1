如何将小数转换为分数

转换阶段

  1. 将小数部分写为小数点（分子）和 10 的幂（分母）右边数字的分数。
  2. 找出分子和分母的最大公约数 (gcd)。
  3. 通过将分子和分母除以 gcd 来减少分数。

示例#1

将 0.35 转换为分数：

0.35 = 35/100

所以找到分子和分母的最大公约数 (gcd)：

gcd(35,100) = 5

因此，通过将分子和分母除以 gcd 来减少分数：

0.35 = (35/5) / (100/5) = 7/20

例子#2

将 2.58 转换为分数：

2.58 = 2+58/100

所以找到分子和分母的最大公约数 (gcd)：

gcd(58,100) = 2

因此，通过将分子和分母除以 gcd 来减少分数：

2+58/100 = 2 + (58/2) / (100/2) = 2+29/50

例子#3

将 0.126 转换为分数：

0.126 = 126/1000

求分子和分母的最大公约数 (gcd)：

gcd(126,1000) = 2

通过将分子和分母除以 gcd 来减少分数：

0.126 = (126/2)/(1000/2) = 63/500

如何将循环小数化为分数

示例#1

将 0.333333... 转换为分数：

x = 0.333333...

10x = 3.333333...

10x - x = 9x = 3

x = 3/9 = 1/3

例子#2

将 0.0565656... 转换为分数：

x = 0.0565656...

100 x = 5.6565656...

100 x -  x = 99 x = 5.6

990 x = 56

x = 56/990 = 28/495
