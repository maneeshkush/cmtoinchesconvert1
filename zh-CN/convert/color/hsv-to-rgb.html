HSV转RGB公式

当 0 ≤ H < 360、0 ≤ S ≤ 1 且 0 ≤ V ≤ 1 时：

C = V × S

X = C × (1 - |(H / 60°) mod 2 - 1|)

m = V - C

(R,G,B) = ((R'+m)×255, (G'+m)×255, (B'+m)×255)

2. RGB转HSV颜色

RGB 到 HSV 颜色转换是将 RGB 颜色空间中指定的颜色转换为 HSV 颜色空间中的颜色的过程。HSV 颜色空间是一个三维颜色空间，它根据色调、饱和度和明度来指定颜色。色相是光的颜色，饱和度是颜色的强度，明度是颜色的亮度。

RGB 到 HSV 颜色转换算法将 RGB 颜色作为输入并将其转换为 HSV 颜色。该算法首先将 RGB 颜色转换为 24 位十六进制颜色。然后它将十六进制颜色分成三种 6 位颜色，红色、绿色和蓝色。然后它将红色、绿色和蓝色转换为 HSV 颜色。该算法然后重新组合三种 HSV 颜色以创建最终的 HSV 颜色。
