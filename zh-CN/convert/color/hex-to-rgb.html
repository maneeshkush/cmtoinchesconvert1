十六进制到 RGB 转换

  1. 获取十六进制颜色代码的左 2 位数字并转换为十进制值以获得红色级别。
  2. 获取十六进制颜色代码的中间 2 位数字并转换为十进制值以获得绿色级别。
  3. 获取十六进制颜色代码的右2位并转换为十进制值以获得蓝色级别。

示例#1

将红色十六进制颜色代码 FF0000 转换为 RGB 颜色：

Hex = FF0000

所以RGB颜色是：

R = FF16 = 25510

G = 0016 = 010

B = 0016 = 010

或者

RGB = (255, 0, 0)

例子#2

将金色十六进制颜色代码 FFD700 转换为 RGB 颜色：

Hex = FFD700

所以RGB颜色是：

R = FF16 = 25510

G = D716 = 21510

B = 0016 = 010

或者

RGB = (255, 215, 0)
