使用 Javascript 进行屏幕分辨率检测

您可以使用screen.width和screen.height属性获取屏幕尺寸。

您可以使用screen.availWidth和screen.availHeight属性获取可用的屏幕尺寸。

例如：

<script>
alert(screen.width+' x '+screen.height);
</script>
