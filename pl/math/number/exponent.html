Co to jest wykładnik

Podstawa a podniesiona do potęgi n jest równa iloczynowi a, n razy:

a n = a × a × ... × a

                    n razy

a to podstawa, a n to wykładnik.

Przykłady

31 = 3

32 = 3 × 3 = 9

33 = 3 × 3 × 3 = 27

34 = 3 × 3 × 3 × 3 = 81

35 = 3 × 3 × 3 × 3 × 3 = 243

Reguły iloczynu wykładników

Reguła produktu z tą samą podstawą

an ⋅ am = an+m

Przykład:

23 ⋅ 24 = 23+4 = 27 = 2⋅2⋅2⋅2⋅2⋅2⋅2 = 128

Reguła iloczynu z tym samym wykładnikiem

an ⋅ bn = (a ⋅ b)n

Przykład:

32 ⋅ 42 = (3⋅4)2 = 122 = 12⋅12 = 144

Reguły ilorazu wykładników

Reguła ilorazu o tej samej podstawie

an / am = an-m

Przykład:

25 / 23 = 25-3 = 22 = 2⋅2 = 4

Reguła ilorazu z tym samym wykładnikiem

an / bn = (a / b)n

Przykład:

43 / 23 = (4/2)3 = 23 = 2⋅2⋅2 = 8

Reguły potęgowe wykładników

Zasada władzy I

(an) m = a n⋅m

Przykład:

(23)2 = 23⋅2 = 26 = 2⋅2⋅2⋅2⋅2⋅2 = 64

Reguła władzy II

a nm = a (nm)

Przykład:

232 = 2(32) = 2(3⋅3) = 29 = 2⋅2⋅2⋅2⋅2⋅2⋅2⋅2⋅2 = 512

Rządy władzy z radykałami

m√(a n) = a n/m

Przykład:

2√(26) = 26/2 = 23 = 2⋅2⋅2 = 8

Reguła wykładników ujemnych

b-n = 1 / bn

Przykład:

2-3 = 1/23 = 1/(2⋅2⋅2) = 1/8 = 0.125
