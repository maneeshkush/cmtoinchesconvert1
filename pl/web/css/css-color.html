Kolor CSSa

Kod koloru może być jednym z:

Format szesnastkowy: #rrggbb

Format RGB: rgb (czerwony, zielony, niebieski)

Format nazwy: imię

Przykład

Dla koloru pomarańczowego:

Format szesnastkowy: #FFA500

Format RGB: RGB(255,165,0)

Format nazwy: pomarańczowy

Ustawienie koloru elementu

Dla określonego elementu:

<element style="color: code;"></element>

Dla wszystkich elementów tego samego typu.Umieść kod w tagu <style> w sekcji head:

<style>
   element { color: code; }
</style>

Przykład

Ustawienie koloru tekstu akapitu na czerwony:

Ustawienie koloru określonego akapitu:

<p style="color: #FF0000;">Some text ...</p>

Pogląd:

Jakiś tekst ...

Ustawienie koloru wszystkich akapitów

<style>
   p { color: #FF0000; }
</style>

Ustawienie koloru tła elementu

element { background: code; }

Przykład

Ustawienie koloru tła akapitu na czerwony:

Ustawienie koloru tła określonego akapitu:

<p style="background: #FF0000;">Some text ...</p>

Pogląd:

Jakiś tekst ...

Ustawienie koloru tła wszystkich akapitów:

<style>
   p { background: #FF0000; }
</style>

Ustawienie koloru obramowania elementu

element { border-color: topcode rightcode bottomcode leftcode }

Przykład

Ustawienie koloru obramowania akapitu na czerwony, zielony, niebieski i czarny:

Ustawianie koloru obramowania określonego akapitu:

<p style="border-color: #FF0000 #00FF00 #0000FF #000000; border-style:solid">Some text ...</p>

Pogląd:

Jakiś tekst ...

Ustawianie koloru obramowania wszystkich akapitów:

<style>
   p { border-color: #FF0000 #00FF00 #0000FF #000000; }
</style>
