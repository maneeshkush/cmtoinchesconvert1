ls polecenie w systemie Linux/Unix

ls to polecenie powłoki systemu Linux, które wyświetla zawartość katalogów plików i katalogów.

  • Składnia ls
  • ls opcje
  • l przykłady
  • ls generator kodu

Składnia ls

$ ls [options] [file|dir]

ls opcje poleceń

Główne opcje polecenia ls:

opcja opis
ls-a lista wszystkich plików, w tym plików ukrytych zaczynających się od „.”
ls --kolor kolorowa lista [=zawsze/nigdy/auto]
ls -d lista katalogów - z „*/”
ls-F dodaj jeden znak */=>@|do wpisów
ls -i lista numer indeksu i-węzła pliku
ls -l lista z długim formatem - pokaż uprawnienia
ls-la długi format listy, w tym ukryte pliki
ls-lh długi format listy z czytelnym rozmiarem pliku
ls-ls lista o długim formacie z rozmiarem pliku
ls -r listę w odwrotnej kolejności
ls-R wyświetla rekurencyjnie drzewo katalogów
ls -s rozmiar pliku listy
ls-S sortować według rozmiaru pliku
ls-t sortuj według czasu i daty
ls-X sortuj według nazwy rozszerzenia

ls przykłady poleceń

Możesz nacisnąć przycisk tabulatora , aby automatycznie uzupełnić nazwy plików lub folderów.

Lista katalogów Dokumenty/Książki ześcieżką względną :

$ ls Documents/Books

 

Wyświetl katalog /home/user/Documents/Books ześcieżką bezwzględną .

$ ls /home/user/Documents/Books

 

Katalog główny listy:

$ ls /

 

Wyświetl katalog nadrzędny:

$ ls ..

 

Wyświetl katalog domowy użytkownika (np.: /home/user):

$ ls ~

 

Lista w długim formacie:

$ ls -l

 

Pokaż ukryte pliki:

$ ls -a

 

Lista z długim formatem i pokaż ukryte pliki:

$ ls -la

 

Sortuj według daty/godziny:

$ ls -t

 

Sortuj według rozmiaru pliku:

$ ls -S

 

Wyświetl listę wszystkich podkatalogów:

$ ls *

 

Rekurencyjna lista drzewa katalogów:

$ ls -R

 

Wyświetlaj tylko pliki tekstowe z symbolami wieloznacznymi:

$ ls *.txt

 

ls przekierowanie do pliku wyjściowego:

$ ls > out.txt

 

Wyświetl tylko katalogi:

$ ls -d */

 

Wyświetl listę plików i katalogów z pełną ścieżką:

$ ls -d $PWD/*
