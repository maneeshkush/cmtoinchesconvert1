ఘాతాంకాలను జోడిస్తోంది

ఘాతాంకాలను ఎలా జోడించాలి.

  • ఘాతాంకాలతో సంఖ్యలను కలుపుతోంది
  • ప్రతికూల ఘాతాంకాలను కలుపుతోంది
  • పాక్షిక ఘాతాంకాలను కలుపుతోంది
  • ఘాతాంకాలతో వేరియబుల్స్ జోడించడం

ఘాతాంకాలతో సంఖ్యలను కలుపుతోంది

ముందుగా ప్రతి ఘాతాంకాన్ని లెక్కించి, ఆపై జోడించడం ద్వారా ఘాతాంకాలను జోడించడం జరుగుతుంది:

an + bm

ఉదాహరణ:

42 + 25 = 4⋅4+2⋅2⋅2⋅2⋅2 = 16+32 = 48

అదే స్థావరాలు b మరియు ఘాతాంకాలు n జోడించడం:

bn + bn = 2bn

ఉదాహరణ:

42 + 42 = 2⋅42 = 2⋅4⋅4 = 32

ప్రతికూల ఘాతాంకాలను కలుపుతోంది

ప్రతికూల ఘాతాంకాలను జోడించడం ప్రతి ఘాతాంకాన్ని ముందుగా లెక్కించి, ఆపై జోడించడం ద్వారా జరుగుతుంది:

a-n + b-m = 1/an + 1/bm

ఉదాహరణ:

4-2 + 2-5 = 1/42 + 1/25 = 1/(4⋅4)+1/(2⋅2⋅2⋅2⋅2) = 1/16+1/32 = 0.09375

పాక్షిక ఘాతాంకాలను కలుపుతోంది

పాక్షిక ఘాతాంకాలను జోడించడం అనేది ముందుగా ప్రతి ఘాతాంకాన్ని పెంచడం ద్వారా మరియు తర్వాత జోడించడం ద్వారా జరుగుతుంది:

an/m + bk/j

ఉదాహరణ:

33/2 + 25/2 = √(33) + √(25) = √(27) + √(32) = 5.196 + 5.657 = 10.853

 

అదే స్థావరాలు b మరియు ఘాతాంకాలు n/m జోడించడం:

bn/m + bn/m = 2bn/m

ఉదాహరణ:

42/3 + 42/3 = 2⋅42/3 = 2 ⋅ 3√(42) = 5.04

ఘాతాంకాలతో వేరియబుల్స్ జోడించడం

ముందుగా ప్రతి ఘాతాంకాన్ని లెక్కించి, ఆపై జోడించడం ద్వారా ఘాతాంకాలను జోడించడం జరుగుతుంది:

xn + xm

అదే ఘాతాంకాలతో:

xn + xn = 2xn

ఉదాహరణ:

x2 + x2 = 2x2
