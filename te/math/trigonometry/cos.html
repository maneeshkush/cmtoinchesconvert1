కొసైన్ నిర్వచనం

లంబ త్రిభుజం ABCలో α, sin(α) యొక్క సైన్ α కోణానికి ప్రక్కనే ఉన్న వైపు మరియు లంబ కోణానికి ఎదురుగా ఉన్న వైపు (హైపోటెన్యూస్) మధ్య నిష్పత్తిగా నిర్వచించబడింది:

cos α = b / c

ఉదాహరణ

b = 3"

c = 5"

cos α = b / c = 3 / 5 = 0.6

విలోమ కొసైన్ ఫంక్షన్

x యొక్క ఆర్కోసిన్ -1≤x≤1ఉన్నప్పుడు x యొక్క విలోమ కొసైన్ ఫంక్షన్‌గా నిర్వచించబడింది.

y యొక్క కొసైన్ xకి సమానంగా ఉన్నప్పుడు:

cos y = x

అప్పుడు x యొక్క ఆర్కోసిన్ x యొక్క విలోమ కొసైన్ ఫంక్షన్‌కి సమానం, ఇది yకి సమానం:

arccos x = cos-1 x = y

ఉదాహరణ

arccos 1 = cos-1 1 = 0 rad = 0°
