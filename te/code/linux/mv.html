Linux/Unixలో mv కమాండ్

Linux mv కమాండ్.

mv కమాండ్ ఫైల్స్ మరియు డైరెక్టరీలను తరలించడానికి ఉపయోగించబడుతుంది.

mv కమాండ్ సింటాక్స్

$ mv [options] source dest

mv కమాండ్ ఎంపికలు

mv కమాండ్ ప్రధాన ఎంపికలు:

ఎంపిక వివరణ
mv -f ప్రాంప్ట్ లేకుండా గమ్యం ఫైల్‌ని ఓవర్‌రైట్ చేయడం ద్వారా బలవంతంగా తరలించండి
mv -i ఓవర్‌రైట్ చేయడానికి ముందు ఇంటరాక్టివ్ ప్రాంప్ట్
mv -u నవీకరణ - గమ్యస్థానం కంటే మూలం కొత్తది అయినప్పుడు తరలించండి
mv -v వెర్బోస్ - ప్రింట్ సోర్స్ మరియు డెస్టినేషన్ ఫైల్స్
మనిషి mv సహాయం మాన్యువల్

mv కమాండ్ ఉదాహరణలు

main.c def.h ఫైల్‌లను /home/usr/rapid/ డైరెక్టరీకి తరలించండి:

$ mv main.c def.h /home/usr/rapid/

 

ప్రస్తుత డైరెక్టరీలోని అన్ని C ఫైల్‌లను సబ్ డైరెక్టరీ bakకి తరలించండి:

$ mv *.c bak

 

సబ్ డైరెక్టరీ బాక్‌లోని అన్ని ఫైల్‌లను ప్రస్తుత డైరెక్టరీకి తరలించండి:

$ mv bak/* .

 

ఫైల్‌ని main.cని main.bakగా మార్చండి:

$ mv main.c main.bak

 

డైరెక్టరీని bak అని bak2గా మార్చండి:

$ mv bak bak2

 

నవీకరణ - main.c కొత్తది అయినప్పుడు తరలించండి:

$ mv -u main.c bak
$

 

bak/main.cని ఓవర్‌రైట్ చేయడానికి ముందు main.cని తరలించి ప్రాంప్ట్ చేయండి:

$ mv -v main.c bak
'bak/main.c' -> 'bak/main.c'
$
