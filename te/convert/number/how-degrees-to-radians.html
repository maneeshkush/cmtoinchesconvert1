డిగ్రీలను రేడియన్‌లుగా మార్చడం ఎలా

డిగ్రీలు రేడియన్స్ మార్పిడి సూత్రం

ఒక డిగ్రీ 0.01745329252 రేడియన్‌లకు సమానం:

1° = π/180° = 0.005555556π = 0.01745329252 rad

కాబట్టి రేడియన్‌లలోని కోణం α డిగ్రీల సమయాలలో α కోణానికి సమానం pi స్థిరాంకం 180 డిగ్రీలతో విభజించబడింది:

α(radians) = α(degrees) × π / 180°

లేదా

radians = degrees × π / 180°

ఉదాహరణ 1

25 డిగ్రీల కోణాన్ని రేడియన్‌లుగా మార్చండి:

α(radians) = α(degrees) × π / 180° = 25° × 3.14159 / 180° = 0.436332313 rad

α(radians) = α(degrees) × π / 180° = 90° × 3.14159 / 180° = 1.5707963268 rad

pi పరంగా రేడియన్‌లకు డిగ్రీలు

కాబట్టి రేడియన్‌లలోని కోణం α డిగ్రీల సమయాలలో α కోణానికి సమానం, pi స్థిరాంకం [180] డిగ్రీలతో భాగించబడుతుంది.

α(radians) = (α(degrees) / 180°) × π

α(radians) = (α(degrees) / 180°) × π = (40° / 180°) × π

 = (1/6) × π = π/6 rad = 0.22222222222π rad
