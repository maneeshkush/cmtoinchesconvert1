జావాస్క్రిప్ట్ దారిమార్పు URL

జావాస్క్రిప్ట్‌తో URL పేజీని ఎలా దారి మళ్లించాలి.

Javascript దారి మళ్లింపు 301 శాశ్వత దారి మళ్లింపు స్థితి కోడ్‌ని అందించదు.

జావ్‌స్క్రిప్ట్ దారిమార్పు

మీరు దారి మళ్లించాలనుకుంటున్న పేజీ యొక్క URLతో పాత పేజీని దారి మళ్లింపు కోడ్‌తో భర్తీ చేయండి.

old-page.html:

<!DOCTYPE html>
<html>
<body>
<script type="text/javascript">
    // Javascript URL redirection
    window.location.replace("http://www.mydomain.com/new-page.html");
</script>
</body>
</html>

 

పాత URL నుండి కొత్త URLకి పేజీ ర్యాంక్‌ని బదిలీ చేయడానికి శోధన ఇంజిన్‌లు 301 స్థితి కోడ్‌ని ఉపయోగిస్తాయి.

Javascript దారి మళ్లింపు తిరిగి http ప్రతిస్పందన స్థితి కోడ్: 200 సరే.

కాబట్టి జావాస్క్రిప్ట్ మళ్లింపు శోధన ఇంజిన్ అనుకూలమైనది కాదు మరియు స్థితి కోడ్‌ని అందించే ఇతర దారి మళ్లింపు పద్ధతులను ఉపయోగించడం ఉత్తమం: 301 శాశ్వతంగా తరలించబడింది.
