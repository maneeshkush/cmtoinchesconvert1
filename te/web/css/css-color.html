ఉదాహరణ

నారింజ రంగు కోసం:

హెక్స్ ఫార్మాట్: #FFA500

RGB ఫార్మాట్: rgb(255,165,0)

పేరు ఫార్మాట్: నారింజ

మూలకం రంగును సెట్ చేస్తోంది

నిర్దిష్ట మూలకం కోసం:

<element style="color: code;"></element>

ఒకే రకమైన అన్ని అంశాల కోసం.హెడ్ ​​సెక్షన్‌లోని <style> ట్యాగ్‌లో కోడ్‌ను ఉంచండి:

<style>
   element { color: code; }
</style>
