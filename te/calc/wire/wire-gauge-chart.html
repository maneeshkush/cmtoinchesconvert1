వైర్ వ్యాసం లెక్కలు

n గేజ్ వైర్ వ్యాసం d n inches (in) 0.005in సార్లు 92కి సమానం 36 మైనస్ గేజ్ సంఖ్య n శక్తికి పెంచబడింది, 39 ద్వారా విభజించబడింది:

dn (in) = 0.005 in × 92(36-n)/39

n గేజ్ వైర్ వ్యాసం d n మిల్లీమీటర్లలో (mm) 0.127mm రెట్లు 92కి సమానం 36 మైనస్ గేజ్ సంఖ్య n శక్తికి పెంచబడింది, 39 ద్వారా విభజించబడింది:

dn (mm) = 0.127 mm × 92(36-n)/39

వైర్ క్రాస్ సెక్షనల్ ఏరియా లెక్కలు

కిలో-వృత్తాకార మిల్స్ (kcmil)లో n గేజ్ వైర్ యొక్క క్రాస్ సెర్షినల్ ప్రాంతం A n అంగుళాలలో (ఇన్) చదరపు వైర్ వ్యాసం d కంటే 1000 రెట్లు సమానం:

An (kcmil) = 1000×dn2 = 0.025 in2 × 92(36-n)/19.5

కాబట్టి n గేజ్ వైర్ యొక్క క్రాస్ సెర్షినల్ ఏరియా A n చదరపు అంగుళాలలో ( 2 లో ) pi కి సమానం 4 సార్లు చదరపు వైర్ వ్యాసం d అంగుళాలలో (in) భాగించబడుతుంది.

An (in2) = (π/4)×dn2 = 0.000019635 in2 × 92(36-n)/19.5

కాబట్టి n గేజ్ వైర్ యొక్క క్రాస్ సెర్షినల్ ఏరియా A n స్క్వేర్ మిల్లీమీటర్లలో (mm 2 ) pi కి సమానం 4 సార్లు చదరపు వైర్ వ్యాసం d మిల్లీమీటర్లలో (mm) భాగించబడుతుంది.

An (mm2) = (π/4)×dn2 = 0.012668 mm2 × 92(36-n)/19.5

వైర్ రెసిస్టెన్స్ లెక్కలు

కాబట్టి n గేజ్ వైర్ రెసిస్టెన్స్ R కిలోఫీట్‌కి ఓంలు (Ω/kft) 0.3048×1000000000 రెట్లు వైర్ రెసిస్టివిటీ ρ ఓమ్-మీటర్‌లలో ( Ω·m) 25.4 2 రెట్లు క్రాస్ సెక్షనల్ ప్రాంతం A n చదరపు అంగుళాలలో భాగించబడుతుంది.( 2 లో ).

Rn (Ω/kft) = 0.3048 × 109 × ρ(Ω·m) / (25.42 × An (in2))

కాబట్టి n గేజ్ వైర్ రెసిస్టెన్స్ R ప్రతి కిలోమీటర్ (Ω/km)లో 1000000000 రెట్లు వైర్ రెసిస్టివిటీకి సమానం ρ ఓం-మీటర్లలో (Ω·m) క్రాస్ సెక్షనల్ ప్రాంతం A n ద్వారా చదరపు మిల్లీమీటర్‌లలో (mm 2 ) భాగించబడుతుంది.

Rn (Ω/km) = 109 × ρ(Ω·m) / An (mm2)

మా వైర్ గేజ్ సైజు కాలిక్యులేటర్ వినియోగదారులను వైర్ గేజ్ పరిమాణాన్ని లెక్కించడానికి అనుమతిస్తుంది.ఈ యుటిలిటీ యొక్క కొన్ని ప్రముఖ లక్షణాలు క్రింద వివరించబడ్డాయి.

నమోదు లేదు

వైర్ గేజ్ సైజు కాలిక్యులేటర్‌ని ఉపయోగించడానికి మీరు ఎలాంటి రిజిస్ట్రేషన్ ప్రక్రియను నిర్వహించాల్సిన అవసరం లేదు.ఈ యుటిలిటీని ఉపయోగించి, వినియోగదారులు మీకు కావలసినన్ని సార్లు వైర్ గేజ్ పరిమాణాన్ని ఉచితంగా లెక్కించవచ్చు.

వేగవంతమైన మార్పిడి

ఈ వైర్ గేజ్ సైజు కాలిక్యులేటర్ వినియోగదారులకు వేగవంతమైన గణనను అందిస్తుంది.వినియోగదారు ఇన్‌పుట్ ఫీల్డ్‌లో వైర్ గేజ్ సైజు విలువలను నమోదు చేసి, లెక్కించు బటన్‌ను క్లిక్ చేసిన తర్వాత, యుటిలిటీ మార్పిడి ప్రక్రియను ప్రారంభించి, ఫలితాలను వెంటనే అందిస్తుంది.

సమయం మరియు కృషిని ఆదా చేస్తుంది

కాలిక్యులేటర్ వైర్ గేజ్ సైజు యొక్క మాన్యువల్ విధానం అంత తేలికైన పని కాదు.ఈ పనిని పూర్తి చేయడానికి మీరు చాలా సమయం మరియు కృషిని వెచ్చించాలి.వైర్ గేజ్ సైజు కాలిక్యులేటర్ అదే పనిని వెంటనే పూర్తి చేయడానికి మిమ్మల్ని అనుమతిస్తుంది.మాన్యువల్ విధానాలను అనుసరించమని మిమ్మల్ని అడగరు, ఎందుకంటే దాని స్వయంచాలక అల్గారిథమ్‌లు మీ కోసం పని చేస్తాయి.

ఖచ్చితత్వం

మాన్యువల్ కాలిక్యులేషన్‌లో సమయం మరియు కృషిని పెట్టుబడి పెట్టినప్పటికీ, మీరు ఖచ్చితమైన ఫలితాలను పొందలేరు.గణిత సమస్యలను పరిష్కరించడంలో ప్రతి ఒక్కరూ నిష్ణాతులు కాదు, మీరు అనుకూలమని భావించినప్పటికీ, మీరు ఖచ్చితమైన ఫలితాలను పొందే మంచి అవకాశం ఉంది.వైర్ గేజ్ సైజు కాలిక్యులేటర్ సహాయంతో ఈ పరిస్థితిని తెలివిగా నిర్వహించవచ్చు.ఈ ఆన్‌లైన్ సాధనం ద్వారా మీకు 100% ఖచ్చితమైన ఫలితాలు అందించబడతాయి.
