How to convert ppb to ppm

ppm and ppb are defined as:

1ppm = 1/106 = 10-6

1ppb = 1/109 = 10-9

So

1ppm = 1000ppb

So The number of parts-per million xppm is equal to the number of parts-per billion xppb divided by [1000].

xppm = xppb / 1000

 

Example 1

4000ppb is equal to 4ppm:

xppm = 4000ppb / 1000 = 4ppm

Example 2

5000ppb is equal to 5ppm:

xppm = 5000ppb / 1000 = 5ppm

Example 3

6000ppb is equal to 6ppm:

xppm = 6000ppb / 1000 = 6ppm

Example 4

8000ppb is equal to 8ppm:

xppm = 8000ppb / 1000 = 8ppm
