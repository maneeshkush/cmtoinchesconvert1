JavaScript yönlendirme URL'si

Javascript ile URL sayfası nasıl yönlendirilir.

Javascript yönlendirmesi, 301 kalıcı yönlendirme durum kodunu döndürmez.

JavaScript yönlendirmesi

Eski sayfayı yönlendirme koduyla, yönlendirmek istediğiniz sayfanın URL'siyle değiştirin.

eski sayfa.html:

<!DOCTYPE html>
<html>
<body>
<script type="text/javascript">
    // Javascript URL redirection
    window.location.replace("http://www.mydomain.com/new-page.html");
</script>
</body>
</html>

 

Arama motorları, sayfa sıralamasını eski URL'den yeni URL'ye aktarmak için 301 durum kodunu kullanır.

Javascript yeniden yönlendirme http yanıtı durum kodu döndürür: 200 Tamam.

Bu nedenle, Javascript yönlendirmesi arama motoru dostu değildir ve şu durum kodunu döndüren diğer yönlendirme yöntemlerini kullanmak daha iyidir: 301 Kalıcı Olarak Taşındı.

<script type="text/javascript">
