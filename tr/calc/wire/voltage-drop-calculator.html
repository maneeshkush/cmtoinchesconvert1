Gerilim düşümü hesaplamaları

DC / tek fazlı hesaplama

Volt (V) cinsinden voltaj düşüşü V, amper (A) cinsinden tel akımı I çarpı 2 kez fit (ft) cinsinden tek yönlü kablo uzunluğu L çarpı 1000 fit başına kablo direncinin ohm (Ω/kft) cinsinden bölünmesine eşittir 1000'e kadar:

Vdrop (V) = Iwire (A) × Rwire(Ω)

= I kablosu (A) × (2 × L (ft) × R kablosu (Ω/kft) / 1000 (ft/kft) )

Volt (V) cinsinden voltaj düşüşü V, amper (A) cinsinden tel akımı I çarpı metre (m) cinsinden 2 kez tek yönlü kablo uzunluğu L çarpı 1000 metrede kablo direncinin ohm cinsinden R (Ω/km) bölünmesine eşittir 1000'e kadar:

Vdrop (V) = Iwire (A) × Rwire(Ω)

= I kablosu (A) × (2 × L (m) × R kablosu (Ω/km) / 1000 (m/km) )

3 fazlı hesaplama

Volt (V) cinsinden hatlar arası voltaj düşüşü V, amper (A) cinsinden tel akımı I'in 3 katının kareköküne eşittir. (Ω/kft) bölü 1000:

Vdrop (V) = √3 × Iwire (A) × Rwire (Ω)

= 1,732 × I kablosu (A) × ( L (ft) × R kablosu (Ω/kft) / 1000 (ft/kft) )

Volt (V) cinsinden hatlar arası voltaj düşüşü V, amper (A) cinsinden tel akımı I'in 3 katı kareköküne eşittir. (Ω/km) bölü 1000:

Vdrop (V) = √3 × Iwire (A) × Rwire (Ω)

= 1,732 × I kablosu (A) × ( L (m) × R kablosu (Ω/km) / 1000 (m/km) )

Tel çapı hesaplamaları

İnç (in) cinsinden n ayar tel çapı d n , 0,005 inç çarpı 92'nin 36 eksi gösterge sayısı n'nin 39'a bölünmesine eşittir.

dn (in) = 0.005 in × 92(36-n)/39

Milimetre (mm) cinsinden n ayar tel çapı d n , 0,127 mm çarpı 92'nin 36 eksi gösterge sayısı n bölü 39'a yükseltilmiş halidir.

dn (mm) = 0.127 mm × 92(36-n)/39

Tel kesit alanı hesaplamaları

Kilo-dairesel mil (kcmil) cinsinden n çaplı telin kesit alanı A n , inç (inç) cinsinden kare tel çapı d'nin 1000 katına eşittir:

An (kcmil) = 1000×dn2 = 0.025 in2 × 92(36-n)/19.5

Yani n ölçülü telin inç kare ( 2'de ) olarak enine kesit alanı An, pi bölü inç (inç) cinsinden kare tel çapı d'nin 4 katına eşittir.

An (in2) = (π/4)×dn2 = 0.000019635 in2 × 92(36-n)/19.5

Yani n ayarlı telin milimetre kare (mm2 ) cinsindenenine kesit alanı An , pi bölü milimetre ( mm) cinsinden kare tel çapı d'nin 4 katına eşittir.

An (mm2) = (π/4)×dn2 = 0.012668 mm2 × 92(36-n)/19.5

Tel direnci hesaplamaları

Yani, kilofeet başına ohm (Ω/kft) cinsinden n ölçü tel direnci R, 0,3048×1000000000 çarpı ohm-metre (Ω·m) cinsinden telin özdirencinin ρ bölü 25,4 2 çarpıinç kare cinsinden A n'ye eşittir. (2'de ) :

Rn (Ω/kft) = 0.3048 × 109 × ρ(Ω·m) / (25.42 × An (in2))

Dolayısıyla, kilometre başına ohm (Ω/km) cinsinden n ölçü tel direnci R, telin ohm-metre (Ω·m) cinsinden özdirencinin ρ 1000000000 katının milimetre kare (mm 2 ) cinsinden A n kesit alanına bölünmesine eşittir.

Rn (Ω/km) = 109 × ρ(Ω·m) / An (mm2)
