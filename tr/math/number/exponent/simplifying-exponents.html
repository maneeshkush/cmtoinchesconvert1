Rasyonel üsleri basitleştirme

n/m'nin gücüne yükseltilen b tabanı şuna eşittir:

bn/m = (m√b)n = m√(bn)

Örnek:

3/2'nin gücüne yükseltilen 2 tabanı, 1'in 3'e yükseltilmiş taban 2'ye bölünmesine eşittir:

23/2 = 2√(23) = 2.828

Üslü kesirleri basitleştirme

Üslü kesirler:

(a / b)n = an / bn

Örnek:

(4/3)3 = 43 / 33 = 64 / 27 = 2.37

Negatif üsleri basitleştirme

Eksi n'nin gücüne yükseltilen b tabanı, 1 bölü taban b'nin n gücüne yükseltilmesine eşittir:

b-n = 1 / bn

Örnek:

Eksi 3'ün gücüne yükseltilmiş 2 tabanı, 1'in 3'ün gücüne yükseltilmiş 2 tabanına bölünmesine eşittir:

2-3 = 1/23 = 1/(2⋅2⋅2) = 1/8 = 0.125

Üstellerle radikalleri basitleştirme

Üslü radikal için:

(m√a)n = an/m

Örnek:

(√5)4 = 54/2 = 52 = 25
