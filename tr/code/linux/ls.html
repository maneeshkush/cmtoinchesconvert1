Linux/Unix'te ls komutu

ls , dosya ve dizinlerin dizin içeriklerini listeleyen bir Linux kabuk komutudur.

  • ls sözdizimi
  • seçenekler
  • örnekler
  • ls kod üreteci

ls sözdizimi

$ ls [options] [file|dir]

ls komut seçenekleri

ls komutu ana seçenekler:

seçenek Tanım
ls-a ile başlayan gizli dosya dahil tüm dosyaları listeler.
ls --renk renkli liste [=her zaman/asla/otomatik]
ls-d dizinleri listele - ' */' ile
ls -F bir karakter */=>@| ekleyingirişlere
ls -i liste dosyasının inode dizin numarası
ls-l uzun biçimli liste - izinleri göster
ls-la gizli dosyalar dahil olmak üzere uzun formatı listele
ls-lh okunabilir dosya boyutunda uzun formatı listele
ls-ls dosya boyutu ile uzun biçimli liste
ls-r ters sırada listelemek
ls -R özyinelemeli dizin ağacını listele
ls-s dosya boyutunu listele
ls -S dosya boyutuna göre sırala
ls-t saat ve tarihe göre sırala
ls -X uzantı adına göre sırala

ls komut örnekleri

Dosya veya klasör adlarını otomatik olarak tamamlamak için sekme düğmesine basabilirsiniz .

Göreli yolu olan Belgeleri/Kitapları listele:

$ ls Documents/Books

 

/home/user/Documents/Books dizinini mutlak yolile listeleyin.

$ ls /home/user/Documents/Books

 

Liste kök dizini:

$ ls /

 

Üst dizini listele:

$ ls ..

 

Kullanıcının ev dizinini listeleyin (örn: /home/user):

$ ls ~

 

Uzun biçimli liste:

$ ls -l

 

Gizli dosyaları göster:

$ ls -a

 

Uzun biçimli listele ve gizli dosyaları göster:

$ ls -la

 

Tarihe/saate göre sırala:

$ ls -t

 

Dosya boyutuna göre sırala:

$ ls -S

 

Tüm alt dizinleri listele:

$ ls *

 

Özyinelemeli dizin ağacı listesi:

$ ls -R

 

Yalnızca joker karakter içeren metin dosyalarını listele:

$ ls *.txt

 

ls çıktı dosyasına yönlendirme:

$ ls > out.txt

 

Yalnızca dizinleri listele:

$ ls -d */

 

Dosyaları ve dizinleri tam yol ile listeleyin:

$ ls -d $PWD/*
