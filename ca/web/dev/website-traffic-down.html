Trànsit del lloc web baixa

Per què el trànsit del meu lloc web baixa?

Consulta el calendari

Els dies festius i els caps de setmana poden reduir el trànsit.

El trànsit tornarà a la normalitat quan acabi la festa.

Compara amb l'any passat

Utilitzeu Google Analytics per mostrar el gràfic de visites de l'any passat.

Comproveu si les visites també van baixar fa un any.

Error de Google Analytics

Si feu servir un codi antic de Google Analytics amb el fitxer urchin.js, és possible que els dos dies recents mostrin menys trànsit que el trànsit real.

El trànsit no ha baixat realment, però sembla que només ha baixat.

Problema del servidor

Intenta navegar pel teu lloc web, si no hi pots accedir, tens un problema amb el servidor web o el servidor DNS.

Intenteu accedir al vostre servidor web i comproveu si està actiu.

Comproveu la integritat de la vostra base de dades o fitxers html.

Utilitzeu l'eina de prova de ping per comprovar la resposta del vostre servidor web.

Cerca un problema nou al servidor DNS. El 9/2012, aquest lloc web amb molts altres no va poder respondre (vegeu: GoDaddy piratejat ).

La classificació dels resultats de la cerca de Google ha baixat

La majoria del trànsit dels llocs web prové dels motors de cerca i el motor de cerca principal és Google.

Si la majoria de les visites del vostre lloc web es generen per una sola paraula clau, la competència pot ser que la prengui.

  • Cerqueu la paraula clau a Google per determinar si hi ha un altre lloc web que si es troba per davant del vostre lloc i ofereix un millor valor a l'usuari.
  • Cerqueu notícies per canviar l'algoritme de classificació de Google. Per exemple, l'actualització de Google Panda va danyar el trànsit de molts llocs web.

Lloc web prohibit per Google

L'ús de mètodes prohibits per promocionar el vostre lloc a Google garantirà que el vostre lloc web sigui prohibit per Google.

Cerqueu a Google amb les vostres paraules clau principals i comproveu si apareix com és habitual als resultats de la cerca.

Si el vostre lloc web no apareix en absolut, hauríeu de:

  1. Llegeix les directrius per a administradors web de Google i arregla el teu lloc web.
  2. Envieu la sol·licitud de reconsideració a Google.
