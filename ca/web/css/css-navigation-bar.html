Barra de navegació CSS

Barra de navegació CSS: horitzontal i vertical.

Barra de navegació horitzontal

Codi CSS:

<style>
    #h_nav_bar a { padding:15px; font-weight:bold; float:left; }
    #h_nav_bar a:link { color:#d0d0d0; background-color:#0000a0; }
    #h_nav_bar a:visited { color:#c0c0c0; background-color:#0000a0; }
    #h_nav_bar a:hover { color:#ffffff; background-color:#000060; }
    #h_nav_bar a:active { color:#f0f0f0; background-color:#00ff00; }
</style>

codi HTML:

<div id="h_nav_bar">
    <a href="html-anchor-link.htm">Anchor link</a>
    <a href="html-link-color.htm">Link color</a>
    <a href="../mailto.htm">Email link</a>
    <a href="html-image-link.htm">Image link</a>
    <a href="html-text-link.htm">Text link</a>
</div>

Visualització:

Enllaç d'àncora Color de l'enllaç Enllaç de correu electrònic Enllaç d'imatge Enllaç de text

 

 

 

#link_bar a és l'estil per a tots els estats de l'enllaç.

#link_bar a:link és l'estil d'enllaç normal.

#link_bar a: visitat és l'estil de l'enllaç visitat.

#link_bar a:hover és l'estil de l'enllaç del ratolí.

#link_bar a: actiu és l'estil d'enllaç quan es pressiona amb el ratolí.
