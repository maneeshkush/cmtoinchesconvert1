Color CSS

El codi de colors pot ser un dels següents:

Format hexadecimal: #rrggbb

Format RGB: rgb (vermell, verd, blau)

Format del nom: nom

Exemple

Per al color taronja:

Format hexadecimal: #FFA500

Format RGB: rgb (255,165,0)

Format del nom: taronja

Configuració del color de l'element

Per a un element específic:

<element style="color: code;"></element>

Per a tots els elements del mateix tipus. Posa el codi a l'etiqueta <style> a la secció del cap:

<style>
   element { color: code; }
</style>

Exemple

Configurant el color del text del paràgraf en vermell:

Configuració del color d'un paràgraf específic:

<p style="color: #FF0000;">Some text ...</p>

Visualització:

Algun text...

Configuració del color de tots els paràgrafs

<style>
   p { color: #FF0000; }
</style>

Configuració del color de fons de l'element

element { background: code; }

Exemple

Configuració del color de fons del paràgraf en vermell:

Configuració del color de fons d'un paràgraf específic:

<p style="background: #FF0000;">Some text ...</p>

Visualització:

Algun text...

Configuració del color de fons de tots els paràgrafs:

<style>
   p { background: #FF0000; }
</style>

Configuració del color de la vora de l'element

element { border-color: topcode rightcode bottomcode leftcode }

Exemple

Configuració del color de la vora del paràgraf en vermell, verd, blau i negre:

Configuració del color de la vora d'un paràgraf específic:

<p style="border-color: #FF0000 #00FF00 #0000FF #000000; border-style:solid">Some text ...</p>

Visualització:

Algun text...

Configuració del color de la vora de tots els paràgrafs:

<style>
   p { border-color: #FF0000 #00FF00 #0000FF #000000; }
</style>
