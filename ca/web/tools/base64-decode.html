Entrada de text:

L'entrada de text es codifica primer com a flux de bits binaris de codis ASCII de cada caràcter.

Cada 6 bits del flux de bits es codifiquen en un dígit base64.

Entrada d'imatge:

Esquema d'URI de dades d'imatge amb sortida de dades codificades en base64:

 

data:image/jpeg;base64,és la capçalera de l'esquema d'URI de dades.

 

/9j/4AAQSkZ...són les dades codificades en base64.

 

HTML <img> amb URI de dades base64

<img src="data:image/jpeg;base64,/9j/4AAQSkZ...">

 

Fons CSS amb URI de dades base64

body {
    background-image: url('data:image/jpeg;base64,/9j/4AAQSkZ...');
}
