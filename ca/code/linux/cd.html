comanda cd a Linux/Unix

cd és una ordre de Linux per canviar el directori/carpeta del shell del terminal.

Podeu prémer el botó de pestanya per completar automàticament el nom del directori.

  • sintaxi de cd
  • exemples de cd

sintaxi de cd

$ cd [directory]

exemples d'ordres cd

Canvi al directori inicial (determinat per la variable d'entorn $HOME):

$ cd

 

També canvieu al directori d'inici:

$ cd ~

 

Canviar al directori arrel:

$ cd /

 

Canviar al directori principal:

$ cd ..

 

Canvi al subdirectori Documents :

$ cd Documents

 

Canvi al subdirectori Documents/Llibres :

$ cd Documents/Books

 

Canviar al directori amb la ruta absoluta /home/user/Desktop :

$ cd /home/user/Desktop

 

Canvia al nom del directori amb espai en blanc - Les meves imatges :

$ cd My\ Images

O

$ cd "My Images"

O

$ cd 'My Images'
