Com convertir hexadecimal a color rgb

Com convertir d'un codi de color hexadecimal a un color RGB.

codi de color
hexadecimal El codi de color hexadecimal és un nombre hexadecimal de 6 dígits (base 16):

RRGGBB 16

Els 2 dígits de l'esquerra representen el color vermell.

Els 2 dígits del mig representen el color verd.

Els 2 dígits de la dreta representen el color blau.

Color RGB

El color RGB és una combinació de colors vermell, verd i blau:

(R, G, B)

El vermell, el verd i el blau utilitzen cadascun 8 bits, amb valors enters que van de 0 a 255.

Així, el nombre de colors que es poden generar és:

256×256×256 = 16777216 = 1000000 16

conversió hexadecimal a rgb

1. Agafeu 2 dígits de l'esquerra del codi de color hexadecimal i convertiu-los a valor decimal per obtenir el nivell de color vermell.
2. Obteniu 2 dígits mitjans del codi de color hexadecimal i convertiu-los a valor decimal per obtenir el nivell de color verd.
3. Trobeu 2 dígits correctes del codi de color hexadecimal i convertiu-los a valor decimal per obtenir el nivell de color blau.

Exemple 1
Converteix el codi de color hexadecimal vermell FF0000 a color RGB:

hex = FF0000

Així, els colors RGB són:

R = FF16 = 25510

G = 0016 = 010

B = 0016 = 010

O

RGB = (255, 0, 0)

Exemple #2
Converteix el codi de color hexadecimal d'or FFD700 a color RGB:

Hex = FFD700

Així, els colors RGB són:

R = FF16 = 25510

G = D716 = 21510

B = 0016 = 010

O

RGB = (255, 215, 0)
