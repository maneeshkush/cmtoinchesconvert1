Com convertir RGB a color hexadecimal

Com convertir de color RGB a codi de color hexadecimal.

Color RGB

El color RGB és una combinació de colors vermell, verd i blau:

(R, G, B)

El vermell, el verd i el blau utilitzen cadascun 8 bits, amb valors enters que van de 0 a 255.

Així, el nombre de colors que es poden generar és:

256×256×256 = 16777216 = 100000016

codi de color hexadecimal

El codi de color hexadecimal és un nombre hexadecimal de 6 dígits (base 16):

RRGGBB 16

Els 2 dígits de l'esquerra representen el color vermell.

Els 2 dígits del mig representen el color verd.

Els 2 dígits de la dreta representen el color blau.

conversió rgb a hexadecimal

1. Converteix els valors vermell, verd i blau de decimal a hexadecimal.
2. Combina els 3 valors hexadecimals de vermell, verd i blau junts: RRGGBB.

Exemple 1
Converteix vermell (255,0,0) a codi de color hexadecimal:

R = 25510 = FF16

G = 010 = 0016

B = 010 = 0016

Així, el codi de color hexadecimal és:

Hex = FF0000

Exemple #2
Converteix el color daurat (255,215,0) a un codi de color hexadecimal:

R = 25510 = FF16

G = 21510 = D716

B = 010 = 0016

Així, el codi de color hexadecimal és:

Hex = FFD700

Què fa aquest convertidor RGB a hexadecimal?

Pren com a entrada valors de color vermell, verd i blau que van de 0 a 255 i després els converteix en una cadena hexadecimal que es pot utilitzar per especificar colors en codi html/css. El programari d'edició de fotografies normalment representa el color en RGB i, per tant, si voleu utilitzar els colors utilitzats al vostre programari d'edició de fotos com a fons del vostre element HTML, heu d'obtenir la representació hexadecimal dels valors RGB. Aquesta eina us permet obtenir aquests valors.

Proveu la nostra nova eina de cerca de colors.

Converteix un valor hexadecimal a RGB

Potser heu vist un codi hexadecimal en una pàgina web i voleu utilitzar aquest color al vostre programari d'edició de fotografies. En aquest cas, necessitareu valors RGB si el vostre programari d'edició de fotos no admet valors HEX.
