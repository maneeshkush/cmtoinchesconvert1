Výpočet váženej známky

Vážená trieda sa rovná súčtu súčinu váh (w) v percentách (%) krát trieda (g):

Weighted grade = w1×g1+ w2×g2+ w3×g3+...

Ak váhy nie sú v percentách (hodiny alebo body...), mali by ste ich vydeliť aj súčtom váh:

Weighted grade = (w1×g1+ w2×g2+ w3×g3+...) / (w1+w2+w3+...)

Príklad 1

3 body Kurz matematiky so známkou 74.

5 bodov Biologický kurz so známkou 87.

2 body Kurz dejepisu so známkou 71.

Vážená priemerná známka sa vypočíta takto:

Weighted grade =

 = (w1×g1+ w2×g2+ w3×g3) / (w1+w2+w3)

 = (3×74+ 5×87+ 2×71) / (3+5+2) = 79.90

Príklad 2

3 body Kurz matematiky so známkou 72.

5 bodov Biologický kurz so známkou 88.

2 body Kurz dejepisu so známkou 70.

Vážená priemerná známka sa vypočíta takto:

Weighted grade =

 = (w1×g1+ w2×g2+ w3×g3) / (w1+w2+w3)

 = (3×72+ 5×88+ 2×70) / (3+5+2) = 79.60
