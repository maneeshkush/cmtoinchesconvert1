presmerovanie jQuery

Ako presmerovať stránku na URL pomocou jQuery.

Vyhľadávače používajú stavový kód 301 na prenos hodnotenia stránky zo starej adresy URL na novú adresu URL.

jQuery redirection return http odpoveď stavový kód: 200 OK.

Takže presmerovanie jQuery, podobne ako presmerovanie Javascript, nie je priateľské pre vyhľadávače a je lepšie použiť iné metódy presmerovania, ktoré vrátia stavový kód 301 Presunuté natrvalo.

Presmerovanie jQuery je vlastne presmerovanie JavaScriptu .Presmerovanie pomocou jQuery sa považuje za prehnané, pretože môžete vykonať čisté presmerovanie Javascript s menším množstvom kódu.

presmerovanie jQuery

Nahraďte starú stránku kódom presmerovania adresou URL stránky, na ktorú chcete presmerovať.

old-page.html:

<!DOCTYPE html>
<html>
<body>
<script src="http://ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script>
<typ skriptu ="text/javascript">
   // presmerovanie adresy URL jQuery
   $(document).ready( function() {
      url = "http://www.mydomain.com/new-page.html";
      $( location ).attr( "href", url);
   });
</script>
</body>
</html>

Príklad presmerovania jQuery

jquery-redirect-test.htm

<!DOCTYPE html>
<html>
<body>
<script src="//ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script>
<script type="text/javascript">
   $(document).ready( function() {
      url = "https://www.cmtoinchesconvert.com/sk/web/dev/jquery-redirect.htm";
      $(location).attr("href", url);
   });
</script>
</body>
</html>
