príkaz pwd v systéme Linux/Unix

Unix/Linux príkaz pwd.

pwd - tlač pracovného adresára, je príkaz Linuxu na získanie aktuálneho pracovného adresára.

  • syntax pwd
  • pwd príklady

syntax pwd

$ pwd [option]

príklady príkazov pwd

Zmeňte adresár na adresár /usr/src a vytlačte pracovný adresár:

$ cd /usr/src
$ pwd
/user/src

 

Zmeňte adresár na domovský adresár a vytlačte pracovný adresár:

$ cd ~
$ pwd
/home/user

 

Zmeňte adresár na nadradený adresár domovského adresára a vytlačte pracovný adresár:

$ cd ~/..
$ pwd
/home

 

Zmeňte adresár na koreňový adresár a vytlačte pracovný adresár:

$ cd /
$ pwd
/
