Príznaky voľby gcc -L / -l

gcc -l sa prepojí so súborom knižnice.

gcc -L hľadá v adresári súbory knižnice.

Syntax

$ gcc [options] [source files] [object files] [-Ldir] -llibname [-o outfile]

 

Link -l s názvom knižnice bez predpony lib a prípon .a alebo .so .

Príklady

Príklad 1

Pre súbor statickej knižnice libmath. použitie -lmath :

$ gcc -static myfile.c -lmath -o myfile

 
Príklad2

Pre súbor zdieľanej knižnice libmath. takže použite -lmath :

$ gcc myfile.c -lmath -o myfile

 
Príklad 3

súbor1.c:

// file1.c
#include <stdio.h>

void main()
{
    printf("main() run!\n");
    myfunc();
}

 

súbor2.c:

// file2.c
#include <stdio.h>

void myfunc()
{
    printf("myfunc() run!\n");
}

 

Zostavte súbor2.c , skopírujte objektový súbor file2.o do adresára libs a archivujte ho do statickej knižnice libmylib.a :

$ gcc -c file2.c
$ mkdir libs
$ cp file2.o libs
$ cd libs
$ ar rcs libmylib.a file2.o

 

Zostavte súbor1.c so statickou knižnicou libmylib.a v adresári libs .

Zostaviť bez výsledkov -L s chybou:

$ gcc file1.c -lmylib -o outfile
/usr/bin/ld: cannot find -llibs
collect2: ld returned 1 exit status
$

Zostavte s -L a spustite:

$ gcc file1.c -Llibs -lmylib -o outfile
$ ./outfile
main() run!
myfunc() run!
$
