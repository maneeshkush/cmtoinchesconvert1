CSS-kleur

De kleurcode kan een van de volgende zijn:

Hex-formaat: #rrggbb

RGB-formaat: RGB (rood, groen, blauw)

Naamformaat: naam

Voorbeeld

Voor oranje kleur:

Hex-formaat: #FFA500

RGB-indeling: RGB (255.165.0)

Naamformaat: oranje

Elementkleur instellen

Voor specifiek element:

<element style="color: code;"></element>

Voor alle elementen van hetzelfde type.Zet code in <style> tag in het head-gedeelte:

<style>
   element { color: code; }
</style>

Voorbeeld

Alineatekstkleur instellen op rood:

Kleur van specifieke paragraaf instellen:

<p style="color: #FF0000;">Some text ...</p>

Weergave:

Wat tekst...

Kleur instellen van alle alinea's

<style>
   p { color: #FF0000; }
</style>

Achtergrondkleur van het element instellen

element { background: code; }

Voorbeeld

Alinea-achtergrondkleur instellen op rood:

Achtergrondkleur van specifieke alinea instellen:

<p style="background: #FF0000;">Some text ...</p>

Weergave:

Wat tekst...

Achtergrondkleur van alle alinea's instellen:

<style>
   p { background: #FF0000; }
</style>

Randkleur van element instellen

element { border-color: topcode rightcode bottomcode leftcode }

Voorbeeld

Alinearandkleur instellen op rood, groen, blauw en zwart:

Randkleur van specifieke alinea instellen:

<p style="border-color: #FF0000 #00FF00 #0000FF #000000; border-style:solid">Some text ...</p>

Weergave:

Wat tekst...

Randkleur van alle alinea's instellen:

<style>
   p { border-color: #FF0000 #00FF00 #0000FF #000000; }
</style>
