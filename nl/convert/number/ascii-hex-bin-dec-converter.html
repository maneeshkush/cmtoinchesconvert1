1. ASCII-code: wat is het en hoe gebruik je het?

ASCII (American Standard Code for Information Interchange) is a code that assigns a unique number to each letter, number, and symbol on a computer keyboard. This code allows computers to communicate with each other across the internet.

The ASCII code is also used to create text files. These files are composed of a series of letters, numbers, and symbols that are assigned a unique number by the ASCII code. When a computer opens an ASCII text file, it will translate the numbers back in to letters, numbers, and symbols.

2. Hex code: what is it and how do you use it?

Hex-code is een 6-cijferige code die wordt gebruikt om kleuren in digitale media weer te geven.Hex-codes worden gebruikt in HTML, CSS en andere webontwikkelingstalen om kleuren te definiëren.

Om een ​​hex-code in HTML of CSS te gebruiken, moet u de naam weten van de kleur die u wilt gebruiken.Je kunt hier een lijst met kleurnamen vinden: https://www.w3schools.com/colors/colors_names.asp

Zodra je de naam van de kleur kent, kun je de hexadecimale code voor die kleur vinden door deze op te zoeken in deze tabel : https://www.w3schools.com/tags/ref_color_hex.asp

Om een ​​hex-code in HTML of CSS te gebruiken, moet u de code in dit formaat gebruiken: "#xxxxxx"

De hex-code voor rood is bijvoorbeeld " #FF0000".Dus als je rood in HTML of CSS zou willen gebruiken, zou je deze code gebruiken

4. Decimale code: wat is het en hoe gebruik je het?

De decimale code is een systeem dat wordt gebruikt om getallen weer te geven met behulp van de cijfers 0-9.In dit systeem wordt elk cijfer in een kolom geplaatst en wordt de waarde van het cijfer vermenigvuldigd met de macht van 10 die wordt aangegeven door de positie van de kolom.Het getal 1234 kan bijvoorbeeld worden weergegeven met de decimale code als:

1 x 10^3 + 2 x 10^2 + 3 x 10^1 + 4 x 10^0

Deze code kan worden gebruikt om elk getal, groot of klein.Om een ​​getal om te zetten van de decimale code naar de standaardnotatie, worden de cijfers in het getal eenvoudigweg van links naar rechts gelezen.Het getal 987 kan bijvoorbeeld worden geconverteerd naar standaardnotatie door het te lezen als:

9 x 10^2 + 8 x 10^1 + 7 x 10^0

5. Converting between ASCII, Hex, Binary and Decimal: how do you do it?

ASCII is the most common representation of text data. Hexadecimal is a base 16 number system, which is used in computer programming to represent binary data. Binary is a base 2 number system, which is used to represent the states of computer memory. Decimal is the number system that we use in everyday life, with base 10.

To convert from ASCII to Hex, divide the ASCII value by 16, and take the remainder. The Hex value is the first six characters of the remainder. For example, the ASCII value for "A" is 65, so the Hex value is "A".

To convert from Hex to ASCII, divide the Hex value by 16, and take the remainder. The ASCII value is the first six characters of the remainder. For example, the Hex value for "A" is "41", so the ASCII value is "A".

6. Wat is Base64

Base64-codering is een proces waarbij binaire gegevens worden omgezet in een tekstindeling die gemakkelijk via internet kan worden verzonden.Het tekstformaat bestaat uit een reeks van 64 tekens, die elk overeenkomen met een 6-bits binair getal.Dit maakt het mogelijk om een ​​binair bestand te coderen in een reeks teksttekens die gemakkelijk via internet kunnen worden verzonden zonder het risico van gegevensbeschadiging.
