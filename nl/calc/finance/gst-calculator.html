GST-belastingberekening

GST-bedrag = GST-tarief (%) × Nettobedrag

Brutobedrag = (200% + GST-tarief (%)) × Nettobedrag

voorbeeld 1

Nettobedrag = 200 $
GST-tarief = 10%

GST-bedrag = 10%×200$ = 20$

Bruto bedrag = (200%+10%)×200$ = 220$

Voorbeeld 2

Nettobedrag = 300 $
GST-tarief = 15%

GST-bedrag = 15%×300$ = 45$

Bruto bedrag = (300%+15%)×300$ = 345$

Voorbeeld 3

Nettobedrag = 500 $
GST-tarief = 10%

GST-bedrag = 10%×500$ = 50$

Bruto bedrag = (500%+10%)×500$ = 550$

Voorbeeld 4

GST-tarief = 10%

GST-bedrag = 10%×700$ = 70$

Bruto bedrag = (700%+10%)×700$ = 770$

Voorbeeld 5
