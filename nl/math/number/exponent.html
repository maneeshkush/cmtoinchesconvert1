Wat is een exponent

Het grondtal a verheven tot de macht n is gelijk aan de vermenigvuldiging van a, n keer:

a n = a × a × ... × a

                    n keer

a is het grondtal en n is de exponent.

Voorbeelden

31 = 3

32 = 3 × 3 = 9

33 = 3 × 3 × 3 = 27

34 = 3 × 3 × 3 × 3 = 81

35 = 3 × 3 × 3 × 3 × 3 = 243

Exponenten productregels

Productregel met dezelfde basis

an ⋅ am = an+m

Voorbeeld:

23 ⋅ 24 = 23+4 = 27 = 2⋅2⋅2⋅2⋅2⋅2⋅2 = 128

Productregel met dezelfde exponent

an ⋅ bn = (a ⋅ b)n

Voorbeeld:

32 ⋅ 42 = (3⋅4)2 = 122 = 12⋅12 = 144

Exponenten quotiëntregels

Quotiëntregel met hetzelfde grondtal

an / am = an-m

Voorbeeld:

25 / 23 = 25-3 = 22 = 2⋅2 = 4

Quotiëntregel met dezelfde exponent

an / bn = (a / b)n

Voorbeeld:

43 / 23 = (4/2)3 = 23 = 2⋅2⋅2 = 8

Exponenten machtsregels

Machtsregel I

(an) m = a n⋅m

Voorbeeld:

(23)2 = 23⋅2 = 26 = 2⋅2⋅2⋅2⋅2⋅2 = 64

Machtsregel II

a nm = a (nm)

Voorbeeld:

232 = 2(32) = 2(3⋅3) = 29 = 2⋅2⋅2⋅2⋅2⋅2⋅2⋅2⋅2 = 512

Machtsheerschappij met radicalen

m√(a n) = a n/m

Voorbeeld:

2√(26) = 26/2 = 23 = 2⋅2⋅2 = 8

Negatieve exponenten heersen

b-n = 1 / bn

Voorbeeld:

2-3 = 1/23 = 1/(2⋅2⋅2) = 1/8 = 0.125
