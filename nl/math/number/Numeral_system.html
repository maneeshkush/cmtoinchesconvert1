Cijfersysteem

b - cijfersysteembasis

d n - het n-de cijfer

n - kan beginnen met een negatief getal als het getal een breukdeel heeft.

N +1 - het aantal cijfers

Binair cijfersysteem - Base-2

Binaire getallen gebruiken alleen 0 en 1 cijfers.

B staat voor binair voorvoegsel.

Voorbeelden:

101012 = 10101B = 1×24+0×23+1×22+0×21+1×20 = 16+4+1= 21

101112 = 10111B = 1×24+0×23+1×22+1×21+1×20 = 16+4+2+1= 23

1000112 = 100011B = 1×25+0×24+0×23+0×22+1×21+1×20 =32+2+1= 35

Octaal cijfersysteem - Base-8

Octale getallen gebruiken cijfers van 0..7.

Voorbeelden:

278 = 2×81+7×80 = 16+7 = 23

308 = 3×81+0×80 = 24

43078 = 4×83+3×82+0×81+7×80= 2247

Decimaal cijfersysteem - basis-10

Decimale getallen gebruiken cijfers van 0..9.

Dit zijn de reguliere nummers die wij gebruiken.

Voorbeeld:

253810 = 2×103+5×102+3×101+8×100

Hexadecimaal cijfersysteem - Base-16

Hex-nummers gebruiken cijfers van 0..9 en A..F.

H staat voor hex-voorvoegsel.

Voorbeelden:

2816 = 28H = 2×161+8×160 = 40

2F16 = 2FH = 2×161+15×160 = 47

BC1216 = BC12H = 11×163+12×162+1×161+2×160= 48146
