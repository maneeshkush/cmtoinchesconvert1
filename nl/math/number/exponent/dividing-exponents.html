Delen van exponenten met hetzelfde grondtal

Voor exponenten met dezelfde basis moeten we de exponenten aftrekken:

a n / a m = a n-m

Voorbeeld:

26 / 23 = 26-3 = 23 = 2⋅2⋅2 = 8

Het delen van exponenten met verschillende bases

Als de basissen verschillend zijn en de exponenten van a en b hetzelfde zijn, kunnen we eerst a en b delen:

a n / b n = (a / b) n

Voorbeeld:

63 / 23 = (6/2)3 = 33 = 3⋅3⋅3 = 27

 

Als de basissen en de exponenten verschillend zijn, moeten we elke exponent berekenen en vervolgens delen:

a n / b m

Voorbeeld:

62 / 33 = 36 / 27 = 1.333

Negatieve exponenten delen

Voor exponenten met dezelfde basis kunnen we de exponenten aftrekken:

a-n / a-m = a-n-(-m) = am-n

Voorbeeld:

2-3 / 2-5 = 25-3 = 22 = 2⋅2 = 4

 

Als de basissen verschillend zijn en de exponenten van a en b hetzelfde zijn, kunnen we eerst a en b vermenigvuldigen:

a-n / b-n = (a/b)-n = 1 / (a/b)n = (b/a)n

Voorbeeld:

3-2 / 4-2 = (4/3)2 = 1.7778

 

Als de basissen en de exponenten verschillend zijn, moeten we elke exponent berekenen en vervolgens delen:

a-n / b-m = bm / an

Voorbeeld:

3-2 / 4-3 = 43 / 32 = 64 / 9 = 7.111

Breuken delen met exponenten

Breuken delen met exponenten met dezelfde breukbasis:

(a / b)n / (a / b)m = (a / b)n-m

Voorbeeld:

(4/3)3 / (4/3)2 = (4/3)3-2 = (4/3)1 = 4/3 = 1.333

 

Breuken delen met exponenten met dezelfde exponent:

(a / b)n / (c / d)n = ((a / b)/(c / d))n = ((a⋅d / b⋅c))n

Voorbeeld:

(4/3)3 / (3/5)3 = ((4/3)/(3/5))3 = ((4⋅5)/(3⋅3))3 = (20/9)3 = 10.97

 

Breuken delen met exponenten met verschillende bases en exponenten:

(a / b) n / (c / d) m

Voorbeeld:

(4/3)3 / (1/2)2 = 2.37 / 0.25 = 9.481

Delen van fractionele exponenten

Delen van fractionele exponenten met dezelfde fractionele exponent:

a n/m / b n/m = (a / b) n/m

Voorbeeld:

33/2 / 23/2 = (3/2)3/2 = 1.53/2 = √(1.53) = √3.375 = 1.837

 

Delen van fractionele exponenten met hetzelfde grondtal:

a n/m / a k/j = a (n/m)-(k/j)

Voorbeeld:

23/2 / 24/3 = 2(3/2)-(4/3) = 2(1/6) = 6√2 = 1.122

 

Delen van gebroken exponenten met verschillende exponenten en breuken:

a n/m / b k/j

Voorbeeld:

23/2 / 24/3 = √(23) / 3√(24) = 2.828 / 2.52 = 1.1222

Variabelen delen met exponenten

Voor exponenten met dezelfde basis kunnen we de exponenten aftrekken:

xn / xm = xn-m

Voorbeeld:

x5 / x3 = (x⋅x⋅x⋅x⋅x) / (x⋅x⋅x) = x5-3 = x2
