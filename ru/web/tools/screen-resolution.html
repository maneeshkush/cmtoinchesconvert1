Определение разрешения экрана с помощью Javascript

Вы можете получить размер экрана с помощью свойств screen.width и screen.height .

Вы можете получить доступный размер экрана с помощью свойств screen.availWidth и screen.availHeight .

Например:

<script>
alert(screen.width+' x '+screen.height);
</script>
