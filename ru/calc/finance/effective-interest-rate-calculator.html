Расчет процентной ставки за период действия

Таким образом, эффективная процентная ставка за период равна номинальной годовой процентной ставке, деленной на количество периодов в году n:

Ставка за период  = Номинальная годовая ставка / n

Пример 1

Какова эффективная процентная ставка за период для номинальной годовой процентной ставки 3%, начисляемой ежемесячно?

Решение:

Effective Period Rate = 3% / 12months = 0.03 / 12 = 0.250%

Пример 2

Какова эффективная процентная ставка за период для номинальной годовой процентной ставки 7%, начисляемой ежемесячно?

Решение:

Effective Period Rate = 7% / 12months = 0.07 / 12 = 0.583%

Пример 3

Какова эффективная процентная ставка за период для номинальной годовой процентной ставки 9%, начисляемой ежемесячно?

Решение:

Effective Period Rate = 9% / 12months = 0.9 / 12 = 0.750%

Расчет эффективной годовой процентной ставки

Таким образом, эффективная годовая процентная ставка равна 1 плюс номинальная процентная ставка в процентах, деленная на количество периодов начисления сложных процентов в году n в степени n минус 1.

Effective Rate = (1 +  Nominal Rate /  n)n - 1

Пример 1

Какова эффективная годовая процентная ставка для номинальной годовой процентной ставки 3%, начисляемой ежемесячно?

Решение:

Effective Rate = (1 + 3% / 12)12 - 1

      = (1 + 0,03 / 12) 12  - 1

      = 0,03042 = 3,042%

Пример 2

Какова эффективная годовая процентная ставка для номинальной годовой процентной ставки 7%, начисляемой ежемесячно?

Решение:

Effective Rate = (1 + 7% / 12)12 - 1

      = (1 + 0,07 / 12) 12  - 1

      = 0,07229 = 7,229%

Пример 3

Какова эффективная годовая процентная ставка для номинальной годовой процентной ставки 9%, начисляемой ежемесячно?

Решение:

Effective Rate = (1 + 9% / 12)12 - 1

      = (1 + 0,9 / 12) 12  - 1

      = 0,09381 = 9,381%
