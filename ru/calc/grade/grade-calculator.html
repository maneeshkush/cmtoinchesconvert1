Расчет взвешенной оценки

Взвешенная оценка равна сумме произведений весов (w) в процентах (%) на оценку (g):

Weighted grade =  w1×g1+ w2×g2+ w3×g3+...

Пример 1

Математический курс с оценкой 77 и весом 30%.

Курс биологии с оценкой 85 и весом 50%.

Курс истории с оценкой 73 и весом 20%.

Средневзвешенная оценка рассчитывается по формуле:

Weighted grade =

 =  w1×g1+ w2×g2+ w3×g3

 = (30×77+50×85+20×73) = 80.20

Пример 2

Математический курс с оценкой 79 и весом 30%.

Курс биологии с оценкой 89 и весом 50%.

Курс истории с оценкой 76 и весом 20%.

Средневзвешенная оценка рассчитывается по формуле:

Weighted grade =

 =  w1×g1+ w2×g2+ w3×g3

 = (30×79+50×89+20×76) = 83.40

 

Когда веса не в процентах (часы или баллы...), вы также должны разделить на сумму весов:

Weighted grade = (w1×g1+ w2×g2+ w3×g3+...) / (w1+w2+w3+...)

Пример 1

Курс математики на 3 балла с оценкой 77.

5 баллов по курсу биологии с оценкой 85.

2 балла по курсу истории с оценкой 73.

Средневзвешенная оценка рассчитывается по формуле:

Weighted grade =

 = (w1×g1+ w2×g2+ w3×g3) / (w1+w2+w3)

 = (3×77+ 5×85+ 2×73) / (3+5+2) = 80.20

Пример 2

Курс математики на 3 балла с оценкой 79.

5 баллов по курсу биологии с оценкой 79.

2 балла по курсу истории с оценкой 76.

Средневзвешенная оценка рассчитывается по формуле:

Weighted grade =

 = (w1×g1+ w2×g2+ w3×g3) / (w1+w2+w3)

 = (3×79+ 5×89+ 2×76) / (3+5+2) = 83.40
