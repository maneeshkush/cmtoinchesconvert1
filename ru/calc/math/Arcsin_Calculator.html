Определение арксинуса

Функция арксинуса являетсяобратной функцией y = sin(x).

arcsin(y) = sin-1(y) = x + 2kπ

Для каждого

k = {...,-2,-1,0,1,2,...}

 

Например, если синус 30° равен 0,5:

грех(30°) = 0,5

Тогда арксинус 0,5 равен 30°:

arcsin(0,5) = sin -1 (0,5) = 30°
