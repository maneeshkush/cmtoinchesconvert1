Калькулятор экспоненциального роста/распада

Онлайн-калькулятор экспоненциального роста/убывания.

Формула экспоненциального роста/распада

x(t) = x0 × (1 + r) t

x (t) — значение в момент времени t.

x 0 — начальное значение в момент времени t=0.

r — скорость роста, когда r>0, или скорость затухания, когда r<0, в процентах.

t — время в дискретных интервалах и выбранных единицах времени.

Калькулятор экспоненциального роста

Введите начальное значение ( x 0 ):  
Введите скорость роста/затухания ( r ): %
Введите время ( t ):  
   
Значение в момент времени t ( x (t)):  

Пример 1

x0 = 40

r = 3% = 0.04

t = 80 hours

x(t) = x0 × (1 + r) t = 40×(1+0.03)80 = 425.63

Пример 2

x0 = 40

r = 5% = 0.05

t = 80 hours

x(t) = x0 × (1 + r) t = 40×(1+0.05)80 = 1982

Пример 3

x0 = 60

r = 3% = 0.05

t = 80 hours

x(t) = x0 × (1 + r) t = 60×(1+0.05)80 = 2973
