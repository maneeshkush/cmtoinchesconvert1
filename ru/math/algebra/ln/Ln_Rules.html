Натуральный логарифм Правила и свойства

 

Название правила Правило Пример
Правило продукта

ln(x ∙ y) = ln(x) + ln(y)

ln(3 ∙ 7) = ln(3) + ln(7)
Частное правило

ln(x / y) = ln(x) - ln(y)

ln(3 / 7) = ln(3) - ln(7)
Силовое правило

ln(x y) = y ∙ ln(x)

ln(28) = 8 ∙ ln(2)
производная Ln

f (x) = ln(x) ⇒ f ' (x) = 1 / x

 
Ln интеграл

∫ ln(x)dx = x ∙ (ln(x) - 1) + C

  
Ln отрицательного числа

ln(x) is undefined when x ≤ 0

  
Ln нуля

ln(0) is undefined

 
 
Ln одного

ln(1) = 0

  
Ln бесконечности

lim ln(x) = ∞ , when x→∞

  

 

Производная функции натурального логарифма (ln)

Производная функции натурального логарифма является обратной функцией.

Когда

f (x) = ln(x)

Производная f(x):

f ' (x) = 1 / x
