Что такое показатель

Основание a, возведенное в степень n, равно произведению a, n раз:

a n = a × a × ... × a

                    n раз

а - основание, а n - показатель степени.

Примеры

31 = 3

32 = 3 × 3 = 9

33 = 3 × 3 × 3 = 27

34 = 3 × 3 × 3 × 3 = 81

35 = 3 × 3 × 3 × 3 × 3 = 243

Экспоненты правила продукта

Правило продукта с той же базой

an ⋅ am = an+m

Пример:

23 ⋅ 24 = 23+4 = 27 = 2⋅2⋅2⋅2⋅2⋅2⋅2 = 128

Правило произведения с одинаковым показателем

an ⋅ bn = (a ⋅ b)n

Пример:

32 ⋅ 42 = (3⋅4)2 = 122 = 12⋅12 = 144

См.: Умножение показателей.

Частные правила экспоненты

Частное правило с тем же основанием

an / am = an-m

Пример:

25 / 23 = 25-3 = 22 = 2⋅2 = 4

Частное правило с тем же показателем

an / bn = (a / b)n

Пример:

43 / 23 = (4/2)3 = 23 = 2⋅2⋅2 = 8

См.: Разделение показателей .

Правила мощности экспонентов

Правило силы I

(an) m = a n⋅m

Пример:

(23)2 = 23⋅2 = 26 = 2⋅2⋅2⋅2⋅2⋅2 = 64

Правило мощности II

a nm = a (nm)

Пример:

232 = 2(32) = 2(3⋅3) = 29 = 2⋅2⋅2⋅2⋅2⋅2⋅2⋅2⋅2 = 512

Силовое правило с радикалами

m√(a n) = a n/m

Пример:

2√(26) = 26/2 = 23 = 2⋅2⋅2 = 8

Правило отрицательных показателей

b-n = 1 / bn

Пример:

2-3 = 1/23 = 1/(2⋅2⋅2) = 1/8 = 0.125
