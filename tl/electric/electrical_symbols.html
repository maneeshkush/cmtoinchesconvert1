Mga Simbolo ng Elektrisidad at Mga Simbolo ng Elektroniko

Ang mga simbolong elektrikal at mga simbolo ng elektronikong circuit ay ginagamit para sa pagguhit ng schematic diagram.

Ang mga simbolo ay kumakatawan sa mga electrical at electronic na bahagi.

Mga Simbolo ng Diode / LED
Diode Pinapayagan ng diode ang kasalukuyang daloy sa isang direksyon lamang - kaliwa (anode) hanggang kanan (cathode).
Zener diode Pinapayagan ang kasalukuyang daloy sa isang direksyon, ngunit maaari ding dumaloy sa reverse na direksyon kapag nasa itaas ng breakdown boltahe
Schottky Diode Ang Schottky diode ay isang diode na may mababang boltahe drop
Varactor / Varicap Diode Variable capacitance diode
Tunnel Diode  
Light Emitting Diode (LED) Ang LED ay naglalabas ng liwanag kapag dumadaloy ang kasalukuyang
Photodiode Pinapayagan ng Photodiode ang kasalukuyang daloy kapag nakalantad sa liwanag
Mga Simbolo ng Transistor
NPN Bipolar Transistor Pinapayagan ang kasalukuyang daloy kapag mataas ang potensyal sa base (gitna)
PNP Bipolar Transistor Pinapayagan ang kasalukuyang daloy kapag mababa ang potensyal sa base (gitna)
Darlington Transistor Ginawa mula sa 2 bipolar transistor.May kabuuang pakinabang ng produkto ng bawat pakinabang.
JFET-N Transistor N-channel na field effect transistor
JFET-P Transistor P-channel na field effect transistor
NMOS Transistor N-channel MOSFET transistor
PMOS Transistor P-channel na MOSFET transistor

Mga Simbolo ng Logic Gates
HINDI Gate (Inverter ) Mga Output 1 kapag ang input ay 0

XOR Gate Mga Output 1 kapag iba ang mga input.(Eksklusibo o)
