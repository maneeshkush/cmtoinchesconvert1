Decibel (dB) na kahulugan

Kaya ang Decibel (Simbolo: dB) ay isang logarithmic unit na nagsasaad ng ratio o gain.

Kaya ginagamit ang Decibel upang ipahiwatig ang antas ng mga acoustic wave at electronic signal.

Kaya Ang logarithmic scale ay maaaring maglarawan ng napakalaki o napakaliit na mga numero na may mas maikling notasyon.

Kaya Ang antas ng dB ay maaaring tingnan bilang kamag-anak na nakuha ng isang antas kumpara sa iba pang antas, o ganap na antas ng logarithmic scale para sa mga kilalang antas ng sanggunian.

Ang decibel ay isang walang sukat na yunit.

Ang ratio sa bels ay ang base 10 logarithm ng ratio ng P 1 at P 0 :

RatioB = log10(P1 / P0)

Ang decibel ay isang ikasampu ng isang bel, kaya ang 1 bel ay katumbas ng 10 decibel:

1B = 10dB

Power ratio

Kaya Ang ratio ng kapangyarihan sa decibels (dB) ay 10 beses base 10 logarithm ng ratio ng P 1 at P 0 .

RatiodB = 10⋅log10(P1 / P0)

Amplitude ratio

Kaya Ang ratio ng mga dami tulad ng boltahe, kasalukuyang at antas ng presyon ng tunog ay kinakalkula bilang ratio ng mga parisukat.

Kaya Ang ratio ng amplitude sa decibels (dB) ay 20 beses base 10 logarithm ng ratio ng V 1 at V 0 :

RatiodB = 10⋅log10(V12 / V02) = 20⋅log10(V1 / V0)

Power ratio sa conversion ng dB

Ang gain G dB ay katumbas ng 10 times base 10 logarithm ng ratio ng power P 2 at ang reference power P 1 .

GdB = 10 log10(P2 / P1)

 

Ang P 2 ay ang antas ng kapangyarihan.

Ang P 1 ay ang tinutukoy na antas ng kapangyarihan.

Ang G dB ay ang power ratio o gain sa dB.

 
Halimbawa

Kaya Hanapin ang pakinabang sa dB para sa isang system na may input power na 5W at output power na 10W.

GdB = 10 log10(Pout/Pin) = 10 log10(10W/5W) = 3.01dB

conversion ng dB sa power ratio

Kaya Ang power P 2 ay katumbas ng reference power P 1 beses 10 na itinaas ng gain sa G dB na hinati ng 10.

P2 = P1  ⋅ 10(GdB / 10)

 

Ang P 2 ay ang antas ng kapangyarihan.

Ang P 1 ay ang tinutukoy na antas ng kapangyarihan.

Ang G dB ay ang power ratio o gain sa dB.

Amplitude ratio sa conversion ng dB

Para sa amplitude ng mga alon tulad ng boltahe, kasalukuyang at antas ng presyon ng tunog:

GdB = 20 log10(A2 / A1)

 

Ang A 2 ay ang antas ng amplitude.

Ang A 1 ay ang tinutukoy na antas ng amplitude.

Ang G dB ay ang amplitude ratio o gain sa dB.

conversion ng dB sa amplitude ratio

A2 = A1  ⋅ 10(GdB/ 20)

Ang A 2 ay ang antas ng amplitude.

Ang A 1 ay ang tinutukoy na antas ng amplitude.

Ang G dB ay ang amplitude ratio o gain sa dB.

 
Halimbawa

Hanapin ang output boltahe para sa isang sistema na may input boltahe ng 5V at boltahe makakuha ng 6dB.

Vout = Vin⋅ 10 (GdB / 20) = 5V ⋅ 10 (6dB / 20) = 9.976V ≈ 10V

Pagkuha ng boltahe

Kaya Ang boltahe na nakuha ( G dB ) ay 20 beses ang base 10 logarithm ng ratio ng output boltahe ( V out ) at ang input boltahe ( V in ):

GdB = 20⋅log10(Vout / Vin)

Kasalukuyang pakinabang

Kaya Ang kasalukuyang pakinabang ( G dB ) ay 20 beses ang base 10 logarithm ng ratio ng kasalukuyang output ( I out ) at ang input current ( I in ):

GdB = 20⋅log10(Iout / Iin)

Acoustic gain

Kaya Ang acoustic gain ng isang hearing aid ( G dB ) ay 20 beses ang base 10 logarithm ng ratio ng output sound level ( L out ) at ang input sound level ( L in ).

GdB = 20⋅log10(Lout / Lin)

Signal to Noise Ratio (SNR)

Kaya Ang ratio ng signal sa ingay ( SNR dB ) ay 10 beses ang base 10 logarithm ng signal amplitude ( A signal ) at ang noise amplitude ( A noise ).

SNRdB = 10⋅log10(Asignal / Anoise)

Sound level meter

Ang sound level meter o SPL meter ay isang device na sumusukat sa sound pressure level (SPL) ng sound waves sa decibels (dB-SPL) units.

Ginagamit ang SPL meter upang subukan at sukatin ang lakas ng mga sound wave at para sa pagsubaybay sa polusyon ng ingay.

Ang yunit para sa pagsukat ng antas ng presyon ng tunog ay pascal (Pa) at sa logarithmic scale ang dB-SPL ay ginagamit.
