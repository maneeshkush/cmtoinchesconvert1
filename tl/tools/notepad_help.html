Tulong sa Notepad

Mayroong 2 paraan upang i-save ang teksto

  1. Ang teksto ay nai-save sa tuwing isasara mo ang tab na Notepad, sa lokal na cache ng browser.Kapag pumasok ka muli sa pahina ng Notepad, lalabas muli ang teksto.
  2. Ang teksto ay nai-save/na-backup sa hard drive kapag pinindot mo ang pindutang I-save.Upang muling buksan ang text mula sa hard drive, pindutin ang Open button at piliin ang text file na iyong ginawa.

Mga parameter ng pagsasaayos


Mahalagang impormasyon

  • Kung nawawala ang text ng nakaraang session :
    • Kung mayroong teksto, piliin at kopyahin ito gamit ang Ctrl+C at i-paste ito gamit ang Ctrl+V sa pahina ng Notepad :

    • Kopyahin ang text mula sa na-update na text file na nabuo ng Auto save operation sa Download folder (kung mayroon).
    • Suriin kung tinitingnan mo ang parehong computer na ginamit mo dati.
    • Paganahin ang cookies at history ng browser.
    • Huwag gumamit ng pribado/incognito mode ng browser.Ang tekstong lokal na imbakan ay tatanggalin ng browser kapag isinara mo ang window ng browser.
    • Subukang idagdag/alisin ang www mula sa URL sa address bar ng browser.
  • Ang teksto ng Notepad ay hindi mase-save sa incognito/private mode na pag-browse !!!
  • Maaaring matanggal ang text ng naka-save na Notpadkapag tinanggal mo ang iyong kasaysayan sa pag-browse/cache o nagpatakbo ng application sa paglilinis ng disk (Hal. Windows Disk cleanup / CCleaner)!!!
  • Kunghindi gumana ang pindutang Buksan ang file, mangyaring i-reload ang pahina at subukang muli.
  • Gamitin ang Notepad na may na-update na bersyon ng browser.Kung gumagamit ka ng Internet Explorer , kopyahin ang iyong teksto sa isang modernong browser (hal. Chrome/Edge/Firefox ).
  • Ang teksto ng notepad ay awtomatikong nai-save sa lokal na cache ng browser (hindi ligtas).
  • Maaaring i-auto save (backup) ang text ng notepad sa hard drive, ayon sa tagal ng auto save sa View > Preferences menu.
  • Maaari mong i-backup ang text ng Notepad sa hard drive, gamit ang button na I-save o menu File > Save.
  • Para sa Mac gamitin ang ⌘ Command sa halip na Ctrl key.
  • Upang buksan ang naka-save na file, hanapin ang file sa folder ng Mga Download.
  • Kung ang mga backgound na linya ay hindi mag-scroll, itago ang mga linya: alisan ng check ang menu View > Preferences > Text lines
  • I-save ang button o menu File > I-save i-save ang file sa folder ng Mga Download .Tingnan: Saan napupunta ang mga file kapag na-download?
  • Kung hindi nakikita ang mga linya ng Text , pakisubukang gumamit ng Chrome browser.
  • Kung hindi gumagana ang spell check, subukang paganahin ito sa seksyong Mga Setting>Mga Wika ng iyong browser.Kung hindi tinukoy, maaari mo ring subukang itakda ang English (United States) sa setting ng wika ng iyong browser.
