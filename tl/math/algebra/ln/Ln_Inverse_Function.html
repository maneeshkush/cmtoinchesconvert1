Baliktad na Function ng ln(x)

Ano ang inverse function ng natural logarithm ng x?

Ang natural na logarithm function na ln(x) ay ang inverse function ng exponential function e x .

Kapag ang natural na logarithm function ay:

f (x) = ln(x),  x>0

 

Pagkatapos ang kabaligtaran na pag-andar ng natural na logarithm function ay ang exponential function:

f -1(x) = ex

 

Kaya ang natural na logarithm ng exponent ng x ay x:

f (f -1(x)) = ln(ex) = x

 

O kaya

f -1(f (x)) = eln(x) = x
