Ano ang isang exponent

Ang base a na itinaas sa kapangyarihan ng n ay katumbas ng pagpaparami ng a, n beses:

a n = a × a × ... × a

                    n beses

a ang base at n ang exponent.

Mga halimbawa

31 = 3

32 = 3 × 3 = 9

33 = 3 × 3 × 3 = 27

34 = 3 × 3 × 3 × 3 = 81

35 = 3 × 3 × 3 × 3 × 3 = 243

Mga panuntunan sa produkto ng exponent

Panuntunan ng produkto na may parehong base

an ⋅ am = an+m

Halimbawa:

23 ⋅ 24 = 23+4 = 27 = 2⋅2⋅2⋅2⋅2⋅2⋅2 = 128

Panuntunan ng produkto na may parehong exponent

an ⋅ bn = (a ⋅ b)n

Halimbawa:

32 ⋅ 42 = (3⋅4)2 = 122 = 12⋅12 = 144

Tingnan ang: Multiplying exponents

Exponent quotient rules

Quotient rule na may parehong base

an / am = an-m

Halimbawa:

25 / 23 = 25-3 = 22 = 2⋅2 = 4

Quotient rule na may parehong exponent

an / bn = (a / b)n

Halimbawa:

43 / 23 = (4/2)3 = 23 = 2⋅2⋅2 = 8

Mga panuntunan sa kapangyarihan ng mga exponent

Panuntunan ng kapangyarihan I

(an) m = a n⋅m

Halimbawa:

(23)2 = 23⋅2 = 26 = 2⋅2⋅2⋅2⋅2⋅2 = 64

Panuntunan ng kapangyarihan II

a nm = a (nm)

Halimbawa:

232 = 2(32) = 2(3⋅3) = 29 = 2⋅2⋅2⋅2⋅2⋅2⋅2⋅2⋅2 = 512

Pamumuno ng kapangyarihan na may mga radikal

m√(a n) = a n/m

Halimbawa:

2√(26) = 26/2 = 23 = 2⋅2⋅2 = 8

Panuntunan ng mga negatibong exponent

b-n = 1 / bn

Halimbawa:

2-3 = 1/23 = 1/(2⋅2⋅2) = 1/8 = 0.125
