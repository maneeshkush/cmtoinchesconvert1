Pinapasimple ang mga fractional exponents

Ang base b na itinaas sa kapangyarihan ng n/m ay katumbas ng:

bn/m = (m√b)n = m√(bn)

Halimbawa:

Ang base 2 na nakataas sa kapangyarihan ng 3/2 ay katumbas ng 1 na hinati ng base 2 na nakataas sa kapangyarihan ng 3:

23/2 = 2√(23) = 2.828

Pagpapasimple ng mga fraction gamit ang mga exponents

Mga fraction na may exponents:

(a / b)n = an / bn

Halimbawa:

(4/3)3 = 43 / 33 = 64 / 27 = 2.37

Mga negatibong fractional exponent

Ang base b na itinaas sa kapangyarihan ng minus n/m ay katumbas ng 1 na hinati ng base b na itinaas sa kapangyarihan ng n/m:

b-n/m = 1 / bn/m = 1 / (m√b)n

Halimbawa:

Ang base 2 na itinaas sa kapangyarihan ng minus 1/2 ay katumbas ng 1 na hinati ng base 2 na nakataas sa kapangyarihan ng 1/2:

2-1/2 = 1/21/2 = 1/√2 = 0.7071

Mga fraction na may negatibong exponent

Ang base a/b na itinaas sa kapangyarihan ng minus n ay katumbas ng 1 na hinati ng base a/b na itinaas sa kapangyarihan ng n:

(a/b)-n = 1 / (a/b)n = 1 / (an/bn) = bn/an

Halimbawa:

Ang base 2 na itinaas sa kapangyarihan ng minus 3 ay katumbas ng 1 na hinati ng base 2 na itinaas sa kapangyarihan ng 3:

(2/3)-2 = 1 / (2/3)2 = 1 / (22/32) = 32/22 = 9/4 = 2.25

Pagpaparami ng fractional exponents

Pagpaparami ng fractional exponent na may parehong fractional exponent:

a n/m ⋅ b n/m = (a ⋅ b) n/m

Halimbawa:

23/2 ⋅ 33/2 = (2⋅3)3/2 = 63/2 = √(63) = √216 = 14.7

 

Pagpaparami ng fractional exponents na may parehong base:

a n/m ⋅ a k/j = a (n/m)+(k/j)

Halimbawa:

23/2 ⋅ 24/3 = 2(3/2)+(4/3) = 7.127

 

Pagpaparami ng fractional exponents na may iba't ibang exponents at fractions:

a n/m ⋅ b k/j

Halimbawa:

23/2 ⋅ 34/3 = √(23) ⋅ 3√(34) = 2.828 ⋅ 4.327 = 12.237

Pagpaparami ng mga fraction sa mga exponents

Pagpaparami ng mga fraction na may mga exponents na may parehong base ng fraction:

(a / b) n ⋅ (a / b) m = (a / b) n+m

Halimbawa:

(4/3)3 ⋅ (4/3)2 = (4/3)3+2 = (4/3)5 = 45 / 35 = 4.214

 

Pagpaparami ng mga fraction na may mga exponent na may parehong exponent:

(a / b) n ⋅ (c / d) n = ((a / b)⋅(c / d)) n

Halimbawa:

(4/3)3 ⋅ (3/5)3 = ((4/3)⋅(3/5))3 = (4/5)3 = 0.83 = 0.8⋅0.8⋅0.8 = 0.512

 

Pagpaparami ng mga fraction na may mga exponents na may iba't ibang base at exponents:

(a / b) n ⋅ (c / d) m

Halimbawa:

(4/3)3 ⋅ (1/2)2 = 2.37 ⋅ 0.25 = 0.5925

Paghahati ng mga fractional exponent

Paghahati ng mga fractional exponent na may parehong fractional exponent:

a n/m / b n/m = (a / b) n/m

Halimbawa:

33/2 / 23/2 = (3/2)3/2 = 1.53/2 = √(1.53) = √3.375 = 1.837

 

Paghahati ng mga fractional exponents na may parehong base:

a n/m / a k/j = a (n/m)-(k/j)

Halimbawa:

23/2 / 24/3 = 2(3/2)-(4/3) = 2(1/6) = 6√2 = 1.122

 

Paghahati ng mga fractional exponents na may iba't ibang exponents at fractions:

a n/m / b k/j

Halimbawa:

23/2 / 34/3 = √(23) / 3√(34) = 2.828 / 4.327 = 0.654

Paghahati ng mga fraction sa mga exponent

Paghahati ng mga fraction sa mga exponents na may parehong base ng fraction:

(a / b)n / (a / b)m = (a / b)n-m

Halimbawa:

(4/3)3 / (4/3)2 = (4/3)3-2 = (4/3)1 = 4/3 = 1.333

 

Paghahati ng mga fraction sa mga exponent na may parehong exponent:

(a / b)n / (c / d)n = ((a / b)/(c / d))n = ((a⋅d / b⋅c))n

Halimbawa:

(4/3)3 / (3/5)3 = ((4/3)/(3/5))3 = ((4⋅5)/(3⋅3))3 = (20/9)3 = 10.97

 

Paghahati ng mga fraction sa mga exponents na may iba't ibang base at exponents:

(a / b) n / (c / d) m

Halimbawa:

(4/3)3 / (1/2)2 = 2.37 / 0.25 = 9.481

Pagdaragdag ng mga fractional exponent

Ang pagdaragdag ng mga fractional exponent ay ginagawa sa pamamagitan ng pagtataas muna ng bawat exponent at pagkatapos ay pagdaragdag ng:

an/m + bk/j

Halimbawa:

33/2 + 25/2 = √(33) + √(25) = √(27) + √(32) = 5.196 + 5.657 = 10.853

 

Pagdaragdag ng parehong mga base b at exponents n/m:

bn/m + bn/m = 2bn/m

Halimbawa:

42/3 + 42/3 = 2⋅42/3 = 2 ⋅ 3√(42) = 5.04

Pagbabawas ng mga fractional exponent

Ang pagbabawas ng mga fractional exponent ay ginagawa sa pamamagitan ng pagtataas muna ng bawat exponent at pagkatapos ay pagbabawas:

an/m - bk/j

Halimbawa:

33/2 - 25/2 = √(33) - √(25) = √(27) - √(32) = 5.196 - 5.657 = -0.488

 

Pagbabawas ng parehong mga base b at exponents n/m:

3bn/m - bn/m = 2bn/m

Halimbawa:

3⋅42/3 - 42/3 = 2⋅42/3 = 2 ⋅ 3√(42) = 5.04
