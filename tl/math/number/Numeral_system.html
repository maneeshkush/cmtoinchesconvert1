Sistema ng Numero

b - base ng sistema ng numeral

d n - ang n-th digit

n - maaaring magsimula sa negatibong numero kung ang numero ay may bahaging bahagi.

N +1 - ang bilang ng mga digit

Binary Numeral System - Base-2

Ang mga binary na numero ay gumagamit lamang ng 0 at 1 digit.

Ang B ay nagsasaad ng binary prefix.

Mga halimbawa:

101012 = 10101B = 1×24+0×23+1×22+0×21+1×20 = 16+4+1= 21

101112 = 10111B = 1×24+0×23+1×22+1×21+1×20 = 16+4+2+1= 23

1000112 = 100011B = 1×25+0×24+0×23+0×22+1×21+1×20 =32+2+1= 35

Octal Numeral System - Base-8

Ang mga Octal na numero ay gumagamit ng mga digit mula sa 0..7.

Mga halimbawa:

278 = 2×81+7×80 = 16+7 = 23

308 = 3×81+0×80 = 24

43078 = 4×83+3×82+0×81+7×80= 2247

Decimal Numeral System - Base-10

Ang mga desimal na numero ay gumagamit ng mga digit mula sa 0..9.

Ito ang mga regular na numero na ginagamit namin.

Halimbawa:

253810 = 2×103+5×102+3×101+8×100

Hexadecimal Numeral System - Base-16

Gumagamit ang mga hex na numero ng mga digit mula sa 0..9 at A..F.

Ang H ay nagsasaad ng hex prefix.

Mga halimbawa:

2816 = 28H = 2×161+8×160 = 40

2F16 = 2FH = 2×161+15×160 = 47

BC1216 = BC12H = 11×163+12×162+1×161+2×160= 48146
