Depinisyon ng sine

Sa isang kanang tatsulok na ABC ang sine ng α, sin(α) ay tinukoy bilang ang ratio sa pagitan ng gilid na katapat ng anggulo α at ang gilid na kabaligtaran sa tamang anggulo (hypotenuse):

sin α = a / c

Halimbawa

a = 3"

c = 5"

sin α = a / c = 3 / 5 = 0.6

Inverse sine function

Ang arcsine ng x ay tinukoy bilang inverse sine function ng x kapag -1≤x≤1.

Kapag ang sine ng y ay katumbas ng x:

sin y = x

Pagkatapos ang arcsine ng x ay katumbas ng inverse sine function ng x, na katumbas ng y:

arcsin x = sin-1(x) = y

Tingnan ang: Arcsin function
