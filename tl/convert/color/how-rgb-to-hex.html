Paano i-convert ang RGB sa hex na kulay

Paano i-convert mula sa kulay ng RGB hanggang sa hexadecimal na code ng kulay.

Kulay ng RGB

Ang kulay ng RGB ay kumbinasyon ng Pula, Berde at asul na mga kulay:

(R, G, B)

Pula, berde, at asul ang bawat isa ay gumagamit ng 8 bits, na may mga halaga ng integer na mula 0 hanggang 255.

Kaya ang bilang ng mga kulay na maaaring mabuo ay:

256×256×256 = 16777216 = 100000016

hex na code ng kulay

Ang hex color code ay isang 6 na digit na hexadecimal (base 16) na numero:

RRGGBB 16

Ang 2 kaliwang digit ay kumakatawan sa pulang kulay.

Ang 2 gitnang digit ay kumakatawan sa berdeng kulay.

Ang 2 kanang digit ay kumakatawan sa asul na kulay.

rgb sa hex conversion

1. I-convert ang pula, berde, at asul na mga halaga mula sa decimal patungo sa hex.
2. Pagsamahin ang 3 hex value ng pula, berde at asul na magkasama: RRGGBB.

Halimbawa 1 I-
convert ang pula (255,0,0) sa hex na color code:

R = 25510 = FF16

G = 010 = 0016

B = 010 = 0016

Kaya ang hex color code ay:

Hex = FF0000

Halimbawa #2
I- convert ang kulay na ginto (255,215,0) sa hex na color code:

R = 25510 = FF16

G = 21510 = D716

B = 010 = 0016

Kaya ang hex color code ay:

Hex = FFD700

Ano ang ginagawa nitong RGB to Hex converter?

Kinukuha ito bilang input ng pula, berde at asul na mga halaga ng kulay mula 0 hanggang 255 at pagkatapos ay iko-convert ang mga halagang iyon sa isang hexadecimal string na maaaring magamit upang tukuyin ang mga kulay sa html/css code.Ang software sa pag-edit ng larawan ay karaniwang kumakatawan sa kulay sa RGB at kaya kung gusto mong gamitin ang mga kulay na ginamit sa iyong software sa pag-edit ng larawan bilang background ng iyong HTML na elemento kailangan mong makuha ang hexadecimal na representasyon ng mga halaga ng RGB.Binibigyang-daan ka ng tool na ito na makuha ang mga halagang iyon.

Subukan ang aming bagong tool sa paghahanap ng kulay.

I-convert ang isang Hex Value sa RGB

Marahil ay nakakita ka ng hex code sa isang web page at gusto mong gamitin ang kulay na iyon sa iyong software sa pag-edit ng larawan.Sa kasong iyon, kakailanganin mo ang mga halaga ng RGB kung ang iyong software sa pag-edit ng larawan ay hindi sumusuporta sa mga halaga ng HEX.
