Pagkalkula ng GPA

Kaya ang GPA ay kinakalkula bilang weighted average ng mga grado, kapag ang bilang ng credit/oras ay ang timbang at ang numeric na grado ay kinuha mula sa talahanayan ng GPA.

Kaya Ang GPA ay katumbas ng kabuuan ng produkto ng timbang ng oras ng kredito (w) na beses sa grado (g).

GPA = w1×g1+ w2×g2+ w3×g3 + ... + wn×gn

Ang timbang ng mga oras ng kredito (w i ) ay katumbas ng mga oras ng kredito ng klase na hinati sa kabuuan ng mga oras ng kredito ng lahat ng mga klase:

wi= ci / (c1+c2+c3+...+cn)
