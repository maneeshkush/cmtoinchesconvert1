Kahulugan ng Arctangent

Ang arctangent function ay ang inverse function ng y = tan(x).

arctan(y) = tan-1(y) = x+ kπ

Sa bawat

k = {...,-2,-1,0,1,2,...}

 

Halimbawa, Kung ang tangent ng 45° ay 1:

tan(45°) = 1

Kung gayon ang arctangent ng 1 ay 45°:

arctan(1) = tan -1 (1) = 45°
