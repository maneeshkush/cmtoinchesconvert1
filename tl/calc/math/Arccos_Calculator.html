Kahulugan ng Arccos

Ang arccosine function ay ang inverse function ng cos(x).

arccos(x) = cos-1(x)

 

Halimbawa, Kung ang cosine ng 60° ay 0.5:

cos(60°) = 0.5

Pagkatapos ang arccos ng 0.5 ay 60°:

arccos(0.5) = cos -1 (0.5) = 60°
