ls -l command sa Linux

Mga listahan ng bandila ng opsyon na ls -l na may mahabang format ng listahan.

Syntax

$ ls -l [options] [file|dir]

Mga halimbawa

ls , default na listahan na may maikling format:

$ ls
Desktop   Downloads Pictures Templates Videos
Documents Music     Public   todo.txt
$
