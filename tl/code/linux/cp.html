cp command sa Linux/Unix

Ang cp ay isang Linux shell command upang kopyahin ang mga file at direktoryo.

  • cp syntax
  • mga pagpipilian sa cp

cp command syntax

Kopyahin mula sa pinagmulan hanggang sa dest

$ cp [options] source dest

mga pagpipilian sa utos ng cp

cp command pangunahing mga pagpipilian:

opsyon paglalarawan
cp -a archive ng mga file
cp -f pilitin ang pagkopya sa pamamagitan ng pag-alis ng patutunguhang file kung kinakailangan
cp -i interactive - magtanong bago mag-overwrite
cp -l mag-link ng mga file sa halip na kopyahin
cp -L sundin ang mga simbolikong link
cp -n walang file overwrite
cp -R recursive copy (kabilang ang mga nakatagong file)
cp -u update - kopyahin kapag ang pinagmulan ay mas bago kaysa sa dest
cp -v verbose - mag-print ng mga mensaheng nagbibigay-kaalaman

mga halimbawa ng utos ng cp

Kopyahin ang solong file main.c sa patutunguhang direktoryo bak :

$ cp main.c bak

 

Kopyahin ang 2 file na main.c at def.h sa direktoryo ng absolute path ng patutunguhan /home/usr/rapid/ :

$ cp main.c def.h /home/usr/rapid/

 

Kopyahin ang lahat ng C file sa kasalukuyang direktoryo sa subdirectory bak :

$ cp *.c bak

 

Kopyahin ang directory src sa absolute path directory /home/usr/rapid/ :

$ cp src /home/usr/rapid/

 

Kopyahin ang lahat ng mga file at direktoryo sa dev recursively sa subdirectory bak:

$ cp -R dev bak

 

Pilitin ang kopya ng file:

$ cp -f test.c bak

 

Interactive na prompt bago i-overwrite ang file:

$ cp -i test.c bak
cp: overwrite 'bak/test.c'? y

 

I-update ang lahat ng mga file sa kasalukuyang direktoryo - kopyahin lamang ang mas bagong mga file sa patutunguhang direktoryo bak:

$ cp -u * bak
