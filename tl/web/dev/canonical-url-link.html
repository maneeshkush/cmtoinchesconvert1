Link ng Canonical URL

Canonical na tag ng URL.Canonical na link tag.

HTML canonical link tag redirect

Hindi nagre-redirect ang canonical link sa gustong URL, ngunit maaari itong maging alternatibo sa pag-redirect ng URL para sa mga website kung saan nagmumula ang karamihan sa trapiko sa mga search engine.

Ang HTML na tag para sa mga canonical na link ay maaaring gamitin kapag mayroong maraming mga pahina na may katulad na nilalaman at gusto mong sabihin sa mga search engine kung aling pahina ang gusto mo sa mga resulta ng paghahanap.

Ang canonical link tag ay maaaring mag-link sa parehong domain at gayundin sa maramihang mga domain .

Idagdag ang canonical link tag sa lumang page para mag-link sa
bagong page.

Idagdag ang canonical link tag sa mga page na mas gusto mong hindi  i-link ang trapiko  ng search  engine  sa gustong page.

Dapat idagdag ang canonical link tag sa seksyon.

Halimbawa #1

Kapag lumipat na ang page, maaari tayong magdagdag ng canonical link sa lumang page para ipaalam sa mga search engine na maabot ang bagong page.

old-page1.html:

<!DOCTYPE html>
<html>
<head>
   ...
   <link rel="canonical" href="http://www.mydomain.com/new-page1.html">
</head>
<body>
   ...
</body>
</html>

Halimbawa #2

Kapag may ilang page na may katulad na convetnt, kung mas gusto namin ang mga search engine na maabot ang page3.html, dapat kaming magdagdag ng canonical link sa page3.html sa head section ng page1.html at page2.html.

page1.html:

<!DOCTYPE html>
<html>
<head>
   ...
   <link rel="canonical" href="http://www.mydomain.com/page3.html">
</head>
<body>
   ...
</body>
</html>

page2.html:

<!DOCTYPE html>
<html>
<head>
   ...
   <link rel="canonical" href="http://www.mydomain.com/page3.html">
</head>
<body>
   ...
</body>
</html>
