Bumaba ang trapiko sa website

Bakit bumababa ang trapiko ng aking website?

Suriin ang kalendaryo

Maaaring bawasan ng mga holiday at weekend ang iyong trapiko.

Babalik sa normal ang traffic kapag natapos na ang holiday.

Ikumpara noong nakaraang taon

Gamitin ang Google Analytics upang ipakita ang graph ng mga pagbisita noong nakaraang taon.

Suriin kung bumaba rin ang mga pagbisita noong nakaraang taon.

Bug ng Google Analytics

Ang paggamit ng lumang Google Analytics code na may urchin.js file, ay maaaring magpakita ng kamakailang 2 araw na may mas mababang trapiko kaysa sa totoong trapiko.

Hindi naman talaga down ang traffic, pero mukhang down lang.

Problema sa server

Subukang i-browse ang iyong website, kung hindi mo ito ma-access, mayroon kang problema sa web server o DNS server.

Subukang i-access ang iyong web server at tingnan kung ito ay aktibo.

Suriin ang integridad ng iyong database o mga html na file.

Gumamit ng ping test tool upang suriin ang tugon ng iyong web server.

Maghanap ng bago sa problema sa DNS server.Noong 9/2012, hindi makatugon ang website na ito kasama ang marami pang iba (tingnan ang: Na-hack ang GoDaddy ).

Bumaba ang ranggo ng mga resulta ng paghahanap sa Google

Karamihan sa trapiko sa mga website ay nagmumula sa mga search engine at ang pangunahing search engine ay Google.

Kung ang karamihan sa mga pagbisita ng iyong website ay nabuo ng isang keyword, maaari itong makuha ng kumpetisyon.

  • Hanapin ang keyword sa Google upang matukoy kung may isa pang website na kung matatagpuan sa unahan ng iyong site at nagbibigay ng mas mahusay na halaga sa user.
  • Maghanap ng balita para sa pagbabago ng algorithm ng ranking ng Google.Halimbawa, sinira ng pag-update ng Google panda ang trapiko ng maraming website.

Ang website ay pinagbawalan ng Google

Ang paggamit ng mga ipinagbabawal na paraan upang i-promote ang iyong site sa Google ay titiyakin na ang iyong website ay ipagbabawal ng Google.

Maghanap sa Google gamit ang iyong mga pangunahing keyword at tingnan kung lumilitaw ito gaya ng dati sa mga resulta ng paghahanap.

Kung hindi lalabas ang iyong website, dapat mong:

  1. Basahin ang mga gabay sa webmaster ng Google at ayusin ang iyong website.
  2. Isumite ang kahilingan sa muling pagsasaalang-alang sa Google.
