URL ng pag-redirect ng JavaScript

Paano i-redirect ang pahina ng URL gamit ang Javascript.

Ang pag-redirect ng Javascript ay hindi nagbabalik ng 301 permanenteng redirect status code.

Pag-redirect ng Javscript

Palitan ang lumang page ng redirection code ng URL ng page na gusto mong i-redirect.

old-page.html:

<!DOCTYPE html>
<html>
<body>
<script type="text/javascript">
    // Javascript URL redirection
    window.location.replace("http://www.mydomain.com/new-page.html");
</script>
</body>
</html>

 

Ginagamit ng mga search engine ang 301 status code upang ilipat ang ranggo ng pahina mula sa lumang URL patungo sa bagong URL.

Ang pag-redirect ng Javascript ay nagbabalik ng http response status code: 200 OK.

Kaya ang Javascript redirection ay hindi search engine friendly at mas mainam na gumamit ng iba pang paraan ng pag-redirect na nagbabalik ng status code: 301 Moved Permanently.
