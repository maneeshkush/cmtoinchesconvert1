Halimbawa ng pag-encode ng larawan sa base64

Input ng larawan:

Skema ng URI ng data ng imahe na may naka-encode na base64 na output ng data:

data:image/jpeg;base64,ay ang data URI scheme header.

/9j/4AAQSkZ...ay ang naka-encode na base64 data.

HTML <img> na may base64 data URI

<img src="data:image/jpeg;base64,/9j/4AAQSkZ...">

CSS background na may base64 data URI

body {
background-image: url('data:image/jpeg;base64,/9j/4AAQSkZ...');
