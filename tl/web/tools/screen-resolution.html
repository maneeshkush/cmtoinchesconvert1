Pagtukoy sa resolution ng screen gamit ang Javascript

Makukuha mo ang laki ng screen gamit ang mga katangian ng screen.width at screen.height .

Makukuha mo ang available na laki ng screen gamit ang mga katangian ng screen.availWidth at screen.availHeight .

Halimbawa:

<script>
alert(screen.width+' x '+screen.height);
</script>
