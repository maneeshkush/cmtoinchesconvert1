Kulay ng CSS

Ang color code ay maaaring isa sa:

Hex na format: #rrggbb

RGB format: rgb(pula, berde, asul)

Format ng pangalan: pangalan

Halimbawa

Para sa kulay kahel:

Hex na format: #FFA500

RGB format: rgb(255,165,0)

Format ng pangalan: orange

Pagtatakda ng kulay ng elemento

Para sa partikular na elemento:

<element style="color: code;"></element>

Para sa lahat ng elemento ng parehong uri.Ilagay ang code sa <style> tag sa head section:

<style>
   element { color: code; }
</style>

Halimbawa

Pagtatakda ng kulay ng teksto ng talata sa pula:

Pagtatakda ng kulay ng partikular na talata:

<p style="color: #FF0000;">Some text ...</p>

Tingnan:

Ilang text...

Pagtatakda ng kulay ng lahat ng talata

<style>
   p { color: #FF0000; }
</style>

Pagtatakda ng kulay ng background ng elemento

element { background: code; }

Halimbawa

Pagtatakda ng kulay ng background ng talata sa pula:

Pagtatakda ng kulay ng background ng partikular na talata:

<p style="background: #FF0000;">Some text ...</p>

Tingnan:

Ilang text...

Pagtatakda ng kulay ng background ng lahat ng talata:

<style>
   p { background: #FF0000; }
</style>

Pagtatakda ng kulay ng hangganan ng elemento

element { border-color: topcode rightcode bottomcode leftcode }

Halimbawa

Pagtatakda ng kulay ng hangganan ng talata sa pula, berde, asul at itim:

Pagtatakda ng kulay ng hangganan ng partikular na talata:

<p style="border-color: #FF0000 #00FF00 #0000FF #000000; border-style:solid">Some text ...</p>

Tingnan:

Ilang text...

Pagtatakda ng kulay ng hangganan ng lahat ng talata:

<style>
   p { border-color: #FF0000 #00FF00 #0000FF #000000; }
</style>
