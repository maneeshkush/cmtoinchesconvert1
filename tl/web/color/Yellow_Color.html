Dilaw na RGB color code

Dilaw na RGB code = 255*65536+255*256+0 = #FFFF00

RED=255, GREEN=255, BLUE=0

Dilaw na code ng kulay ng HTML

HTML na talata na may mga dilaw na font

Code:

<p style=" color:yellow; background:black">Dilaw ang mga font na ito!</p>

Resulta:

Dilaw ang mga font na ito!

O

<p style=" color:#FFFF00; background:black">Ang mga font na ito ay dilaw din!</p>

Resulta:

Ang mga font na ito ay dilaw din!

O

<p style=" color:rgb(255,255,0); background:black">Ang mga font na ito ay dilaw din!</p>

Resulta:

Ang mga font na ito ay dilaw din!

HTML na talata na may mga itim na font at kulay dilaw na background

Code:

<p style="color:black; background:yellow ">Dilaw ang kulay ng background</p>

Resulta:

Dilaw ang kulay ng background
