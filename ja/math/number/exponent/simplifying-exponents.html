指数の単純化

指数を単純化する方法。

  • 有理指数の単純化
  • 指数による分数の簡約
  • 負の指数の単純化
  • 指数による根号の簡略化

有理指数の単純化

基数 b を n/m 乗すると、次のようになります。

bn/m = (m√b)n = m√(bn)

例：

底 2 の 3/2 乗は、底 2 の 3 乗で 1 を割った値に等しくなります。

23/2 = 2√(23) = 2.828

指数による分数の簡約

指数のある分数:

(a / b)n = an / bn

例：

(4/3)3 = 43 / 33 = 64 / 27 = 2.37

負の指数の単純化

基数 b をマイナス n 乗した値は、基数 b を n 乗した値で 1 を割った値に等しくなります。

b-n = 1 / bn

例：

基数 2 のマイナス 3 乗は、基数 2 の 3 乗で 1 を割った値に等しくなります。

2-3 = 1/23 = 1/(2⋅2⋅2) = 1/8 = 0.125

指数による根号の簡略化

指数付き根号の場合:

(m√a)n = an/m

例：

(√5)4 = 54/2 = 52 = 25
