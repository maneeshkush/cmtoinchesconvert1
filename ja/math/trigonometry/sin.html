正弦定義

直角三角形 ABC では、α の正弦である sin(α) は、角 α の反対側と直角の反対側 (斜辺) の比として定義されます。

sin α = a / c

例

a = 3"

c = 5"

sin α = a / c = 3 / 5 = 0.6

逆正弦関数

xの逆正弦は、 -1≤x≤1の場合、x の逆正弦関数として定義されます。

y のサインが x に等しい場合:

sin y = x

次に、x のアークサインは、y に等しい x の逆サイン関数に等しくなります。

arcsin x = sin-1(x) = y
