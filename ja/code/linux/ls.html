Linux/Unix の ls コマンド

lsは、ファイルとディレクトリのディレクトリの内容を一覧表示する Linux シェル コマンドです。

  • ls 構文
  • ls オプション
  • ls の例
  • ls コード ジェネレーター

ls 構文

$ ls [options] [file|dir]

ls コマンドのオプション

ls コマンドの主なオプション:

オプション 説明
ls -a 「.」で始まる隠しファイルを含むすべてのファイルを一覧表示します
ls --色 カラーリスト [=常に/決して/自動]
ls -d ディレクトリの一覧表示 - ' */' を使用
ls -F */=>@| の 1 文字を追加します。エントリーへ
ls -i リストファイルの inode インデックス番号
ls -l 長い形式のリスト - 権限を表示
ls-la 隠しファイルを含む長い形式のリスト
ls -lh 読み取り可能なファイルサイズのロングフォーマットをリストする
ls -ls ファイルサイズのある長い形式のリスト
ls-r 逆順に並べる
ls -R ディレクトリツリーを再帰的にリストする
ls -s リスト ファイル サイズ
ls -S ファイルサイズで並べ替え
ls -t 時間と日付で並べ替え
ls-X 拡張機能名で並べ替え

ls コマンドの例

タブボタンを押すと、ファイルまたはフォルダーの名前を自動補完できます。

ディレクトリDocuments/Booksを相対パスで一覧表示:

$ ls Documents/Books

 

ディレクトリ/home/user/Documents/Booksを絶対パスで一覧表示します。

$ ls /home/user/Documents/Books

 

ルート ディレクトリを一覧表示します。

$ ls /

 

親ディレクトリを一覧表示:

$ ls ..

 

ユーザーのホーム ディレクトリを一覧表示します (例: /home/user):

$ ls ~

 

長い形式のリスト:

$ ls -l

 

隠しファイルを表示:

$ ls -a

 

長い形式でリストし、隠しファイルを表示します。

$ ls -la

 

日付/時刻で並べ替え:

$ ls -t

 

ファイルサイズで並べ替え:

$ ls -S

 

すべてのサブディレクトリを一覧表示します。

$ ls *

 

再帰的なディレクトリ ツリー リスト:

$ ls -R

 

ワイルドカードを含むテキスト ファイルのみを一覧表示します。

$ ls *.txt

 

出力ファイルへの ls リダイレクト:

$ ls > out.txt

 

ディレクトリのみを一覧表示します。

$ ls -d */

 

ファイルとディレクトリをフル パスで一覧表示します。

$ ls -d $PWD/*

ls コード ジェネレーター

lsオプションを選択し、[コードの生成] ボタンを押します。

オプション 
  長いリスト形式 (-l)
  すべてのファイル/隠しファイルを一覧表示する (-a)
  ディレクトリ ツリーを再帰的に一覧表示する (-R)
  逆順リスト (-r)
  フルパス付きリスト (-d $PWD/*)
