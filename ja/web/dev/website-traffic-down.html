ウェブサイトのトラフィックがダウン

Web サイトのトラフィックが減少しているのはなぜですか?

カレンダーをチェック

休日と週末は、トラフィックが減少する可能性があります。

休業日が明けると渋滞は通常に戻ります。

去年と比べて

Google アナリティクスを使用して、昨年の訪問数グラフを表示します。

訪問数が 1 年前にも減少したかどうかを確認します。

Google アナリティクスのバグ

urchin.js ファイルで古い Google アナリティクス コードを使用すると、実際のトラフィックよりも少ないトラフィックで最近 2 日間が表示される場合があります。

トラフィックは実際にはダウンしていませんが、ダウンしているように見えるだけです。

サーバーの問題

Web サイトを閲覧してみてください。アクセスできない場合は、Web サーバーまたは DNS サーバーに問題があります。

Web サーバーにアクセスして、アクティブかどうかを確認してください。

データベースまたは html ファイルの整合性を確認してください。

ping テスト ツールを使用して、Web サーバーの応答を確認します。

DNS サーバーの問題で新しいものを検索します。2012 年 9 月、他の多くの Web サイトと同様に、この Web サイトは応答できませんでした (参照: GoDaddy がハッキングされました)。

Google 検索結果のランキングが下がった

ほとんどの Web サイト トラフィックは検索エンジンから来ており、主要な検索エンジンは Google です。

Web サイトへのアクセスのほとんどが 1 つのキーワードによって生成されている場合、競合他社に奪われる可能性があります。

  • Google でキーワードを検索して、あなたのサイトより先に位置し、ユーザーにより良い価値を提供する別の Web サイトがあるかどうかを判断します。
  • Google ランキング アルゴリズムの変更に関するニュースを検索します。たとえば、 Google パンダの更新により、多くの Web サイトのトラフィックが損なわれました。

Googleによって禁止されたウェブサイト

禁止されている方法を使用して Google でサイトを宣伝すると、そのサイトは Google によって確実に禁止されます。

主なキーワードで Google を検索し、検索結果に通常どおり表示されるかどうかを確認します。

Web サイトがまったく表示されない場合は、次のことを行う必要があります。

  1. Google ウェブマスター向けガイドラインを読み、ウェブサイトを修正してください。
  2. Google に再審査リクエストを送信 します。
