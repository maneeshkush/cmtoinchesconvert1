PHP リダイレクト

ページから URL への PHP リダイレクト。PHP 301 リダイレクト。

この PHP リダイレクトは、HTTP 応答ステータス コードを返す必要があります: 301 Moved Permanently.

検索エンジンは、301 応答ステータス コードを使用して、ページ ランクを古い URL から新しい URL に移します。

PHP ヘッダー リダイレクト

old-page.php コードを new-page.php へのリダイレクト コードに置き換えます。

古いページ.php:

<?php
// PHP permanent URL redirection
header("Location: http://www.domain.com/new-page.php", true, 301);
exit();
?>

古いページには .php ファイル拡張子が必要です。

新しいページには任意の拡張子を付けることができます。

PHP リダイレクトの例

例 #1

php-リダイレクト-test.php

<?php
// PHP permanent URL redirection test
header("Location: https://www.cmtoinchesconvert.com/ja/web/dev/php-redirect.html", true, 301);
exit();
?>

 

このリンクを押すと、php-redirect-test.php からこのページにリダイレクトされます。

 

PHP リダイレクト テスト - PHP ファイル

例 #2

php-リダイレクト-test.htm

<?php
// PHP permanent URL redirection test
header("Location: https://www.cmtoinchesconvert.com/ja/web/dev/php-redirect.html", true, 301);
exit();
?>

 

HTML ファイル php-redirect-test.htm からの PHP リダイレクトは、通常、.htaccess または httpd.conf ファイルで有効にされていない限り、.html ファイル拡張子のために機能しません。

 

PHP リダイレクト テスト - HTML ファイル

 

HTML ファイルで PHP を有効にするには、次のコードを.htaccess または httpd.conf ファイルに追加します。

Addtype application/x-httpd-php .htm .html
