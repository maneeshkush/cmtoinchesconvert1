CSS カラー

カラー コードは次のいずれかです。

16 進形式: #rrggbb

RGB フォーマット: rgb (赤、緑、青)

名前の形式: 名前

例

オレンジ色の場合:

16 進形式: #FFA500

RGB フォーマット: rgb(255,165,0)

名前の形式: オレンジ

要素の色を設定する

特定の要素の場合:

<element style="color: code;"></element>

同じタイプのすべての要素。head セクションの <style> タグにコードを挿入します。

<style>
   element { color: code; }
</style>

例

段落テキストの色を赤に設定:

特定の段落の色の設定:

<p style="color: #FF0000;">Some text ...</p>

意見：

いくつかのテキスト...

すべての段落の色を設定する

<style>
   p { color: #FF0000; }
</style>

要素の背景色の設定

element { background: code; }

例

段落の背景色を赤に設定:

特定の段落の背景色の設定:

<p style="background: #FF0000;">Some text ...</p>

意見：

いくつかのテキスト...

すべての段落の背景色の設定:

<style>
   p { background: #FF0000; }
</style>

要素の境界色の設定

element { border-color: topcode rightcode bottomcode leftcode }

例

段落の境界線の色を赤、緑、青、黒に設定:

特定の段落の境界線の色を設定する:

<p style="border-color: #FF0000 #00FF00 #0000FF #000000; border-style:solid">Some text ...</p>

意見：

いくつかのテキスト...

すべての段落の境界線の色を設定する:

<style>
   p { border-color: #FF0000 #00FF00 #0000FF #000000; }
</style>
