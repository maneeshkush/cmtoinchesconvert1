RGBを16進数に変換する方法

RGB カラーから 16 進数のカラーコードに変換する方法。

RGB カラー

RGB カラーは、赤、緑、青の色の組み合わせです。

（R、G、B）

赤、緑、青はそれぞれ 8 ビットを使用し、整数値は 0 から 255 の範囲です。

したがって、生成できる色の数は次のとおりです。

256×256×256 = 16777216 = 100000016

16 進数の色コード

16 進数のカラー コードは、6 桁の 16 進数 (基数 16) の数字です。

RRGGBB 16

左の 2 桁は赤色を表します。

真ん中の 2 桁は緑色を表します。

右 2 桁は青色を表します。

RGBから16進への変換

1. 赤、緑、青の値を 10 進数から 16 進数に変換します。
2. 赤、緑、青の 3 つの 16 進値を組み合わせます: RRGGBB。

例 1
赤 (255,0,0) を 16 進数のカラー コードに変換します。

R = 25510 = FF16

G = 010 = 0016

B = 010 = 0016

したがって、16 進数のカラー コードは次のとおりです。

Hex = FF0000

例 2
金色 (255,215,0) を 16 進数のカラー コードに変換します。

R = 25510 = FF16

G = 21510 = D716

B = 010 = 0016

したがって、16 進数のカラー コードは次のとおりです。

Hex = FFD700

この RGB から Hex へのコンバーターは何をしますか?

0 ～ 255 の範囲の赤、緑、青の色の値を入力として受け取り、それらの値を 16 進数の文字列に変換します。この文字列は、html/css コードで色を指定するために使用できます。写真編集ソフトウェアは通常 RGB で色を表現するため、写真編集ソフトウェアで使用される色を HTML 要素の背景として使用する場合は、RGB 値の 16 進数表現を取得する必要があります。このツールを使用すると、これらの値を取得できます。

新しいカラー検索ツールをお試しください。

16 進数値を RGB に変換する

Web ページで 16 進コードを見て、その色を写真編集ソフトウェアで使用したいと思うかもしれません。その場合、写真編集ソフトウェアが HEX 値をサポートしていない場合は、RGB 値が必要になります。
