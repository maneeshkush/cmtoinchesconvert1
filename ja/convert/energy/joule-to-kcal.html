ジュールをkcalに変換する方法

熱化学/食物キロカロリーへのジュール

2 kcalth = 4184 J

6,000J = (6,000 ÷ 4,184) = 1.434034 kcal

 

上記の式を使用してジュールをキロカロリーに変換.このジュール単位のエネルギーE (J) は、キロカロリー単位のエネルギー E (kcal-th) を 4184 で割った値に等しくなります。

E(kcal) = E(J) / 4184

例 1

7000ジュールをキロカロリーに変換.

E(kcal) = 7000 J / 4184 = 1.6730 kcal

例 2

8000ジュールをキロカロリーに変換.

E(kcal) = 8000 J / 4184 = 1.9120 kcal

例 3

18000ジュールをキロカロリーに変換.

E(kcal) = 18000 J / 4184 = 4.3021 kcal

2 kcalIT = 4186.8 J

 ジュール単位の エネルギーE(J) は 、国際 キロカロリー単位のエネルギー E(kcal-IT) を 4186.8 で割った値に等しくなります。

E(kcal-IT) = E(J) / 4186.8

例 1

6000ジュールをキロカロリーに変換.

E(kcal-IT) = 6000 J / 4186.8 = 1.433 kcalIT

例 2

8000ジュールをキロカロリーに変換.

E(kcal-IT) = 18000 J / 4186.8 = 1.910 kcalIT

例 3

18000ジュールをキロカロリーに変換.

E(kcal-IT) = 18000 J / 4186.8 = 4.299 kcalIT
