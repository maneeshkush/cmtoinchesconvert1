BTU/hr を kW に変換する方法

1 時間あたりの英国熱量単位 (BTU/hr) を キロワット (kW)に変換する方法。

1 BTU/hr は 0.00029307107 キロワットに相当します。

1 BTU/hr = 0.00029307107 kW

BTU/hr から キロワットへの換算式

キロワット 単位の電力P (kW) は、  1 時間あたり の BTU IT単位の電力P ( BTU /hr ) を 3412.142 で割った値に等しくなります。

P(kW) = P(BTU/hr) / 3412.142

例 1

10000 BTU/hr をキロワットに変換します。

P(kW) = 10000 BTU/hr / 3412.142 = 2.930kW

例 2

30000 BTU/hr をキロワットに変換します。

P(kW) = 30000 BTU/hr / 3412.142 = 8.792kW

例 3

40000 BTU/hr をキロワットに変換します。

P(kW) = 40000 BTU/hr / 3412.142 = 11.722kW

例 4

60000 BTU/hr をキロワットに変換します。

P(kW) = 60000 BTU/hr / 3412.142 = 17.584kW

例 5

110000 BTU/hr をキロワットに変換します。

P(kW) = 110000 BTU/hr / 3412.142 = 32.237kW
