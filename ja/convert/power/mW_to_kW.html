ミリワットをキロワットに変換する方法

したがって、キロワットP (kW) 単位の電力は、ミリワット P (mW ) 単位の電力を 100 万で割った値に等しくなります。

P(kW) = P(mW) / 1000000

それで

1mW = 0.000001kW

5mW = 0.000005kW

例 1

200 ミリワットをキロワットに変換します。

P(kW) = 200mW / 1000000 = 0.0002kW

例 2

400 ミリワットをキロワットに変換します。

P(kW) = 400mW / 1000000 = 0.0004kW

例 3

3000 ミリワットをキロワットに変換します。

P(kW) = 3000mW / 1000000 = 0.003kW

例 4

5000 ミリワットをキロワットに変換します。

P(kW) = 5000mW / 1000000 = 0.005kW

例 5

7000 ミリワットをキロワットに変換します。

P(kW) = 7000mW / 1000000 = 0.007kW
