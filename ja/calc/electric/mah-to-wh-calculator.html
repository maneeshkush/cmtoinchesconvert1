ミリアンペア時からワット時への計算

したがって、ワット時で表したエネルギーE (Wh)は、ミリアンペア時で表した  電荷 Q (mAh) にボルト (V) で表した電圧V (V) を [1000] で割った値に等しくなります。

E(Wh) = Q(mAh) × V(V) / 1000

したがって、ワット時は、ミリアンペア時×ボルトを 1000 で割った値に等しくなります。

watt-hours = milliamp-hours × volts / 1000

また

Wh = mAh × V / 1000

例: 1

100Mah と 5V のバッテリーのワット時間を計算しますか?

解決：

ワット時 = ( 100 × 5 ) / 1000
= 0.5Wh

例: 2

500Mah と 5V のバッテリーのワット時間を計算しますか?

解決：

ワット時 = ( 500 × 5 ) / 1000
= 2.5Wh

例: 3

5000Mah と 5V のバッテリーのワット時間を計算しますか?

解決：

= 25Wh

例: 4

10000Mah と 5V のバッテリーのワット時間を計算しますか?

解決：

ワット時 = ( 10000 × 5 ) / 1000
= 50Wh
