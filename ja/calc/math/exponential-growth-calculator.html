指数関数的成長/減衰計算機

オンラインの指数関数的成長/減衰計算機。

指数関数的成長/減衰式

x(t) = x0 × (1 + r) t

x (t) は時刻 t における値です。

x 0は時刻 t=0 での初期値です。

r は、r>0 の場合は成長率、r<0 の場合は減衰率 (%) です。

t は、離散間隔および選択された時間単位の時間です。

指数関数的成長計算機

初期値を入力 ( x 0 ):  
成長率/減衰率 ( r ) を入力してください: %
時間 ( t ) を入力してください:  
   
時間 t の値 ( x (t)):  

例 1

x0 = 40

r = 3% = 0.04

t = 80 hours

x(t) = x0 × (1 + r) t = 40×(1+0.03)80 = 425.63

例 2

x0 = 40

r = 5% = 0.05

t = 80 hours

x(t) = x0 × (1 + r) t = 40×(1+0.05)80 = 1982

例 3

x0 = 60

r = 3% = 0.05

t = 80 hours

x(t) = x0 × (1 + r) t = 60×(1+0.05)80 = 2973
