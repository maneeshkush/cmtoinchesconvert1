期末試験の成績計算

したがって、最終試験の成績は、必要な成績に 100% を掛け、100% から最終試験の重み (w) を引いて現在の成績 (g) を掛け、最終試験の重み (w) で割った値に等しくなります。

Final exam grade =

=  ( 100%×required grade - (100% - w)×current grade ) / w

現在のグレードは 70% (または C-) です。

最終試験の重みは 50% です。

必要なグレードは 80% (または B-) です。

Final exam grade =

= ( 100%×required grade - (100% - w)×current grade ) / w

= ( 100%×80% - (100% - 50%)×70% ) / 50% = 90%

したがって、最終試験の成績は 90% (または A-) である必要があります。
