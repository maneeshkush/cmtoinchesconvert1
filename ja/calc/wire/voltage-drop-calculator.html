電圧降下の計算

直流・単相計算

ボルト (V) で表した電圧降下 V は、アンペア (A) で表したワイヤー電流 I に、フィート (ft) で表した一方向のワイヤーの長さ L を 2 倍し、1000 フィートあたりのワイヤー抵抗 R をオーム (Ω/kft) で割った値に等しい1000までに：

Vdrop (V) = Iwire (A) × Rwire(Ω)

= Iワイヤー(A) × (2 × L (フィート) × Rワイヤー(Ω/kft) / 1000 (ft/kft) )

ボルト (V) で表した電圧降下 V は、アンペア (A) で表したワイヤー電流 I に、メートル (m) で表した一方向のワイヤーの長さ L を 2 倍し、1000 メートルあたりのワイヤー抵抗 R をオーム (Ω/km) で割った値に等しい1000までに：

Vdrop (V) = Iwire (A) × Rwire(Ω)

＝Ｉ線（Ａ） ×（２×Ｌ（ｍ） ×Ｒ線（Ω／ｋｍ）／１０００（ｍ／ｋｍ））

三相計算

ボルト (V) 単位の線間電圧降下 V は、アンペア (A) 単位のワイヤ電流 I の 3 倍の平方根に等しい(Ω/kft) を 1000 で割った値:

Vdrop (V) = √3 × Iwire (A) × Rwire (Ω)

= 1.732 × Iワイヤ(A) × ( L (ft) × Rワイヤ (Ω/kft) / 1000 (ft/kft) )

ボルト (V) 単位の線間電圧降下 V は、アンペア (A) 単位のワイヤ電流 I の 3 倍の平方根に等しい(Ω/km) を 1000 で割った値:

Vdrop (V) = √3 × Iwire (A) × Rwire (Ω)

= 1.732 × Iワイヤー(A) × ( L (m) × Rワイヤー(Ω/km) / 1000 (m/km) )

線径の計算

したがって、n ゲージ ワイヤの直径 d n (インチ) は、0.005 インチ x 92 の 36 乗 - ゲージ数 n を 39 で割った値に等しくなります。

dn (in) = 0.005 in × 92(36-n)/39

したがって、ミリメートル (mm) 単位の n ゲージ ワイヤの直径 d nは、0.127mm に 92 を掛けて、36 の累乗からゲージ数 n を引いて、39 で割った値に等しくなります。

dn (mm) = 0.127 mm × 92(36-n)/39

ワイヤー断面積の計算

キロサーキュラー ミル (kcmil)単位の n ゲージ ワイヤの断面積 A nは、インチ (インチ) 単位の平方ワイヤ直径 d の 1000 倍に等しくなります。

An (kcmil) = 1000×dn2 = 0.025 in2 × 92(36-n)/19.5

そのため、n ゲージ ワイヤの断面断面積 A n (平方インチ (インチ2 )) は、平方インチ ワイヤ直径 d (インチ (インチ)) の 4 倍で割った pi に等しくなります。

An (in2) = (π/4)×dn2 = 0.000019635 in2 × 92(36-n)/19.5

したがって、n ゲージ ワイヤの断面積 A n (平方ミリメートル (mm 2 )) は、平方ミリメートル (mm) 単位のワイヤ直径 d の 4 倍で割った pi に等しくなります。

An (mm2) = (π/4)×dn2 = 0.012668 mm2 × 92(36-n)/19.5

配線抵抗の計算

したがって、キロフィートあたりのオーム (Ω/kft) 単位の n ゲージ ワイヤ抵抗 R は、ワイヤの抵抗率 ρ (オームメートル単位(Ω·m)) を 0.3048 × 1000000000 倍し、25.4 で割った値に等しくなります。断面積 An (平方インチ) の2倍(2で):

Rn (Ω/kft) = 0.3048 × 109 × ρ(Ω·m) / (25.42 × An (in2))

したがって、キロメーターあたりのオーム (Ω/km) 単位の n ゲージ ワイヤ抵抗 R は、オームメートル (Ω・m) 単位のワイヤの抵抗率 ρ を 1000000000 倍し、平方ミリメートル (mm2 ) 単位の断面積 A nで割った値に等しくなります。

Rn (Ω/km) = 109 × ρ(Ω·m) / An (mm2)

電圧降下計算機の機能

当社の電圧降下計算機を使用すると、ユーザーは電圧降下を計算できます。このユーティリティの優れた機能のいくつかを以下に説明します。

登録なし

電圧降下計算機を使用するために登録プロセスを行う必要はありません。このユーティリティを使用すると、ユーザーは無料で何度でも電圧降下を計算できます。

高速変換

この電圧降下計算機は、ユーザーに最速の計算を提供します。ユーザーが入力フィールドに電圧降下値を入力して [計算] ボタンをクリックすると、ユーティリティは変換プロセスを開始し、すぐに結果を返します。

時間と労力を節約

Calculator Voltage Drop の手動手順は簡単な作業ではありません。このタスクを完了するには、多くの時間と労力を費やす必要があります。電圧降下計算機を使用すると、同じタスクをすぐに完了することができます。自動化されたアルゴリズムが作業を行うため、手動の手順に従う必要はありません。

正確さ

手計算に時間と労力を費やしても、正確な結果を得ることができない場合があります。誰もが数学の問題を解くのが得意というわけではありません。プロだと思っていても、正確な結果が得られる可能性は十分にあります。この状況は、電圧降下計算機を使用してスマートに処理できます。このオンライン ツールにより、100% 正確な結果が提供されます。

互換性

オンラインの電圧降下コンバーターは、すべてのオペレーティング システムで完全に動作します。Mac、iOS、Android、Windows、または Linux デバイスのいずれであっても、このオンライン ツールを手間をかけずに簡単に使用できます。

100% 無料

この電圧降下計算機を使用するために登録プロセスを行う必要はありません。このユーティリティは無料で使用でき、無制限に電圧降下計算を行うことができます。
