음수의 로그

음수의 로그는 무엇입니까?

대수 함수

y = logb(x)

지수 함수의 역함수

x = by

밑수 b가 양수(b>0)이므로, 밑수 b는 y의 거듭제곱으로 모든 실수 y에 대해 양수(by >0)여야 합니다 .따라서 숫자 x는 양수(x>0)여야 합니다.

음수의 실수 밑수 b 로그는 정의되지 않습니다.

logb(x) is undefined for x ≤ 0

예를 들어, 밑이 10인 -5의 로그는 정의되지 않습니다.

log10(-5) is undefined

 Log z = ln r + iθ
