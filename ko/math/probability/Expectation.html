기대 가치

확률과 통계에서 기대 값 또는 기대값 은 임의 변수의 가중 평균값입니다.

연속 확률 변수의 기대

E ( X )는 연속 확률 변수 X 의 기대값입니다.

x 는 연속 확률 변수 X 의 값입니다.

P ( x )는 확률 밀도 함수입니다.

불연속 확률 변수의 기대치

E ( X )는 연속 확률 변수 X 의 기대값입니다.

x 는 연속 확률 변수 X 의 값입니다.

P ( x )는 X 의 확률 질량 함수입니다.

기대의 속성

선형성

a가 상수이고 X,Y가 랜덤 변수인 경우:

E(aX) = aE(X)

E(X+Y) = E(X) + E(Y)

끊임없는

c가 상수일 때:

E(c) = c

제품

X와 Y가 독립 랜덤 변수인 경우:

E(X ⋅Y) = E(X) ⋅ E(Y)
