탄젠트 정의

직각 삼각형 ABC에서 α의 탄젠트, tan(α)는 각도 α의 반대쪽 변과 각도 α에 인접한 변 사이의 비율로 정의됩니다.

tan α = a / b

예

a = 3"

b = 4"

tan α = a / b = 3 / 4 = 0.75

역탄젠트 함수

x 의 아크탄젠트 는 x가 실수일 때(x ∈ℝ ) x의 역 탄젠트 함수로 정의됩니다.

y의 탄젠트가 x와 같을 때:

tan y = x

그런 다음 x의 아크탄젠트는 y와 같은 x의 역탄젠트 함수와 같습니다.

arctan x = tan-1 x = y

예

arctan 1 = tan-1 1 = π/4 rad = 45°

참조: Arctan 함수
