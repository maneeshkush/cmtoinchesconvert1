코사인 정의

직각 삼각형 ABC에서 α의 사인, sin(α)는 각도 α에 인접한 변과 직각의 반대 변(빗변) 사이의 비율로 정의됩니다.

cos α = b / c

예

b = 3"

c = 5"

cos α = b / c = 3 / 5 = 0.6

역코사인 함수

x 의 아크코사인 은 -1≤x≤1일 때 x의 역코사인 함수로 정의됩니다.

y의 코사인이 x와 같을 때:

cos y = x

그런 다음 x의 아크코사인은 x의 역코사인 함수와 같으며 이는 y와 같습니다.

arccos x = cos-1 x = y

예

arccos 1 = cos-1 1 = 0 rad = 0°

참조: Arccos 함수
