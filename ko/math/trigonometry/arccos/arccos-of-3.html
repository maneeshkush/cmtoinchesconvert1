3의 arccos는 무엇입니까?

3의 아크코사인은 무엇입니까?

arccos 3 = ?

실제 arccos 기능

아크코사인은 역코사인 함수입니다.

코사인 함수는 -1에서 1까지의 출력값을 가지므로,

arccosine 함수는 -1에서 1까지의 입력 값을 가집니다.

따라서 arccos x는 x=3에 대해 정의되지 않습니다.

arccos 3 is undefined

복잡한 arccos 함수

x = arccos(3)

cos(x) = cos(arccos(3))

cos(x) = 3

오일러 공식에서

cos(x) = (eix + e-ix) / 2

(eix + e-ix) / 2 = 3

eix + e-ix = 6

eix 로 곱하기

e2 ix + 1 = 6eix

y = eix

우리는 이차 방정식을 얻습니다.

y2 - 6 y + 1 = 0

y1,2 = (6 ± √32)/2

y1 = 5.828427 = eix

y2 = 0.171573 = eix

양쪽에 ln을 적용하면 arccos(3)에 대한 솔루션이 제공됩니다.

x1 = ln(5.828427) / i

x2 = ln(0.171573) / i
