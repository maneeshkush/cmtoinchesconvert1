사인 정의

직각 삼각형 ABC에서 α의 사인, sin(α)는 각도 α의 반대쪽 변과 직각 반대쪽 변(빗변) 사이의 비율로 정의됩니다.

sin α = a / c

예

a = 3"

c = 5"

sin α = a / c = 3 / 5 = 0.6

역사인 함수

x 의 아크사인 은 -1≤x≤1일 때 x의 사인 역함수로 정의됩니다.

y의 사인이 x와 같을 때:

sin y = x

그런 다음 x의 아크사인은 x의 사인 역함수와 같고 이는 y와 같습니다.

arcsin x = sin-1(x) = y

참조: Arcsin 함수
