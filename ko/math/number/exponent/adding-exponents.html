지수로 숫자 더하기

지수 추가는 먼저 각 지수를 계산한 다음 다음을 추가하여 수행됩니다.

an + bm

예:

42 + 25 = 4⋅4+2⋅2⋅2⋅2⋅2 = 16+32 = 48

같은 밑수 b와 지수 n을 더하기:

bn + bn = 2bn

예:

42 + 42 = 2⋅42 = 2⋅4⋅4 = 32

음수 지수 추가

음의 지수를 추가하려면 먼저 각 지수를 계산한 다음 다음을 추가하면 됩니다.

a-n + b-m = 1/an + 1/bm

예:

4-2 + 2-5 = 1/42 + 1/25 = 1/(4⋅4)+1/(2⋅2⋅2⋅2⋅2) = 1/16+1/32 = 0.09375

분수 지수 추가

분수 지수를 더하는 것은 먼저 각 지수를 올린 다음 다음을 더함으로써 이루어집니다.

an/m + bk/j

예:

33/2 + 25/2 = √(33) + √(25) = √(27) + √(32) = 5.196 + 5.657 = 10.853

 

같은 밑수 b와 지수 n/m을 더하기:

bn/m + bn/m = 2bn/m

예:

42/3 + 42/3 = 2⋅42/3 = 2 ⋅ 3√(42) = 5.04

지수로 변수 추가

지수 추가는 먼저 각 지수를 계산한 다음 다음을 추가하여 수행됩니다.

xn + xm

지수가 같은 경우:

xn + xn = 2xn

예:

x2 + x2 = 2x2
