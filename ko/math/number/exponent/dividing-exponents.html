밑이 같은 지수 나누기

밑이 같은 지수의 경우 지수를 빼야 합니다.

a n / a m = a n-m

예:

26 / 23 = 26-3 = 23 = 2⋅2⋅2 = 8

밑이 다른 지수 나누기

밑이 다르고 a와 b의 지수가 같으면 먼저 a와 b를 나눌 수 있습니다.

a n / b n = (a / b) n

예:

63 / 23 = (6/2)3 = 33 = 3⋅3⋅3 = 27

 

밑과 지수가 다른 경우 각 지수를 계산한 다음 나누어야 합니다.

a n / b m

예:

62 / 33 = 36 / 27 = 1.333

음수 지수 나누기

밑이 같은 지수의 경우 지수를 뺄 수 있습니다.

a-n / a-m = a-n-(-m) = am-n

예:

2-3 / 2-5 = 25-3 = 22 = 2⋅2 = 4

 

밑이 다르고 a와 b의 지수가 같으면 a와 b를 먼저 곱할 수 있습니다.

a-n / b-n = (a/b)-n = 1 / (a/b)n = (b/a)n

예:

3-2 / 4-2 = (4/3)2 = 1.7778

 

밑과 지수가 다른 경우 각 지수를 계산한 다음 나누어야 합니다.

a-n / b-m = bm / an

예:

3-2 / 4-3 = 43 / 32 = 64 / 9 = 7.111

지수로 분수 나누기

분수 밑이 같은 지수로 분수 나누기:

(a / b)n / (a / b)m = (a / b)n-m

예:

(4/3)3 / (4/3)2 = (4/3)3-2 = (4/3)1 = 4/3 = 1.333

 

지수가 같은 지수로 분수 나누기:

(a / b)n / (c / d)n = ((a / b)/(c / d))n = ((a⋅d / b⋅c))n

예:

(4/3)3 / (3/5)3 = ((4/3)/(3/5))3 = ((4⋅5)/(3⋅3))3 = (20/9)3 = 10.97

 

밑과 지수가 다른 지수로 분수 나누기:

(a / b) n / (c / d) m

예:

(4/3)3 / (1/2)2 = 2.37 / 0.25 = 9.481

분수 지수 나누기

동일한 분수 지수로 분수 지수 나누기:

a n/m / b n/m = (a / b) n/m

예:

33/2 / 23/2 = (3/2)3/2 = 1.53/2 = √(1.53) = √3.375 = 1.837

 

밑이 같은 분수 지수 나누기:

a n/m / a k/j = a (n/m)-(k/j)

예:

23/2 / 24/3 = 2(3/2)-(4/3) = 2(1/6) = 6√2 = 1.122

 

지수와 분수가 다른 분수 지수 나누기:

a n/m / b k/j

예:

23/2 / 24/3 = √(23) / 3√(24) = 2.828 / 2.52 = 1.1222

지수로 변수 나누기

밑이 같은 지수의 경우 지수를 뺄 수 있습니다.

xn / xm = xn-m

예:

x5 / x3 = (x⋅x⋅x⋅x⋅x) / (x⋅x⋅x) = x5-3 = x2
