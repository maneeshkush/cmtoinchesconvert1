Facebook에 Infinity 기호를 입력하는 방법

Facebook에 무한대 기호 텍스트 입력.

  1. 아래의 무한대 기호( ∞ )를 마우스로 선택합니다.
  2. 아래 무한대 기호를 복사합니다. Ctrl + C 키보드 키를 누르거나 마우스 오른쪽 버튼을 클릭하고 복사 메뉴 항목을 선택합니다.
  3. Facebook 페이지에 붙여넣기: Ctrl + V 를 누르 거나 마우스 오른쪽 버튼을 클릭하고 붙여넣기 메뉴 항목을 선택합니다.
