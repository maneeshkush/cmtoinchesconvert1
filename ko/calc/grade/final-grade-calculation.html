기말고사 성적 산정

따라서 기말시험 성적은 요구 성적의 100% 곱하기 - 100% - 기말시험 가중치(w) 곱하기 현재 성적(g)을 기말시험 가중치(w)로 나눈 값입니다.

Final exam grade =

=  ( 100%×required grade - (100% - w)×current grade ) / w

예

현재 등급은 70%(또는 C-)입니다.

기말고사 비중은 50%입니다.

필수 등급은 80%(또는 B-)입니다.

계산

따라서 기말시험 성적은 요구 성적의 100% 곱하기 - 100% - 기말시험 가중치(w) 곱하기 현재 성적(g)을 기말시험 가중치(w)로 나눈 값입니다.

Final exam grade =

= ( 100%×required grade - (100% - w)×current grade ) / w

= ( 100%×80% - (100% - 50%)×70% ) / 50% = 90%

따라서 최종 시험 성적은 90%(또는 A-)여야 합니다.
