루멘을 칸델라로 변환하는 방법

루멘 단위의 광속(lm)을 칸델라 단위의 광도(cd)로 변환하는 방법.

계산할 수는 있지만 칸델라와 루멘은 같은 양을 나타내지 않기 때문에 루멘을 칸델라로 변환할 수는 없습니다.

루멘에서 칸델라로 계산

균일한 등방성 광원의 경우  칸델라(cd) 단위의 광도Iv 는 루멘(lm) 단위의 광속 Φv 와 같습니다  .

스테라디안 단위의 입체각 Ω 으로 나눈 값  (sr):

Iv(cd) = Φv(lm) / Ω(sr)

 

따라서 스테라디안(sr) 단위의 입체각 Ω  은 2 곱하기 파이 곱하기 1에서 꼭지각  θ 의 절반 (도(°))의 코사인을 뺀 것과 같습니다.

Ω(sr) = 2π(1 - cos(θ/2))

 

따라서 칸델라(cd) 단위의 광도 I v 는 루멘(lm) 단위의 광속 Φ v  와 같습니다.

2 곱하기 파이 곱하기 1에서 꼭지각  θ 의 절반 을 도(°)로 나눈 코사인을 뺀 값입니다.

Iv(cd) = Φv(lm) / ( 2π(1 - cos(θ/2)) )

그래서

candela = lumens / ( 2π(1 - cos(degrees/2)) )

또는

cd = lm / ( 2π(1 - cos(°/2)) )

예 1

 루멘(lm) 단위 의 광속 Φv  가 340lm이고 꼭지각이 60° 일 때광속  I v 를 칸델라 단위(cd)로 구하십시오.

Iv(cd) = 340 lm / ( 2π(1 - cos(60°/2)) ) = 403.9 cd

예 2

 루멘(lm) 단위 의 광속 Φv  가 360lm이고 꼭지각이 60° 일 때광속  I v 를 칸델라 단위(cd)로 구하십시오.

Iv(cd) = 360 lm / ( 2π(1 - cos(60°/2)) ) = 427.6 cd

예 3

 루멘(lm) 단위 의 광속 Φv  가 380lm이고 꼭지각이 60° 일 때광속  I v 를 칸델라 단위(cd)로 구하십시오.

Iv(cd) = 380 lm / ( 2π(1 - cos(60°/2)) ) = 451.4 cd

예 4

 루멘(lm) 의 광속 Φv  가 440lm이고 꼭지각이 60° 일 때광속  I v 를 칸델라(cd)로 구하십시오.

Iv(cd) = 440 lm / ( 2π(1 - cos(60°/2)) ) = 522.6 cd

실시예 5

 루멘(lm) 의 광속 Φv  가 540lm이고 꼭지각이 60° 일 때광속  I v 를 칸델라(cd) 단위로 구하십시오.

Iv(cd) = 540 lm / ( 2π(1 - cos(60°/2)) ) = 641.4 cd
