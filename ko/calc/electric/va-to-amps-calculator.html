VA에서 암페어로 계산기

볼트 암페어(VA)에서 암페어(A) 로 계산기 및 계산 방법.

위상 번호, 피상 전력(volt-amps), 전압 ( volts )을 입력 하고 Calculate (계산 ) 버튼을 누릅니다.

전류 를 암페어 단위로 얻으려면 :

단계 번호 입력:  
볼트 암페어 입력: 버지니아
선간 전압 입력: V
   
암페어 결과: ㅏ

암페어에서 VA으로 계산기 ►

암페어 계산 공식에 단상 VA

따라서 암페어 단위의 전류 I는 볼트 암페어 단위의 피상 전력 S를 볼트 단위의 전압 V로 나눈 값과 같습니다.

I(A) = S(VA) / V(V)

예 1

질문: 피상 전력이 3000VA이고 전압 공급 장치가 120볼트일 때 전류는 암페어로 얼마입니까?

해결책:

I = 3000VA / 120V = 25A

예 2

질문: 피상 전력이 3000VA이고 전압 공급 장치가 130볼트일 때 전류는 암페어로 얼마입니까?

해결책:

I = 3000VA / 130V = 23.07A

예 3

질문: 피상 전력이 3000VA이고 전압 공급 장치가 150볼트일 때 전류는 암페어로 얼마입니까?

해결책:

I = 3000VA / 150V = 20A

3상 kVA - 암페어 계산 공식

따라서 암페어 단위의 전류 I는 볼트 암페어 단위의 피상 전력 S에 [1000]을 곱하고 선간 전압 VL -L (볼트 단위)의 3배의 제곱근으로 나눈 값 입니다.

I(A) = S(VA) / (√3 × VL-L(V) ) = S(VA) / (3 × VL-N(V) )

예 1

질문: 피상 전력이 3000VA이고 전압 공급 장치가 120볼트일 때 전류는 암페어로 얼마입니까?

해결책:

I = 3000VA / (√3 × 120V) = 14.433A

예 2

질문: 피상 전력이 3000VA이고 전압 공급 장치가 130볼트일 때 전류는 암페어로 얼마입니까?

해결책:

I = 3000VA / (√3 × 130V) = 13.323A

예 3

질문: 피상 전력이 3000VA이고 전압 공급 장치가 150볼트일 때 전류는 암페어로 얼마입니까?

해결책:

I = 3000VA / (√3 × 150V) = 11.547A

VA에서 암페어로의 변환 기능

VA에서 암페어로의 변환을 통해 사용자는 VA에서 암페어로 계산할 수 있습니다.이 유틸리티의 주요 기능 중 일부는 아래에 설명되어 있습니다.

등록하지 않음

VA to amps 변환기를 사용하기 위해 등록 절차를 거칠 필요가 없습니다.이 유틸리티를 사용하면 원하는 만큼 무료로 VA를 암페어로 변환할 수 있습니다.

빠른 변환

이 VA to amps 계산기는 사용자에게 가장 빠른 변환을 제공합니다.사용자가 입력 필드에 VA to amps 값을 입력하고 변환 버튼을 클릭하면 유틸리티가 변환 프로세스를 시작하고 결과를 즉시 반환합니다.

시간과 노력 절약

계산기 VA에서 암페어로의 수동 절차는 쉬운 작업이 아닙니다.이 작업을 완료하려면 많은 시간과 노력을 들여야 합니다.VA to amps 계산기를 사용하면 동일한 작업을 즉시 완료할 수 있습니다.자동 알고리즘이 작업을 수행하므로 수동 절차를 따르라는 요청을 받지 않습니다.

정확성

수동 계산에 시간과 노력을 투자하더라도 정확한 결과를 얻지 못할 수 있습니다.모든 사람이 수학 문제를 잘 푸는 것은 아닙니다. 자신이 전문가라고 생각하더라도 정확한 결과를 얻을 수 있는 좋은 기회가 여전히 있습니다.이 상황은 VA to amps 계산기를 사용하여 현명하게 처리할 수 있습니다.이 온라인 도구는 100% 정확한 결과를 제공합니다.

호환성

온라인 VA-암페어 변환기는 모든 운영 체제에서 완벽하게 작동합니다.Mac, iOS, Android, Windows 또는 Linux 장치가 있더라도 번거로움 없이 이 온라인 도구를 쉽게 사용할 수 있습니다.

100% 무료

이 VA를 앰프 계산기에 사용하기 위해 등록 절차를 거칠 필요가 없습니다.이 유틸리티를 무료로 사용할 수 있으며 제한 없이 무제한 VA에서 암페어로 변환할 수 있습니다.
