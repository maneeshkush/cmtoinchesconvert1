DC 회로 규칙

옴의 법칙

I = V / R

주울의 법칙

P = V · I = I 2 · R = V 2 / R

직렬 회로 규칙

VT = V1 + V2 + V3 + ...

IT = I1 = I2 = I3 = ...

RT = R1 + R2 + R3 + ...

1/CT = 1/C1 + 1/C2 + 1/C3 + ...

LT = L1 + L2 + L3 + ...

병렬 회로 규칙

VT = V1 = V2 = V3 = ...

IT = I1 + I2 + I3 + ...

1/RT = 1/R1 + 1/R2 + 1/R3 + ...

CT = C1 + C2 + C3 + ...

1/LT = 1/L1 + 1/L2 + 1/L3 + ...

전압 분할

V1 = VT ⋅ R1 / (R1+R2+R3+...)

현재 부서

I1 = IT ⋅ (R2+R3+...) / (R1+R2+R3+...)

Kirchhoff의 전압 법칙(KVL)

전류 루프에서 전압 강하의 합은 0입니다.

∑ Vi = 0

키르히호프의 현행법칙(KCL)

여러 회로 요소 사이의 접합을 노드라고 합니다.

노드에서 전류 값의 합은 0입니다.

∑ 나는 나는 = 0

정전 용량

C = 큐 / V

병렬 플레이트 커패시터

C = ε ⋅ A / l

ε는 미터당 패럿(F/m) 단위의 유전율입니다.

유전율

ε = ε0 ⋅ εr

ε 0 은 진공에서의 유전율입니다.

ε r 은 상대 유전율 또는 변증법 상수입니다.

커패시터의 전류

IC(t) = C d VC(t) / dt

커패시터의 전압

VC ( t ) = VC( 0) + 1/C∫IC ( t)⋅dt

인덕터의 전압

VL(t) = L d IL(t) / dt

인덕터의 현재

IL (t) = IL (0) + 1 / L ∫ V L (t)⋅ dt

축전기의 에너지

WC = C⋅V 2 / 2

인덕터의 에너지 

WL = L⋅I 2 / 2
