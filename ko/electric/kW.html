킬로와트 정의

5킬로와트(kW)는 5000와트(W)와 같습니다.

5kW = 5000W

1킬로와트는 1초 동안 1000줄의 에너지 소비로 정의됩니다.

1kW = 1000J / 1s

1킬로와트는 1000000밀리와트와 같습니다.

1kW = 1000000mW

5킬로와트는 5000000밀리와트와 같습니다.

5kW = 5000000mW

킬로와트 예시

예 #1

40000 줄의 에너지가 10초 동안 방출되었을 때 소비 전력은 kW로 얼마입니까?

해결책:

P(kW) = 40000J / 10s / 1000 = 4kW

예 #2

3시간 동안 6kWh를 소비할 때 소비 전력은 얼마입니까?

해결책:

P(kW) = 6kWh / 3h = 2kW
