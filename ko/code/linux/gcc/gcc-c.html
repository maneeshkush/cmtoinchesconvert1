gcc -c 옵션 플래그

gcc -c 링크 없이 소스 파일을 컴파일합니다.

통사론

$ gcc -c [options] [source files]

예

소스 파일 myfile.c 쓰기 :

// myfile.c
#include <stdio.h>
 
void main()
{
    printf("Program run\n");
}

 

myfile.c 컴파일:

$ gcc -c myfile.c

이 컴파일은 myfile.o 개체 파일을 생성했습니다.
