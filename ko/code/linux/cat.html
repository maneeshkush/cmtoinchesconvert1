Linux/Unix의 cat 명령

리눅스 고양이 명령.

cat  명령은 텍스트 파일의 내용을 표시하고 여러 파일을 하나의 파일로 결합하는 데 사용됩니다.

cat 명령은 디렉토리를 허용하지 않습니다.

고양이 명령 구문

$ cat [options] file1 [file2...]

고양이 명령 옵션

cat 명령 기본 옵션:

옵션 설명
고양이 -b 비어 있지 않은 줄에 줄 번호 추가
고양이 -n 모든 줄에 줄 번호 추가
고양이 -s 빈 줄을 한 줄로 짜기
고양이 -E 줄 끝에 $ 표시
고양이 -T 탭 대신 ^I 표시

고양이 명령 예

텍스트 파일 데이터 보기:

$ cat list1.txt
milk
bread
apples

$ cat list2.txt
house
car

$

 

2개의 텍스트 파일 결합:

$ cat list1.txt list2.txt
milk
bread
apples

house
car

$

 

2개의 텍스트 파일을 다른 파일에 결합:

$ cat list1.txt list2.txt > todo.txt
$
