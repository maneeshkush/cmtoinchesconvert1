CSS 색상

색상 코드는 다음 중 하나일 수 있습니다.

16진수 형식: #rrggbb

RGB 형식: rgb(빨강, 녹색, 파랑)

이름 형식: 이름

예

주황색의 경우:

16진수 형식: #FFA500

RGB 형식: rgb(255,165,0)

이름 형식: 주황색

요소 색상 설정

특정 요소:

<element style="color: code;"></element>

동일한 유형의 모든 요소에 대해.헤드 섹션의 <style> 태그에 코드를 입력합니다.

<style>
   element { color: code; }
</style>

예

단락 텍스트 색상을 빨간색으로 설정:

특정 단락의 색상 설정:

<p style="color: #FF0000;">Some text ...</p>

보다:

일부 텍스트 ...

모든 단락의 색상 설정

<style>
   p { color: #FF0000; }
</style>

요소 배경색 설정

element { background: code; }

예

단락 배경색을 빨간색으로 설정:

특정 단락의 배경색 설정:

<p style="background: #FF0000;">Some text ...</p>

보다:

일부 텍스트 ...

모든 단락의 배경색 설정:

<style>
   p { background: #FF0000; }
</style>

요소 테두리 색상 설정

element { border-color: topcode rightcode bottomcode leftcode }

예

단락 테두리 색상을 빨강, 녹색, 파랑 및 검정으로 설정:

특정 단락의 테두리 색상 설정:

<p style="border-color: #FF0000 #00FF00 #0000FF #000000; border-style:solid">Some text ...</p>

보다:

일부 텍스트 ...

모든 단락의 테두리 색상 설정:

<style>
   p { border-color: #FF0000 #00FF00 #0000FF #000000; }
</style>
