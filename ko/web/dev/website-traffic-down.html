웹사이트 트래픽 감소

내 웹사이트 트래픽이 감소하는 이유는 무엇입니까?

캘린더 확인

공휴일과 주말에는 트래픽이 감소할 수 있습니다.

휴일이 끝나면 트래픽이 정상으로 돌아갑니다.

작년과 비교

Google Analytics를 사용하여 작년 방문 그래프를 표시합니다.

방문 횟수가 1년 전에도 떨어졌는지 확인하세요.

구글 애널리틱스 버그

urchin.js 파일과 함께 이전 Google Analytics 코드를 사용하면 실제 트래픽보다 트래픽이 적은 최근 2일을 표시할 수 있습니다.

트래픽이 실제로 다운된 것은 아니지만 다운된 것처럼 보일 뿐입니다.

서버 문제

웹사이트를 탐색해 보십시오. 액세스할 수 없으면 웹 서버 또는 DNS 서버에 문제가 있는 것입니다.

웹 서버에 액세스하여 활성 상태인지 확인하십시오.

데이터베이스 또는 html 파일의 무결성을 확인하십시오.

ping 테스트 도구를 사용하여 웹 서버 응답을 확인하십시오.

DNS 서버 문제에서 새 항목을 검색합니다.2012년 9월에 다른 많은 웹사이트와 함께 이 웹사이트에서 응답할 수 없었습니다(참조: GoDaddy 해킹 됨).

구글 검색 결과 순위 하락

대부분의 웹사이트 트래픽은 검색 엔진에서 발생하며 주요 검색 엔진은 Google입니다.

대부분의 웹사이트 방문이 단일 키워드에 의해 생성된 경우 경쟁업체가 이를 차지할 수 있습니다.

  • Google에서 키워드를 검색하여 귀하의 사이트 앞에 위치할 경우 사용자에게 더 나은 가치를 제공하는 다른 웹사이트가 있는지 확인하십시오.
  • Google 순위 알고리즘 변경에 대한 뉴스를 검색합니다.예를 들어 Google Panda 업데이트는 많은 웹사이트의 트래픽을 손상시켰습니다.

Google에서 금지한 웹사이트

Google에서 귀하의 사이트를 홍보하기 위해 금지된 방법을 사용하면 귀하의 웹사이트가 Google에 의해 금지됩니다.

주요 키워드로 Google을 검색하고 검색 결과에 평소와 같이 나타나는지 확인하십시오.

웹사이트가 전혀 나타나지 않으면 다음을 수행해야 합니다.

  1. Google 웹마스터 가이드라인 을 읽고 웹사이트를 수정하세요.
  2. Google에 재검토 요청 을 제출 하세요.
