HTML 표준 링크 태그 리디렉션

표준 링크는 기본 URL로 리디렉션되지 않지만 대부분의 트래픽이 검색 엔진에서 오는 웹 사이트의 경우 URL 리디렉션의 대안이 될 수 있습니다.

표준 링크용 HTML 태그는 유사한 콘텐츠가 있는 여러 페이지가 있고 검색 결과에서 선호하는 페이지를 검색 엔진에 알리고 싶을 때 사용할 수 있습니다.

표준 링크 태그는 동일한 도메인 및 여러 도메인에 연결할 수 있습니다 .

이전 페이지에 표준 링크 태그를 추가하여
새 페이지에 연결합니다.

 검색 엔진  트래픽을 원하는 페이지 에 연결 하지 않으려는 페이지에 표준 링크 태그를 추가합니다  .

표준 링크 태그는 섹션에 추가해야 합니다.

예 #1

페이지가 이동되면 이전 페이지에 표준 링크를 추가하여 새 페이지에 도달하도록 검색 엔진에 알릴 수 있습니다.

old-page1.html:

<!DOCTYPE html>
<html>
<head>
   ...
   <link rel="canonical" href="http://www.mydomain.com/new-page1.html">
</head>
<body>
   ...
</body>
</html>

예 #2

유사한 convent가 있는 여러 페이지가 있는 경우 검색 엔진이 page3.html에 도달하도록 선호하는 경우 page1.html 및 page2.html의 헤드 섹션에 page3.html에 대한 표준 링크를 추가해야 합니다.

page1.html:

<!DOCTYPE html>
<html>
<head>
   ...
   <link rel="canonical" href="http://www.mydomain.com/page3.html">
</head>
<body>
   ...
</body>
</html>

page2.html:

<!DOCTYPE html>
<html>
<head>
   ...
   <link rel="canonical" href="http://www.mydomain.com/page3.html">
</head>
<body>
   ...
</body>
</html>
