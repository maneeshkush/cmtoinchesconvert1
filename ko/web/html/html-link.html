HTML 버튼 링크

자바스크립트 없이:

<form action="link/html-button-link.htm">
    <input type="submit" value="A button link">
</form>

자바스크립트로:

<input type="button" value="A button link" onclick="window.location.href='link/html-button-link.htm'">
