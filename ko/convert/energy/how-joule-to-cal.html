줄을 칼로리로 변환하는 방법

줄(J) 단위의 에너지를 칼로리(cal)로 변환하는 방법.

작은 칼로리와 큰 칼로리

소칼로리(cal)는  1기압에서 물 1g을 1°C 올리는 데 필요한 에너지입니다.

큰칼로리(cal)는  물 1kg을 섭씨 1도 올려 1기압까지 올리는 데 필요한 에너지다.

큰칼로리는 식품칼로리라고도 하며 식품에너지의 단위로 사용된다.

줄에서 칼로리로 변환하는 방법

줄에서 열화학 칼로리으로

1 calth = 4.184 J

열화학적 칼로리  의 에너지는 줄  단위의 에너지를 다음으로 나눈 값 과 같습니다.E(calth)E(J) 4.184:

E(cal) = E(J) / 4.184

예 1

500줄을 열화학 칼로리로 변환합니다.

E(cal) = 500J / 4.184 = 119.5 calth

예 2

700줄을 열화학 칼로리로 변환합니다.

E(cal) = 700J / 4.184 = 167.3 calth

예 3

900줄을 열화학적 칼로리로 변환합니다.

E(cal) = 900J / 4.184 = 215.1 calth

줄 ~ 15°C 칼로리

1 cal15 = 4.1855 J

15°C 칼로리  E (cal15)  의 에너지는 줄  E (J) 단위의 에너지를  4.1855로 나눈 값과 같습니다.

E(cal15) = E(J) / 4.1855

예 1

500줄을 15°C 칼로리로 변환합니다.

E(cal15) = 500J / 4.1855 = 119.460 cal15

예 2

700줄을 15°C 칼로리로 변환합니다.

E(cal15) = 700J / 4.1855 = 167.244 cal15

예 3

900줄을 15°C 칼로리로 변환합니다.

E(cal15) = 900J / 4.1855 = 215.028 cal15

줄에서 대형/음식 칼로리로

1 Cal = 4.184 kJ = 4184 J

대형/식품 칼로리  E (Cal)  의 에너지는 줄  E (J) 단위의 에너지를  4184로 나눈 값과 같습니다.

E(Cal) = E(J) / 4184

예 1

500줄을 식품 칼로리로 변환합니다.

E(Cal) = 500J / 4184 = 0.1195 Cal

예 2

700줄을 식품 칼로리로 변환합니다.

E(Cal) = 700J / 4184 = 0.1673 Cal

예 3

900줄을 식품 칼로리로 변환합니다.

E(Cal) = 900J / 4184 = 0.2151 Cal
