화씨

화씨는 온도 측정 단위입니다.

1기압에서 물의 어는점/녹는점은 약 32 °F입니다.

화씨 온도의 기호는 °F입니다.

화씨에서 섭씨로 변환

따라서 화씨 0도는 섭씨 -17.77778도입니다.

0 °F = -17.77778 °C

따라서섭씨(°C) 단위  의 온도  T  는 화씨(°F) 단위의 온도 T에서 32를 뺀 값 [5/9]입니다.

T(°C) = (T(°F) - 32) × 5/9

예 1

화씨 70도를 섭씨로 변환:

T(°C) = (70°F - 32) × 5/9 = 21.1111 °C

예 2

화씨 80도를 섭씨로 변환:

T(°C) = (80°F - 32) × 5/9 = 26.6667 °C

예 3

화씨 100도를 섭씨로 변환:

T(°C) = (100°F - 32) × 5/9 = 37.7778 °C

화씨에서 랭킨으로 변환기

따라서 화씨 0도는 랭킨[459.67]도와 같습니다.

0 °F = 459.67 °R

 랭킨(°R) 단위의 온도 T  는 화씨(°F) 단위의 온도 T에 [459.67]을 더한 것과 같습니다 .

T(°R) = T(°F) + 459.67

예 1

화씨 70도를 랭킨도로 변환:

T(°R) = 70°F + 459.67 = 529.67 °R

예 2

화씨 90도를 랭킨도로 변환:

T(°R) = 90°F + 459.67 = 549.67 °R

예 3

화씨 100도를 랭킨도로 변환:

T(°R) = 100°F + 459.67 = 559.67 °R

화씨에서 켈빈으로 변환기

따라서 켈빈 온도 T  (  K) 는 화씨 온도  T 에 459.67을 곱한 [5/9]와 같습니다.

T(K) = (T(°F) + 459.67)× 5/9

예 1

화씨 65도를 켈빈도로 변환:

T(K) = (65°F + 459.67)× 5/9 = 291.483 K

예 2

화씨 80도를 켈빈도로 변환:

T(K) = (80°F + 459.67)× 5/9 = 299.817 K

예 3

화씨 100도를 켈빈도로 변환:

T(K) = (100°F + 459.67)× 5/9 = 310.928 K
