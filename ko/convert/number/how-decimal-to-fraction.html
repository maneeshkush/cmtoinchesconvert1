십진수를 분수로 변환하는 방법

변환 단계

  1. 소수점 이하 자릿수(분자)와 10의 거듭제곱(분모)으로 소수점 이하 자릿수를 적습니다.
  2. 분자와 분모의 최대 공약수(gcd)를 구합니다.
  3. 분자와 분모를 gcd로 나누어 분수를 줄입니다.

예 #1

0.35를 분수로 변환:

0.35 = 35/100

따라서 분자와 분모의 최대 공약수(gcd)를 찾으십시오.

gcd(35,100) = 5

따라서 분자와 분모를 gcd로 나누어 분수를 줄입니다.

0.35 = (35/5) / (100/5) = 7/20

예 #2

2.58을 분수로 변환:

2.58 = 2+58/100

따라서 분자와 분모의 최대 공약수(gcd)를 찾으십시오.

gcd(58,100) = 2

따라서 분자와 분모를 gcd로 나누어 분수를 줄입니다.

2+58/100 = 2 + (58/2) / (100/2) = 2+29/50

예 #3

0.126을 분수로 변환:

0.126 = 126/1000

분자와 분모의 최대 공약수(gcd)를 구합니다.

gcd(126,1000) = 2

분자와 분모를 gcd로 나누어 분수를 줄입니다.

0.126 = (126/2)/(1000/2) = 63/500

반복되는 소수를 분수로 변환하는 방법

예 #1

0.333333...을 분수로 변환:

x = 0.333333...

10x = 3.333333...

10x - x = 9x = 3

x = 3/9 = 1/3

예 #2

0.0565656...을 분수로 변환:

x = 0.0565656...

100 x = 5.6565656...

100 x -  x = 99 x = 5.6

990 x = 56

x = 56/990 = 28/495
