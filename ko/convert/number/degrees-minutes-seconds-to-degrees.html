도, 분, 초를 십진수로 변환하는 방법

 따라서 1도는 [60]분과 같고 [3600]초와 같습니다.

1° = 60' = 3600"

 따라서 1분은 [1/60]도와 같습니다.

1' = (1/60)° = 0.01666667°

 따라서 1초는 [1/3600]도와 같습니다.

1" = (1/3600)° = 2.77778e-4° = 0.000277778°

d 정수도 m분 및 s초 각도의 경우.

d° m' s"

십진법 dd는 같습니다.

dd = d + m/60 + s/3600

예 1

35도 20분 50초 각도를 십진수 도로 변환:

35° 20' 50"

십진법 dd는 다음과 같습니다.

dd = d + m/60 + s/3600 = 35° + 20'/60 + 50"/3600 = 35.34722223°
