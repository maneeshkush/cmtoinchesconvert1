밀리와트를 와트로 변환하는 방법

전력 을 밀리와트(mW)에서 와트(W) 로 변환하는 방법.

1와트는 1000밀리와트와 같습니다.

1W = 1000mW

밀리와트에서 와트로의 변환 공식

와트  P (W) 단위  의 전력은 밀리와트  P (mW) 단위의 전력 을 [1000]으로 나눈 값과 같습니다.

P(W) = P(mW) / 1000

예 1

1mW를 와트로 변환:

P(W) = 1mW / 1000 = 0.001W

예 2

2mW를 와트로 변환:

P(W) = 2mW / 1000 = 0.002W

예 3

4mW를 와트로 변환:

P(W) = 4mW / 1000 = 0.004W

예 4

6mW를 와트로 변환:

P(W) = 6mW / 1000 = 0.006W

실시예 5

8mW를 와트로 변환:

P(W) = 8mW / 1000 = 0.008W

실시예 6

9mW를 와트로 변환:

P(W) = 9mW / 1000 = 0.009W
