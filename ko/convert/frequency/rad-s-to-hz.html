라드/s에서 헤르츠로 공식

헤르츠(Hz) 단위의 주파수  f  는 초당 라디안(rad/s) 단위의 각 주파수 또는 각 속도 ω를 2π로 나눈 값과 같습니다.

f(Hz) =  ω(rad/s) / 2π

예 1

200rad/s의 각속도에서 헤르츠 단위로 주파수를 계산합니다.

f(Hz) = 200rad/s / 2π = 31.83 Hz

예 2

400 rad/s의 각속도에서 헤르츠 단위로 주파수를 계산합니다.

f(Hz) = 400rad/s / 2π = 63.66 Hz

예 3

1000rad/s의 각속도에서 헤르츠 단위로 주파수를 계산합니다.

f(Hz) = 1000rad/s / 2π = 159.15 Hz

예 4

5000rad/s의 각속도에서 헤르츠 단위로 주파수를 계산합니다.

f(Hz) = 5000rad/s / 2π = 795.77 Hz
