16진수를 RGB 색상으로 변환하는 방법

16진수 색상 코드에서 RGB 색상으로 변환하는 방법.

16진수 색상 코드
16진수 색상 코드는 6자리 16진수(기본 16) 숫자입니다.

RRGGBB 16

2개의 왼쪽 숫자는 빨간색을 나타냅니다.

2개의 중간 숫자는 녹색을 나타냅니다.

2개의 오른쪽 숫자는 파란색을 나타냅니다.

RGB 색상

RGB 색상은 빨강, 녹색 및 파랑 색상의 조합입니다.

(R, G, B)

빨강, 초록, 파랑은 각각 0에서 255까지의 정수 값으로 8비트를 사용합니다.

따라서 생성할 수 있는 색상의 수는 다음과 같습니다.

256×256×256 = 16777216 = 1000000 16

16진수를 rgb로 변환

1. 16 진수 색상 코드 왼쪽 2 자리를 취하여 십진수 값으로 변환하여 빨간색 수준을 얻습니다.
2. 16진수 색상 코드의 중간 2자리를 가져와 십진수 값으로 변환하여 녹색 수준을 얻습니다.
3. 16진수 색상 코드의 올바른 2자리를 찾고 10진수 값으로 변환하여 파란색 수준을 얻습니다.

예 1
빨간색 16진수 색상 코드 FF0000을 RGB 색상으로 변환:

16진수 = FF0000

따라서 RGB 색상은 다음과 같습니다.

R = FF16 = 25510

G = 0016 = 010

B = 0016 = 010

또는

RGB = (255, 0, 0)

예 #2
Gold Hex 색상 코드 FFD700을 RGB 색상으로 변환:

Hex = FFD700

따라서 RGB 색상은 다음과 같습니다.

R = FF16 = 25510

G = D716 = 21510

B = 0016 = 010

또는

RGB = (255, 215, 0)
