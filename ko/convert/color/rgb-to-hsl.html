RGB에서 HSL로의 변환 공식

R , G , B 값을 255로 나누어 범위를 0..255에서 0..1로 변경합니다.

R' = R/255

G' = G/255

B' = B/255

Cmax = max(R', G', B')

Cmin = min(R', G', B')

Δ = Cmax - Cmin

RGB to HSL 색상 변환기 도구의 기능

  1. RGB 값을 HSL 값으로 변환: 이 도구를 사용하면 RGB 값(빨강, 녹색, 파랑)을 입력하고 해당 HSL 값(색조, 채도, 밝기)으로 변환할 수 있습니다.

  2. HSL 값을 RGB 값으로 변환: 이 도구를 사용하면 사용자가 HSL 값을 입력하고 해당 RGB 값으로 변환할 수도 있습니다.

  3. 사용자 정의 색상 입력: 사용자는 자신의 RGB 또는 HSL 값을 입력하여 다른 형식으로 변환할 수 있습니다.

  4. 색상 선택기: 일부 RGB-HSL 색상 변환기 도구에는 색상 선택기 기능이 포함되어 있어 사용자가 시각적 팔레트에서 색상을 선택하거나 RGB 또는 HSL 값에 대한 슬라이더를 조정하여 색상을 선택할 수 있습니다.

  5. 결과 색상 미리보기: 도구는 변환 후 결과 색상의 미리보기를 표시해야 사용자가 색상이 어떻게 보이는지 확인할 수 있습니다.

  6. 다중 색상 변환: 일부 도구를 사용하면 여러 값 세트를 입력하거나 색상 견본 또는 팔레트를 사용하여 사용자가 여러 색상을 한 번에 변환할 수 있습니다.

  7. 색상 라이브러리 또는 팔레트: 일부 도구에는 사용자가 선택하거나 참조로 사용할 수 있는 미리 정의된 색상의 라이브러리 또는 팔레트가 포함될 수 있습니다.

  8. 반응형 디자인: 도구는 반응형이어야 하며 데스크톱 컴퓨터, 태블릿 및 스마트폰과 같은 다양한 장치에서 잘 작동해야 합니다.

  9. HSL 색상 공간 시각화: 일부 도구에는 HSL 색상 공간의 시각화가 포함되어 있어 사용자가 다양한 HSL 값이 다양한 색상에 어떻게 대응하는지 이해하는 데 도움이 될 수 있습니다.

  10. HSL 값을 백분율 또는 각도로 조정하는 옵션: 일부 도구에서는 사용자가 기본 설정에 따라 HSL 값을 백분율 또는 각도로 입력할 수 있습니다.
