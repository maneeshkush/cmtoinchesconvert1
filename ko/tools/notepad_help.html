메모장 도움말

텍스트를 저장하는 방법은 2가지가 있습니다.

  1. 텍스트는 메모장 탭을 닫을 때마다 브라우저의 로컬 캐시에 저장됩니다.메모장 페이지에 다시 입력하면 텍스트가 다시 나타납니다.
  2. 저장 버튼을 누르면 텍스트가 하드 드라이브에 저장/백업됩니다.하드 드라이브에서 텍스트를 다시 열려면 열기 버튼을 누르고 생성한 텍스트 파일을 선택합니다.

구성 매개변수


중요한 정보

  • 이전 세션의 텍스트가 누락된 경우 :
    • 텍스트가 있으면 Ctrl+C로 선택하여 복사하고 Ctrl+V로 메모장 페이지에 붙여넣습니다.

    • 자동 저장 작업으로 생성된 업데이트된 텍스트 파일의 텍스트를 다운로드 폴더(있는 경우)에 복사합니다.
    • 이전에 사용한 것과 동일한 컴퓨터를 보고 있는지 확인합니다.
    • 브라우저 쿠키 및 기록을 활성화합니다.
    • 브라우저의 비공개/시크릿 모드를 사용하지 마세요.브라우저 창을 닫으면 텍스트 로컬 저장소가 브라우저에서 삭제됩니다.
    • 브라우저의 주소 표시줄에 있는 URL에서 www를 추가/제거해 보십시오.
  • 메모장의 텍스트 는시크릿/비공개 모드 브라우징 으로 저장되지 않습니다 !!!
  • 검색 기록/캐시를 삭제하거나 디스크 정리 응용 프로그램(예: Windows 디스크 정리 / CCleaner)을 실행하면저장된 Notpad의 텍스트 가 삭제될 수 있습니다 !!!
  • 파일 열기 버튼이 작동하지 않는 경우 페이지를 새로고침한 후 다시 시도하십시오.
  • 업데이트된 브라우저 버전으로 메모장 을 사용 하십시오. Internet Explorer 를 사용하는 경우 최신 브라우저(예: Chrome/Edge/Firefox )에 텍스트를 복사합니다.
  • 메모장의 텍스트는 브라우저의 로컬 캐시에 자동으로 저장됩니다(안전하지 않음).
  • 메모장의 텍스트는 보기 > 기본 설정 메뉴의 자동 저장 기간에 따라 하드 드라이브에 자동 저장(백업)될 수 있습니다 .
  • 저장 버튼이나 파일 > 저장 메뉴를 사용하여 메모장의 텍스트를 하드 드라이브에 백업 할 수 있습니다 .
  • Mac 의 경우Ctrl 키 대신 ⌘ Command 를 사용 하십시오.
  • 저장된 파일을 열려면 다운로드 폴더에서 파일을 찾으십시오.
  • 배경 라인이 스크롤되지 않으면 라인을 숨깁니다. 보기 > 기본 설정 > 텍스트 라인 메뉴를 선택 취소합니다.
  • 저장 버튼 또는 메뉴 파일 > 저장 파일을 다운로드 폴더에 저장합니다.참조: 파일을 다운로드하면 어디로 가나요?
  • 텍스트 줄 이 보이지 않을 경우 크롬 브라우저를 이용해주세요.
  • 맞춤법 검사가 작동하지 않으면 브라우저의 설정>언어 섹션에서 활성화해 보십시오.정의되지 않은 경우브라우저의 언어 설정에서 영어(미국) 를 설정해 볼 수도 있습니다.
