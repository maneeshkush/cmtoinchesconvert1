mv в Linux/Unix

Команда Linux mv.

Команда mv використовується для переміщення файлів і каталогів.

синтаксис команди mv

$ mv [options] source dest

параметри команди mv

Основні параметри команди mv:

варіант опис
mv -f примусово перемістити, перезаписавши цільовий файл без підказки
мв -і інтерактивна підказка перед перезаписом
мв -у оновити - перемістити, коли джерело є новішим за призначення
мв -в verbose - друкувати файли джерела та призначення
людина мв довідковий посібник

приклади команд mv

Перемістіть файли main.c def.h докаталогу /home/usr/rapid/ :

$ mv main.c def.h /home/usr/rapid/

 

Перемістіть усі файли C у поточному каталозі до підкаталогу bak:

$ mv *.c bak

 

Перемістіть усі файли в підкаталозі bak до поточного каталогу:

$ mv bak/* .

 

Перейменуйте файл main.c на main.bak:

$ mv main.c main.bak

 

Перейменувати каталог bak на bak2:

$ mv bak bak2

 

Оновити - перемістити, коли main.c новіший:

$ mv -u main.c bak
$

 

Перемістіть main.c і запитайте перед перезаписом bak/main.c:

$ mv -v main.c bak
'bak/main.c' -> 'bak/main.c'
$
