символ нескінченності

Символ нескінченності - це математичний символ, який представляє нескінченно велике число.

Символ нескінченності записується символом Lemniscate:

∞

Він являє собою нескінченно позитивне велике число.

Коли ми хочемо записати нескінченно від’ємне число, ми повинні написати:

-∞

Коли ми хочемо написати нескінченно мале число, ми повинні написати:

1/∞

Чи є нескінченність дійсним числом?

Нескінченність - це не число.Він представляє не конкретне число, а нескінченно велику величину.

Правила та властивості нескінченності

Ім'я Тип ключа
Позитивна нескінченність ∞
Негативна нескінченність -∞
Нескінченна різниця ∞ - ∞ не визначено
Нульовий продукт 0 ⋅ ∞ не визначено
Коефіцієнт нескінченності ∞ / ∞ не визначено
Сума дійсних чисел x + ∞ = ∞, для x ∈ℝ
Добуток додатних чисел x ⋅ ∞ = ∞, для x >0
