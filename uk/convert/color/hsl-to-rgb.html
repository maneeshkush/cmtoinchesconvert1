Формула перетворення HSL в RGB

Коли 0 ≤ H < 360, 0 ≤ S ≤ 1 і 0 ≤ L ≤ 1:

C = (1 - |2L - 1|) × S

X = C × (1 - |(H / 60°) mod 2 - 1|)

m = L - C/2

(R,G,B) = ((R'+m)×255, (G'+m)×255,(B'+m)×255)

Перетворення кольорів HSL в RGB

Колірний простір RGB — це додатковий колірний простір, який використовує три основні кольори, червоний, зелений і синій, для створення всіх інших кольорів.Значення кольорів RGB задаються за допомогою трьох 8-бітних цілих чисел, по одному для кожного основного кольору.Це створює діапазон можливих кольорів від 0 (без освітлення) до 255 (повне освітлення).

HSL (відтінок, насиченість, світлота) — це колірний простір, який є більш інтуїтивно зрозумілим способом визначення кольорів, ніж RGB.Значення HSL вказуються за допомогою трьох чисел з плаваючою комою, по одному для кожного компонента.Діапазон можливих значень HSL — від 0 (відтінок відсутній) до 1 (повна насиченість і освітленість).

Перетворення RGB на HSL — це процес перетворення значень кольорів RGB на значення HSL.Формула перетворення RGB у HSL така:

тон = (червоний - зелений) / (червоний + зелений + синій)
насиченість = (синій - зелений) / (синій + зелений + червоний)

Особливості інструменту перетворення кольорів HSL в RGB

  1. Введення HSL (відтінок, насиченість і освітленість): інструмент дозволяє вводити кольори в колірний простір HSL, який базується на трьох характеристиках відтінку, насиченості та освітленості.

  2. Вивід RGB (червоний, зелений, синій): інструмент перетворює кольори HSL у колірний простір RGB, який базується на трьох основних кольорах: червоному, зеленому та синьому.

  3. Попередній перегляд кольору: інструмент зазвичай містить функцію попереднього перегляду кольорів, яка дозволяє побачити представлення вибраного кольору HSL, як він буде відображатися в колірному просторі RGB.

  4. Регульовані повзунки: багато інструментів перетворення кольорів HSL у RGB включають регульовані повзунки або поля введення, які дозволяють точно налаштувати значення кольору HSL, щоб отримати бажаний результат RGB.

  5. Шістнадцятковий вивід: інструмент також може надати кінцевий колір RGB у шістнадцятковому форматі кольорів, який є стандартним представленням кольорів, що використовуються у веб-дизайні та інших цифрових програмах.

  6. Палітра кольорів. Деякі інструменти для перетворення HSL у RGB включають функцію палітри кольорів, яка дозволяє вибирати з діапазону попередньо встановлених кольорів або створювати власні кольори.

  7. Історія кольорів: деякі інструменти також можуть мати функцію історії кольорів, яка дозволяє відстежувати кольори, які ви перетворили, що полегшує використання тих самих кольорів у кількох проектах.

  8. Сумісність із різними колірними просторами. Деякі інструменти перетворення HSL у RGB сумісні з іншими колірними просторами, такими як CMYK (блакитний, пурпуровий, жовтий і чорний) або HSB (відтінок, насиченість і яскравість), що дозволяє конвертувати між цими різними кольорами. моделі також.
