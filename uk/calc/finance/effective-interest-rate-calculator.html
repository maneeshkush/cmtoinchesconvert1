Розрахунок процентної ставки за ефективний період

Отже, ефективна процентна ставка за період дорівнює номінальній річній процентній ставці, поділеній на кількість періодів на рік n:

Ефективна ставка за період  = номінальна річна ставка / n

Приклад 1

Яка процентна ставка за ефективний період для номінальної річної процентної ставки 3% щомісяця?

рішення:

Effective Period Rate = 3% / 12months = 0.03 / 12 = 0.250%

Приклад 2

Яка процентна ставка за ефективний період для номінальної річної процентної ставки 7% щомісяця?

рішення:

Effective Period Rate = 7% / 12months = 0.07 / 12 = 0.583%

Приклад 3

Яка ефективна процентна ставка за період для номінальної річної процентної ставки 9% щомісяця?

рішення:

Effective Period Rate = 9% / 12months = 0.9 / 12 = 0.750%

Розрахунок ефективної річної процентної ставки

Отже, ефективна річна відсоткова ставка дорівнює 1 плюс номінальна відсоткова ставка у відсотках, поділена на кількість періодів нарахування на рік n у ступені n мінус 1.

Effective Rate = (1 +  Nominal Rate /  n)n - 1

Приклад 1

Яка ефективна річна процентна ставка для номінальної річної процентної ставки 3% щомісяця?

рішення:

Effective Rate = (1 + 3% / 12)12 - 1

      = (1 + 0,03 / 12) 12  - 1

      = 0,03042 = 3,042%

Приклад 2

Яка ефективна річна відсоткова ставка для номінальної річної відсоткової ставки 7% щомісяця?

рішення:

Effective Rate = (1 + 7% / 12)12 - 1

      = (1 + 0,07 / 12) 12  - 1

      = 0,07229 = 7,229%

Приклад 3

Яка ефективна річна відсоткова ставка для номінальної річної відсоткової ставки 9% щомісяця?

рішення:

Effective Rate = (1 + 9% / 12)12 - 1

      = (1 + 0,9 / 12) 12  - 1

      = 0,09381 = 9,381%
