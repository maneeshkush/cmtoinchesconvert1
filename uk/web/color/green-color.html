Код зеленого кольору RGB

Зелений шістнадцятковий/RGB код кольору = #00FF00 = 0*65536+255*256+0 = (0,255,0)

ЧЕРВОНИЙ=0, ЗЕЛЕНИЙ=255, СИНІЙ=0
