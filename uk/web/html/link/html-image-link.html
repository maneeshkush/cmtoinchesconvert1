Посилання на зображення HTML

Як зробити зображення посиланням.

Код посилання зображення HTML

<a href="../html-link.htm"><img src="flower.jpg" width="82" height="86" title="White flower" alt="Flower"></a>

Або краще використовувати стилі CSS для визначення ширини та висоти.

<a href="../html-link.htm"><img src="flower.jpg" style="width:82px; height:86px" title="White flower" alt="Flower"></a>

Код створить це посилання:

 

Код складається з таких частин:

  • <a> — це тег посилання.
  • Атрибут href встановлює URL-адресу для посилання.
  • <img> — початковий тег зображення.
  • атрибут src встановлює файл зображення.
  • атрибут title встановлює текст підказки зображення.
  • alt — атрибут альтернативного тексту тегу зображення.
  • атрибут style встановлює за допомогою css ширину та висоту зображення.
  • </a> є кінцевим тегом посилання.
