Посилання для завантаження HTML

Як написати посилання для завантаження в HTML.

Посилання для завантаження — це посилання, яке використовується для завантаження файлу з сервера в каталог браузера на локальному диску.

Код посилання для завантаження записується так:

<a href="test_file.zip" download>Download File</a>

Код створить це посилання:

Завантажити файл

Тестовий файл має бути на веб-сервері.

Код складається з таких частин:

  • <a> — тег посилання.
  • атрибут href встановлює файл для завантаження.
  • Завантажити файл - це текст посилання.
  • </a> є кінцевим тегом посилання.
