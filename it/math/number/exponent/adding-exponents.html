Somma di numeri con esponenti

L'aggiunta di esponenti viene eseguita calcolando prima ciascun esponente e quindi aggiungendo:

an + bm

Esempio:

42 + 25 = 4⋅4+2⋅2⋅2⋅2⋅2 = 16+32 = 48

Sommando le stesse basi b e gli esponenti n:

bn + bn = 2bn

Esempio:

42 + 42 = 2⋅42 = 2⋅4⋅4 = 32

Somma di esponenti negativi

L'aggiunta di esponenti negativi viene eseguita calcolando prima ciascun esponente e quindi aggiungendo:

a-n + b-m = 1/an + 1/bm

Esempio:

4-2 + 2-5 = 1/42 + 1/25 = 1/(4⋅4)+1/(2⋅2⋅2⋅2⋅2) = 1/16+1/32 = 0.09375

Somma di esponenti frazionari

L'aggiunta di esponenti frazionari viene eseguita aumentando prima ciascun esponente e quindi aggiungendo:

an/m + bk/j

Esempio:

33/2 + 25/2 = √(33) + √(25) = √(27) + √(32) = 5.196 + 5.657 = 10.853

 

Sommando le stesse basi b e gli esponenti n/m:

bn/m + bn/m = 2bn/m

Esempio:

42/3 + 42/3 = 2⋅42/3 = 2 ⋅ 3√(42) = 5.04

Somma di variabili con esponenti

L'aggiunta di esponenti viene eseguita calcolando prima ciascun esponente e quindi aggiungendo:

xn + xm

Con stessi esponenti:

xn + xn = 2xn

Esempio:

x2 + x2 = 2x2
