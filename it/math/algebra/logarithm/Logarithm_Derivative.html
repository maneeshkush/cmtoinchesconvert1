Derivata del logaritmo

Quando la funzione logaritmica è data da:

f (x) = logb(x)

La derivata della funzione logaritmica è data da:

f ' (x) = 1 / ( x ln(b) )

x è l'argomento della funzione.

b è la base logaritmica.

ln b è il logaritmo naturale di b.

 

Ad esempio quando:

f (x) = log2(x)

f ' (x) = 1 / ( x ln(2) )
