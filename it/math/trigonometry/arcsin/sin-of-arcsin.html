Qual è l'arcoseno del seno di x?

arcsin( sin x ) = ?

 

Poiché l'arcoseno è la funzione inversa del seno, il seno dell'arcoseno di x è uguale a x:

sin( arcsin x ) = x

x ha valori da -1 a 1:

x∈[-1,1]

 

Poiché il seno è periodico, l'arcoseno del seno di x è uguale a x più 2kπ quando k è intero:

arcsin( sin x ) = x+2kπ
