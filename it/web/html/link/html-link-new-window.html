Collegamento HTML in una nuova finestra

Come aprire un collegamento in una nuova finestra o in una nuova scheda.

Apri un collegamento in una nuova finestra o scheda

Per aprire un collegamento in una nuova finestra/scheda, aggiungi target="_blank" all'interno del tag <a>:

<a href="../html-link.htm" target="_blank">Open page in new window</a>

Nuova finestra o nuova scheda

Non è possibile impostare se il collegamento verrà aperto in una nuova finestra o in una nuova scheda.Dipende dalle impostazioni del browser. 

Apre un collegamento in una nuova finestra con le dimensioni specificate

Per aprire un collegamento in una nuova finestra, aggiungi il comando Javascript onclick="window.open('text-link.htm', 'name','width=600,height=400') all'interno del tag <a>:

<a href="../html-link.htm" target="popup" onclick="window.open('../html-link.htm','name','width=600,height=400')">Open page in new window</a>
