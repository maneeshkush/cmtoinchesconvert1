Colore CSS

Il codice colore può essere uno tra:

Formato esadecimale: #rrggbb

Formato RGB: rgb (rosso, verde, blu)

Formato del nome: nome

Esempio

Per il colore arancione:

Formato esadecimale: #FFA500

Formato RGB: rgb(255,165,0)

Formato del nome: arancione

Impostazione del colore dell'elemento

Per elemento specifico:

<element style="color: code;"></element>

Per tutti gli elementi dello stesso tipo.Inserisci il codice nel tag <style> nella sezione head:

<style>
   element { color: code; }
</style>

Esempio

Impostare il colore del testo in paragrafi su rosso:

Impostazione del colore del paragrafo specifico:

<p style="color: #FF0000;">Some text ...</p>

Visualizzazione:

Qualche testo...

Impostazione del colore di tutti i paragrafi

<style>
   p { color: #FF0000; }
</style>

Impostazione del colore di sfondo dell'elemento

element { background: code; }

Esempio

Impostare il colore di sfondo del paragrafo su rosso:

Impostazione del colore di sfondo del paragrafo specifico:

<p style="background: #FF0000;">Some text ...</p>

Visualizzazione:

Qualche testo...

Impostazione del colore di sfondo di tutti i paragrafi:

<style>
   p { background: #FF0000; }
</style>

Impostazione del colore del bordo dell'elemento

element { border-color: topcode rightcode bottomcode leftcode }

Esempio

Impostare il colore del bordo del paragrafo su rosso, verde, blu e nero:

Impostazione del colore del bordo di un paragrafo specifico:

<p style="border-color: #FF0000 #00FF00 #0000FF #000000; border-style:solid">Some text ...</p>

Visualizzazione:

Qualche testo...

Impostazione del colore del bordo di tutti i paragrafi:

<style>
   p { border-color: #FF0000 #00FF00 #0000FF #000000; }
</style>
