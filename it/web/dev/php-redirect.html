Reindirizzamento PHP

Reindirizzamento PHP dalla pagina all'URL.Reindirizzamento PHP 301.

Questo reindirizzamento PHP dovrebbe restituire il codice di stato della risposta HTTP: 301 Moved Permanently.

I motori di ricerca utilizzano il codice di stato della risposta 301 per trasferire il page rank dal vecchio URL al nuovo URL.

Reindirizzamento dell'intestazione PHP

Sostituisci il codice old-page.php con il codice di reindirizzamento a new-page.php.

vecchia-pagina.php:

<?php
// PHP permanent URL redirection
header("Location: http://www.domain.com/new-page.php", true, 301);
exit();
?>

La vecchia pagina deve avere estensione file .php.

La nuova pagina può essere con qualsiasi estensione.

Esempi di reindirizzamento PHP

Esempio 1

php-test-di-reindirizzamento.php

<?php
// PHP permanent URL redirection test
header("Location: https://www.cmtoinchesconvert.com/it/web/dev/php-redirect.html", true, 301);
exit();
?>

Esempio #2

php-redirect-test.htm

<?php
// PHP permanent URL redirection test
header("Location: https://www.cmtoinchesconvert.com/it/web/dev/php-redirect.html", true, 301);
exit();
?>

 

Il reindirizzamento PHP dal file html php-redirect-test.htm di solito non funziona a causa dell'estensione del file .html, a meno che non sia abilitato nel file .htaccess o httpd.conf:

 

Test di reindirizzamento PHP - File HTML

 

Per abilitare PHP nei file HTML aggiungi questo codice al file .htaccess o httpd.conf:

Addtype application/x-httpd-php .htm .html
