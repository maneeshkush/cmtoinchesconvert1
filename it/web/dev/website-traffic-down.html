Traffico del sito Web in calo

Perché il traffico del mio sito Web sta diminuendo?

Controlla il calendario

I giorni festivi e i fine settimana potrebbero ridurre il traffico.

Il traffico tornerà alla normalità al termine della vacanza.

Confronta con l'anno scorso

Utilizza Google Analytics per visualizzare il grafico delle visite dell'anno scorso.

Controlla se anche le visite sono diminuite un anno fa.

Errore di Google Analytics

L'utilizzo di un vecchio codice di Google Analytics con il file urchin.js potrebbe mostrare gli ultimi 2 giorni con un traffico inferiore rispetto al traffico reale.

Il traffico non è realmente diminuito, ma solo apparentemente.

Problema del server

Prova a navigare nel tuo sito Web, se non riesci ad accedervi, hai un problema con il server Web o il server DNS.

Prova ad accedere al tuo server web e controlla se è attivo.

Controlla l'integrità del tuo database o dei file html.

Usa lo strumento di test del ping per controllare la risposta del tuo server web.

Cerca nuovo problema sul server DNS.Il 9/2012, questo sito Web con molti altri non ha potuto rispondere (vedi: GoDaddy hackerato ).

Il ranking dei risultati di ricerca di Google è diminuito

La maggior parte del traffico dei siti Web proviene dai motori di ricerca e il motore di ricerca principale è Google.

Se la maggior parte delle visite del tuo sito Web è generata da una singola parola chiave, potrebbe essere presa dalla concorrenza.

  • Cerca la parola chiave in Google per determinare se c'è un altro sito web che si trova prima del tuo sito e offre un valore migliore all'utente.
  • Cerca notizie per una modifica dell'algoritmo di ranking di Google.Ad esempio, l'aggiornamento di Google Panda ha danneggiato il traffico di molti siti web.

Sito bannato da Google

L'utilizzo di metodi proibiti per promuovere il tuo sito su Google garantirà che il tuo sito web venga bandito da Google.

Cerca su Google con le tue parole chiave principali e vedi se appare come al solito nei risultati di ricerca.

Se il tuo sito web non appare affatto, dovresti:

  1. Leggi le linee guida per i webmaster di Google e correggi il tuo sito web.
  2. Invia richiesta di riconsiderazione a Google.
