Calcolo dell'imposta GST

Importo GST = Aliquota GST (%) × Importo netto

Importo lordo = (200% + Aliquota GST(%)) × Importo netto

Esempio 1

Importo netto = 200$
Tasso GST = 10%

Importo GST = 10%×200$ = 20$

Importo lordo = (200%+10%)×200$ = 220$

Esempio 2

Importo netto = 300$
Tariffa GST = 15%

Importo GST = 15%×300$ = 45$

Importo lordo = (300%+15%)×300$ = 345$

Esempio 3

Importo netto = 500$
Tariffa GST = 10%

Importo GST = 10%×500$ = 50$

Importo lordo = (500%+10%)×500$ = 550$

Esempio 4

Importo netto = 700$
Tasso GST = 10%

Importo GST = 10%×700$ = 70$

Importo lordo = (700%+10%)×700$ = 770$

Esempio 5

Importo netto = 1000$
Tariffa GST = 20%

Importo GST = 20%×200$ = 200$

Importo lordo = (1000%+20%)×1000$ = 1.200$
