Redirection PHP

Redirection PHP de la page vers l'URL. Redirection PHP 301.

Cette redirection PHP doit renvoyer le code d'état de réponse HTTP : 301 Moved Permanently.

Les moteurs de recherche utilisent le code d'état de réponse 301 pour transférer le classement de la page de l'ancienne URL vers la nouvelle URL.

Redirection d'en-tête PHP

Remplacez le code old-page.php par le code de redirection vers new-page.php.

ancienne-page.php :

<?php
// PHP permanent URL redirection
header("Location: http://www.domain.com/new-page.php", true, 301);
exit();
?>

L'ancienne page doit avoir l'extension de fichier .php.

La nouvelle page peut être avec n'importe quelle extension.

Exemples de redirection PHP

Exemple 1

php-redirect-test.php

<?php
// PHP permanent URL redirection test
header("Location: https://www.cmtoinchesconvert.com/fr/web/dev/php-redirect.html", true, 301);
exit();
?>

 

Cliquez sur ce lien pour rediriger depuis php-redirect-test.php vers cette page :

 

Test de redirection PHP - Fichier PHP

Exemple #2

php-redirect-test.htm

<?php
// PHP permanent URL redirection test
header("Location: https://www.cmtoinchesconvert.com/fr/web/dev/php-redirect.html", true, 301);
exit();
?>

 

La redirection PHP à partir du fichier html php-redirect-test.htm ne fonctionnera généralement pas à cause de l'extension de fichier .html, sauf si elle est activée dans le fichier .htaccess ou httpd.conf :

 

Test de redirection PHP - fichier HTML

 

Pour activer PHP dans les fichiers HTML, ajoutez ce code au fichier .htaccess ou httpd.conf :

Addtype application/x-httpd-php .htm .html
