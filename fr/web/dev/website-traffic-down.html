Trafic du site Web en baisse

Pourquoi le trafic de mon site Web diminue-t-il ?

Consultez le calendrier

Les jours fériés et les week-ends peuvent diminuer votre trafic.

Le trafic reviendra à la normale une fois les vacances terminées.

Comparer avec l'année dernière

Utilisez Google Analytics pour afficher le graphique des visites de l'année dernière.

Vérifiez si les visites ont également chuté il y a un an.

Bogue de Google Analytics

L'utilisation d'un ancien code Google Analytics avec le fichier urchin.js peut afficher les 2 derniers jours avec un trafic inférieur au trafic réel.

Le trafic n'est pas vraiment en baisse, mais il semble seulement en baisse.

Problème de serveur

Essayez de parcourir votre site Web, si vous ne pouvez pas y accéder, vous avez un problème de serveur Web ou de serveur DNS.

Essayez d'accéder à votre serveur Web et vérifiez s'il est actif.

Vérifiez l'intégrité de votre base de données ou de vos fichiers html.

Utilisez l'outil de test ping pour vérifier la réponse de votre serveur Web.

Rechercher un nouveau problème de serveur DNS. Le 9/2012, ce site Web avec beaucoup d'autres n'a pas pu répondre (voir : GoDaddy piraté ).

Le classement des résultats de recherche Google a chuté

La plupart du trafic des sites Web provient des moteurs de recherche et le principal moteur de recherche est Google.

Si la plupart des visites de votre site Web sont générées par un seul mot-clé, il se peut qu'il soit pris par la concurrence.

  • Recherchez le mot-clé dans Google pour déterminer s'il existe un autre site Web qui, s'il est situé avant votre site, offre une meilleure valeur à l'utilisateur.
  • Rechercher des nouvelles pour un changement d'algorithme de classement Google. Par exemple, la mise à jour Google Panda a endommagé le trafic de nombreux sites Web.

Site interdit par Google

L'utilisation de méthodes interdites pour promouvoir votre site dans Google garantira que votre site Web sera interdit par Google.

Recherchez Google avec vos mots-clés principaux et voyez s'ils apparaissent comme d'habitude dans les résultats de recherche.

Si votre site Web n'apparaît pas du tout, vous devriez :

  1. Lisez les lignes directrices de Google pour les webmasters et corrigez votre site Web.
  2. Envoyez une demande de réexamen à Google.
