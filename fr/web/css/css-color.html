Couleur CSS

Le code couleur peut être l'un des suivants :

Format hexadécimal : #rrggbb

Format RVB : RVB (rouge, vert, bleu)

Format du nom : nom

Exemple

Pour la couleur orange :

Format hexadécimal : #FFA500

Format RVB : RVB (255,165,0)

Format du nom : orange

Réglage de la couleur de l'élément

Pour un élément spécifique :

<element style="color: code;"></element>

Pour tous les éléments du même type. Mettez le code dans la balise <style> dans la section head :

<style>
   element { color: code; }
</style>

Exemple

Réglage de la couleur d'un paragraphe spécifique :

<p style="color: #FF0000;">Some text ...</p>

Voir:

Du texte ...

Définir la couleur de tous les paragraphes

<style>
   p { color: #FF0000; }
</style>

Définir la couleur d'arrière-plan de l'élément

element { background: code; }

Exemple

Définir la couleur d'arrière-plan du paragraphe sur rouge :

Définition de la couleur d'arrière-plan d'un paragraphe spécifique :

<p style="background: #FF0000;">Some text ...</p>

Voir:

Du texte ...

Réglage de la couleur d'arrière-plan de tous les paragraphes :

<style>
   p { background: #FF0000; }
</style>

Définir la couleur de la bordure de l'élément

element { border-color: topcode rightcode bottomcode leftcode }

Exemple

Définir la couleur de la bordure du paragraphe sur rouge, vert, bleu et noir :

Définition de la couleur de bordure d'un paragraphe spécifique :

<p style="border-color: #FF0000 #00FF00 #0000FF #000000; border-style:solid">Some text ...</p>

Voir:

Du texte ...

Définition de la couleur de bordure de tous les paragraphes :

<style>
   p { border-color: #FF0000 #00FF00 #0000FF #000000; }
</style>
