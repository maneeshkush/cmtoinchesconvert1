Barre de navigation CSS

Barre de navigation CSS - horizontale et verticale.

Barre de navigation horizontale

Code CSS :

<style>
    #h_nav_bar a { padding:15px; font-weight:bold; float:left; }
    #h_nav_bar a:link { color:#d0d0d0; background-color:#0000a0; }
    #h_nav_bar a:visited { color:#c0c0c0; background-color:#0000a0; }
    #h_nav_bar a:hover { color:#ffffff; background-color:#000060; }
    #h_nav_bar a:active { color:#f0f0f0; background-color:#00ff00; }
</style>

Code HTML:

<div id="h_nav_bar">
    <a href="html-anchor-link.htm">Anchor link</a>
    <a href="html-link-color.htm">Link color</a>
    <a href="../mailto.htm">Email link</a>
    <a href="html-image-link.htm">Image link</a>
    <a href="html-text-link.htm">Text link</a>
</div>

Voir:

Lien d'ancrage Couleur du lien Lien e-mail Lien vers l'image Lien texte

 

 

 

#link_bar a est le style pour tous les états du lien.

#link_bar a:link est le style de lien normal.

#link_bar a : visité est le style du lien visité.

#link_bar a:hover est le style de lien survolé par la souris.

#link_bar a: active est le style de lien lorsqu'il est pressé par la souris.
