Définition arcsinus

La fonction arcsinus est la fonction inverse de y = sin(x).

arcsin(y) = sin-1(y) = x + 2kπ

Pour chaque

k = {...,-2,-1,0,1,2,...}

 

Par exemple, si le sinus de 30° est de 0,5 :

sin(30°) = 0.5

Alors l'arc sinus de 0,5 vaut 30° :

arcsin(0.5) = sin -1 (0.5) = 30°
