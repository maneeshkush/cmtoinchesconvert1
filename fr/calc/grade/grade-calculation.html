Calcul de la note pondérée

La note pondérée est égale à la somme du produit des poids (w) en pourcentage (%) par la note (g) :

Weighted grade = w1×g1+ w2×g2+ w3×g3+...

Lorsque les poids ne sont pas en pourcentage (heures ou points...), il faut aussi diviser par la somme des poids :

Weighted grade = (w1×g1+ w2×g2+ w3×g3+...) / (w1+w2+w3+...)

Exemple 1

Cours de mathématiques à 3 points avec une note de 74.

Cours de biologie de 5 points avec une note de 87.

2 points Cours d'histoire avec une note de 71.

La note moyenne pondérée est calculée par :

Weighted grade =

 = (w1×g1+ w2×g2+ w3×g3) / (w1+w2+w3)

 = (3×74+ 5×87+ 2×71) / (3+5+2) = 79.90

Exemple 2

Cours de mathématiques à 3 points avec une note de 72.

Cours de biologie de 5 points avec une note de 88.

2 points Cours d'histoire avec une note de 70.

La note moyenne pondérée est calculée par :

Weighted grade =

 = (w1×g1+ w2×g2+ w3×g3) / (w1+w2+w3)

 = (3×72+ 5×88+ 2×70) / (3+5+2) = 79.60
