Comment déplacer des fichiers sous Linux/Unix

Fichiers de déplacement Unix/Linux.

 

Déplacer la syntaxe du fichier :

$ mv [options] sourcefiles destdir

 

Déplacez les fichiers main.c def.h vers le répertoire /home/usr/rapid/

$ mv main.c def.h /home/usr/rapid/

 

Déplacer tous les fichiers C du répertoire courant vers le sous-répertoire bak

$ mv *.c bak
