Symbole de l'infini

Le symbole de l'infini est un symbole mathématique qui représente un nombre infiniment grand.

Le symbole de l'infini s'écrit avec le symbole Lemniscate :

∞

Il représente un grand nombre infiniment positif.

Lorsque nous voulons écrire un nombre infiniment négatif, nous devons écrire :

-∞

Lorsque nous voulons écrire un nombre infiniment petit, nous devons écrire :

1/∞

L'infini est-il un nombre réel ?

L'infini n'est pas un nombre. Il ne représente pas un nombre spécifique, mais une quantité infiniment grande.

Règles et propriétés de l'infini

Nom Type de clé
Infini positif ∞
Infini négatif -∞
Différence infinie ∞ - ∞ est indéfini
Zéro produit 0 ⋅ ∞ est indéfini
Quotient infini ∞ / ∞ n'est pas défini
Somme des nombres réels x + ∞ = ∞, pour x ∈ℝ
Produit de nombre positif x ⋅ ∞ = ∞, pour x >0
