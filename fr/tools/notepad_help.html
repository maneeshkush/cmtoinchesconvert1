Aide du bloc-notes

Il y a 2 façons d'enregistrer le texte

  1. Le texte est enregistré chaque fois que vous fermez l'onglet Bloc-notes, dans le cache local du navigateur. Lorsque vous entrez à nouveau dans la page Bloc-notes, le texte réapparaîtra.
  2. Le texte est enregistré/sauvegardé sur le disque dur lorsque vous appuyez sur le bouton Enregistrer. Pour rouvrir le texte du disque dur, appuyez sur le bouton Ouvrir et sélectionnez le fichier texte que vous avez créé.

Paramètres de configuration


Une information important

  • Si le texte de la session précédente est manquant :
    • Si du texte existe, sélectionnez-le et copiez-le avec Ctrl+C et collez-le avec Ctrl+V dans la page Bloc -notes :

    • Copiez le texte du fichier texte mis à jour qui a été généré par l'opération d'enregistrement automatique dans le dossier de téléchargement (le cas échéant).
    • Vérifiez si vous regardez le même ordinateur que celui que vous avez utilisé précédemment.
    • Activer les cookies et l'historique du navigateur.
    • N'utilisez pas le mode privé/incognito du navigateur. Le stockage local du texte sera supprimé par le navigateur lorsque vous fermerez la fenêtre du navigateur.
    • Essayez d'ajouter/supprimer le www de l'URL dans la barre d'adresse du navigateur.
  • Le texte du Bloc-notes ne sera pas enregistré avec la navigation en mode incognito/privé !!!
  • Le texte du notpad enregistré peut être supprimé lorsque vous supprimez votre historique de navigation/cache ou exécutez une application de nettoyage de disque (par exemple, nettoyage de disque Windows / CCleaner) !!!
  • Si le bouton Ouvrir le fichier ne fonctionne pas, veuillez recharger la page et réessayer.
  • Utilisez le Bloc -notes avec la version mise à jour du navigateur. Si vous utilisez Internet Explorer , copiez votre texte dans un navigateur moderne (par exemple Chrome/Edge/Firefox ).
  • Le texte du bloc-notes est automatiquement enregistré dans le cache local du navigateur (non sécurisé).
  • Le texte du bloc-notes peut être enregistré automatiquement (sauvegarde) sur le disque dur, selon la durée d'enregistrement automatique dans le menu Affichage > Préférences .
  • Vous pouvez sauvegarder le texte du Bloc-notes sur le disque dur, en utilisant le bouton Enregistrer ou le menu Fichier > Enregistrer.
  • Pour Mac, utilisez la commande ⌘ au lieu de la touche Ctrl .
  • Pour ouvrir le fichier enregistré, recherchez le fichier dans le dossier Téléchargements.
  • Si les lignes de fond ne défilent pas, masquez les lignes : décochez le menu Affichage > Préférences > Lignes de texte
  • Le bouton ou le menu Enregistrer Fichier > Enregistrer enregistre le fichier dans le dossier Téléchargements . Voir : Où vont les fichiers une fois téléchargés ?
  • Si les lignes de texte ne sont pas visibles, veuillez essayer d'utiliser le navigateur Chrome.
  • Si le correcteur orthographique ne fonctionne pas, essayez de l'activer dans la section Paramètres>Langues de votre navigateur. S'il n'est pas défini, vous pouvez également essayer de définir l'anglais (États-Unis) dans les paramètres de langue de votre navigateur.
