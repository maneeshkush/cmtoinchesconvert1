Comment utiliser le tableau de bord

  • Réglez le temps en minutes et secondes.
  • Définissez les noms des joueurs/équipes.
  • Appuyez sur le bouton Démarrer pour démarrer le jeu.
  • Cliquez sur la zone de texte du score pour définir le score.
  • Appuyez sur le bouton + pour augmenter le score.
  • Appuyez sur le bouton - pour décrémenter le score.
  • Appuyez sur le bouton × pour réinitialiser le score.
